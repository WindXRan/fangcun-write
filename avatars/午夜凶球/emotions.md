# 午夜凶球 情绪模块

## 爽点模式

### 模式1：打脸逆袭
- **触发**：被看不起/被欺负
- **释放**：用实力/智慧反击
- **例子**：李观澜高考700分，全省第一，打脸所有看不起她的人

### 模式2：身份揭露
- **触发**：隐藏身份被发现
- **释放**：震惊+反转
- **例子**：李观澜被发现是被抱错的真千金

### 模式3：亲情爆发
- **触发**：家庭矛盾/误解
- **释放**：和解+温暖
- **例子**：两个家庭最终和解，李观澜成为两个家庭的核心

## 泪点模式

### 模式1：父母之爱
- **触发**：父母为孩子付出
- **释放**：孩子理解父母的苦心
- **例子**：李建国凌晨送饭，李观澜看到父亲的白发

### 模式2：离别场景
- **触发**：分离/告别
- **释放**：不舍+祝福
- **例子**：李观澜去上大学，家人送别

### 模式3：和解时刻
- **触发**：长期矛盾化解
- **释放**：原谅+拥抱
- **例子**：两个家庭最终和解

## 笑点模式

### 模式1：互怼日常
- **触发**：日常互动
- **释放**：毒舌对话
- **例子**："毒妇，我跟你拼了！""人渣，同归于尽吧！"

### 模式2：误会乌龙
- **触发**：信息不对称
- **释放**：真相大白
- **例子**：刘芳误会李观澜和江流早恋，结果发现两人在打架

### 模式3：东北式幽默
- **触发**：日常生活
- **释放**：方言+梗
- **例子**：李建国的"男人不能说自己不行"

## 情绪弧线模板

### 日常型章节
开头(轻松) → 中段(小冲突) → 结尾(温馨)

### 冲突型章节
开头(紧张) → 中段(升级) → 高潮(爆发) → 结尾(解决)

### 高潮型章节
开头(铺垫) → 中段(蓄力) → 高潮(释放) → 结尾(余韵)

### 结局型章节
开头(收束) → 中段(告别) → 结尾(新开始)
