# 去AI味规则库

> 所有分身共享的去AI味知识。写作时自动检查。

## Tier 1 — 必须替换（出现即标记）

### 路标词
- 首先、其次、最后、总而言之、综上所述、值得注意的是
- 一方面...另一方面...
- 让我们、接下来、现在我们
- 与此同时、不仅如此、更重要的是

### 空洞修饰词
- 深入、全面、有效、显著、重要
- 积极、主动、认真、努力
- 不断、持续、进一步

### 模板句式
- "这是一个...的过程"
- "通过...的方式"
- "在...的基础上"
- "为了...的目的"

## Tier 2 — 集群标记（同一段落2+个）

### 高频副词
微微、轻轻、淡淡、缓缓、悄悄、默默、静静、慢慢、渐渐

### 情绪形容词
温暖、感动、幸福、快乐、紧张、焦虑、担心、害怕、愤怒、悲伤、失望、无奈

### 动作描写
点头、摇头、叹气、微笑、皱眉、瞪眼、抿嘴、眨眼

## Tier 3 — 密度标记（超过3%）

### 通用词
问题、情况、方面、时候、事情、东西、地方、样子、一下、一点、一些、一会儿

## 修复策略

### 路标词替换
- "首先...其次...最后" → 删除，直接说
- "与此同时" → "同时" / "也"
- "不仅如此" → "而且" / "还"

### 空洞修饰词删除
- "深入研究" → "研究"
- "有效解决" → "解决"
- "显著提升" → "提升"

### 模板句式替换
- "这是一个...的过程" → 删除"这是一个...的过程"
- "通过...的方式" → 删除"通过...的方式"
