# 大纲

## 基本信息
- **书名**：重生后，我把冷血侯府搅翻天了
- **题材**：古代言情 / 重生 / 宅斗 / 医女
- **预计字数**：50-80万字
- **预计章数**：200-300章

## 卷级结构

### 第一卷：入府（约5万字，20-25章）
- **功能**：铺垫+起步+第一个大爽点
- **核心事件**：沈青禾重生，主动进入侯府，揭开身世
- **起始状态**：重生医女，孤身一人
- **结束状态**：身份初步确认，在侯府站稳脚跟
- **关键节点**：
  - 第1章：重生觉醒
  - 第2章：救治世子亲卫，获得入府机会
  - 第3章：进入侯府，身世初露
  - 第4-6章：王氏暗算，沈青禾反杀
  - 第7-10章：治愈老太太，赢得认可
  - 第11-15章：身世正式揭开
  - 第16-20章：与王氏第一次正面交锋

### 第二卷：立足（约10万字，40-50章）
- **功能**：巩固地位+扩展势力+感情线推进
- **核心事件**：沈青禾用医术在临安城打出名声，与谢昭宁关系升温
- **起始状态**：侯府千金，但根基不稳
- **结束状态**：医术闻名，与世子互生情愫
- **关键节点**：
  - 开设医馆/药铺
  - 治愈达官贵人，积累人脉
  - 与谢昭宁多次合作
  - 沈青蕙再次设局，被彻底打脸

### 第三卷：争斗（约15万字，60-75章）
- **功能**：主线矛盾激化+大爽点
- **核心事件**：王氏联合外部势力反扑，沈青禾与谢昭宁联手应对
- **起始状态**：地位稳固，感情萌芽
- **结束状态**：铲除王氏势力，感情明确
- **关键节点**：
  - 王氏身世秘密揭露
  - 外部势力介入（王氏娘家）
  - 谢昭宁正式表态护妻
  - 侯府权力更迭

### 第四卷：登顶（约10万字，40-50章）
- **功能**：高潮+收尾
- **核心事件**：沈青禾成为侯府主母，医术闻名天下
- **起始状态**：侯府核心人物
- **结束状态**：大团圆
- **关键节点**：
  - 谢昭宁承袭爵位
  - 沈青禾开设医学堂
  - 前世仇人全部清算
  - 大婚+番外

## 节奏设计

### 爽点密度
- 每3-5章一个小爽点（打脸/治病/反杀）
- 每15-20章一个中爽点（身世/感情/权力）
- 每卷结尾一个大爽点（对决/胜利/升级）

### 情绪曲线
- 第一卷：低谷→上升（重生觉醒→站稳脚跟）
- 第二卷：平稳上升（扩展势力→感情萌芽）
- 第三卷：波动→高潮（危机→反击→胜利）
- 第四卷：持续高潮→圆满（登顶→大团圆）

## 伏笔规划

### 全书伏笔
1. 沈青禾的真实身世（第一卷揭开）
2. 王氏的秘密（第三卷揭开）
3. 谢昭宁的调查（贯穿全书）
4. 前世毒杀真相（第二卷查明）

### 单卷伏笔
- 第一卷：香囊麝香、顾嬷嬷的态度、老太太的旧疾
- 第二卷：医馆背后势力、谢昭宁的身世调查
- 第三卷：王氏娘家、外部势力
