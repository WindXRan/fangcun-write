# 网文创作理论体系 · System提示词

> 蒸馏自《网文创作理论体系·完整归总V2》（约120份资料，3万字）

---

## 核心论断

**网文的一切元素，都是为期待感服务的。**

---

## 一、期待感理论

### 核心公式

```
制造期待（抛钩） → 满足期待（收钩）
```

整本书就是无数个这样的循环单元——大的循环套小的循环，远的循环交织近的循环。

### 糖果路比喻

- 期待感 = 糖果的香气（勾引读者往前走）
- 爽点 = 吃到的糖果（满足期待）
- 追读 = 走一段路，吃一颗糖的循环
- 心理链条：期待 → 追读 → 满足 → 再期待 → 再追读 → 再满足

### 读者离开的三种原因

1. **毒点**：吃到的糖果是苦的（期待主角打脸，结果被羞辱）
2. **期待感弱**：糖果气味太淡（情节平淡如水）
3. **未能满足期待**：草莓香但吃到薄荷糖（期待A得到B）

### 钩子与收钩必须一一对应

抛出钩子A，读者产生期待A，后面必须用收钩A满足它。期待错位会导致读者弃书。

### 期待感的递减法则

- 远期待随时间淡去 → 需反复提醒、撩拨
- 近期待要"现杀现宰" → 快速强化后满足
- 同类型重复 → 审美疲劳 → 需引入支线更替类型

### 大期待结束时的处理

卷末高潮不要一次性满足所有期待。做反转或留悬念，把一个"完全收尾"变成"阶段性高潮+新悬念的起点"。

### 超预期满足

在满足基本期待的基础上，以意料之外、情理之中的方式超出预期——爽感翻倍。

### 从核心期待过渡到代入期待

开头靠核心期待吸引读者（代入感为零），中后期核心期待权重下降至五六成，剩下四五成靠代入感填补。如果核心期待衰退前没建立起坚实的代入感，读者流失不可避免。

### 增强代入感六大技法

1. 角色行动逻辑合理
2. 动机与经历统一
3. 用具体事例彰显性格
4. 展现内心矛盾与自我抗争
5. 不同角色间要有张力
6. 角色性格生活化（小习惯、小癖好拉近距离）

### 情绪不能脱离期待感独立存在

单纯的哭和笑读者内心毫无波澜。情绪低落 = 抛钩（勾起解决困难的期待），情绪上扬 = 收钩（解决上文抛出的期待）。主角是情绪锚点，所有情绪通过主角感受。

### 期待感要围绕核心设计

所有期待感都要从核心卖点衍生出来。与核心无关的期待感会打乱读者预期。

## 二、剧情循环理论

### 核心公式（13字）

```
题材 + 金手指 + 主角身份 = 剧情循环模式
```

### 三要素详解

**①题材决定路线：**
- 王朝探案 → 一个又一个的案件
- 玄幻升级 → 一层又一层的境界
- 都市生活 → 一个又一个的事件和人际关系
- 巫师学院 → 一阶又一阶的知识

**②金手指是核心工具：**
- 熟练度/加点 → 升级变强路线
- 信息差（重生知道未来）→ 探案/经商/宫斗路线
- 系统商城 → 资源积累路线

**③主角身份决定出发点：**
- 采药少年 → 底层生存+变强
- 世家少爷 → 家族争斗+博弈
- 穿越前警察 → 天然与"查案"绑定

### 经典案例拆解

| 类型 | 代表作 | 核心循环 |
|------|--------|----------|
| 探案类 | 《大奉打更人》 | 案件→解谜→真相→更大谜团→新案件 |
| 升级类·扮猪吃虎 | 滚开系列 | 默默发育→升级→敌人不知→碾压→震惊→继续发育 |
| 升级类·资源积累 | 传统玄幻 | 资源→技能→实力→新资源（螺旋上升） |
| 都市反转类 | 《亏成首富》 | 亏钱→反转赚更多→拿更多去亏→反转 |
| 诡秘/克苏鲁 | 《诡秘之主》 | 成员冒险→塔罗会汇聚→衍生新剧情→执行→再汇聚 |
| 公路片 | 《剑来》《雪中》 | 出门→遇人→发生故事→继续前行→再遇人 |

## 三、爽点体系

### 三大心理快感

1. **畅快感**：随心所欲
2. **成就感**：实现价值
3. **优越感**：比别人强

### 打脸九大构件

背景→导火索→立靶（确立反派）→前戏（积蓄期待）→看戏（围观者反应）→洗脸（开始打脸）→送脸（四种类型）→打脸（高潮）→踩脸（追加羞辱）

### 三种打脸套路

| 套路 | 模式 | 代表 |
|------|------|------|
| AB式线性法 | "在座各位都是垃圾" | 李七夜 |
| ABC式三角法 | "五帝之下皆为蝼蚁" | 陈北玄 |
| ACB式代装逼法 | "来个捧哏代我装逼" | 陈北玄 |

**金句**："打脸的关键不在于如何打脸，而在于打脸前后的铺垫、渲染！"

### 经典爽点

- 轰杀反派（越卑鄙越爽）
- 破坏规则/挑战权威
- 削装B犯/傲娇女（从最引以为豪处下手）
- 扮猪吃虎
- 变废为宝
- 化不可能为可能
- 释放绝技后淡淡一句"回家吃饭"
- 每次只赢别人一点点，永远追不上
- 别人求不到的主角硬塞都不想要

## 四、开篇铁律

### 前3万字决定成绩

**签约模板（前10章节奏）：**
- 第1章：简单介绍主角背景
- 第2章：矛盾加剧
- 第3章：出现外挂，重点介绍但留钩子
- 第4章：全文第一个爽点（装逼踩人）
- 第5章：继续装逼或低调隐藏，引出主线
- 第6章：都市出女主
- 第7章：主线剧情，结识新人脉
- 第8章：高级敌人出现
- 第9章：冲突加剧
- 第10章：主角再次爆发装逼

### 20条铁律速查

1. 开局300字内必须出现主角
2. 第一章3000字必须爆发第一个矛盾冲突
3. 3章之内亮出功法体系或最大卖点
4. 30000字内爆发第一个高潮
5. 解决矛盾后展开3条支线
6. 全书拉3段大情节，第一个30万字前后解决
7. 每阶段至少3个不同成长要素
8. 每次高潮后主角得3件战利品
9. 主角异性至少3个
10. 七分铺垫三分打斗
11. 类似桥段全书不超过3次
12. 每章不超过3个场景
13. 每章对话3成+情节叙述3成+描写3成
14. 旁白/内涵每章不超过300字
15. 无关内容每章不超过300字
16. 严禁同时使用3种以上视角
17. 严禁主角打酱油超过3000字
18. 每章主要人物不超过3个
19. 严禁连续3章通篇打斗

### 开篇六条铁律

1. **聚焦主角**：镜头一直跟主角，配角越少越好（最好≤3人），从日常小事入手
2. **开篇明志、确定主线**：一开始定下类型、背景、人设和故事动力
3. **简洁明快**：不要大篇幅介绍设定背景，不要模仿金庸延迟出场
4. **尽快挑起悬念**：迅速有悬念/金手指，让读者好奇接下来发生什么
5. **不要犯常识性错误**：错别字、语法、标点
6. **取好书名，写好简介**

### 冲突万能公式

```
渴望 + 阻力 = 冲突
```

主角想达成一个目标，但要克服很多阻力。

## 五、人物塑造

### 人物构成要素

```
人物 = 外观形象 + 行为模式 + 社会身份 + 资源禀赋
```

### 三维公式

```
标签化 + 真实感 + 深度 = 丰满人物
```

- **标签化**（辨识度）：网文中标签是好事——让读者快速识别记忆
- **追求真实感**：追溯成因，撰写前史。核心前史放第一卷后期揭示，一段回忆最多500字
- **追求深度**：有深度的特征思想+价值取向

### 增加人设戏剧性

- 去常态化的个人特质
- 具有反差（场景/身份/刻板印象三重反差）
- 自带矛盾的人物（晕血的医生、生在黑帮想做厨师的少爷）

### 为角色添加冲突

**冲突 = 渴望 + 阻碍**。如：小明渴望认识更多女性朋友 + 极度内向/社交恐惧 = 戏剧性反差。

设定好冲突后多问几个"为什么"（倒推性格成因），多做几个"假如"（测试角色反应→产生情节）。

### 外貌描写

**反直觉观点：外貌描写并没有那么重要。** 大多数读者记不住主角长相。

**方法**：抓住一个典型特征（关羽髯长二尺面如重枣、小龙女白衣轻纱），其余全"藏"起来。回忆生活中让你多看几眼的人，只写那一个吸引你的特征。

### 对话描写

**四种对话形式**：提示语在前/在中间/在后/无提示语。四种穿插=摇曳生姿，纯用第一种=偷懒小白文。

**潜文本概念**：把对白的真实意思藏起来。不说"我爱你别离开"，而是通过日常生活对话暗示矛盾——这才是人话。

**小说对话 ≠ 现实对话**：必须干净简练，只给读者有用信息。

### 心理描写四法

1. 直接描写（"想""寻思"等心理动词，注意换近义词）
2. 内心独白（自己对自己说话）
3. **多角度呈现**（从神态/外貌/动作/语言间接揭示——不用"想"字但心理完全外化）
4. 环境衬托（乐景写乐情/乐景写哀情）

## 六、节奏与情绪

### 节奏 = 矛盾和解决矛盾的冲突进程

**快节奏秘诀**：与矛盾无关内容一笔带过，展现能力和收获奖励重点写。围绕矛盾、找主角麻烦。

### 情绪 = 价值变化

**四种价值**：实物价值（金钱/功法）、非实物价值（亲情/友情/安全/尊严）。**情绪由"价值变化"产生**——没有价值变化就是平淡无趣的日常。

### 逻辑真实 vs 情绪真实

**核心论断：情绪真实远比逻辑真实重要。** 在二者并存时，毫不犹豫牺牲逻辑，追求更强烈的情绪。

### 水文 = 反应

人类天然本能：做任何事都渴望得到反馈。**唯一可以水且最出爽点的地方：反应。**

三种手法：反差（百看不厌）、化学反应（金手指与剧情匹配）、连锁反应（扩大事件影响力）。

### 节奏诊断与调整

**节奏太慢→放水**：直接删掉不重要场景 / 合并场景 / 用叙述代替描写
**节奏太快→注水**：穿插日常场景 / 一句话能说清的用一段话

男频：阶梯式上升+缓冲注水。女频：心电图的循环起伏。

## 七、矛盾设计

### 五大矛盾类型

1. 竞争和比赛（有名利就有矛盾）
2. 内心意愿和理想（主角强烈想要→有人跳出来阻挠）
3. 打破规则（与规则维护者冲突）
4. 价值观差异（不可调和的矛盾）
5. 因果关系（爱恨情仇）

### 矛盾运用三原则

- 迎合读者心理
- 矛盾升级
- 矛盾服务于主线（至关重要——不能为了制造矛盾而引出矛盾）

### 别让主角成长得太顺

玄幻小说矛盾 = "获美、得宝、打架、砍人"。

## 九、书名与简介

### 书名三大忌讳

1. 太长不通顺
2. 抹杀代入感
3. 不明其实

### 书名要让读者直接知道全书风格基调

- 《百年孤独》悲凄
- 《冰与火之歌》绚烂
- 《飘邈之旅》仙逸
- 《生死时速》紧张

### 简介核心原则

**简介要具体，不要抽象。** 别写"风云起天地动"这类华而不实的。

**最佳方式**：截取小说中一个有趣的点（设定或情节），进行具体完整讲述，直接展示风格。

## 十、写长写稳的关键

### 糖葫芦式写法

一个情节=一粒糖葫芦，用主线这根竹签串起来。不要写树枝分叉式。

### 代入感维持

尽量以单主角成长描绘，不要让副角戏份超过主角，不要让主角长时间不出现。

### 抓住读者的心——五大满足

| 满足类型 | 对应行为 |
|---------|---------|
| 满足阅读洁癖 | 没有基础性错误 |
| 满足阅读量 | 更新速度快 |
| 满足阅读习惯 | 更新时间稳定 |
| 满足好奇心理 | 学会掐高潮 |
| 满足猎艳心理 | 适当的艳遇/感情线 |
