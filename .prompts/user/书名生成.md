# 书名生成提示词

## 核心原则

**书名要让读者直接知道全书风格基调。**

## 三大忌讳

1. 太长不通顺
2. 抹杀代入感（如"我的认知世界"）
3. 不明所以

## 好书名示例

| 书名 | 风格 |
|------|------|
| 《百年孤独》 | 悲凄 |
| 《冰与火之歌》 | 绚烂 |
| 《飘邈之旅》 | 仙逸 |
| 《生死时速》 | 紧张 |

## 生成要求

根据以下信息生成书名：
- **题材**：{题材}
- **核心爽点**：{核心爽点}
- **目标平台**：{平台}
- **情绪基调**：{基调}

## 书名风格参考

| 类型 | 特点 | 示例 |
|------|------|------|
| 直白型 | 直接点明核心设定 | 《我有一座恐怖屋》 |
| 悬念型 | 制造疑问引好奇 | 《诡秘之主》 |
| 情绪型 | 传递情绪氛围 | 《怒晴湘西》 |
| 身份型 | 突出主角特殊身份 | 《赘婿》 |
| 反差型 | 制造认知冲突 | 《这个修仙太难了》 |

## 输出格式

生成5个备选书名，每个附带简短说明：
1. 书名 - 说明为什么好
2. ...
