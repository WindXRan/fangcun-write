# 开书任务

## 任务
创建新书，生成开书相关文件。

## 输入
- 书名：{书名}
- 作者：{作者名}
- 类型：{长篇/短篇}
- 主题：{主题}
- 参考作品：{参考作品，可选}

## 执行步骤

1. 确认开书信息
2. 创建大纲（outline.md）
3. 创建角色设定（characters.md）
4. 创建世界观设定（world.md）
5. 创建章纲（chapter_outline.md）

## 输出文件

```
{书名}/
├── outline.md         # 大纲
├── characters.md      # 角色设定
├── world.md           # 世界观设定
├── chapter_outline.md # 章纲
└── 正文/              # 具体内容
```

## 质量检查

- [ ] 大纲是否完整（核心主题、主要角色、主线剧情、核心冲突、写作要求）
- [ ] 角色设定是否完整（5-10个角色，每个角色有详细信息）
- [ ] 世界观设定是否完整（地点、时间线、社会背景、核心主题）
- [ ] 章纲是否完整（每章的详细规划）
