
## 期待感——网文的根本

**核心论断：网文的一切元素，都是为期待感服务的。**

### 什么是期待感？

读者内心深处有一个东西在吊着他——想知道：
- 主角能不能打赢
- 误会何时解开
- 身份何时暴露

这一个又一个"想知道"就是期待感。

### 糖果路比喻

- 期待感 = 糖果的香气（勾引读者往前走）
- 爽点 = 吃到的糖果（满足期待）
- 追读 = 走一段路，吃一颗糖的循环

**心理链条：** 期待 → 追读 → 满足 → 再期待 → 再追读 → 再满足

### 核心公式

```
制造期待（抛钩） → 满足期待（收钩）
```

整本书就是无数个这样的循环单元——大的循环套小的循环，远的循环交织近的循环。

## 爽点体系

### 三大心理快感

1. **畅快感**：随心所欲
2. **成就感**：实现价值
3. **优越感**：比别人强

### 打脸九大构件

背景→导火索→立靶→前戏→看戏→洗脸→送脸→打脸→踩脸

### 经典爽点

- 轰杀反派（越卑鄙越爽）
- 破坏规则/挑战权威
- 扮猪吃虎
- 变废为宝
- 化不可能为可能
- 隐藏实力曝光
- 身份反转
