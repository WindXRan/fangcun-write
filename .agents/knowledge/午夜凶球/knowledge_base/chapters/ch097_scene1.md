# 第97章 - 场景1：齐奶奶撮合顾伟峰和小张

犀利的眼光如影随形，顾伟峰感觉如芒在背，他终于忍不住回头看向母亲，"妈，您到底想说什么？"

"儿子，你觉得小张咋样？"，齐奶奶再次问出了这个问题。

"挺好的，刚才那种情况都没打我。"，顾伟峰答道，还谢谢我。

有戏！

齐奶奶心里一动，"我听说小张要离婚了……"

未尽之意两人都懂。

顾伟峰仔仔细细的打量了一番亲妈，得出结论，"妈，您年轻时候就是我的榜样。工作的时候多次获得过先进工人的表彰，获得大家一致认可的'铁娘子'称号。"

"说这个干什么？你知道我想听啥。"，亲奶奶打断他。

"我就想说您年纪大了依然拥有如此气魄推陈出新，人家婚还没离完您就惦记上了您觉得合适吗？您这是让我去当小三儿？您看张壮士那个英雄气概，发起火来咱娘俩绑在一起都打不过吧。"，顾伟峰随便找了个理由搪塞齐奶奶，就是离完了他也不同意。

也不是没结过婚，结过了就知道，不过如此，哪有自己过的自在，想起那几年鸡飞狗跳的婚姻生活他就头疼。

他可一点儿不希望他妈看见个差不多的就帮他惦记上。张女士固然很好，可他更爱自由。

"小张也不是精神病逮谁打谁，我看她平时为人处世是个正派的，俩人要是一条心的话有这么个老婆你就偷着乐吧。我这不就是让你先做个准备吗。"，这性格出去遇见什么事儿你就知道了，那才提气呢，不先打算着还能轮的上你？

这倒是，顾伟峰点头认可，不过他心意已定，多好都跟她没关系。

"妈，您说的对，但是我不想结婚，谈个恋爱还行，谈恋爱出门遇事儿也提气。您看成吗？您同意我明天就去追她。"

成个屁！

一切不以结婚为目的的恋爱行为都是耍流氓。

齐奶奶白了儿子一眼，她还不明白儿子的套路吗？看来他是铁了心做孤家寡人了。

算了，他爱怎么地怎么地，真是皇帝不急太监急，她也不想那些有的没的耽误人家小张了，人家离了婚想不想再婚还不一定呢。

"哎呀，我不管了，随你的心意吧，我也不劝了。"，齐奶奶背着手离开，背影有些萧索。

顾伟峰有些内疚，这个愿望他真满足不了母亲。

"小杜，晚上炖个肘子。"

母亲洪亮的声音从厨房传来。

哦，脑补多了。

顾伟峰心里的内疚迅速的飞走了。
