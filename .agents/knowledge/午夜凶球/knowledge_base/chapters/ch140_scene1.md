# 第140章 - 场景1：门口尬聊

李建国和李卫东站在门口一起迎宾，因为时间尚早，来的人还不算太多，两人相对无言的时间比较多，气氛一时有些尴尬。

按说李建国也是个健谈的人，但跟李卫东他真不知道说啥。

咋起话头啊？

你离婚以后开心吗？这是人问出来的话吗？

跟那个女的一起过没有啊？人家隐私你瞎问多尴尬。

孩子最近学习紧张不？他也不咋管啊。

算了，以老人问候身体开头吧。

"那个，你家大姨身体最近还行啊，刚才我看挺有精神头的。"

"嗯，挺好的。"，就是想孩子，知道他要来一定要跟来看看李昭。礼尚往来，李卫东也礼貌回问，"你家大姨身体也……"

李卫东闭上嘴，忘了李建国母亲早就去世了的事儿了。

你奶奶个罗圈屁的。

"去世好多年了，哈哈。"，李建国皮笑肉不笑，不会唠嗑不唠也行。

"我这记性。对不起啊。"，李卫东讪讪。

"没事儿。"，你别说话就行。

气氛更尴尬了。

哎呀，终于来人了，不用在这儿尬聊了。

"哎，老宋，够意思，来真早啊。"，李建国开心的迎了过去。

本来正闷头走路的老宋抬头，看着大老远迎过来的李建国一脸懵圈，啥情况，李建国咱俩不好几年不说话了吗？！你家办事情啊？也没人通知我啊？

本来准备去市场买点儿菜的老宋稀里糊涂的被李建国送进饭店包间，老宋面上笑呵呵，心里暗骂，李建国，还他妈是你老小子有心机啊，在这儿堵我，今天这钱我是非掏不可了是吧。踏马的，我家孩子上大学我都没通知你！

李建国突然忙碌了起来，李卫东就在一边静静站着，觉得这样倒很好，气氛比之前自然多了。

"要不你先进屋，我给安排个地方。"，李建国抽空问他。

李卫东摇摇头，他想想看，李观澜每天挂在嘴边，有点儿什么好东西都想着他，惦记他。这样父亲是怎么做的，他想学学。
