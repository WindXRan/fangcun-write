# 第101章 - 场景3：早市遇摊主父女

“哎，老哥，来了啊。”

李观澜就看她爸离那个摊位还有老远呢摊主就迎上来了。

她爸看样子跟人家也很是熟稔，“嗯呢，早上起来没啥事儿出来溜达溜达。”

“这是你姑娘啊，长得真漂亮，就是不咋像你，像嫂子吧？”，摊主啧啧赞叹道，这姑娘长得真好，罕见的好看。

这真是哪壶不开提哪壶，好在李建国早已经接受现实，并且发现除了多了个孩子也没影响啥，要不就这句话他就得翻脸。

“嗯，是。”，李建国随意应着，“我买点儿青菜，你今天都有啥啊，有好的给我来点儿。”

“好嘞。”，摊主的倒骑驴上依旧盖着破棉被，他今天的手轻柔了许多，轻轻掀起一个被角。

但他的动作还是惊扰到被子底下的人，小姑娘缩缩脖子睁开了眼睛。

“你这是给孩子带来了啊？”，李建国眼神儿好，一眼就看到了被子下面的小姑娘。小姑娘大概七八岁，脸色带着些病态的苍白。

“嗯呢，她妈有事儿今天没在家，她非得跟我出来，我就带出来了。”，摊主眼神温柔的摸摸孩子脑袋，“上回你问爸爸的事儿还记得不啦？这个就是之前帮过咱们的人，你叫大爷就行。”

小姑娘眼神儿怯怯的，但听说是曾经帮过她们的人还是鼓起勇气直视着眼前这个陌生人，“大爷好，谢谢您以前帮助我们家。”

“诶呀，这小姑娘可真招人稀罕，不客气啊，都举手之劳。”，李建国也过去想摸摸她的头，懂礼貌的小孩儿可讨人喜欢了。

隔的有点儿远，李建国够着费劲，小姑娘往他那边儿挪了挪，把头凑过去让他摸。

这把李建国稀罕的，赶紧掏兜儿。

“不要啊，老哥。”摊主赶紧摆手。

“没出正月还是年，第一回见孩子应该的。”

“那你这么说我也得给大侄女点儿。”，摊主也开始翻他小破兜儿。

“行行，那不支吧了，你赶紧给孩子捂严实喽，还挺冷的呢。”

他们两个在那儿客气的热火朝天，李观澜在玩儿小孩儿。

她用被把孩子裹严实了，只露出个头，两双大眼睛对视着卡巴卡巴。

“姐姐，你可真好看，长大我也准备长这样儿。”

小女孩儿的童言童语逗笑了李观澜，“不行，你长大得比我好看。我小时候没有准备还长这样儿呢，你都有准备了长大肯定比我好看。”

小女孩儿歪头想了想，“也是，那我长大比你好看。”

“行，就这么定了啊。来，拉钩。”，李观澜伸出小拇指。

小女孩儿由于对自己未来的长相十分满意，开心的张开了缺了两颗牙的嘴巴，小拇指伸出和李观澜的勾在一起。

“拉钩，上吊一百年不许变。”

盖章，结契完成。

看着笑得合不拢嘴的闺女，摊主擦了擦湿润的眼角，“大哥，要不人说龙生龙凤生凤呢，这孩子跟你一样儿，仁义。”

李建国面上笑着点头，心里合计哎呀，你可得了吧，看的也不准，别瞎看了。
