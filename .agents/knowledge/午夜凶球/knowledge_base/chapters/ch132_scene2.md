# 第132章 - 场景2：各方反应

"喂，老班，我700分！"

"单科呢？哦哦哦，好好好。"，老班的声音激动狂喜，"知道了。"

'啪'，老班的电话也没有丝毫拖泥带水的挂了，还得问别人呢，没时间夸你了，等我闲下来的的。

"萱萱，700，不知道第几呢，哈哈哈哈哈哈，好的。"

李萱挂了电话，对着笑得眉目舒展的张自珍激动道，"妈，好好考了700。"

张自珍摸摸她的头，"知道了，你怎么比她还激动。"

李萱想，大概是因为，在她心里，好好就应该这样一路熠熠生辉的走下去才对，她本就该耀眼夺目。

李昭在一旁张大了嘴巴，脑子里只剩一个想法: 卧槽，女魔头这么牛逼，难怪看着我写作业的时候骂的我狗血淋头。

李观澜的电话铃声简直是无缝衔接，刚挂了李萱的电话谢玉露的电话也进来了。

"观澜，成绩出来没？"

"700，哈哈，姐牛逼不？"，李观澜最初的激动过去，现在情绪已经稳定下来了，还有闲心自吹自擂。

"牛哇！"，谢玉露挂了电话开心的在屋里转圈圈，闺蜜考了700分她与有荣焉到处炫耀。

"谢飞星，你知道吗，观澜考了700分！"

谢飞星写签字的手一顿，"嗯，这下知道了。"

这喇叭花还挺厉害的，他挑挑眉挂断了电话。

"林霆，你知道吗，观澜她，哦，你知道了，那拜拜。"

谢玉露最后炫无可炫想起了当初给她打电话的两个同学。

"你们知道吗，我和林霆的小宝贝高考考了700分，是700分，你们知道700分什么概念吗？让你对着书抄你都抄不上700！"

切，我有鸡毛大仙儿加持我也能考700，考得好就考得好呗，呜呜呜呜，谢玉露你特么最后两句怎么还人身攻击。

李卫东挂了电话露出了最近的第一个舒心笑容，孙老太追问，"考多少分啊，笑这样儿。"

"700。"，李卫东答得简练。

孙老太听了十分震惊，知道她学习好，不知道她学习这么好啊！难怪我干不过她，这个流氓有文化！

那边林秋敏也接到家里各路亲戚的问候，首当其冲的就是刘淑芬。

"多少分儿，700啊，那考的还行。"，刘淑芬淡定的挂了电话，对着凉亭里一群纳凉的老头老太太淡然说道，"考700。"

凉亭众人听得十分艳羡，有人低声道，"我的妈呀，这刘淑芬的外孙女这么霸道啊，难怪她平时那么能得瑟呢。"

她邻座看了她一眼，刘淑芬嘚瑟那完全就是因为她是那样人，她外孙女学习好跟她嘚瑟有啥关系，你就是单纯想说讲究她吧。

"淑芬，你外孙女真给你长脸啊。"

"还行吧。"，刘淑芬摇起大蒲扇，一脸不以为然，"这还有点儿没发挥好呢，原来我以为她能考760。"

众人无语，满分才750那10分是看她好看给她加的啊，真是想捧她刘淑芬的臭脚都没法儿捧。

李建国这边问候的主要是朋友，李建国嘚瑟的情绪已经从电话线传递过去了，"哎呀，孩子考这成绩我也没想到，我都没想这事儿，她平时在家也不学习，没成想还挺给我长脸，哈哈哈哈哈哈哈哈。"

李建国朋友撂了电话跟媳妇儿感叹，"你说这人没哪儿看去啊，当初老李家那孩子我合计混好了也就是个理发一条街总瓢把子，谁能寻思她高考能考700分。"

"多少分儿？"，朋友媳妇震惊的面膜都掉了吗那孩子她就见过两回，记忆还停留在那五颜六色的造型上，记忆里她就没看清楚过那孩子长啥样儿，那大头帘子给脸遮去二分之一，她高考考700分儿？

她回头看向在沙发上玩儿手机玩的入迷的儿子，要不回头带孩子去做个造型？

电话陆陆续续一直持续到傍晚，李建国和林秋敏总算有时间歇口气儿了，今天才发现接电话是个体力活儿。

"总觉得忘点啥呢？"，李建国一拍头，"忘了问问江流打多少分儿了。"

话音刚落，江青山的电话打了进来:

"好好考多少分啊，700啊，700好啊，江流，江流712哈哈哈哈哈哈哈哈。"
