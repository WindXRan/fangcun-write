# 第126章 - 场景2：朱文静威胁孙老太被反击

过了两天，李卫东那边一直没有消息，朱文静有些着急，干脆到李家去堵他。

小邓一如既往的用家里没人的理由搪塞她，这次朱文静也没了之前的好态度，似笑非笑的塞给她一张照片。

"还是让你们家老太太看看再说吧！"

小邓瞄了一眼，吓了一大跳，把照片扣在身上生怕露出一点缝隙。

"奶奶，这个，那个，唉，你自己看吧。"，小邓把照片往孙老太手里一放。

什么事儿这么慌张，朱文静撵走没有啊？年纪小就是不遭不住事儿，孙老太随意的瞥了眼照片，瞬间瞪大双眼，差点儿从沙发上滑下来。

"她人呢？"

"在门口。"

"让她进来，快点儿。"

孙老太后悔啊，自己一眼没看住，儿子就被那个煞星缠住了，他糊涂啊！怪不得这两天失魂落魄的，回家就在书房窝着，有时候一天也说不上一句话。

朱文静不急不缓的走进来，看见孙老太还带着一丝笑意，"孙阿姨，今天在家啊？"

孙老太也没不好意思，表情带着防备道，"你到底想干什么？"

朱文静坐到她旁边，拉住孙老太的手，"孙阿姨，你看李哥现在自己一个人，家里没个人照顾也不像话。我呢，恰好也是单身，我们又认识这么多年，再合适不过了。"

朱文静这话孙老太完全不赞同，呸，哪儿合适，你一根头发丝儿都配不上我儿子。

朱文静看到孙老太的表情也不恼，从她手里抽出照片，指给她看，"孙阿姨，你说我要是闹到李哥单位去，说他假借帮扶报恩的名义强迫我，会不会对他有什么影响啊。就算是没有影响，这照片要是满天飞也不好看是不是。"

她也不怕话说的太绝，以后真在一起生活，她再小意温存，早晚能拢住李卫东的心。

孙老太想想那个画面头皮都要炸，刚想说话，李卫东的声音便传了过来，他下班刚推开门就听到朱文静威胁他母亲的谈话。

"你去试试看就知道好看不好看了。"

"李哥，你想清楚了吗？"，朱文静不可置信的看着他，弹了弹手中的照片，"这种照片，我可有很多张呢。"

"想清楚了。"，李卫东思索了几息，平静的回答道。

"那就好，那……"，朱文静期待的声音传过来。

"你告诉去吧。"，李卫东无所谓道。

"……你说什么？"，朱文静怀疑自己听错了，李卫东这么看重脸面的人会说出这种话，"你不怕对你声誉有影响？"

"那咋啦？"，李卫东说出了这句话才发现，原来这话说起来有一种不管他人死活的爽感，"你爱上哪儿告上哪儿告，你也没有证据。"

"那照片……"

"照片里我不睡着了吗？我怀疑是你非礼我，你长那么丑，我长这么帅，你对我有觊觎之心完全可能啊，反正我是看不上你。"

朱文静快疯了，李卫东这话是人能想出来的吗？

"我再给你最后一次机会……"，朱文静强撑着让自己看起来淡定些，她意识到自己做了个最差的选择。

那就是，如果李卫东不那么要脸了，那么它的要挟对他将毫无威慑力。单位那边可能会因此有些影响，但绝对不致命。

"我不同意。"，李卫东毫不畏缩直视着她，"看在陈刚的份上，我也给你最后一次机会，你把照片销毁了，当没这个事儿，我就当这个事儿没发生过，以后我们就当不认识。"

朱文静的心落地了，心里暗自好笑，说来说去你不是还是害怕，"我也不同意……"

"我报警了。"，李卫东淡淡道。

朱文静的笑意僵在脸上。

"你说什么？"

"我说那天我报警了。"，李卫东心里涌上一股快意，"检测到我身体里确实有药物残留，虽然因为我没有实际损失不予立案，但报案过程已经留痕了，你现在的行为已经可以确认为胁迫，你确定要这么做吗？"

朱文静脸色煞白，脚一圈瘫倒在沙发上。

完了，全完了。
