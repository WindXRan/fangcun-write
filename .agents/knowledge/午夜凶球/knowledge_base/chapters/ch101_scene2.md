# 第101章 - 场景2：父女早起去早市

过了没多久，外面响起了悉悉索索的声音。

李建国起来了，他琢磨着去趟早市。怕担心打扰那娘俩睡觉，他灯也没开，摸黑穿衣服。

“爸，你干啥去？”

幽幽的声音自背后响起。

李建国一回头，吓得心脏都骤停。身后李观澜一身白色睡裙，披头散发的站在他身后。

“我的妈呀！”，李建国非常不男子汉的拍拍自己的小心脏，“小犊子你吓死我得了。”

“那多不好，私房钱藏哪儿还没告诉我呢。”，李观澜张开手臂，像只蝴蝶一样在屋里飘来飘去，配上她的小白脸效果非常惊悚。

李建国一闭眼，转身往门口走，“我去早市，你自个儿在家玩儿吧，别吓唬你妈啊，她胆儿小。”

“我也去。”，反正也睡不着了，出去溜达溜达。

“不睡回笼觉了？”，平时上学时叫起来这个费劲，放假让她睡她还不睡了，真没整。

“睡，一会儿溜达回来睡。”，李观澜随便套了几件衣服就往外走。
