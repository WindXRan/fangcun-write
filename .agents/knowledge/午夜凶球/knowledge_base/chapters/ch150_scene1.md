# 第150章 - 场景1：谢玉露升学宴·主持人宣读"重生小说"

"还得是大城市节目多啊。"，李观澜给林秋敏和李建国打电话聊八卦。

"那可真是，我活了四十多年在咱这儿都没听过这样式儿的。"，林秋敏听着都觉得惊心动魄，"露露还有一场吧，她那不能出啥问题吧。"

"不能。"，李观澜笃定，"露露家里没有乱七八糟的事儿，比萱萱这场还消停呢。"

萱萱这场还有仨老太暗流涌动，李卫东争风吃醋，李建国携美逃跑，刘佳佳贼心不死等娱乐项目呢，谢玉露那估计风平浪静的，她家应该就是纯显摆。

太好了，这场参加完就没有了，去吃个饭就可以去旅游啦~

说早了！

谢玉露这场升学宴她就不应该来。

倒是没有什么狗血的事发生，但是提前没告诉我还有煽情环节啊！

感谢天感谢地，感谢父母感谢老师就算了，你感谢我干嘛？

还写这么长一段！

"这次转学改写了谢家二小姐的人生轨迹，在那个地方，她遇到了她人生中的导师，灵魂伙伴……"，主持人声情并茂，说到这儿停顿了一下。

一束光'chua'的投射到李观澜的位置，给足大家看到谢家幼女人生导师的时间。

面对全场四面八方发射过来的好奇目光，李观澜脚趾抓地。

谢玉露，你们家怎么恩将仇报！

"看到二小姐不思进取的模样，她心痛，她愤怒。放弃自己是一种不负责任的表现，是思想不成熟的行为！她一定要带这位新来的可爱同学共同进步！"

别看了，不是我，我没那么高尚，我开始的目的纯为钱！

"她灵活机变，因材施教。与好友连夜编撰了一套适合新同学学习的教材。"

有一束光打下来，刚才还呲个大牙笑的徐楠楠闭上了嘴。这高档酒店射灯这么多吗？！

"我给大家念一下。"，光线回到台上，主持人不知道从哪儿变出来一本书，封皮很简单，白纸上打印着标题。虽然书很简陋，但显然谢玉露有好好保管，最外面她包上了透明书皮，整本书看起来崭新如初，连个卷边也没有。

亚埋喽！（住手！）

西兹卡尼！（安静！）

乌鲁赛一！（吵死了，住嘴！）

李观澜和徐楠楠惊恐的尔康手。

"秋雨—重生……"，然而遗憾的是主持人并没有接收到她们的信号，用堪比央视主持人般标准的磁性声音读出了里面的内容。

随着主持人的声音将故事娓娓道来，五彩的光线再次打到两人头上。

李观澜和徐楠楠阵亡。

谢玉露，你们家的钱是真难拿啊，死了都要挖出来鞭尸！

终于把这个流程熬过去，两人好像生了一场大病，面无血色。
