# 第129章 - 场景2：谢飞星配合李观澜演戏

谢飞星和警察同时到达，运营小姐姐因为职业的敏锐度，第一时间开始了摄像。

老太太振振有词，"警察同志，她们讹人，撞了我不说，还要我赔他们钱。"

警察皱眉看着老太太，又是她，这老太太他们也太熟了，经常碰瓷儿，不过因为年纪大也不能把她怎么样。

"警察叔叔，我们没有，我们说送奶奶去医院她不肯。"，李观澜她们齐刷刷的变了态度。

"真的，她们还有一个装晕的。"，老太太指向谢玉露。

警察看过去，六个人都站的笔直。这老太太，真能胡说八道。

老太太气啊，撩起裤腿，上面还有红印，"不说别的，这个总是你们压的吧！"

"唉～你说是就是吧！"，李观澜叹气，"你就说多少钱吧，我们赔就是了。也别给警察叔叔添麻烦了，他们一天天的上班多辛苦。"

"是啊，是啊。"，好友团纷纷点头赞同，林霆甚至已经开始掏兜儿了。

警察小吴刚上班没两年，正是正义爆棚的时候，听李观澜这么一说更觉得不能让好人吃亏，"调监控吧！"

为了方便碰瓷儿，监控让老太太砸坏了，她欲哭无泪，真说不清了。

"别较真儿了，奶奶一定也是有她的苦衷。奶奶，别在地上坐着了，太凉对身体不好，我们给你钱就是了。"，李观澜过去扶老太太起来。

地上不凉，我心凉！

老太太这么多年纵横江湖，就没有有苦说不出的时候，今天算我被鹰啄了眼，这钱我不要了！

她用力推开李观澜，你给我起开。

李观澜蹲着重心有些不稳，直接摔到了地上。她一坐下就感觉完了，谁喝的袋装饮料没喝完就扔地上了，有水顺着裤子底下流出来了。

老太太幸灾乐祸的看着李观澜，该！

嘿！这老太太，不吓唬吓唬你让你长长记性还不行了。

"我的孩子！"，李观澜捂住肚子，表情痛苦。

林霆，江流，谢玉露，徐楠楠，李萱: 这么牛逼，孩子说来就来？谁的啊？

谢飞星顿住想要扶起李观澜的手，但已经晚了，手已经被李观澜抓住。

"我们的孩子。"，李观澜抬头看见谢飞星一愣，他啥时候来的。算了，随机诬陷一个，这个孩子是你的了。

也赶巧了，李观澜坐的饮料是红色的，老太太被刺目的红吓了一跳，腿脚利落的站了起来，"不是我，我不故意的。"

"李秘书，给我抓住那个老太太，我让她给我的孩子陪葬！"，谢飞星握住李观澜的手，语气悲痛。

老太太以不符合常理的速度逃跑了，警察也没去追，因为他们看到了李观澜屁股底下的番茄汁。

小吴翻了个白眼，又是工作热情减退的一天，所有人都是骗子。

运营小姐姐看着手机里的素材，眼睛放光，这些够她剪出两集电视剧了。
