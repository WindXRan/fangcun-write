# 第112章 - 场景1：学生给张凡的惊喜告别

然而再多的不舍也不能让时光的流速变慢，剩余的高中时光依旧像匹脱了缰的野马般头也不回的离开了。

高中在校的最后一天，晚自习结束后，张凡在办公室里抚摸着已经洗出来的班级毕业照，他看着照片上一张张稚气的脸，充满了不舍。每天早晨起床上班的时候都感觉时间很难熬，毕业了怎么觉得时间一下就过去了呢。

还不待他感慨完毕，胡野突然就慌慌张张的跑进了他的办公室，"凡哥，快，咱班灯管炸了，砸到人了，你快来看看。"

张凡心都要蹦出来了，还有两天就高考了，这时候可别出什么问题啊！他来不及收起毕业照，跟着胡野狂奔了起来。

果然，灯火通明的教学楼中，只有他们班教室一片漆黑。

他着急地推开门，教室里默然无声，黑板前的投影仪却亮了起来。

上面一排硕大的字映照进他的眼帘。

'你知道吗？'

后面跟着一个硕大的问号。

'一中有名叫张凡的老师。'

这句话下面配着他的一寸照。

投影仪切换到下一张。

'他不高。'

是他和胡野的照片，胡野站在他身边比他高了一截。

'也不帅。'

是他运动会拿着矿泉水跟着参加长跑的王煜的照片，因为超过他平时的运动量，他的脸有些涨红，眼镜也跑歪了，看起来有些滑稽。

'他的脾气没那么好。'

上面是他拿着教鞭气的脸红脖子粗的模样，不知道哪个混球偷拍的。

'可他会关心每一个学生，有教无类四个字在他身上体现的淋漓尽致。'

这句话配的是他在讲台上的照片。

'即将分别，我们想说:'

'高中三年，感谢有您！'

最后出现的照片是他们毕业的集体照。

'凡哥，我们爱你，你是最棒的！'

这句话是结束语。

张凡摘下眼镜儿擦了擦眼睛，这群孩子。

屋子里再次恢复了黑暗，但马上又亮起一束束手机凑成的灯光秀。

"我怕我没有机会……"胡野大声吼道。

胡野这一开头，班级里马上响起合唱声。

"我怕我没有机会

跟你说一声再见

因为也许就再也见不到你

明天我要离开

熟悉的地方和你

要分离

我眼泪就掉下去

我会牢牢记住你的脸

我会珍惜你给的思念

这些日子在我心中

永远都不会抹去

……"

歌声慢慢变得哽咽，教室里响起同学们的啜泣声。

教室里的灯不知道什么时候亮了起来，同学们每人拿着一支鲜花依次走到他面前递给他。

张凡:谁想的这死出儿……

"老班，祝你以后身体健康，还能带二十届学生。"

张凡:哎呦，那可带不动。

"净说那吓人的话，老班我祝你暴富，财富自由赶紧回家躺平，带学生太累了。"

张凡:你也知道啊！

"老班，我高考肯定好好考，给你长脸。"

张凡:还得是李观澜啊，终于说句让我高兴的话了。

张凡费力地捧着一大捧花走到讲台上，"谢谢各位同学，那我也讲两句吧！

高中毕业只是你们人生中一次短暂旅途的结束，以后你们还有一次又一次的开始。

未来你们会遇到更多问题，面临更多选择。请你们始终保持对世界的好奇，对真理的敬畏，不随波追流，不轻易妥协。真正的成长，跟年龄无关，是责任，是担当。无关高考成绩，老师相信我的学生，每一个人，都会按照自己的期许，成为更好的人！

千言万语，凝缩成一句——

祝各位同学，长风破浪会有时，直挂云帆济沧海！"

教室里响起热烈的掌声和欢呼声，张凡伸出双手往下压了压，"我宣布，三年二班，全体同学，正式毕业！"

班级里的学生静默一瞬，突然乌泱泱的跑到讲台把张凡围在中间。

"凡哥，抱一下！"

"起开，凡哥，亲一口！"

"哎哎哎，我的老腰……"

"不管了，凡哥我爱你……"

"咦～"

张凡本就不高的身子逐渐被淹没在人群中。
