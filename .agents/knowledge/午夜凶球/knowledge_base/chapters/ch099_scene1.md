# 第99章 - 场景1：李观澜和江流楼下发疯

李观澜在万众瞩目下灰溜溜的回来了，谢玉露在后面亦步亦趋的跟着她她，生怕一眼没看到她不定又跑哪儿去了。

这也正常，高三学生哪有不疯的。

比如江流，看似正常，但李建国有一天早上送孩子上学看见他站楼下跟大树说话。

他搂着大树指向太阳升起的地方，露出神秘的笑容，"阿树，你知道吗？早上起床，面对太阳。前面东，后面是西……"

先下楼的李观澜正兴致勃勃的看热闹，她用看傻子的目光看着江流。

看了两分钟她过去一脚把江流蹬开，"你给我正常点儿。"

李建国点点头，心想还好我闺女还正常。但他马上就发现他放心的有点儿早了。

只见李观澜搂住大树，回头对江流鄙视道，"你说的这些跟树哥都没关系，它不想知道！还是我来说吧。

树哥，你想不想知道你今年多高了？想知道很简单，只要知道我的身高，量出影子的长度，再量出你影子的高度。同一时间太阳光线与地面夹角相同，那么我们把已知条件带入……"

"李观澜你是不是弱智，那大树一看都长到五楼了，根据举架高度直接乘以……"

"闭嘴！就你会！就你能！"，李观澜在地上捡起一根树枝，"老贼吃我一剑。"

江流沉腰坐马，一掌挥出，"看我降龙十八掌。"

两人你来我往过了几招，那小损招儿李建国看的直辣眼睛，赶紧叫停，"二位少侠，时间不早了，此地不宜久留，快些上马吧。"

给两个孩子送回学校，他回家把这个事儿一讲，林秋敏也觉得最近孩子们压力太大精神有些紧张。

"让他们发泄发泄吧，一天就那十分八分的，再不玩一玩闹一闹那根儿神经都要崩断了。坚持坚持，再有三个月就孩子们全都解放了。"

李建国不赞同，他觉得学习就得劳逸结合，天天这么学也太费孩子了。还有两天月假了，他不能让孩子在家闷头学了，这两天眼瞅着都打蔫儿。

"秋敏，你同事开那个农家乐在哪儿来着？现在营业不？"

"人家孩子快高考了都争分夺秒的学，你可倒好，拖后腿的玩意。"，林秋敏吐槽，但她也就是嘴上说，她一边吐槽边掏出手机，"小吴啊，你家那个农家乐现在营业吧，后天还有房间不？对，定个烤全羊，你再给配几个菜。对，带孩子去，最近学习太累了……"

李建国看着媳妇儿嘴硬心软的样子忍不住露出笑，他觉得自己可真有福啊，他说的话媳妇儿都当个事儿给办。
