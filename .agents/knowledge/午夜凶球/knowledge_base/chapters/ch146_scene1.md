# 第146章 - 场景1：吃席吃腻了，准备回京

吃席这个事儿，吃一顿两顿的还行，天天吃，日日吃谁也受不了。

李观澜这两天接电话都打怵，看见肘子都恶心。李建国打电话问她晚上吃啥，他看见有家排骨不错想买点儿，晚上炖豆角。

李观澜赶紧制止他，"爸，你爱就买啊，不用问我，我现在就想吃大米饭配咸鸭蛋。"

她得清清肠胃，过两天去京市还得接着吃。

张自珍要准备李萱的升学宴，带两个孩子先回去了，李观澜想着上大学以后没什么时间在家陪陪爸妈了多留了两天，准备到时候跟林霆他们一起回京。

沈景明的秘书这次来还带了点儿别的任务，抽查子公司的运行状况，需要晚两天回去。

眼看着李萱的升学宴在即，检查终于结束，一行人踏上了回京的路上。
