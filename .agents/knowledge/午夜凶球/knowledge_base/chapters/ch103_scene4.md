# 第103章 - 场景4：谢玉露学坏

谢玉露看了全程，回来叭叭的讲给林霆听，“这啥人呐，我真是没见过这款的。”

胡野辣评，“裤裆里揣啤酒。”

谢玉露好奇的问徐楠楠，“楠楠，啥意思？”

啊，这，徐楠楠不知道该怎么解释。

“der喝的。”

谢玉露听了先是一愣，然后红着脸低下了头，身子却一颤一颤的。

林霆眼前一黑，谢玉露你别啥都学啊，你哥知道又想杀了我了。
