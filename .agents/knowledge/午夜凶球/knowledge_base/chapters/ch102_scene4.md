# 第102章 - 场景4：晚自习被拦，江流当秘书

饶是李观澜以僵尸的姿态做了两天操，还是没能躲避过对方的追杀，她终于在一天晚自习下课时不幸被对方拦住了。

“李同学，你好我是……”

“安静男孩。”，一旁的谢玉露和徐楠楠接话。

？你们俩怎么知道，果然是暗恋我。但我现在有事，暂时回应不了你们，你们俩先等等。

“是的，我确实平时很安静，我的本名是……”

“不好意思，你有事儿吗？”，李观澜着急上厕所，她快憋疯了。

“是这样的，我想问一下你下次大休的时候有没有时间……”

“那个，我还有点事儿，你有啥问题跟我秘书说吧。”，李观澜抛弃两个好友一马当先的离开了。

“我也有事儿，你跟我秘书说吧。”，徐楠楠率先抛弃了谢玉露。

“我也有事儿，你跟我秘书说吧。”，谢玉露左顾右盼，正好看到江流过来，一把薅住江流推过去，“这是秘书。”

江流一脸懵的被推到一个男生面前，发生啥事儿了？

对面男生面对江流马上换了一个表情，傲慢的打量了他一番，“还不错，但我不会因为对手的强大而惧怕。”

江流:？？？

“李同学现在可能喜欢的是你，但她很快就会知道，男人，不能只图有外表，还要有强大的内核。”

江流:哪个李同学？你怎么还人身攻击呢？

“男人，不能只有强大的内核，还要有正常的脑壳，不行你去看看病呢？”，江流虽不明就里，但不可能被骂了不还嘴。

“什么素质？！李同学竟然喜欢你这样儿素质低下的小白脸儿，真是让人痛心疾首。”，对方恼羞成怒。

“谢谢夸奖，你也是脑袋有病的小黑脸儿。”

“我要向李同学揭穿你的真面目。”

“不是，李同学到底是谁啊？”

“别装了，李观澜同学你不认识吗？”

江流一梗，刚才应该想到的。他进前端详这位勇士，拍拍他的肩膀，“你是真牛逼啊，啥人你都敢追。”

“怎么的，威胁我是不是？”，安静男孩防备的后退了一步。

“不，我只是敬佩你。”

给李观澜惹毛了她不把你粑粑打出来算你拉的干净。
