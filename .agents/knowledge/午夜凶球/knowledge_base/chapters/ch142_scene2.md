# 第142章 - 场景2：李观澜用"重生文学"说服谢玉露

李观澜:"最主要是我觉得女明星和金融巨子的配置太好磕了，露露，势均力敌的爱情，你想不想试试？"

谢玉露那颗恋爱脑的灯泡'chua'的一下就亮了，看过的小说情节自动在脑海中播放，想的双眼亮晶晶，听起来很不错诶。

李观澜:"'她逃他追，她插翅难飞'和'她陪着青梅竹马熬过了最难的那几年，等日子终于好了起来，他却变了。'，这两个故事你喜欢哪个？"

林霆:"我就非得当贱人吗？"

李观澜:"故事情节需要，你忍一下。"

谢玉露:"那……我还是报表演叭。"

yes！

谢震元握拳。这个小同学能力很强啊，露露如今能跟林霆暂时解绑大大超出了他的预料。少女的心思谁说的准，分开这几年说不定就移情别恋了。换谁都行，林霆那个家庭，他不喜欢。

仅仅支付协议上的金额已经不能表达他的感激和兴奋了，他得派个人出差去一趟了。

李观澜:"对嘛，而且到时候你周围的同学都是帅哥，你就游戏人间一下，林霆的危机感一下就起来了，到时候惦记你惦记的抓心挠肝的。"

林霆:"你们编故事能不能管管男主的死活，这个故事不需要我的回应吗？"

他对谢玉露的感情十分复杂，你要说不喜欢吧，听见李观澜让谢玉露跟别人游戏人间一下他心里还有点慌。你说喜欢吧，他一想到要跟谢玉露在一起，他还暗戳戳的害羞，一想将来有一天他要牵谢玉露的手，亲……他就害羞到原地爆炸。

李观澜耿直的回复道，"是的，不需要，这就是工具人的命运。"

林霆:"枉我把你当自己人，你就这么对我。"

李观澜想想也有点理亏，林霆在她嘴里变成了一个渣男，话风一转:"……，话又说回来，我觉得林霆不是那种人，江流和楠楠你们俩说的有点太过分了。"

江流:"……本场对话全场只有一个坏人，不是我。"

徐楠楠:"也不是我。"

林霆:"也不是我。"

谢玉露:"难道是我？"，我抛弃了林霆，说好了一直陪他却中途跑路了，我真自私。

谢震元:……，我的女儿真单纯啊～
