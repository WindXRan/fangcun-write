# 第55章 - 场景3：李卫东的反应

李卫东看完了视频一时不知道如何评价，虽然不太体面，但知道了原委还挺解气的。这孩子确实是受了委屈，他不常跟孩子见面也就算了，她妈怎么也不清楚情况，这孩子是怎么带的。他皱眉想着，外面突然有人喊“报告”。

“进。”，他放下手机，表情不怒自威。

“报告首长，李昭的老师给您打电话了。”

李卫东皱眉，看向才回来的勤务兵。

勤务兵秒懂，敬礼转身一气呵成。他心想，这个不能给李建国他们打电话知会一声儿吗？哦，这个不是他们俩生的。
