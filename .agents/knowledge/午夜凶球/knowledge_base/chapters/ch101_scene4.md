# 第101章 - 场景4：回家后问李萱近况

到家吃完饭李观澜跑屋睡回笼觉去了，林秋敏一边儿刷碗一边儿骂李建国，“你说大早上你给她整起来干啥，本来觉就不够睡。”

李建国感觉自己比窦娥都怨，早上起来买菜让人吓一跳，回来做完饭还让人骂一顿。

算了，就这命。

“哎，你这两天跟萱萱发短信没，孩子现在状态咋样儿？”，李建国问老婆，他也不知道什么问题，平时挺能白话的一个人跟李萱说话有点儿不好意思。

“挺好，孩子说她们那边儿没咱这儿压力这么大，让咱们不用担心，多关心关心好好就行。”

“哦，那个叫啥佳佳那个孩子现在她俩还玩儿不，那孩子还跟她玩心眼儿不，这样式儿人得防着点儿啊，不定啥时候阴你一下子。”

“都说了，孩子心里有谱儿，知道咋处理。没你想的那么完蛋，比咱强。”

“哦，那她在家……”

“你自己问得了，也不是没有联系方式，这个费劲。”

“不是，我问问你咋的了，你是不是烦我了。”

……

她爸太黏牙，不怪她妈有时候受不了，李观澜翻了个身继续睡。
