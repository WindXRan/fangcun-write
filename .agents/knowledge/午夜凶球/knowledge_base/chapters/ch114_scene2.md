# 第114章 - 场景2：李观澜在考场答题

南阳（我随便编的）

李观澜把准考证，身份证和文具都摆到桌子上，静静的等待马上到来的考试。

广播中传来指令声，"现在开始分发试卷。请考生在试卷密封线内填写姓名、准考证号，不得提前答题，否则按违纪处理。"

监考老师用美工刀撕开带有红色印章的封条，拿出答题卡和卷子。监考老师点清试卷份数，确保与考场考生人数一致后依次分发。

李观澜拿到卷子先翻到了最后，看了作文题目又翻到前面，心里有数了便安心的等待开始答题。

"现在开始答题。"

考场内立刻响起哗啦啦翻卷子的声音。
