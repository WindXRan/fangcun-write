# 第98章 - 场景2：李萱反击刘佳佳

"真的吗？好像是长了一点。"，李萱低头看着自己，校服照比上个学期是短了一些。

"哇，放假你是吃了什么灵丹妙药啊。"，前桌女生羡慕的看着李萱。

自从上次李观澜来闹了一通后，好多人知道了韩宁的猥琐做法都很同情李萱，接触之下，发现她也并没有她们想象的那么高傲，她的人缘倒比以前好了许多。

也并没有吃什么灵丹妙药，只是在东北的那段时间每天睁眼睛就是吃饭，睡觉之前还在吃饭，每天除了吃睡以外就是玩儿，她还学会了打麻将。大概是吃得饱睡得好，心情愉悦，她竟然长个儿了。

"萱萱，我们去吃饭吧！"，刘佳佳的声音插进来。

前桌女生不怎么喜欢刘佳佳，总感觉她身上有种绿茶味儿，一直装傻哭穷欺负李萱。

"哦，好呀。"，李萱笑眯眯的跟前桌女生道别，跟着刘佳佳离开。

前桌女生怒其不争的看了李萱一眼转过身去。

自从李观澜揍了韩宁以后，刘佳佳很是消停了一段时间，看见李萱都要躲着走，但李萱却好像对她做的事情毫无察觉，与她相处起来也看不出什么异常。

久而久之刘佳佳又放下了警惕心，心里依旧拿李萱当成当初那个傻白甜大小姐。

"观澜以后还会不会来京市啊？"，刘佳佳吃饭时仿佛闲聊般问了一句。

"近期应该不会来了吧，她最近忙的天昏地暗，没什么时间聊天。"

马上要一模了，最近李观澜进入了癫狂的学习状态，李萱有时给她发消息要等很久她才能看到。久而久之李萱也不怎么打扰她了，论学习压力李观澜还是要比她大很多。

刘佳佳听着李萱有些怅然的语气暗喜，心想离得远就是不方便联络感情，久而久之感情也就淡了。

她感叹道，"萱萱，我没有别的意思啊，只是觉得有些不公平。你是不是付出的太多了，我们也是高三，也没见谁忙到这种程度的。你看到什么好东西都惦记着给她邮过去，她倒好回去就跟忘了你这么个人似的，连信息都不愿意回你。"

刘佳佳话音刚落，就见到黑影迎面袭来。

不锈钢餐盘落地的声音清晰又刺耳，刘佳佳的身上脸上挂着她中午买的素菜和鸡蛋汤，勾了芡的汤汁黏黏糊糊的顺着她的头发往下淌，她不敢置信的摸了摸头拿下一根菜叶。

周围静默了一瞬，然后欲盖弥彰的吵闹了起来，只是暗中不知道多了多少眼睛在偷看她们这边。

李萱的脸色让刘佳佳有些害怕，不知道为什么以前那个可以任她任意拿捏的人现在的气场强的摄人心魄。

李萱盯了她一会儿又恢复到刘佳佳熟悉的样子，有些抱歉的站起来帮她擦脸，"对不起，佳佳，我不是故意的，只是听不了有人说观澜不好，刚才那一瞬间我好像不受控制了，好半天才回过神儿，你没事儿吧？"

这话要是让李建国听见那肯定得感叹，这孩子随她姥儿了，无师自通学会顶仙儿了，而且大仙属性一样，属于定制款，李萱这个大仙儿看来是专门收拾刘佳佳。

刘佳佳瑟缩着不敢抬头，胡乱的擦了几下勉强笑道，"我真不是那个意思，你别多心。我得去洗手间收拾一下，你中午先自己吃吧。"，说完再也忍受不了来自四面八方的异样目光匆匆离开。

李萱慢悠悠坐回座位，插起盘子里鱿鱼圈举了起来，她眯起一只眼睛透过中间的中控位置看着刘佳佳远去的背影，嘴里喃喃道，"你可真是吃一百个豆儿不嫌腥。"

跟她背靠背听了全程的同学脑袋上缓缓升起一个问号。这位貌似精神分裂的女同学打完人又说的什么咒语，起到什么功效？
