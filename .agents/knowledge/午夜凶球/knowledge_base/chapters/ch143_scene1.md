# 第143章 - 场景1：两位秘书暗中较量随礼金额

"张庆才，五百。"

什么人啊？！占着茅坑不拉屎。他白了穿西装的俩人一眼，穿的人模狗样的掏钱这么费劲。

两个秘书意识到不能再拖了，谢震元的秘书率先打电话咨询老总的意见。

"沈景明也派人来了？"，谢震元冷哼一声，还想用金钱腐蚀我女儿的朋友？做梦！我们露露必须永远是李观澜心中的南波万！"不管他们出多少钱，比他们多加十万。"

得到领导的确定的指令，秘书立刻知道该怎么做了，他决定，先出手！然后一会儿悄悄地回来看对方随了多少分子。

"五十万，谢玉露。"，秘书礼貌的从背后背的电脑包里掏出几扎捆的整整齐齐的现金。

现金重奖不轻，放在木质桌上发出不轻的一声"咚"的声音。

"多，多少？"，高慧云卡壳了。

这么多现金她以前只在警匪电影和社会新闻里见过，谁没事儿拎着这么多现金来随礼啊。

"五十万。"，秘书肯定道。

"哦，等一下，你先别走。"，高慧云看了看秋敏给她准备装钱的黑兜子，装不下了。她得跟秋敏说一声，这太吓人了，谢玉露是谁？是不是有人想坑她家，这架势像洗/钱来了。

高慧云给林秋敏打了电话，"秋敏啊，你来一下，现在这个情况我有点说不清。"

秘书无所畏惧，正好老板要求他当面对李观澜家长表达一下他的感谢。

林秋敏急匆匆的来了，看着账桌上罗成小山的钱也吓了一跳，什么情况？她得给李建国打个电话。

"不是，你们是谁啊？"，李建国也懵圈。

"李先生您好，我是谢玉露父亲的秘书，听闻贵府今日升学宴，谢总特意派我来感谢李同学平日对我们小姐的照顾与提点，小小敬意不值一提，请您务必收下。"，谢震元的秘书礼貌又真诚。

"不用不用，她们朋友之间那都应该的。"，李建国连连推拒，这有钱人太吓人了。

"请您收下吧，对你们来说的举手之劳已经改变我们小姐的人生轨迹，我们谢总真的非常感谢，若不是太忙还想要亲自前来感谢您的。"，谢震元的秘书说道。

"我们沈董也是这么想的。"，沈景明的秘书非常鸡贼的跟在谢家秘书后面当应声虫。

谢震元的秘书在心里翻白眼。

"哎呀，那也不要！"，李建国非常坚定。

"那我回去将失业。"，谢震元的秘书表情淡定，嘴角泄出一丝苦笑。

"俺也一样！"，沈景明的秘书在后面疯狂+1。

谢震元的秘书快被他气死了，在李建国面前还要维持体面，"就这样吧，我先走了。"

"我也要走了，对了这是我们沈董的份子钱，六十万。"，沈景明的秘书把钱扔在桌上也跑了。

你妈的，这时候你怎么不'俺也一样了'！谢震元的秘书想打死他。

他出了饭店在附近找了家银行又取了二十万，确认没看到沈景明秘书后，又到账桌上重新上账，"您好，谢玉露再加二十万。"

高慧云已经麻木了，把他的钱收下，随手扒拉到一边的编织袋里，把谢玉露名下的金额直接勾了一下，五改成七。

没多大功夫，沈景明的秘书又来了，"我感觉不太保险，刚才那个衣冠禽兽回来没？"

他眼尖的看到谢玉露名字下面金额的改动，庆幸自己还好自己生性多疑，回来看了一眼。

"林霆那个改一下，我们加二十。"

谢震元的秘书从洗手间里出来就看到了这一幕，他等着沈家秘书离开，再次来到帐桌前，"谢玉露，加二十。"

高慧云无语的看着勾的乱七八糟的账本，我真是不了解有钱人的想法。

满意的看到金额超过了沈家，他出门直奔他停车的地方，拿起手机打电话，"喂？交警吗？我举报，有人违章停车拒不挪车。"
