# 第102章 - 场景2：好友吐槽告白信

“哎呀，我俩还没看完呢。”

谢玉露一屁股把李观澜挤中间去了，不大的椅子她俩一人坐一半。

李观澜挪了挪位置，有点儿硌屁股。谢玉露和徐楠楠夹着她你一言我一语的念了起来。

“希望你能知道曾经有我这样一个安静的男孩子，呕，默默喜欢过你。”，谢玉露一边儿看一边吐槽，“文字略显油腻。”

“希望你知道有这么一个男孩儿曾经会注视着你的一颦一笑，因为你的开心而开心，你的难过而伤心。他这一天也没别的事儿了，净瞅你了。”，徐楠楠为这段话下批注。

“我知道这是我的奢望，你是天上的明月，是我可望而不可及的存在。楠楠，看我脚下的迪士尼城堡。”，谢玉露脚趾抓地。

“但你是那么的温柔又美好，一定不会轻视我这份真心。对了，你喜欢喝奶茶吗？如果你喜欢，那明天上午间操时间，你往右手边五点钟方向看一下，跟我比个欧尅的手势，我就站那儿，跟你隔四排的就是我，下课我去给你买奶茶。

不是，这个大哥怎么突然就自说自话起来了。”，谢玉露皱眉，啥意思，不轻视你就得跟你做朋友啊。

李观澜把信拿过来揉吧揉吧扔垃圾桶里了，“别管他，现在我们的任务是学习，不要给他眼神。”

谢玉露和徐楠楠对视一眼，不管不行啊，我们现在可好奇这个安静的男孩子长啥样儿了。
