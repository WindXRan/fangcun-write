# 第103章 - 场景2：李观澜被围观拒绝

李观澜溜溜达达的回来了，一看这个场面转身就走。

但怎么可能走得了，校草硬刚拆cp的第三者，多么激动人心的时刻。此时以江流两人为中心前后左右都是看热闹的，有眼尖的一下就看到她了。

“女主回来了。”

众人如摩西分海一般给她让出一条路，呃，人也没那么多，严谨点儿，摩西分沟吧。

她傻眼了，这个剧情我是不是在哪儿看过，这时候她是不是应该站出去说一句，你们不要再吵了啦。

安静男孩眼睛一亮，马上告状，“李同学，我要跟你告白，他不让。”

李观澜被推到两人中间，抬眼看向江流，日行一善，救他狗命，你人还怪好嘞。

“不是，有我啥事儿啊？你表白你的呗，我哪句话不让你表白了。”，江流不明白他的脑回路，我只是让你去看病而已。

“你说我牛逼，啥人都敢追，这我还听不明白吗，你这意思就是李同学是你的呗，我敢追你就揍我。”，安静男孩觉得自己解读的很明白，江流就是这个意思。“他还讽刺有病，让我知难而退。”

同学A:“校草这是吃醋了，我说啥来的，面上淡定，其实早就气疯了。”

同学B:“你别说，平时看着阳光大男孩，遇到这种事儿还挺霸道的呢。”

同学C:“占有欲确实强，我就说我的cp是真的。”

李观澜看了眼百口莫辩的江流，她敢肯定全场除了她自己没一个人听明白江流其实是什么意思的？

“这位同学，他不是那个意思，他的意思是你不够了解我，你不要生气。”，李观澜合计赶紧给他劝走得了，他们仨站这儿跟猴儿似的给人围观。“我们并不认识，对彼此的认知都很表面，你喜欢的可能只是你表面看到的我，实际上的我跟你认知到的还是有一定差别的，我们不合适。而且马上就要高考了，现在我们最重要的任务是学习，希望你把精力还是放在学习方面。”

“这么说你们俩也没在一起？”，安静男孩只提炼自己想要的信息，他又高兴起来，“那行，没事儿，你学你的不用给我回应，我会默默对你好的，以后慢慢就能互相了解了，别的事儿高考以后再说。”

李观澜倒吸一口凉气，这怎么油盐不进的，考虑到有人听不懂委婉的拒绝，她决定直接一点，“对不起，我不喜欢你，你也不用默默对我好，离我远点儿就行，你影响我学习了。也不用等到高考以后再说，不可能。”

“是不是你年纪太小了家里不同意……”

他好像听不懂我在说啥，已经打预备铃了，李观澜有点儿着急。她托着下巴思索了一下，决定下点儿猛药，“这么说吧，我大学毕业，上班了，处对象了，离婚了，退休了你都不用等我，因为我有点儿烦你，是我就是死了化成灰被风吹走看见你都绕着飘，懂？”

安静男孩不敢置信的呆愣在原地，李同学怎么跟他想象的不一样，说话这么冷酷无情。就算不喜欢他，难道在这么多人面前不应该给他留点面子吗？她在他心中温柔又善解人意的形象瞬间崩塌了。

他愤愤的看向李观澜，“没想到你是这种人，算我瞎了眼，以后你求我我都不会看你一眼。”

我有病啊！李观澜无语，她露出一个非常mean的笑容，“那说好了啊，说话不算数的是狗。”

江流一脸‘我说啥来的？’的得意表情，这回知道我善良了吧。

“你也不是什么好东西！”安静男孩临走还没忘了江流，骂了句后恼怒的离开，完全不给江流反驳的机会，留下看热闹的众人和李江二人。
