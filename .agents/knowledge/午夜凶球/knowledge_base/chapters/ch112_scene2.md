# 第112章 - 场景2：李萱与刘佳佳的告别

李萱的学校也是差不多时间放的假，随大流走完感谢恩师的流程后，李萱收拾好了书包没有半分留恋的离开了教室。

刘佳佳在心里左右衡量之后追上了李萱，她不确定自己以后还能不能再遇到一个同李萱一样身份背景优越的同学了。她知道李萱和以前不一样了，但她还是想试试。

"萱萱，萱萱。"

李萱站定，讶异的看了眼刘佳佳，"佳佳，还有什么事吗？"

"没有。"刘佳佳红了眼眶，"我只是很舍不得你而已。"

"哦。"李萱歪头沉思。

既没有感动不舍，也没有不屑嘲讽。

这和刘佳佳想象中的任何一种情况都不相同，她没由来的更加紧张了。

"那就舍不得好了。"李萱似乎是想明白了，她露出如往常一般的和煦笑容，"我想了一下，你无论从学习成绩还是家庭背景来说，以后都很难有可能与我有交集了，我们的友谊就到此结束吧！"

"你，你说什么？"刘佳佳有些回不过神来，这么直白，不像李萱会说出的话。

"哈哈，惊讶吗？我还是很感谢你这三年里的陪伴，无论给我带来的是开心还是难过，过去的时光都有你的存在。如果未来你的身份阶层可以实现跳跃的话，我还是很欢迎你来找我玩的啊～"李萱想了想，又认真道，"不过，依我之见，这种可能性微乎其微。你既没有让人佩服的品德，又没有刻苦学习的勇气，你这一辈子可能也就这样了。"

刘佳佳如遭雷击，"你这三年，一直都把我当跳梁小丑吗？"

"没有。"李萱神色复杂，"我开始真的把你当做的好朋友的，可你似乎不这么想。刚发现我还是有点伤心的，可当我真正感受到了什么是无条件的爱后，你曾经给过我那一点点微不足道友爱已经无足轻重了。"

刘佳佳再说不出什么，失魂落魄地站在原地。

"那么，看在你曾经对我的照顾，高中结束的最后一天，我祝你，"李萱斟酌了一下用词，又露出了她的标志性笑容，"我祝你，在未来没有我的日子里，生活无波无澜，止步不前。那么，拜拜啦！"
