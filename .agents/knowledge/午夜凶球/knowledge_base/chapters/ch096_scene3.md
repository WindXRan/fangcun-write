# 第96章 - 场景3：李卫东见到顾伟峰与张自珍

他并不想跟张自珍说关于财产的话题，但他有点想看看她，看看她不在这个家里时的模样。

原本他总想着，往后还有很多时间可以看，但现在才惊觉，以后想再看到她已经是奢望了。她的家不会再容他随便进入，拿到离婚证后他们很有可能成为两条再不相交的平行线。

今天是周末，张自珍有很大可能在家，他没控制住自己的欲望来到了她家附近。

算他运气好，刚到她家门口就看到母女二人拎着大包小包往回走。他想上前帮忙，却看张自珍带着李萱一个转身进到隔壁家愿院子。

她们没有进屋，只是在门口跟对方一个跟母亲年纪一般的阿姨聊了几句，放下了东西后就离开了。

"顾伟峰！"，那个阿姨大声喊着，"顾伟峰，你给我出来替我送送小张她们。"

一个年纪应该跟她差不多大的男人从门里出来，但跟他常年沉稳的穿着不同，对方穿了件连帽衫和运动裤就出来了，这让他看起来比自己有活力的多。

张自珍最怕这种场面，嘴里拒绝着带着李萱匆匆往家走，就这两步路送什么送。

天公不作美，一阵邪风把张自珍的丝巾吹了回去。

顾伟峰捡起丝巾，想叫住匆忙离开的张自珍，到称呼这儿他卡壳儿了，叫什么好呢？

叫小张俩人没那么熟。叫小姐有点轻浮。对，叫女士好了，虽然他更想叫她壮士。顾伟峰这么想着，也顺嘴这么说了，"张壮士，请留步。"

张自珍回头:叫我？

啊，这！

顾伟峰只能当做这事儿没发生，尴尬嫌笑道，"那个你丝巾掉了。"

他想把丝巾赶紧还给张壮士，下台阶时急了点儿，一个趔趄把丝巾甩了出去。

丝巾好死不死被风严严实实的呼在了张自珍的脸上。

严丝合缝。

我就说今天这风是邪风吧！顾伟峰定在原地思索，她不会以为我在嘲笑她吧。

"非常感谢。"，张自珍面无表情的拿下脸上的丝巾。

"哈哈，哈。壮士慢走。"

张自珍已经回到自己的院子里了，顾伟峰忍不住抽了自己的嘴一下。

齐奶奶冷眼看他，心里对自己产生了怀疑，"我生的？"

李卫东站在原地看完了全程，原来没有家庭困扰的张自珍，是那么的鲜活啊。

他看着两人回家欢快的背影，望而却步。
