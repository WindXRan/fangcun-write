# 第98章 - 场景1：孙老太熏艾草

李卫东在楼道就闻到一股刺鼻的味道，他寻着味道的来源，三步并两步推开家门推开家门，只见家里浓烟滚滚滚隐约能见到孙老太端着盆干艾草在家里来回溜达。

"妈，你干什么呢？"，他一把拽住孙老太。

孙老太看见儿子眼神躲闪了一下，说去晦气儿子肯定又说她封建迷信，还是跟他讲科学吧，"我杀杀菌，你别管，回房间待着去。"

不对，她估摸着儿子回来的时间，会不会和朱文静碰见。不好说，她端着艾草盆围着李卫东转了两圈。

李卫东疑惑的看着孙老太。

孙老太实在编不出来瞎话了，扔下句就躲去了厨房。

"没事儿，我看你好像长个儿了。"

李卫东:？
