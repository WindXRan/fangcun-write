# 第127章 - 场景1：刘淑芬汇报，李观澜继续忽悠

'我劝你想想陈觅'。

李卫东说的话言犹在耳，朱文静深一脚浅一脚的离开了李卫东家。

没了，这下全没了。朱文静恨自己，为了心里那么一点儿念想，想来个一石二鸟之计，没想到偷鸡不成蚀把米。陈刚留给她们最后的财产，终于也没有了。

李卫东大发神威，孙老太松了口气，"卫东啊，这是……"

"妈，我回屋消停消停。"，李卫东揉了揉太阳穴，一脸疲惫的回了房间。

孙老太望着儿子的背影，心疼的拍了下大腿，帮人帮出个白眼狼，换谁不心寒。

除当天在现场的三个人外，第一个得到消息的人是刘淑芬。

第二天李卫东一去上班孙老太就拨了电话给'上神'，她学的惟妙惟肖，唉声叹气的问道，"上神，你说这事儿是不是就解决了，以后那个天煞星不能再来我家了吧？"

刘淑芬听八卦听的叹为观止，第一反应是得告诉好好一声儿，这以后应该不能有啥牵扯了，诶嘛，那我岂不是要失业。

她有点儿溜号儿，说话的声音就忘记了伪装，"那谁知道了，也不好说。"

孙老太一愣，这声音是谁啊，"你是？"

刘淑芬这才反应过来，随机应变道，"大仙儿有点事儿要处理，你一会儿再打吧。"

刘淑芬挂了电话紧接着就给李观澜打了过去，把事儿这么一说，李观澜都诧异了，朱文静她们娘俩还真豁得出去啊，最后还失败了。

这可真挺，喜大普奔的。

"好好啊，一会儿她还得给我打电话，你说我咋说啊？"，刘淑芬咨询顶仙儿办总指挥接下来的战略部署？

李观澜脑筋一转，做出了一个煎熬孙老太内心的决定。

"接下来也不好说啊，这次这个危机算是度过了，谁知道她会不会卷土重来。我观天象，原本你家宅中也是有一个镇宅之人的，所以之前那个煞星也惹不出什么大乱子，最近那人似乎离你家越来越远了，所以镇不住那个煞星才惹出乱子。"

孙老太虽不愿意承认，但左思右想也只有张自珍是这个时间段离开的，她悔不当初，当时怎么就鬼迷了心窍了，非要他们两个离婚，一定是受天煞星影响。

晚上她在床上翻来覆去的睡不着，越想越后悔，一宿没睡好，熬到天亮才迷迷糊糊的睡着。再睁眼已经九点多了，她收拾收拾自己一个人去了张自珍家，准备劝说一下她和李卫东和好，她亲自去够诚意了吧。
