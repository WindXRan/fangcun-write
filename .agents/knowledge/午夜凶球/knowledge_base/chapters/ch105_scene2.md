# 第105章 - 场景2：周文雪翻书包

李观澜迟疑的回了教室，周文雪蹲在她课桌边上翻动她的书包看来看去，李观澜轻手轻脚的走到她后面，贴着她的耳边儿轻声道，“看啥呢？”

周文雪被吓得一个激灵，一屁股坐到地上，她有些尴尬的拍拍脏了的裤子站起来，“没事儿，观澜，我看看你的书包，看着挺好看的。”

李观澜今天背的的包是齐奶奶送的新年礼物，不知道是什么品牌但应该不便宜。

她当时怂的不敢收，因为齐奶奶买东西没轻没重的，逛街那次看到支手表非说适合她，李观澜扫了一眼，那款最基础的标价四万多，她当时害怕极了，用尽全身解术才按住齐奶奶。

这个包是她临走时四奶奶送来了，不收就要断交，张自珍看她那个怂样忍不住鄙视，让她放心收下，她不会让齐奶奶吃亏的。

这个包通体黑色，容量也不错，只有一个小小的logo在不起眼的地方，看起来低调大方，李观澜还挺喜欢的，最近一直背着。

“观澜，这个包多少钱买的啊？”，周文雪又问道。

“别人送的，我也不知道。”，李观澜把书包摆正。

“哦。”，周文雪的眼神闪烁了两下，迪迪的马鞍包，挺贵的呢，李观澜刚背来的时候她就注意到了。

她之前还以为是假的，但是听到最近流传甚广的八卦她忍不住过来仔细摸了摸，手感很好，不像假的。她又说是别人送的，周文雪更怀疑了。

李观澜看着周文雪离开的背影沉吟，问了问前后左右有没有觉得最近学校哪里不对劲儿，都得到否定的答案后她才按下心头的疑惑。

这种消息，正主往往是最后一个知道的，消息滞后的还有她的朋友们，毕竟没人会作死问到她们头上‘听说李观澜被包养了，真的假的？’，那也是很愣头青了了。
