# 第104章 - 场景4：安静男孩开始造谣

中午在学校栅栏那边给孩子送饭的家长还挺多的，李观澜也没注意到她和谢飞星聊天和接信封这一幕被人注意到了。

安静男孩把这一切看到眼里，不屑道，说的那么义正言辞的，不过就是喜欢有钱的。

谢飞星来时开的是宾利添越，停在送饭的不远处，不少人看到了。安静男孩开始还羡慕的看着谢飞星，后来发现竟然她是奔着李观澜去的。看着李观澜收信封时的笑容，他内心不禁存了鄙视的想法。

里面装的什么？钱吗？

他阴恻恻的笑了笑，走到同学旁边，“诶，你猜我刚才看见什么了。”
