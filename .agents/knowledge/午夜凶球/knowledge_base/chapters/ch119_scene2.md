# 第119章 - 场景2：请李昭的小弟们吃麦当当

那边李昭脸也不知道怎么红起来了，谁？谁是你们家的？！咱俩根本不熟好不好，你打我时候下手比黄毛还狠呢！

李萱拉着李昭上下左右看了一番，确认他没事儿才放心下来，从包里掏出纸巾给他擦脸。

李昭看着周围的小弟，非常要面子的一把夺下纸巾，"不用擦，男子汉大丈夫不拘小节。"

李昭的脸上带着些擦伤，并不严重。

"快点儿的吧，别死要面子了，你那脸像刚从灶坑里钻出来。"，李观澜扯过纸巾了，用力在他脸上擦了几把。她嘲讽的话像刀子一样扔过来，下手也毫不留情，完全不像李萱那么温柔。

几个懵圈的小弟这才反应过来，纷纷迎上来，七嘴八舌的问道，"老大，你没事儿吧？"

"我没事儿。"，李昭面上镇定，还妄图保持老大的威风，"别看她们大惊小怪的，她们女人就那样儿，烦得很。"

哦，小弟看着他，那老大你的嘴怎么一直在抽搐，忍不住你就笑出来吧。

简单给李昭收拾了一下，李观澜开始查人，看看李昭到底有几个小弟，查来查去算李昭才七个人，跟葫芦娃似的。

李昭有些紧张，她想干嘛？不会去告状吧？告状也没用，我谁也不怕！

其实还是有点怕的，李卫东板起脸他还是毛毛的。

"我告诉你哦，李观澜，我们江湖儿女可是……"

李观澜怪异的看了他一眼，说什么梦话呢？

"谢谢大家今天陪李昭出来，你们吃饭没呢？"

大家你看看我我看看你，什么意思？难道要请他们吃饭，这群不良少年可从来没有过这种待遇，一时不敢置信。

"没有！"，里面有个出了名的二愣子答道，别人都不说话，他可是真的饿了，才不管那么多。

"走，请你们吃饭去，刚才打架都累坏了。"，李观澜带着一串葫芦娃往外走。

李昭愣住，他抠了抠耳朵，没听错吧！没骂他就算了，还要请他吃饭？！

"林霆，你们先去游乐场吧，我和萱萱安排好这群孩子再过去。"

林霆点点头，他们在这儿也没有用，反而还让人不自在，于是只留下李观澜和李萱两人，先去了游乐场。

吃饭的地方是他们自己选的，麦当当，刚开始大家还有些不好意思，李昭带头给大家选了东西他们才放开些手脚。

东西很快就都上来了，这群孩子刚开始还有些顾忌李观澜和李萱，过了一会儿看她们一直笑眯眯的好脾气的样子才彻底放开了。

这这孩子看穿着条件应该都不太好，吃起东西有些狼吞虎咽的，明显的兴奋。李昭偷看了她们几次，见她们没有嫌弃的样子偷偷松了口气

他用手肘碰了碰李观澜，有些难为情的说道，"那个，谢谢了啊。"

说完以后他迅速转头，避免和她有眼神交汇。

真是的，他很少说这种话。
