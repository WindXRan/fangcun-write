# 第147章 - 场景2：李萱升学宴·同学入座与刘佳佳后悔

被服务人员引导到座位上，李萱的同学打量着宴会厅里富丽堂皇的设施。

"哇，知道李萱家里有钱，不知道这么有钱啊~这酒店我就听过，还第一次来。"，同学A感叹。

"她还真低调，我刚才听见有人说这里不算场地费，餐标都要两万多起了。"，同学B八卦。

"失敬失敬，原来富二代就在我们身边，我们没能发现，早知道抱个大腿就好了。"，同学C打趣道。

"不是，她家这么有钱当初还能被……"，同学D的话戛然而止，她突然想起刘佳佳也来了。

原来两个人关系那么好，刘佳佳要是不作死，之后的人生路也算是开启easy模式了。还好还好，这好日子差点就被她过上了。

别人也就是感叹一下，刘佳佳是真切的后悔到肠子都悔青了。曾经有一个摆脱原生家庭的机会摆在她面前，她没有珍惜，如果再给她一次机会，她想说，现在当你狗还来不来得及？！

这李萱也真是的，家里这么有钱藏着掖着的，给她的东西没有一样值钱的，想来也没真心把她当过朋友。刘佳佳的脸色变幻莫测，下定决心一会儿看见李萱试试能不能修复关系，毕竟已经分开很久了，也许已经不那么生气了。

大家聊得热火朝天，只有卫骁在看李观澜。

啊~我的女神！

同学没眼看，还做梦呢，人家看保安的眼神都比看你的热情。
