# 第130章 - 场景2：江流给李观澜买裤子

"李观澜，拿着。"，江流不知道什么时候走的，盛夏的天他跑的大汗淋漓，他一抹额头的汗递过来一个袋子。

李观澜打开一看，哦，是裤子，感动。再一看，一言难尽。

"你给我买奶奶裤就算了，"李观澜把裤子从袋子里拎出来，展示在江流面前，"你非得给我买跟她同款的吗？"

裤子一拿出来大伙儿就觉得眼熟，李观澜这么一说才对上号儿，挺好，跟刚才碰瓷儿的老太太同款的。

"我说怎么看着眼熟。"，江流自言自语，又对她的不识好歹非常不忿，"事儿这么多，你就凑合着穿吧。"

李观澜咬牙，感动这种情绪跟你真是多一秒钟都维持不了，踢死你得了。

谢飞星着看两人打打闹闹，勾起嘴角，呵呵，感情还挺好的。

李观澜换了裤子还是不舒服，于是让她们先去野餐，她要回家洗个澡。

然后在谢玉露的安排下，李观澜坐上了谢飞星的车。

"让谢飞星送你回去，反正他公司卖不出去东西也不忙。"

车门"咣"的一声被谢玉露砸上了。
