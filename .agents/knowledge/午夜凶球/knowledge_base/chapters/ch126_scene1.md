# 第126章 - 场景1：李卫东街头反思

李卫东游魂一般走在大街上，看着街上的车水马龙，脑子里不期然想起的竟然是李观澜那句，'没人的地方都会都骂你傻逼的'。

哈哈哈哈，还背后什么啊，都被人怼到脸上了。他回忆过去种种，突然觉得荒谬的厉害。

"小心！"，一个男声突然响起。

李卫东沉浸在自己的思绪中毫无反应，一个六七岁的孩子踩着滑板撞到他身上，他被撞的一个趔趄，被人扶了一把才稳住身体。

"对不起对不起。"，男人一手拽着孩子，一边对他致歉。

一个年纪相仿的女人也匆匆走了过来，两人一起对李卫东道歉。

李卫东摇摇头，目送着他们远去。

女人对男人抱怨道，"让你看个孩子都看不住，废物点心。"

男人好脾气的一笑，"哎哟，错了错了，我看孩子高兴，就让她撒欢儿玩一会儿嘛！这次我一定看住她!"

他恍惚记起李萱小时候，他们家里似乎也有过这样的画面。后来，后来就再也没见过了啊。

对于朱文静家，他总觉得因为自己的原因陈刚的家散了，媳妇没了老公，孩子没了爸爸。这么多年对他们有求必应，却养大了她们的野心。

自己家，又把责任都放到的张自珍身上，忽略孩子的成长和家庭的矛盾，他的家也散了，被自己搞散的。

他怀疑最初的李卫东是不是在当初那场变故中随陈刚一起死去了，现在的他是什么？是只为了自己精神上满足的虚荣自私鬼！

他涌起强烈的欲望，他要打电话给张自珍道歉！

"道歉？！"，张自珍接起电话，"道歉也没用，道歉也起诉！"

"妈？咋了？"，李观澜她们都在客厅看电影，突然听到张自珍语气不善的接了电话，她八卦的凑过去，"谁给你打电话啦？"

"没事儿，李卫东不知道发什么癫，别管他。"，张自珍丝毫不受影响。

那边的李卫东被挂断电话也不意外，这不都是他活该，他停住回家的脚步转身向警局走去。
