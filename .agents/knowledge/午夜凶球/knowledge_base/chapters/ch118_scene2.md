# 第118章 - 场景2：孩子们在张自珍家住宿

张凤霞开门看见雇主带着六个孩子回家的时候，整个人都震惊了，雇主这是已经想开了，男人没啥用准备收养孩子培养死侍了？

"张姐，这些都是好好的同学，今晚在这儿住，你去把收拾两间客房给她们住。"

"哦，好。"，张凤霞赶紧上楼去收拾，吓死了，这么多人她一个人可伺候不过来，还好就住一天。

"李观澜，你家就一个人儿干活儿啊，能干过来吗？我给你再推荐几个，她们公司不错，来的人在我家干的挺好的。"，谢玉露看张凤霞忙乎的脚不沾地，好心道。

不用！我自己能干过来！张凤霞在心里暗暗瞪说话的小姑娘！

真是物以类聚人以群分，第一回来就威胁我。

第二天早上一看，她多余收拾那么多屋，四个女生挤一张床上睡的，真不知道她们怎么睡着的。

因为昨天聊到太晚，几个人睡到日上三竿才起床。睁眼一看，张自珍已经不在家了。

李建国给李观澜发了短信，'我跟你妈二人世界去了，然后下午我们就回去，你们爱干啥干啥吧。'

是的，这次李观澜不跟他们俩回去了，回家他们俩上班不在家她也是无聊，已经跟小伙伴们商量好在这边住几天了。
