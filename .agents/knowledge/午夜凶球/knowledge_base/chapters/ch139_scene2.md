# 第139章 - 场景2：升学宴迎宾

升学宴这天，李建国和林秋敏早早就到了饭店安排乱七八糟的杂事。

处理的差不多，亲朋好友也开始陆陆续续的到来了，李建国让林秋敏在屋里忙乎，他要去外面迎宾了。

他站了没多一会儿迎接到一个意想不到的人。

"李先生？"，李建国看着带着一个老太太前来的李卫东，他也没告诉李卫东他们家什么时候办事情啊，他从哪儿得到的消息。

李卫东点点头，"我和我妈过来看看孩子，尽量不给你添麻烦，你忙你的。"

"这话说的，你能给我添啥麻烦。"，今天你敢整事儿，张自珍能弄死你。

李建国琢磨着给这娘俩安排在哪儿合适，扫视了一圈，诶，放这儿指定没毛病。

她过去拍拍刘淑芬，"妈，这桌还有空位置吧。"

这桌现在就刘淑芬一家和二姨姥一家，正好还有两个空位，她刚被熟人追捧过，心情正好，立刻热情道，"有地儿，坐这儿。"

李卫东说要在门口陪他待一会儿，李建国没拒绝，于是只带了孙老太过来。安排她坐下，孙老太有些倨傲的跟刘淑芬点点头算是打过招呼了。

够能装的，刘淑芬心想。

能被李建国安排到这儿，应该是沾亲带故的，刘淑芬现在心情好，也没在意主动挑起话头，"姐们，看着你眼生，你跟他家啥亲戚啊？"

孙老太看她一眼，觉得声音有点熟悉，但确定没见过这人，她有点着急炫耀，也没多想便回道，"李观澜是我孙女。"

怕刘淑芬不知道李观澜是谁，又补了一句，"就今天办事情这个孩子，高考状元。"

谁？刘淑芬搓搓身上的鸡皮疙瘩，李建国他妈不早没了吗？这也太吓人了。

不对，刘淑芬马上反应过来，是李观澜那个不上馅儿（不讲理）的亲奶。

她'噶'一下止住话头，我的妈呀，这个死李建国真能安排，咋安排我这儿来了。

"啊，呵呵。"，刘淑芬心虚的闭上了嘴，生怕孙老太认出她的声音。

这是啥反应，咋不羡慕我呢，孙老太暗自生气，没想到对面这老太太更过分，竟然干脆转过身不理她了。

她坐了一会儿，周遭没一个认识的实在无聊，她没忍住又捅咕刘淑芬，"你跟李建国啥亲戚啊？"

刘淑芬刚开始假装没听见，架不住孙老太锲而不舍一直捅咕她，她挠挠鼻子，转过身来，声音照刚才低了两度，"我是他丈母娘。"

"哦。"，丈母娘就丈母娘呗，咋跟做贼似的，孙老太没话扯话，"咱俩是不是见过，我听你声音有点儿耳熟呢。"

刘淑芬深吸一口气，声音更低沉了，"不可能，我就没出过东北。"

可是我以前在东北待过啊，还不待孙老太说话，刘淑芬匆匆起身说要去趟卫生间。

不是，你这是什么意思。

真有意思，你不乐意搭理我，我还不乐意搭理你呢。东北我还有人脉呢！
