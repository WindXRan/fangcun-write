# 网文知识库

可复用的网文知识库，越蒸馏越强。

## 目录结构

```
.agents/knowledge/
├── index.md              # 索引
├── {作者名1}.md          # 风格文件
├── {作者名2}.md          # 风格文件
└── ...
```

## 蒸馏方式

使用 `style-distill` skill，一次性提取完整风格：

```bash
# 蒸馏风格
/distill {书名}

# 或
/蒸馏风格 {书名}
```

## 蒸馏内容

一次蒸馏，提取所有内容：
1. **风格类型**：题材、基调、幽默风格、情绪基调
2. **风格锚点**：5-8个必须保留的源文独有特征
3. **写法指令**：10条左右通用写法特征
4. **技法提取**：对话技法、场景描写、情绪表达等
5. **套路提取**：常用的情节模式、冲突模式等
6. **模板提取**：角色模板、剧情模板等
7. **场景片段索引**：按类型分类的场景片段（打脸、甜蜜、搞笑、虐心等）
8. **示例提取**：最能体现风格的原文片段

## 使用方式

1. 查看索引：读取 `index.md`
2. 查找作者：在索引中查找作者名
3. 使用风格：读取对应的 `{作者名}.md`

## 越蒸馏越强

每次蒸馏都积累知识，系统越来越强：
- 风格库越来越丰富
- 技法库越来越全面
- 套路库越来越完善

## 多作者协同

不同作者可以协同完成一部小说：
- 作者A负责风格蒸馏
- 作者B负责技法提取
- 作者C负责套路提取
