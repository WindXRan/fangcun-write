# story-publish · 蛙蛙写作投稿助手

## 功能

自动提取项目中的小说信息，生成蛙蛙写作投稿表单的完整填写清单。用户确认后逐一指导提交。

## 触发

```
/story-publish
/story-publish --config configs/xxx.json
/story-publish --book "书名"
```

## 投稿表单字段（12 项）及自动提取源

| 字段 | 必填 | 约束 | 自动提取源 |
|------|------|------|-----------|
| 作品名称 | ✅ | 不带书名号 | `config.json` → `book_name` 或 `book_info.md` |
| 笔名 | ✅ | ≤40字 | `config.json` → `author` 或 `book_info.md` |
| 作品字数 | ✅ | 数字 | `chapters/` 全部 .txt 字数之和 |
| 作品简介 | ✅ | ≤500字 | `concept.md` 或 `book_info.md` → 简介段落 |
| 作品分类 | ✅ | 长篇（唯一） | 固定为「长篇」 |
| 作品频道 | ✅ | 男频/女频/全频 | `concept.md` → 赛道分析 → 频道 |
| 作品状态 | ✅ | 连载中/已完结 | `state.json` → 完成度 |
| 小说类目 | ✅ | 三级级联 | `book_info.md` → 分类标签 → 类目映射 |
| 作品标签 | ✅ | 多选+自定义，≤10个，每标签≤10字 | `concept.md` → 题材/赛道 → 标签映射 |
| 上传文件 | ✅ | .docx/.doc/.txt | `export/` 合并版 TXT |
| 作品封面 | ✅ | jpg/png，≤5MB | `{书名}_封面.png` |
| 历史成绩 | ❌ | 可选 | — |

## 流程

### 1. 定位项目

```
/story-publish --config configs/xxx.json
```

读取 `config.json` → 定位 `rewrites_dir` → 自动提取所有表单字段。

### 2. 提取 + 校验

逐字段提取并展示预览，标记缺失项：

```
## 投稿预览

| 字段 | 提取值 | 状态 |
|------|--------|------|
| 作品名称 | 临水小厨娘 | ✅ |
| 笔名 | 宜莀 | ✅ |
| 作品字数 | 245,000字 | ✅ |
| 作品简介 | （从 concept.md 提取） | ✅ |
| 作品分类 | 长篇 | ✅（固定） |
| 作品频道 | 女频 | ✅ |
| 作品状态 | 连载中 | ✅ |
| 小说类目 | 古代言情 > 穿越架空 > 种田经商 | ✅ |
| 作品标签 | 种田、美食、甜宠、穿越、日常 | ✅（5/10） |
| 上传文件 | export/临水小厨娘.txt | ✅ |
| 作品封面 | 临水小厨娘_封面.png | ❌ 缺失 |
| 历史成绩 | — | ⬚ 可选 |
```

### 3. 用户确认

缺失项逐一询问：
- **封面**：有则复用，无则 `/story-cover` 生成
- **类目**：如果自动映射不准，展示 3 个最匹配的候选让用户选
- **标签**：从预设标签池中匹配题材相关的前 10 个，用户可删减

### 4. 生成提交清单

```
## 投稿提交清单

1. 打开 https://wawawriter.com/app/submission/create
2. 按以下顺序填写：
   - 上传文件 → export/XXX.txt
   - 作品封面 → XXX_封面.png
   - 作品名称 → XXX（不带书名号）
   - 笔名 → XXX
   - 作品字数 → XXX
   - 作品简介 → （见下方，复制粘贴）
   - 作品分类 → 长篇
   - 作品频道 → XX
   - 作品状态 → XX
   - 小说类目 → XX > XX > XX
   - 作品标签 → XX、XX、XX...
3. 检查无误后点击「提交」
```

## 标签映射规则

从 `concept.md` 的题材/赛道描述自动匹配预设标签池：

| 赛道关键词 | 建议标签 |
|-----------|---------|
| 古代/古言 | 古代、宫斗宅斗、仙侠 |
| 现代/都市 | 现代、校园、职场、娱乐圈 |
| 种田/美食 | 种田、美食、日常、治愈 |
| 甜宠/言情 | 甜宠、言情、日久生情、HE |
| 穿越/重生 | 穿越、重生、古代、系统 |
| 悬疑/惊悚 | 悬疑、惊悚、反转、脑洞 |
| 女频/大女主 | 大女主、锦鲤、团宠、爽文 |
| 男频 | 热血、争霸、杀伐果断、打脸 |
| 沙雕/喜剧 | 沙雕、爽文、脑洞 |
| 虐恋/悲剧 | 虐恋、追妻火葬场、BE |

每类取 2-3 个标签，总数不超过 10 个。

## 文件准备

### TXT 导出

```bash
python .agents/skills/fangcun-novel/tools/merge_chapters.py \
  chapters/ export/{book_name}.txt
```

或直接从 `export/` 目录取已存在的合并版。

### 封面

已有 `{book_name}_封面.png` → 直接使用
无 → 提示用户 `/story-cover`

## 注意事项

- 蛙蛙写作当前仅开放**长篇**投稿
- 作品名称**不要带书名号《》**
- 封面必须 `jpg`/`png`，≤5MB
- 标签最多 10 个，每标签 ≤10 字

## 输出

执行完毕输出 `投稿清单_{书名}.md`，可直接对照提交。
