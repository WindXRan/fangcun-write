# fangcun-editor

全自动。调用即读全文、逐段诊断、出编辑意见、改能改的问题、跑验证。

---

## 诊断前准备

确定项目频道（男频/女频），这决定节奏预期的基准线：

**女频基准：** 节奏偏慢、情绪密度高、内心活动多、感官细节多、对话占比中高、开头可以铺氛围但不能超过200字无信息
**男频基准：** 节奏偏快、冲突密度高、动作多、对话占比高、开头必须在10字内给出压力信号

读总纲和套路分析，知道这本书的核心爽点公式和读者预期。

---

## 逐段诊断

从正文第一句开始，逐段往下读，每段标注：

```
[段落1]
载体：动作/对白/内心/环境/旁白
功能：推进剧情/立人设/释放信息/铺垫情绪/造氛围
信息量：读者读完这段知道了什么新事实
问题：这段是否有效？无效理由

[段落2]
...
```

连续3段以上同载体 → 节奏平了，建议插入不同载体。
连续2段以上零信息量 → 这段可以删或合并。

**读完全文后，归纳三个数字：**
- 不同叙事载体种类数（<3种 = 叙事手法单一）
- 前200字释放的关键事实数（>5个 = 信息倒灌，<1个 = 欠信息）
- 情绪转折次数（章纲标的次数 ±1 以内正常）

---

## 开场（前500字，逐句分析）

对标类型：

| 开场类型 | 女频用法 | 男频用法 |
|---------|---------|---------|
| 内心独白 | ✅ 常见，用来建立语音和亲近感 | ❌ 除非是吐槽文 |
| 动作冲突 | ⚠️ 可以但有情绪铺垫 | ✅ 第一选择，10字内给压力 |
| 环境描写 | ⚠️ 不超过3句，必须带情绪色彩 | ❌ 禁止，网文男频不认环境开场 |
| 对话切入 | ✅ 有效的快开场方式 | ✅ 有效的快开场方式 |
| 反常细节 | ✅ 很好的女频钩子 | ✅ 很好的男频钩子 |

检查第一句话是否可以被删掉而不影响理解。如果可以 → 这句话无效。

**前500字的有效信息密度曲线：** 应该在5句内给出至少1个"让读者想知道更多"的信号。如果前500字全是氛围铺设没有信息推进，标记为"开场过慢"。

---

## 情绪弧线追踪

把正文按读者情绪变化切段，和章纲的情绪弧线逐段对比：

```
正文实际情绪：好奇 → 温暖 → 警觉 → 恐惧 → 决心
章纲标注情绪：好奇 → 欢喜 → 疑惑 → 惊恐 → 警觉
差异：第2段（正文"温暖" vs 章纲"欢喜"）——正文这里写了母亲抱孩子的场景，读者感受到的是温暖不是欢喜。章纲的情绪目标不对，或者正文的写法没打到目标情绪。
```

对标不上的位置，判断是"章纲标的情绪不对"还是"正文没写到"：
- 正文有那场戏但情绪感受不同 → 写法问题（用身体外化代替抽象词）
- 正文没有那场戏 → 章纲设计没执行

---

## 对话质量评估

挑选3段对话，每段检查：
1. 把角色名遮住，能分辨出谁在说话吗？→ 不能 = 角色声线没立起来
2. 这段对话删掉后，情节推进受影响吗？→ 不影响 = 水字数
3. 对话是"解释"还是"冲突"？→ 解释性对话必须改（角色不会在对话里解释世界观）

对话三功能检验：推进情节 / 制造期待 / 传递情绪。一项都不满足就删。

---

## 章尾评估

钩子强度衰减曲线（引用开头，读者在第N章结尾的感受）：

| 钩子类型 | 举例 | 强度 |
|---------|------|------|
| 突然揭示 | "他手里的信，是三年前自己写给自己的" | ★★★★★ |
| 倒计时 | "距离封印破碎还有 72 小时" | ★★★★ |
| 承诺/威胁 | "明天太阳升起之前，我会让所有人知道真相" | ★★★★ |
| 情感悬念 | "她最后看他的那个眼神，他没看懂" | ★★★ |
| 留白省略 | "然后，他消失了" | ★★★ |
| 意象钩子 | "窗外枯了一个月的茉莉，突然冒出了花苞" | ★★ |
| 无钩子 | 故事自然结束，没有下文推力 | ☆ 必须重写 |

章尾必须有至少 ★★★ 强度的钩子。否则标记重写。
