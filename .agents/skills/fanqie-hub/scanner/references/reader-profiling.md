# 读者画像分析

## 番茄小说读者画像

### 男频读者

| 维度 | 特征 |
|------|------|
| 年龄 | 18-35岁为主 |
| 学历 | 高中/大专为主 |
| 阅读场景 | 通勤、睡前、碎片时间 |
| 付费意愿 | 低（免费阅读为主） |
| 核心需求 | 爽感、代入感、节奏快 |

#### 男频读者偏好

1. **开篇要快**：3章内出金手指、打脸
2. **节奏要快**：每章都有爽点
3. **代入要强**：主角是普通人逆袭
4. **不要虐**：可以有小挫折，但不能太虐
5. **更新要稳**：日更4000+字

### 女频读者

| 维度 | 特征 |
|------|------|
| 年龄 | 16-30岁为主 |
| 学历 | 高中/本科为主 |
| 阅读场景 | 睡前、午休、碎片时间 |
| 付费意愿 | 中等 |
| 核心需求 | 情感共鸣、甜宠、逆袭 |

#### 女频读者偏好

1. **情感要细腻**：感情线要慢热
2. **人设要好**：男主要专一、女主要聪明
3. **不要太虐**：可以有误会，但要HE
4. **节奏适中**：不要拖沓，也不要太快
5. **更新稳定**：日更3000+字

## 读者需求分析

### 核心需求（按优先级）

1. **爽感**：打脸、逆袭、装逼
2. **代入感**：主角是普通人、有成长
3. **节奏快**：不拖沓、每章有看点
4. **更新稳**：日更、不断更
5. **人设好**：主角讨喜、配角有特点

### 避坑需求

1. **毒点**：绿帽、圣母、无脑
2. **虐心**：太虐、悲剧结局
3. **拖沓**：水字数、剧情注水
4. **断更**：更新不稳定
5. **烂尾**：结局仓促、逻辑不通
