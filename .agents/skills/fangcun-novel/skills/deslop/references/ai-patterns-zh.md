# 中文AI痕迹检测规则

## Tier 1 — 必须替换（出现即标记）

### 路标词
| 词汇 | 替换建议 |
|------|----------|
| 首先 | 删除，直接说 |
| 其次 | 删除，直接说 |
| 最后 | 删除，直接说 |
| 总而言之 | 删除 |
| 综上所述 | 删除 |
| 值得注意的是 | 删除，直接说 |
| 一方面...另一方面 | 拆成两句 |
| 让我们 | 删除 |
| 接下来 | 删除 |
| 现在我们 | 删除 |
| 与此同时 | 同时 / 也 |
| 不仅如此 | 而且 / 还 |
| 更重要的是 | 而且 / 还 |

### 空洞修饰词

| 全面 | 删除或具体化 |
| 有效 | 删除或具体化 |

| 重要 | 删除或具体化 |
| 积极 | 删除 |
| 主动 | 删除 |

| 努力 | 删除 |
| 不断 | 删除或具体化 |
| 持续 | 删除或具体化 |

### 模板句式
| 句式 | 修复方式 |
|------|----------|
| 这是一个...的过程 | 删除"这是一个...的过程" |
| 通过...的方式 | 删除"通过...的方式" |
| 在...的基础上 | 删除"在...的基础上" |
| 为了...的目的 | 删除"为了...的目的" |
| 从...到... | 直接说结果 |
| 不仅...而且... | 拆成两句 |

## Tier 2 — 集群标记（同一段落2+个）

### 高频副词
微微、轻轻、淡淡、缓缓、悄悄、默默、静静、慢慢、渐渐

### 情绪形容词
温暖、感动、幸福、快乐、紧张、焦虑、担心、害怕、愤怒、悲伤、失望、无奈

### 动作描写
点头、摇头、叹气、微笑、皱眉、瞪眼、抿嘴、眨眼

## Tier 3 — 密度标记（超过3%）

### 通用词
问题、情况、方面、时候、事情、东西、地方、样子、一下、一点、一些、一会儿

---

## 特殊模式

### 对话标签
- "XXX说道" → 直接写对话，不加标签
- "XXX轻声道" → 删除"轻声道"
- "XXX低声道" → 删除"低声道"

### 省略号滥用
- "......" → 减少使用，用动作/表情代替
- "……" → 同上

### 感叹号滥用
- "！" → 减少使用，用动作代替
