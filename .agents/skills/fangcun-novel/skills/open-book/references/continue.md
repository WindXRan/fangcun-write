# 续写模式

## 核心原则

**续写 = 延续原作精髓，创造全新故事**

- 角色名保留
- 延续原作核心爽点
- 延续原作金手指
- 延续原作情绪基调
- 延续原作冲突类型

## 设定文件

### concept.md

```
# {续写书名} 设定

## 风格类型
{与原作一致}

## 定位
续写/第二部，延续原作的核心卖点和情感基调。

## 核心卖点
{从原作故事核提取}

## 续写方案摘要
{续写方向}
```

## 续写铁律

- 核心爽点必须与原作一致
- 金手指必须与原作一致
- 情绪基调必须与原作一致
- 冲突类型必须与原作一致
- 不要擅自改变原作风格
