# 章间衔接方式

> 来源：@旅行的巨人 高级仿写系列

## C1-引导型

**手法**：时间、地点、发问、设定等引导词
**正向**：「再说林冲踏着那瑞雪，迎着北风，飞也似奔到草料场门口。」「次日正是上元节候，天色晴明得好。」
**违规**：「接下来林冲去了草料场。」→ 过于平淡，缺乏画面感
**常用词**：「再说」「话说」「且说」「却说」「次日」「翌日」

## C2-暗示渲染型

**手法**：环境描写、赋比兴，暗示下文走向
**正向**：「……既见复关，载笑载言。尔卜尔筮，体无咎言。以尔车来，以我贿迁。于嗟女兮，无与士耽！」（先写甜蜜，再写警示，暗示悲剧）
**违规**：「他们结婚了。后来她后悔了。」→ 直接叙述，无暗示和铺垫

## C3-标识型

**手法**：切词、空段、分割线
**正向**：「——」「***」「第一章」「（一）」
**违规**：在同一段落内直接写"后来""然后"进行转换 → 缺乏明确的场景切割感

## C4-议论型

**手法**：带有价值判断的叙述，总结上文，为下段提供自由度
**正向**：「张小敬故意没有问任何问题，让囚犯在心理上产生错觉，以为拷问方无求于自己。这样才会让他愈加惶恐，愈加急切地想证明自己的价值。刑求这门艺术，和房事一样，精髓在于前戏。安排好之后，张小敬转身离开告解室……」
**违规**：「张小敬离开了房间，囚犯很害怕。」→ 缺乏价值判断和过渡感

## C5-蒙太奇

**手法**：以共有对象作为衔接点，切换镜头（时间/空间/视角）
**适用**：压缩时间跨度、对比场景、快速推进
**模板**：
```
[场景A，聚焦物象X]。
[场景B，物象X出现/呼应]。
```
**正向**：「林钰赶着下班到菜市场里买菜，照例在菜市口买了半斤猪肉，一朵菜花，一把韭菜和一托鸡蛋……酒足饭饱后，桌上只留下残羹剩菜。林钰到头在沙发上刷着手机。」（买菜→吃饭，跳过做饭过程）
**违规**：「林钰买了菜。然后他洗菜。然后切菜。然后炒菜。然后吃饭。」→ 流水账叙述，无跳跃感
