# 开篇钩子技巧

> 来源：@旅行的巨人 高级仿写系列

## K1-冲突前置法

**结构**：直接进入矛盾场景 → 用对话/动作引爆冲突 → 补充背景
**适用**：第1章开篇，快速抓住读者
**模板**：
```
[角色]正在[困境场景]。
[冲突方]出现/联系，抛出[矛盾]。
[角色反应]，情绪[压抑/愤怒/无奈]。
→ 补充背景（不超过3句）
```
**正向**：「林瑾蜷缩在急诊室的椅子上，左手手背扎着针头。手机亮了，赵秀芬的声音像刀片刮过铁皮：'今晚家宴你准备得怎么样了？'」
**违规**：「林瑾从小就喜欢文物修复，她在这个行业工作了八年……」→ 大段背景开场，无冲突

## K2-朋友圈/社交媒体引爆法

**结构**：女主发现关键信息（朋友圈/短信/消息）→ 情绪爆点 → 展开冲突
**适用**：现代都市文，快速揭露矛盾
**模板**：
```
[角色]刷到/看到[社交媒体内容]。
内容显示[矛盾信息]（背叛/羞辱/隐瞒）。
[角色情绪反应]，[身体反应]。
→ 冲突展开
```
**正向**：「盛惜看了眼手机屏幕，呼吸一滞。是林蓁的朋友圈，上面那条动态正是早上六点，姜霖发的动态。配文：下载进度停在了99%和0%有什么区别。」
**违规**：「盛惜发现姜霖最近总是很晚回家，她怀疑他有问题……」→ 慢慢铺垫，无爆点

## K3-悬念开场法

**结构**：抛出一个谜团/异常现象 → 引发好奇 → 逐步揭晓
**适用**：悬疑/逆袭类，制造好奇心
**模板**：
```
[场景中出现异常]。
[角色]注意到[细节]，但[原因不明]。
[悬念]：为什么/怎么回事/谁做的？
→ 后续揭晓
```
**正向**：「垃圾桶里塞满了东西。她走过去，低头一看——捡碎瓷片的刷子、那把自己用了十年的修复刀、还有半盒她收藏的矿物颜料，全被扔了。修复刀的刀刃有裂痕。」
**违规**：「林瑾回到家，发现婆婆把她的东西扔了。」→ 平铺直叙，无悬念

## K4-情绪共鸣法

**结构**：展现女主的委屈/困境 → 引发读者共情 → 给出希望/转机
**适用**：虐文/逆袭文，建立读者情感连接
**模板**：
```
[角色处于困境]，[具体委屈细节]。
[内心独白/身体反应]，展现[压抑/无奈]。
[转机出现]：电话/消息/事件 → 悬念
```
**正向**：「林瑾握着手机的手在发抖。不是因为生气，是因为冷。输液管里的液体冰凉，从手背进到血管，整个人都像被扔进了冰窖里。手机又震了。这次是个陌生号码。」
**违规**：「林瑾很难过，她觉得自己的人生很失败。然后她接到了一个电话。」→ 直接告诉读者情绪，无代入感
