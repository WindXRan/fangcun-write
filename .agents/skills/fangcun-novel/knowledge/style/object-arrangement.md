# 物象排列模板

> 来源：@旅行的巨人 高级仿写系列

可直接套用的物象排列格式。

## 修真秘籍版

```
[角色]作为[身份]，[行为]，[心情]。
[物象1]，[物象2]，[物象3]，[物象4]，[物象5]。
[更甚的物象]，[更甚的物象]。凡所应有，无所不有。
```

## 嫁妆/礼物版

```
[容器]里东西真不少。[数量]+[物品1]及[物品2]。
[物品3]、[物品4]、[物品5]、[物品6]。
[珍品描述]，泛着[质感]。侧边还[附加物]：[具体内容]。
```

## 集市/场景版

```
[位置]的就是[排列方式]的[场景]。
[感官描述]，[物象1]，[物象2]。
一旁[人物]干[行为]着，[肖像]，[特点]。
```
