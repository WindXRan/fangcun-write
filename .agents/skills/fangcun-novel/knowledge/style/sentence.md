# 句式技巧

> 来源：@旅行的巨人 高级仿写系列

## 短句占比 ≥ 35%

**规则**：
- 用于情绪爆发、测试宣布、简评
- 连续3句以上超过20字 → 违规
**正向**：「"斗之力，三段！"」「转身。离开。」
**违规**：「萧炎缓缓转过身去，他的目光扫过那些嘲讽的面孔，嘴角露出一丝苦涩的自嘲。」→ 连续长句，无短句节奏

## 段落节奏

**规则**：
- 大部分段落只有一句话（特别是对话段落）
- 叙事段落不超过三句
- 对话和叙事交替出现，不要连续超过三个叙事段
**正向**：
```
"阿雯，出来呀！"
雯听到楼下的呼喊，拉开窗户，黎明的拂晓分外清冷。
圣司骑着自行车停靠在路边，朝着二楼挥手。
是他！雯欣喜万分。
```
**违规**：连续5段叙事，中间无对话打断 → 节奏拖沓

## 情绪外化

**规则**：用动作/神态/身体感受表达情绪，禁止直接写情绪词
**正向**：「紧握的手掌，因为大力，而导致略微尖锐的指甲深深的刺进了掌心之中」「脸红到脖子根」「手都在抖」「眼眶微热」
**违规**：「他感到无比愤怒」「心中涌起一股杀意」「不禁感到失落」→ 直接情绪词

## 环境描写密度

**规则**：
- 每章不超过3处，每处不超过1句
- 仅放在人物出场或场景转换时
- 连续3段大段环境描写 → 违规
**正向**：「人头汹涌的广场」「青衣少女快速跑出」（各1句，分散在不同位置）
**违规**：连续3段写「天空如何、地面如何、周围如何」→ 环境堆砌
