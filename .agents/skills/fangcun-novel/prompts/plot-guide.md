<methodology>
## 核心原则：灵魂换血

**仿写 ≠ 换名字换细节。仿写 = 保留功能，重新设计事件。**

### 读源文的方法（重要）：

**读源文时，只关注两个问题：**
1. 本章释放了什么信息？（新角色出场、关系揭示、背景交代）
2. 本章在全书中的功能是什么？（建立关系/制造冲突/推进剧情）

**读完后，合上源文，自己设计事件。** 不要参考源文的具体情节、道具、台词。

### 什么是"换皮"（禁止）：
- 源文"母亲撞破打架" → 仿写"母亲撞破抢书" ❌（只是换了道具）
- 源文"送菜被误会中暑" → 仿写"送水果被误会脚崴" ❌（只是换了细节）
- 源文"用50万奖金激励" → 仿写"用2万奖学金激励" ❌（只是换了金额）
- 源文"编故事忽悠富家女" → 仿写"编故事忽悠转校生" ❌（核心创意相同）

### 什么是"灵魂换血"（正确）：
- 源文"母亲撞破打架" → 仿写"母亲发现女儿偷偷打工" ✅（完全不同事件）
- 源文"送菜被误会中暑" → 仿写"女儿帮父亲摆摊被同学撞见" ✅（完全不同事件）
- 源文"用50万奖金激励" → 仿写"女儿为帮父亲还债主动接近转校生" ✅（完全不同动机）
- 源文"编故事忽悠富家女" → 仿写"带富家女体验底层生活让她觉醒" ✅（完全不同方法）

### 自检方法：
1. 列出源文的情节步骤（如：①母亲撞破 → ②变脸装乖 → ③糖醋排骨收尾）
2. 列出你仿写的情节步骤
3. **如果超过2个步骤的功能相同，必须重写**
4. **把仿写的人名换回源文，如果读者觉得是同一本书，必须重写**
</methodology>

<blacklist>
## 禁用清单（从源文动态提取）

**读源文第{N}章，提取以下元素，列入禁用清单：**

### 1. 禁用情节骨架
问自己：源文的情节推进顺序是什么？仿写能用完全不同的顺序吗？
- 源文"A→B→C→D" → 仿写不能是"A'→B'→C'→D'"（只换细节，步骤相同）
- 必须重新设计事件的起因、过程、收尾

### 2. 禁用标志性道具
问自己：源文中有没有让读者一看就能联想到这本书的道具？
- 如果有，列入禁用，仿写必须用完全不同的道具

### 3. 禁用标志性台词
问自己：源文中有没有高辨识度的名台词？
- 如果有，列入禁用，仿写必须用完全不同的话术

### 4. 禁用核心设定
问自己：源文中有没有独特的核心设定（金额、身份、机制）？
- 如果有，列入禁用，仿写必须用不同的设定

### 5. 禁用"换皮式"场景
问自己：我的场景是不是只是换了名字/道具/地点，但核心冲突和源文一样？
- 如果是，必须重写整个场景
</blacklist>
