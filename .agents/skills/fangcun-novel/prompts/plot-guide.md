<methodology>
## ⚠️ 仿写方法论（核心）

**仿写 = 功能级模仿，不是内容级模仿。**

**模仿的是"为什么这样写"（底层逻辑），而不是"写了什么"（具体内容）。**

### 正确示范
- 源文通过"冤家互坑"展现人物关系 → 仿写也设计一个互坑事件，但事件内容完全不同
- 源文通过"父亲试探"展现父女关系 → 仿写也设计父亲试探，但试探方式完全不同

### 错误示范
- 源文用"装中暑"互坑 → 仿写也装病 ❌
- 源文父亲说"男人不能说自己不行" → 仿写父亲也说类似的话 ❌

### 第一步：提取功能（生成章纲前必做）

列出源文第{N}章承担的核心功能：
1. **场景功能**：这个场景在全书中承担什么功能？（建立关系/制造冲突/推进剧情/情感转折）
2. **情感节奏**：情绪如何推进？（先紧张后温馨/先压抑后爆发）
3. **人设展现**：角色通过什么方式立起来？（动作/对话/选择/反应）
4. **信息释放**：读者在这章获得了什么信息？

### 第二步：设计全新场景（基于功能，不是基于内容）

为实现上述功能，设计**具体场景**：
1. **核心冲突**的起因、过程和收尾均不得与源文雷同
2. **人物**通过新的细节和动作来展现相同性格底色
3. **关键戏剧性桥段**必须用功能相同但形式完全不同的设计替代
4. **每个场景**必须有具体地点、人物、事件、关键台词

**自检：** 你的事件与源文有何不同？如果只是换了人名和细节，但事件相同，那就是内容级模仿，不是功能级模仿！
</methodology>

<blacklist_rules>
## ⚠️ 仿写禁区（违反=失败）

**禁止内容级模仿：**
- 源文中"A事件→B冲突→C收尾"的具体事件，不可在仿写中以相同或相似形式重现
- **自检：** 列出你的事件，与源文对比，确保完全不同

**禁止标志性道具复用：**
- 源文中承担关键戏剧功能的道具，仿写中不可使用相同或高度相似的道具来实现相同功能
- **自检：** 列出你使用的道具，确保与源文不同

**禁止标志性台词移植：**
- 源文中具有高辨识度的台词设计，仿写中不得出现同义替换版本
- **自检：** 列出你的台词，确保与源文不同

**完成后必须逐条说明你的仿写是如何规避以上禁区的。**
</blacklist_rules>
