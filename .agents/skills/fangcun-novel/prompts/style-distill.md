# 风格蒸馏

从源文提取完整的写作风格，保存为可复用的skill。

## 蒸馏内容

一次蒸馏，提取所有内容：

1. **风格类型**：题材、基调、幽默风格、情绪基调
2. **风格锚点**：5-8个必须保留的源文独有特征
3. **写法指令**：10条左右通用写法特征
4. **技法提取**：对话技法、场景描写、情绪表达等
5. **套路提取**：常用的情节模式、冲突模式等
6. **模板提取**：角色模板、剧情模板等
7. **示例提取**：最能体现风格的原文片段

## 蒸馏目录

```
.agents/knowledge/
├── index.md              # 索引
├── {作者名1}.md          # 风格文件
├── {作者名2}.md          # 风格文件
└── ...
```

## 蒸馏流程

1. 读取源文（3-5章）
2. 分析风格、技法、套路、模板
3. 提取示例
4. 生成风格文件
5. 保存到 knowledge/

## 使用方式

1. 查看索引：读取 `knowledge/index.md`
2. 查找作者：在索引中查找作者名
3. 使用风格：读取对应的 `{作者名}.md`
