## ⚠️ 结构锁定铁律（最高优先级）

**仿写 = 换皮不换骨。你必须严格保留源文的剧情骨架，只换表面细节。**

### 必须保留（不可换）
- **主线冲突走向**：源文的主要矛盾是什么，新书就是什么
- **人物弧线**：源文主角从A状态到B状态的成长路径，新书必须一致
- **关键转折点**：源文在哪几章有重大转折，新书就在对应位置有同样功能的转折
- **冲突升级顺序**：源文的冲突是怎么一步步升级的，新书必须同序
- **情绪节奏曲线**：源文哪里压抑、哪里爆发、哪里缓和，新书必须对齐

### 可以换（表面层）
- 人名、地名、组织名
- 时代背景、地理设定
- 具体场景、道具、细节
- 台词措辞（但情绪功能不变）
- 角色外貌、职业（但性格内核不变）

### 铁律验证
每写一个章节锚点，必须回答：**源文这一章的核心事件是什么？新书对应章的核心事件是否同一功能？**

## 输出内容

### 1. 主线概述（一句话）
格式：{主角}在{新书设定}中，面对{与源文同构的核心冲突}，通过{与源文同构的行动路径}，最终{与源文同构的结局走向}。

### 2. 情感内核（源文 vs 新书）
用表格对比，**关系本质和情感功能必须一致**，只换表面设定。

### 3. 人物关系图
**角色数量、关系结构必须与源文一致。** 只换名字和设定，不换功能位。
**必须使用上面的角色名映射表中的名字，不可自编。**

### 4. 前10章锚点
**每章的核心事件必须与源文对应章同功能。** 格式：
| 章 | 源文核心事件 | 新书对应事件 | 功能 |

### 5. 节奏分段
**卷/幕划分必须与源文的情绪转折点对齐。**

## 自查清单（生成后内部校验，不输出）

- [ ] 主线冲突与源文同构（不是"灵感来自"，是"结构一致"）
- [ ] 人物弧线与源文同路径
- [ ] 前10章锚点每章都能指出源文对应章
- [ ] 关键转折点位置与源文对齐
- [ ] 角色数量和关系结构与源文一致
- [ ] 角色名全部使用映射表中的名字，没有自编
- [ ] 没有引入源文不存在的主线冲突
- [ ] 没有跳过源文的关键剧情节点
