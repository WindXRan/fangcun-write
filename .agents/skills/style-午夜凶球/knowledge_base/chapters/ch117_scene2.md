# 第117章 - 场景2：李观澜被抽中唱歌

演唱会进入高潮，随机抽取一名观众跟歌手合唱，刚才跟林霆一起接李观澜的那个工作人员示意灯光师，选那个小姑娘。

一束光线照到了李观澜头上，已经混进人群的她无比惊讶的指着自己，我吗？我吗？

哦，不！

江流和徐楠楠痛苦的闭上双眼。

"对，就是你！"

得到肯定的答复李观澜兴高采烈的站了起来，可不是我非要唱的哦！

音乐响起，歌手唱出第一句，"剪一段时光缓缓流淌……"

"流进了月色中微微荡漾"，李观澜唱出第一句大家还没什么感觉，等她继续唱下去场馆诡异的静默了。

平衡了，长得好看怎么啦，被偶像抽到咋的了，我们唱歌起码不这么抽象。

老天爷是公平的。

给了她美貌就就一定不会再让她拥有百灵鸟一样的歌声。

好了，过过瘾得了，快把麦克风还回去吧，再唱下去演唱会门票都降价了。

看着旁人投来打量的目光，林秋敏用荧光棒往旁边一指，"其实这个是她闺女。"

张自珍:……

到底是谁闺女，那人目光怪异的看了她们俩一眼，不会是拉拉吧，怪时髦的。

张自珍肩膀后甩，面无表情的把林秋敏的胳膊抖了下去。

才见过三回，不熟。
