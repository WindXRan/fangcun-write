# 第99章 - 场景3：大鹅追人事件

谢玉露发现了了一个铁丝网小门儿，门上有个挂钩勾住了铁栅栏，她好奇的把钩子拿下来，往里面看了一眼。

一声厚重的'嘎嘎'声从拐角里面传来。

谢玉露有些害怕，她伸出手里的树枝捅了一下，什么东西？

一只白色的大鹅从拐角里面愤怒的冲出来，"嘎嘎嘎！"

"妈呀！"，谢玉露扭头就跑。"快跑。"

她还算有良心，逃跑不忘告诉几个小伙伴一声。几人回头一看一只翅膀炸开快有一米宽的大鹅正弓着身子跟在谢玉露后面穷追不舍。

李观澜眼睛都要瞪出来来了，谢玉露怎么把从哪儿把这玩意整出来的。大鹅堪称家禽中的战斗机，拧人一下能原地升天。

显然大家也都听说过大鹅的战斗力，所有人都没有丝毫犹豫，撒腿就跑。

院子里顿时一片鸡飞狗跳。

但大鹅显然非常执，它只想拧最开始捅咕它的那个小丫蛋儿。谢玉露连上体育课都没这么跑过，她觉得自己现在至少也能达到国家二级运动员的标准。

大家都不跑了，全蹲一边儿看热闹。

江流从兜里掏出一把瓜子儿分给大伙，"看不出来谢玉露怪能跑的啊。"

林霆紧盯着大鹅，手上却也没闲着，接过瓜子儿，"也挺出乎我的意料，看来不逼到绝境谁也不知道自己有什么潜能。"

"可不，这么看着身体素质比我强。"，李观澜看着身形矫健与以前判若两人的谢玉露感叹，好好加以培养以后是块女杀手的好料子。

"我们光看着是不是有点过分啊。"，徐楠楠蹙眉，觉得这样不太仗义，"我们给她加油吧！"

……

你这个行为很友好吗？！！

"露露，加油！"

"谢玉露，加油！"

"一二三四五六七，谢玉露你最牛逼！"

"山水无情人有情，谢玉露你一定行！"

谢玉露欲哭无泪，她马上就要没有力气了，就在大鹅马上要扑上来的的时候，林霆把瓜子儿往江流手里一塞跑过去一个飞踹救出了谢玉露。

是的，大鹅改追他了。

谢玉露中场休息，徐楠楠贴心的递过来一瓶水。

谢玉露擦了擦头上的汗，咕嘟嘟的喝下去半瓶水。缓过气儿她捅咕江流，"该你了。"

江流一脸无语，这玩意又不是接力赛，你爱玩儿你玩儿。

林霆在大鹅的全力追逐中放空了脑子里所有的想法。竭尽全力的奔跑，放空的大脑，竟然让他有一种格外轻松的感觉。

他越跑越高兴，在院子里大放厥词，"没有精疲力尽的奔跑过，怎么能算热血的青春！"

"该我了，该我了！"

李观澜看的眼热，她也想要热血的青春。
