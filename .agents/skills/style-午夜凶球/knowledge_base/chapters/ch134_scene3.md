# 第134章 - 场景3：刘小华回家

林秋敏挂了电话喜气洋洋的回了办公室，坐了一会儿实在没忍住，对着电脑屏幕发出笑声。

刘小华死对头郑雪非常善解人意的发出了疑问，"林姐，华清大学都打电话了，孩子这排名肯定很高吧？"

林秋敏笑着答道，"啊，刚才打电话我才知道，省里第一。"

"哎哟，我的天啊，林姐，你家这孩子文曲星下凡啊。办事情有啥要帮忙的不，那天我第一个到啊，我也沾沾文曲星的仙气儿。"，郑雪什么好听说什么，人家这孩子眼看就要起飞了，处好关系只有好处没有坏处，以后背不住有用的着地方呢，谁傻啊这时候泼冷水。

办公室里的赞美声此起彼伏，不绝于耳。

这群马屁精！刘小华咬牙，正巧有人看过来，刘小华马上变了表情，露出一个还算自然的笑容，"林姐，恭喜你啊。"

刘小华忍了一天，憋了一肚子气。推门一看老公正在沙发上躺着玩儿手机，家里锅碗瓢盆早上走的时候什么样儿现在还是什么样儿，把包往沙发上一扔。

"你回来这么早，把饭做上不行啊？"

"你吃炸药了？"，刘小华老公眼睛都没离开手机，翻了个身背对她。

"我在单位忙了一天了，回来想吃口现成饭都不行啊。人家老公回家做这做那的，你倒好天天在家养大爷。"，刘小华越看他越生气，李建国那一天天像是缺媳妇儿似的给林秋敏供上，他在家酱油瓶子倒了都不知道扶一下。

"你差不多得了。"，刘小华老公皱眉，语气明显不悦，"一天天供你吃供你穿的，干点活儿没完没了的。"

"不是冲你。"，刘小华看到老公发火，语气明显缓和下来。

她那个工作也就够她自己每个月吃喝，家里赚钱的主力还是她老公，她也很有眼色的只在她老公心情好的时候才敢作一下。

"还不是那个林秋敏，孩子考了700分这把她嘚瑟的，我看单位快容不下她了。"

"多少分儿？700！"，刘小华老公诧异的转过身来，"那人家咋嘚瑟也没毛病，咱家小雪考那些分，我能回老家摆一个月流水席。"

刘小华老公是个小包工头，一年不少赚，说这话完全有底气。小雪要考这些分，他能回老家给她单开一页族谱，看谁还敢笑他没儿子。

刘小华老公没跟她统一战线，但她也不敢再作了，闷闷的说道，"我去厨房做饭。"

"不过我咋记得，上回听谁说李建国家那孩子抱错了呢？"，他们之间业务有重合，朋友之间交集不少，他恍惚记得听谁提过一嘴。

刘小华听见老公云淡风轻的声音传来，她猛的抬头，"你说啥？！"
