# 第102章 - 场景1：收到告白信

要说学校给学生休大礼拜还是非常正确的决定，两天下来李观澜觉得自己脑子都清醒了，看起书来事半功倍，做什么事儿都反应贼快。

“李观澜，给你的信。”

李观澜一边儿做练习册一边唰的接过了信压在底下，动作行云流水一气呵成。

反应就是这么快！

然后就给忘了。

她做完最后一道题，把练习册卷起来放进书架，收拾好才桌子才看见这封信。

她回忆了一下，恍惚是有这么个事儿来的哈。打开粉色的信封，里面是一封告白信。

“李观澜同学你好，我是三年X班的XXX，喜欢你已经很长时间了，我知道在这个关键的时刻打扰你学习的行为非常不好，但我还是希望你能知道……”

“唰”，李观澜把信叠起来了，她不想知道。
