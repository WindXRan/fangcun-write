# 第113章 - 场景3：高考当天早上出发

高考那天早上，李建国特意给李观澜准备的考试必备套餐。

一根香肠，两个荷包蛋。

李观澜看着有点儿一言难尽，要不要跟老爸说现在满分是一百五。算了，吃吧。

林秋敏在镜子前整理衣服，灰色旗袍，深黄色的盘扣。ok，没问题。

"闺女，吃完没？"

"吃完了。"

"走，出发。"

他们出发的都够早的了，到了考场那人还是乌泱乌泱的。

门口随处可见家长嘱咐孩子的画面。

"答完卷不着急交啊，多检查两遍。"

"一定检查答题卡涂的对不对啊！"

"那个铅笔，咱买的是不是2B铅笔，快掏出来瞅一眼。"

"是这个考场吧？准考证，身份证掏出来准备好。"

当然也有画风迥异的。

"儿子，你要是不会也不行睡觉啊，蒙你也把选择题给我蒙上。"

一句句叮嘱孩子上没上心不知道，李建国上心了，他更紧张了。

"快点儿，你也掏出来看看。"

"爸你别紧张，昨天晚上不都检查好几遍了嘛！"李观澜为了让他爸安心掏出来又让他检查一遍。

"你别给孩子添乱了，一点忙都帮不上净扯后腿。"林秋敏看见有人进考场了，赶紧把他拉开，"好好，可以进考场了，进去吧。我和你爸在那边儿那个大树底下等你，你考完上这儿来找我俩。"

"好嘞！"李观澜背着小书包，马尾一甩一甩的进了学校。

"秋敏啊，我心咋这么慌呢？"李建国坐在大树底下手里的扇子摇啊摇。

"你不都求好几个大仙了吗，肯定稳了。"林秋敏非常敷衍的说道。

"我现在有点儿后悔了，一下求好几个能不能犯冲啊？"跟求人办事一样，一件事儿不能求好几个人，要不容易办叉劈。

墨迹，林秋敏懒得理他，头一歪，靠大树上假装睡着了。
