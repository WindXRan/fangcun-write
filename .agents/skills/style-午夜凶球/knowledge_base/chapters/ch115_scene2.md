# 第115章 - 场景2：演唱会票的巧合

林霆:"不行你们上京市来吧，我家公司承办了个演唱会，后天开场，你们要是来我去给你们要几个工作证。"

江流:"不太想去，谁啊？"

林霆:"XXXX。"（之前有个宝子不让写是谁的演唱会，说是怕对家哈哈哈哈，笑发财了，这个我不信有对家。）

那很权威了。

这个组合，在国内的火爆程度和她们的组合名一样传奇，粉丝年龄横跨三岁到八十岁。

江流:"车票已买，兄弟等我。"

李观澜:"装货。"

徐楠楠:"+1"

谢玉露:"+2"

李观澜兴奋的在床上一个鲤鱼打挺，之前售票的时候她没抢上，这不是瞌睡就有人送枕头。

李观澜买了票准备等李建国他们俩下班了再告诉他们，谁知道晚上李建国回来的时候还不等她开口就欢天喜地的跟她显摆，"哎呀，这求爷爷告奶奶的，好不容易给你买回来三张票，你和萱萱还有你妈去看吧。"

不会吧，这么巧，李观澜瞪大了眼睛，"什么票？"

"演唱会的，京市场，你不想看吗？XXXX！"，李建国等着闺女惊喜的表情。

"你咋不去呢？"，李观澜先问出了她最好奇的问题。

"我那天有事儿。"，李建国不自然的把头撇向一边儿。

其实是门票太贵了，他买的本身就是前排，因为没抢到票找的黄牛又加了好几千，他有点儿心疼钱，自己就不去了，门票路费住宿加一起能省不少呢。

李观澜一眼就看穿了她爸那点儿小心思，"门票能不能退，退一张。"

"谁不去啊？"，李建国疑惑的看着她。

"都去，我上边有人。"，李观澜嘚瑟。

"谁啊？"

"你好大儿。"

？？？

谁啊？

"林霆他家是主办方，我和萱萱有工作证，能进去。你跟我妈坐着看吧。"，她们跟林霆说好了，不白看，帮忙干活儿。

她爸妈这个岁数就算了，消停坐那儿看吧。看他们给别人干活儿她闹心，还是纯享受吧。

最后那张票也没退成，原因是林秋敏说都到京市了，也带萱萱了，你咋好意思不带你亲妈呢？

李观澜对此表示不是不带她看，是想象无能，张总跟演唱会那种闹哄哄的地方一点也不适配啊。

林秋敏说那也得问问，你不问咋知道呢。

于是这个活儿李观澜就分配给她了，张总最近这么猛万一哪句话没说好喷我怎么办。

林秋敏用看废物的眼神看了眼她，直接打了语音给张自珍。
