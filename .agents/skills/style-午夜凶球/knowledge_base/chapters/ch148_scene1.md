# 第148章 - 场景1：刘佳佳试图修复关系被李萱拒绝

"李萱！恭喜你！"

"李萱！祝贺你！"

李萱的高中同学纷纷站起来跟她打招呼。

李萱笑眯眯点头谢过他们，"吃好，喝好，有什么需要跟服务人员说，不要客气。"

说罢她转身要走。

刘佳佳急了，起身拉住她的裙摆，"萱萱！"

李萱侧头垂下眼皮看着她的手，刘佳佳慌张的松开手，"那个，萱萱我就是想说你今天真漂亮！"

"哦？"，李萱勾起嘴角，"这裙子你也看上了？"

这个也字信息量很大，大家都看向刘佳佳。饶是她脸皮不薄，此刻在众人探究的眼神下，她也涨红了脸。

"我不是这个意思。"

"那你放开我啊，我还有事。"，李萱从她手里拽出裙子，对着大家点点头离开了。

再次试图抱大腿失败，刘佳佳知道她们是没什么缓和的余地了。她坐下恼羞成怒的说道，"有钱人家的小姐真是现实，毕业了就不认老同学了。"
