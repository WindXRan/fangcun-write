# 第140章 - 场景2：老赵来了

"老李，恭喜恭喜啊。"，身上雕龙画凤的老赵跟李建国一碰面就笑，他是个直肠子，不像别人都挑好话说，他想说啥说啥。

"诶呀，你是熬出头了。谁能合计小好好能像今天这么出息啊，早几年我看她是块混社会的好料子，都准备一辍学就收拢到我旗下，给我要帐去。后面学习一下好了，我还挺失望的呢，痛失一员爱将！"

"去你的吧！"，李建国上去就是一拳，"让你儿子给你要去吧，我闺女不干这活儿。"

"你看你还不乐意，咱好好姐搁我公司那条街都有一号，打架下手那叫一个黑，我看见都害怕。还好这孩子有良心，看着他爹在外面跪着挣钱她心疼啊，你是功德圆满了，以后都是好日子啦。"，老赵给递给李建国根烟，自己也点了一根儿，回首看见一边儿的李卫东示意他，你也来一根儿？

李卫东摇摇头，他好像随着两人的对话回到了李观澜以前最叛逆的时期。

"说那个干啥，哪个老爷们不养家，钱难挣屎难吃，亘古不变的道理。这点事儿还带家去让媳妇儿孩子跟着闹心？"

李卫东膝盖中了一箭。

"是啊，媳妇儿孩子的，哪个不得惦记着，要不那还叫个爷们儿？"，老赵猛吸了口烟，别看他看着像，社会似的，做的也是擦边生意，看着风光，进了社会，哪个能没有闹心事儿。

李卫东膝盖又中了一箭。

俩人正在这儿感慨，又有熟人上来了，是个四十多岁打扮的花枝招展的中年女性。

"建国，恭喜你啊。"

这女人有些自来熟，上来就握住了他的手，热切道，"孩子太棒了，都是你教育的好啊。"

"哎呀，客气了客气了。"，李建国感觉背后一寒，赶紧甩掉她的手，"赶紧进屋坐着去，让你嫂子给安排个地儿。"

"秋敏啊～"，李建国一转身回去发现林秋敏就在她后面站着，他吓一跳，你那什么眼神儿，我表现挺好的。

林秋敏用眼神表示'你等回家的'后就换了一副惊喜的表情，亲亲热热的搂住来人，"呀，弟妹来啦，快往里进，有日子没见着你了。"

老赵看的在一边儿偷笑，"那小柴以前不还帮过你吗？你咋跟见了洪水猛兽似的呢？"

"她帮我我不也帮她了吗，我还把自己搭她啊，真有意思，这回家都得挨呲儿。"，李建国在裤子上蹭了蹭手，自己又闻闻有没有味儿。万一沾点护手霜啥的呢，那老林得干我。

李卫东膝盖再次中箭。
