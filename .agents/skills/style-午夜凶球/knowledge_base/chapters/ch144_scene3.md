# 第144章 - 场景3：孙老太找李昭被拒与饭桌争执

孙老太气的吃不下饭，起身给自己新晋闺蜜打电话，添油加醋的把事情讲了一遍，最后结尾，"齐姐，来者是客，你说有她这样儿的吗？"

"没有，过分了！"，这事儿搁我身上我得打她，来干啥的不知道吗？还敢当面说我外孙女，真让人捶一顿你就老实了。"这你都没还口？你这素质真没得说，换我我早上手了，不给她们点教训真不知道马王爷长几只眼！"

还得我得老姐妹，孙老太感动的抹抹眼，一天的委屈都在此刻被人理解后释放了，"算了，她们没素质，我不跟她们一样儿，跟你说说我就好多了。"

这个怂货，就说打不过得了，齐奶奶在电话的另一头翻白眼。"那行吧，好人就是容易受委屈，咱素质高不跟她计较。都这个点儿了，你吃饭了吗？"

没有！孙老太气消了才觉出饿来，出来半天了应该上菜了，"没吃呢，那我先回去吃饭，咱下回聊。"

孙老太抱着我大度，跟你们这群乡下人不一般见识的心态回去吃饭了。

到了饭桌，心态又崩了。

吃饭吃的这么快啊！！那两个人干嘛呢？！都开始甩塑料袋准备打包了？！

孙老太赶紧坐下划拉两口菜，今天早上出来的本来就早，早就饿了，中午再不让她吃口饭就要撅过去了。

吃完饭孙老太才想起来她此次出行的目的，看大孙子。这一上午光顾着跟这缺德老太太较劲了，都忘找孩子了。

孙老太眺目远望，一眼就找到了人群中惹眼的李昭，她倒腾着小碎步快速走到他面前。

"昭昭，昭昭。"

"奶奶？"，李昭看见奶奶来了也很惊喜，说实话，许久不见他还有点儿想孙老太。虽然孙老太不讲理，但对李昭是实打实的好，他没办法一边享受着奶奶带来的利益一边唾弃她的人品。

虽然偶尔有不耐烦的时候，但祖孙相处这么多年的感情是实打实的。

孙老太把李昭搂进怀里，看着他今天的造型泪眼婆娑，"昭昭啊，跟奶奶回家吧！你在这边儿她们也不把你当人看啊。"

这头发一绺一绺的，都几天没洗了？

还有那衣服裤子，上面漏的都是洞，这破衣服都是哪儿捡回来的。张自珍真不是人啊这么有钱就给孩子穿破烂，李萱怎么就穿的干干净净的呢？不是自己养大的就是不亲！

"不要！"，李昭一口拒绝。

孙老太的眼神充满了不可置信，才在外面住几天就不回家了。

李昭也有点后悔自己拒绝的太快，伤了孙老太的心，赶紧找补，"那个，奶奶，我在这边还有点事儿，等我都处理完就回家住两天奥。"

"是不是你跟我孙子说啥了？！之前在家你就兴风作浪，搅得全家不安宁，离婚了你更肆无忌惮了……"，孙老太恶狠狠的看向张自珍，肯定是她说啥坏话了，要不孩子怎么家都不回了。

张自珍眼皮都没撩一下，直接找能解决孙老太的人，"李卫东，你能不能管管你妈？大好的日子别让她来添堵。人家李建国他们正开心呢，非要这时候来解决你家这点儿破事儿吗？"

李卫东早在孙老太把矛头转向张自珍的时候就站了起来，但还是晚了一步，他强行把孙老太带到一边，"妈，您要这样那我们还是回去吧。"

孙老太还没看够孙子不想走，只好憋屈的住了嘴。
