# 第140章 - 场景3：孩子们到来

这时候李观澜领着一群人热热闹闹的来了，她刚接到林霆和谢玉露她们，看见李建国在门口一群人一窝蜂的就上来了。

"爸，你咋不找个阴凉地方待着啊这块儿多晒，似不似傻。"

"哎呀知道了，这太阳才照到这儿。没看见你赵叔啊，咋不打招呼。"，李建国赶紧教育孩子，看见人不打招呼呢，怪没礼貌的。

"哎呀，是我赵叔啊，我合计我爸哪儿认识的大帅哥呢，刚才都没敢认。好久不见了，赵叔，今天咋没给我小弟带来呢？"，李观澜立刻拍马屁加打招呼，并自动开始拉家常功能。

老赵被李观澜哄的笑逐颜开，哎呀，这孩子真是人才啊，当初要是跟他混现在也成就一番事业了。

那边一群孩子围着李建国一会儿'义父'，一会儿'李叔'的，吵的他头大。

"咋样儿，今天用的我姐的发泥，她给我抓的，炫酷不？"，李昭指着自己颜色更乱套的发型说道。

"哎呀好看好看，太好看了，还得你们年轻人的审美。"，我们这岁数的看完要心梗，李观澜你真是能祸祸人，给你弟脑袋上整几个色儿啊。那有的地方发泥还没搓开呢，跟鸡粑粑似的。

老赵诧异的在李观澜和李昭脸上看来看去，最后得出结论，"老李，你啥时候要的二胎？"

李昭满眼憧憬的看着老赵，叔叔你真霸气。

李萱看到老赵也礼貌的上前打招呼，感谢赵叔叔上次特意来给她送东西，下次到京市一定联系她，她要请赵叔叔吃饭。

老赵稀罕稀罕这个，稀罕稀罕那个，一时间都稀罕不过来了。

"建国啊，你家这几个孩子真好呀，是真好呀！"

没有一个人注意到一边的李卫东，孩子都成别人家的了，李卫东身上插满了箭。
