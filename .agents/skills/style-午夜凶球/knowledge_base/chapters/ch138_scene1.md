# 第138章 - 场景1：爱马仕店

"不行，必须要。"，张自珍霸总上身，语气不容置疑，看着刘小华恍惚的眼神她都有点爱上这种感觉了，"这孩子你养的不容易，给你买东西也是应该的。"

她非常有自知之明，李观澜如果给她养，应该就是战力值和智力值拉满的女版李昭，这么一想她都觉得东西买少了。

"走，看看包去。"

"不用不用，小珍呐，我根本用不上。"，林秋敏两只手上挂满了袋子，惊恐的往后退了两步。

看啥包啊看包，金子好歹还保值，还钱就算了。那破包到手就贬值一半，我可不要，到时候还钱得闹心死我。

"别说话。"，张自珍拽住林秋敏，回头看了眼刘小华，"这你同事是吧，一起来吧，帮忙参谋参谋。"

刘小华咬牙，去就去，这给你装的，家里有矿啊，我看你装完这波一会儿心疼的呲牙不。

"行啊，既然您都邀请我了，那我去看看。"

"小珍呐，差不多得了。"，林秋敏凑近张自珍低声说，"已经给她刺激够呛了，咱收手吧。"

"没你的事儿，我看她不爽，说话阴阳怪气的死样儿跟朱文静有一拼。"，张自珍左右环顾一圈，看到一家品牌店，就你吧，她拉住林秋敏的手腕。

"拉倒吧，这破地方地球仪敢卖十多万，我可不……"，林秋敏感觉背后一股推力，直接进了爱马仕大门。

进去吧你！刘小华推了林秋敏一把，俩人唱上双簧了是吧，我看结账时候谁尴尬。

张自珍赞赏的看了她一眼，你还算有点儿用。

这是什么眼神，刘小华哽住。

导购在门口已经打量她们半天了，三个都是生面孔，在门口拉拉扯扯的，进来都要犹豫半天。估计是进来看热闹的，这种不是她们的目标客户。

"欢迎光临，请问看些什么？"导购不冷不热的打了招呼。

"看看包。"，张自珍带着林秋敏往沙发上一坐，抬头示意导购取包过来。

刘小华站在一边抱胸看着她们两个，我就静静的看着你装。买黄金和买包可不一样，黄金放两年还兴许赚点儿在这儿买包还要配货，纯赔。

"您要看哪一款呢？有的款我们柜台只是用来展示，没有现货的。"，张嘴就要包，懂不懂想在我们这里买包是要配货的啊。

"铂金25，大象灰银扣。"，这个容量还行，样式背着也不跳脱，估计林秋敏能用上。

导购露出职业假笑，"真不好意思，正巧这款没有了呢。"

刘小华:然后你就借坡下驴可以走了吧，呵呵。

张自珍心下了然，姿态的疏懒的靠到沙发上，"我可以配货。"

导购微笑，"女士，我们没有配货这一说呢。"

张自珍笑笑，"我建议你先去查查我的账户再说话。"

导购半信半疑的去查了账户，没多久就一脸热切的走了回来，态度跟刚才比判若两人，"张女士，真的很抱歉刚才怠慢您了，您想要的包我可以给您调货过来，不过也需要一两天的时间呢，您看您可以等吗？"

"这样吧，你去给我选一些适合年轻女孩穿的衣服啊，鞋啊什么的，每样双份。还有配饰丝巾挂饰什么你看着选好了，我今天就想取货，能办到吗？"，张自珍也不为难她，都是心照不宣的规则，配就好了。

"这……张女士，既然您这么着急的话我再给您想想办法。"，导购看似一脸为难，实则内心都要乐开花了，她就喜欢跟爽快人办事。"那我再去请示一下领导，看能不能先把别的客户的包先给您。"

导购离开后林秋敏死死的按住张自珍，咬牙切齿的低声道，"我可没钱还你。"

张自珍鄙视的看了她一眼，"说了送你的了，不用你还。"

刘小华已经觉得事情不对了，她心里惴惴，差不多得了，你们两个来真的啊，那她可要怄死了。

但天不遂人愿，导购喜气洋洋的走回来，一脸惊喜道，"张女士，真巧呢，另一位客户最近来不了，这支包包您今天就可以带走。鞋子什么的选好了拿过来给您过目可以吗？"

张自珍看了配货，扫到一条丝巾的时候手顿了下，这条丝巾怎么这么丑，送林秋敏同事吧。

张自珍利落的结了账，留了地址把乱七八糟的东西让她们送过去，包包则是直接挂在了林秋敏的手上。

林秋敏和刘小华都陷入了沉默。
