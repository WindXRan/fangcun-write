# 第120章 - 场景3：李昭回家遇到朱文静

他恍恍惚惚的回到了家，感觉今天的一切都完美的不真实，他甚至怀疑今天是自己臆想出来的。

直到在家门口遇到朱文静，他才回归现实，不是梦啊，又看到这个讨厌的女人。

"小邓，我没别的事儿，就是买了点水果过来看看孙阿姨。"，朱文静手里拎着几个袋子，面上带着些讨好。

"朱阿姨，奶奶真的不在家。"，小邓在门口挡着，拒绝之意非常明显，"你的好意心领了，等奶奶回来我跟她说。"

"那我把水果放进去吧，都是孙阿姨爱吃的。"，她说着就要往屋里进。

"不用了。"，小邓因为太紧张，声音大了起来，听起来显得非常不耐烦。奶奶已经给她下了死命令，绝对不允许朱文静进去。再说她一进去，奶奶在家这事儿不就露馅儿了。

朱文静的表情有些维持不住了，勉强笑道，"那行，那我就放门口，一会儿……"

"说不要你听不懂吗？"，李昭从后面推开她，带着些青紫的脸上满是嘲讽，"脸皮真厚。"

李观澜想这招儿是真好用，那之后朱文静来了好几次，愣是一次都没进来过。

李昭脱鞋进屋，小邓看着他脸上的伤，注意力都转移走了，敷衍的对朱文静笑笑便关上了门。

"昭昭，脸怎么弄的，怎么又打架了，身上有没有伤。"

小邓关切的声音从门里传出来。

"怎么了？我看看，哎呀，昭昭你怎么搞的，疼不疼，奶奶看看。"

孙老太慌里慌张的声音清晰的响起，朱文静攥紧了被袋子坠的生疼的手。

真是同人不同命，这小混混的命怎么这么好。她的陈觅那么乖，那么优秀，却只能拼命讨好别人，才能得到人家手里漏的那么一点儿好处。

现在连这么一点儿好处都要失去了，这怎么行！
