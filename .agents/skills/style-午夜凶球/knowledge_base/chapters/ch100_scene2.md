# 第100章 - 场景2：家长们吃饭聊天

"孩儿们，吃饭了。"

李建国的声音传来，天大地大吃饭最大，一群人拍拍屁股往回走。

众人依次坐好，开场的仍是成年人之间的商业互吹。

"来来来，边吃边说，一会儿都凉了。"，李建国招呼大家吃饭。他给江青山和楠楠爸都倒了酒，"少喝一口，领导不都会开车吗？"

看看媳妇儿脸色都没有勉强的意思，两人都高兴的拿起酒杯。

徐楠楠的父母跟李建国两口子还算熟悉，跟江青山两口子倒是第一次见，不过刚才一起忙忙活活的也算熟悉起来了。

刚开始大家还有些端着，随着酒过三巡，大家说话就没了顾忌。这顿饭大家吃的也是热热闹闹，宾主尽欢。

"哎呀，李哥，我平时就跟楠楠说，楠楠，你就看住李观澜，她干啥你干啥，指定没毛病，人那孩子心里有谱儿，靠得住。"，楠楠爸酒量一般，开始往外秃噜大实话。"你喝水上厕所频率都跟她保持一致，整不好有啥诀窍完事儿学习能好。"

楠楠妈的脸腾的一下红了，在桌子下面掐住老公的大腿。

"咋了，媳妇儿，你掐我干啥，我说的有啥毛病吗？"，楠楠爸看着媳妇儿一脸纳闷儿。

"没有啊，谁掐你了，你喝多了吧。"，楠楠妈让自己尽量看起来淡定一些，下次再也不带这玩意出来吃饭了，丢人现眼。

"哎呀，都希望自己家孩子好，咱们心情都是一样的。"，李建国怕楠楠妈尴尬死，赶紧打圆场，还不忘拉上一直没说话的江青山，"江哥家孩子也优秀啊，我都告诉李观澜，你没事儿跟江流学学。"

李观澜看他爸，啥时候的事儿？

李建国的手指在嘴上迅速晃了一下，让她别往这儿瞅，赶紧转过去！

当着孩子面儿说假话他还有点儿不好意思，那他也不能说我觉得我闺女最好，不用跟别人学。

"净说那个。"，有人夸他儿子江青山也高兴，花花轿子众人抬嘛！你夸夸我家的孩子，我夸夸你家的孩子，多好。"我就觉得好好这孩子比江流强，有礼貌贴心，啥时候看见都笑盈盈的。"

江流:谁没礼貌咋的，我看见人也不翻白眼啊！

三个老爷们越喝兴致越高，媳妇儿在外面都给留着面子谁也没说啥。到后面仨人唠嗑儿越唠越没边儿她们也懒得听了，干脆换座位坐一起聊天儿去了。
