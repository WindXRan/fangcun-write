# 第116章 - 场景3：咖啡厅里林秋敏与张自珍聊天

坐到咖啡厅里，指望张自珍先起话头是不可能了，于是e人林秋敏先出声儿了。

"小珍呐，上回萱萱回去过年大家伙儿都夸她呢，懂事儿，你培养的真不错啊。"，这话林秋敏说的违心，她其实挺生气的，给孩子养太好说话了一点儿脾气没有，在她姥家里谁都能指使她干活儿。

"哦，谢谢夸奖，李观澜也不错。"，张自珍现在只学会了怼人，人情世故这方面还是差一点。让林秋敏听起来感觉她好像说的挺勉强的，她就有点儿不乐意了。

咋的，嫌弃我们没给孩子培养好啊？你培养那个方式我们还看不上呢，于是她说话就有点儿变味儿了。

"就是太软了，这性格容易挨欺负，也不知道小时候吃过多少亏，像之前那个韩啥来的。"

"这是我的疏忽，我已经深刻反省过了。"，张自珍非常认真的说道。

林秋敏没招儿了，感情她就是这性格，她叹息道，"你这性格，容易吃亏。"

她这话有点交浅言深了，张自珍有些不解，怎么就说到这儿了。

"算了，你要是不嫌弃我，以后有事儿可以跟我说说，咱俩也当朋友处。"，林秋敏看她还有些懵懂的状态主动邀请道。其实她也怪可怜的，情商低，外表看着冷，其实还挺好说话，这样的人最容易吃亏。毕竟是好好亲妈，有啥事儿她可以给出出主意。

怎么又说到这这儿了，张自珍觉得林秋敏的思维跳跃的真快。

她先将这个想法抛之脑后，对林秋敏指了指响铃的电话，示意先中止一下聊天，"喂？"

"张总，何氏医药又来人了，想跟您谈一下技术授权的事儿，他们愿意再提高一千万的技术使用费。"

电话那头应该是杨秘书，这个声音林秋敏听过。

"不用谈了，他们家做生意的方式我不喜欢。"，张自珍果断拒绝道。

多少钱，林秋敏咋舌。

差点忘了她是有钱人，算了，我还是可怜一下我自己吧。

张自珍挂断电话发现林秋敏不再说那些让她摸不着头脑的话，剩下的时间只有一搭没一搭的拉起了家常，直到进场。
