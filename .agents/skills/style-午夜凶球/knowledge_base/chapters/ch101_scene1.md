# 第101章 - 场景1：噩梦惊醒

看到这个场面，林霆不知道触动了哪根儿神经，他也凑上去了，“我结，还是我结。给我个机会感谢叔叔阿姨，义父义母。”

江流一把抱住他，那都不是一辈儿人，有你啥事儿，别瞎掺和了。

传承，这就是传承。

李观澜目瞪口呆。

江流两人拉扯着拉扯着就变了味儿，仿佛生出了火气，打起来了。

李观澜在一边儿劝架:

“别打了，别打了。”

两个人离李观澜越来越近，她感觉眼前一黑，不知道谁的拳头抡上了她的眼眶。

李观澜倏然坐起，右手捂住眼眶，摸了两下放下心来。还好，还好，原来是做梦，可能是昨天他们结账的画面给她的印象太深刻了。她的眼眶还是完好无损的，周一不用顶着熊猫眼去上学。

外面的天还黑着，大概是刚才的梦实在是太真实，让人心有余悸，她躺在床上睡不着了。
