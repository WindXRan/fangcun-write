# 第116章 - 场景2：演唱会场外汇合

大概是刚高考完，来旅游的人格外多。一行人费力挤出车站，打车来到会场，李观澜给林霆打电话成功汇合。

谢玉露和林霆是回京市参加高考的，几个人大概十几天没见了。谢玉露像是跟她们分别了半辈子似的，见到了就黏一起撕不开了。

"别黏糊了，赶紧进去吧。"，林霆给大家分工作牌，又埋怨李观澜，"义父义母来你咋不说一声呢，我要两张票就完了，还花那冤枉钱买。"

一旁跟着的工作人员对大少爷刮目相看，还知道花冤枉钱这个词儿，这真是我见过最接地气儿的大少爷了。

还有这个姨父姨母是谁家亲戚，不管了，都笑脸相迎就对了。

"那也不能逮着只羊可劲儿薅，差不多得了呗。"，李观澜戴好工作牌，四处张望了一下，没发现李萱的踪迹。她拨了电话给李萱，没几秒那边就接了电话。

"萱萱，你到了吗？"，李观澜提高了音量，这里到处是各地来的粉丝，打卡合影干什么的都有，不大点儿声根本听不见。"哦，我们在入口，你快来。"

不一会儿李萱和张自珍急匆匆的赶来了，李观澜只来得及跟亲妈打个招呼就进场了。演唱会七点半开始，她们得进去干点儿活儿，不能给林霆丢面儿。

"认识一下，我妹妹李萱。萱萱，这几个都是我朋友，徐楠楠，谢玉露，林霆，这个你应该认识不介绍了。"，李观澜给她们互相介绍了一下。

"你们好啊。"，李萱乖乖得跟大家打招呼。

"你好。"，徐楠楠看着李萱，感觉很奇妙，这个才是真正的'李观澜'啊。

"我告诉你，你不许欺负李观澜哦，我家很有钱的，你欺负她我就让你家破产。"，谢玉露绷起小脸，电视剧里都这么演的，假千金都不是省油的灯，先给她个下马威。

"不好意思，我给她控控水。"，林霆歪了歪谢玉露的头，李观澜那个性格不欺负别人就不错了，谁能欺负她。

"正品义妹你好，一看你就跟赝品不一样。"，单看李观澜还没什么感觉，李萱一来就很明显了，她跟林秋敏七八分相似，一看就是一家人。

"是呢，当初那护士也是缺德，要没抱错李叔他俩省老心了。"，江流补充。

"你俩少说话吧，萱萱跟我是一伙儿的，你俩说我她不乐意。"

"真能给自己脸上贴金，我看人家是有点害怕你吧……"

一群人打打闹闹的进去了，场外徒留李建国夫妇和张自珍面面相觑。

"咱喝点儿啥去啊，别搁这傻站着，还有俩点儿开场呢。"，李建国提议。
