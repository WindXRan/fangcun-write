# 第149章 - 场景3：沈琳母子被送出国

林霆把沈飞拯救出来，放到地上。他罕见的对着他露出一个笑来，"去找你妈吧。"

沈飞被他笑的后背发毛，飞奔着向宴会厅门口跑去，门口的沈琳面色苍白的跟注视着这边的林霆对视上。

沈景明根本不拿沈飞当回事儿，按这个趋势下去，她和沈飞连汤都喝不到。她本想搏一搏，没想到被林霆的朋友搅和了。

沈琳低下头，搂过沈飞匆匆离开。

沈景明特意来她们这桌跟林霆的几个朋友打招呼，看到亭亭玉立的李观澜又有些心动。

谢玉露虎视眈眈的看着他，叔叔，你看啥呢？！林霆可配不上我们观澜啊！配我还行！

沈景明叹气，行吧，过河拆桥的事儿也确实不地道。

当晚，沈琳看到沈景明的来电，再难升起什么期待的心情了。

她忐忑的接起电话，"景明，今天沈飞说想去参加哥哥的升学宴，我看他太可怜了才……"

沈景明等她说完了才开口，"你这几天收拾一下东西，我已经派人给沈飞申请了国外的学校，你们准备一下，过两天就走吧。"

沈琳有如被五雷轰顶，这不就是变相的流放。她说话都开始结巴起来，"景……"

沈景明只给她一次开口的机会，打断她道，"别人不知道，你是知道的。沈飞的沈，是沈琳的沈。当初你能在林霆他妈面前耀武扬威，是我允许你耀武扬威。让她同意跟我离婚是你唯一的用处，也是你唯一的功劳，你不应该一而再，再而三的试探我的底线。你对自己的定位不是很准确，把自己养得胃口太大了，不适合在这儿留着。以前我懒得讲，这次话我已经跟你说明白了，你带沈飞在那边好好过吧，生活费会给够，别的就不要想了。有合适的你在那边结婚也行，到时候我会给你一笔钱。"

沈景明说了一大段话有些累，休息了几息继续道，"别回来了。"

沈景明已经挂断了电话，沈琳大脑一片空白，好半天才想起来放下手机。她突然对着手机痴痴笑了起来，笑得上气不接下气，笑得涕泗横流。
