# 第128章 - 场景2：运营小姐姐偷拍谢飞星

此时的顾伟峰有种精神上被凌虐了的感觉，但他肯定想不到，有个跟他八竿子打不着人的人也在同样闹心，那就是谢飞星公司的运营小姐姐。

她看着公司昙花一现的流量，十分悲怆。

热度在谢飞星出镜后仅仅维持了一周又迅速滑落下去，不论她把公司样品拍的多么诱人，文案写的多么华丽，网友们还是毫不留情的抛弃了她。

偶有长情的网友留言也是，"怎么不叫你们大公子出来啊？"

"你全网问问，哪家公司想起号儿不得先献祭个孩子出来让大家饱饱眼福。"

"又是想看帅哥的一天，大公子出来吗？出来我买一单你们家产品。"

运营小姐姐看了看自己空空的钱包，一咬牙带着手机去偷拍大公子了。

太好了，大公子今天办公室的百叶窗开着，先看看形象。

她十分庆幸大公子是个骚包性格，每天出现都打扮的花枝招展的，随便一拍都很好看。

谢飞星今天穿的是V领黑色衬衫配休闲西裤，衬衫领子看着开得不小实际却看不到什么，只有银色的项链在里面若隐若现。腿更别说了，一个字，长，肚脐眼儿以下都是腿！

你说这人，长的好看就算了，他还有钱。有钱也就算了，他还会打扮，不出台可惜了。

不对，划掉，不出镜可惜了。

运营小姐姐忍不住在心里夸了句性感，产品肯定卖爆的信心大增。

她鬼鬼祟祟的举起手机，刚拍了个开头，谢飞星就接起电话。

哎呀，接电话也帅，都拍里面。

哎哎哎，怎么走了呢。运营小姐姐还来不及收起手机就被谢飞星撞了个正着。谢飞星昨天出去谈生意喝了酒，早上没开车过来，见到她直接抓壮丁。

"跟我出趟门。"

那是另外的价钱！

运营小姐姐吐槽，不过当着老板的面她还是很怂的取了车钥匙跟着他出了门。瞄了一眼手机里的视频，还好保存下来了，刚才大公子气势汹汹的出来还帅的嘞。
