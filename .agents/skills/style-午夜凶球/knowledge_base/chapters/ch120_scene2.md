# 第120章 - 场景2：烟花下的许愿

"砰！"

巨大的烟花在游乐场上空炸开，点点星光闪烁着四散开来。

谢玉露:"快许愿！"

林霆摸摸她的额头:"你怎么看见什么都能许愿，灵吗你就许。"

昨天演唱会上碎个灯泡她也能许个愿，一天天的愿望可多了。

谢玉露十指交叉，握拳闭眼，"不管不管，反正就要许愿。我希望我们大家能够永远在一起，不要分开。"

林霆好笑的看了她一眼，望向天空，那我就许愿，谢玉露的愿望成真吧。

徐楠楠闭眼秒跟: "过一本分数线，过一本分数线，过一本分数线。"

李萱在心里默念:好像已经很好了，没什么需要许愿了，那就希望大家都身体健康吧。

李观澜仰望星空，表情虔诚，"希望世界和平。"

江流嗤之以鼻，"假大空。"

李观澜睁眼看他，"你不假，你许啥愿望了？"

江流不语，他要留点悬念。

李昭拆穿他，"他刚才许愿说晚上想吃肘子，说好几遍。"

"费那个事，跟我说啊，观澜女神会满足你的。"

"她不会，她只会分走我一半肘子。"

"……"

李昭笑嘻嘻的在一旁看热闹，李萱在看他，你许了什么愿望呢？

李昭许了什么愿望？

他许愿，今天这一切不是梦就好。
