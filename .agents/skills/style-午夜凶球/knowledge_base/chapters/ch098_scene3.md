# 第98章 - 场景3：高三学习压力与疯癫状态

李观澜那边现在气氛确实紧张，还有不到四个月高考，班里的同学状态肉眼可见的与上学期不同。大家抓紧一切时间学习，不学习的也安静了许多，尽量不打扰学习的同学。胡野和王煜甚至无师自通了用手语吵架，但显然除了他们俩别人都看不懂，猛的一看还以为他俩手在抽筋。

谢玉露那边她哥又给她找了两个老师每天晚上给她补课。正常像谢玉露和林霆这种有专人给上课的情况，他们不来在家学习的效果会更好。

但是林霆觉得他俩有病，在家名师辅导环境舒适的学习他们俩学不进去，反倒是回学校跟着一屋子像是精神错乱的同学们一起倒是能静下心来。

"啊！为什么为什么为什么！这道题为什么不对，为什么我看答案看不懂，我已经连答案都看不懂了吗？！"

李观澜憋了一节课，终于忍到下课想上个厕所被吓了一跳。

这咋又疯一个。

他的同桌默默换了两张答案给他，"你拿错了。"

"哦，谢特！各位兄弟姐妹对不住了。"，他拱手对大家抱歉。

"没事儿，理解。"

说话这位仁兄昨天刚疯完，战绩为在大庭广众之下怒吼，"shabby,shabby，为什么选这个，I look shabby吗？"，一战成名，喜提外号shabby哥。

李观澜摇摇头，抗压能力太差，不像她，心理素质强大。不说了，她得去厕所了。

没一会儿林霆甩着手狂奔进教室，"谢玉露你赶紧去看着点儿李观澜，她刚才迷瞪的差点儿进男厕所。"
