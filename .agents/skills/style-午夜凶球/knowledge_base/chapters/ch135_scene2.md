# 第135章 - 场景2：张自珍买房

"小珍呐，这回来住哪儿啊？"，林秋敏问道。

她们也是突然才收到张自珍来的消息，还没来得及给她订酒店。不过酒店再干净豪华也不如家好。

李建国他们俩买了套新房子已经装修完大半年了，放味儿也放的差不多，她想要是张自珍不嫌弃就让她们在那里凑合住几天。位置就在凤凰小区对面，也方便她们有点事儿一起行动方便。

"你要是不嫌弃……"

"我在你们小区对面买了几套房，这次住那儿。"，张自珍的语气好像她买的是几颗白菜一样平淡。

"你这是啥时候买的啊？"，林秋敏被张自珍的大手笔震惊到，她把这话的内容成功消化后问道。

"不记得了。"，张自珍看起来不怎么在乎的样子。

张自珍不愿意多说，杨秘书却非常有眼力见的接话，"差不多一年了，装修都已经装完了半年。一共买了四套，萱萱和好好每人两套。都在一栋楼里，随她们喜欢是住是租。"

林秋敏和李建国对视一眼，那不是才知道孩子抱错了就买了。别看她平时态度很冷淡，其实她也在用她的方式在关心孩子。

除此以外，李建国还生出了一些对自己恨铁不成钢的心思，他开始还为给俩孩子每人攒点钱儿沾沾自喜，这一下就有了危机感，张女士这也太阔气了。不行，虽然张女士有钱，我也得尽量多给孩子们攒点儿，不能让她比下去了。

李萱和李观澜也皆是对自己身家财产大幅度上涨感到震惊。

李萱脸色微红，感动道，"谢谢妈妈。"

李观澜则非常欢快，"谢谢妈妈！"

付出有被人看到，并发自内心的真诚回应，这种感觉让张自珍很愉悦。

吃过饭，李建国两口子跟着张自珍去她买的房子参观了一下，也算是认认门。

张自珍的原则是既然买了就一步到位，几套房子都是大平层，装修也没少往里砸钱。

一进屋，李建国和林秋敏就能感受到金钱的气息扑面而来。
