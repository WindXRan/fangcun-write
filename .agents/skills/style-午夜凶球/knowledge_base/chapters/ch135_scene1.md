# 第135章 - 场景1：张自珍到来

"开了这么久的车很累吧，辛苦了辛苦了。"李建国握住杨秘书的手摇了摇，"快往屋里进，102那个房间是咱们的。"

"不辛苦不辛苦，您太客气了。"，杨秘书露出发自内心的笑容，这趟出行按出差标准每天补贴一千块。不止如此，想也知道这两天他的任务就是陪着少爷小姐们逛吃玩儿，她可以蹭吃蹭喝蹭玩儿，多么美妙的一次出行啊，我爱出差。

张自珍在李萱和李昭期待的眼神下，莫名其妙就提前了几天来了李观澜这儿。她都不知道自己是怎么脑子一热就答应的，来了也没什么事儿做，她来这么早干什么？算了，就当是度假了。

"小珍呐，你来的正好啊。"，林秋敏挎上张自珍的胳膊往饭店里走，"正好这几天我都忙不过来了，你来了还能帮帮我。"

？？？

张自珍感觉自己已经有点习惯了林秋敏的自来熟，她们的关系已经进到next level了吗？不知道，反正她现在不会抗拒林秋敏跟她的肢体接触了。

李昭像个跳马猴子似的在李观澜身边叽叽喳喳，"啥时候能赶海，李萱说这边儿有大海，能捡到海鲜。"

李昭在视频上看到过好多赶海的视频，他急着尝试那种用小夹子把海鲜一个个捡到桶里的美妙感觉，一定又解压又成就感。要不然他也不会这么急着来，急到每天张自珍下班他都要在她附近转上几圈，知道张自珍福至心灵，提出要带她们早几天来。

"唔。"，李观澜看了眼李萱，赶海是行，但是人比海鲜还多这个事儿怎么解决，以现人们对赶海的热情来看，能捡到两只螃蟹都属于赶海圣体了，一般只能捡到贝壳。对，纯贝壳，里面没有肉那种。

李萱笑眯眯回看过来，我也没说能捡到什么啊，他听见赶海两个字就开始着急了就我也没办法。

李建国这是第一次见李昭，不过虽然是初见面，他还是无可避免的对李昭生出一种熟悉感，这简直就是初中时候李观澜的翻版啊，看到他甚至有种时空错乱的感觉。

"叔，叔叔你看什么呢？"，李昭虽然被李建国的眼神看的有些发怵，但也还算礼貌。因为出门之前李萱已经三令五申，不许对李建国夫妻出言不逊，不然你将见识到李观澜引爆地球。

"哦，没事儿。"，李建国这才反应过来，他给人孩子吓到了，"那个，发型不错，你姐还有两罐发泥你要吗？彩色的。用起来造型更炫酷一些。"

李建国絮絮叨叨说了许多，中心一直围绕着给李昭的发型建议，完全没有任何对他的偏见或说教。

李昭瞬间慕了，李观澜你过的都是什么好日子啊？当初抱错的怎么不是我！
