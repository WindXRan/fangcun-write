# 第130章 - 场景3：李观澜坐谢飞星的车回家

李观澜和谢飞星在后座面面相觑，她颇有些不好意思的对谢飞星说道，"孩子小，不懂事儿哈，您别和她计较。"

谢飞星侧头看她，那个到底是谁妹妹啊？！

"嗯，习惯了，你对她倒是真好啊。"，谢飞星食指在腿上无意识敲击，白皙又骨节分明的手与黑西裤的对比格外强烈。

"那当然了，露露是我朋友嘛！"，她语气理所当然道。

"哦，既然是朋友那，那个奖金……"，谢飞星语气带出一丝迟疑。

"这个，一码归一码，接这活儿的时候我俩还不是朋友。"，李观澜赶紧截住他的话，老话说的真没错，越有钱的人越抠，说话咋还能不算数呢。

"哈哈哈哈哈，知道了。"，谢飞星没忍住笑出声，"你还挺有原则的，别急，等出分了就打给你。"

李观澜被他笑得摸不着头脑，有这么好笑吗，说到出分，高考成绩还有两天就要出来了，这两天她就得回家了。

谢飞星一只手拄着头，目不转睛的看着她，李观澜背后发毛，"你这么看着干什么？"

"我看看钱串子成精长什么样儿。"，谢飞星调侃道。

嘿，你这人，难怪谢玉露烦你！李观澜转身侧坐，用后脑勺对着他。

"不让看！"

谢飞星觉得她这个造型更好笑了，"李观澜，你转过来呗，这么坐着你裤子后面的大喇叭花对着我，我想笑。"

李观澜转过来恶狠狠瞪了他一眼，你能不能别说话。

终于到家了，李观澜飞速的下了车，把还想说话的谢飞星隔绝在车里。

住嘴，男人话太密有性缩力！
