# 第142章 - 场景3：两位家长偷听与秘书博弈

李观澜:"谢玉露，你上大学以后也尽量只能跟我一起玩儿哦，你能不能让叔叔暗箱操作一下把咱俩放一个寝室。"

江流:"同意。"

林霆:"同意。"

徐楠楠:"同意。"

谢玉露:"……同意。"

这么一看跟观澜一个学校还有个好处，她俩能住一起。跟林霆一个学校也不能住一起，咦，我在想什么，哈哈，害羞。

谢震元脸色绯红的女儿:好的，一定。我给你们两个在学校周边买套房子，你俩住外面。

他没忍住出声儿:"露露啊~"

谢玉露正在幻想中，看见老爸吓了一跳，恼羞成怒，"爸，你喝水喝了半个小时了，怎么还没喝完。"

谢震元:我是你老子，我爱喝多长时间就喝多长时间，房子是我赚钱买的，你管的着？

"喝完了喝完了，这就上楼。"

儿女都是债啊，谢玉露这笔债他一定欠的巨款。

沈景明也听了全程，最后感叹道，"你这个同学是个人才，爸爸觉得你们在一起很合适。"

沈景明虽然现在是个商人，但曾经文人的思想还刻在他骨子里，他就喜欢学习好的孩子，这个女孩子他很喜欢。聪明机灵，也不吃亏，跟林霆在一起一定能给他很大帮助。

"叔叔！"，谢玉露气急败坏，你怎么乱点鸳鸯谱呢？！

"……好吧！"，沈景明叹气，怎么原来语音没挂断吗？

这个'好吧'的语气就很神。

林霆凉凉的接了一句:"爸，忘了告诉你了，这个人才就是你的野生闺女。"

沈景明倒吸一口凉气，想起去年过年有朋友给他打电话，说在商场看到林霆好像被一个太妹挟持了，问他需不需要帮忙。

朋友拍的照片还在他手机里，虽然后来问了林霆得到否定的答案，但李观澜独立特行的形象依然深深的印在了他的心里。

沈景明:……那还是做朋友吧。

不过，他知道，在林霆离开家这段时间对他态度的转变，跟他这些朋友应该有很大关系。

既然如此，野生闺女他也认了，升学宴，他也凑个热闹吧。

沈景明此时的思想和谢震元默契的接轨。

于是，就出现了两个秘书站在帐桌前职业假笑互相谦让的一幕。

"你们俩……"，李卫东不知道两个人葫芦里卖的什么药，抬头看着两人表示疑问。

"你们俩不写我写。"，后面人急坏了，把两个职场精英挤开。
