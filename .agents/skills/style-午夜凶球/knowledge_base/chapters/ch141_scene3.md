# 第141章 - 场景3：两位秘书随礼与群聊回忆

这边高慧云刚嘀咕完，那边又上来两个互相谦让西装革履的商务人士。

"您先请。"，左边的小伙儿礼貌谦让，我看看你们出多少。

"还是您先请。"，右边的小伙儿温文尔雅，还是你先出价吧。

两人分别是林霆父亲沈景明和谢玉露父亲谢震元派来的秘书，两人突发奇想要来随份子的起因是昨天李观澜她们在群里打语音被双方父亲听到。

让我们回溯时光到昨天晚上，彼时这些年轻人在群里闲聊。

李观澜:露露，你明天快到的时候告诉我一声，我去接你们。

谢玉露:好嘞。

李观澜:"露露，你大学报哪个学校想好了吗？"

谢玉露:"我正在纠结，你和林霆我只能选一个。"

谢父到楼下喝水，听到林霆两个字马上竖起耳朵。

李观澜:"你先确定金融和表演你喜欢哪个。"

谢玉露:……那当然不喜欢金融。

李观澜:"唉～那你跟我吧。"

跟林霆分开四年，谢玉露又有些犹豫。

李观澜:"你看哈，咳，林霆在吗？"

林霆:"嗯。"

李观澜:"以下我所有的言论都是基于我个人对人性洼地的阴暗揣测，与你本人的道德水平没有直接关系哈。"

林霆:"放。"

李观澜:"好滴，露露，你看哈，假设你觉得未来你一直都会喜欢林霆的情况下，我们推测一下林霆未来的生活轨迹。

从继承权这方面来说，有两种可能。一，他成功夺权，过上了春风得意的生活。二，夺权失败，过上了穷困潦倒的生活。

那么你们两个就有四种可能。一，他春风得意的和你在一起了。二，他春风得意的和别人在一起了。三，他穷困潦倒的和你在一起了。四，他穷困潦倒的和别人在一起了。你选吧！"

谢玉露肯定选第一种啊。

林霆:"哎，我有话要说。"

李观澜:"不行，你现在是道具，不样说话。

好，他春风得意的跟你在一起了。那么又有两种可能，一未来你俩一直幸福。二，他出轨了。"

谢玉露扣1。

李观澜:"好的，那恭喜你的未来有12.5%的机会过上了每天无所事事，送林霆上班，等林霆下班的美好生活。"

谢玉露一听，这也没有很美好，"那我可以干点儿别的啊。"

李观澜:"对呀，你可以干点儿别的，你想我要做点我喜欢的事，于是……"

谢玉露:我喜欢啥嘞，喜欢林霆。

李观澜:"那么，你决定做点你擅长的事，大学四年，你跟林霆学了一样的专业，金融学。于是……"

谢玉露:我擅长啥嘞，擅长跟着林霆。金融学，学不明白一点。

李观澜:"总结:如果未来有一天林霆不干人事儿，你的世界将天崩地裂。那么你现在重生回来了，这个世界再给你一次机会，你的大学四年选择如何度过。"

谢玉露:"我重生了，这次我一定要远离渣男。我的命运由我自己掌握，就从选择自己喜欢的大学开始。前世的债，我要一笔一笔的讨回来。"

林霆:……你俩配合的还挺好，能不能给编乎卸载喽。

徐楠楠:"好好，你这两天又出书啦？"

李观澜:"电子评书，有语气渲染带入感更强。那个，露露，我就是随便假设一下，你自己想想大学这四年你到底想怎么过吧。"

林霆:"谢玉露，去做你想做的事儿吧，我们都该长大了。"

谢玉露:"我想想。"

江流:"谢玉露你好好想想吧，我都看不下去了，他今天敢内涵你幼稚，明天就敢打你，你还是离他远点儿吧。"

林霆:……江流的报复虽迟但到。
