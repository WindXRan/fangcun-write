# 第97章 - 场景2：朱文静被拒与李卫东摊牌

朱文静眉头紧锁的从李家出来，出了什么问题，怎么孙老太突然这么不待见她，厌恶之情几乎溢于言表。

她想起孙老太见她如了鬼的神情，连门都没让她进，说她一会儿要出门就不招待她了，言下之意不就是要她赶紧滚蛋。

她也没再多说，放下水果就识趣的离开了。对门的邻居出门扔垃圾，在门口磨迹了好一会儿才走，不就是为了看她笑话，她才不会给她们机会。

她想来想去没个头绪，正心烦时远远看到了李卫东的身影，她心头一喜，表情却变得黯然。

李卫东不知道在想些什么，竟然看也没看她一眼从她身边擦肩而过。

她愣了一瞬马上换了策略叫住了他。

"李哥。"

李卫东这才看到她停住脚步，他有些诧异，不是说以后会保持距离吗，他还没开导她们呢，怎么不到半个月就来了？他问道，"文静，你怎么来了，是有什么困难了吗？"（大家不要觉得叫名字很亲密啊，就算在单位里一些表面关系过得去的同事也这么叫。）

朱文静用偷偷掐了自己一把，眼眶顿时变得通红，"李哥，我听说你和嫂子要离婚？是不是我的原因，这些年我拖累你们太多了，用了你们那么多钱嫂子一定不高兴了吧。"

她擦了擦眼泪，透过缝隙看着李卫东的表情，以往她只要这么问，李卫东就会觉得张自珍是因为钱的问题在跟他闹别扭，然后安抚她和她没关系，此刻她依旧用了老办法，老套但管用。

"唉，跟你是有点关系。"

不是，你怎么不按套路来，这叫我怎么说话。朱文静一时间无语，尴尬在原地没有接话。

好在李卫东也没有要她说话的意思，他接着说道，"最近我也听了一些风言风语，确实意识到以前我的做法有些过于没有边界感，非常容易引起别人的误会，给你也带来了一些不好的影响。出于这方面的考虑，以后我们如无必要还是尽量不要见面，家里有困难你就给我打电话，能解决的我还是会帮忙。"

这话在朱文静听来如晴天霹雳，李卫东这意思不就是以后要和她们泾渭分明，她回过神来立刻以退为进，羞愧道，"李哥，这些年你照顾我们已经够多了。你的意思我明白了，以后我们一定不麻烦你。"

朱文静想着李卫东这么要面子的人听了这话，肯定会给她解释一下他不是这个意思。

"你不要多心。"

果然，李卫东这话好像给朱文静吃了一颗定心丸，可没想到他后面的话让她有些惊恐。

"老陈的儿子我一定供他念到大学，但我现在经济状况也确实不比从前，家里负担重了许多，可能没办法像以前一样承担你们的各项开支。"

他已经意识到自己在朱文静母子身上投入超过正常水平的钱和关注，甚至因此引起了别人的猜疑。他应该让自己的头脑冷静下来，不能再继续毫无底线的供给了。这种供给或许对他和朱文静母子来说都并不是一件好事。

而且，他要攒些钱了，他至少应该给孩子们留下一些什么。这虽然弥补不了过往他在孩子们生活中的缺失，但至少能让们感受到他的态度。

"那是当然，我们本来也是这么想的，已经很麻烦你了。"，朱文静脸色苍白的应对着，心里慌张的不能自已，不停的想着下一步该怎么做。

李卫东认为两人已经达到共识，欣慰的点点头，"那好，那就这样，我先回去了。"

朱文静无意识的点头，眼神涣散的盯着李卫东离开的背影。
