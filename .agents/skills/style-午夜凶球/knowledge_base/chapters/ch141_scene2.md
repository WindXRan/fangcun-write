# 第141章 - 场景2：李卫东主动请缨写账

吴欣从饭店里匆忙走出来，急火火的对李建国说道，"姐夫，没看见我姐，你找个人替我一下，我车让人刮了，我去处理一下。"

"你去你去。"，李建国四处踅摸人，吴欣走了，写账的就剩高慧云一个人了，那收不过来。

老赵？拉倒吧！认识的字儿还没有他脸上的痦子多呢。

李卫东在门口站不住了，总算能帮个忙主动请缨道，"我来写吧。"

那多不好意思啊，这么大个文化人给他家写账，李建国拉着他往里走，"行，就在门口，我带你过去。"

到了账桌儿那儿，高慧云果然已经手忙脚乱了，先收你的，再收你的，再收你的，一个一个来，都能收到啊。

紧急安排了李卫东帮入座写账，高慧云的压力骤减，一个写账，一个数钱，速度马上快了起来。

李卫东写了一笔好字，高慧云看了立刻惊为天人，"哎哟，您这字写的真不错啊，李建国还认识文化人呐。"

李卫东笑笑没说话，从包里掏出一个牛皮纸袋，"上帐吧。"

他现在手里没太多钱，给李昭留了一部分，剩余的就分给萱萱和好好上大学，正好每人十万。

高慧云瞪大眼睛，还有这么随礼的呢，一般这么大额都直接转账了，这是怕人家不收，他是不是干啥亏心事儿了。
