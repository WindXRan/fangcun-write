# 第99章 - 场景2：农家乐聚会

李观澜还想着放假在家刷几套题呢，突然得到通知老爸要带她出去放风儿。

"问问楠楠她们谁有时间，都让过来，爹妈也都带过来。孩子上学不轻松，陪读的也不轻巧。我跟你江大爷说好了，那天他们也来。"

李观澜第二天到学校一问，很好，不愧是她的朋友，一听连吃带玩一条龙都有时间。

于是周六那天一群人浩浩荡荡到了农家乐，大人们一挥手，玩儿去吧，饭好了叫你们。

初春的天气并不暖和，不过对十七八岁的孩子来说，温度跟玩儿比不值一提。她们几个在偌大的院子里撩猫逗狗，每个隐蔽的犄角旮旯都要捅咕一下，看看里面到底有什么。
