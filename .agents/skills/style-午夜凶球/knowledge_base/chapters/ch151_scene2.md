# 第151章 - 场景2：登山途中的诗词接力

碎嘴小分队往上走，走了一会儿刚刚恢复的那点儿体力再次消失不见。

"累——"，徐楠楠哀嚎。

"别说这个，越说越累，我们说点儿别的。"，李观澜被她一说也觉得累。

"说什么啊？"，谢玉露扶着山壁往上蹭，越走越慢。

"说点让人听了有力气的，比如……会当凌绝顶，一览众山小。"，李观澜说道。

"好，那我说少年自当扶摇上，揽星衔月逐日光。"，江流接上。

不知道是被吸引了注意力还是文字真的有力量，每个人都觉得脚步轻快了些。

李萱想了想说道:"少年负壮气，奋烈自有时。"

还好还好，我想的那句没被说到，徐楠楠松了口气，"恰同学少年，风华正茂，书生意气，挥斥方遒。"

林霆挠挠头，:"须知少日拏云志，曾许人间第一流。"

谢玉露磕磕巴巴，"少年，少年自有少年狂，身似山河挺脊梁。"

Umm……

行，也算。

该你了，大家一起看李昭。

李昭把头撇开，"我能走动，我不玩儿。"

他在心里哭唧唧，谢玉露，你怎么抢我台词儿，我拢共就会这么一句。

"哈哈，好吧好吧。"，李观澜摸摸李昭毛茸茸的头发，手感还怪好的。

李昭脸红，学习了不起啊，回去我就学。

李观澜低头在他耳边嘀咕了一句，李昭立刻反口，"我也有，我也有，我的是再见少年拉满弓，不惧岁月不惧风。"

"好，说得好。"

"小昭弟弟说的不错啊~"

"哟，你看人家孩子……"

赞美声自四面八方响起，甚至还有不认识的人夸奖起李昭。

李昭即不好意思又骄傲，不好意思这句话其实不是他想起来的，骄傲但是是从我嘴里说出来的。

他高高扬起下巴，接受了大家的赞扬。

大不了回去我背一背好了，下次我就会了。

路人看着这群你拉着我，我拽着你，看着脚步已经疲劳至极，脸上却仍然意气风发的少年人，皆是露出会心一笑。

年轻真好啊~

神采飞扬，豪情万丈。
