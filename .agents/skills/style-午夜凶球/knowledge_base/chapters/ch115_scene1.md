# 第115章 - 场景1：高考后的无聊假期群聊

李观澜开始了她醉生梦死的假期生活。

睡觉睡到自然醒，醒了玩手机，玩饿了吃饭，吃完接着在卧室、客厅、厕所玩手机。

刚开始的两天李观澜还觉得终于过上了她梦寐以求的生活，可没超过一个星期她就觉得无聊透了。

李观澜: "啊，无聊。"

徐楠楠: "+1"

谢玉露: "+2"

林霆: "+3"

谢玉露:"+2"

李观澜: "啊，被发现了，真是没有办法啊～"

江流: "真是给你闲出屁了。"

李观澜 :"愚蠢的人类，我在楼下开你车锁开了八百个来回，车铃都快被我扒拉坏了你都不下楼打我，你是不是聋了。"

江流 :"我带耳机打游戏呢，谁能合计你这么无聊，有事儿你打电话呗！"

李观澜:"我没事儿，我就是想吵架。"

江流:"……不行你上网接单代吵呢，你还能挣点儿钱。"

李观澜:"骂别人我没有灵感。"
