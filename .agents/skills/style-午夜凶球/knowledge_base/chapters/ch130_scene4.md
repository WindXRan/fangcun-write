# 第130章 - 场景4：齐奶奶汇报，江青山夫妇聊天

好不容易到了家，李观澜进门前又被齐奶奶拦住，她神神秘秘的问道，"好好啊，你奶奶今天来了，让我劝走了，她好像有低头的意思，你妈什么意见啊。"

孙老太？她姥儿是真会忽悠啊，效果这么显著。

能有什么意见啊，张自珍和李卫东的婚姻走到了尽头也不完全是孙老太的问题，更多的问题出在李卫东身上。她好不容易和李卫东离了婚，不会再因为一两句话再跳进火坑，这些年她已经对李卫东失望透了。

而且她不认为孙老太会发自内心的觉得自己错了，估计见面了没两句话又会吵起来。对那个家，张自珍一点儿留恋也没有，应该不会想见到孙老太。

"我妈现在没什么想法，走了挺好的，谢谢齐奶奶。"，李观澜感激的看着这个可爱的老太太。

"嗨，这算什么事儿啊，没事儿。"，得到肯定的回复齐奶奶心里踏实了许多，说话也有了底气，"去吧，回家吧。"

既然有这话，那我就放心的给那老太太上课了。

"嗡。"

是手机信息，齐奶奶看了一眼，是孙老太发过来的。

"齐姐，忘跟你说了，你帮我盯着点儿有没有男人进出我们小珍家里啊。"

"好嘞好嘞。"，齐奶奶立刻就回了信息。

"小杜，晚上那个汤你多炖一些。"，等伟峰回来让他去给小张送点儿。

李观澜洗了澡就不愿意再动了，打了电话给李萱说了一声，躺在家吹空调。

李观澜不来了？

江流总感觉缺点儿啥，想了半天没想明白，最后总结一下是缺了个打手。

他走这两天江青山两口子也闲下来了，晚上两口子还有时间在家小酌一杯。

"媳妇儿，干一个，咱俩也算是跟着孩子毕业了。"

"是啊。"，高慧云感叹，"这一年把咱俩累的，冷不丁闲下来还有点儿不习惯。"

"你啊，就是操心命，闲下来有啥不习惯的。到时候你就跟秋敏你们俩，也办个健身卡啥的，锻炼锻炼身体。要不出去逛逛街，看看电影吧多舒服。"，江青山一杯冰镇啤酒下肚，浑身舒坦，"你说咱们两家也是缘分啊，孩子都差不多大，脾气也相仿以后要是能当亲家就好了。"

"别做梦了。"，高慧云一点儿没给面子的击碎老公的幻想，"你儿子根本没长那根儿筋，高考前两天我还看他往好好书包里塞橘子皮呢。好好那么出挑儿呢大学不得老鼻子人追了啊。等你儿子开窍，你等着吧。"
