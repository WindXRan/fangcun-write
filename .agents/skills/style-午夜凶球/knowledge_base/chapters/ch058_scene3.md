# 第58章 - 场景3：江流的反应

“乱成一锅粥了。”，江流评价。

“妈呀，吓死我了。”，李观澜回手就是一个铁拳，“你来咋不出声儿。”

“我给你送点儿水果，我看你也没关门啊！”，江流捂着胸口呲牙咧嘴，这一身牛劲，你咋不打死我呢。

“你们怎么知道我煮粥了，”，高慧云进屋招呼俩人吃宵夜，就听见什么粥不粥的了，“快趁热喝了吧。”

李建国:……

什么乱七八糟的。
