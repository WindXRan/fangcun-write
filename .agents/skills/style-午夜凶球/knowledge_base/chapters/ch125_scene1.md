# 第125章 - 场景1：李昭入住妈妈家

"让我想想。"

李卫东这次的打击可能有些大，约好了陈觅出院时他会来送她们回家就走了。

"妈，怎么办？"，陈觅这回是真的快哭了。

这件事显然也不在朱文静的意料之内，她喃喃道，"再等等，再等等。"

但再等些什么她也不知道，是等张自珍其实只是吓唬她们，还是等李卫东解决这件事。

朱文静'唰'的一声拉上了帘子，隔绝了周围若有若无的刺探目光。

"妈，你今天有点帅了哦！这怎么青出于蓝了呢，我今天都没插上话。我这点小技俩在你面前根本微不足道，这以后不全都得靠你罩着我了吗。"，李观澜的彩虹屁从上车开始就连绵不绝。

张自珍发现维持表情原来这么难，怎么样才能让自己笑得不那么得意呢。

"姆……"

什么声儿，三人寻找这奇怪的声源。

"妈。"

哦，原来是李昭。

"妈，我能不能去你那儿住两天。"

那当然可以啦，于是李昭背着小书包成功的入住了。
