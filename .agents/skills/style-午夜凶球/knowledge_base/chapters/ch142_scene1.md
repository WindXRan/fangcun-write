# 第142章 - 场景1：群聊中众人劝谢玉露

谢震元在一边听的心花怒放的，林霆唯一干的好事儿就是当初转学的飞镖扎到东北了，这才能认识这么一堆好朋友。再劝劝，给他俩劝的老死不相往来才好呢。

徐楠楠:"露露，我也不是那挑事儿的人，但是你俩还没确定关系他就敢跟你吆五喝六的，确定完关系以后你就是他的奴隶。"

林霆:"没啥唠的不行咱这群解散吧。。"

沈景明:……还好毕业了，口音能改回来吧？！

谢玉露:"那我们这四年都不好见面了。"

谢震元:不好见面？不好见面就对了！

徐楠楠继续:"都在京市有什么不好见面的，远香近臭听过没？你天天在他眼前转悠他将来就不知道珍惜了，男人，都那样！"

"楠楠啊？你跟谁唠嗑儿呢？男人咋的了？"

徐楠楠爸爸的疑惑声音从语音中传出来。

"没事儿，爸，我说男人都有担当。"，徐楠楠紧急避险。

"啊，那对，那是应该的……"

"我爸回来了，我换个屋，等我一下。"

林霆:"不是，你们劝她就劝她呗，老踩嗒我干什么玩意。"

江流:"听着吧，我还陪你挨了一脚没说啥呢。"

徐楠楠:"谁挨了一脚？江流，谁踹你了？"

江流:"没人踹我，你可别打岔了，还有啥要说的没有？"

徐楠楠:"哦，没有了。"

谢震元扼腕，咋就这么两句，露露看着还没下定决心呢，那个开始最能说的呢，再说两句啊。
