# 第129章 - 场景1：李观澜等人遇碰瓷老太太

李观澜她们几个围着躺在地上不动的老太太面面相觑，共享单车这么大劲儿吗，碰一下就死啦？

今天林霆说要带他们体验一下京市的胡同文化，一群人扫了共享单车就大街小巷的四处乱窜。

谢玉露听他们聊天听的正开心呢，忽然不知道从哪儿窜出个老太太到她前面，谢玉露来不及刹车，直接从老太太腿上压了过去。

老太太'啊'了一声，便倒地不起没了声息。

于是就有了开头那一幕。

谢玉露挂了电话，伸手摸了摸老太太的鼻息，还行，喘气儿呢。

"奶奶，奶奶？"，谢玉露轻轻戳了戳她。

老太太毫无反应。

"这情况得送医院去吧？"，徐楠楠绕着老太太转了一圈，没发现她有什么外伤。

"再看看。"，李观澜看这架势，觉得她们是遇到了碰瓷儿的面儿大。

大家都给面子的一起看着老太太，没人再提去医院的事儿。

她看见老太太控制不住抽动的眼皮，没说话，给了大家一个眼神，大家秒懂，又要开演了呗。

"露露，这事儿大了，我看可能是你一个寸劲儿把她撞坏了。"

"啊？那怎么办？"，谢玉露慌张道。

"送医院她万一讹咱们怎么办，要不咱跑吧。"，徐楠楠出主意。

"我看行，我刚才看了，这儿没监控。"，李萱赞同。

"但也不能把她扔这儿啊，一会儿她醒了报警怎么办？"，李观澜提出异议。

"那好说，我和林霆先把她搬那个死胡同的杂物堆里，那边儿乱七八糟的东西挺多呢，压里面出不来，先埋了吧。等晚上咱们再过来把她扔河里。"，江流无所谓的说道。

"行，反正也不是第一回了。"，林霆上手拽住老太太的手腕。

"哎呀～"，老太太慢慢睁开眼睛，"我这是在哪儿啊？"

老太太是这个胡同的住户，平时没事儿就出来碰瓷儿，她专挑年纪不大的学生，她们年纪小没注意，还心地善良，一般一碰一个准儿，三百二百的他们说给就给。

没想到今天碰到这么一群狼崽子，她这还喘气儿呢就要给她埋喽，她怕再不清醒就被抬走了，赶紧'醒了'。

"哎哟，是我老太太走路没看路，吓到你们了吧，不好意思啊。"，老太太见他们不好惹，赶紧换了策略，希望能引起她们的同情心，少赔点，赔个一二百就当她的出场费了。

"知道就好。"

一个漂亮的晃眼的小姑娘站了出来，表情恶狠狠的看着她，"给我姐妹吓坏了，现在还没回过神儿呢，掏钱，我姐妹得去医院看看。"

啥？老太太想抠抠耳朵，我没听错吧。

是的，你没听错！

李观澜推了谢玉露一下，谢玉露顺势倒在林霆怀里，晕了过去。

"你瞅瞅！给我们孩子吓啥样儿。"

"露露，你醒醒，你醒醒啊你才高考完，还有大好的前程等着你呢。"，徐楠楠拼命摇晃谢玉露，哭的凄惨。

终于有人收拾这老太太了，她算是遇到对手了，透过门缝儿看热闹的街坊四邻感叹。

"不要你多，什么医药费，护理费，交通费，精神损失费加一起赔二十万算了。"，李观澜掰着手指查数，一脸便宜你了的表情。

老太太瞠目结舌，不是，我好歹还出了条腿才想要你们二百，你这上下嘴皮子一碰就要我二十万，还是你们黑啊。

她脾气也上来了，想黑吃黑是吧，门儿都没有，我报警！好歹我还占个岁数的优势。
