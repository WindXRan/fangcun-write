# 第121章 - 场景1：孙老太检查李昭伤口

"昭昭，让奶奶看看。"，孙老太急的团团转，把李昭翻过来调过去的看了又看，检查了两遍看他身上没有别的伤口才放下心来，怒骂道，"谁家的小王八羔子下手没轻没重的，怎么专挑脸打，你跟奶奶说，我找他家长好好说道说道。"

李昭无奈的拿开孙老太的手，"奶奶，我没事，是擦伤，没有跟人打架，你放心吧。"

从小到大，李昭但凡跟别人家孩子有一点摩擦，孙老太都要兴师动众的去找人家理论理论。

一次两次也就算了，时间久了家属院儿的人都嫌麻烦，警告家里的孩子都离李昭远一点。

家属院的孩子那么多，多李昭一个不多，少他一个不少，也没人非要跟他一起玩儿，从那时开始李昭在院儿里已经是被孤立的状态。

情况变成大家玩儿的正热闹，只要李昭一过去大家马上原地解散，有过那么几次这种情况李昭也懂了，不再自讨没趣。

"真的？"，孙老太不怎么相信。

"真的，我累了，回屋睡觉了。"，李昭强忍着心里的不耐烦，回房间关上了房门。

呼，整个世界清静了。

他靠在床头上把玩着今天李观澜买给他们的挂坠，毛茸茸的，手感不错。

我挂着会不会显得娘们唧唧的，李昭想了半天，还是把它挂到了书包最显眼的地方。

算了，挂着吧，下次她见到我没挂又要揍我。
