# 第150章 - 场景2：谢震元送包致谢与谢飞星

谢震元携妻子儿子特意来感谢，看到两人难看的脸色关切到，"怎么了？不舒服吗？"

"没有没有。"，两人原地复活，并对谢震元的特别感谢表示感谢。

"听说你们喜欢包啊？"，谢震元招呼后面的秘书过来，"我特意给你们买了几个做伴手礼，来，一人一个。"

谢震元非常自然的跳过林霆，"自己选啊，看喜欢哪个。"

林霆:叔叔，你这个行为就特别不友好。

谢震元:你小子的存在更不友好，怎么没点儿自觉呢，请你了吗，你就来。

"好了，我也不在这儿久留了，你们年轻人也不自在。"，谢震元去跟其他人寒暄了，谢飞星驻足看了眼李观澜。

"看我干嘛？想拜我为师？！不收徒！"，是不是看我反应来了，今天这损招是谁想出来的，你们家你嫌疑最大。

"飞星，来跟你钱伯伯打个招呼。"，谢震元在不远处喊他。

谢飞星没说话，伸出手指点了点眉毛就去找谢震元了。

李观澜苦思许久，他是不是说我的脑袋有问题。

徐楠楠伸手给她蹭了一下，眉毛上有奶油，大概是刚才李观澜阵亡在桌子上的时候蹭到蛋糕了。

呵呵，尴尬。

是我小人之心度小人之腹了。
