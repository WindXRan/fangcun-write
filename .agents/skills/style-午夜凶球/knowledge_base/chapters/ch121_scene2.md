# 第121章 - 场景2：朱文静的野心计划

朱文静把水果放到地上，用力甩上了门。

陈觅听到声音从房间走了出来，看到地上的水果，大概知道今天又是他妈自讨没趣的一天。

他过去从里面掏出个橘子扒开，塞到嘴里，还挺甜的。

朱文静坐到沙发上，不知道在想些什么，两只手无意识的松开又握住。

陈觅撕下一瓣橘子塞到她嘴里，朱文静这才回过神儿，看向他问道，"最近给你李叔叔打过电话吗？"

"打了。"，陈觅选着月考过后的时间给李卫东打过两次电话，明着是报告学习成果，实际是试探李卫东的态度。

但结果显然不太好，李卫东听到他的成绩依然很高兴，但提到陈觅状似无意般提到的夏令营问题，他已经不再像从前一般大包大揽了。

今年暑假班级里有同学要参加新西兰研学营，因为陈觅平时在学校给人一种家境很不错的感觉，同学便问他去不去，可以一起搭个伴。

这不是一笔小钱，靠他家自己肯定是掏不出这笔钱，陈觅模棱两可的说看看有没有时间再说。

同学一时都围到要出去研学那人的周围，羡慕之声不绝于耳。陈觅眼红的很，李卫东怎么偏这个时间出了岔子，要不然他很有把握自己可以去上，到时候大家羡慕的人也有他一个。

朱文静看着儿子落寞的模样，把自己思索了许久的办法跟儿子坦白，破釜沉舟的问道，"你想不想让李叔叔当你爸爸。"

陈觅有些惊讶，他妈竟然这么敢想。他确实是曾经有过'李叔叔要是我爸爸就好了'这种想法，不过也只是想想而已。

不是他看不起他妈，他打量了一下母亲的外貌，精心打扮过后也只能说是个精致的普通人。个人条件方面，农村出身，高中都没念完。现在的工作还是别人看在李叔叔面子，让他妈在公司里做一个内勤做一些跑跑腿，打打文件的活儿，一个月几千块钱饿不死而已。

原来李叔叔的老婆，外貌就不说了，毋庸置疑是漂亮的。学历高，家世也很好，听说父母都在国外有大公司，她自己在国内背靠这座大山，每年光授权费就拿到手软。

陈觅对比了一下，觉得他妈可能是气疯了，他要是李叔叔离婚了也会有更好的选择，而不是选他妈这种带着拖油瓶的普通中年妇女。

朱文静看儿子的表情就知道他在想什么，她也没跟儿子发火，问道，"你就说你想不想就行了。"

那当然想，陈觅迟疑的点了点头。

"那好，你按我说的做。"，朱文静招招手让儿子靠过来。

陈觅听着他妈的计划，眼睛越瞪越大。

"行吗，儿子。"，朱文静问他意见，毕竟要陈觅遭点儿罪，她怕他这边出岔子。

陈觅想了想以后的美好生活，一咬牙，点头道:"行。"
