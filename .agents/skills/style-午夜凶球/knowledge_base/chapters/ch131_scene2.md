# 第131章 - 场景2：李观澜回家

"咋回来了呢？"，林秋敏推开门发现家里空调打着，她往客厅一探就看到了歪在沙发上的李观澜。"也不说一声儿，让你爸接你去多好。"

"这不要出分儿了吗，我合计得回来跟你俩抱头痛哭一下。"，李观澜起来挂到林秋敏的后背上。"妈妈，你有没有想我啊？"

"一边儿去，这么大了还黏呲人。"，林秋敏甩了两下没甩下去，也任由她挂着，她心里美着呢。"小珍没留你啊？"

怎么没有呢！

张自珍昨天试图挽留了一下，但被李观澜无情的拒绝了，查分的高光时刻她要回去跟建国和秋敏共享一下。

张自珍有点醋，但也知道李观澜跟她们感情深厚，帮她准备了点儿特产让她一起带回去。

你回去吧，萱萱在我这儿跟我共享高光时刻，两个我也抱不过来。

此时的江青山两口子也看到了到家的江流，江青山看着江流气鼓鼓呵呵脸问道，"回来咋还不高兴了呢，谁惹你了。"

"李观澜呗！还能有谁！"

江青山来了兴致，"因为啥啊？"，有时候闹脾气也能增进感情。

"她净占我便宜，我说玩儿扇嘴巴子的游戏，她不跟我玩儿。"，江流义愤填膺。

高慧云看着江青山，脸上写着，你看我说啥来着。

江青山看着一脸吃瘪的儿子，心道，算了，没有太高要求了，能找着媳妇儿就行，不挑别的。

"那个，媳妇儿晚上吃啥？"

"家里还有点儿白菜，炖个干豆腐得了。"，高慧云打开冰箱，就剩半颗白菜了，今天没买菜。

哎，不是，李观澜的事儿先不说，我不才回来吗？不给我接风洗尘啥的啊，但凡给我炒个鸡蛋呢？高考完事儿待遇降这么多吗？

李观澜回来的第一天晚上，在建国同志哀怨的眼神中，抢夺走秋敏同志晚上的归属权。

她搂着林秋敏，"妈，明天出分儿了你紧张不？"

林秋敏咋不紧张呢，虽说理智上知道已经考完了，紧张也没用，孩子也差不了。但情绪也不受理智控制啊，她想不紧张就不紧张也不可能，但她怕影响到李观澜的情绪，云淡风轻的说道，"不紧张啊，有啥好紧张的。"

"那你晚上切西瓜时候为啥把瓤扔了，给皮留下了。"，李观澜完全不给她妈面子。

"我爱吃皮！"，林秋敏气哼哼的说道，明知故问。

"嘿嘿！那以后西瓜皮我都给你留着。"

"我看你是皮痒。"，林秋敏哭笑不得，"你想好报哪个学校没？还学那个导演不？"（同学们，完啦，考导演好像要校考，我妹写，就当不用考吧。）

"还没有，报京电的话浪费不少分儿呢。"，到时候说出去没那么好听，她还想让她们嘚瑟一回呢。

"那没事儿，看你喜欢啥呗，兴趣爱好才是最好的老师，你都不爱学的东西咋能学好呢？还得上四年呢，多痛苦，"，林秋敏真不在意这个事儿，她就想让孩子快快乐乐，以后能有个谋生的本事就行，别的她不在乎。

"真的吗？"

"真的，别想那么多，这有啥的呢。那老马家孩子考七百多分最后上的体育大学人家不也没说啥吗。"

"……，妈，你等会儿，这个我确实听着有点耳熟，是真事儿吗？"

"真真儿的，都上电视了。"

"呃……"
