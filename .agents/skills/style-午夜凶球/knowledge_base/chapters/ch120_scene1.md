# 第120章 - 场景1：带李昭去游乐场

送走了一个个吃饱喝足的小屁孩儿，李观澜按住要跑的李昭，"干嘛去？"

"回家。"，李昭心虚的答道，他现在就是不想跟她们两个在一起，那个，也不知道因为什么，就是羞涩。他从来没想过羞涩这个词儿，竟然能跟他联系在一起。

"回什么家。"，李观澜和李萱一左一右的按住他，"走，玩儿去。"

啊？

李昭瞪大了眼睛，我不是做梦吧。打了架，好吃好喝的招待我不说还要带我去玩儿？

"去哪儿？"

"游乐场！"，李观澜秉承着李建国的思想，反正的学不进去，那就玩儿去呗，能咋的。先玩儿好了，别的以后再说。

李昭的脚被黏在原地，游乐场啊，好想去啊，还没去过呢。

他看向李萱，你去过吗？

李萱微笑摇头，没有啊，所以我们一起去吧！

听说他们两个都没去过游乐场，李观澜又出离愤怒了。她们家那个小地方没什么正经游乐场，但是能去的地方她基本都快玩儿吐了，李萱她们竟然没去过游乐场。

她二话不说打电话给李卫东，"歪，李先生，打钱！弥补童年！"

李卫东对着说完诉求就挂断的电话陷入沉思，弥补童年？缺钱了？听说她小时候家里条件是挺不好的。

手机转账已到账，李观澜一手拉一个，站在游乐场门口，表情严肃，"准备好了吗？"

"准备好了。"，李萱举手。

"嗯。"，李昭哼哼。

"啪！"

"呃，准备好了。"

挨了李观澜一个降龙十八掌后，李昭终于学会了正常回话。

先和江流他们汇合，确认江流依然恐高什么都玩不了后，她拉着江流开始漫长的排队之旅。

李萱和李昭一队，她和江流一队。等李萱她们玩完一个项目后，她们这队也排的差不多了。

换上李萱她们两个，她和江流换一个项目继续排队。

就这样，李萱和李昭基本上做到了项目和项目之间的无缝衔接，想玩儿的都玩儿到了。

呃，不想玩儿的也玩儿到了。

李观澜和江流排队已经排疯了，人多的项目她们都要去排一下，然后等李萱她们过来再不由分说的把两人塞进去。

"好好，好好。"，李萱在后面深情呼唤她。

不用谢！李观澜帅气的甩头，这是我应该做的。

江流狐疑的转过头，"我感觉李萱好像快要哭了。"

"没事，感动罢了，我也只能为他们做一些小事了。快走，我们去排大摆锤。"

"啧……我怀疑。"，江流怀疑李观澜在报复她们两个，也不知道她俩怎么惹她了。

"呕～"

"呕～"

从大摆锤下来，李萱和李昭一左一右的扒住路灯杆，低头干呕。

这么一看，游乐场也没那么好玩，呕～

李观澜递给他们矿泉水，嘴上不赞同的批评两姐弟，"你们玩儿的也太疯了，再好玩儿也不能这样啊，下回注意。"

"呕～知道了。"，李萱强忍住恶心答道。

"呕～"，李昭看了眼李萱，没想到你真是好脾气，以前我还以为你装的。
