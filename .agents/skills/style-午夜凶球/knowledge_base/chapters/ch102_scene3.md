# 第102章 - 场景3：间操时间的观察

第二天间操时间，李观澜的每一个动作都做的一丝不苟，就是脖子仿佛锈死了，坚决不转头。

昨天刚送完信的小伙儿心不在焉的跟着做操，眼睛一直盯着李观澜那边儿看她有没有什么回应。

“别看了，人家根本不想理你。”，他的同班同学吐槽道。

“不对，你仔细看她的动作，多么僵硬，没有以前灵动了，我看她八成是昨晚上没睡好落枕了。”，该小伙子坚信自己这封信写的情真意切，一定能打动李观澜。

我的天呀，人家看天看地就是不看你这边还不够明显吗？再说人家的cp是江流，江流又高又帅学习又好，你有点自知之明好不好。

徐楠楠和谢玉露一眼就发现了那个探头探脑的安静男孩子，太明显了，那脸就看着这边儿都不带换个方向的。

长得还行，算是清秀，就是说话恶心点儿。

两人满足了好奇心就不看了，转过身来认真做操。

“诶，那俩女生怎么总看我呢。”感受到两道视线，安静男孩问同学。

同学看了一眼，心道，那还用想，肯定是八卦来了。

“难道她们俩喜欢我？我这该死的魅力，但我已经心有所属了。哎呀，忘了观察李同学了，她刚刚有没有看我？”

同学转头跟后面的人商量，“咱俩换个位置。”

换完位置他吐了口气，这回好多了，要不本来上学就烦，还要应付二逼。
