# 第136章 - 场景3：李观澜来办公室

回到办公室，坐在刘小华旁边小声说话的人突然住了嘴，"那个，小华，我先回去了啊，有时间我再过来找你聊。"

这种背后嘀咕的事儿最恶心，你还不能上去问她'你是不是讲究我呢'。

林秋敏不管，有流言一律按刘小华传出来的处理，她看着刘小华似笑非笑，"刘小华，说啥呢，让我也听听呗。"

"哦，也没说啥，不知道从哪儿听来的消息，不知真假，听着真气人。林姐，正好我问问你……"，刘小华这时看着与平日的刁钻判若两人，好脾气的说道。

'乌蒙山连着山外山……'

林秋敏看了一眼，是李观澜，她没搭理刘小华接起了电话，"喂，闺女，咋的了？"

"妈，我和萱萱在这附近溜达，中午你出来跟我俩一起吃呗，我请你们俩搓顿好的。"

林秋敏的手机音量不小，李观澜的声音整个办公室都听的清清楚楚。

办公室里的人都竖起耳朵，孩子声音听着没啥异常，还是跟林姐很亲热，那消息到底真的假的。

"那你俩等我一会儿，我手里有点活儿还有二十分钟差不多完事儿。"，闺女来了，林秋敏懒得搭理刘小华，干脆把手里活儿干完，下午就请假不来了。

"那我俩去你办公室待会儿呗。"，李观澜的声音带着点儿跃跃欲试。

她要干啥？林秋敏感觉她要搞幺蛾子。

"那你俩上来呗。"，林秋敏无所谓，反正单位已经传的很乱套了，大大方方的让他们看呗。

林秋敏放下电话也不理刘小华，闷头干起活儿来，早完事儿早走。

李观澜和李萱没两分钟就上来了，"妈。"

办公室众人只觉眼睛一亮，两个青春靓丽的女孩子出现在办公室，跟大家打过招呼后一左一右围在林秋敏身边。

单看李观澜不觉得，两个孩子都在家一眼就能发现谁跟林秋敏长得像。

"林姐，哪个孩子是你家好好来着。"，刘小华故作不知的问道。

"都是我闺女。"，林秋敏也没瞒着。

"啊？"，刘小华的惊讶的很浮夸，"我记得你家就一个孩子来的。"

"嗯，分裂了，变俩了，等再有分裂的就变成仨。"，明天就给李昭带来，吓死你。

"呵呵，林姐你可真会开玩笑。"，李小华毫不气馁，指着继续问道，"是这个吧，跟你一个模子扒下来的似的。"
