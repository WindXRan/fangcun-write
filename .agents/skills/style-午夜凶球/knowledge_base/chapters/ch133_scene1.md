# 第133章 - 场景1：群里互怼

李观澜也同时接到消息，江流在群里猖狂的很。

'仰天大笑出门去，我辈岂是蓬蒿人！'

'海到无边天作岸，山登绝顶我为峰！'

'待到来年九月八，我花开后百花杀！'

林霆:才发现你挺能装的。

谢玉露:叽里咕噜说啥呢，高考都完事了别整这个，看着害怕。

李观澜:天不生你江流，二逼届万古如长夜。

江流:嫉妒，赤裸裸的嫉妒。李观澜你照镜子看看自己的嘴脸，多么丑陋！

李观澜:微笑，别逼我在这么开心的日子里抽你。

江流:切～哥们，我考了712@林霆

林霆:你好棒哦，我也想把喜悦分享给你，我爸今天给我提了台兰博基尼。

江流:……不得意，那玩意卡底盘。

林霆:Urus。

江流:……李观澜有人炫富你管不管？！

李观澜:林霆同学请注意，你金钱的利刃刺伤到穷人那根敏感的神经了。

林霆:他也没管我的死活啊。

徐楠楠:啊啊啊啊啊啊啊——

徐楠楠:我524，524！我过一本线啦！

江流，林霆:恭喜！

李观澜，谢玉露:撒花！楠楠你太棒了！！

徐楠楠:感谢天，感谢地，感谢我的老师我的朋友我的父母，感谢cctv,mtv……

李观澜看到徐楠楠的好消息开心的在床上滚来滚去，在高考这场人生的小试炼中，她的好友无人掉队。

京市明天出分，林霆已经觉醒不用担心，谢玉露自我感觉也很良好，反正卷子上面能写的都写上了。
