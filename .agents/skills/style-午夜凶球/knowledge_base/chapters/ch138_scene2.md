# 第138章 - 场景2：送丝巾

"这条丝巾送你吧，谢谢你陪我们半天了。"，张自珍递过丝巾给刘小华，"你们都是一个单位的，以后麻烦在单位多照顾林……小敏了。"

林秋敏马上清醒了，好几千的东西你送她干嘛，她伸出手就要拿回丝巾。

张自珍眼疾手快的塞到刘小华的手里，拉着林秋敏走了。

林秋敏的看到刘小华已经看不到了，赶紧要把袋子塞回到张自珍手里，"我不要啊，你全拿走。"

张自珍斜睨了她一眼，"不值什么，拿着吧，你背着好看。"

哦，这视金钱为粪土的语气。

林秋敏看着身上挂满的袋子，叹气，李建国，我后悔了，下辈子首富跟你我想选首富了，太他妈过瘾了。

"那你也不能给刘小华啊，我可膈应她了，给她花一分钱我都难受。"

"放心吧，这礼物她收了比捅她一刀都难受，今晚上能不能睡着觉都两说。"

"能吗？哎呀，孩子行李箱还没买呢。"

"行李箱我觉得那个白的好看。"

"那个不禁脏。"

"咳。"

"行行行，你说是啥就是啥。"

林秋敏收回自己的意见，真是吃人的嘴短，拿人的手软。

就像张自珍说的一样，看着两人远去的背影，想起刚才导购阿谀奉承的嘴脸，刘小华气的攥紧了手里的袋子，这好日子怎么就让她林秋敏过上了呢！

装丝巾的袋子发出'哗哗'的响声，刘小华低头看了一眼这条昂贵的丝巾，想扔又不舍得，有一种吃到包了屎的蛋糕的感觉。

踏马的！
