# 第144章 - 场景1：李卫东感慨被张自珍怼

李卫东的位置挨着张自珍，看到这一幕他不禁感叹道，"好的家庭教育让人受益一生，他们真的很棒！"

"你能不能不说话。"，刚刚的气氛一下被破坏了，张自珍一秒钟切换到不耐烦的态度。对比完知道自己什么样儿就得了，非得说话找存在感。

张自珍可以说是一点面子也没给他留，今天能坐在一起都是看在不明就里的高慧云份上，两个账房先生因为最后入座，都是安排到亲属这一桌的，高慧云直接带着李卫东坐到了这里，张自珍也不好意思说什么。

李卫东被怼的面色发红，李昭和李萱默契的低下头研究起桌子，诶？你说这桌子腿怎么就不多不少刚好四条呢。

江青山不知道两个人什么关系，刚想张嘴劝一劝，被高慧云一杵子打后腰上，马上闭嘴了。

啥也不知道，别瞎叭叭。他自己都没说话呢，显着你了，这么爱说明天江流升学宴你发言。

这桌的气氛陷入尴尬的沉寂，另一桌孙老太觉得李观澜今天礼貌的让她陌生。
