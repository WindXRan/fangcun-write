# 第114章 - 场景4：李观澜被记者采访

没等李建国找到孩子呢，记者先找到了。

李观澜长得多扎眼啊，记者一下就相中她了，这孩子肯定上镜。他拦住李观澜问道，"同学，考完这一科感觉怎么样？题目难度和平时模拟比如何？

"还好，照比平时稍微简单一点儿。"，李观澜礼貌微笑，整个人又漂亮又自信，往那儿一站就是对青春洋溢，亭亭玉立这些词语最好的诠释。

"看来这位同学考的不错啊，有什么话想对别的同学说的吗？"，记者很高兴，又漂亮又不怯场，真是个好素材。

"那就祝大家都考到一个心仪的好成绩吧！"

"谢谢这位同学，也祝你考到理想的成绩！再见！"

李建国和林秋敏骄傲的看着落落大方的李观澜，这是我闺女。

李观澜按约定往大树这边走来，李建国在人群中跳脚，"大闺女，我们在这儿。"

林秋敏捂脸，这把你嘚瑟的，生怕别人不知道刚才采访的是你家孩子。

三天高考下来，李观澜感觉整个人都被吸干了精气神儿一样。

李建国夫妻又何尝不是，考完试那天晚上，精神松懈下来三口人对付吃了口中午的剩菜，齐齐躺下睡觉。

补足了觉，第二天李观澜还得去学校估分儿。

她一下楼碰上也要去学校的江流，直接一屁股坐他自行车上了。李观澜对她爸挥挥手，"回家吧，碰见专车了。"

李建国丝滑的转身上楼，太好了，回来还挺不好找停车位的。

江流难得的没和李观澜拌嘴，等她上了车，长腿一蹬往学校方向飞驰而去。

"诶呀，江屎蛋，今天咋这么消停。"，李观澜也是惊讶。

"以后应该也没啥机会了，也不差这一回半回的了。"，江流会怀疑自己是贱人，咋还有点儿依依不舍了呢。

李观澜沉默了半晌，突然开口问道，"你是不是得绝症了。"

人言否？

果然刚才是错觉。

"你给我滚下去！"，江流一个急刹车把李观澜从车上甩下去。

"别激动。"，李观澜又跳上来，"我就问问，没得就没得呗，咋还不高兴了呢！"

"这话我听着怎么更来气呢！"

"那谁知道了，可能是你小心眼儿吧！"

"诶，李观澜，你大学想去哪儿上啊，国外你看怎么样？尽量别回来了……"

"那你还不想死我。"

"我想你死！"

"反弹！"

"反弹你妹啊！"

"什么虎狼之词，我给萱萱打电话。"

"……你牛逼，我错了。"
