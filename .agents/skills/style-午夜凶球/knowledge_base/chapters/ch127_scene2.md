# 第127章 - 场景2：孙老太去张自珍家被拦

这边她也来过两回，别墅区附近没有公交车站，孙老太斥巨资打车到了，结果被拦在大门之外。

原因:保安不认识她。

孙老太急的直跳脚，跟保安怎么说都说不通，打张自珍的电话又打不进去，大夏天的她热的浑身是汗。

能打进去就有鬼了，张自珍早就把她拉黑了。

齐奶奶遛弯儿刚回来就看到有人在小区门外跳脚，她看了一眼也没当回事儿，以前也见过，不知道谁家的穷亲戚没打招呼又过来了。

"我记不得是几号了，房主叫张自珍，四十多岁，长得挺漂亮的，我是她婆婆，你给她打个电话问问。"

保安不为所动，不可能，我干了好几年了，就没见过他们家有男主人，别说婆婆了。

齐奶奶定住，回头看了她一眼，是你啊！

她露出惊讶的笑容凑过去，"你是小张的婆婆啊？"

孙老太很惊喜，"你认识张自珍？"

"我是她邻居，她今天不在家啊。"，齐奶奶早上看到她们上班的上班，上学的上学，出去玩的出去玩，没一个在家。

孙老太这才想起来，不是人人都跟她一样，每天都在家里，她傻眼了。出租车已经走了，离车站那么远的路程，她可怎么回去啊？

齐奶奶恰到好处的出声，"要不您到我家坐会儿，落落汗再走？"

孙老太赶紧同意了，这么大太阳，等凉快点儿再走也好。
