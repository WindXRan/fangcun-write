# 第111章 - 场景3：拍毕业照的离别泪水

"右边的女生往中间靠一靠，对。"

"咔嚓！"

谢玉露觉得时间过得飞快，看电影的日子好像就在昨天，但实际已经过了大半个月了。

今天学校组织她们照毕业照，她原来还只顾着臭美，务必要求自己艳压群芳以最美的形象出镜。可照着照着，她突然意识到，分别就在眼前，以后如现在一般，每天在一起的时间比家人还要长，说的话比家人还要多的亲密时光将要越来越少了。

她站在林霆和李观澜中间，两个人每人在她的脑后比了个兔耳朵，她笑着笑着突然掉下眼泪。

几个人都慌了手脚，七手八脚的给她擦眼泪。谢玉露刚开始还有些不好意思，但眼泪就是控制不住的越擦越多。

"哇——我舍不得你们！"，她干脆不忍了，放声大哭。

"你干嘛啊！分开就不是朋友啦！"，徐楠楠大声说道，好像这样就能控制住她马上就要流出的眼泪，但这果然是徒劳，谢玉露一人独嚎的情况马上变成了二人抱头痛哭。

"真是受不了你们两个。"，李观澜露出无奈的微笑，我们江湖中人流血不流泪，你们两个搞煽情这套。怎么回事，口水从眼睛里溜出来了。

江流和林霆站在一边束手无策。

江流挠头，要不我上去给李观澜一脚？那她肯定马上原地复活，跳起来打我。

气氛这不一下就活跃了，就是我要遭点罪。
