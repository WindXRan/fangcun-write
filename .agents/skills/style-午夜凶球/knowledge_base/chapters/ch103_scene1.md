# 第103章 - 场景1：安静男孩挑战江流

安静男孩没听出来敬佩，他觉得江流阴阳怪气的厉害。

粉丝多了不起啊，你俩又没官宣。

“你占有欲是不是太强了？你俩也没在一起凭啥不让别人跟李同学表白。”

江流词穷，你从哪儿听出来我占有欲强的，“我没不让你表白啊，我还支持你呢，快去快去，用我给你带路不。”

完事儿让李观澜暴露她的真实面目，一拳给你从三楼楼梯口打到一楼器材室。

“你以为你说反话我听不出来是吧，大男人说话阴阳怪气的，是男人就凭实力说话，敢不敢和我公平竞争。”，安静男孩对自己还是有这个自信的，他也处过几个女朋友，都是她们先追的他。

我疯啦，我跟你竞争李观澜，我活够了是咋的！我要是追她，她敢天天骑我脖颈上来学校。

“同学，现在你说的这些并不重要。马上要高考了，在这个重要的时刻你主要的任务应该是去看看脑子。”，江流一脸真诚，看好了你以后会来感谢我的。

而且马上就高考了，你挑这个时间点儿也够烦人的，让老师知道你影响李观澜学习，你看他看干不干你就完了。
