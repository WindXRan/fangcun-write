# 第139章 - 场景1：生日惊喜

逛街逛到傍晚才回家，张自珍觉得这比上一天班还要辛苦，她疲惫的推开家门，瘫在沙发上。

李观澜和李萱不在，杨秘书带着李昭去赶海了还没回来，家里静悄悄的。

她伸出手腕看着上面的钻石手链心里不忿，切～看着比林秋敏那个大金镯子好看多了，这俩孩子什么审美。

灯光突然熄灭，零星的灯光从卧室里泄出。

"祝你生日快乐，祝你生日快乐……"，女孩子清脆的声音在房间里响起。

李萱捧着蛋糕，后面跟着一串人走了出来。

张自珍愣了，这才想起今天是她的生日。

"妈妈，生日快乐！"李萱笑着把蛋糕放在桌上，烛光映着她甜甜的笑脸。

张自珍看着这几个人一张张期待的小脸儿，眼底有些酸涩，跟我玩儿这套是吧。

"快点，小珍，许愿。"，李观澜把生日帽戴在她头上。

"没大没小。"，话虽然这么说，张自珍还是闭上双眼认真许愿。

希望我的孩子们健康。

希望我的孩子们快乐。

希望……好像没什么想许愿的了。

她的眼睛欠了一条缝看了几个孩子一眼，嘶，算了还是闭上吧。

第三个愿望，希望李昭换个发型。

"看看我们的礼物！"，李观澜和李萱急不可耐的拿出礼品盒。

是全套的珍珠饰品，成色一般，设计倒是挺好看。

张自珍在两人期待的眼神中戴在了手上，脸上忍不住流露出笑意，"嗯，好看。"
