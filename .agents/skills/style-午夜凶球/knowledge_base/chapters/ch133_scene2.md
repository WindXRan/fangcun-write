# 第133章 - 场景2：筹备升学宴

李建国和林秋敏还要再经历一次这种体验，第二天掐着时间给李萱打电话，李萱考了历史成绩最高点，627。这下李建国和林秋敏是彻底放心了，两人开始研究起办升学宴的问题。

"好点儿的酒店现在订不着了吧。"，林秋敏后悔自己怎么没早点儿订酒店。

"订完了，高考头两天我就订完了。"，李建国早就订完了酒店，后来忙忙叨叨的忘记跟媳妇儿说了，林秋敏一提这个事儿他才想起来。

"行啊，老李，靠谱。"，林秋敏又想起李建国一贯的尿性，提醒道，"你别整太张扬了啊。"

李建国嗤之以鼻，我闺女考700我还不张扬我啥时候张扬，但凡我认识的人都得通知到。别说人了，路边儿狗李建国看见了都想给甩张请帖。

再说了，之前随那么多礼，现在不通知啥时候通知，别说考700了，李观澜就是考70我也得都通知到了。

"对了，你问问张自珍，她要是有时间就来，也见证一下孩子辉煌时刻。"

他们和张自珍商量了，还按原来的轨迹，李观澜在这边办，李萱张自珍会在京市替她办升学宴，没有必要这个时候大张旗鼓的对全世界宣告孩子抱错了这件事儿。

有人问的话，他们的态度是，不回避也不刻意，一切自然而然，水到渠成就好。重磅炸弹似的宣布这个消息除了成为他人的谈资外毫无意义，润物细无声就好了。

"嗯，行，我现在跟小珍关系还行，我问问。"，打从上回演唱会结束，林秋敏对自己和张自珍的关系有了重新的评估。

李观澜挂了电话，林霆671，她接受良好。让她惊讶的是谢玉露，417分，谢玉露从最开始数学符号都认不全到今天这个分数，真是很艰难了。

谢父谢母对这个结果都已经很满意了，表示要大摆筵席招待亲朋好友，广而告之。

我们家谢玉露出息了。

谢飞星一言难尽，我考706你们也没这么高兴啊，是开心终于能摆脱小女儿不学无术的传言了吗？

谢玉露还在水群，李观澜在邀请他们去参加她的升学宴。

谢玉露:好呀好呀，然后你们跟我回来参加我的升学宴。

江流:我也办，既然要来都来啊。

徐楠楠:我也是。

李观澜掐指一算，哦，7月，吃席的一个月。

还有别的同学邀请她呢，她要是都去，有时候一天甚至能赶两场。
