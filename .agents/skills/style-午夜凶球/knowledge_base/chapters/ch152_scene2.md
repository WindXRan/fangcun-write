# 第152章 - 场景2：李观澜大学报到

李观澜压着学校开学的最后一天到校此时到学校报到的人已经寥寥无几了。

入学手续什么的她们已经很熟悉了，昨天送李萱去学校的时候已经办过一次了，今天给李观澜处理起来游刃有余。

寝室的人已经都到了，她是最后一个。

林秋敏爬上爬下的给她收拾床铺，李建国一趟一趟的打水给她擦桌子擦柜子，张自珍不会干活儿，但挑毛病很在行，半天下来李观澜的铺位焕然一新。

吃完早饭回来的几个同学震惊的看着这一幕，心里不约而同的想法是，这姑娘得多娇气啊，会不会很难相处。寝室里已经有一个难相处的了，再有一个简直是地狱模式。

李观澜见有人回来了，赶紧拿出刚才在外面买的水果分给她们，"不好意思啊，我来晚了，没影响你们吧？"

"不用不用，没事的。"，两人赶紧说道，还好，现在看起来还算好相处。

"拿着吧，买了好多呢，我自己也吃不完。"，李观澜笑盈盈的把水果塞到她们手上。

"谢谢。"，两人有些不好意思的收下。

三人很快熟悉了起来，聊上天，她们发现新同学既活泼又健谈，两人很快放下了戒心，竟然分享起她们听到的八卦来。

"你听说了吗？"其中圆脸戴眼镜儿的女孩儿神神秘秘道。

"什么？"，李观澜十分配合。

"听说有个考了700分的神人报了我们学校导演系。"，她颇为自豪自己的消息灵通，这个消息目前学校知道的人还不多。

"真的假的？我不信。"，另外一个女生说道。

"真的。"，李观澜点头。

"你怎么知道？"，圆脸女生疑惑问道。

"因为我就是那个神人。"，李观澜用大拇指比划自己。

"是吗？哈哈，哈……"，圆脸女生闹个大红脸，"那个，我还有点事儿，出去一趟啊。"

"这么巧，哈哈，哈……"，另一个女生跟着尴尬。

两个人随便找个了理由走出去了。

李观澜还能听到她们强忍到门口才奔跑起来的脚步，她忍不住笑出声，看来她这个事儿传播范围还挺广的。
