# 第146章 - 场景2：回京路上的车位之争与车上斗嘴

临行前，随车人员配置问题出现一些小问题。

林霆因为家里派的是一辆商务车，除了司机还剩六个位置。谢玉露家是轿车，只有四个位置。

林霆数了数，一二三四五，上山，不是，这车里正好够坐。你，你们，都去坐谢玉露家的车。

此次回去除了林霆秘书以外，车里还有两位财务人员做完子公司的财务审查后一同回京。两人听了大公子的命令，从善如流的换了车。

沈景明的秘书不想去，这不正好六个座位吗？我可以挤……

谁跟你挤一挤，林霆把他推下去了，走你吧！

他郁闷的打开谢家的车门，就剩一个一个座位了，他前面坐着的就是谢震元的秘书。

他坐到座位上，没一会儿车子平稳的行驶起来。

他心中一直有个疑问，忍了一会儿，最终没忍住，凑近谢家秘书试探道，"前几天随份子的时候……"

谢震元秘书表情波澜不惊，"嗯，怎么了？"

"没什么。"，沈景明秘书依旧怀疑他，换了个方法试探他，"这边的交管还挺严格的啊？"

谢震元秘书缓慢的把头转向窗户，状似不在意的答道，"是吗？没注意诶。"

沈景明秘书不语，火热的视线如有实质一般紧紧盯着他。

谢震元秘书抱住公文包的手缓缓收紧，车窗里反射出他无辜的眼神。

妈的，放屁瞅别人，其实就是你放的，肯定是你！

沈景明秘书秘书冷笑一声，膝盖并拢，死死的抵在座椅靠背上。

谢震元秘书把座椅往前调。

沈景明秘书往前挪，仅剩半个屁股在座椅上。

好气，有钱人的车为什么空间这么大，从来没觉得自己腿这么短过。

谢震元秘书再次调整座椅。

沈景明秘书:！！！

随同的财务人员瞪大了眼睛，大秘，你怎么在车上扎起马步了？！！

那边气氛剑拔弩张，林霆这边舌剑唇枪。

"我靠，江流你专业报的临床医学？"，林霆十分惊恐，华清临床医学3+2+3，毕业就二十七了，我敬你是条汉子。

"小场面，不要惊慌。"，江流把头发背到脑后，表情十分自得，"以后我就是胃不好的总裁的医生朋友。"

"我胃不可能不好，每顿我都按时吃饭，偶尔还加宵夜。"，林霆鄙视他。

"我知道，但是总吃撑了胃也可能不好。"，江流觉得自己想的非常周全，"再说我就不能认识别的总裁？"

林霆思索良久，答道，"不可能有第二个脾气这么好的总裁跟你当朋友了，严格来说咱俩也不算朋友，我有时候挺烦你。"

江流翻白眼，"你有时候也很装，我也是强忍着的。"

李观澜无语插话，"你们两个小学鸡无不无聊，林霆你快别气江流了，赶紧给他照相留念吧，他头发浓密的日子不多了。"

"你才秃呢，我天赋异禀，头发茂密到永远!"

"你不用嘴硬，那多少医生上学和上班判若两人的……"

"我走管理岗。"

"管理保安和保洁。"

"我！你！"

"诶，吴明远学的什么专业？"

"计算机，怎么啦。"

"哦，那你们俩可以组个组合，组合名叫跨界双秃。"

"……"

"我照完了，林霆我给你发过去哦～怎么样，很有眼力吧，毕业我准备给你当秘书。"，徐楠楠学的汉语言文学，还没上学已经给自己看好工作岗位了。

"我不招女秘书。"，哼，徐楠楠你也没少落井下石，现在想起我了。

"那好叭。"，徐楠楠也不生气，回头就跟谢玉露嘀咕，"露露，那我给你当助理好了。"

"行，我一个月给你开五万。"，谢玉露爽快的答应。

"还是你好，小气的男人真的很减分，我跟你说……叽叽咕咕稀里哗啦……"，徐楠楠这一路嘴就没停下来过。

林霆想抢夺方向盘的心思到达了顶点，同归于尽吧！

司机:这是我职业生涯中最闹腾的一次出差。

到了京市，沈景明秘书是被人扶下来的，谢震元秘书这一路也憋屈够呛。

两人分别之前对视一眼，皆是面带微笑，眼含杀气。

"有机会再见！"

两人异口同声。

林霆诧异，"看不出你们俩关系处的怪好，要不你俩再聊一会儿？你腿咋的啦？"

秘书:……大公子此话威力不亚于'难道要我也喜欢你'……
