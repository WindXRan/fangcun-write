# 第104章 - 场景1：谢飞星与父亲通话

“谢飞星，明天去看看你妹妹。”，电话那头传来老父亲的命令。

“我很忙的好吧。”，谢飞星叹口气，“你想看你去呗。”

“忙什么？我听听。”，论文写完了，工作也不用他找，他有什么可忙的。

“不是给我安排的活儿吗？搞了一个效益不好的破公司给我美其名曰让我锻炼，我天天忙的焦头烂额，都快要到网上去出卖美色谈合作了。”

是的，他真的有这个想法。接手这个破公司已经一个月了，每天几乎连轴转。为了做出成绩这个月他几乎事无巨细都亲力亲为，但依然收效见微。

公司的运营多次明里暗里的提示他，某某家的公子在稳音上发视频，现在公司效益好的嘞，他已经快要向现实低头了。

“是哦，我给忘了。”，谢父想起来确实有这么回事儿，这事儿确实是他理亏。“那正好我给你放两天假，你去看看你妹妹回来再去出卖美色。”

……

谢飞星怀疑自己不是亲生的。

“老头子，你快要失去我这个儿子了。”

“真的吗？”，谢父在电话里一本正经，“我这么有钱，外面想当我儿子的人已经排队到巴黎了，你不再考虑考虑了吗？”

“我错了，爸。儿子还是亲生的好，听话。我马上出发去看你亲爱的女儿，我亲爱的妹妹。”识时务者为俊杰，谢飞星反口也很快。
