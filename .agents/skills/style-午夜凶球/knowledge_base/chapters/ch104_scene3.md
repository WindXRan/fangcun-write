# 第104章 - 场景3：送信封给李观澜

谢玉露带着大包小包的到达投喂点的时间赶的正好，正是李建国来给他们送饭的时间。

她把李建国挤到一边儿，“义父，让一下。观澜，我给你们带好吃的啦。”

李建国被她撞的的一个趔趄，抬头就和谢玉露一起来的小伙子对视上了。

哎呀，这不是谢玉露她哥吗，死孩崽子净瞎叫，李建国尴尬的对他笑笑，“玉露她哥来啦。”

“叔叔，您好，又见面了。露露回家都说了，谢谢您平时对她的照顾。”，谢飞星笑笑，当做没听见。

谢飞星没提这茬，李建国感觉自在了不少，这小伙子看着可比他妹妹精明稳健多了。

“说那个干啥，那都顺手的事儿，也不费啥功夫。”，李建国跟他不熟，也没什么可说的了就抬手看了看手表，“哎呀，我得走了，回头有时间到家里坐坐啊。”

李建国走了，谢飞星来到还在叽叽喳喳的谢玉露几人面前，李观澜的笑容马上变得虚伪了起来，“玉露她哥来啦。”

谢飞星一顿，这谁能想到你不是亲生的，“叫不出来哥叫我谢飞星也可以。”

“那多不好啊。”李观澜不好意思道，“那显得我多没有礼貌啊，是吧，谢飞星？”

谢飞星失笑，看着跟他隔了一个学校栅栏的李观澜，有种探监的感觉，“这么叫着吧，舒服多了。给你拿着。”

谢飞星拿出个信封递给李观澜。

李观澜苍蝇搓手，脸上的快乐都挡不住了还要装作为难的样子，“现在就给啊，这还没高考完事儿呢，我受之有愧。”

话虽这么说，但她还是利落的接过了信封喜滋滋的揣在兜里。

“……”，谢飞星知道她误会了，也没解释，“拿着吧，你应得的，没有你就没有它。”

“那也是。”李观澜点头道，“谢飞星你做事这么爽快以后肯定大有一番作为。”

“客气了客气了，一定拿到没人的地方再看啊。”，不然容易吓到别人。

“那当然了。”李观澜深以为然，要不被人惦记上的了怎么办。她看了眼谢飞星，怎么总感觉他脸上的笑容不怀好意呢。

她心情忐忑的送走了谢家兄妹，在洗手间打开了谢飞星给的信封，她和谢玉露的纸扎人照片从里面滑了出来，李观澜翻白了个眼，奶奶个腿，白高兴了，就知道谢飞星没那么好心！
