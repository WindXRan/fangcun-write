# 第115章 - 场景4：火车上分享食物

就这样，李建国夫妻带着三个孩子进京去看演唱会了。

车次是下午三点半到京市的，她们中午没来得及吃饭就上了车。

放好行李，李建国开始从背包里往外掏东西，一兜兜一瓶瓶的摆的小桌板满满登登。

"谁吃茶叶蛋？"，这一堆东西能掏出来，但塞回去费劲。这玩意占地方，赶紧吃了（liao）了好扔喽。

果然这几个胃口堪比成年非洲象的孩子都齐刷刷的举起了手。

李建国给她们一人拿纸巾包了两个茶叶蛋，"不够我这儿还有啊。有没有吃干豆腐的，我给你们卷一份儿。"

说话功夫江流都吃完一个茶叶蛋了，"李叔，我要，给我卷个大的，不要葱，多放黄瓜。"

"要鸡蛋酱还是肉酱。"，李建国准备的挺全乎。

"肉酱！"

"行，你排一号啊。"

她们五人当时买票买的是一排的，一排六个座，三个孩子坐一边，他和林秋敏坐另外一边。

林秋敏边儿上挨得是个二十多岁的小姑娘，一身职业装，小桌板上摆着笔记本电脑，一直在上面敲敲打打。

从他们分茶叶蛋开始她就不着痕迹的皱了皱眉，她戴上了耳机，继续工作。

他们虽然不怎么说话，但因为动作比较多，她还是不由自主的分神。

她合上笔记本，决定先吃饭。她掏出盒方便面，还没等站起来手里一沉被塞了什么进来。

"吃吧，没给你放葱。"，坐她旁边的中年女人说道。

"呃……"，她看着手里多出来的干豆腐卷。

"你叔戴一次性手套了，干净的，放心吃。"

"谢谢！"，其实我不是这个意思。她有些受不住对方热切的目光，把干豆腐卷放进嘴里。

还挺好吃。

林秋敏得意的看着李建国，就咱这敏锐性，她看我茶叶蛋第二眼我就发现了。
