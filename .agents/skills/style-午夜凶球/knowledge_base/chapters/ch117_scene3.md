# 第117章 - 场景3：演唱会结束后讨论住处

林秋敏不枉在家练了好几天，跟唱整场演唱会，几乎就没有她不会的。但后遗症就是一场演唱会下来，现在说话的嗓音像唐老鸭。

"李建国，你慢点儿，等会儿我。"，林秋敏随着人流往外走，她上前一把拉住李建国的手。人太多了，她们好几次差点儿被冲散。

李建国也很无奈，不是他想走这么快的。他感觉双脚已经离地，去哪儿完全不受控制了。刚才有一瞬间差点儿想破罐子破摔，他就看看这群人到底能给他带哪儿去。

张自珍被林秋敏的另一只手拉住，三个人就像吊桥上的锁链似的，一会儿被放下来，一会儿又被拉直。

她这辈子有没有过这种体验，不对，还有上次去超市那次。

在肺部空气被挤压殆尽之前他们终于成功出了会场，找到一块相对清净的地方。

"喂，好好啊，我们在出口东南角第二棵大树底下等你啊。"，林秋敏给孩子打电话报位置。

"啊？你谁啊？"，李观澜又看了眼手机来电才确定是她妈，这会儿功夫说话就自带低音混响了。

成功汇合后，大家又因为晚上住哪儿发生了分歧。

"在跟前儿找个酒店住得了。"，住酒店最省事儿，谁也不麻烦，她们心里也没负担，林秋敏觉得住酒店挺好。

"别介啊，都来了让我尽一下地主之谊啊。"，林霆不同意，吃了人家那么多顿饭，来京市还不让他报答一下，酒店都给找好了。

谢玉露紧紧搂住李观澜，别人她不管，晚上她们仨肯定要住一起。

李萱拉住李观澜，都来京市了不回家住不行，晚上她要和好好一被窝儿，好几个月都没见着了。

江流倒是挺好说话，他心说我上哪儿住都行，看这胶着的气氛一会儿别给我落下就行，李观澜这人缘还挺好，跟她走没错儿，我也得拉住她。

但一打量，李观澜左右手都被占住了，他灵机一动拉住了她的马尾辫。

哦，我可真是个天才。

李观澜回头怒视他，"你有病啊。"

江流悻悻的松开手，退而求其次的拉住她的包带，她这人真是的，玩儿玩儿就急眼。

李建国看着一群人，你拉我我拉你的，干啥呢这是，准备玩儿老鹰抓小鸡儿啊？

他拍板订了，"林霆给咱订完了，咱就去他订那个，不去也花完钱了。"，完事儿明天在这儿溜达溜达给孩子买点儿啥呗，都一回事儿。

既然李建国定好了林秋敏就也不再说了，她看着拉的紧紧的几个孩子，"你们四个上哪儿住自己定，跟我们住酒店还是怎么的赶紧商量一下。"

"要不……"，一直充当背景板的张自珍说话了，她要是再不出声儿这几个孩子都研究要一起到酒店去打地铺了。好几个月没见了，她也挺想孩子的，"孩子们都上我那儿去住吧。"

李萱眼睛一亮，张自珍不出声儿她都没敢往那儿想，她妈最不喜欢人多了。
