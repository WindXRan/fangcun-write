# 第137章 - 场景3：张自珍买黄金

"使个大劲就买这么个玩意？"，三人此时正在黄金柜台前面，张自珍在柜台前面巡视了一番，"这个，这个，这个还有那些都拿出来我看看。"

她气势逼人，一时间柜台里的几个柜姐过来了好几个，一起忙乎了起来。

林秋敏还愣着，一个巨沉的手镯'嗖'就套上了她的手腕，脖子一沉，一个吊坠又挂在了上面。

张自珍给她换了好几套，才满意的点点头，递出一张黑卡，"刚才试过的，都要了，刷卡。"

先不说林秋敏两人的反应，柜姐颤颤巍巍的接过了银行卡，小心确认道，"女士，货物离柜是不退不换的。"

"知道。"，张自珍还是那么淡然，语气里充满了有钱人的自信。

"好的，您请坐。"，柜姐心花怒放，手脚麻利的开始打包起来。

"那个，小珍呐，你这是干啥呢？"，林秋敏都没敢往是给自己买的那个方向合计，有些诧异的问。

"你喜欢这玩意不早说，多买点儿你换着戴。"，张自珍答道，俩孩子没给她买东西其实她是有点儿醋，不过她反应过来了刚才接近她这女的不是啥好东西。对内，先不说她们关系有多瓷实。对外，谁也不行欺负我的人。

这是李观澜最近给她上的霸总小课堂，用这儿没错吧。

"啊？不要不要，你们别结账啊，我们不要。"，林秋敏慌张的拦下了要刷卡的柜姐，这么多得多少钱啊。这回装把大的挺爽，以后可咋还。
