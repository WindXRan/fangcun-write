# 创意孵化

## 流程

用户说"想写小说"时，不要急着出大纲。先问：

1. 什么类型？（古言/现言/玄幻/都市/悬疑）
2. 男主什么人设？（用一句话描述）
3. 女主什么人设？（用一句话描述）
4. 核心冲突是什么？（用一句话描述）
5. 你想让读者感受到什么？（爽/虐/甜/燃）

**至少问3个问题再动手。** 不要假设用户想要什么。

## 输出格式

用户回答后，整理成设定卡：

```
【类型】：XXX
【男主】：XXX（一句话人设）
【女主】：XXX（一句话人设）
【核心冲突】：XXX（一句话）
【情感基调】：XXX（爽/虐/甜/燃）
```

用户确认后，进入 outline 阶段。

## 注意

- 不要一次问5个问题，分2-3轮问
- 用户说"随便"时，给2-3个选项让他选
- 用户说"不知道"时，从热门类型里推荐
