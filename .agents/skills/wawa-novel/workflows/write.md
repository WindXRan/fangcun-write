# 正文生成

## 流程

用户确认章纲后，写正文。

## 写作规则

- 字数：±10%误差
- 章末必须有钩子
- 禁止AI味（心中涌起、不禁、仿佛）
- 对话占比≥40%
- 内心戏≤30%

## 自查清单

写完后自查：
- [ ] 字数达标吗？
- [ ] 有钩子吗？
- [ ] 有AI味词吗？
- [ ] 对话占比够吗？
- [ ] 内心戏太多吗？

自查不通过→修正后输出。

## 加载知识库

根据场景加载：
- 写打脸场景 → knowledge/beats/爽点/打脸.md
- 写日常过渡 → knowledge/beats/日常/感情升温.md
- 写章末钩子 → knowledge/beats/钩子/信息反转.md
- 用户选了风格 → knowledge/style/对应.json
