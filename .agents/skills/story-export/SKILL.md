# story-export · 标准化导出

## 功能

合并章节为完整小说，格式与导入格式一致，支持互转。

## 使用

```
/story-export <项目目录>
```

## 执行步骤

1. 读取 chapters/ 目录下的所有章节
2. 读取 fanwai/ 目录下的所有番外
3. 读取 _header.txt 获取书名、作者等信息
4. 合并为完整小说
5. 输出到 export/ 目录

## 输出格式

```
书名：xxx
作者：xxx
状态：完结
字数：xxx
章节：xxx
番外：xxx
分类：xxx
标签：xxx

简介：
========================================

【第一卷】

第1章 xxx
...

—————全文完—————
```

## 互转说明

- 导入 → 导出 → 导入：结果一致
- 导出 → 导入 → 导出：结果一致
- 格式保持：书名、作者、章节、番外
