# 样板库索引

情节结构技巧 + 文笔技巧。生成 plot_guide / style_guide 时按需参考。

## Plot — 情节结构技巧

| 文件 | 核心内容 | 适用场景 |
|------|----------|----------|
| `plot/character-entry.md` | 人物入场4式：正面/侧面/全面/忽入 | 角色首次登场 |
| `plot/scene-cut.md` | 场景切入3式：人物场景结合/交替/聚焦递进 | 场景转换、开篇 |
| `plot/chapter-link.md` | 章间衔接5式：引导/暗示/标识/议论/蒙太奇 | 章际过渡 |
| `plot/hook.md` | 开篇钩子4式：冲突前置/社交引爆/悬念/情绪共鸣 | 第1章开篇 |
| `plot/side-character.md` | 配角功能2式：嘴替/信息传递 | 前3章配角 |
| `plot/relationship.md` | 关系突破3式：合作契约/闪婚契约/情感铺垫 | 第3章必备 |
| `plot/meet-cute.md` | 相遇方式4式：专业场合/目的接近/引荐/工作重逢 | 避免撞梗 |
| `plot/original-plot.md` | 原创框架4式：开篇/分手/新男主出场/闪婚动机 | 避免换皮 |

### 各文件关键规则速查

**character-entry.md** — T1正面：肖像+经历+侧面。T2侧面：借他人之口建立神秘感。T3全面：时间+身份+背景+引子。T4忽入：危急场景→突然入场。

**scene-cut.md** — S1人物场景结合：全景→烘托→聚焦。S2人物环境交替：段1交替→段2环境→段3人物。S3画面聚焦递进：情感色彩→展开详写。

**chapter-link.md** — C1引导词（再说/次日）。C2环境暗示下文。C3切词/空段。C4价值判断过渡。C5共有物象蒙太奇。

**hook.md** — K1冲突前置：直接矛盾。K2社交引爆：发现关键信息。K3悬念开场：异常现象。K4情绪共鸣：委屈→共情→转机。

**side-character.md** — P1嘴替：替读者骂人调节情绪。P2信息传递：配角带来线索推动情节。

**relationship.md** — R1合作契约：条件交换绑定。R2闪婚契约：互惠理由充分。R3情感铺垫：长期关注证明。

**meet-cute.md** — M1专业场合。M2目的性接近。M3配角引荐。M4工作重逢。

**original-plot.md** — F1开篇（工作/社交/日常场景）。F2分手（冷静摊牌/公开决裂/自我觉醒）。F3新男主出场（专业/合作/引荐）。F4闪婚动机（专业合作/家族压力/意外）。

## Style — 文笔技巧

| 文件 | 核心内容 | 关键规则 |
|------|----------|----------|
| `style/description.md` | 描写4式：肖像/五感/物象排列/连贯动作 | 动态状态+对比+旁观者视角 |
| `style/dialogue.md` | 对话技巧：标签方式+口语化 | 动作标签替代"说"，短句对轰 |
| `style/sentence.md` | 句式：短句占比/段落节奏/情绪外化/环境密度 | 短句≥35%，禁用直抒情词 |
| `style/pronoun-density.md` | 代词密度控制 | 每100字"她/他"≤3 |
| `style/metaphor.md` | 比喻规则 | 具体可感，嵌入叙事≤1句 |
| `style/object-arrangement.md` | 物象排列模板 | 修真/嫁妆/集市3种格式 |

### 各文件关键规则速查

**description.md** — D1肖像：神情+面容+年龄对比+旁观者收尾。D2五感：单一感官为主。D3物象排列：短词并列制造画面密度。D4连贯动作：动词具体化，A→B→C链条。

**dialogue.md** — 不用"他说/她道"，用动作标签。口语化：打断、抢话、短句对轰。

**sentence.md** — 短句≥35%，连续3句超20字违规。叙事段≤3句，对话叙事交替。情绪用动作外化，禁止写"感到无比XX"。环境描写每章≤3处。

**pronoun-density.md** — 名称替代、省略主语、称呼轮换、合并动作、句式改写。

**metaphor.md** — 具体可感，日常物象，≤1句嵌入叙事。禁止抽象比喻和陈词滥调。

**object-arrangement.md** — 三种格式模板直接套用。
