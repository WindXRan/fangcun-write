# 配角功能技巧

> 来源：@旅行的巨人 高级仿写系列

## P1-闺蜜/助手"嘴替"功能

**结构**：女主受委屈 → 闺蜜/助手替读者骂人 → 调节情绪节奏
**适用**：压抑场景后的情绪释放
**模板**：
```
[女主遭遇不公]，[隐忍/沉默]。
[闺蜜/助手出现]，[愤怒反应]："他怎么敢/她凭什么/这也太过分了！"
[女主反应]：苦笑/摇头/沉默 → 展现人物性格
```
**正向**：「"这个老太婆把你当猴耍呢！"小余气不过，声音都高了，"什么叫'展示才艺'？什么叫'评判'？您可是……"她说到一半，林瑾按住了她的手腕，轻轻摇了摇头。」
**违规**：「林瑾很难过，但没人理解她。」→ 缺少情绪出口，读者憋屈

## P2-配角信息传递功能

**结构**：配角带来关键信息 → 推动情节 → 展现女主处境
**适用**：需要外部信息介入时
**模板**：
```
[配角出现]，带来[信息]。
[信息内容]：揭露真相/提供机会/制造冲突
[女主反应]：震惊/愤怒/期待
```
**正向**：「"姐，你看。"小余把手机递到她面前。屏幕上是朋友圈截图。赵秀芬发的："今晚家宴，准儿媳要来展示她的'修复才艺'。"」
**违规**：「林瑾发现婆婆在朋友圈发了不好的内容。」→ 直接叙述，无场景感
