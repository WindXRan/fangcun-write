# 相遇方式技巧

> 来源：@旅行的巨人 高级仿写系列

## M1-专业场合相遇

**结构**：女主参加专业活动 → 男主也在场 → 因专业互动而相识
**适用**：职业线强的作品，建立专业认可
**模板**：
```
[女主]参加[专业活动：拍卖会/展览/学术会议]。
[男主]也在场，[身份/角色]。
[专业互动]：男主注意到女主的专业能力/观点
→ 建立第一印象
```
**正向**：「林瑾在拍卖预展上指出那件青花瓷是赝品，全场哗然。角落里一个穿白大褂的男人放下咖啡杯，目光落在她脸上。」
**违规**：「林瑾走在路上，差点被一辆车撞到。」→ 过于常见，缺乏专业性

## M2-目的性接近法

**结构**：男主主动来找女主 → 带着明确目的 → 建立合作关系
**适用**：需要快速建立男女主联系
**模板**：
```
[男主]出现在[女主的工作场所/住所]。
目的：[合作/求助/调查]
[对话]：男主说明来意，展示对女主的了解
→ 建立联系
```
**正向**：「顾西洲的宾利停在修复室门口。他推开车门，手里拿着一份修复方案：'林小姐，这个方案是你的？'」
**违规**：「顾西洲在路上遇到了林瑾。」→ 缺乏目的性，显得巧合

## M3-配角引荐法

**结构**：配角介绍男女主认识 → 背景交代 → 建立初步联系
**适用**：需要自然的相遇方式
**模板**：
```
[配角]安排[男女主]见面。
[配角介绍]：说明双方身份/背景
[男女主互动]：初次对话，建立印象
```
**正向**：「陈屿拉着林瑾走进拍卖行VIP室：'顾教授，这就是我跟您提过的林瑾，修远的金奖修复师。'」
**违规**：「林瑾和顾西洲在街上偶遇。」→ 缺乏自然性和目的性

## M4-工作场景重逢法

**结构**：男女主在工作场景重逢 → 回忆过去 → 建立新的联系
**适用**：有前史设定的作品
**模板**：
```
[女主]在[工作场景]工作。
[男主]出现，[身份/目的]。
[回忆]：两人曾经的交集
→ 建立新的关系
```
**正向**：「林瑾正在修复室整理碎片，门被推开。她抬头，看见三年前在修远协会见过的那个男人——顾西洲。他手里拿着一块永乐官窑碎片：'林小姐，这个你能修吗？'」
**违规**：「林瑾和顾西洲在咖啡馆偶遇。」→ 缺乏工作场景的专业性
