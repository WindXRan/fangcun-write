# 描写技巧

> 来源：@旅行的巨人 高级仿写系列

每条必须有：规则 + 正向示例 + 违规示例

## D1-肖像描写

**结构**：神情 + 发型/面容 + 年龄对比 + 状态反差
**规则**：
- 不写静态属性（"他有一头黑发"），写动态状态（"满头黑色短发失去了光泽"）
- 用对比制造反差：往日意气风发 vs 当前死灰
- 旁观者视角收尾：「[旁观者]心想，[角色]已经[判断]」
**正向**：「父亲神情肃穆地骑在马 上，满头棕色长发在风中飞扬。他修剪整齐的胡子里冒出几缕白丝，看起来比三十五岁的实际年龄要老些。」
**违规**：「他长得很帅，眼睛很大，有一头黑发。」→ 纯静态描述，缺乏动态和情感

## D2-五感描写

**结构**：前提概要 + 五感之一 + 场景描写 + 语言（可选）+ 感觉陈述
**规则**：选一个感官为主，不要五感全上
**五感正向**：
- 嗅觉：「社区带着警察刚一推开门，众人连忙捂住了鼻子，老单身汉的床底下传出一阵阵恶臭」
- 触觉：「林钰顿时耳鸣目眩，脸上火辣辣的，又疼又麻」
- 视觉：「衔远山，吞长江，浩浩汤汤，横无际涯」
- 听觉：「宿舍的水龙头失修已久时不时会渗水滴下，全身的精力都被滴水声缠上」
- 味觉：「一股腥臊味在味蕾上炸开，席卷了五脏六腑」
**违规**：「房间里很臭，很难受。」→ 缺乏感官细节和身体反应，直接告诉读者感受

## D3-物象排列

**结构**：人物行为 + 接续物象 + 环境刻画
**规则**：
- 物象用短词并列，制造画面密度
- 形容词+名词的结构：「通红的辣椒，翠绿的油菜，肥美的鸭子」
- 物象服务于情绪：欢快场景用明亮物象，压抑场景用灰暗物象
**正向**：「炼体道门，法术神通，符箓咒文，心经，密宗偏门，奇门遁甲。缩地成寸，撒豆成兵。真言，手决。更甚人生果嫁接法，蟠桃熏烟御寒术。凡所应有，无所不有。」
**违规**：「秘籍有很多，各种各样的都有。」→ 无具体物象，无画面密度

## D4-连贯动作描写

**结构**：一系列带有动词的连贯描写，形成动作流
**规则**：
- 动词要具体：不说"走过去"，说"贴着墙边缘小心挪动"
- 动作链条不断：A→B→C，中间不插议论
- 可在动作流末尾加一句身体反应（汗珠、喘息等）
**正向**：「市长一板一眼，一家一家看过来……还爬上了农家的小楼，掀开农民储粮、储米、盛油、盛酒的缸盖，然后伸手进去摸一摸……末了，市长还随手抓一把稻谷出来，挑几颗放进嘴里咬咬……」
**违规**：「市长检查了农民的储粮，发现情况还不错。」→ 动作被叙述替代，缺乏连续性
