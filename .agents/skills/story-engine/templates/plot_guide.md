### 结构功能参考
开场钩子 / 建立困境 / 推进冲突 / 心理活动 / 转折点 / 释放爽点 / 埋钩子 / 场景过渡

### 冲突类型参考
身份冲突 / 利益冲突 / 信息差冲突 / 情感冲突 / 目标冲突 / 道德冲突 / 时间压迫 / 空间限制

⚠️ 规则：
- "冲突类型"必须换（源文身份→新书利益/信息差/道德，不能相同）
- "社交场景"必须换（源文饭局→新书不能饭局；源文在家→新书不能在家）
- "动作"和"反应"必须全换（不能只改人名地名，事件本身要不同）
- "结构功能"和"情绪+强度"照搬（一一对应，不可改）

## 换皮检验 ⚠️

把新书所有节拍剥掉人名地名，用一句话概括：
→ （概括）
→ 读者看到这句话能想起源文吗？能→换皮，重做。不能→合格。

## 角色高光时刻 ⭐（每章至少一个）

| 类型 | 说明 | 示例 |
|------|------|------|
| **甜** | 两人之间不经意的心动瞬间 | 他把她做的糊鸡蛋全吃了 |
| **虐** | 不得不隐瞒/伤害对方的时刻 | 嘴上说分手，手在发抖 |
| **笑** | 轻松幽默的桥段 | 女主逞强做饭差点烧厨房 |
| **反差** | 角色做出人意料的事 | 一直冷漠的人突然护短 |
| **擦边** | 暧昧肢体接触 | 帮她系安全带靠太近 |
| **打脸** | 被嘲笑后实力碾压 | 众人嘲笑废物，主角一招震全场 |
| **装逼** | 低调展示实力 | 淡淡一句"还行" |
| **绝境翻盘** | 濒临失败时逆转 | 快死时觉醒新能力 |
| **热血** | 激发斗志的宣言/行动 | "这一战，我接了！" |

**格式**：类型 + 节拍#N + 一句话描述 + 为什么读者会记住
