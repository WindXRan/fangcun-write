**仿写规则**：剥掉人名地名后认不出源文（场景/冲突/道具/对话换类不换名）。品类特征对标源文。禁止连续8字相同，冲突/动作/反应全换。角色名严格用设定。

**段落节奏**：对标源文的叙事节奏——紧张段落不拉长，平缓段落不压缩。段落数量和每段长度不作硬性对齐。

**字数控制**：以对应 phase 的 prompt 中字数规则为准。若明显超出或不足，自行删补。

**输出格式**：只输出要求内容，无解释/标记/元评论。字数标注按对应 phase prompt 的格式要求执行。

**禁用路标词**（包括但不限于——且包括任何类似结构的短语）：
值得注意的是、毋庸置疑、不可否认、不言而喻、换言之、具体来说、显而易见、换句话说、正是如此、这意味着、简而言之、总的来说、总之、最后、首先、其次、此外、另外、当然、事实上。
也不得使用任何类似的总结、过渡、强调性短语（如"重要的是"、"关键是"、"一般来说"、"也就是说"、"从某种意义上说"、"在这其中"、"归根结底"、"由此可见"）。
**最终输出前，必须逐句扫描是否含有以上任何一词或与其结构相同的词（如"值得一提的是""不可忽略"）。若发现，立即替换为具体动作/感官/对话描写。**

**反抄袭**：不得与源文连续8字相同。对话场景/冲突/动作链不得照搬源文结构。

**角色名**：严格使用设定中的角色名，不得使用源文角色名。
