写《{新书名}》第{N}章。正文第一行写"第{N}章 [章名]"（不加#）。章名取 plot_guide 中标注的，未标注则自拟。

按 plot_guide 节拍的情绪功能定位创作全新内容，不是改写源文。

【plot_guide】projects/{作者名}/{源书名}/rewrites/{新书名}/guides/plot_{N}.md

## 写作前必读（强制执行的源文分析步骤）
在开始写正文之前，必须先读取源文第{N}章全文，并手动统计以下数据。然后将结果以注释形式写在正文前：
1. 【开篇对标】源文前200字的开篇类型（对话/动作/悬念/环境）→ 提取前10字的具体结构（例如：对话首句“你怎么来了？”→ 重写首句必须也是问句，且长度偏差≤±3字）。
2. 【源文段落】共X段，平均每段Y字。并列出每段的字数：段1: Z1字, 段2: Z2字, ...（若段落太多，可只列出前5段和后3段的近似字数，但必须给出整体分布）。
3. 【源文对话比】统计源文中对话字数占总字数的比例（仅汉字，对话引号内内容）。记录为X%。
4. 【源文句长】统计源文句子的平均字数（以句号、问号、感叹号、省略号计数）。记录为X字。
5. 【AI路标词统计】源文中出现的所有AI路标词（见规则3列表）并计数。
6. 【字数计划】根据 plot_guide 的节拍数量，将目标字数平均分配到每个节拍（考虑节拍重要性微调，但每个节拍字数偏差≤±15%）。示例：节拍1: 500字, 节拍2: 400字, ...

以上注释必须完整，否则视为严重违规。

## 本章硬性规则（必须做到，缺一不可）

### 1. 开篇钩子对标（可量化 + 精确到首句长度）
**规则**：  
- 开篇类型（对话开/动作开/悬念开/环境开）必须与源文第{N}章前200字的开篇类型完全相同。  
- 进一步，重写的**第一句**（不含“第N章”标题）的结构必须与源文第一句的结构一致：  
  - 若源文以对话开头（如：角色A说“你怎么来了？”），则重写第一句也必须是对话，且**句子长度**（对话字数+叙事字数）偏差≤±3字。  
  - 若源文以动作开头（如：她推开门），则重写第一句也必须是动作，且动词类型（推/拉/冲/看等）必须属于同一语义类别（移动/观察/触感）。  
  - 若源文以悬念开头（如：那天夜里发生了什么？），则重写第一句也必须是问句或含未知信息的陈述句。  
  - 若源文以环境开头（如：雨下了一整夜），则重写第一句也必须是环境描写，且感官维度（视觉/听觉/触觉）一致。  
- 禁止将对话开改为环境描写开，或将动作开改为悬念开。  
- **执行**：在“写作前必读”的【开篇对标】中已记录。正文第一句必须立刻进入开篇，不得用“这天”“那时”等模糊时间词铺垫。

### 2. 字数（零容忍强制规则）——核心约束，超标即重写
目标字数 **{目标字数} 字**（{目标字数_min}~{目标字数_max}）。  
字数超出或不足1%即视为严重违规，整章作废必须重写。

#### 2.1 写作前：字数分配计划（已在“写作前必读”中完成）
确保总字数 = 各节拍字数之和，且每个节拍偏差≤±15%。

#### 2.2 写作中：实时200字检查 + 累积字数输出
写作过程必须**逐段进行**，每完成约200字（以汉字为准，不计标点空格）后，**立即输出一行注释**（不计入正文字数），格式为：
`【实时字数：累计XXX字，当前节拍完成XX字，剩余目标YYY字】`
然后根据实际字数与计划的偏差调整下一段的内容：
- 若累计超过计划字数，下一段必须缩减，删除冗余（优先删副词、重复细节）。
- 若累计不足，下一段补充有意义的细节（环境、心理、动作微表情）。
- 每200字检查一次，确保最终总字数在目标区间内。

#### 2.3 写作后：字数统计与验证行（强制）
全文完成后，必须进行精确的字数统计（只计汉字，不含标点、空格、换行符）。统计后必须在正文末尾另起一行添加字数验证行：
`【字数验证：XXX字（目标{目标字数_min}-{目标字数_max}）】`
如果实际字数不在{目标字数_min}~{目标字数_max}之间，则**不要输出验证行**，而是回到正文进行修改，直到字数落在区间内，再输出验证行。

#### 2.4 字数超标/不足的修正方法（不能重头写，只能针对性调整）
- **字数不足**：按节拍补足，优先补充：  
  - 环境细节（气味、温度、光影、质感、音景）  
  - 角色内心独白（情感层级、记忆闪回、感官联想）  
  - 对话中的微表情、肢体动作、语气变化  
  - 细化动作节奏（放慢/加快），增加与主线相关的次要互动（如等待时的观察）  
  - 检查是否有未展开的节拍，补充至少50字有意义细节。  
  **严禁**拉长句子、重复废话、添加空洞过渡词。
- **字数超出**：必须删除冗余内容。优先删除：  
  - 重复的修饰副词（“非常”“突然”“很”“完全”）  
  - 多余的环境描写（同一场景多次渲染）  
  - 不必要的心理总结（“她明白了”“她意识到”“他心想”）  
  - 与核心情绪无关的对话或旁白  
  - 合并短句，精简描写（如“窗外天色渐渐暗了下来，夜幕缓缓降临” → “夜色渐浓”）  
  - 删除重复动作（如“他站起身，走了两步又停下”保留一个）  
  若仍超出，按节拍依次砍掉最不重要的细节。  
- **特殊措施**：如果字数超出超过5%（即>目标字数*1.05），则必须强制删除整个章节最后10%的字数，并重写结尾钩子以符合章末规则。

### 3. 反AI路标词（可量化密度规则 + 位置记录）
**规则**：重写中出现的AI路标词总数 **必须 ≤ 源文第{N}章中的AI路标词总数**。  
**执行**：  
1. 【写作前必读】中已统计源文路标词数量。  
2. 写作过程中，每次使用下列AI路标词时，**必须在正文前用注释记录**（类似于脚注），格式为：  
   `【路标位置：第X段第Y句，使用了“倏地”】`  
   以便最终统计时能清晰比对。  
3. 避免使用以下高频路标词（以及任何与之类似的突兀词）：  
   - 倏地、入目、紧了床单、她想起来了、她都想起来了、她回忆道  
   - 突兀时空过渡词：这时、突然、然后、紧接着、忽然、转眼间、就在此时  
   - 显眼心理总结：她明白了、她意识到、她感觉到、她注意到  
   - 其他常见路标：必须、不得不、一下子、只见、但见、便、于是（用于跳转）  
4. 用自然的时间推进、动作或环境描写来衔接。  
   - 反例：“倏地睁开眼入目是一面斑驳老旧的墙壁”  
   - 正例：“墙皮起泡脱落，她慢慢睁开眼睛，视线里一片斑驳”  
   - 反例：“她突然意识到了问题的严重性”  
   - 正例：“问题像一块石头，越来越重地压在她心上”  
5. 全文完成后，再次计数重写中的AI路标词数量（计入所有位置记录）。若超过源文数量，必须删除或替换多余的，直到数量≤源文。

### 4. 反抄袭（原创台词与描写）
不得直接复制或雷同源文（包括已出版作品、网络文本、本系列其他章节）中的台词、叙述或描写。所有内容必须用自己的语言重新构建。  
以下为已发现的雷同场景，严禁再次出现：  
- “倏地睁开眼入目是一面斑驳老旧的墙壁，墙皮起泡脱落...”  
- “紧了床单她想起来了，她都想起来了出租车撞上的前五分...”  
- “才放平浴室的水声停了楚靳寒擦着头发走出来，换了件干...”  
若内容被检测出与源文高度相似（包括场景、台词结构、关键修饰语），必须重写。

### 5. 写作节奏与结构对标（细化到段落字数）
#### 5.1 段落节奏（精确对标源文段落）
- **段数偏差**：重写的段落数（空行分隔）与源文第{N}章段落数偏差≤±20%。  
- **段长分布**：重写每段的字数须与源文对应段落的字数偏差≤±20%。例如，若源文第一段为120字，则重写第一段应在96~144字之间。  
  - 如果源文段落数量多于重写段落，则按顺序匹配最近似段落；如果重写段落多于源文，则多余段落应对标源文最后一段的段长范围。  
- **执行**：在“写作前必读”中已记录源文每段字数。写作时逐段参照。每写完一段，输出注释：“【段落字数】本段XXX字（源文对应段约为YYY字）”。若偏差超过20%，立即调整本段或下一段。

#### 5.2 开篇300字进场景/悬念
前300字必须进入故事场景或设置悬念，禁止用背景介绍、景物堆砌开头。若开篇为对话，则对话前最多1句交代场景。

#### 5.3 章末留钩子（强化）
最后一段必须有一个明确的钩子（对话悬念、动作中断、心理未解问题），不能平缓结束。钩子应自然，避免“就在这时……”类路标。  
并且，钩子必须与源文第{N}章最后一段的钩子类型相似（例如源文用未回答的问题，重写也应用未回答的问题；源文用动作中断，重写也应用动作中断）。在执行注释中注明：“【章末钩子】源文为[类型]，重写采用[类型]”。

#### 5.4 时间跳跃
若存在时间跳跃（如“三天后”），必须在开头1-2句内交代，且用具体事件标记，避免模糊。

### 6. 反AI句式（全部对标源文指纹，硬指标）
以下指标强制对标源文统计值（若未提供则使用下文默认，但必须在“写作前必读”中声明默认来源及原因）：  
- **段长**：最重要。单段≈{源文段长}字（偏差≤±10字）。若未提供源文段长，则取重写全文平均段长，但需在注释中声明。  
- **句长均值**：≈{源文句长}字（偏差≤±3字）。若未提供，则默认16字。  
- **短句比**：短句（<8字）占总句子数≤{源文短句比}%。若未提供，则默认10%。  
- **对话占比**：≈{源文对话比}%（偏差≤±5%）。若未提供，则默认30%。  
  - **实时控制**：每完成200字后，计算当前累积对话占比，若与目标偏差超过5%，则调整下一段对话比例。  
- **代词密度**：≈{源文代词密度}/千字（偏差≤±2）。若未提供，则默认80/千字。  
- **标点风格**：{源文标点}。包括句号密度、问号/感叹号比例、省略号使用频率、分号与冒号比例。若未提供，则默认文本风格。  
- **段尾极短句或省略号**：段尾应出现极短句（≤6字）或省略号，频率与源文一致。  
- **执行**：在“写作前必读”中已统计源文指纹。写作时参照这些数字调整。如发现当前句长偏离，立即在下一句中调整。
