### 🔴 擦边浓度对标（都市后宫/男频擦边类专用）

**源文的擦边不是缺点，是赛道特征。新书必须对标，不能降级。**
**终极标准：让读者起生理反应——男读者勃起，女读者高潮。达不到就是擦边不到位。**

| 对标维度 | 源文标准 | 新书标准 |
|---------|---------|---------|
| 男性凝视 | 每章至少2处女性身体部位描写 | 同等浓度 |
| 性张力 | 借体检/检查/护送之名行暧昧 | 同级别场景设计 |
| 多女好感圈 | 多名女性对男主有明确兴趣 | ≥2个女配，同等浓度 |
| 系统爽感 | 系统任务+奖励+惩罚循环 | 必须有同级别系统互动 |
| 直白口吻 | 口语化+粗口+自嘲 | 同等直白度，不文学化 |
| 肢体接触 | 女性主动贴贴 | 同级别接触，不降级 |
| **系统命名** | 系统名本身就是擦边玩梗 | 系统名必须擦边/谐音/玩梗，禁止学术命名 |

### 🔴 系统命名硬规则（都市擦边专用）

**系统名字就是第一道擦边。** 禁止用"XX检测系统""XX辅助分析"这种学术命名，必须用擦边/谐音/玩梗的名字，让读者看到系统名就笑/就硬。

**命名技法**：
- **谐音梗**：「好色之图」（好色之徒）、「摸鱼系统」（摸鱼+摸人）、「心有所鼠」（心有所属）
- **双关语**：「体检报告单」（体检=检查身体=暧昧）、「贴贴系统」「好感探测器」
- **直白擦边**：「后宫管理系统」「红颜攻略器」「桃花运检测仪」
- **反差萌**：系统名正经，实际功能擦边——「情绪分析仪」实际功能是"检测女性对男主的好感度+身体接触意愿"
- **玩梗**：结合当下网络热梗——「牛头人警报器」「绿茶识别器」「白月光探测仪」

**系统命名必须满足**：
1. 名字本身就有擦边/暧昧暗示
2. 读出来就能让男读者会心一笑
3. 与系统实际功能（擦边相关）匹配

❌ 禁止：「情绪检测系统」「微表情辅助分析」——学术腔，零擦边
✅ 示例：「贴贴雷达」——看到名字就知道系统功能是检测女性对男主的身体接触倾向

**🔴 擦边展开硬规则（写入 style_guide 和 write-chapter，每章必须执行）：**
- 每章至少1段完整的擦边展开（≥3句感官细节），不能一笔带过
- 擦边节拍的感官五件套：触觉（皮肤温度/质感）、视觉（身体部位特写）、嗅觉（靠近才闻到的体香）、听觉（呼吸/心跳/衣物摩擦）、心理/生理（心跳加速/喉咙发干/身体僵硬）
- 男主必须有生理反应描写，不能全程"面无表情"或"无动于衷"
- ❌禁止：「小李大腿贴过来，擦过郝仁的膝盖。」一句带过就走
- ✅必须：写出温度、质感、男主的身体反应、心跳变化、空气中的暧昧气息

### 🔴 都市擦边开书阶段强制输出（settings/ 中必须包含）

**以下内容必须在开书阶段产出，写入 settings/ 文件，不能留到写章阶段再补。**

#### 1. 系统设计（写入 settings/book_info.md 或 settings/world.md）

系统是都市擦边的核心爽点引擎，开书阶段必须设计完成：

| 设计项 | 要求 |
|--------|------|
| **系统名** | 必须擦边/谐音/玩梗（见上方命名规则），不能是「XX检测系统」 |
| **系统定位** | 系统的核心身份——是"贴身保镖""红娘""后宫管家"还是"任务发布器"？ |
| **核心功能** | 系统能做什么？列出3-5个主要功能（必须与擦边挂钩） |
| **触发条件** | 什么条件下触发系统？（身体接触/情绪波动/特定场景/定时触发） |
| **奖励机制** | 完成任务奖什么？（技能/属性/特权/信息/金钱） |
| **惩罚机制** | 不完成任务罚什么？（扣属性/强制执行/尴尬场景/失去能力） |
| **擦边功能** | 系统如何制造暧昧机会？（任务设计/场景触发/信息提示/强制接触） |
| **系统提示风格** | 系统弹窗的措辞风格——必须擦边（见 write-chapter 规则） |

**示例（非直接使用，仅供理解）**：
- 系统名：「贴贴雷达」
- 定位：女性好感度+身体接触倾向检测器
- 功能：①检测女性对男主的好感度百分比 ②检测近距离体温变化 ③发布"贴贴任务" ④解锁技能作为奖励
- 触发：男主与女性距离<1米时自动激活
- 奖励：完成任务解锁格斗/推理/医疗等技能
- 惩罚：超时不完成→24小时内失去该技能
- 擦边功能：任务强制要求肢体接触（"拥抱""牵手""近距离对视"），不完成就扣能力
- 提示风格：「她的心跳比你快了17%」「检测到近距离体温升高」「警告：距离过近，荷尔蒙浓度超标」
