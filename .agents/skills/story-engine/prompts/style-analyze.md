读这一章，输出仿写操作手册。不要描述文本，要给出可以直接执行的写法指令。

## 风格类型（🔴 仿写必须保持一致，不可偏离）

从本章判断源文的风格类型：
- **题材类型**：{如：都市家庭轻喜剧、校园甜宠、悬疑推理等}
- **叙事基调**：{如：轻松诙谐、温馨治愈、暗黑沉重等}
- **幽默风格**：{如：吐槽玩梗、冷幽默、反差萌、毒舌互怼等，无则写"无"}
- **情绪基调**：{如：暖心、虐心、爽感、紧张等}

**⚠️ 铁律：** 仿写必须保持相同风格类型。源文是喜剧就不能写成悬疑，源文是轻松日常就不能写成暗黑沉重。风格类型是读者选择这本书的根本原因。

## 场景运作机制（🔴 仿写必须遵循这个机制，不是参考）

分析本章 2-3 个关键场景，每个场景拆解"这个场景是怎么运作来达到效果的"。

### 场景1：[场景名]
- **功能**：这个场景在章中承担什么角色？（建立关系/制造笑点/推进冲突/情感爆发/节奏缓冲）
- **结构**：这个场景是怎么运作的？
  - 开头：用什么方式引入？（误会前置/对话切入/动作引入/环境铺垫/旁观者视角）
  - 中段：怎么推进？（冲突升级/情绪反转/信息揭示/角色互动升级）
  - 收尾：怎么收束？（点睛句/旁观者反应/动作收尾/留白悬念/情绪落点）
- **节奏**：这个场景的情绪曲线是什么？（紧张→释放/轻松→爆发/温馨→反转/压抑→爽感）
- **技法**：用了什么具体技法制造效果？（反差萌/毒舌互怼/误会乌龙/夸张动作/旁白吐槽/信息差/欲扬先抑）

### 场景3：[场景名]

**⚠️ 铁律：** 仿写时，必须用同样的场景结构（开头方式→推进方式→收尾方式）和同样的节奏曲线（如紧张→释放），但场景内容、对话、人物必须是全新的。不是模仿源文的内容，是模仿源文的运作方式。

## 正面（写的时候这样做——每条一条指令+一个原文例句）

挑 7-8 条本章独有的写法特征，每条格式为：
- 指令：[具体可执行的写作动作]
- 例句：从本章摘一句话作为示范

注意：
- 如果特征是"多用省略号"，不必说——锚点里已经有了
- 专注锚点看不出的特征：怎么起句、怎么收段、怎么让对话带上情绪、怎么在叙述中插入画面
- **必须包含**：本章句首词分布规律（哪些句子以人名开头、哪些以代词/环境/动作/对话开头，比例大致如何）

## 反面（写的时候绝对不能这样做——每条一个具体踩雷点+为什么）

挑 2-3 条本章写法最容易在仿写时走样的点，每条格式为：
- 踩雷：[具体的错误写法]
- 原因：为什么仿写时容易犯

## 多样性指标（必须统计，用于校准仿写的多样性目标）

统计本章的以下指标，输出为表格：

| 指标 | 数值 | 说明 |
|------|------|------|
| 对话标签种类 | {N}种 | 如：说/道/问/开口/喊/嘟囔/冷笑/叹气... |
| 说/道/问 占比 | {N}% | "XX说""XX道""XX问"占全部对话标签的比例 |
| 动作替代标签占比 | {N}% | 用动作描写替代对话标签的比例 |
| 情绪反应种类 | {N}种 | 如：攥紧手指/咬嘴唇/眼眶红/肩膀抖/愣住... |
| 结尾手法 | {类型} | 如：对话收尾/动作收尾/心理收尾/环境收尾/悬念收尾 |
| 环境意象 | {列举} | 如：石榴树、窗外的鸟、井水的光 |
| 高频词(≥3次) | {列举} | 如：顿了顿(3次)、攥紧(4次) |
| 回忆插入 | {N}次 | "她想起"出现次数 |
| 伪格言句 | {N}句 | 独立短句（"陈述句+。"）数量 |

**用途：** 仿写时，对话标签种类数和情绪反应种类数必须≥源文数值，说/道/问占比必须≤源文数值。如果源文本身多样性不足（如标签种类只有2种），仿写时应提升到至少4种。
