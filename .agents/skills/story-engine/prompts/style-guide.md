## 任务

上面的【风格模板】已经由脚本自动生成了定量数据（字数、句长、比喻、AI词等），但"LLM 补充"部分是空的。

你的任务：读源文，填充模板中所有"LLM 补充"的占位符。**不要改动脚本已生成的数据部分。**

### 需要填充的内容

1. **句式规则**（2-3条）：从源文提取句式特点，每条含 ✅正向示例 + ❌反向示例
2. **对话规则**（2条）：对话标签方式、口语化程度，每条含示例
3. **描写规则**（2-3条）：情绪表达、环境描写、五感使用，每条含示例
4. **代词密度**：统计源文中以"他/她"开头的句子占比，仿写以此为上限（+2%容差）
5. **本章情绪节奏**：开篇→发展→高潮→结尾的情绪变化
6. **🔴 字数配比规则**（必须）：读【源文】，统计以下比例，作仿写硬约束：
   - **对话占比**：源文对话字数/总字数 = X%，仿写目标 X% ± 5%
   - **内心戏占比**：源文内心独白字数/总字数 = X%，仿写目标 X% ± 5%（括号内的内心活动算内心戏）
   - **叙事/描写占比**：源文叙事描写字数/总字数 = X%
   - **示例**：源文对话62% + 内心12% + 叙事26% → 仿写对话57~67% + 内心7~17% + 叙事21~31%
   - **统计方式**：按字符数统计，不是按行数
7. **🔴 段落结构规则**（必须）：读【源文】，统计段落数，作仿写约束：
   - **段落数**：源文= X段，仿写目标 X段 ± 20%
   - **单句段落比**：源文= X%，仿写目标 X% ± 10%
   - **示例**：源文43段/单句段49% → 仿写34~51段/单句段39~59%
   - **目的**：防止段落过度碎片化（源文写一大段你拆成三小段 = 节奏变拖）

### 填充要求

- 每条规则必须是可验证的（写完能对着正文打勾）
- 示例必须从源文原句中提取（不要编造）
- 保持模板原有格式，只填充占位符部分
- 控制在 100 行以内

### 输出格式

直接输出填充完成的完整 style_guide，包含：
1. 脚本自动生成的定量锚点 + 修辞特征 + 句首高频模式 + 防AI检测 + 字数控制
2. 你补充的：句式规则 / 对话规则 / 描写规则 / 代词密度 / 情绪节奏
3. 你补充的：🔴擦边规则（如品类有要求，6条）
4. **你新增的**：字数配比规则（对话占比X% ±5%、内心戏占比X% ±5%、叙事描写占比X%）
5. **你新增的**：段落结构规则（段落数X ±20%、单句段落比X% ±10%）
