# Guide 衔接修复

以下为同一部小说的多章章纲（plot_guide），请检查相邻章之间的衔接问题。

## 检查项

1. **角色名一致性**：各章使用的角色名是否完全一致（注意：第一部可能有配角名变化是因为角色正常登场，只要不冲突即可）
2. **人设一致性**：角色的身份、性格、行为模式在跨章时是否连贯
3. **时间线连贯性**：相邻章的时间顺序是否合理，有无跳跃或回溯
4. **场景连续性**：章与章之间的场景转换是否有交代
5. **主线一致性**：各章是否在同一条主线/冲突线上推进，有无另起新线
6. **钩子呼应**：上一章结尾的悬念/约定/未完成事件，下一章是否有回应

## 修复要求

- 对**有断裂**的章节，重写其角色列表和节拍映射表中的事件，使其与前后章连贯
- 对**无断裂**的章节，保持原样不动
- **必须保持各章节拍数量、结构功能、情绪曲线与源文的对应关系不变**（这是仿写骨架）
- 只改情节事件、角色名、场景名，不改字数分配、冲突类型标签
- 禁止引入新角色（除非已在本批其他 guide 中出现）
