番茄小说平台审查标准。在统一审查基础上追加以下维度：

## 番茄专项维度

### 1. 开头钩子（前 500 字）
- [ ] 是否在 500 字内建立了可感知的核心冲突/悬念/情绪锚点？
- [ ] 是否有「一眼看得懂」的核心设定？（读者不需要积累前情就能进入）
- [ ] 是否有「想翻下一页」的即时钩子？（好奇/共鸣/爽感/害怕，至少占一个）

### 2. 冲突频率
- [ ] 每 2000 字至少有一个冲突推进点（身份/利益/信息差/道德）
- [ ] 冲突不能「打了就完」——每次冲突必须有后续涟漪
- [ ] 小冲突的频率是否保持了「翻一章就有事发生」的节奏？

### 3. 情绪兑现
- [ ] 每章至少有一个情绪兑现点（甜/虐/爽/乐，匹配题材定位）
- [ ] 情绪兑现不能铺垫后落空（例：三章铺垫了打脸，结果被打脸对象消失了）
- [ ] 高频情绪轮换：相邻两章的情绪基调应有差异（不能连续三章都是「甜」）

### 4. 理解门槛
- [ ] 世界观/设定是否在 3000 字内完成基础交代？（不用术语堆砌）
- [ ] 角色关系是否一眼看得懂？（不用记复杂的族谱/帮派名称）
- [ ] 每章的开头 200 字是否能快速定位到「当前发生了什么」？

### 5. 打脸节奏（番茄特色）
- [ ] 是否建立了「被低估/被嘲讽」→「打脸」的爽感循环？
- [ ] 打脸间隔：小打脸 3-5 章一次，大打脸 10-15 章一次
- [ ] 打脸的对象是否有足够的「可恨度」让读者期待打脸发生？

## 评分规则
每项 0-2 分（0=不达标，1=部分达标，2=完全达标），满分 10。

- 8-10 分：番茄适配优秀
- 5-7 分：需调整节奏/冲突密度
- 0-4 分：建议换平台或大改节奏
