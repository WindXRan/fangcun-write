基于源文分析，为《{新书名}》生成角色设定。

## 源文分析（已锁定，不可更改）

{源文分析}

## ⚠️ 角色锁定铁律

**仿写角色 = 换名字不换灵魂。**

### 必须保留
- **功能位数量**：源文有几个主要角色，新书就有几个
- **关系结构**：谁和什么关系、谁推动什么剧情，功能位不变
- **性格内核**：表面性格可以微调，但核心驱动力和行为模式不变
- **行为模式卡片**：应激模式、决策方式、情感表达必须与源文对应角色一致

### 可以换
- 姓名、外貌、职业、住处
- 说话风格（但情绪功能不变）
- 具体记忆点（但类型不变，如源文用"口头禅"，新书也用"口头禅"）

## 起名铁律

- 名字像真人，禁用AI审美
- 禁用字：男主(寒琛霆辰砚辞屿淮洲舟泽衍瑾瑜墨熙尧翊宸) / 女主(晚念栀眠清意棠予初浅瑶萱璇颖苒若汐诺涵) / 通用(婳翊珩彧旻昶堇蓁莞芊筱妤淇沄璟)
- 现代男主单名、古代男主双名、年代朴素
- 配角常见姓普通名，禁用全员诗意双名
- **角色名必须与源文完全不同**

## 输出内容

每个角色必须标注**源文对应角色**，格式：

```
【角色名】（源文对应：{源文角色名}）
- 功能位：{与源文一致}
- 性格内核：{与源文一致}
- 五要素：身份/特征/境遇/行动/结局
- 行为模式卡片：应激/决策/情感/弱点/动机/能力
```

- [ ] 角色数量与源文一致
- [ ] 每个角色都能指出源文对应角色
- [ ] 性格内核与源文对应角色一致
- [ ] 行为模式卡片与源文对应角色一致
- [ ] 关系结构与源文一致（不新增、不删除功能位）
