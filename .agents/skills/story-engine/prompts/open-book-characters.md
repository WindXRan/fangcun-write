## ⚠️ 角色锁定铁律

**仿写角色 = 换名字不换灵魂。每个角色都必须有新名字，一个都不能漏。**

### 输出要求

**第一步：角色名映射表**（必须包含源文角色清单中的所有角色，不可遗漏）

格式：
```
| 源文名 | 新名 | 性别 | 功能位 |
|--------|------|------|--------|
| 李观澜 | {新名} | 女 | 女主 |
| 江流 | {新名} | 男 | 男主 |
| 高慧云 | {新名} | 女 | 养母 |
| 赵萌萌 | {新名} | 女 | 闺蜜 |
| ...每一个角色都要列出... |
```

**起名铁律：**
- 新名必须与源文名完全不同（不是加字、不是换一个字、不是换姓）
- 名字像真人，禁用AI审美
- 禁用字：寒琛霆辰砚辞屿淮洲舟泽衍瑾瑜墨熙尧翊宸 / 晚念栀眠清意棠予初浅瑶萱璇颖苒若汐诺涵 / 婳翊珩彧旻昶堇蓁莞芊筱妤淇沄璟
- 合并条目必须拆开（如"林建华/林远征/林兴源"必须分别起3个不同新名）
- **名字区分度**：不同角色的名字不能有相同字、不能谐音、姓氏要多样化、长度要混合（单名+双名）

**第二步：角色卡**（每个角色的详细设定）

## 起名铁律（最高优先级，违反即不合格）

- **每个角色名必须与源文完全不同** — 不可以保留源文的姓或名，不可以在源文名前加字（如源文"张凡"不能起"苏张凡"），不可以只换一个字（如"李建国"→"李建民"）
- 名字像真人，禁用AI审美
- 禁用字：男主(寒琛霆辰砚辞屿淮洲舟泽衍瑾瑜墨熙尧翊宸) / 女主(晚念栀眠清意棠予初浅瑶萱璇颖苒若汐诺涵) / 通用(婳翊珩彧旻昶堇蓁莞芊筱妤淇沄璟)
- 现代男主单名、古代男主双名、年代朴素
- 配角常见姓普通名，禁用全员诗意双名
- **合并条目必须拆开**：如源文有"林建华/林远征/林兴源"，必须分别起3个不同的新名

## 输出内容

**先输出映射表，再输出角色卡。每个角色都必须列出，不分主次。**

映射表之后，每个角色的详细设定格式：

```
【新角色名】（源文对应：{源文角色名}）
- 功能位：{与源文一致}
- 性格内核：{2-3句话描述性格本质}
- 核心动机：{他/她最想要什么}
- 关系：{与主角/其他角色的关系}
```

**注意：不要写"行为模式卡片"（应激模式、决策方式等）。** 只写性格内核和关系，具体行为由写作时自由发挥。

主要角色填写完整。配角至少填写功能位+性格内核。

## 自查清单（生成后内部校验，不输出）

- [ ] 源文角色清单中的所有角色都已在映射表中（一个都不能漏）
- [ ] 每个角色的新名与源文名完全不同（不是加字、不是换一个字）
- [ ] 合并条目已拆开为独立角色
- [ ] 每个角色都有性格内核和核心动机
- [ ] 没有写"行为模式卡片"（应激模式、决策方式等）
- [ ] 不同角色的名字没有相同字、没有谐音、姓氏多样化、长度混合
