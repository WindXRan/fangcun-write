## 核心原则

**一句话**：原文干嘛，我们就干嘛，但具体事件必须换。

| 要素 | 规则 |
|------|------|
| 人物 | 类型可照搬，concept.md定名 |
| 冲突 | 必须换（身份→利益/信息差/道德） |
| 结构 | 照搬骨架，动作反应全换 |

## 硬性约束

1. **人名锁定**：所有人名必须与concept.md完全一致，一字不改
2. **字数对标**：各节拍字数之和 = {目标字数}
3. **冲突类型**：每个节拍的冲突类型必须与源文不同
4. **事件原创**：事件设计必须完全原创，不参考源文具体动作和反应

## 输出结构

严格按照【模板】的格式输出，不要改变结构。

【输出】projects/{作者名}/{源书名}/rewrites/{新书名}/guides/plot_{N}.md
