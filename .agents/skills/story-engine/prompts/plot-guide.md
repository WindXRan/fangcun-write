# 为《{新书名}》第{N}章生成创作指令

**你的任务：分析源文，提取这一章的「情绪功能」和「写法技巧」，然后输出创作指令。不要输出源文的节拍映射，不要逐段对应源文。**

## 角色（本章出场，必须用这些名字）

{角色约束}

## 世界观
{世界观}

## 源文分析任务

读源文后，分析以下内容：

### 1. 情绪功能
这一章在全书中承担什么角色？
- 是建立关系、激化冲突、情感爆发、还是节奏缓冲？
- 读者读完这章应该有什么感受？（如：心疼、好笑、紧张、温暖、愤怒）

### 2. 人物状态变化
- 开始时：各角色的关系/处境/心态是什么？
- 结束时：发生了什么变化？（必须有变化，不能平进平出）

### 3. 核心冲突
- 这一章的主要矛盾是什么？（谁和谁之间？因为什么？）
- 冲突怎么升级的？（不能一上来就最激烈，要有梯度）

### 4. 写法技巧（从源文学）
- 源文用了什么技巧让这场戏好看？（如：用旁观者反应烘托、用小动作代替直接说情绪、用对话节奏制造张力）
- 列出 2-3 个可以复用的技巧

### 5. 场景要求
- 需要几个场景？每个场景的环境/氛围是什么？
- 有哪些必须出现的元素？（如：某个道具、某个环境细节）

## 输出格式

```
# 第{N}章 创作指令

## 情绪功能
{一句话}

## 人物状态变化
- 开始：{状态}
- 结束：{状态}

## 核心冲突
{描述}

## 写法技巧
1. {技巧 + 说明}
2. {技巧 + 说明}

## 场景要求
- 场景1：{环境/氛围/必须出现的元素}
- 场景2：{环境/氛围/必须出现的元素}

## 字数目标
{目标字数}字（{目标字数_min}~{目标字数_max}）
```

## 铁律

1. **不输出源文的节拍映射** — 不要逐段对应源文，不要列出"源文第X段→新文第Y段"
2. **不输出源文的台词** — 源文对话仅供分析用，不得出现在输出中
3. **只输出创作指令** — 写手（write-chapter）看到你的输出后，自己设计场景和对话
4. **情绪功能必须具体** — 不要写"推进剧情"，要写"让读者第一次对男主心动"
