## 🔴 擦边是赛道核心卖点——节拍必须落地

**终极标准：让读者起生理反应——男读者勃起，女读者高潮。节拍设计达不到这个效果就是擦边没到位。**

**规则不是写在注释里就算了，必须写到节拍映射表的每一行中。**

### 硬性约束
1. **每章至少 2 个节拍**标为「擦边」冲突类型（列于冲突类型列）。可以是纯擦边，也可以是擦边+其他类型混合
2. 第 1 章必须有擦边节拍（不是从第 2 章才开始）
3. 节拍事件描述中必须明确出现擦边手法（凝视/贴贴/越界接触/暧昧公事），不能只写"产生暧昧感"
4. **浓度对标源文本章**：打开源文第{N}章，数源文有几个擦边节拍 → 新书不能少于源文

### 擦边节拍的设计优先级
1. **借公事之名**（检查/治疗/问话/护送/值班）行暧昧之实 — 优先选这个
2. **多对一包围**：≥2 女角色同时对男主有兴趣/互动
3. **肢体接触**：女角色主动贴/靠/摸/拉
4. **男性凝视**：女性身体部位描写，用被动/画面叠放手法
5. **直白口吻**：吐槽/粗口/自嘲

### 输出要求
在「本章高光」后新增一节：
```
## 🔴 擦边节拍统计
- 本章擦边节拍数：X个（需≥2，且≥源文本章擦边节拍数）
- 源文本章擦边节拍数：X个
- 手法：借公事之名 | 多对一包围 | 肢体接触 | 男性凝视 | 直白口吻（勾选本使用的手法，至少选2）
```
