# 角色起名提示词模板

> 来源：@旅行的巨人。用于为小说角色起名，避免AI味。

---

## 核心原则

名字不是追求"第一眼惊艳"，而是追求"真实、贴脸、有记忆点"。名字要像这个角色的父母真的会给他取出来的，而不是作者为了装高级临时拼出来的。

**判断名字有没有"AI味"的标准**：
1. 像不像真实户口本上会出现的名字？
2. 是不是过度追求文艺感？
3. 是否和角色家庭背景匹配？
4. 放进现代都市言情里，会不会显得悬浮？
5. 是否和其他角色撞风格？
6. 读者看到这个名字，能不能快速记住？
7. 如果人物被长辈、同事、爱人称呼，这个名字是否自然？

## 防翻车词（追加到任意prompt后面）

```text
请严格避开以下问题：

1. 不要全员名字都像诗句拆出来的。
2. 不要用过度热门的言情男主字，例如：寒、琛、霆、辰、砚、辞、屿、淮、洲。
3. 不要用过度热门的言情女主字，例如：晚、念、栀、眠、清、意、棠、予、初、浅。
4. 不要让女主名字一看就像小白花，也不要让男主名字一看就是霸总。
5. 不要出现"姓氏很常见，名字却悬浮得像艺名"的情况。
6. 不要只给好听名字，要给适合人物的名字。
7. 如果名字不适合，请直接说不适合，不要硬夸。
```

## 起名5维度（必须判断）

1. **出生年代**：70后/80后/90后/00后命名习惯不同
2. **家庭阶层**：县城家庭/小康家庭/书香家庭/豪门家庭
3. **父母审美**：文化水平、地域、审美倾向
4. **角色气质**：温柔/坚韧/冷淡/活泼/沉稳
5. **小说功能**：女主/男主/白月光/前任/情敌/反派

## 高频禁用字清单

**男主禁用**：寒、琛、霆、辰、砚、辞、屿、淮、洲、舟、泽、衍、瑾、瑜

**女主禁用**：晚、念、栀、眠、清、意、棠、予、初、浅、瑶、萱、璇、颖

**通用禁用**：婳、翊、珩、彧、旻、昶（生僻字）
