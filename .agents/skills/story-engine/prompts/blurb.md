## 🔴 总纲

**书名和简介是读者的第一印象。直接决定点击率。**
- 书名要「一眼看懂+有冲动点进去」
- 简介要「读前三句就想看正文」
- **必须对标番茄当前在榜的爆款语感，不能按传统出版或起点风格写**

## 书名生成（输出5个候选）

### 番茄爆款书名公式（2024-2025趋势）

| 公式 | 说明 | 番茄真实案例 |
|------|------|-------------|
| **身份+反差** | 一个身份+意想不到的附加 | 《我的弟子全是大帝之资》《让你当保安，你泡了女神总裁》 |
| **动词+悬念** | 动作为开头，制造"然后呢" | 《重生后，我成了渣男他叔》《开局一把刀，装备全靠砍》 |
| **人设+处境** | 人物标签+麻烦/关系 | 《七零娇气包被疯狗盯上了》《年代文作精她不想当炮灰》 |
| **关系+冲突** | 两个人之间的关系+张力 | 《继兄他不肯放手》《退婚后，他追妻火葬场了》 |
| **短促有力型** | 4-7字，干净利落 | 《北王霸刀》《晚风热恋》《偏偏心动》 |
| **场景+情绪** | 场景触发情绪联想 | 《在暴雨天失恋》《你说别离》《无人区玫瑰》 |

### 必须做到
- 书名和源文书名不能有相似感
- 书名的题材指向必须清晰（读者一眼知道这是什么类型）
- 有情绪/画面/冲突感，至少占一个
- 字数4-10字最佳，不超过12字
- **禁用以下AI起名模板**：
  - 《重生之XX》《穿越之XX》《我在XX当XX》
  - 《被XX后我XX了》《XX后，我XX了》
  - 《XX：XX》冒号结构（除非是身份+反差类如《七零娇气包：被疯狗盯上了》）
  - 《XX大佬XX》《XX王爷XX》《XX总裁XX》

### 必须避免
- 文艺过头（读者看不懂）
- 含蓄绕弯（"一曲离殇"、"花开半夏"之类）
- 和源文撞梗（人设/关系/设定雷同）
- 太长（超过12字）
- 两种风格混搭不伦不类

### 输出示例
1. 《继兄他不肯放手》
2. 《回城后，黑市大佬赖上我》
3. 《穿书七零：偏执大佬的掌心娇》

## 简介生成（输出5个候选，5种钩子类型各一版）

### 番茄爆款简介特征（死记）

**第一条铁律：前三句决定生死。**
开头三句必须有至少一个：冲突/悬念/情绪/反转。不能铺垫背景。

**第二条铁律：口语化到像作者在跟你讲故事。**
- 用短句。能断就断。
- 用生活化语言，不用书面语。
- "她心想"→删掉。"命运的齿轮开始转动"→杀了也不能用。
- "悬着的心终于死了"、"两眼一黑"、"好家伙"、"这波操作"→AI烂梗，禁用。

**第三条铁律：有画面感。**
- 不说"他很生气"，说"他捏碎了手里的杯子，碎片扎进掌心"
- 不说"她很害怕"，说"她后背贴紧了墙，手指抖得攥不住衣角"

**第四条铁律：结尾留钩子。**
最后一句必须是悬念/对话/反转，让读者忍不住点"下一章"。

**第五条铁律：代词密度控制。**
- 100字内"她"≤3次，"他"≤3次
- 连续出现的"她/他"要立刻打断
- 用名字/外号/称呼/省略主语来降低

### 代词密度控制技法

| 技法 | 反面例子 | 正确写法 |
|------|---------|---------|
| 用名字替换 | 他扣住她的手腕 | 魏长河扣住她的手腕 |
| 省略主语 | 她想要什么，他就不择手段 | 想要什么，魏长河就不择手段 |
| 称呼轮换 | 她……他……她……他 | 那女人、姓魏的、继兄、黑市商人交替用 |
| 合并动作 | 她利用完想踢开他，他扣住她的手腕 | 利用完想踢开人，手腕反被扣住 |

### 5种钩子类型（每版选一种，不准重复）

**版本A — 悬念型**
结构：穿书/重生背景 → 原著命运节点 → 觉醒 → 男主出场 → 章末金句
适用：穿书、重生、替嫁、有原著对照的设定
示例：
```
【穿书+年代+偏执男主】

一朝穿越，宋云绯成了假冒失忆总裁女友的女配。
原书中她下场凄惨——坐牢五年，出狱后惨死街头。
一睁眼，看着刚送外卖回来的男人，宋云绯只能将错就错。
一边跟他过贫穷日子，一边策划跑路。
终于等到跑路那天，男人却愤愤地将她抓了回来：
"骗完我就想跑？"
宋云绯震惊——这人什么时候偷偷恢复记忆的？
```

**版本B — 冲突型**
结构：标签句 → 两个选择/三角关系 → 核心矛盾 → 男主占有宣言
适用：言情拉扯、追妻火葬场、强取豪夺
示例：
```
【双洁+追妻火葬场+强取豪夺】

宋明锦有两个选择。
继兄严卫东老实本分，给不了她想要的生活。
黑市商人魏长河疯得让人害怕，但想要什么，姓魏的就不择手段捧到面前。
她选了魏长河。
利用完想踢开那天，手腕反被扣住贴在男人脸上：
"宋明锦，我这条命是你捡的，你就得负责到底。"
```

**版本C — 情感型**
结构：场景切入（冬天/雨夜/医院/黄昏） → 情感记忆 → 双向拉扯 → 暗示执念
适用：久别重逢、暗恋成真、破镜重圆
注意：场景描写不超过2句，迅速转入情绪/情节
示例：
```
【久别重逢+暗恋成真+双向奔赴】

再见到陆倦，是在深夜的急诊室。
他白大褂上沾着血，眉骨一道旧疤，看她的眼神像看陌生人。
七年前他一声不吭消失，她以为这辈子不会再见了。
可第二天，他出现在她办公室门口，递过来一盒胃药：
"你胃不好，记得吃。"
好像他们从没分开过。
```

**版本D — 反转型**
结构：背景 → 困境 → 男主递条件 → 发现门被焊死
适用：契约婚姻、各取所需、先婚后爱
示例：
```
【先婚后爱+蓄谋已久+甜宠】

陈薇需要钱，很多钱。
霍言需要个老婆，应付家里。
两人一拍即合，签了三年契约——不谈感情，到期离婚。
第一年，他准时转账，从不过夜。
第二年，他开始回家吃饭。
第三年，契约到期那天，霍言把合同撕了：
"不好意思，我没打算放你走。"
```

**版本E — 口语型**
结构：吐槽式开场 → 现实打脸 → 男主出场 → 对话收尾
适用：轻松喜剧、沙雕甜宠、日常流
示例：
```
【甜宠+双洁+日常向】

沈棠穿进了一本虐文，成了开局就死的炮灰女配。
她当场躺平：死就死吧，穿越太累了。
结果男主不按剧本走——不但不杀她，还每天端茶倒水嘘寒问暖。
沈棠慌了，揪住他衣领：
"你是不是拿错剧本了？你应该杀我啊！"
男人沉默片刻：
"你死了，我娶谁？"
```

### 通用规则（5个版本都遵守）

**必须做**：
- 开头亮标签：【标签1+标签2+标签3】，精准定位读者
- 用短句打节奏，长句铺信息。一句短的，一句长的，交替来
- 有画面感的具体描写（动作/表情/环境细节）
- 人名出现位置合理（第一句女主名+关系，后面用名字/外号轮换）
- 字数200-350字
- 结尾留悬念，引导读者点下一章
- **代词自检**：写完后数一遍"她""他"，超标必须改

**绝对禁止**：
- "她心想"、"她暗想"、"命运的齿轮"、"悬着的心终于死了"、"好家伙"、"两眼一黑"、"这波操作"、"一顿操作猛如虎"
- 铺垫超过2句还没进入情节
- 剧透结局
- 和源文简介结构一样
- 太长（超过400字）
- 太文艺（"他眼底暗涌着情愫"、"空气凝固了"）
