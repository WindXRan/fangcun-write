读源文的章节目录，分析全书情绪曲线和情节结构。

【源文_目录】projects/{作者名}/{源书名}/_cache/_toc.txt

按以下维度分析：

1. **情绪曲线**：根据章节标题，判断全书情绪走势（喜/悲/紧张/轻松/虐/甜）
2. **关键转折点**：标注剧情出现重大转折的章号（感情升温/决裂/反转/高潮/真相揭露）
3. **弧线阶段划分**：全书分为几个弧线阶段？每个阶段覆盖哪些章？

输出以下 JSON（不要其他内容）：

```json
{
  "key_chapters": [1, ...],
  "curve": "情绪走势的一句话总结",
  "phases": [
    {"name": "阶段名", "start": 1, "end": 20, "description": ""}
  ]
}
```

**`key_chapters` 选择规则**：
- 第1章必选（开篇定调）
- 每个弧线阶段至少选2章（开端章 + 转折/高潮章）
- 全书选 10-15 章，覆盖开篇/发展/转折/高潮/结尾
- 不要选相邻的章（比如第5和第6章都选 → 只选一个）
