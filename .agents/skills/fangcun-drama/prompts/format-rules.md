# 剧本格式规范（硬性约束）

本文件是剧本编写的**格式红线**，违反任何一条 = 不合格。agent 写完剧本后必须逐条自查。

---

## 一、△ 行规则

### 1.1 △ 行必须用全名，禁用代称

```
❌ △他抬手一看。
❌ △她愣住了。
❌ △他摸摸自己的脸。

✅ △张景辰抬手一看。
✅ △于兰愣住了。
✅ △张景辰抬手摸了摸脸颊。
```

### 1.2 △ 行必须有丰富的视觉画面感

△ 行是**镜头能拍到的东西**，不是干巴巴的动作描述。

```
❌ △他摸摸自己的脸。
✅ △张景辰抬手摸了摸脸颊——皮肤紧绷，胡茬扎手。

❌ △雪很大。
✅ △推开门，积雪呼啦涌进来，堆到膝盖深。整个胡同白茫茫一片。

❌ △于兰肚子疼。
✅ △于兰蜷在炕上，嘴唇咬出一排白印子，额头上一层细汗，手指死死抠着炕沿。

❌ △张景辰和孙久波进山。
✅ △大雪没到脚脖子。张景辰和孙久波在林子里深一脚浅一脚地走，眉毛上结了一层霜，哈气跟白烟似的。
```

### 1.3 △ 行禁用破折号（——）

破折号是偷懒的连接符。每个破折号都必须用句号/逗号/冒号替换。

```
❌ △他跌撞下炕，冲到镜子前——一张浓眉大眼的脸。
✅ △他跌撞下炕，冲到镜子前。镜子里一张浓眉大眼的脸。

❌ △张景辰过来看了一眼煤筐——见底了。
✅ △张景辰过来看了一眼煤筐。见底了，剩几块碎煤渣。

❌ △他猛地睁开眼——院子里有动静。
✅ △他猛地睁开眼。院子里有动静。

❌ △脚下一滑，凳子倒了——她整个人仰面摔在地上。
✅ △脚下一滑，凳子倒了，她整个人仰面摔在地上。
```

**唯一例外**：剧情梗概（### 剧情梗概）段落中可以使用破折号。

### 1.4 △ 行里的主观感知必须拆成 OS

△ 行用冒号引入人物感知 = 违规。必须拆成 △动作 + OS。

```
❌ △他摸摸自己的脸：年轻的，紧实的。
✅ △张景辰摸了摸自己的脸。
   张景辰OS：好紧实的皮肤…我重返年轻了？

❌ △抬手一看：手没皱纹，有劲儿。
✅ △张景辰抬手一看。
   张景辰OS：这手没皱纹，有劲儿…二十多岁的手！

❌ △张景辰跟出来看了一圈：窗户纸烂得一条一条的。
✅ △张景辰跟出来看了一圈。
   张景辰OS：窗户纸烂得一条一条的…上辈子她就是这么熬过来的。
```

**例外**：外部声音/画外音可以用冒号（"△远处传来断裂声：电线杆子倒了！"）。

## 二、对话行规则

### 2.1 对话行必须纯净：只有 人物名：台词

对话行**禁止**夹带任何动作、神态、语气描述。这些内容必须放在前面的 △ 行。

```
❌ 于兰（小声）：他爹…你醒了？
✅ △于兰声音压得低低的，靠着门框。
   于兰：他爹…你醒了？

❌ 张景辰（嗓子发干）：嗯。
✅ △张景辰嗓子发干，喉咙里挤出闷闷的一声。
   张景辰：嗯。

❌ 王全发（拍桌）：你出千！
✅ △王全发一掌拍在桌上。
   王全发：你出千！

❌ 于兰（攥住他衣袖不松手，眼泪啪嗒往下掉）：他们那么多人…
✅ △于兰一把攥住张景辰的衣袖不松手，眼泪啪嗒往下掉。
   于兰：他们那么多人…你会被打死的…

❌ 孙久波（画外）：二哥！
✅ △孙久波的声音从外头传来。
   孙久波（画外）：二哥！
```

**注意**：`（画外）` 是技术标记，标识声音来源，可以保留。但其他括号一律拆到 △ 行。

### 2.2 对话必须像真人说话

```
❌ 我看到的不是现在。是下一次进化路线。
✅ 为什么明明有了金手指，而我还是这么倒霉？

❌ 你这样做是不对的，你应该反思一下自己。
✅ 你脑子有病吧？
```

## 三、OS（内心独白）规则

### 3.1 OS 格式

```
张景辰OS：内心话内容
于兰OS：内心话内容
```

OS 前面**不加**任何括号、语气描述。

### 3.2 OS 必须短：每段约 10 字

一个动作配一段 OS，约 10 个字。情绪有变化就要加新动作 + 新 OS。

```
❌ △张景辰摸摸自己的脸。
   张景辰OS：好紧实的皮肤，我这是重返年轻了？老天爷真让我重来一回？这辈子一定要好好活！

✅ △张景辰摸了摸脸颊，皮肤紧实。
   张景辰OS：好紧实的皮肤…
   △张景辰低头看了看手，虎口有老茧。
   张景辰OS：我重返年轻了？
```

### 3.3 OS 和 VO 的区别

- **OS（内心独白）**：角色当下的内心想法。格式：`人物名OS：内容`
- **VO（画外音/旁白）**：配合闪回画面的旁白，或回忆性叙述。格式：`人物名VO：内容`

```
❌ （闪回画面中）于兰OS：他以前总是打我骂我…
✅ （闪回画面中）于兰VO：他以前总是打我骂我…
```

**使用场景**：
- OS = 角色此刻心里在想什么
- VO = 配合闪回/回忆的旁白叙述

## 四、场景规则

### 4.1 每集最多 4 个场景

场景太多观众跟不上。合并相关场景。

```
❌ 一集 6-10 个场景
✅ 一集 3-4 个场景
```

### 4.2 场景头格式

```
1-1 日/内 张景辰家
人物：张景辰、于兰

△场景描写…
```

## 五、角色塑造规则

### 5.1 角色转变必须有根据

角色的怀疑/恐惧/转变必须有闪回+VO解释，不能凭空产生。

```
❌ 于兰OS：他是不是又想要钱了？
（没有任何根据，于兰还没看到任何异常行为）

✅ △于兰愣住了——以前张景辰见她第一句就是骂。
   （画面闪回：年轻张景辰掀翻桌子指着于兰骂。）
   于兰VO：以前他见我第一句话不是骂就是打…
   （闪回结束。于兰站在门口，手指在门框上蹭了两下。）
```

### 5.2 省主语 = 偷懒

```
❌ 他站起来走了。（"他"是谁？）
✅ 张景辰站起来走了。
```

## 六、自查清单（写后必查）

- [ ] △ 行无代称（他/她/她们/我们）
- [ ] △ 行有视觉画面感（不是干巴巴的动作）
- [ ] △ 行无破折号（——）
- [ ] △ 行无冒号引入主观感知（拆成 OS）
- [ ] 对话行无括号动作神态（拆到 △）
- [ ] OS 每段约 10 字，情绪变化有新动作
- [ ] 闪回用 VO 不用 OS
- [ ] 每集 ≤4 个场景
- [ ] 角色转变有闪回+VO 根据
- [ ] 无省主语现象
