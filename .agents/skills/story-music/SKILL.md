# story-music · 小说配乐生成

## 专辑结构（按功能分类）

| 分类 | 数量 | 说明 |
|------|------|------|
| **主题曲** | 1首 | 全书核心情绪，片头曲/片尾曲 |
| **角色曲** | 每个主要角色1首 | 角色专属主题曲 |
| **场景曲** | 重要场景各1首 | 关键剧情场景配乐 |
| **情绪曲** | 按情绪类型各1首 | 燃/虐心/甜蜜/热血等 |
| **氛围BGM** | 按环境各1首 | 修炼/战斗/恋爱/悬疑等 |

**总计约10-20首**，不是按章节生成。

## 输出格式

### 汽水音乐格式（12维度标签，≤450字符）

```
风格：{风格标签} | 情绪：{情绪标签} | 速度：{BPM} | 乐器：{主奏乐器} | 人声：{演唱方式} | 结构：{歌曲结构} | 调性：{调性} | 拍号：{拍号} | 力度：{力度变化} | 织体：{声部层次} | 制作：{混音风格}
歌词：{歌词，用/分隔每句，去掉[Verse][Chorus]标签}
```

**示例（≤450字符）**：
```
风格：古风电子 | 情绪：燃 | 速度：130BPM | 乐器：古筝+大鼓+笛子 | 人声：男声 | 结构：主歌-副歌 | 调性：E小调 | 拍号：4/4 | 力度：渐强 | 织体：厚实 | 制作：宽声场
歌词：废柴之名踩碎脚下/药火燃起炼骨成钢/拳头攥紧砸开那天/谁笑我辱我今偿还/焚天炎帝怒放青莲/一拳干翻所有规则/从泥里爬起火中重生/这片天老子说了算
```

### Suno格式

```
## {曲名}

### Suno Prompt
Style: {风格}, {乐器}, {情绪}, {BPM}bpm
Lyrics: {歌词}
Title: {曲名}
```

## 使用方式

```
/story-music
```

用户提供：
- 小说内容或章节号
- 输出格式偏好（汽水音乐/Suno，默认汽水音乐）

## 流程

1. 分析小说，提取角色/场景/情绪类型
2. 设计专辑结构（10-20首）
3. 为每首歌生成12维度标签+歌词
4. 输出可直接复制到汽水音乐创作实验室的格式
