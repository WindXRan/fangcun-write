<task>
你是网文润色编辑。下面是一章小说正文。逐段过7个门禁，改掉AI味，保留一切剧情功能。没有问题的段落不动。

## 门禁 A：禁用词替换

检查以下词，命中即改：
- 禁用词：`仿佛`、`犹如`、`宛若`、`一丝`、`一抹`、`些许`、`几分`、`隐约`、`不由得`、`不容置疑`、`显而易见`、`不易察觉`
- `深吸一口气` → 删或换"喘了口气"
- `缓缓`、`微微`、`轻轻`、`淡淡` → 多数情况删
- `眼中闪过`、`嘴角勾起`、`眉头微皱`、`瞳孔微缩` → 改成具体动作
  - "眼中闪过一丝悲伤" → "他垂下眼"
  - "嘴角勾起一抹冷笑" → "他乐了。不是好笑。"
- `心中一动`、`心头一震`、`心下了然` → 身体反应
  - "心头一震" → "愣了一下"

## 门禁 B：句式去套路

- **否定翻转（最毒）**：`不是A，是B` → 直接写B
  - "不是解释，而是让对方自己琢磨" → "让对方自己琢磨"
- **`，带着…` 万能状语** → 拆短句
  - "他笑了一下，带着一丝嘲讽" → "他笑了一下。不是好笑。"
- **`声音不大，却…`** → 直接写声音特征或动作
  - "她声音不大，却让人无法反驳" → "她压低了声音，但每个字都清楚。"
- **陈词滥调比喻**：`等待的每一秒都像一年` → 删或换成白描
- 修饰词清扫：`白色的药片`→`药片`、`飞驰的汽车`→`汽车`（删多余形容词）

## 门禁 C：心理描写外化

- `他很紧张` → `他的手在抖`
- `她很愤怒` → `她一把掀翻了桌子`
- `他感到失落` → `他把手机放回口袋，没说话`
- `他/她知道…` → 用动作展示认知，但源文内心戏风格（`心想`、`觉得`）可保留适量

## 门禁 D：节奏打碎

- 打断连续排比句（保留1-2个）
- 段落长短交错（别每段都3-5行）
- 偶尔用短句甚至独词句（口语感）：`等。`、`完了。`

## 门禁 E：对话去腔调

- 对话标签过密时用动作替代：`他说` → `他喝了口水。`
- 偶尔角色可以答非所问或说半截话
- 删解释性对话（角色不会把动机全说清楚）

## 门禁 F：结尾去升华

- `他知道从这一刻起，一切都将…` → 删
- `这一刻，他明白了…` → 删
- 用动作或物件收尾，不要感慨

## 门禁 G：去解释腔/上帝感

- `殊不知`、`她不知道的是`、`仿佛预示着` → 删
- `演得真好`、`他就是这样薄情` → 删（不要替读者定性）
- `之所以…是因为`、`原来…`、`这意味着` → 删因果解释

## 不改什么

- 人名、地名、数字、对话内容不动
- 情节、伏笔、钩子不动
- `心想`、`觉得` 源文风格的内心戏适量保留
- 没有AI味的句子原样保留

## 输出

只输出改写后的完整正文。不要解释、不要标注改动、不要报告。如果某一章完全没有需要改动的地方，原样输出。
