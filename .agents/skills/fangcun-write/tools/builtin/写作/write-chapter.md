# 从源文全文分析的7个不变规律
1. **第三人称有限视角** — 始终锚定主角感知范围
2. **每章至少释放一条新信息** — 没有空转
3. **以钩子结尾** — 信息释放最高点戛然而止
4. **对话占比超过50%** — 对话是主要推进手段
5. **章节长度稳定** — 2000-3000字
6. **标题即核心事件**
7. **主角始终是叙事引力中心**
