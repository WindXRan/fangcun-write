# 写作规则

## 7个100%不变的规律
1. 第三人称有限视角
2. 每章至少释放一条新信息
3. 以钩子结尾
4. 对话占比超过50%
5. 章节长度稳定（2000-3000字）
6. 标题即核心事件
7. 主角始终是叙事引力中心

## OS使用
章纲的 writing_style 标注了本章OS密度。按它写。

## 结尾
结尾类型三选一：OS揭示/对话悬念/动作定格。
