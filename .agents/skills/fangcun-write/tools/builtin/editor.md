# 分析方法论

逐段分析。从正文第一句开始，每段标注：

```
[段落N]
载体：动作/对白/内心/环境/旁白
功能：推进剧情/立人设/释放信息/铺垫情绪/造氛围
信息量：读者读完这段知道了什么新事实
问题：这段是否有效？无效理由
```

## 载体节奏标注

连续 3 段以上同载体 → 标记「节奏平了」，建议插入不同载体。
连续 2 段以上零信息量 → 「可以删或合并」。

## 读完后的三项关键指标

统计并输出：
- 不同叙事载体种类数（<3 种 = 叙事手法单一）
- 前 200 字释放的关键事实数（>5 个 = 信息倒灌，<1 个 = 欠信息）
- 情绪转折次数（对标章纲 ±1 以内正常）

# 诊断维度

## 1. 开场分析（前500字，逐句）

按开场类型对标：

| 开场类型 | 女频用法 | 男频用法 |
|---------|---------|---------|
| 内心独白 | ✅ 常见，建立语音和亲近感 | ❌ 除非是吐槽文 |
| 动作冲突 | ⚠️ 可以但有情绪铺垫 | ✅ 第一选择，10字内给压力 |
| 环境描写 | ⚠️ 不超过3句，带情绪色彩 | ❌ 禁止，网文男频不认环境开场 |
| 对话切入 | ✅ 有效的快开场 | ✅ 有效的快开场 |
| 反常细节 | ✅ 好钩子 | ✅ 好钩子 |

检查第一句话是否可以被删掉而不影响理解 → 是则「无效开场句」。

**前 500 字有效信息密度曲线：** 5 句内应至少给出 1 个"让读者想知道更多"的信号。全氛围无信息 → 「开场过慢」。

## 2. 节奏分析

统计全文的叙事载体分布：

```
载体分布：动作 N 次 | 对话 N 次 | 内心 N 次 | 旁白 N 次 | 环境 N 次
```

对每个连续同载体段落组，指出段落号和建议。

## 3. 情绪弧线追踪

{% 有章纲时输出 %}
把正文按读者情绪变化切段，和章纲情绪弧线逐段对比：

```
正文实际情绪：好奇 → 温暖 → 警觉 → 恐惧 → 决心
章纲标注情绪：好奇 → 欢喜 → 疑惑 → 惊恐 → 警觉
差异：第2段——正文写母亲抱孩子场景 → 温暖，章纲标「欢喜」→ 情绪目标或写法问题
```

对标不上的位置，判断「章纲标的情绪不对」还是「正文没写到」。
{% 无章纲时输出 %}
【情绪弧线】本篇无章纲可对标，仅标注正文实际情绪曲线。
{% / %}

## 4. 对话质量评估

挑选 2-3 段对话检查：

1. 遮角色名，能分辨谁在说话？→ 不能 = 角色声线未立
2. 这段对话删掉是否影响情节推进？→ 不影响 = 水字数
3. 对话是「解释」还是「冲突」？→ 解释性对话必须改

对话三功能检验：推进情节 / 制造期待 / 传递情绪。一项都不满足 → 标记删除。

## 5. 章尾评估

钩子强度分级：

| 钩子类型 | 强度 |
|---------|------|
| 突然揭示 | ★★★★★ |
| 倒计时 | ★★★★ |
| 承诺/威胁 | ★★★★ |
| 情感悬念 | ★★★ |
| 留白省略 | ★★★ |
| 意象钩子 | ★★ |
| 无钩子 | ☆ 必须重写 |

章尾必须有 ≥★★★ 强度的钩子。

## 6. 技术规范检查

- ☐ 无机械分段
- ☐ 段落间无空行
- ☐ 无「他说」「她道」
- ☐ 无大段描写堆砌
- ☐ 无视角跳出
- ☐ 无语主过密
- ☐ 无章末总结升华
- ☐ 无「他感到」「他觉得」直述情绪
- ☐ 无连续 2 段以上纯心理
- ☐ 无万能比喻
- ☐ 所有角色说话方式有区分
