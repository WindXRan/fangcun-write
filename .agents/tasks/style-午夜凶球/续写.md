# 续写Task：午夜凶球风格

## 任务
续写{书名}的{章节号}，约{字数}字。

## 输入
- 书名：{书名}
- 章节号：{章节号}
- 字数：{字数}
- 前文内容：{前文内容或章节号}

## 输出结构

### 1. 章节大纲
```markdown
## 第X章 章节名

### 场景1：场景名
- 地点：
- 角色：
- 事件：
- 冲突：
- 结果：

### 场景2：场景名
...

### 本章钩子
让读者想看下一章的悬念是什么
```

### 2. 正文内容
```markdown
# 第X章 章节名

[正文内容，约{字数}字]
```

### 3. 元数据
```markdown
---
章节号：X
字数：XXX
场景数：X
主要角色：角色1, 角色2
情节类型：日常/冲突/感情线
---
```

## 执行步骤

### Step 1: 读取风格理解
读取以下文件了解风格特征：
- `.agents/skills/style-午夜凶球/SKILL.md` - 风格特征、写作原则、绝对禁区、反AI检测要点

### Step 2: 检索知识库找参考片段
在 `.agents/skills/style-午夜凶球/knowledge_base/chapters/` 中找类似场景：
1. 找包含"{主要角色}"的场景
2. 找包含"{情节类型}"的场景
3. 找包含"{场景地点}"的场景

读取3-5个参考片段，学习写法。

### Step 3: 分析参考片段
从参考片段中分析：
1. 句子长度变化：有很短的句子，也有很长的句子
2. "毛边"：打断、未说完的话、突然转折
3. 真实感：具体细节、情感变化、因果关系
4. "意外"：突然的转折、意外的反应
5. 细节：小动作、小表情、小细节

### Step 4: 写章节大纲
基于参考片段，先写章节大纲：
1. 本章要推进什么剧情
2. 有哪些场景
3. 每个场景的冲突点是什么
4. 本章的钩子是什么（让读者想看下一章）

### Step 5: 基于大纲写正文
基于大纲，写正文内容：
- 对话比例60-70%
- 短句为主，节奏明快
- 东北方言自然融入
- 内心独白碎片化
- 点到即止，煽情不超过三句话
- 有"毛边"，不要太顺
- 细节多，小动作、小表情
- 句子长度变化大
- 有真实感，像是"经历"过的
- 有"意外"，有突然的转折

### Step 6: 验证输出
写完后验证是否符合风格：
- 对话比例是否60-70%
- 是否有东北方言
- 内心独白是否碎片化
- 是否有点到即止的煽情
- 是否有"毛边"（不要太顺）
- 是否有细节（小动作、小表情）
- 是否避免了绝对禁区
- 句子长度是否变化大
- 是否有真实感
- 是否有"意外"
