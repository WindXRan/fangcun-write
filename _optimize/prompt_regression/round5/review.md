## 根因

**根本问题：prompt指令缺少"克制写作"原则，导致模型倾向信息倾泻和过度解释。**

具体分解：

1. **结构化缺数据**: 原文的优秀特征(渐进信息释放、留白、感官细节优先于解释)没有被抽象成可执行的写作规则。prompt只告诉模型"写个转世+偷听+系统+危机"的章节，没有约束"信息释放速率""留白比例""解释 vs 展示"这类写作质量参数。

2. **prompt指令不够**: prompt没有明确指出：
   - 不允许使用现代/网络用语(沙丁鱼罐头、穷光蛋、我特么)
   - 不允许在叙述中替读者做情感解读("浑身上下写满了'我是个正经人'")
   - 不允许角色跳过正常心理反应阶段(母亲必须经历怀疑否认期)
   - 章节结尾必须是行动/悬念/未完成态，不能是情感总结
   - 信息释放要遵循"一章节只揭开一个谜"原则

3. **执行偏差**: 即便prompt有够用的约束，模型在执行时选择了"效率优先"的路径——用最短的篇幅把最多的设定塞进去。这是大模型生成长文本的典型病：它知道结局，所以失去了"慢慢走向结局"的耐心。

## 修复建议

### 必须改的文件

**文件**: `projects\全家偷听心声\_optimize\prompt_regression\round5\output_ch1.xml`

需要重写以下内容：

1. **开场第1-3行**: 删除"不对，不是痛，是被挤的"自我纠正。直接写痛点开场。参考原文写法：用感官细节(冰凉、清脆、啼哭)定调，不要用现代比喻。

2. **父亲外貌描写第56-58行**: 删除作者旁白"浑身上下写满了'我是个正经人'"。改为具体细节(官服、露水、动作笨拙如拆炸弹——保留"拆炸弹"但这是个现代比喻，仍需替换)。

3. **系统余额第106-116行**: 删掉"穷光蛋""我特么连张脸都换不起"等吐槽。余额为0本身就有戏剧张力，不需要主角替读者吐槽。冷处理，让读者自己感受。

4. **母亲反应第120-144行**: 重写母亲从"听到声音"到"行动"之间的心理过程。必须加入：
   - 怀疑/否认阶段("产后虚弱，幻听了")
   - 试探阶段(不是直接问"你能听到吗"，而是用动作测试)
   - 确认阶段(手在发抖，但不能在下人面前失态)
   - 隐忍到独处时才行动

5. **王妈妈剧透第139-142行**: 心声内容从"巫蛊案的导火索就在今晚——王妈妈！皇长子安插的眼线！巫蛊娃娃！"改为更模糊的暗示。让读者和母亲一样只知道"王妈妈有问题"，不知道"具体什么问题"。悬念靠信息差维持。

6. **结尾第228-229行**: 删除"我沈妙妙好不容易有娘了——谁也别想让我没家"情感总结。改为场景/行动收束——比如脚步声靠近书房，母亲抱着孩子隐入黑暗，或者母亲的手在摇篮边握紧了发簪。必须是一个"未完待续"的姿势，不能是句号。

7. **全文范围**: 搜索并删除所有现代/网络用语(老阎王你又坑我、穷光蛋、我特么、清福套餐、灭门套餐等)。保留口语感但不能跳出古风语境。

### prompt改进建议（下一轮）

在写作prompt中加入以下约束：

```
## 写作纪律
- 禁止使用现代/网络用语
- 禁止在叙述中替读者做情感解读（不要写"浑身上下写满了X"、"眼神里充满了Y"）
- 禁止在100行内释放超过2条核心信息
- 禁止角色跳过正常心理反应阶段
- 章节结尾必须是行动或场景，禁止以情感总结收束
- 信息释放遵循"一章节只揭开一个谜"原则
- 感官细节优先于解释说明：用"做什么"展示人物，不要用"是什么样的人"定义人物
```
