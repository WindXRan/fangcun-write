# Prompt 修复补丁 — Round 2

## 根因定位

审稿报告指出 **prompt指令不够（prompt缺少关键约束和优化指令）**，具体表现为三个层次：

| 层次 | 审稿差距 | 根因 |
|------|---------|------|
| G1 (P0) | 输出199/200行与原文逐字一致 — 疑似记忆复现而非推理生成 | prompt无"不得逐字复制"约束，模型默认以"最大相似度"作为奖励信号 |
| G2 (P0) | 原文缺陷完整继承（开场平淡、infodump生硬、情绪转变过急） | prompt只有"复刻"指令，没有"分析原文缺陷并优化"指令 |
| G3 (P1) | 唯一的自主编辑（删"冗余句"）反而降低了叙事微节奏 | prompt缺少"保留有功能的冗余/过渡拍"的节奏意识约束 |

**关键判定**：这不是结构化数据的问题（模式卡已在R1补全了 protagonist_voice 等字段）。模型有足够数据做判断，但 prompt 没有告诉它"什么时候不要复制"和"怎样做得更好"。

## 补丁 A — 不得逐字复制原文 🔴

**目标文件**: `fanxie-chapter.md`
**对应差距**: G1（输出与原文几乎完全一致）
**位置**: 第13行"# 角色"段末尾，"# 工作流程"之前，新增一整个约束块

**插入内容**:

```
### 🔴 禁止逐字复制（第一约束）

这是仿写任务**最高优先级约束**，违反则输出作废：

- ❌ **禁止**连续5个字以上与原文完全相同（标点符号除外）
- ❌ **禁止**复制原文的句式结构+同一组关键词（如原文"从混沌中醒来"→仿写不能也写"从混沌中醒来"）
- ❌ **禁止**保持原文的措辞选择——即使情绪和节奏一致，也必须换一套词汇

✅ **允许**：保留情绪弧线、信息释放节奏、调性风格、段落节奏模式（句均字数、对话占比等量化指标）
✅ **必须**：换全部措辞。源文用"混沌"→仿写可以用"黑暗/虚无/迷雾"；源文用"浮上来"→仿写用"散开/褪去/抽离"

**自检方法**：把仿写和原文并列运行文本对比工具（diff），如果相似度 > 60%，说明违反了本约束。必须重写。
```

## 补丁 B — 分析原文缺陷并修复优化 🔴

**目标文件**: `fanxie-chapter.md`
**对应差距**: G2（原文缺陷被完整继承）+ 修复建议F1~F4
**位置**: 补丁A之后，"# 工作流程"之前

**插入内容**:

```
### 🔴 仿写即优化：每个段落问自己"原文这里有没有问题？"

仿写不是"越像原文越好"。如果原文有缺陷，仿写应当**在保持风格一致的前提下修复**。

**逐段扫描以下问题，有则必改：**

| 问题类型 | 扫描点 | 修复原则 |
|---------|-------|---------|
| **开场冲击力不足** | 第1~5行是否有具象画面/金句/独特感知？ | 必须是"读者一眼能看到/听到/感受到"的具体东西，替换抽象描写（"意识像从深水里浮上来"→改为"鼓膜里还残留着阎王殿的回音，眼皮沉得撑不开"） |
| **叙事性infodump** | 是否有连续5行以上的纯背景介绍插入进行时叙事？ | 将背景碎片化：拆成2~3段穿插在后续的感知/对话/动作中释放。禁止出现"记忆轰地涌上来"后接一整段简历式叙事。 |
| **情绪转折过急** | 角色从A情绪到B情绪是否缺少过渡拍？ | 在转折点前加一个"犹豫/怀疑/自问"步骤。例如：母亲从震惊→相信之间，必须有至少1次内心挣扎或客观观察。 |
| **情绪表达太直白** | 角色情绪是否用"很XX""太XX了""她感到XXX"直接说出？ | 用身体反应/微动作/环境映射代替直述。例："她鼻子忽然有点酸"→"她眨了眨眼，睫毛湿了"（不是本质修改，是措辞优化） |
| **冗余分解** | 是否有两个句子功能重叠？ | 优先保留功能差异明显的句子。当两句看似相似时，检查是否构成"收→散"呼吸节奏——如果是，两句都保留。 |

**自检方法**：逐段读，每读完一段问"原文这段有没有上述5个问题？有则修复，无则保持。" 修复后不影响调性、节奏、冲突层级的对齐。
```

```
- [ ] **微节奏检查**：是否有两行看似"冗余"但构成"收→散"呼吸节奏的句子？如果有——不要合并。保留递进（先关注外部→再内心游离），这是读者沉浸感的"呼吸空间"。
```
