# 章纲 第31-40章

## 第31章 套了个祖宗回来

### 源文对应
- 源文第31章：套了个祖宗回来
- 核心事件类型：夜市日常 + 情感升温 + 宠物收编
- 情绪弧线：轻松→兴奋→惊喜→无奈→羞涩
- 钩子类型：浴室玻璃暴露钩子

### 新书情节（内容全部原创）
- **核心事件**：省城展会多待一天，逛夜市套金鱼。顾西洲百发百中套中金鱼缸。
- **利益冲突**：无直接冲突，但酒店浴室磨砂玻璃暴露——外面能看清里面轮廓 → 暧昧尴尬
- **情绪弧线**：无聊（独自逛）→ 兴奋（套圈）→ 惊喜（顾西洲帮她套中）→ 羞涩（发现浴室玻璃）
- **动作链**：沈念念逛夜市→套圈全空→顾西洲一次套中→回酒店发现玻璃问题→快速洗澡→顾西洲坐床边转鱼缸
- **钩子设计**：沈念念洗完澡出来，看到顾西洲坐在床边，手里转着那只金鱼缸，不知道在想什么
- **爽点设计**：顾西洲百发百中的反差技能；对路边摊不屑一顾却乖乖陪玩

### 关键细节（确保和源文不同）
- 场景：省城夜市（非源文编手绳夜市）
- 宠物：金鱼（非源文的狗），金鱼缸成为本章核心道具
- 核心道具：金鱼缸+酒店磨砂玻璃
- 差异化：源文是编手绳+套狗，新书是套金鱼；源文狗后续还要处理，新书金鱼好携带

## 第32章 要死啊

### 源文对应
- 源文第32章：要死啊！
- 核心事件类型：浴室暧昧 + 心理拉扯 + 初梦
- 情绪弧线：尴尬→心猿意马→慌张→羞耻→无措
- 钩子类型：亲密关系钩子（第一场春梦）

### 新书情节
- **核心事件**：沈念念发现玻璃能看到人影，偷看顾西洲洗澡→做春梦→醒来在他怀里
- **利益冲突**：自我克制 vs 身体欲望的第一次正面冲突
- **情绪弧线**：好奇（试玻璃）→ 心跳加速（偷看）→ 羞耻（梦）→ 慌张（被问）
- **动作链**：沈念念试玻璃→看到人影→偷看→睡觉→做春梦→被吵醒→发现在他怀里→被追问梦到什么
- **钩子设计**：顾西洲问「你昨晚梦到什么了？一直在说胡话」「叫得很急」
- **爽点设计**：偷看被察觉的尴尬感+春梦惊醒式反差

### 关键细节
- 场景：酒店房间
- 差异化：源文是宋偷看+楚拿吹风机逼她吹头发；新书是顾西洲直接过来同床+第二天追问
- 关键道具：磨砂玻璃（延续31章的道具）

## 第33章 走了狗屎运

### 源文对应
- 源文第33章：买了房子就结婚
- 核心事件类型：展会工作 + 商业能力展示
- 情绪弧线：轻松→被调侃→投入→小得意→惊喜
- 钩子类型：买房话题钩子

### 新书情节
- **核心事件**：展会上沈念念帮顾西洲改造展区，用审美吸引德国客户下单，提成三十万
- **利益冲突**：沈念念想攒钱跑路 vs 顾西洲想买房安家（利益方向不同）
- **情绪弧线**：轻松（早餐）→ 投入（改造展区）→ 得意（效果被夸）→ 惊喜（三十万提成）→ 僵硬（买房话题）
- **动作链**：沈念念早起→去展馆→看不下去帮改造展区→德国客户满意下单→孙总想拉顾西洲入股→顾西洲说买房子
- **钩子设计**：顾西洲说「这下有钱了。买套房子吧。」沈念念笑容僵住
- **爽点设计**：沈念念的审美天赋被印证；德国客户当场下单的爽感；孙总态度大变

### 关键细节
- 场景：省城展馆
- 差异化：源文是做房地产的搭配审美，新书是沈念念上辈子做室内设计的布局能力
- 关键转折：顾西洲第一次说「买房子」——他认真了

## 第34章 顾总？

### 源文对应
- 源文第34章：楚总？您怎么会在这？
- 核心事件类型：身份危机 + 高潮冲突
- 情绪弧线：紧张→愤怒→震惊→恐惧→机智化解
- 钩子类型：身份暴露钩子 + 间谍钩子

### 新书情节
- **核心事件**：展会上遇到江州来的李总，盯着顾西洲试探性喊「顾总？」
- **利益冲突**：沈念念的谎言面临被揭穿 vs 必须保持冷静
- **情绪弧线**：紧张（对手厂抢客户）→ 震惊（李总认人）→ 恐惧（完了完了）→ 配合演戏（机智化解）
- **动作链**：对手厂抢单→孙总发火→李总出现盯着顾西洲→喊「顾总？」→旁边爱看小说的女生脑补「富二代体验生活」救场→李总侄女加沈念念微信
- **钩子设计**：李总离开时意味深长地看沈念念一眼。她微信收到好友申请：「你好，我是李总的侄女，想跟你请教下买房的事。」
- **爽点设计**：顾西洲用德语怼人；李总侄女的小说脑补解围（meta自嘲）

### 关键细节
- 场景：省城展馆VIP室
- 差异化：源文是华悦集团周明远，新书是江州来的李总；源文被认出后配合演戏是宋云绯说「亲爱的你什么时候成楚总了」，新书是旁边人帮忙圆场
- 新角色加入：周敏（李总侄女）——以后会成为接近沈念念的间谍

## 第35章 买房子

### 源文对应
- 源文第35章：去买套房子怎么样？
- 核心事件类型：感情压力 + 未来规划冲突
- 情绪弧线：放松→高兴→僵硬→心虚
- 钩子类型：买房冲突钩子 + 身份危机持续

### 新书情节
- **核心事件**：顾西洲认真研究安城房价，沈念念慌了——买房=关系绑定=跑路更难
- **利益冲突**：沈念念想轻装上阵方便跑路 vs 顾西洲想扎根安家
- **情绪弧线**：放松（订单爆完的疲惫）→ 高兴（提成）→ 僵硬（买房）→ 心虚（搪塞）
- **动作链**：展会结束→吃饭时顾西洲提买房→沈念念反应过度→她找借口搪塞→内心纠结「拿他的钱跑路等于嫌命长」
- **钩子设计**：「你不想跟我一起买房？」顾西洲问。沈念念张了张嘴，说不出「不想」两个字。
- **爽点设计**：顾西洲说「才一万」的淡定对比沈念念的崩溃

### 关键细节
- 场景：省城餐馆
- 差异化：源文是楚说「买套房子」被宋误解为求婚，新书是顾西洲认真研究房价+沈念念怕关系绑定

## 第36章 初吻

### 源文对应
- 源文第36章：我感觉，你并不是很想和我在一起
- 核心事件类型：关系本质对话 + 第一次接吻
- 情绪弧线：僵持→压迫→激烈→沉沦→恐惧退缩
- 钩子类型：关系定义钩子 + 身体关系钩子 + 孙磊消息钩子

### 新书情节
- **核心事件**：顾西洲直接问「你到底想怎么样」，沈念念被逼到墙角，他吻了她。她情动时闪过原主惨死结局，推开他。
- **利益冲突**：沈念念真实感情 vs 怕死的恐惧
- **情绪弧线**：僵持（买房分歧）→ 压迫（顾西洲质问）→ 激烈（初吻）→ 沦陷→ 恐惧（闪回原主结局）→ 推开
- **动作链**：回酒店→浴室玻璃话题被点破→顾西洲关灯洗澡（承认看到）→质问→吵架→他把她抵在墙上吻→她回应→闪回原主惨死→推开→手机震动（孙磊消息）
- **钩子设计**：手机震动——吴凯发来消息：「姐，有个叫孙磊的医生在打听你们的事。」
- **爽点设计**：顾西洲从隐忍到主动出击的爆发反差；初吻后推开的情感撕裂

### 关键细节
- 场景：酒店房间
- 差异化：源文初吻在客厅，新书在酒店房间（延续31-32章酒店场景）
- 关键道具：手机（孙磊的消息打断）

## 第37章 孙磊

### 源文对应
- 源文第37章：确认是真爱无疑了
- 核心事件类型：关系停顿 + 新角色引入
- 情绪弧线：混乱→内疚→温暖→轻松
- 钩子类型：孙磊登场钩子 + 身份危机升级（顾家已在查）

### 新书情节
- **核心事件**：第二天关系尴尬，但顾西洲默默照顾。吴凯介绍孙磊——原主的朋友，当初帮忙处理过「救人」的事。孙磊说顾家已经查到他了。
- **利益冲突**：沈念念想保护顾西洲 vs 怕真相暴露
- **情绪弧线**：混乱（被吻后）→ 内疚（推开他）→ 温暖（顾西洲带饭）→ 紧张（孙磊消息）
- **动作链**：尴尬的早餐→顾西洲先回厂里→沈念念见孙磊→孙磊警告顾家查到→沈念念说「出事就推到我身上」
- **钩子设计**：沈念念说：「如果出事，就说是我一个人的主意。」孙磊看了她一眼：「你认真的？」
- **爽点设计**：顾西洲虽然被拒绝但不闹不冷战，默默照顾

### 关键细节
- 场景：安城小餐馆
- 新角色：孙磊——原主的医生朋友，知道「救人」真相
- 差异化：源文有宠物（狗）的起名梗（套套→圈圈），新书用金鱼替代（延续31章的金鱼道具）

## 第38章 周敏的接近

### 源文对应
- 源文第38章：你这是何必呢
- 核心事件类型：外部压力升级 + 职场伏笔
- 情绪弧线：疑惑→紧张→恐惧→小心→温暖
- 钩子类型：顾家追查钩子 + 间谍钩子 + 客户被挖钩子

### 新书情节
- **核心事件**：周敏约沈念念看房套近乎。孙磊的追查压力让她焦躁。顾西洲主动伸手搂她。小霜发来黎姗挖客户的消息。
- **利益冲突**：沈念念要应付周敏的接近+孙磊的压力+黎姗的挖角
- **情绪弧线**：疑惑（周敏加微信）→ 紧张（孙磊来电）→ 焦躁（压力太大）→ 温暖（顾西洲伸手搂她）→ 愤怒（客户被挖）
- **动作链**：周敏约看房→套近乎→孙磊来电说顾家在查→沈念念在顾西洲面前撒谎→顾西洲伸手搂她问「昨晚的事害怕？」→小霜发来黎姗挖客户照片
- **钩子设计**：小霜发来一张照片——黎姗和赵太太坐在一起喝茶。
- **爽点设计**：周敏拙劣的借口「抽奖活动抽中你」——明目张胆地讨好沈念念

### 关键细节
- 场景：安城房产中介门店+咖啡馆
- 差异化：源文是周敏直接说买房，新书是周敏用「抽奖」为借口接近

## 第39章 客户被撬

### 源文对应
- 源文第39章：我天天得罪他
- 核心事件类型：职场日常 + 关系修复 + 冲突爆发
- 情绪弧线：迷糊→温暖→尴尬→放松→愤怒
- 钩子类型：客户被撬钩子 + 孙总套路

### 新书情节
- **核心事件**：回到安城，沈念念发现赵先生被黎姗撬走。顾西洲在厂里地位上升，老板不敢对他大声说话。
- **利益冲突**：沈念念的业绩被抢 vs 必须冷静应对
- **情绪弧线**：迷糊（醒来顾西洲不在）→ 温暖（他安排好金鱼）→ 尴尬（想起昨晚）→ 愤怒（客户被撬）
- **动作链**：沈念念醒来发现顾西洲在喂金鱼→回安城→上班发现黎姗背着她接触赵先生→顾西洲来中介接她→她看到黎姗和赵太太喝茶的照片
- **钩子设计**：沈念念盯着手机上黎姗和赵太太喝茶的照片，攥紧了手机。
- **爽点设计**：顾西洲在厂里底气足——做三个人的活拿一个人的工资，孙总都不敢对他大声

### 关键细节
- 场景：安城
- 差异化：源文有训狗情节，新书没有宠物只有金鱼；源文是返程+孙总套路，新书是回安城后直接触发职场冲突

## 第40章 反击

### 源文对应
- 源文第40章：竟然还会上这种当
- 核心事件类型：职场反击 + 客户争夺战
- 情绪弧线：愤怒→冷静→策略布署→赶去对峙
- 钩子类型：职场冲突爆发 + 客户抢回悬念

### 新书情节
- **核心事件**：沈念念冷静布局，找同事周姐打配合，用赵先生真正在意的学区房信息差截胡黎姗
- **利益冲突**：沈念念 vs 黎姗（职场公平竞争被破坏）
- **情绪弧线**：愤怒→冷静下来→策略布署→对峙→胜利
- **动作链**：沈念念冷静→分析赵太太在意学区→找周姐帮忙（算共同业绩）→在签约前一小时拦下→三方对峙→走出VIP室看到顾西洲靠在墙边等她
- **钩子设计**：沈念念走出VIP室，看到顾西洲靠在墙边等她。他不知道什么时候来的——但他在。
- **爽点设计**：沈念念不是自己冲过去吵，而是用策略（学区房信息差）智斗黎姗

### 关键细节
- 场景：安城房产中介门店VIP室
- 差异化：源文是赵太太被尾盘稀缺性话术忽悠，新书用学区房信息差作为截胡武器
- 核心道具：学区房资料（沈念念提前准备的功课）
- 关系变化：顾西洲第一次在沈念念的职场露面接她——关系半公开化
