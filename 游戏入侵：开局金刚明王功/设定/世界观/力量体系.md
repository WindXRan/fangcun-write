# 力量体系

## 等级系统

### 玩家等级
- **范围**：1-100级
- **升级方式**：获取经验值（挖矿、杀怪、完成任务）
- **升级奖励**：每级获得1点自由属性点
- **转职门槛**：10级

### 等级与实力对应
| 等级 | 实力阶段 | 说明 |
|------|----------|------|
| 1-10 | 新手期 | 矿工、流民 |
| 11-30 | 成长期 | 武者、普通弟子 |
| 31-50 | 崛起期 | 精英弟子、管事 |
| 51-70 | 称霸期 | 长老、城主 |
| 71-100 | 传奇期 | 宗主、国师 |

## 属性系统

### 基础属性
| 属性 | 作用 | 获取方式 |
|------|------|----------|
| 力量 | 影响攻击力、挖矿速度 | 升级加点、装备、武学 |
| 敏捷 | 影响速度、闪避 | 升级加点、装备、武学 |
| 精神 | 影响修炼速度、武学门槛 | 升级加点、装备、武学 |
| 体质 | 影响生命值、防御 | 升级加点、装备、武学 |
| 幸运 | 影响掉落、暴击 | 装备、特殊道具 |

### 属性计算
- **总属性** = 基础属性 + 装备加成 + 武学加成
- **生命值** = 体质 × 10 + 基础生命值
- **攻击力** = 力量 × 武学倍率

## 武学体系
