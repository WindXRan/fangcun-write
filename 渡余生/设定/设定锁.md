# 《渡余生》设定锁

## 一级锁定（不可更改）

### 核心地名
- **京城**：大庆朝都城，权力中心
- **青州**：边境重镇，战事频发之地
- **药王谷**：隐世医门所在，女主出身之地
- **七宿司**：皇帝直属暗卫机构，男主执掌
- **寒山寺**：京城郊外古寺，藏有关键秘密

### 关键物件
- **玄铁令牌**：七宿司首领信物，可调动暗卫
- **青囊针**：药王谷传承医针，女主随身之物
- **血玉佩**：男主母亲遗物，牵扯身世之谜
- **金丝楠木匣**：藏于寒山寺，内有机密卷宗
- **紫金丹**：宫廷秘药，实为慢性毒药

### 人物关系锁
- **沈清棠**（女主）：药王谷传人，医术精湛，性格坚韧
- **裴行舟**（男主）：七宿司首领，冷面心热，身世成谜
- **裴老夫人**：男主祖母，慈祥但心思深沉
- **皇帝**：多疑猜忌，沉迷丹药
- **太子**：表面温和，实则阴狠
- **药王**：女主师父，隐世高人

### 背景事件锁
- **药王谷灭门案**：十五年前，药王谷被神秘势力灭门，女主被师父救出
- **裴家军覆灭案**：十年前，裴家军在青州全军覆没，唯男主幸存
- **七宿司清洗**：五年前，男主执掌七宿司后大肆清洗异己
- **丹药案**：近年，太子进献丹药，皇帝沉迷其中

### 时间线锁
- 故事开始：大庆历永安二十年春
- 女主年龄：十七岁
- 男主年龄：二十三岁
- 药王谷灭门：女主两岁时
- 裴家军覆灭：男主十三岁时

## 反调色盘自检

### 与《惜花芷》差异化

| 维度 | 《惜花芷》 | 《渡余生》 | 是否通过 |
|------|-----------|-----------|---------|
| 核心设定 | 世家被抄家 | 医女卷入宫廷阴谋 | ✓ |
| 核心冲突 | 皇权打压家族 | 医术与权力的博弈 | ✓ |
| 角色原型 | 世家千金被迫撑家 | 医女主动掌控命运 | ✓ |
| 开篇方式 | 抄家 | 乱葬岗救人 | ✓ |
| 破冰方式 | 主动承担责任 | 用医术建立联系 | ✓ |

### 差异化说明
- 女主身份从世家千金变为医女，有独立技能
- 男主从世子变为身世成谜的暗卫首领
- 冲突从家族兴衰变为医术与权力的博弈
- 开篇从被动受灾变为主动救人
- 感情线从相识相知变为救命之恩+身份对立
