- **UGC Prompt**: Prompt创建、分享、收藏
- **支付/会员**: 预支付、金币、会员、订单
- **邀请/提现**: 邀请码、排行榜、奖励
- **收藏/历史**: 收藏管理、历史记录
- **标签/模板**: 标签、模板、配置
- **统计/事件**: 用量统计、事件追踪
- **文件上传**: 简单上传、分块上传
- **营销/运营**: 活动、优惠券、签到、任务中心
- **排行榜**: UGC排行榜、工作流排行榜

#### v2 API (~55个端点)
- **小说v2**: 结构化编辑 (卷/章/节点树)、排序、回收站、草稿系统
- **角色v2**: 角色CRUD、恢复
- **设定v2**: 设定、背景设定、势力设定CRUD
- **回收站v2**: 批量恢复、已删除内容管理
- **草稿v2**: 草稿夹、草稿排序
- **封面v2**: AI封面图生成、上传、保存
- **视频v2**: 视频生成、视觉风格、TTS、素材管理
- **UGC签约v2**: 签约状态、资格、签约操作
- **CDK v2**: CDK验证
