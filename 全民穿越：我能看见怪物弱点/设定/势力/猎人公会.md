# 势力档案：猎人公会

## 基本信息

| 项目 | 内容 |
|------|------|
| 名称 | 猎人公会 |
| 类型 | 组织 |
| 阵营 | 中立 |
| 核心理念 | 以猎为生，以猎为荣 |

## 组织结构

### 高层
- **会长**：李云龙（老猎人，传奇猎人）
- **副会长**：2-3名，负责不同区域
- **长老**：5-10名，资深猎人

### 中层
- **精英猎人**：30级以上，可接高级任务
- **资深猎人**：20级以上，可接中级任务
- **正式猎人**：10级以上，可接普通任务

### 基层
- **见习猎人**：1-10级，只能接低级任务
- **学徒**：未转职，跟随猎人学习

## 入会条件

1. 等级达到10级
2. 通过猎人考核（击杀指定怪物）
3. 有两名猎人推荐
4. 无不良记录

## 公会福利

| 等级 | 福利 |
|------|------|
| 见习猎人 | 基础任务、低价药水 |
| 正式猎人 | 中级任务、公会装备 |
| 资深猎人 | 高级任务、公会技能 |
| 精英猎人 | 稀有任务、公会驻地 |
| 长老 | 传奇任务、决策权 |

## 公会任务

| 类型 | 奖励 | 说明 |
|------|------|------|
| 采集任务 | 金币、材料 | 采集指定材料 |
| 狩猎任务 | 金币、装备 | 击杀指定怪物 |
| 护送任务 | 金币、经验 | 护送NPC或玩家 |
| 探索任务 | 稀有道具 | 探索未知区域 |
| 讨伐任务 | 传奇装备 | 讨伐BOSS |

## 与林默的关系

- **第5章**：林默遇到李云龙，获得猎人指导
- **第15章**：林默加入猎人公会，成为见习猎人
- **第30章**：林默成为精英猎人，获得公会认可
- **第50章**：林默成为长老，参与公会决策
