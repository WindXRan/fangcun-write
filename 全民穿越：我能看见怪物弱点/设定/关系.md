# 角色关系

## 核心团队

### 林默 ↔ 苏瑶
- **关系类型**：队友 → 暧昧 → 恋人
- **情感**：信任、依赖、好感
- **描述**：林默救下苏瑶，两人逐渐建立信任。苏瑶是队伍核心辅助，林默是核心输出。两人互相欣赏，但都不轻易表达。
- **演变轨迹**：
  - 第1-10章：救命恩人 → 队友
  - 第11-30章：队友 → 互相信任
  - 第31-50章：信任 → 好感
  - 第51章后：好感 → 暧昧 → 恋人

### 林默 ↔ 赵铁柱
- **关系类型**：队友 → 兄弟
- **情感**：忠诚、义气、信任
- **描述**：林默救下赵铁柱，赵铁柱死心塌地跟随。赵铁柱是队伍核心肉盾，林默是核心输出。两人以兄弟相称。
- **演变轨迹**：
  - 第1-10章：救命恩人 → 队友
  - 第11-30章：队友 → 兄弟
  - 第31章后：兄弟 → 生死之交

### 苏瑶 ↔ 赵铁柱
- **关系类型**：队友
- **情感**：尊重、信任
- **描述**：同为林默队友，互相配合。苏瑶负责治疗，赵铁柱负责抗伤害。
- **演变轨迹**：
  - 第1-10章：陌生人 → 队友
  - 第11章后：队友 → 默契搭档

## 对手关系

### 林默 ↔ 王霸天
- **关系类型**：对手 → 敌人
- **情感**：竞争、敌意
- **描述**：王霸天是另一个势力的首领，与林默争夺资源和地盘。两人多次冲突，逐渐成为死敌。
- **演变轨迹**：
  - 第11-20章：陌生人 → 竞争对手
  - 第21-40章：竞争对手 → 敌人
  - 第41章后：敌人 → 死敌

## 恩人关系

### 林默 ↔ 李云龙
- **关系类型**：恩人 → 师徒
- **情感**：尊敬、感激
- **描述**：李云龙是老猎人，教会林默很多技巧。林默视其为恩人和师父。
- **演变轨迹**：
  - 第5-15章：陌生人 → 恩人
  - 第16章后：恩人 → 师父
