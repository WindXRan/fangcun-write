# 全书大纲

## 基本信息

| 项目 | 内容 |
|------|------|
| 书名 | 全民穿越：我能看见怪物弱点 |
| 题材 | 玄幻脑洞（全民穿越+升级流） |
| 目标平台 | 番茄小说 |
| 预计字数 | 150-200万字 |
| 预计章数 | 500-600章 |
| 目标读者 | 18-35岁男性，喜欢升级流、爽文 |

## 一句话梗概

全民穿越异世界，猎户林默获得"弱点透视"金手指，能看见怪物弱点，从一个普通猎人成长为传奇猎人，带领人类在蛮荒大陆建立新秩序。

## 核心卖点

1. **弱点透视金手指**：能看见怪物弱点，攻击弱点造成暴击
2. **猎户开局**：比矿工更有战斗性，爽点更密集
3. **全民穿越**：所有人都在争抢资源，紧迫感强
4. **升级可视化**：属性面板+弱点显示，爽感具象化

## 卷级结构

### 第一卷：蛮荒求生（第1-50章，约15万字）
- **功能**：铺垫+建立优势
- **核心事件**：林默熟悉世界，组建队伍，击杀第一个BOSS
- **起始状态**：普通猎户，等级1
- **结束状态**：小有名气，等级15，队伍成型

### 第二卷：崛起之路（第51-150章，约30万字）
- **功能**：崛起+建立势力
- **核心事件**：林默加入猎人公会，击杀多个BOSS，组建猎人小队
- **起始状态**：小有名气，等级15
- **结束状态**：一方强者，等级35，猎人小队成型

### 第三卷：争霸天下（第151-300章，约45万字）
- **功能**：争霸+势力扩张
- **核心事件**：林默建立势力，与其他势力争霸，击杀世界BOSS
- **起始状态**：一方强者，等级35
- **结束状态**：顶级玩家，等级55，势力成型

### 第四卷：登顶之路（第301-450章，约45万字）
- **功能**：登顶+对抗怪物潮
- **核心事件**：林默统一势力，对抗怪物潮，击杀传奇BOSS
- **起始状态**：顶级玩家，等级55
- **结束状态**：传奇猎人，等级75，人类势力统一

### 第五卷：传奇终章（第451-550章，约30万字）
- **功能**：终局+找到回去方法
- **核心事件**：林默击杀世界BOSS，找到回去方法，建立新秩序
- **起始状态**：传奇猎人，等级75
- **结束状态**：传奇猎人，等级100，找到回去方法
