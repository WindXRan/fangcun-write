仿写时调性一致性是红线级要求。开书预设（open-book / enhanced-open-book）生成的任何内容——简介、总纲、角色卡、卷纲——都必须严格匹配源文的题材、语气、风格定位。

**为什么会偏离：** LLM 默认偏"稳"偏"正"，容易把权谋文写成正剧、把轻松文写成严肃、把口语化写成书面腔。每次输出前必须自检调性。

**如何应用：**
- 输出前对照源文正文前三章，检查语感是否一致
- 简介/卖点的措辞要像真人作者写的，不用网感词（"拉满""爆了""直接封神"）
- 角色语言风格 7 维必须与题材匹配：古风文不说现代梗，轻松文不说书面腔
- 系统设定严格忠于源文，不自行添加源文没有的代价/规则
