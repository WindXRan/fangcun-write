# Coze工作流部署指南

## 小说转剧本 Bot 配置

### 1. 创建Bot

1. 登录 [coze.cn](https://www.coze.cn)
2. 点击「创建Bot」
3. 填写信息：
   - 名称：小说转剧本助手
   - 描述：一键将小说转换为短剧剧本，支持自动提取事件、生成骨架、编写剧本
   - 图标：上传一个剧本相关图标

### 2. 配置工作流

进入Bot编辑页面 → 工作流 → 新建工作流

#### 节点1：开始节点
- 输入变量：`novel_text`（用户上传的小说文本）
- 输入变量：`episode_count`（生成集数，默认10）

#### 节点2：事件提取（LLM节点）
```
模型：豆包/DeepSeek
Prompt：
你是一个专业的剧本策划。请从以下小说中提取关键事件，每个事件包含：
- 章节来源
- 事件描述
- 涉及角色
- 冲突类型
- 情绪强度（1-10）

小说内容：
{{novel_text}}

请输出JSON格式的事件列表。
```

#### 节点3：骨架生成（LLM节点）
```
模型：豆包/DeepSeek
Prompt：
你是一个专业的剧本架构师。根据以下事件列表，生成{{episode_count}}集短剧骨架。

要求：
1. 每集包含：集数、标题、核心冲突、情绪曲线、钩子
2. 前3集免费，第4集开始付费
3. 每集结尾要有悬念钩子
4. 节奏要快，每集2分钟

事件列表：
{{node2.output}}

请输出JSON格式的骨架。
```

#### 节点4：剧本编写（LLM节点）
```
模型：豆包/DeepSeek
Prompt：
你是一个专业的短剧编剧。根据以下骨架，编写完整的短剧剧本。

格式要求：
- 场景描述用△标记
- 对话格式：角色名：台词
- 每集目标300字台词
- 优先用画面和动作叙事

骨架：
{{node3.output}}

请输出完整剧本，每集用---分隔。
```

#### 节点5：结束节点
- 输出变量：`script`（node4.output）

### 3. 添加开场白

在Bot设置 → 开场白：

```
欢迎使用小说转剧本助手！

请发送小说文本，我会自动帮你：
1. 提取关键事件
2. 生成剧本骨架
3. 编写完整剧本

支持格式：
- 直接粘贴小说文本
- 上传txt文件

示例：
"帮我把这段小说转成10集短剧剧本：[小说内容]"
```

### 4. 添加预置对话

在Bot设置 → 预置对话：

| 问题 | 回答 |
|------|------|
| 你是做什么的？ | 我是小说转剧本助手，可以帮你把小说自动转换为短剧剧本。 |
| 支持什么格式？ | 支持直接粘贴文本或上传txt文件。 |
| 生成剧本要多久？ | 根据小说长度，通常1-3分钟。 |
| 可以生成多少集？ | 建议10-30集，每集2分钟。 |

### 5. 发布

1. 点击「发布」
2. 选择发布范围：商店/私有链接
3. 填写商店信息：
   - 分类：写作工具
   - 标签：剧本、短剧、小说、AI写作
   - 简介：一键将小说转换为短剧剧本

## 注意事项

1. **模型选择**：推荐豆包或DeepSeek，性价比高
2. **Token限制**：单次对话有token限制，长小说需要分段处理
3. **审核时间**：商店上架需要1-3天审核
4. **内容安全**：确保小说内容合规，避免敏感题材
