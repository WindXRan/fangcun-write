# 全书大纲

## 书名
《每天模拟一次，阿斗逆天了》

## 题材
历史脑洞 + 系统模拟器

## 一句话梗概
穿越成刘禅，绑定人生模拟器，每次模拟都能获得奖励，预知未来，算无遗策，从"扶不起的阿斗"变成"千古一帝"。

## 卷级结构

### 第一卷：开局（约30章，12万字）
- 功能：铺垫+起步+第一个大爽点
- 核心事件：穿越成刘禅，绑定模拟器，用模拟器改变第一次历史事件，让诸葛亮等人刮目相看
- 起始状态 → 结束状态：刘禅从"扶不起的阿斗"变成"让人刮目相看的聪明人"

### 第二卷：救关羽（约50章，20万字）
- 功能：发展+第一个大高潮
- 核心事件：用模拟器预知关羽败走麦城，提前布局救援，改变历史
- 起始状态 → 结束状态：刘禅从"聪明人"变成"蜀汉支柱"，关羽从"败走麦城"变成"守住荆州"

### 第三卷：救张飞（约50章，20万字）
- 功能：发展+第二个大高潮
- 核心事件：用模拟器预知张飞被部下杀害，提前防范，改变历史
- 起始状态 → 结束状态：刘禅从"蜀汉支柱"变成"蜀汉未来"，张飞从"被部下杀害"变成"善终"

### 第四卷：刘备称帝（约30章，12万字）
- 功能：转折+第三个大高潮
- 核心事件：用模拟器帮助刘备称帝，建立蜀汉，同时对抗曹魏和孙吴
- 起始状态 → 结束状态：刘禅从"蜀汉未来"变成"蜀汉太子"，蜀汉从"割据势力"变成"正式王朝"

### 第五卷：诸葛亮北伐（约80章，32万字）
- 功能：高潮+核心对决
- 核心事件：用模拟器帮助诸葛亮北伐，对抗司马懿，改变五丈原命运
- 起始状态 → 结束状态：刘禅从"蜀汉太子"变成"蜀汉实际掌权者"，诸葛亮从"五丈原去世"变成"北伐成功"

### 第六卷：统一三国（约60章，24万字）
- 功能：最终高潮+收尾
- 核心事件：用模拟器统一三国，成为千古一帝，建立长治久安的王朝
- 起始状态 → 结束状态：刘禅从"蜀汉实际掌权者"变成"千古一帝"，三国从"分裂"变成"统一"

## 总字数
约200章，100万字

## 核心爽点
1. 第一卷：绑定模拟器，改变第一次历史事件，让诸葛亮刮目相看
2. 第二卷：救关羽，改变败走麦城命运
3. 第三卷：救张飞，改变被部下杀害命运
4. 第四卷：帮助刘备称帝，建立蜀汉
5. 第五卷：帮助诸葛亮北伐，改变五丈原命运
6. 第六卷：统一三国，成为千古一帝

## 核心冲突
1. 刘禅 vs 司马懿：终极对手，智谋对决，改变历史
2. 蜀汉 vs 曹魏：主要敌人，统一战争
3. 蜀汉 vs 孙吴：潜在盟友或敌人，根据形势变化

## 核心反转
1. 刘禅从"扶不起的阿斗"变成"让人刮目相看的聪明人"
2. 关羽从"败走麦城"变成"守住荆州"
3. 张飞从"被部下杀害"变成"善终"
4. 诸葛亮从"五丈原去世"变成"北伐成功"
5. 蜀汉从"灭亡"变成"统一三国"
