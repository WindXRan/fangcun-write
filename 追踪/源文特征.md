## === SECTION: world_rules ===

### 世界规则

**基础世界观**：现代都市，现实世界为底层框架，叠加"穿书"元设定。

**核心机制**：

1. **穿书机制**
   - 宋云绯（现实灵魂）穿越进入一本名为（推测）的小说世界，接管了同名恶毒女配"宋云绯"的身体
   - 穿越时自动继承原主记忆，包括原主的身份信息、人际关系、所作所为
   - 穿越者知道原书剧情走向和角色结局，但不知道当前时间点之后的实时变化

2. **原著剧情框架**
   - 原著是一本典型的"总裁文"：男主楚靳寒是宴金集团总裁，前未婚妻是女主
   - 原主宋云绯是恶毒女配：趁男主失忆假冒女友，骗他到偏远青城过穷日子
   - 原书结局：宋云绯身份暴露后惨死（具体死法未明，但足以让穿越者恐惧）
   - 前未婚妻原设定是"等待型"女主，但本书中她重生后选择不再等待

3. **重生机制**（前未婚妻线）
   - 前未婚妻拥有重生金手指，重生后放弃楚靳寒
   - 这导致楚靳寒在原书主线中从"男主"降格为"追妻火葬场男二"
   - 重生者的视角切换是本书重要的暗线张力来源

4. **信息差规则**
   - 女主知道原书剧情但不知道男主已恢复记忆
   - 男主恢复记忆后知道女主在骗他，但选择配合观察
   - 男主不知道女主是穿书者（只知道她"变了"）
   - 多层信息差构成核心戏剧张力

5. **空间规则**
   - 主场景：青城（偏远小城，经济落后，出租屋密集）
   - 对比场景：宴金集团所在的大城市（豪门、上流社会）
   - 空间落差是身份落差的外化表现

### 逐章情节骨架（前10章）

#### 第1章：穿书开局

**核心事件**：宋云绯穿书醒来，出租屋+外卖服男主，记忆涌入
**场景**：破旧出租屋
**人物状态**：
- 女主：惊恐→快速消化信息→进入"生存模式"
- 男主：失忆状态，被动接受"女友"设定，穿外卖服（白天上班晚上送外卖）
**关键信息揭示**：
- 原主假冒女友的事实
- 男主失忆的原因（被女主撞的）
- 原书结局的暗示（女主惨死）
**叙事功能**：用最短篇幅完成世界观铺设，"失忆总裁穿外卖服洗床单"这个画面本身就是强钩子
**情绪曲线**：震惊→荒诞→恐惧（原书结局）→下定决心

#### 第2章：被迫同床

**核心事件**：出租屋只有一张床，女主拒绝亲密，推男主下床
**场景**：出租屋卧室
**人物状态**：
- 女主：坚持边界感，拒绝假戏真做
- 男主：配合但有微妙不满（被推下床的屈辱感）
**关键冲突**：物理空间的亲密 vs 心理距离的疏远
**叙事功能**：建立两人关系的核心张力——名义上是情侣，实际上陌生人
**喜剧元素**：推人下床的肢体冲突，"失忆总裁睡地板"的荒诞画面

#### 第3章：套套与查岗

**核心事件**：发现套套（暗示原主计划），男主公司被称"查岗哥"，胡瑶撩拨
**场景**：出租屋 + 公司（侧写）
**人物状态**：
- 女主：尴尬→警觉→决定改变
- 男主：被动承受"查岗"标签
**关键信息**：
- 原主对男主的控制欲极强
- 公司同事对男主的关注（胡瑶等）
**叙事功能**：通过"套套"这个道具揭示原主的居心，通过"查岗哥"标签展现原主的行为模式
**调性**：轻松喜剧为主，暗藏危险信号

#### 第4章：女主开始改变

**核心事件**：买菜做饭，不再好吃懒做
**场景**：出租屋厨房（简陋）

**叙事功能**：转折点——穿书者的行为模式开始与原主产生偏差
**伏笔**：男主的"观察"暗示他并非完全失忆/被动，为后续"早已恢复记忆"埋线

#### 第5章：找工作+无意识缠抱

**核心事件**：女主找到房产销售工作，睡觉时无意识抱住男主

**人物状态**：

#### 第7章：不查岗+冰箱

**核心事件**：女主不再监控男主，男主反而在意；商场抽奖中冰箱
**场景**：出租屋 + 商场
**人物状态**：
- 女主：主动放弃"女友"控制权
- 男主：被"不查岗"后反而关注女主的动态
**关系转折**：权力结构开始反转——从"女主控制男主"到"男主关注女主"
**冰箱事件**：生活改善的小里程碑，轻喜剧调性
**叙事功能**：通过"不查岗"这个行为变化，推动男主从被动变为主动观察

#### 第8章：电动车与宝马

**核心事件**：推电动车回家遇宝马女炫耀，男主说"委屈你了"
**场景**：青城街道
**人物状态**：
- 女主：对阶层差距无感（她知道男主真实身份）
- 男主：愧疚+心疼（他知道真相但不能说）
**阶层对比**：电动车 vs 宝马 = 当前困境 vs 未来回归
**"委屈你了"**：男主首次流露真实情感，这句话有四重含义：
1. 他知道女主在"受苦"（表面）
2. 他知道自己的真实身份能给她更好的生活（隐层）
3. 他知道女主在骗他但仍心疼（更深）
4. 他对这段"假关系"产生了真感情（最深）
**叙事功能**：情感高点，男主的情感从"观察"转向"介入"

#### 第9-10章：持续发展

**推测方向**：
- 女主在房产行业继续挣扎或找到新出路
- 两人日常相处中更多"无意识亲密"细节
- 男主可能开始暗中布局（恢复记忆后的行动线）
- 胡瑶或其他外部角色制造新的冲突
- 女主"跑路计划"与"感情发展"的矛盾加剧

## 附录：核心叙事引擎

### 三重信息差结构

```
女主视角：我知道原书剧情 → 但不知道男主已恢复记忆
男主视角：我已恢复记忆   → 但不知道女主是穿书者
读者视角：知道双方的秘密  → 等待"摊牌时刻"
```

### 爽点公式

```
身份反差（总裁装穷）× 信息反差（互相隐瞒）× 情感反差（假戏真做）= 高密度爽感
```

### 调性定位

**轻松喜剧为表，暗流涌动为里**。日常笑料（推下床、查岗哥、邪教早会）掩盖深层张力（身份隐瞒、原书结局恐惧、感情与自保的拉锯）。
