# Project: 三国杀：我的牌库成神路

## 状态
世界观设定文件已创建 at `三国杀：我的牌库成神路/设定/世界设定.md` (2026-05-17)

## 核心设定摘要
- **主角**: 林夜，穿越者，SSS级天赋「技能融合」
- **世界法则**: 三国杀卡牌游戏为底层规则，16岁觉醒获得本命武将牌
- **卡牌品阶**: 白→绿→蓝→紫→橙→红(传说)→金(神话)
- **修为等级**: 1-9星 → 史诗 → 传说 → 神话 → 神
- **三大势力**: 魏(铁血秩序)、蜀(义气羁绊)、吴(世家博弈)、中立区洛阳(天机阁)
- **金手指**: 技能融合(1+1>2，可跨阵营融合)，初期限制，中期复式融合，后期多维融合
- **核心梗**: 品阶决定论 vs 后天突破，打破阶级天花板
