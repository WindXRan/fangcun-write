# 魔王只养露娜 第3章扩展

## 扩展内容
第3章"第一次挑战"从约1020汉字扩展到约3500+汉字。

### 新增六大模块
1. **世界频道挑衅场景扩展**：林墨刚睡醒，眯眼看面板，露娜站在门口看到消息后平静等待，林墨查看羁绊值37后决定接战
2. **广场场景扩展**：方圆三十米黄土广场，押注开盘、嗑瓜子、起哄的围观群众，烈火战神和火鬃豹的细节描写
3. **战斗10回合制展开**：从简单叙述扩展为回合制战斗——
   - 1-2回合：火鬃豹猛攻，露娜闪避受伤
   - 3-4回合：林墨观察出3种攻击模式，开始分析
   - 5-6回合：露娜反击，第一剑命中前腿
   - 7-8回合：烈焰冲击大招，露娜擦火焰边缘滑过，侧脸燎伤
   - 9-10回合：刺中后腿关节，火鬃豹跪地
4. **观战反应层次**：嘲笑→惊讶→沉默→震惊的渐进变化，押注人划掉账本，戴眼镜玩家分析"不是巧合"
5. **系统设定暗示**：玩家发现露娜战斗数据被隐藏，为后续"锁"的概念铺垫
6. **战后温情收尾**：林墨帮露娜摘烧焦的头发（摘不掉），陌生玩家喊"挺牛的"，强化羁绊

### 格式约束遵守情况
- 一句一段 ✓
- 每段≤60汉字 ✓
- 段落间无空行 ✓
- 对话用动作引出，无「他说」「她道」 ✓
- 不跳出林墨限知视角 ✓
- 禁用词排查通过 ✓

## 扩展原则
- 战斗写策略和反转，非"你一拳我一脚"流水账
- 每位扩展都推动情绪（爽/温情/悬念）或铺垫（锁的暗示）
- 林墨的观察和分析贯穿战斗，强化"限知视角"特点
- 收尾用动作和细节（焦发按不下去），无总结升华

**Why:** 原章约1020字过于单薄，战斗过程太短，缺乏情绪积累。扩展后达到网文章节基本体量。

**How to apply:** 未来写战斗章节时，先规划回合数（建议8-12回合），在每回合间插入主角观察和观众反应，形成"打一下→停一下→想一下"的节奏。
