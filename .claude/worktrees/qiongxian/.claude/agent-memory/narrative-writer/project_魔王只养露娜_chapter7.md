# 魔王只养露娜 第7章写作记录

**文件名**: `魔王只养露娜/正文/第7章-第一个朋友.md`
**字数**: ~2200汉字
**章名**: 第一个朋友

## 情节要点
- 胖子（王宇）追上林墨，自我介绍，诉苦二十多发无SSR
- 林墨戒备拒绝，但因论坛/系统/买号压力不想惹麻烦
- 胖子跟了一整个上午，死缠烂打
- 露娜对胖子的2星盾卫产生兴趣，主动拉林墨袖子做"笑"表情——首次对外人展现兴趣
- 林墨松口给暗示："把资源留给第一个英雄"
- 胖子半信半疑离开，林墨触动——第一个来请教而非嘲讽的人
- 章尾：露娜再次做"笑"表情，林墨笑着回"走吧，回去"

## 写作技法
- 开头用"陌生人喊住主角"钩子，直接切入
- 胖子语言特点：语速快、急切、带结巴（"可……可算找着你了"）
- 露娜全程用肢体语言（拉袖子、指方向、做表情）——不说话但比说话更有表现力
- 林墨的拒绝动机衔接第6章伏笔（系统检测异常、有人买号）
- 章尾无升华总结，用露娜的轻快步子暗示情绪
- 胖子离开时走几步回头的画面"节奏锚"——首次呼应时确认角色

- 前接：第6章系统检测异常+有人买号→林墨压力
- 后续：第8章新手魔王大赛→胖子为友方角色
