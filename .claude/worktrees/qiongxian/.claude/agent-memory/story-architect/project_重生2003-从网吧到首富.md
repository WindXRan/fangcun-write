# 重生2003：从网吧到首富 — 项目记忆

**书名**：重生2003：从网吧到首富（暂定）
**类型**：都市重生 + 互联网行业文
**目标平台**：番茄小说 | 男性向 | 预计150万字

**核心梗**：前独角兽CTO被资本踢出局后重生2003年，用20年产品手感从网吧起步，截胡传奇私服、抢魔兽代理、抄底移动互联网——做一个无法被抹去的人。

**主角**：林远（前世38岁/重生18岁），金手指=20年肌肉记忆级产品直觉+看人眼光+谈判手感，记忆随蝴蝶效应逐渐失效。

**卷结构**：
1. 第一卷（2003-2004）：网吧起家
2. 第二卷（2005-2007）：游戏切入
3. 第三卷（2008-2010）：平台扩张
4. 第四卷（2011-2014）：移动转型
5. 第五卷（2015-2020）：帝国时代

**已产出**：
- 设定/世界观/背景设定.md — 世界观完整设定（2026-05-20）

**关键创作决策**：
- 蝴蝶效应规则：林远的记忆优势分阶段递减，后期靠能力非记忆
- 核心冲突三层：个人恩怨（被踢出局）→ 行业对抗（巨头围剿）→ 时代对决（改变互联网格局）
- 情绪弧线：卷一递进+V形、卷二W形、卷三递进+急转

**Why**: 用户提供核心设定后，需要细化的世界观文档支撑后续大纲排布和章节写作。

**How to apply**: 后续进行大纲排布、开篇设计、钩子/反转设计时，以本文档的世界观规则和情绪弧线为基础。
