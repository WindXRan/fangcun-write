# 项目：重生2003-从网吧到首富 — 主角角色设计

- 类型：都市重生行业文（番茄男频）

- 设计日期：2026-05-20
- 角色档案路径：`重生2003-从网吧到首富/设定/角色/林远.md`

## 核心设计决策

### 1. 金手指定位
不是超能力/系统，是"20年操盘手感"——前世经验的肌肉记忆。
- 产品直觉（知道什么产品会成功）
- 看人眼光（知道一个人能不能合作）
- 谈判直觉（知道底牌在哪）
- 技术架构能力（知道什么技术路线会赢）
- 关键时间节点（非典、电商崛起、移动互联网爆发等）
关键约束：蝴蝶效应会导致经验逐渐失效，不是100%准确。

### 2. 核心驱动力
不是复仇驱动——是"为自己活"驱动。
前世被踢出局+酗酒致死 → 重生后目标是建立自己的商业帝国 + 学会做"人" + 弥补对家人的亏欠。
对前世仇人的态度：用商业成功碾压，不是低级报复。

### 3. 核心成长弧线
"前世技术宅→这一世学会做'人'"。
五卷对应五个认知升级阶段：找帮手→放权→处理信任危机→学会灰度→人格圆满。

### 4. 前世关键人物
- 陈启明（大学同学+联合创始人，背叛者。这一世第四卷以新人身份出现）
- 李国宏（方略资本合伙人，资本方代表。这一世第五卷终极对手）
- 周教授（大学计算机教授，2008年病逝。这一世林远要弥补遗憾）
- 苏晴（前世初恋，因林远太忙分手。这一世感情线核心）

### 5. 这一世核心团队
- 王磊"王胖子"（大学室友，白手套角色，社交天才）
- 张恒（大学同学，技术天才，社交零分，林远前世的镜像）
- 王叔（网吧老板，第一卷导师/助力）

### 6. 三层标签反差
- 身份标签：18岁县城少年
- 表现标签：沉默寡言、不争不抢
- 内核标签：38岁互联网老兵灵魂
反差 = 立体感，亮牌时刻 = 爽点。

### 7. 语言风格核心
- 短句连击，数据驱动
- 口头禅："看数据""这个逻辑不对""等等，让我想想"
- 标志动作：决策前敲桌面三下
- 权力博弈：掌控者时对话极短，被触怒时语气变冷变慢

### 8. 爽感六大来源
1. 信息差碾压（知道别人不知道的趋势）
2. 看人准（慧眼识珠，挑的人后来全成大佬）
3. 前世仇人被碾压（不是你死我活，是"你不配做对手了"）
4. 技术直觉从不翻车
5. 弥补前世遗憾（家人、恩师、初恋）
6. 校园装逼（18岁身体+38岁灵魂的降维打击）
