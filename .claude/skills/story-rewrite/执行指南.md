# 执行指南：Phase 2 写作流程

**重要**：此文件用于指导主agent如何正确执行Phase 2写作流程。

---

## 核心原则

1. **每个区间必须独立执行完整流程** - 蒸馏、结构映射、章纲都要重新生成
2. **必须使用Task工具启动子agent** - 不能自己直接写
3. **必须传递完整提示词** - writer-system.md作为system prompt
4. **必须并行执行** - 每区间10章，启动10个并行agent

---

## 每区间完整流程（必须按顺序执行）

```
Phase 2：写作（每区间 10 章）
  ├── Step 1: 提取源文第x-y章 → 追踪/源文_{x-y}.txt
  ├── Step 2: 源文逐章分析 → 追踪/源文特征_{x-y}.md
  ├── Step 3: 文风蒸馏 → 追踪/蒸馏_{x-y}.md  ⚠️ 必须重新蒸馏
  ├── Step 4: 结构映射 → 追踪/结构映射_{x-y}.md  ⚠️ 必须重新生成
  ├── Step 5: 章纲细化 → 大纲/章纲_{x-y}.md  ⚠️ 必须重新生成
  ├── Step 6: 写新章（10 并行）→ 正文/第N章.txt
  ├── Step 7: 校验 → pass/fail
  └── Step 8: Observer + Settler → 更新 truth files
```

---

## 详细执行步骤

### Step 1：提取源文第x-y章

```bash
# 使用source_chapter_splitter.py提取
python source_chapter_splitter.py extract <源文.txt> <start> <end> <追踪/源文_{x-y}.txt>
```

### Step 2：源文逐章分析

使用 `prompts/source-analyzer.md` 分析提取的源文，生成：
- `追踪/源文特征_{x-y}.md`：包含plot_structure, key_events

### Step 3：文风蒸馏 ⚠️ 关键步骤

**必须对每个区间的源文重新进行文风蒸馏**：

Layer A：统计指纹
```bash
python style_analyzer.py 追踪/源文_{x-y}.txt
```

Layer B：LLM 8维度分析
- 使用 `prompts/style-analysis.md`
- 分析第x-y章的文风特征
- 输出：`追踪/蒸馏_{x-y}.md`

Layer C：写作方法论
- 首次区间生成 `追踪/写作方法论.md`，后续复用

### Step 4：结构映射

从本区间的源文分析结果中提取逐章映射：
- 输出：`追踪/结构映射_{x-y}.md`

### Step 5：章纲细化

基于：
- 卷纲（整体规划）
- 源文分析的plot_structure
- 当前truth files

生成本区间的详细章纲：
- 输出：`大纲/章纲_{x-y}.md`

### Step 6：写新章（并行）

使用Task工具启动10个并行writer agent，**必须使用本区间的蒸馏数据**：

```
⚠️ 关键：文风指南必须指向本区间的蒸馏文件
【文风指南】请先读取：{书名}/追踪/蒸馏_{x-y}.md  ← 必须是本区间的！
```

### Step 7：校验

对每章执行质量检查：
- 字数合规（目标2000-2500字，硬上限3000字，硬下限1800字）
- AI 痕迹检测
- 禁用词/句式检查
- 章节完整性

### Step 8：状态沉淀

启动Observer和Settler agent更新truth files：
- `追踪/current_state.md`
- `追踪/pending_hooks.md`
- `追踪/chapter_summaries.md`
- `追踪/character_matrix.md`
- `追踪/emotional_arcs.md`

## 常见错误

❌ **错误1**：主agent自己直接写
✅ **正确**：使用Task工具启动子agent

❌ **错误2**：不传提示词，只传章纲
✅ **正确**：传递完整的user message模板

❌ **错误3**：串行执行，一章一章写
✅ **正确**：并行执行，同时启动10个agent

❌ **错误4**：不读取参考文件
✅ **正确**：在prompt中明确要求先读取所有参考文件

❌ **错误5**：用第1-10章的蒸馏数据写第11-20章
✅ **正确**：每个区间必须重新蒸馏，用本区间的蒸馏数据

❌ **错误6**：跳过蒸馏步骤，直接写新章
✅ **正确**：必须先完成Step 1-5，再执行Step 6
