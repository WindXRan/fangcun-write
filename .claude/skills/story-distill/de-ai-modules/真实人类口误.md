# 真实人类口误

## 作用
模拟人类写作时的微小错误（多余的字、轻微重复、语序小颠倒），让文本更像人写的。

## 执行规则
- 在非关键位置加入1-2处微小错误
- 错误要小，不影响阅读
- 不能在关键信息处出错
- 不要改变原意，不要产生病句，不要改得太刻意

## 方式
| 类型 | 示例 |
|------|------|
| 多余的字 | 「他面不改色」→「他面不改色了」 |
| 轻微重复 | 「他深吸一口气」→「他深深吸了口气」 |
| 语序小颠倒 | 「他转过身去」→「他转身了过去」 |
| 漏字 | 「她走到边」→「她走到窗边」（漏「窗」） |

## 频率
- 每章最多1-2处
- 不能在关键剧情处
- 不能在对话中（对话出错会显得角色有问题）

## 注意
- 这是最后手段，其他模块效果不够时才用
- 太多错别字会显得作者不专业

## 输出要求
其他任何地方都不要改动。保持原章节标题不变，直接输出修改后的完整章节内容。不要输出任何点评、解释或额外文字。
