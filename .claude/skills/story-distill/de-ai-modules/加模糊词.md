# 加模糊词

## 作用
人类写作会用模糊表达（「大概」「好像」「似乎」「差不多」），AI倾向精确表达。

## 触发条件
- 文中精确表达过多（每个数字/时间/距离都很精确）

## 执行规则
- 在语义通顺、不改变原意的前提下，少量（全文3-5处）随机加入「似乎」「好像」或「莫名」
- 优先加到句首或主语后面，例如：「林凡转过身」→「林凡似乎转过身」；「天空很暗」→「天空好像很暗」
- 非关键信息处，把精确表达改为模糊表达
- 关键信息（剧情转折、人名地名）保持精确

## 替换方式
| 精确表达 | 模糊表达 |
|---------|---------|
| 三年前 | 大前年 / 好几年前 |
| 二十个亲兵 | 几十个亲兵 |
| 午时前 | 午饭前 / 快到中午 |
| 走了三百步 | 走了一段路 |
| 高三尺三寸 | 三尺来高 |

## 示例
- 原文：「三年前的今日，太子带兵围了将军府。」
- 处理：「好几年前的事了。太子带兵围了将军府。」

## 例外
- 关键时间线不能模糊（如「三天后抄斩」必须精确）
- 对话中的数字保留（角色说话会用具体数字）
- 不要加在不该加的地方

## 输出要求
其他任何地方都不要改动。保持原章节标题不变，直接输出修改后的完整章节内容。不要输出任何点评、解释或额外文字。
