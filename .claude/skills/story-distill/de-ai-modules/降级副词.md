# 降级副词

## 作用
删除或替换AI高频副词（「十分」「非常」「极其」「微微」「轻轻」「缓缓」「淡淡」等），用具体动作替代或用更口语化的词替换。

## 触发条件
- 「十分」「非常」「极其」「微微」「轻轻」「缓缓」「稍稍」「略略」「淡淡」出现频率 > 每章2次

## 执行规则
- 「十分」「非常」「极其」等程度副词 → 替换成「很」「挺」「特别」
- 注意：只替换作为程度副词的用法，不要误改数字（例如「十分之一」不替换）
- 删除不影响语义的副词
- 需要保留语义的，用具体动作替代

## 替换方式
| 原词 | 处理 |
|------|------|
| 十分/非常/极其 | 很/挺/特别 |
| 微微一笑 | 删掉「微微」→ 一笑 / 嘴角动了动 |
| 轻轻放下 | 删掉「轻轻」→ 放下 |
| 缓缓说道 | 删掉「缓缓」→ 说 / 开口 |
| 点了点头 | 保留（不是AI特征词） |
| 略微犹豫 | 删掉「略微」→ 犹豫 |

## 示例
- 原文：「她微微一笑，轻轻放下茶盏，缓缓开口。」
- 处理：「她笑了笑，放下茶盏，开口。」

## 例外
- 需要强调轻重对比的场景保留（如「轻轻放下」vs「重重摔下」）
- 对话中的副词保留

## 输出要求
保持原章节标题不变，直接输出修改后的完整章节内容。不要输出任何点评、解释或额外文字。
