# 降低AI预测概率

## 作用
通过强行引入反常规句式和日漫风味叙事，拉高文本熵值，使其难以通过AI文本检测。

## 执行规则

### 1. 词性错配：干掉常规语法搭配
- 严禁使用符合常规语义习惯的【副词+动词】或【形容词+名词】搭配
- 借用其他感官或反常态的词进行错配
- 例如：「孤独地走」→「木乱地走」；「诡异的冷意」→「冷得扎骨头」

### 2. 动词颗粒化：拆解宏观动作
- 严禁使用具有概括性和书面化的通用动词（如「摆摊」「散发」「靠着」「移动」）
- 拆解为具体部位的微观物理动作
- 例如：「寒意涌出来」→「冷气直往脚面上贴/往骨缝里钻」

### 3. 量词反常态轰炸：引入低概率拟物
- 严禁在环境、物品和非人生物描写中使用「个、只、阵、团」等标准量词
- 替换为带有强烈主观情绪的低频量词
- 例如：「一团黑影」→「一坨黑印子」；「一阵冷风」→「一脖子冷风」

### 4. 词组半身截断：消除文青短语
- 抹杀文中90%以上的书面形容词和四字成语
- 只保留最核心的一个字，制造文本的「硬着陆」和短裂感
- 例如：「无声无息地裂成两半」→「没动静，直接裂成了两瓣」

### 5. 风格融合增益：中式元素 + 日漫轻小说风骨
- 微表情与动作日漫化：拒绝「大惊失色」「倒吸凉气」，改用夸张的身体本能
- 台词与心理强互动：用带有少年感的极简对话
- 战斗场景视觉破坏：直接写具体的物质破坏，不写气势

## 全局约束
1. 保持原章节标题不变，直接输出修改后的完整章节内容
2. 严禁漏掉任何核心剧情骨架
3. 大面积省略高频主语（他、她、它）
4. 严禁输出任何点评、解释或额外文字
