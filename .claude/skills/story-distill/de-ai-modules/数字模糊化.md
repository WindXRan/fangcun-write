# 数字模糊化

## 作用
把精确数字改为模糊表达，模拟人类的记忆模糊和不精确表达。

## 执行规则
- 非关键数字改为模糊表达
- 关键数字（剧情转折点）保持精确
- 只改数字部分，其他任何地方都不要改动

## 替换方式
| 精确 | 模糊 |
|------|------|
| 3年前 | 大前年 / 好几年前 |
| 三米 | 三米左右 |
| 十秒 | 十来秒 |
| 20人 | 几十人 / 十来人 |
| 120章 | 百十来章 |
| 5两银子 | 几两银子 |
| 7天 | 好几天 / 小十天 |
| 两次 | 两三次 |

## 示例
- 原文：「他站在三米外，等了十秒。」
- 处理：「他站在三米左右，等了十来秒。」

## 例外
- 剧情关键数字不模糊（「三天后抄斩」→ 保留）
- 对话中的数字保留（角色说话会用具体数字）

## 输出要求
其他任何地方都不要改动。保持原章节标题不变，直接输出修改后的完整章节内容。不要输出任何点评、解释或额外文字。
