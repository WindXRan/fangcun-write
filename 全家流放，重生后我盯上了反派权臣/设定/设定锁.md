# 设定锁 · 全家流放，重生后我盯上了反派权臣

> 一级锁定：核心地名、关键物件、人物关系、背景事件、时间线

## 时代背景

- **时代**：大衍朝（架空虚构）
- **年份**：承安十三年 → 故事开始
- **政治格局**：皇帝病弱，太子年幼，皇后宋氏把持后宫，三皇子、五皇子夺嫡
- **整体氛围**：山雨欲来，朝堂暗流涌动

---

## 核心地名

### 京城
- **朱雀大街**：京城主街，达官贵人聚居
- **裴府**：位于朱雀大街东段，裴砚之府邸
  - 含：正堂、东院（裴砚之起居）、西院（裴蕊起居）、外书房、内书房
- **沈府**：位于宣平坊，沈度府邸（二品官规格）
- **大理寺**：审案机构
- **御史台**：监察机构
- **诏狱**：刑部直属重犯监狱

### 外放 / 流放地
- **沧州**：沈家流放目的地，距京城八百里
- **潞州**：途经大城，沈岁晚在潞州展开自救
- **雁门关**：边境要塞

## 关键人物

### 女主
- **沈岁晚**（17岁）
  - 沈家嫡女，行三
  - 容貌：鹅蛋脸，柳叶眉，肤白，身量纤细
  - 性格：重生后表面温婉内心吐槽机，前世有"恋爱脑"，重生后清醒冷厉
  - 特征：左手腕有一颗小痣（前世与裴砚之相关）
  - 前世记忆节点：承安十二年腊月全家被抄 → 流放途中病死 → 十年后沈家平反，但她已死

### 男主
- **裴砚之**（26岁）
  - 裴家嫡长子，权倾朝野的权臣
  - 官职：御史大夫（从二品）+ 帝师（教导太子）
  - 容貌：清冷，凤眼，左眉尾一道浅疤（少年时伤），常年穿玄色
  - 性格：表面冷厉寡言，实则心思缜密，重情义
  - 旧伤：左手曾中箭伤筋，不能剧烈用力（不外露）
  - 与沈家关系：沈度的远房外甥（母系），两家来往不多
  - 前世：承安十三年曾为沈家上书被贬岭南，五年后才回京

### 沈家
- **沈度**（48岁）：女主父，礼部尚书（二品），为官清正
- **柳氏**（42岁）：女主母，出身书香门第
- **沈怀瑾**（22岁）：女主大哥，少将军（云麾将军），掌北境军
- **沈怀瑜**（19岁）：女主二哥，进士及第，翰林院编修
- **沈家小妹**（7岁）：沈岁宁，全家宠爱的幺女

### 裴家
- **裴蕊**（18岁）：裴砚之亲妹，活泼明朗
- **裴老夫人**（60岁）：裴砚之之母，沈家柳氏的远房表姐（重要！）

### 盟友 / 中立
- **萧蕴**（18岁）：女主闺中密友，皇后侄女（实际与宋皇后不合）
- **卫临**（25岁）：裴砚之亲随，武艺高强
- **老夫人身边的嬷嬷**：宋妈妈

### 对手
- **赵琅**（30岁）：御史中丞，构陷沈家主要推手
- **赵父赵谦**（55岁）：吏部侍郎，赵琅之父
- **三皇子齐王**：觊觎沈家军权
- **五皇子梁王**：表面中立

## 关键物件

### 沈家信物
- **沈家玉佩**（沈度佩）：传家之玉，沈岁晚贴身藏
- **柳氏绣帕**：柳氏亲手所绣梅花帕，前世裴砚之曾珍藏（重要伏笔）

### 裴家信物
- **裴砚之的私印**：刻"砚"字，朱文
- **半块双鱼佩**：裴家祖传，本是一对，裴老夫人当年送了一块给柳氏（重要伏笔）

### 翻案关键
- **赵琅密信**：构陷沈家的原版信，藏在大理寺某处
- **承安十二年腊月初三的起居注**：皇帝病中批阅奏折的真实记录

### 重要数字
- **三百两**：柳氏给沈岁晚的私房钱初始
- **一千两**：裴砚之每月给沈岁晚的零用（用度）
- **八百里**：京城到沧州流放地
- **五天**：流放队伍每日行进约四十里
- **三件大事**：抄家、流放、翻案

## 核心机制

### 重生记忆
- 沈岁晚保留前世全部记忆
- 但前世记忆模糊（细节不一定对），需要重新核实
- 男主不知女主是重生，但察觉她"不对劲"

### 信息差
- 女主知道：前世全部走向 + 男主真实身份（表兄）+ 男主曾救过沈家
- 男主知道：女主反常举动 + 似乎是冲着他来的 + 一言一行都像早就认识他
- 读者知道：女主的全部底牌

### 假婚约
- 表面：缔结婚约冲喜（柳氏身体抱恙）
- 实际：绑定裴家，让赵琅投鼠忌器

## 语感特征（3-5条）

1. **短句动作链**：「她猛地坐起」「翻身下地」「顺手抄起烛台」——动词领起，3-8字
2. **吐槽式内心独白**：古言版用「荒唐」「岂有此理」「简直没天理了」替代现代网梗
3. **反问链推进**：「你娶我？」「……」「你不信？」「……」「我手里有你要的东西。」
4. **感官细节**：嗅觉（药香/茶香）+ 触觉（手冷/手温）+ 视觉（玄衣/凤眼）
5. **微物传情**：帕子、印章、半块玉佩、一盏冷茶——具体物件承载情感
6. **反差萌**：冷面权臣背地里偷偷给未婚妻买糖、塞手炉
7. **延迟满足**：情感高潮前3-5步层层递进（女主先定亲→再同居→再共患难→再表明心意）

## 写作铁律（闻栖风格）

- 章名 5-8 字，口语化
- 开头：对话/声音/感官开场（前3章用对话开场）
- 结尾：悬念/动作/情感断章，不能用句号收尾
- 每章 1-2 个核心事件
- 信息投放不超过 30% 章节篇幅
- 反模式：不用「此外」「然而」「因此」「必然」「显然」「无疑」「微微」「轻轻」「缓缓」
- 长句不超 25 字，必须拆
- 心理描写不超 3 句，必用身体反应替代
- 经济数字具体到个位
- 章尾钩子轮换：悬念 → 甜蜜 → 反转
