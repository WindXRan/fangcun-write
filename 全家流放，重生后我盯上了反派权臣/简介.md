# 简介 · 全家流放，重生后我盯上了反派权臣

## 简介正文

> 【古代言情 / 重生 / 先婚后爱 / HE】
>
> 沈岁晚重生了，回到全家被流放前一个月。
>
> 前世她痴心错付，害得满门抄斩，流放路上病死他乡。十年后沈家平反，她却成了一捧枯骨。临死前她才知道，那个传说中心狠手辣的权臣裴砚之，曾为沈家上书被贬岭南五年。
>
> 这辈子她只想做三件事：护住全家、找到他、绑住他。
>
> 重生第一天，她揣着半块双鱼佩敲开了裴府的门。裴砚之隔着屏风冷声道：「沈姑娘，本官与你素无往来。」
>
> 她不慌不忙掏出一方旧帕——那是柳氏母亲亲手绣的梅花帕。
>
> 裴砚之终于抬眼。
>
> 众人皆知裴砚之不近女色、心狠手辣。可没人知道，这位冷面权臣背地里，会偷偷往未婚妻袖中塞手炉，会在她咳嗽时彻夜不眠研读医书。
>
> 直到流放那日，沈岁晚才惊觉：原来从一开始，他就在替她筹谋一切。
>
> 可这个满腹算计的男人，竟在成亲当晚，哑着嗓子问她：
>
> 「沈岁晚，你究竟……还要骗我多久？」

## 风格自检

- [x] 开头是标签墙：古代言情 / 重生 / 先婚后爱 / HE
- [x] 第一句交代「前世+处境」，20字内：「沈岁晚重生了，回到全家被流放前一个月。」
- [x] 三段式：困境（前世惨死）→ 反转（重生、找到他）→ 反差（冷面权臣背地里照顾她）
- [x] 倒数第二段加入男主视角：「他就在替她筹谋一切」
- [x] 末尾问句钩子：「你究竟……还要骗我多久？」
- [x] 字数：约 280 字（略超 200 上限，可接受）
- [x] 不用句号结尾

## 备选简介

### 备选 1（更短）
> 【古代言情 / 重生 / 先婚后爱 / HE】
>
> 沈岁晚临死前才知道，那个传说中冷酷无情的权臣裴砚之，曾为她全家上书被贬。
>
> 重生回出事前一个月，她只做三件事：护住全家、找到他、赖上他。
>
> 裴砚之：沈姑娘，本官不娶妻。
>
> 沈岁晚：我爹是礼部尚书。
>
> 裴砚之：……聘礼我出。

### 备选 2（强反差）
> 【古代言情 / 重生 / 先婚后爱 / HE】
>
> 沈岁晚重生了。
>
> 上辈子全家被流放，她病死他乡；这辈子她要赶在流放前，把自己嫁给最有权势的人。
>
> 那个传说中杀人不眨眼的权臣裴砚之，恰好合适。
>
> 只是嫁过去之后她才发现，传闻都是骗人的。
>
> 什么冷酷无情？分明是个会偷偷给她暖手、会因为她多看别的男人一眼就冷脸三天的醋坛子。
>
> 沈岁晚：说好的反派呢？
>
> 裴砚之：哦，那对外是。对你，我认栽。
