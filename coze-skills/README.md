# Coze Skills 赚钱计划

## 目标用户
网文作者（番茄小说、起点、晋江等平台）

## 三个核心 Skill

### 1. 小说封面生成器（story-cover-coze）
**功能**：根据书名、作者名、题材风格自动生成专业级网文封面
**变现方式**：按次收费（5-10元/张）或包月订阅（99元/月，50张）

### 2. 番茄排行榜分析（story-scan-coze）
**功能**：自动采集番茄小说排行榜数据，分析市场趋势，提供创作建议
**变现方式**：订阅制（199元/月，每日更新数据）

### 3. 小说仿写助手（story-rewrite-coze）
**功能**：根据爆款小说结构，生成新书大纲和章纲
**变现方式**：按项目收费（299元/本）或订阅制（399元/月，3本）

## 技术实现步骤

### 第一步：安装 Coze CLI
```bash
npm install -g @coze/cli
coze auth login --oauth
```

### 第二步：创建 Coze 项目
```bash
coze project create story-tools
```

### 第三步：开发 Skill
1. 封面生成器：调用 GPT-Image-2 API
2. 排行榜分析：爬虫 + 数据分析
3. 仿写助手：基于现有 story-rewrite 引擎

### 第四步：部署和测试
```bash
coze deploy
```

### 第五步：发布到技能商店
通过 Coze Web 端申请上架
