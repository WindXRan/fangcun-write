# 第3章 深度拆解

## 基本信息
- 章节编号：第3章
- 章节标题：惊险的遭遇
- 字数：2293字
- 基调：轻松→紧张
- 主题标签：爱情、日常

## 章节概要
程枫给何柔清转了调理费。工作中与同事江敏互动，江敏请他吃饭。餐厅附近江敏被醉汉骚扰，程枫出手相救。初恋宋子怡发来见面邀请，程枫未回复。

## 情节点

P1 【过渡】程枫给何柔清多转几百块调理费，何柔清收了但没回复【情感铺垫】
P2 【内心独白】程枫抱怨"酒精害人"，钱花光了，工作不在状态【人物塑造】
P3 【日常】工作中，江敏突然出现在面前，程枫误触她手【日常/暧昧】
P4 【人物揭示】江敏害羞慢热，比程枫大两岁，是岗前培训组长【人物塑造】
P5 【日常】程枫撩江敏："敏姐这样的人美心善大姐姐，不撩都是一种罪过"【暧昧】
P6 【信息揭示】江敏提醒程枫欠她一顿饭，约定下班后请客【日常】
P7 【信息揭示】程枫收到初恋宋子怡的见面邀请，内心纠结但未回复【感情线铺垫】
P8 【场景转换】下班后，江敏换了碎花连衣裙，坐程枫的车去吃饭【日常/暧昧】
P9 【日常】小餐馆吃饭，江敏喝酒上脸，程枫担心她【日常/温馨】
P10 【暧昧】江敏酒后主动挑逗程枫，摸他喉结，程枫反而害羞【暧昧/反转】
P11 【危机】江敏上厕所被醉汉骚扰，程枫赶到踹倒醉汉，拉着江敏跑【紧张/冲突】
P12 【温馨】程枫送江敏回家，安抚她，开玩笑说"喊我名字更管用"【温馨/收尾】
P13 【悬念】宋子怡再次发信息"你是在躲着我吗？"——程枫仍未回复【感情线悬念】

## 写法技巧
1. **多线铺垫**：一条线是何柔清的余波，一条线是江敏的暧昧，一条线是宋子怡的回归
2. **英雄救美**：程枫踹醉汉救江敏，展现男主担当
3. **角色反转**：平时是程枫撩江敏，酒后江敏反撩程枫
4. **悬念收尾**：宋子怡的"你是在躲着我吗？"制造感情线悬念
