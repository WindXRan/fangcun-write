# 背景设定

好的，网络小说结构分析师已就位。基于您提供的章节摘要，我已提取出以下世界观设定：

## 核心设定
- **末世类型**：极寒末世。环境温度极低（零下七度已算温暖），生存环境恶劣，以对抗寒冷为核心挑战。
- **时间线**：末世初期，第三次寒潮即将来临。人类尚在适应与挣扎，秩序尚未完全崩溃，但已显露混乱迹象。
- **核心规则**：
    - **生存指数**：由体质、战力等数值综合决定，是衡量幸存者生存能力的关键指标。指数过低者难以生存，且被视为“冒险者”。
    - **光脑系统**：每个幸存者都配备光脑，用于交易、查看排行榜、通讯等。光脑可被融合，继承死者数值，这已成为公开的秘密，是引发“杀人夺宝”的核心规则。
    - **交易区**：每月开启一次，每次一小时，效率低下，促使线下交易和掠夺行为。

## 力量体系
- **生存指数**：以数值衡量，包括**体质**（决定抗寒能力）、**战力**（决定战斗能力）等。数值提升可通过服用特殊物品（如体质药剂）或融合他人光脑实现。战力榜是公开的实力排名。
- **交易系统**：以**交易币**为硬通货，通过交易区（官方渠道）或线下交易获取。物品序列号（如体质药剂）可追溯转手次数与属性。
- **势力等级**：存在区域性的基地（如幸福基地）和战力榜上的顶尖强者（如余游）。强者拥有话语权，弱者易被掠夺或剥削。

## 地理信息
- **主要地点**：**第八试验区**（故事发生区域），包含**泉水城**（区域频道名称）等行政区划。
- **避难所**：幸存者自行搭建的临时居所，通常为帐篷等简易设施，需进行保温、隐蔽和防御改造。避难所的隐蔽性是生存关键。

## 金手指设定
- **类型**：观察推理 + 信息差 + 信息利用
- **来源**：主角自身谨慎、冷静、善于分析的个性，以及跟随“重生者”（王大伟）的行动轨迹，提前获取关键信息（如体质药剂的稀有性、寒潮时间表、光脑融合秘密）。
- **限制**：
    - **信息不完全**：主角仅能通过观察和分析推导出部分真相（如体质药剂的价值、余游的虚弱状态），无法预知所有细节。
    - **信息滞后**：主角需要时间验证和利用信息差，存在被敌人抢先或反制的风险。
    - **信息风险**：正确信息的获取（如光脑融合）可能将自己置于危险境地，引来他人觊觎。
