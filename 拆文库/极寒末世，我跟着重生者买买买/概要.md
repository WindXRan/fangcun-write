# 概要

- 书名：极寒末世，我跟着重生者买买买
- 作者：雪大

- 总章数：39章

## 故事梗概（200字 thin first-pass）

姜清越是个水产店老板，发现邻居王大伟行为异常：突然大方购物、疯狂健身、大量囤货。她推断王大伟是重生者，末世即将来临。

姜清越果断卖房贷款，跟着王大伟买买买，物资翻倍囤。末世降临后，她发现是极寒末世，气温骤降。前男友叶宏宇发小作文曝光她的住址，迫使她离开小区。

姜清越找到父亲留下的避难所，开始在极寒中生存。她遇到一只神奇的兔子姜清宴，建立庇护所，与其他幸存者交易物资。

随着寒潮一轮轮升级，姜清越不断强化自己的生存能力，最终成为势力榜第一。

## 核心设定
- 金手指：观察推理 + 信息差利用（跟随重生者囤货）
- 主角身份：水产店老板，末世小说爱好者
- 核心循环：观察异常 → 推理判断 → 模仿囤货 → 应对危机
- 情绪基调：紧张、生存、谨慎

## 主要角色
- 姜清越：主角，水产店老板，善于观察和推理
- 王大伟：邻居，重生者，姜清越的信息来源
- 叶宏宇：前男友，反派，发小作文曝光姜清越
- 姜清越的兔子：神奇动物伙伴
