# 金手指：《西游：这是谁把金丹放我府上的？》

## 无名手札

| 项目 | 内容 |
|------|------|
| 类型 | artifact（残缺功法道具） |
| 获取 | Ch2在废旧卷宗堆中发现，无封皮、无署名，写在发黄草纸上 |
| 外观 | 发黄草纸，字迹如醉鬼所划，撇如刀捺如剑 |
| 核心理念 | "截"字诀——截取乱窜灵气，强行斩断暴躁，只留生机 |
| 与正统对比 | 道家正统是"顺势而为"，手札是"逆势截取" |

## 能力演进

1. **首次触发**（Ch3）：陈微服火云丹，木属功法压不住火气。手札无风自动，传授"截"字诀
2. **后续**：每当陈微遇到修炼瓶颈，手札自动翻页提供功法

## 火云丹

| 项目 | 内容 |
|------|------|
| 来源 | 火德星君的封口费，装在火髓玉葫芦中 |
| 数量 | 3颗 |
| 效果 | 火部特供，专补根基。陈微服1颗+百年蟠桃，从散仙五转突破至九转 |

## 百年蟠桃

| 项目 | 内容 |
|------|------|
| 来源 | 与蟠桃园土地张福德交易所得（作为平账的"样本"） |
| 价值 | 一百年修为，比丹药温和无副作用 |

## 孙悟空毫毛

| 来源 | 后期孙悟空赠送 |
|------|--------------|
| 用途 | 保命底牌，陈微去灌江口见杨戬时贴身携带 |

## 金手指评价

本书金手指设计极其克制：
- 无名手札不是万能系统，只是偶尔提供功法的道具
- 火云丹和蟠桃都是"交易"得来，不是白捡的
- 主角真正的金手指是**读懂规则的能力**——所有资源都是靠制度博弈换来的
