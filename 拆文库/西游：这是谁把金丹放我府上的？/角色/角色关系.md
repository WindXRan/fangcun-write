# 角色关系：《西游：这是谁把金丹放我府上的？》

## 核心关系网络

### 陈微 ↔ 王灵官
- **关系**：职场上下级→对立→被反制
- **演变**：Ch1王灵官将陈微贬去废旧处→Ch9试图让陈微背锅→Ch100后陈微权力反超王灵官

### 陈微 ↔ 火德星君
- **关系**：封口费交易→政治同盟
- **演变**：Ch1陈微写报告救场→Ch2火德星君送火云丹→后期成为政治支持者

### 陈微 ↔ 太白金星
- **关系**：伯乐与千里马
- **演变**：Ch9-10陈微的备忘录打动太白→获得赏识→成为陈微最重要的政治靠山

### 陈微 ↔ 张福德（蟠桃园土地）
- **关系**：交易伙伴→长期利益共同体
- **演变**：Ch5以"样本"名义交易百年蟠桃→打开蟠桃园利益通道

### 陈微 ↔ 赵书吏
- **关系**：对照角色（同期入职）
- **演变**：Ch3赵书吏抢占陈微的文书监位置→Ch7炫耀→Ch10后逐渐被陈微碾压

### 陈微 ↔ 杨婵
- **关系**：爱情/政治联姻
- **演变**：中期建立关系→通过杨戬考验→成为陈微最重要的政治盟友和情感支撑

### 陈微 ↔ 杨戬
- **关系**：考验者→认可者
- **演变**：Ch240杨戬刀架脖子→最终认可陈微

### 陈微 ↔ 孙悟空
- **关系**：不直接互动/变数催化剂
- **演变**：Ch7陈微听说"野猴子"→预测其为"搅动死水的棍子"→后期成为重要盟友

## 权力网络

```
    │

    │           │

    │           │
内务府线      稽查院线
```
