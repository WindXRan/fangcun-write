# 第3章 深度拆解：蟠桃园的烂账

**字数**：约 2,132 字 | **核心事件**：陈微吞服火云丹突破至散仙七转，发现蟠桃园20枚桃子账面亏空，被上司甩锅

## 结构拆解

| 段落范围 | 功能 | 字数 | 说明 |
|----------|------|------|------|
| 前情回收 | 过渡 | ~200 | 换领办公耗材，遇旧相识赵书吏被冷落 |
| 换领办公耗材+遇赵书吏 | 铺垫（社会冷暖） | ~400 | 天庭"人走茶凉"的社会规则展示 |
| 服火云丹突破 | 爽点（修炼） | ~500 | 散仙五转→七转，无名手札功法救场 |
| 蟠桃园废档送达 | 转折 | ~200 | 新任务：归档蟠桃园废卷 |
| 发现亏空 | 冲突 | ~400 | 20枚蟠桃对不上账 |
| 上报后遭甩锅 | 冲突升级 | ~400 | 上级回复："由你核查处理"——把黑锅焊死在陈微背上 |

| 项目 | 内容 |
|------|------|
| 类型 | 获得机缘（修为突破） |

## 可借鉴要素

1. **连续机缘释放**：Ch1得手札→Ch2发现丹药→Ch3服丹突破，三章构成完整循环
2. **双重冲突叠加**：修炼突破（内）+官僚甩锅（外），内外交困增加张力
3. **职场潜规则展示**：上级回复"由你核查处理"——典型的职场甩锅话术
4. **无名手札第二次出现**：建立"手札=救命底牌"的预期
