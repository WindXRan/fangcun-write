# 故事线：《西游：这是谁把金丹放我府上的？》

## 故事框架

| 项目 | 内容 |
|------|------|
| 框架类型 | 升级流（制度权力升级） |
| 核心驱动 | 陈微从天庭底层通过制度博弈攀升至权力顶峰 |
| 主轴矛盾 | 个人能力 vs 体制固化——陈微用制度武器打破阶层壁垒 |
| 升级机制 | 散仙→天仙（编制获得）→稽查院长→司法长史→天庭二品 |
| 叙事节奏模式 | 危机→翻条例→写公文→反转→新对手出现 |

|------|------|
| 类型 | 感情线 |

## 副线2：西游事件线

| 项目 | 内容 |
|------|------|
| 类型 | 外部催化线 |
| 功能 | 提供体制裂缝（"风浪越大鱼越贵"） |
| 核心事件 | 石猴出世→大闹天宫→取经事件 |
| 对主线影响 | 每次西游大事件都打破天庭的阶层僵化，为陈微创造上升窗口 |

## 故事线关系

- 主线和感情线：交织——杨婵关系既是感情线也是政治线
- 主线和西游线：并行——西游事件提供外部变数，陈微在其中捕捉机遇
- 感情线和西游线：独立
