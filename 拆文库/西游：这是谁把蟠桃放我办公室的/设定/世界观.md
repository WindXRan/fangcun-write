# 世界观：《西游：这是谁把蟠桃放我办公室的》

## 力量体系

| 等级 | 描述 | 社会地位 |
|------|------|---------|
| 人仙 | 飞升初期 | 施工员 |
| 天仙 | 有编制 | 副经理/经理级 |
| 元仙 | 中高层 | 司长/主任级 |
| 金仙 | 高层 | 部门主管级 |
| 大罗金仙 | 顶级 | 杨戬、观音级 |
| 准圣 | 天花板 | — |

苏元的升级路径：人仙初期（施工员）→天仙初期（副经理）→天仙巅峰（经理）→金仙初期（司长）

## 世界结构

- **凌霄殿**：玉帝朝会+天庭最高权力中心
- **基建处**：天庭建设项目管理部门
- **南天门项目部**：苏元的主场
- **通明殿**：天庭行政中枢
- **监察七司**：苏元后期任职的监察机构
- **雷部**：执掌天劫的军事/执法部门
- **兜率宫**：太上老君
- **瑶池**：王母势力范围
- **财神府**（赵公明、曹宝）：财政审批
- **灌江口**（杨戬道场）
- **西行路线**：车迟国、通天河等

## 关键势力

- **文官体系**：太白金星、许天师、文曲星君——派系林立
- **武将体系**：李靖派系（巨灵神、药叉将）、魔家四将、天蓬元帅
- **佛教暗线**：文殊菩萨（代号"峨眉峰"）、金吒
- **瑶池体系**：王母、七仙女、洪锦
- **财神体系**：赵公明、曹宝、萧升——管钱的神仙

## 核心规则

1. **修为与职务双轨制**：修为提升和职务晋升必须匹配，修为高了职务跟不上会"明升暗降"
2. **灵石即一切**：灵石既是货币又是修行资源（氪金系统）
3. **天庭=公司**：天庭被彻底翻译为集团公司——有项目经理、综合办主任、绩效考核、百日攻坚、新闻稿
4. **西游事件=商业机会**：每次大事件都是套利窗口
