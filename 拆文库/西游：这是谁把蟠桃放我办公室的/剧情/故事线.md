# 故事线：《西游：这是谁把蟠桃放我办公室的》

## 故事框架

| 项目 | 内容 |
|------|------|
| 框架类型 | 多线交织（职场+间谍+意识形态） |
| 核心驱动 | 苏元利用天庭制度漏洞将灾难转化为商业机会 |
| 主轴矛盾 | 个人财富权力积累 vs 天庭/佛门/瑶池多方博弈 |
| 升级机制 | 修为（氪金）+ 职务（运作）+ 影响力（思想传播）三维并进 |
| 叙事节奏模式 | 发现商机→设计方案→上下打点→利益最大化→善后→下一次 |

## 主线：平账帝国

| 阶段 | 章节 | 核心操作 | 收益 |
|------|------|---------|------|
| 南天门一期 | 1-30 | 竣工前被砸→凌霄殿三请→拿到8亿 | 3亿经费+预付款5亿 |
| 南天门二期 | 31-80 | 再次修建→再次被砸→再次申报 | 累计数亿灵石 |
| 蟠桃园项目 | 81-200 | 抢占蟠桃园土地位置→操控蟠桃分配 | 资源+政治筹码 |
| 情报帝国 | 201-350 | 操控佛道暗线→监察七司权力扩张 | 信息优势+多方进贡 |
| 西行传播 | 351-548 | 随取经路扩散思想→建立意识形态影响力 | 成为"苏师" |

## 副线1：佛门暗线

| 项目 | 内容 |
|------|------|
| 类型 | 间谍线 |
| 核心角色 | 文殊菩萨（峨眉峰）、金吒、苏元（佛龛） |
| 机制 | 苏元以"佛龛"代号操控佛门情报网，同时在天庭获取佛门动向 |
| 交织 | 与主线交织——佛门情报帮助苏元在天庭博弈中占优 |

## 副线2：瑶池对抗

| 项目 | 内容 |
|------|------|
| 类型 | 势力冲突线 |
| 核心角色 | 王母、洪锦 |
| 起始 | 中期（苏元收孙婆婆、暴打洪锦后与瑶池交恶） |
| 机制 | 拉拢李家（金吒/木吒）对抗瑶池，形成多方制衡 |

## 副线3：西行取经线

| 项目 | 内容 |
|------|------|
| 类型 | 外部事件线+意识形态传播 |
| 核心角色 | 孙悟空、天蓬、金吒（后期随行） |
| 机制 | 取经路途经过的车迟国→通天河→沿线传播苏元的思想和方法论 |

## 故事线关系

- 主线+佛门线：交织（佛门情报支持主线操作）
- 主线+瑶池线：对抗（利益冲突驱动主线博弈）
- 主线+取经线：后期融合（苏元本人参与取经，沿途传播思想）
- 佛门线+取经线：独立/交叉（取经是佛门的项目，但苏元在其中加入自己的议程）

## 标志性操作

1. **一座南天门吃三回**（Ch2）：全书最核心的商业模型
2. **百万送礼换八亿拨款**（Ch4-5）：ROI达800倍的经典操作
3. **远程操控金吒**（Ch100）：身份套娃的巅峰
4. **保洁阿姨破阵**（Ch200）：搞笑揭示"修为≠安全"
5. **灵感大王成为苏师门徒**（Ch548）：苏元影响力的"去中心化传播"
