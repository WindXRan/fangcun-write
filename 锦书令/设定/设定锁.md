# 设定锁（一级锁定，所有 agent 共享）

## 核心地名
- **锦州**：江南水乡，女主生长的城池。水路发达，商贾云集
- **京城长安**：大燕都城，权谋中心
- **定北关**：大燕与北狄边境要塞，男主驻守之地
- **青云巷**：锦州城中一条窄巷，女主在此租屋开代笔铺子
- **南风渡**：锦州城外最大的渡口

## 关键物件
- **代笔簿**：女主替人写每份文书都留底存档的簿册，她最重要的自保工具
- **青竹笔**：父亲留下的遗物，女主最珍视的毛笔
- **定北军令牌**：男主给女主的信物，可在军中通行
- **盐铁账册**：揭开贪腐案的关键证据

## 人物关系锁

### 沈家
- **沈锦书**（女主）：24岁，代笔先生。父亲曾是衙门书吏，精通律法文书。母亲早逝，父女相依为命。父亲三年前病故后，她接替父亲营生。性格沉稳、心思缜密、外柔内刚。
- **沈敬之**（女主之父）：生前为锦州府衙书吏，为人正直，精通律法。三年前病故。

### 裴家
- **裴衍之**（男主）：28岁，定北侯世子，镇北大将军。镇守定北关七年，战功赫赫。性格冷峻、不苟言笑，但内心有温度。
- **定北侯**（男主之父）：老侯爷，坐镇京城。
- **裴老夫人**（男主祖母）：慈祥明理

### 核心人物
- **陆清和**（男二）：锦州知府之子，温润如玉。曾对沈锦书有意，但因家族阻碍未果。
- **赵师爷**（男主的幕僚之首）：四十余岁，心思缜密，最初看不起女主，后心服口服。
- **春兰**（女主侍女）：十六岁，机灵忠心的丫头，无父无母被女主收留。
- **江望之**（反派）：吏部侍郎，军粮贪腐案主谋，伪善阴狠。
- **小皇帝**：年仅十二岁登基，太后垂帘听政。
- **太后**：小皇帝的生母，心思深沉，为保皇位不择手段。
- **端王**：先帝幼弟，野心勃勃，觊觎皇位。

## 背景事件锁
- 三年前，老燕帝驾崩，十二岁幼子继位，太后垂帘，朝局不稳
- 定北军镇守边境，连年与北狄交战，军粮消耗巨大
- 朝中党争激烈：太后党、端王党、保皇党三派角力
- 军粮贪腐案：有人克扣军粮中饱私囊，以次充好，导致边军补给困难
- 女主父亲生前曾察觉锦州府衙账目有问题，留有手札
