=== SECTION: world_rules ===

## 世界规则

**时代背景**：现代都市，真实城市体系（海市 = 豪门商圈，青城 = 偏远小城）

**社会结构**：
- 上层：豪门世家（楚家 = 宴金集团，掌控商业帝国）
- 中层：小公司职员（月薪3500-5000）、售楼销售
- 底层：外卖骑手、出租屋生活

**核心设定**：
- 穿书机制：女主穿越到正在看的小说中，成为恶毒女配
- 原书剧情：虐文女主重生文，原主假冒失忆总裁女友
- 信息差：女主知道原书剧情，男主不知道自己身份
- 时间线：男主被找到前还有不到一年

**经济细节**（高度具体）：
- 出租屋月租1400，水电约1500/月
- 男主工资+外卖月入5000-7000
- 男主自留500生活费，剩余给女主
- 女主余额1000多，欠网贷2万多
- 二手电脑最低3000+
- 一份外卖20多，买菜20多能吃一天
- 售楼保底工资1500/月

## 逐章情节骨架

### 第1章 穿、穿书了？
1. **核心事件**：宋云绯穿越醒来→接收原主记忆→了解原书剧情→决定跑路
2. **情绪弧线**：懵→震惊→绝望→冷静分析→"先吃饭吧"
3. **钩子**：楚靳寒洗完澡坐在床上，暗示要"履行男友义务"
4. **角色动态**：宋云绯从懵到接受现实；楚靳寒冷漠但默默做事
5. **转折点**：发现穿到"绝食饿死"的节骨眼上
6. **节奏标记**：铺垫（建立困境+世界观）

### 第2章 再不满足你，我怕你明天上吊
1. **核心事件**：楚靳寒主动上床→宋云绯推倒他→化解危机→发现藏的套
2. **情绪弧线**：紧张→尴尬→慌张→松口气→自嘲
3. **钩子**：楚靳寒藏套=心里厌恶；宋云绯决定跑路
4. **角色动态**：楚靳寒"被迫履行义务"→被推倒后困惑；宋云绯急中生智化解
5. **转折点**："没让你睡我"→确立新边界
6. **节奏标记**：发展（亲密接触+人设展示）

### 第3章 查岗哥
1. **核心事件**：楚靳寒上班→胡瑶八卦查岗→宋云绯不查岗引发疑惑
2. **情绪弧线**：日常→八卦→疑惑→好奇
3. **钩子**：宋云绯为什么不查岗了？楚靳寒开始关注
4. **角色动态**：胡瑶挑拨→楚靳寒冷怼；宋云绯反常→楚靳寒在意
5. **转折点**：信息差反转——以前是女主查岗，现在是男主关注女主
6. **节奏标记**：发展（外部世界+信息差）

### 第4章 有觉悟的男人
1. **核心事件**：宋云绯买菜做饭→楚靳寒观察反常→宋云绯找工作→两人吃饭
2. **情绪弧线**：日常→温馨→微妙→敷衍→自嘲
3. **钩子**：宋云绯为什么突然变勤快？楚靳寒狐疑加深
4. **角色动态**：宋云绯主动改变→楚靳寒被动观察；"不胖""很好"=敷衍式关心
5. **转折点**：宋云绯说要上班→楚靳寒意外
6. **节奏标记**：发展（日常+关系微妙变化）

### 第5章 哪里受得了这个
1. **核心事件**：楚靳寒送外卖→宋云绯算账→睡觉时八爪鱼缠人→楚靳寒去浴室
2. **情绪弧线**：算账焦虑→困倦→暧昧→尴尬→自嘲
3. **钩子**：楚靳寒身体反应=在意的信号；宋云绯无意识亲密
4. **角色动态**：宋云绯无意识撩人→楚靳寒克制→浴室冷静
5. **转折点**：八爪鱼缠人→身体先于意识在意
6. **节奏标记**：发展（亲密+喜剧）

### 第6章 邪教仪式
1. **核心事件**：宋云绯入职售楼部→张涛暗示用美色→了解工作规则
2. **情绪弧线**：紧张→无语→警惕→吐槽→接受
3. **钩子**：张涛暗示=潜在冲突线；三个月开单压力
4. **角色动态**：宋云绯进入新环境→面对职场潜规则
5. **转折点**：张涛暗示用美色→建立职场冲突
6. **节奏标记**：铺垫（新冲突线+经济压力）

### 第7章 这算不算冷暴力？
1. **核心事件**：胡瑶八卦查岗→楚靳寒冷怼→宋云绯约商场抽奖→楚靳寒失神
2. **情绪弧线**：八卦→冷怼→好奇→期待→失神
3. **钩子**：楚靳寒摸手机看消息=在意；宋云绯约他=关系推进
5. **转折点**：楚靳寒主动看手机→从被动到在意
6. **节奏标记**：发展（信息差+关系推进）

## 文风特征摘要

### 叙事视角
- 第三人称限知视角，交替聚焦女主/男主
- 内心独白用吐槽式口语（"怎么感觉进了传销""人都要裂开了"）

### 节奏特征
- 短句动作链：3-8字独立成句，句号断开
- 长短交替：动作用短句，描写用长句（15-25字）
- 每章2个核心事件（双事件节拍）
- 章尾必有钩子（悬念/情感/动作）

### 对话特征
- 男主极简："嗯""很好""走吧"
- 女主口语化+网络梗："悬着的心终于死了""狗看了都摇头"
- 反问链推进对话深度
- 对话有潜台词层（说A想B）

### 情感表达
- 具体物件/金额代替抽象情感（50块、三菜一汤）
- 言行不一：嘴上说A，行动做B
- 环境映射情绪（烈日+蝉鸣+风扇=孤独闷热）
- 微表情代替语言（喉结滚动、睫毛颤抖）

### 禁用词/反模式
- 不用：此外、因而、综上所述、必然、显然、无疑
- 不用：微微、轻轻、缓缓
- 不用：俊美无俦、说时迟那时快
- 不用：超过25字的长句
- 不用：心理描写超过3句不打断
