# 逆推法·核心方法

**核心理念：** 不需要人类总结写作规则。读20章原文，提取统计特征，直接编译进 prompt。统计特征不会骗人，也不会遗漏。

---

## 一、方法流程

```
选择一本质量确认的原文（朱雀100%绿）
    ↓
读20章（均匀分布在全书前中后）
    ↓
提取每章的硬指标：
    字数、行数、句数、段落数
    OS条数、OS字数占比
    感叹号数、问号数、省略号数、破折号数
    每行平均字数、短行(<15字)占比、长行(>50字)占比
    ↓
聚合20章数据，找出稳定值
    ↓
稳定值直接编译进 write-chapter / fanxie-chapter 的写作红线
    ↓
稳定值同时进入 checker.py 的检测规则
```

## 二、哪些特征是"通用的"

从统计中提取的特征分为两类：

### 硬规则（20章中0偏差）

这些特征在20章中一致出现，可以写入规则"违反即修"：

| 特征 | 统计值 | 用法 |
|------|--------|------|
| 省略号 | 20章共0次 | 禁止使用 |
| 分号 | 20章共0次 | 禁止使用 |
| 每章字数 | 2000-2500字 | 超出即超长，必须压缩 |
| 每行字数 | 25-35字 | 超过35字的行切开 |
| 段落句数 | 1-2句/段 | 超过3句的段落拆开 |
| 长行占比 | <5% | 超过50字的行砍半 |

### 量化基准（20章统计的稳定值）

这些特征有波动但范围可控，写入规则"参考对齐"：

| 特征 | 统计范围 | 用法 |
|------|---------|------|
| OS密度 | 0-33%（按章节功能浮动） | 情绪章20-33%，事件章0-11% |
| 感叹号密度 | 每章13-54个 | OS必用感叹号收尾 |
| 问号密度 | 每章2-18个 | 正常使用 |
| 短行占比 | 8-21% | 每10行约1-2行极短行，用于强调 |
| 破折号 | 每章0-7次 | 极少用，引出关键信息时 |

### 非特征（统计波动过大，不作为规则）

- 对话占比（波动大，取决于章节场景）
- 具体用词偏好（不同书不同风格）
- 情绪弧线类型（不同章节不同功能）

## 三、分析工具

```bash
# 统计20章的硬指标
python3 << 'PYEOF'
import re, os
from pathlib import Path

d = "projects/全家偷听心声/正文/正文"
pick = [1,2,3,4,5,10,15,20,25,30,35,40,45,50,60,70,80,90,100,150]

for p in pick:
    for f in sorted(os.listdir(d)):
        if not f.endswith('.xml') or not f.startswith(f'第{p}章') or len(f) < 15:
            continue
        text = Path(f"{d}/{f}").read_text(encoding='utf-8')
        m = re.search(r'<content>(.*?)</content>', text, re.DOTALL)
        if not m: continue
        body = m.group(1).strip()
        lines = [l.strip() for l in body.split('\\n') if l.strip() and not l.strip().startswith('第')]
        full = '\\n'.join(lines)
        
        print(f"Ch{p}: {len(full)}字 {len(lines)}行 {full.count('！')}！ {full.count('？')}？ "
              f"OS{len(re.findall(r'【.*?】',full))}条 "
              f"……{full.count('……')} ——{full.count('——')} "
              f"均{round(sum(len(l) for l in lines)/len(lines),1)}字/行")
        break
PYEOF
```

## 四、如何确定"读多少章够"

| 目标 | 最少读 |
|------|--------|
| 标点规则 | 5章即可（标点极端稳定） |
| 行级节奏 | 10章（需覆盖不同章节类型） |
| OS密度规律 | 15章（需覆盖情绪章和事件章） |
| 完整统计 | 20章（覆盖全书前中后） |

20章后新的统计特征趋于稳定，继续读的边际价值递减。

## 五、逆推法的适用范围

| 场景 | 适用 | 说明 |
|------|:----:|------|
| 仿写（已有原文） | ✅ | 有原文直接统计 |
| 原创（无原文） | ❌ | 无统计对象 |
| 同作者多本书 | ✅ | 可提取该作者的通用特征 |
| 跨题材 | ✅ | 标点/行级规则跨题材通用 |
| 跨平台 | ⚠️ | 番茄版和晋江版的字数/节奏不同 |
