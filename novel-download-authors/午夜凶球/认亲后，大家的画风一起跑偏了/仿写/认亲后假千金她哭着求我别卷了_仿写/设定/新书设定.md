# 新书设定

## 书名候选
1. 《认亲后，假千金她哭着求我别卷了》（推荐）
2. 《回豪门后，我靠嘴炮征服全家》
3. 《被抱错十七年，亲爹跪求我回家》

## 类型定位
- 分类：青春甜宠
- 标签：现代言情|校园|豪门|真假千金
- 核心赛道：甜宠+爽文+家庭温情

## 核心卖点（保留的成功因子）
| 源文成功因子 | 新书保留方式 |
|---|---|
| 欢喜冤家CP互怼 | 从陌生对立到互怼暧昧，CP粉自发磕糖 |
| 女主毒舌怼人爽感 | 市井出身的女主更泼辣，怼人更狠更爽 |
| 家庭温情日常 | 养母面馆日常+亲生家庭破冰，双线温情 |
| 身份反转（抱错千金） | 保留核心设定，改为市井→豪门的阶层冲击 |
| 校园青春群像 | 贵族学校+原学校双线，同学互动更丰富 |
| 青梅竹马羁绊 | 改为市井竹马+豪门世家男，三角张力更强 |
| CP粉起哄名场面 | 贵族学校八卦氛围更浓，CP粉戏份加重 |
| 父女亲情（女儿奴） | 养父温情线+亲生父亲破冰线，双倍催泪 |

## 人物设定（与源文差异化）

### 女主：沈念初
- 身份：被抱错的真千金，养母是面馆老板娘
- 背景：从小在南城老街长大，白天上学晚上帮忙端盘子收账，练就察言观色的本事和一张利嘴。高三前暑假被告知身世真相，被接回苏家
- 性格：表面飒爽大方，实际护短记仇。嘴上不饶人但从不主动欺负人，谁惹她她怼谁，怼到对方怀疑人生
- 目标：高考考回养母身边，顺便把假千金的真面目拆穿
- 差异点：源文女主是小康家庭温柔女神路线（外柔内刚）；新书女主是市井泼辣小辣椒（外刚内也刚），更有攻击性和烟火气

### 男主：顾深
- 身份：顾氏集团独子，苏家世交，假千金苏婉宁的"未婚夫"
- 背景：从小被当作接班人培养，待人温和有礼是刻进骨子里的教养。跟苏婉宁一起长大，两家默认联姻，他无所谓，觉得娶谁都一样
- 性格：表面温润如玉、谦和有礼，实际冷淡腹黑。对不感兴趣的人连多看一眼都懒得，但一旦认定就占有欲极强
- 标签：白切黑|腹黑|占有欲|世家公子
- 差异点：源文男主江流是阳光直球型（嘴贱但坦荡）；新书男主顾深是温润腹黑型（笑着把人卖了还让人替他数钱），反差感更强

### 关键配角
- **苏婉宁**（假千金）：表面楚楚可怜引人同情，实际心机深沉，暗中排挤沈念初。与源文胆小好欺负的假千金完全相反
- **沈念初养母·张秀兰**：面馆老板娘，泼辣爽利，视念初如命。知道真相后强忍不舍送她回苏家，是全书最催泪角色
- **陆嘉树**（市井竹马）：念初从小一起长大的发小，烧烤摊的儿子，暗恋念初多年。憨厚直球，是念初在南城的退路和底气
- **苏子轩**（亲弟弟）：叛逆初中生，一开始排斥沈念初，后来被她"收服"，变成护姐狂魔
- **苏明远**（亲生父亲）：地产集团董事长，威严寡言，对沈念初愧疚但不知如何表达，笨拙父爱
- **顾深母亲·周雅芝**：贵妇人，看不起沈念初的出身，暗中撮合顾深和苏婉宁
- **赵萌萌**（CP粉头子）：贵族学校八卦女王，第一时间锁定沈念初×顾深CP，自发组建后援会

## 世界观
- 时代背景：现代都市，2020年代
- 地理：南城老街（市井生活区）↔ 北城别墅区（豪门聚集地），两个世界形成强烈对比
- 势力格局：苏家（地产）× 顾家（金融）× 陆家（市井），三线交织
- 核心设定：
  - 苏家是南城地产龙头，表面光鲜内部暗流涌动，苏婉宁的亲生母亲在暗中觊觎苏家财产
  - 顾家是苏家世交，两家联姻是商业布局的一环
  - 沈念初的回归打破了原有的利益平衡，所有人都在重新站队
- 与源文差异化：
  - 源文是小康↔富裕的温和对比；新书是市井↔豪门的强烈阶层冲击
  - 源文以校园日常为主；新书增加豪门权谋暗线，格局更大
  - 源文家庭温情为主；新书增加商战元素和假千金对抗线，冲突更强

## 简介（三种风格各150-250字）

### 标准版
沈念初在南城老街的面馆里长到十七岁，端盘子收账样样精通，嘴皮子比刀还利。她以为自己这辈子就是考上大学、把妈的面馆开成分店。直到一个西装革履的男人找上门，说十七年前医院抱错了孩子，她才是苏家的千金。

亲生父亲是地产集团董事长，继母笑里藏刀，假千金楚楚可怜，亲弟弟拒她于千里之外。更要命的是，她的新同桌顾深——苏婉宁名义上的未婚夫，人前温润如玉，人后冷淡疏离，看她的眼神像在看一个不速之客。

沈念初心想：行，既然你们都不欢迎我，那就别怪我把这个家搅个天翻地覆。

### 悬念版
十七年前，南城中心医院，一场暴雨夜的混乱，两个女婴被调换了命运。

沈念初在面馆的油烟里长大，苏婉宁在别墅的钢琴声中长大。一个练就了察言观色的本事，一个学会了不动声色的心机。

当真相浮出水面，沈念初被接回苏家，她以为自己终于有了家。可等待她的是继母的冷眼、弟弟的排斥、假千金的暗箭，以及那个温润如玉却从不正眼看她的顾深——苏婉宁的"未婚夫"。

所有人都在等她知难而退。可他们不知道，这个在市井里摸爬滚打长大的女孩，从来不知道"退"字怎么写。

而顾深也在等，等她露出破绽。只是他没想到，等来的不是破绽，是心动。

### 甜宠版
沈念初从小就知道自己跟别人不一样——别的同学放学回家写作业，她放学回面馆端盘子。但她不觉得苦，因为她有个全世界最好的妈妈，还有个从小护着她的竹马陆嘉树。

直到十七岁那年，一辆黑色轿车停在面馆门口，说她其实是苏家走丢的千金小姐。

新家什么都不缺，就是缺温度。亲爸不会表达，继母笑里藏刀，假千金暗中使绊，亲弟把她当外人。就连新同桌顾深，那个所有人眼中温润如玉的世家公子，对她也是客客气气的疏离。

沈念初不怕，她有嘴，能怼。继母阴阳怪气？怼。假千金装可怜？怼。顾深冷淡？怼……等等，他怎么笑了？

后来沈念初才知道，这个腹黑的男人，从第一眼起就没打算放过她。
