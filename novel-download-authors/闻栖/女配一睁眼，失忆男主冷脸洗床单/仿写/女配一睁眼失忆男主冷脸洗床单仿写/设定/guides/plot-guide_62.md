# 第N章写章指南

**【严禁原样搬用源文冲突/动作/反应，每个节拍必须重新设计，通过剥名测试】**

## 一、源文节拍拆解

### 节拍1
**源文段落**：宋云绯说着，抬头看向眼前的男人，试图挽救一下，“又不影响给你擦药。” / “再说了，关心都是相互的……” / “昨天我太困，给忘记了，你可别记在心里啊。” / 楚靳寒微微摇头，将自己的手抽了回来，“没什么大碍，不用擦药……”

**冲突**：宋云绯想通过擦药维持亲密互动 与 楚靳寒拒绝她靠近（情感/身体接触的主动权之争）

**动作**：宋云绯抬头解释、说关心是相互的、为忘记擦药找借口；楚靳寒抽回手、拒绝擦药

**反应**：楚靳寒说“不用擦药，过几天也会好”——用自愈能力削弱她的存在价值

**读者情绪**：憋屈（1/5）——她想修补，他不动声色地拉开了距离

---

### 节拍2
**源文段落**：楚靳寒想到什么，忽然站起身，把桌上的酒拿起，盖上盖子，放回了柜子里。 / 然后又拿出装药的箱子，在里面翻了翻，找出一盒布洛芬拿到她跟前。

**冲突**：楚靳寒想照顾她 与 他不愿接受她的照顾（单向付出 vs 双向互动的不对等）

**动作**：楚靳寒起身收酒、翻药箱、找布洛芬、递给她

**反应**：无直接反应，过渡段

**读者情绪**：甜（2/5）——他嘴上拒绝擦药，但记得她肚子疼

---

### 节拍3
**源文段落**：宋云绯看了眼他手里的药，“不知道啊，吃两粒看看吧。” / 楚靳寒掰了一粒药放在她手心，又去重新接了杯水过来。 / 宋云绯捧着水杯，吃了药，瘫坐在沙发上看电视。

**冲突**：宋云绯对自己身体的陌生感（失忆后遗症）与 楚靳寒细致照料形成的依赖关系

**动作**：宋云绯说“不知道”、吃两粒；楚靳寒掰药、接水递给她；宋云绯吃药、瘫坐看电视

**反应**：楚靳寒主动掰药接水——他用行动在照顾，但全程无表情/无语言

**读者情绪**：甜（3/5）——细节照顾很暖，但她连自己吃什么药都不知道

---

### 节拍4
**源文段落**：楚靳寒去喂狗。 / 不知道是狗粮不符合口味，还是它单纯的不爱吃，圈圈吃了几口就走开了。 / 楚靳寒又去把刚才的剩菜和剩饭和狗粮搅拌一起。 / 小家伙又跑回来，埋头在盆里吃了起来。 / 而且混在一起的狗粮，它竟然都拱到一边去了…… / 楚靳寒狐疑地拎起狗粮袋子查看配料表……

**冲突**：圈圈不吃狗粮（狗的需求 vs 楚靳寒提供的食物） → 折射出“女主不在时他照顾不了这个家”的隐喻

**动作**：楚靳寒喂狗、看狗不吃、搅拌剩饭菜、狗跑回来吃；楚靳寒拎起袋子查看配料表；狗把狗粮拱到一边

**反应**：狗选择性地不吃狗粮——它已经习惯了宋云绯的喂养方式

**读者情绪**：悬念（2/5）——狗为什么不吃？他在检查什么？

---

### 节拍5
**源文段落**：宋云绯注意到他在看狗粮，开口说道，“你别看了，它不吃那个狗粮，这些天我都是喂的我们吃的饭菜。” / 楚靳寒不解，“它为什么不吃？” / 宋云绯摇头，“不知道啊，可能是质量太差了？” / “这袋一百五。”楚靳寒说一百五的时候，还有几分郁闷的意味。 / 他一个月才五百块，这小东西居然嫌弃一百五的狗粮。

**冲突**：宋云绯擅自改变喂养方式 与 楚靳寒的预算/控制感被打破（家庭决策权之争）

**动作**：宋云绯说“别看了”、解释自己一直喂饭菜；楚靳寒问为什么；宋云绯猜是质量差；楚靳寒说出价格，语气郁闷

**反应**：楚靳寒内心：他一个月五百，狗嫌弃一百五的狗粮——经济弱势的自卑瞬间

**读者情绪**：憋屈（3/5）——他连狗都养不起的感觉

---

### 节拍6
**源文段落**：宋云绯有些好笑，“好了，别看了，以后就给它吃饭吧，还省钱了呢。” / 楚靳寒把狗粮放回去，回到沙发上坐了下来。

**冲突**：宋云绯想用“省钱”化解他的郁闷 与 楚靳寒无法反驳（她说的对但更显得他没用）

**动作**：宋云绯笑着说“还省钱了呢”；楚靳寒放回狗粮、坐到沙发上

**反应**：楚靳寒不说话，被动接受——他在这件事上没有发言权

**读者情绪**：憋屈（1/5）——她轻松化解，但他的沉默很重

---

### 节拍7
**源文段落**：宋云绯挑选电视，她有点受不了电视里那几部免费的动画片，心一横，直接开了个会员。 / 在找电视机的时候，宋云绯特意看了眼演员表，确定没有那个叫萧娴的演员，才点开播放。

**冲突**：宋云绯想消费（开会员）与 家里经济紧张（之前刚被扣款一千多）的矛盾；以及她对萧娴的警觉

**动作**：宋云绯受不了免费动画、开会员、检查演员表避开萧娴、点开播放

**反应**：无直接外部反应——这是她内心的决策

**读者情绪**：悬念（2/5）——她为什么避开萧娴？开会员花钱不心疼吗？

---

### 节拍8
**源文段落**：外面下着淅淅沥沥的雨，在屋里看着电视，十分的惬意。 / 如果肚子不疼就好了。 / 她拿出手机，记下来大姨妈的时间，免得下次再遇到这种事。

**冲突**：当下的惬意 与 身体不适（生理期疼痛）的对立

**动作**：宋云绯感受雨声、看电视；记下经期时间

**反应**：环境：外面下雨，屋里看电视——氛围营造

**读者情绪**：甜（1/5）——平淡生活的记录，她在学着管理自己的身体

---

### 节拍9
**源文段落**：电视也不好看，她干脆在二手网上看电脑。 / 现在卡里有三千多，足够买一台做图用的二手电脑了。 / 翻了半天，添加了几台先收藏，打算明天再跟买家好好聊聊。

**冲突**：她想买电脑提升工作能力 与 资金有限（三千多只能买二手）

**动作**：宋云绯觉得电视不好看、打开二手网、看电脑、收藏几台

**反应**：她的内心计算：三千多够买二手做图电脑——在有限资源里规划未来

**读者情绪**：甜（2/5）——她在为工作做准备，有目标感

---

### 节拍10
**源文段落**：坐在旁边的楚靳寒，能时不时听到她嘴里发出“嘶”的声音。 / 过了会儿，他开口，“你要不去床上躺着？” / 宋云绯嗯了一声，收起手机，“那你还要看吗？” / 楚靳寒拿起遥控器关掉了电视，“我去洗澡。”

**冲突**：宋云绯忍着痛不说 与 楚靳寒察觉她的不适（沉默的关怀 vs 不表达的忍耐）

**动作**：宋云绯发出“嘶”声；楚靳寒让她去躺着；宋云绯问他还看不看；楚靳寒关电视说去洗澡

**反应**：楚靳寒关掉电视——她不在客厅，他一个人也不看了

**读者情绪**：甜（2/5）——他一直在注意她，但她不说他也不追问

---

### 节拍11
**源文段落**：宋云绯狐疑，“怎么了？有什么事吗？” / “没事，等你好点再说。” / 说完这句话，楚靳寒就起身去了洗手间。 / 宋云绯茫然地看着他……她现在就如同惊弓之鸟，每次楚靳寒卖关子，她都感觉是自己暴露了。

**冲突**：楚靳寒有事要说但选择隐瞒 与 宋云绯害怕身份暴露（信息不对等引发的恐惧）

**动作**：宋云绯追问；楚靳寒说“等你好点再说”；楚靳寒去洗手间；宋云绯内心恐慌

**反应**：宋云绯心里解读：每次他卖关子，她都觉得自己暴露了——她一直活在恐惧里

**读者情绪**：悬念（4/5）——他要说什么？她为什么这么害怕？

---

### 节拍12
**源文段落**：回到床上躺下，圈圈又在扒拉在床头，蹦跶着两条小短腿试图跳上来…… / 宋云绯无奈，只能将它捞上来。 / 小家伙上来却不是睡觉，它在床上嗅来嗅去，最后停在楚靳寒的枕头旁边，一口咬住枕头。

**冲突**：圈圈想上床 与 它不会自己跳上来（狗的依赖）；狗咬枕头（狗的行为 vs 物品安全）

**动作**：圈圈扒拉床头、蹦跶、嘤嘤嘤；宋云绯捞它上来；圈圈嗅床、咬住楚靳寒的枕头

**反应**：宋云绯无奈捞狗——她一直在照顾周围的生命

**读者情绪**：悬念（2/5）——狗为什么咬他枕头？

---

### 节拍13
**源文段落**：宋云绯大惊，一把抓住枕头，“给我松口，不想活了你！小心等会他出来打死你。” / 她一边说，还一边瞄向洗手间。 / 但她越扯，圈圈就拽得越卖力，撅着屁股，四条腿都绷直了。 / 最终，还是宋云绯更胜一筹，把枕头抢了回来，却也听到“咔嚓”一声。 / 枕头被扯开了一条口子，露出了里面的枕芯。

**冲突**：宋云绯要保护枕头（怕楚靳寒生气）与 圈圈非要咬枕头（狗不懂后果）

**动作**：宋云绯抓枕头、威胁狗、瞄洗手间；圈圈拽枕头、撅屁股绷腿；宋云绯抢回枕头、枕头撕裂

**反应**：枕头破了——物品被损坏的后果已经发生，她没拦住

**读者情绪**：悬念（3/5）——他出来发现枕头破了会怎样？

---

### 节拍14
**源文段落**：宋云绯瞪着它，抬手就要给它一嘴巴。 / 这家伙倒也聪明，许是察觉到闯祸了，摇了摇尾巴，在她身边把自己蜷成一团，睁着水汪汪的大眼睛，无辜的望着她。 / 看到它这模样，宋云绯举在半空的手，愣是没打下去。 / 片刻后，宋云绯无奈地叹了口气，把枕套扒下来，去找了个新的枕套换上。 / “再咬打你！”

**冲突**：宋云绯的愤怒/惩罚冲动 与 狗的卖萌求饶（惩罚与心软的拉扯）

**动作**：宋云绯瞪狗、抬手要打；狗摇尾巴、蜷缩、无辜眼神；宋云绯手停在半空、叹气、换枕套、威胁狗

**反应**：狗成功用卖萌躲过惩罚——它知道怎么拿捏她

**读者情绪**：甜/憋屈（2/5）——她下不去手，狗太精了

---

### 节拍15
**源文段落**：刚换上，楚靳寒从浴室走了出来。 / 宋云绯赶紧钻进被窝睡觉，顺便把狗子一并藏盖住了。 / 没多会儿，旁边响起窸窸窣窣的声音，脚步声由远及近。 / 楚靳寒站在床前，看着蜷在两人中间，只露出一个头的狗，又看了眼换了个颜色的枕头。 / 片刻后，他抬手关掉了灯。 / 宋云绯暗暗替圈圈松了口气，庆幸它躲过一顿毒打。

**冲突**：宋云绯想隐瞒狗咬破枕头 与 楚靳寒注意到枕头换颜色了（隐瞒 vs 察觉）

**动作**：楚靳寒出来；宋云绯钻进被窝、藏狗；楚靳寒走过来的脚步声；楚靳寒站床前看狗、看枕头；关灯；宋云绯松口气

**反应**：楚靳寒看了枕头但没说话——他发现了但不点破

**读者情绪**：悬念/甜（3/5）——他一定看到了，为什么不说？他在忍什么？

---

### 节拍16
**源文段落**：第二天早上。 / 宋云绯醒来，第一件事就是伸手摸向旁边，看狗有没有被自己压死。 / 这次又摸了个空，她倏地睁开眼，从床上坐起来。 / 厨房里有动静，目光一转，看向笼子里，圈圈正坐在里面，眼巴巴地望着她。 / 宋云绯挠了挠头，它什么时候跑笼子里去的？

**冲突**：宋云绯的担心（狗可能被压死）与 狗安然无恙在笼子里（她的担心落空，但产生新疑问）

**动作**：宋云绯摸旁边、摸空、坐起来；看厨房有动静、看笼子里的狗；挠头疑惑

**反应**：狗在笼子里——狗什么时候进去的？楚靳寒早上起来关的？

**读者情绪**：悬念（2/5）——他把狗关起来了？是烦狗还是怕她压死狗？

---

### 节拍17
**源文段落**：还没来得及细想，手机忽然响了起。 / 还以为是自己的工资到账了，随手拿起手机解锁。 / 下一秒，宋云绯猛地从床上站了起来。 / 楚靳寒端着早餐出来，见她又气又怒的站在床上，问了句，“怎么了？” / 宋云绯捏着手机，悲痛欲绝地跪坐在床上：“天杀的自动还款！” / 虽说是最低还款，但是也扣了她一千多。 / 电脑啊……又没了。 / 但转念一想，不扣的话，逾期了还的更多。 / 宋云绯怅然的叹气，也只能认命了。

**冲突**：宋云绯期待工资到账（想买电脑）与 自动还款扣款（生活成本剥夺梦想）

**动作**：手机响；宋云绯以为工资到账；解锁；看到扣款消息站起；楚靳寒问怎么了；宋云绯跪坐、喊出扣款；计算扣款金额；内心挣扎后认命

**反应**：楚靳寒端着早餐出来看到她崩溃——他在旁观她的情绪崩塌

**读者情绪**：憋屈（4/5）——刚计划好的电脑又没了，生活总在打击她

---

### 节拍18
**源文段落**：楚靳寒把早餐放在桌上，“扣了多少？” / “一千零五十！” / “先吃早餐吧。” / 宋云绯欲哭无泪地下了床，去洗手间洗漱出来，有气无力地坐在了小餐桌前。 / 却发现桌上摆放着一杯红糖水。 / 她心中微动，下意识转头看向旁边的男人。 / 刚才被扣款的郁闷，也少了几分，“谢谢啊，今天已经没事了，不用再给我准备。”

**冲突**：宋云绯的钱被扣（生活压力）与 楚靳寒的红糖水（情感抚慰）形成对立，但红糖水同时强化了她的愧疚

**动作**：楚靳寒问扣多少；宋云绯报数字；楚靳寒说吃早餐；宋云绯下床洗漱、坐到餐桌前；看到红糖水；心里感动；转头看他；说谢谢、说不用再准备

**反应**：宋云绯的郁闷因红糖水减少——他的细节照顾在治愈她，但她不知道这治愈的代价

**读者情绪**：甜/憋屈（3/5）——他记得经期要喝红糖水，但她不知道他在想什么

---

### 节拍19
**源文段落**：楚靳寒沉默了下，拿出手机，随便点了几下。 / 宋云绯这边手机立马响了，她拿起来一看，居然是楚靳寒给他转了一千块。 / 她惊愕地看向他，“你哪来的这么多钱？” / 他不是一个月就五百吗？ / 而且这个月他没有工资发，
