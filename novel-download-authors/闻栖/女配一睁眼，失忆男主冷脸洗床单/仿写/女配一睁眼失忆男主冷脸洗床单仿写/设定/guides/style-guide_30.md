# 去AI写作指南 - 第N章：冷脸洗床单

> **核心原则**：定量锚点，平衡定性。避免极端化指令，追求自然波动。

## 1. 定量锚点（数据来源：style_profile_N.json）

| 指标 | 源文数值（第N章） | 仿写目标值（±20%） | 指令说明 |
| :--- | :--- | :--- | :--- |
| **篇幅** | 3300字 | **2600 - 4000字** | 保持中短章节体量。 |
| **平均句长** | 18.5字 | **15 - 22字** | 短句为主，穿插中句。避免连续长难句。 |
| **平均段落长度** | 45-65字（2-4句） | **35 - 80字** | 段落轻短，主打破碎感与呼吸感。 |
| **对话占比** | 18% | **14% - 22%** | 对话精简，功能性强。大段对白或解释性对话是大忌。 |
| **描写密度** | 高（动作/细节导向） | 高 | 环境描写、心理描写必须通过动作和细节展现。 |

## 2. 八维定性分析

**核心驱动：对抗性张力。所有描写都为“她失忆后的疏离”与“他冷脸下的失控”这对矛盾服务。**

#### 节奏控制
*   **源文写法**：快慢交织。冲突和动作场景用连续短句快节奏；单人物内心或环境渲染时节奏放缓，用细节填充。
*   **仿写指令**：**有起有伏**。开头迅速建立场景和张力（快），中间通过多次“无效沟通”和细节积累情绪（慢→快），结尾用一个反转或极具冲击力的画面将情绪推到顶点（快）。

#### 过渡与衔接
*   **源文写法**：大量使用硬切换和动作顺接。场景转换利落，不拖泥带水。前一句还在写她的表情，下一句直接切到他的动作反应。
*   **仿写指令**：**用动作代替连接词**。禁止使用“与此同时”、“过了一会儿”、“他突然想到”。直接用“他拧开水龙头。”、“她转过身。”这类动作开启新段落，驱动叙事。

#### 描写精度
*   **源文写法**：笔力集中于“反常的细节”。不泛泛写气氛，而是写“他手背上的青筋”、“她衣袖下露出的半截纱布”、“搓到发红的指关节”。
*   **仿写指令**：**用一个细节锚定一种情绪**。写他的愤怒，只写他洗床单时过分用力的手；写她的迷茫，只写她下意识摩挲婚戒留下的白痕。每个物件都必须有戏。

#### 对话功能
*   **源文写法**：对话是战争的延续。信息量极低，潜台词极高。一问一答之间充满试探、回避和刺痛。常用不完整的句子或沉默。
*   **仿写指令**：**话不说满，甚至不说人话**。多用反问、省略、答非所问。对话的核心不是交流，是展示两人之间无法逾越的认知鸿沟。

#### 情绪表达
*   **源文写法**：极度克制，外化到动作上。内心汹涌，笔下只写他“把洗干净的床单又浸回水里”这类重复性、无意义的行为。
*   **仿写指令**：**情绪不能“说”出来，只能“做”出来**。她感到厌烦，就去反复开冰箱门。他感到窒息，就解开领口第一颗扣子又立刻扣上。用行为链呈现心理状态。

#### 句子结构
*   **源文写法**：以简单句和并列句为主。主-谓-宾结构清晰，修饰成分少。营造出一种冷静、甚至冷酷的叙事腔调。
*   **仿写指令**：**长短句交替，营造呼吸感**。连续2-3个10字以下的短句制造压迫感后，接一个15-20字的中句稍作舒缓。例如：“水很凉。她缩了一下手。他没看见，转过身去拿洗衣液，动作快得像在躲什么东西。”

#### 信息释放节奏
*   **源文写法**：挤牙膏式，且充满错觉。读者一开始以为男主是施暴者，后来通过细节发现他可能是被伤害的那个；以为女配真失忆，又发现她可能藏有秘密。
*   **仿写指令**：**制造双向误解**。每一次信息的释放，都要同时加深两个角色的悲剧性。比如，他以为她在惩罚自己，其实她只是真的不记得；她以为他天生冷漠，其实他昨晚守了她一夜。

#### 叙事结构
*   **源文写法**：单场景多视角。在一个限定的空间（如洗衣房），通过女主的视角观察男主，再通过男主的动作暗示其视角，让读者在上帝视角和限制视角间拉扯。
*   **仿写指令**：**让读者比男女主多知道一点点**。这一点点知情权，就是读者心痛的来源。写出“她不知道他为什么……，但读者知道”的效果。

## 3. 本章重点（不超过3条）

1.  **核心画面**：定一个贯穿始终的核心动作画面（如“洗床单”），所有情节和情绪都围绕这个动作展开。本章就写这个动作的持续、中断与变奏。
2.  **无效沟通**：设计三次失败的交流尝试。一次她用语言（失败），一次他用行动（被误解），一次两人同时沉默（成功传达了最深的隔阂）。
3.  **双向信息差**：写清楚“他以为的”和“她实际想的”之间的偏差。用动作暗示他的误会，用心理（或其他细节）揭示她的真实状态。

## 4. 附录：源文例句精选（来自第N章，用于模仿语调）

> 1.  **【动作即情绪】** 瓷盆里的水一圈一圈地泛红。他还在搓，指关节用力到发白，像是那块布料跟他有什么深仇大恨。
> *   *（模仿点：用“水泛红”和“发白的指关节”代替“他很痛苦”。）*

> 2.  **【破碎的短句群】** 她站在井边。衣服穿得少。他看着心烦。
> *   *（模仿点：三个短句，三个主语，切割出冰冷疏离的视觉节奏。）*

> 3.  **【对话中的潜台词】** “你手不冷？”
>     “……”
>     “我问你话。”他声音沉下去。
>     “不记得了。”她把视线从水里移开，看向他，“也不记得你。”
> *   *（模仿点：答非所问，用“不记得了”回答“手冷不冷”，并递进到“不记得你”，信息量和伤害值同时拉满。）*

> 4.  **【无用功的情绪外化】** 他把洗干净的床单拎起来，看了一会儿，又浸回血水里。
> *   *（模仿点：用一个无意义的重复动作，精准表达人物此刻内心的偏执与无法处理情绪的精神状态。）*

> 5.  **【结尾的定格画面】** 水声停了。她听见他擦燃打火机的声音，还有一声极轻的，被夜风扯散的叹息。
> *   *（模仿点：一个声音细节（打火机）和一个几乎听不见的细节（叹息），构成一个极具电影感的结尾，余味悠长。）*
