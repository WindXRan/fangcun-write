# plot_guide 第N章

## 步骤1：拆源文节拍

【节拍1】
- **冲突**：宋云绯对突然获得房产的经济安全担忧（宋云绯的谨慎vs楚靳寒的突然赠予）
- **动作**：宋云绯一愣，质疑“过户在我名下？”，狐疑地看着他，担心是贷款买房
- **反应**：楚靳寒微微点头“对”，解释全款购买，陈小姐证明
- **读者情绪**：惊讶→安心
- **强度**：2

【节拍2】
- **冲突**：宋云绯意识到无法再用经济理由推脱结婚（宋云绯的逃避心理vs楚靳寒的推进关系）
- **动作**：宋云绯任由他牵手，机械地跟在他身后，内心觉得“把人骗来又骗房”
- **反应**：楚靳寒握住她的手说“以后想换地方卖了就是”
- **读者情绪**：憋屈（女主被困在不得不得接受的处境）
- **强度**：3
- **感官**：触觉（牵手、阳光暖洋洋但感觉不到暖意）

【节拍3】
- **冲突**：外人视角的美好祝福vs宋云绯内心的苦涩自知（表面幸福vs内在危机）
- **动作**：陈小姐说“你男朋友真好”“要结婚了提前恭喜”，宋云绯勉强一笑
- **反应**：陈小姐不解但没多问，离开
- **读者情绪**：虐（甜蜜祝福包裹着苦涩真相）
- **强度**：3

【节拍4】
- **冲突**：楚靳寒的“保障论”透露出他对未来的不确信（楚靳寒预感到危险vs宋云绯需要安全感）
- **动作**：宋云绯复杂地问“为什么”，楚靳寒拉起她的手低声说“你在名下会更安心，不管将来发生什么都算保障”
- **反应**：宋云绯心里沉甸甸的，心头泛酸，猜想“他是不是也预感到什么了”
- **读者情绪**：悬念+虐
- **强度**：4
- **感官**：听觉（低声说话）

【节拍5】
- **冲突**：楚靳寒提出领证vs宋云绯对未来不确定的恐惧（关系推进vs记忆恢复的威胁）
- **动作**：楚靳寒停车转头看她，深深注视后说“我们去领证吧”，宋云绯攥紧安全带低头不回答
- **反应**：楚靳寒眼底闪过一丝黯然，收回视线重新启动车子，说“抱歉不该这么着急，随口一说”
- **读者情绪**：虐（双方都想靠近却都有顾虑）
- **强度**：4
- **感官**：视觉（深深注视、黯然的眼底）

【节拍6】
- **冲突**：宋云绯的理性权衡vs情感的冲动（理智上不该领证vs感情上想答应）
- **动作**：宋云绯捏着安全带，心里天人交战，想“他现在失忆领证不公平，恢复记忆不承认对两人都是伤害”
- **反应**：没有外在反应，楚靳寒不再提，两人恢复日常
- **读者情绪**：虐
- **强度**：3

【节拍7】
- **冲突**：卫岢催促楚靳寒回去vs楚靳寒拒绝（工作责任vs失忆后的无力感）
- **动作**：卫岢守到凌晨两点拦住楚靳寒“股东会剩两个月必须回去”，楚靳寒蹙眉“你觉得我现在这样回去有什么用”
- **反应**：卫岢蹙眉承认棘手
- **读者情绪**：悬念（压力积累）
- **强度**：3
- **感官**：视觉（凌晨两点、厂区）

【节拍8-鼓点】
- **冲突**：突如其来的车辆撞击vs楚靳寒僵立不动（死亡威胁vs记忆冲击）
- **动作**：强光射来，车直冲而来，楚靳寒被气浪带来烧焦味触发记忆碎片“脑海中某处被引爆轰然炸开”，僵立不动；卫岢惊恐扑倒他
- **反应**：两人倒在绿化带旁，车撞上路灯，车头凹陷冒白烟，楚靳寒脸色难看胸口起伏，额头冒冷汗
- **读者情绪**：燃+悬念（危险中的记忆复苏）
- **强度**：5
- **感官**：视觉（强光）、嗅觉（烧焦味）、触觉（气浪）、听觉（巨响）

【节拍9】
- **冲突**：袭击者的出现vs楚靳寒认出对手（刘茂才的报复vs楚靳寒记忆恢复后的身份觉醒）
- **动作**：楚靳寒喘息着试图起身又无力跌下，从牙缝里挤出“刘、茂、才”，痛苦闭眼；卫岢转头看见胖子从车里下来
- **反应**：胖子拎着水果刀，带着怨毒眼神缓步靠近
- **读者情绪**：悬念+燃（记忆恢复+危险逼近）
- **强度**：5
- **感官**：视觉（怨毒眼神、水果刀）

## 步骤2：设计新事件

**排除项**：房产过户、买车撞人、水果刀、领证提议、刘茂才

【节拍1-新】
- **冲突**：叶桑收到匿名寄来的医院档案袋，里面是江辞的脑部CT和诊断报告（叶桑的隐私被侵犯vs江辞的病情被暴露）
- **动作**：叶桑拆开快递，抽出CT片子对着光看，手指微微发抖
- **反应**：江辞从厨房探出头“谁寄的？”，看到片子后走过来抽走，低声说“别看了”
- **读者情绪**：悬念+惊讶
- **强度**：2

【节拍2-新】
- **冲突**：江辞隐瞒病情的决心vs叶桑想知道真相的恐惧（江辞不愿成为负担vs叶桑要确认他是否安全）
- **动作**：叶桑抓住他抽走片子的手腕，仰头盯着他“上面写的什么？你看过没有？”
- **反应**：江辞把报告折好放进抽屉，转身去关火，背对着她说“老毛病，没什么”
- **读者情绪**：憋屈（被隐瞒的无力感）
- **强度**：3
- **感官**：触觉（抓住手腕、拍片子的冰凉触感）

【节拍3-新】
- **冲突**：快递员的话透露寄件人信息vs叶桑意识到有人故意介入（外来者的别有用心vs叶桑对两人关系的守护）
- **动作**：快递员打来电话“寄件人说是您亲戚，姓周”，叶桑追问“有留全名吗”
- **反应**：快递员说“只写了周女士，说您看到就懂了”
- **读者情绪**：悬念（周女士是谁？）
- **强度**：3

【节拍4-新】
- **冲突**：江辞的病历被周女士掌握vs叶桑对“周”这个姓氏的警觉（周女士可能是江辞过去的人）
- **动作**：叶桑挂了电话，盯着厨房里江辞的背影，脑子里闪过他之前的异常（反复确认日期、问她“如果我不在”）
- **反应**：江辞端着菜出来，看见她表情不对，放下盘子走过来，“桑桑？”
- **读者情绪**：悬念+虐
- **强度**：4
- **感官**：视觉（背影、表情不对）、听觉（电话挂断后的沉默）

【节拍5-新】
- **冲突**：叶桑直接追问病情vs江辞回避（叶桑逼他说实话vs江辞用轻描淡写搪塞）
- **动作**：叶桑站起来把CT片子拍在桌上，“你说老毛病，那我就问你一次，这个病，会死人吗？”
- **反应**：江辞盯着她看了几秒，垂下眼说“医生说可控”，手指无意识地转着手腕上的住院手环
- **读者情绪**：虐（谎言的撕裂感）
- **强度**：4
- **感官**：听觉（拍桌声）、视觉（转手环的细节）

【节拍6-新】
- **冲突**：叶桑的理性推断vs情感的自我欺骗（知道他在说谎vs希望他说的是真的）
- **动作**：叶桑注意到他转手环的动作——那是从医院带回来的，他从来没摘过——她喉咙发紧，最终只说“那就好”
- **反应**：江辞把手放下，转身去盛饭，两人坐下吃饭，谁也不说话
- **读者情绪**：虐
- **强度**：3

【节拍7-新】
- **冲突**：周女士持续施压vs叶桑选择不告诉江辞（外部的入侵vs叶桑想自己查清楚）
- **动作**：吃完饭叶桑借口去倒垃圾，在楼下翻手机通讯录，找到半年前存的江辞助理号码，发了条消息“周女士是谁”
- **反应**：助理秒回“叶姐你怎么知道她？？”，随后又撤回，改成“抱歉不方便说”
- **读者情绪**：悬念（助理的反应确认了周女士不简单）
- **强度**：3
- **感官**：视觉（手机屏幕的撤回提示）

【节拍8-鼓点-新】
- **冲突**：叶桑回拨电话被挂断vs助理突然出现在小区门口（叶桑要追问vs助理主动上门带来更紧急的消息）
- **动作**：叶桑连打三个电话都被挂断，正要上楼，看见小区门口一辆黑色轿车闪了闪双闪，后车窗降下，露出助理苍白焦急的脸，“叶姐，江哥在不在楼上？周女士的人已经到了，你们必须马上走”
- **反应**：叶桑脑子嗡的一声，“已经到了是什么意思？”，助理还没回答，楼上的窗户突然传出玻璃碎裂的声音
- **读者情绪**：燃+悬念（追兵已至）
- **强度**：5
- **感官**：视觉（双闪、苍白焦急的脸）、听觉（玻璃碎裂声）

【节拍9-新】
- **冲突**：叶桑冲上楼vs江辞已被控制（叶桑要救江辞vs周女士的人先一步动手）
- **动作**：叶桑推开家门，客厅灯全灭，只有厨房的应急灯亮着，地上碎了一只盘子，江辞被两个穿西装的男人按住肩膀坐在椅子上，他抬头看她，嘴角有血，眼神却是她从未见过的冷——那种冷不是对她的，是透过眼前的人在看别人
- **反应**：一个女人的声音从厨房阴影里传出来，“叶小姐，请坐。我是江辞的母亲。”
- **读者情绪**：燃+悬念（身份反转——周女士是母亲）
- **强度**：5
- **感官**：视觉（灯灭、碎盘子、嘴角血迹、冷透的眼神）

## 步骤3：验证

剥名测试：
- 源文：男友过户房产→女友纠结→外人祝福→男主“保障论”→提领证→女主理性拒绝→助理催男主回去→车祸→男主恢复记忆认出袭击者
- 新文：男友病历被寄来→女友追问真相→快递员透露“周女士”→女主联想到男主异常→女主逼问病情→男主撒谎→女主私下联系助理→助理示警→男主被控制、母亲现身
- 确认：无一个冲突类型重叠（房产纠纷→病历隐私，车祸冲击→家人突袭，袭击者报复→母亲现身），所有动作反应均已替换剥名后无法认出源文结构
