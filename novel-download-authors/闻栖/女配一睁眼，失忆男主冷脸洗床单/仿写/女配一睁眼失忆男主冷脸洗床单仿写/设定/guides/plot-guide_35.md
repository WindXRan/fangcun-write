# 第N章 写章指南

## 一、源文节拍拆解

### 节拍1
- **原文内容**：李哲说完，期待的看着楚靳寒，希望他能想起自己。
- **冲突**：李哲想要确认楚靳寒身份的利益 vs 楚靳寒隐藏身份的利益（身份确认矛盾）
- **动作**：李哲用期待的眼神注视楚靳寒
- **反应**：（待楚靳寒回应）
- **读者情绪**：悬念，强度2
- **感官**：视觉

### 节拍2
- **原文内容**：楚靳寒短暂的沉默过后，平静地开口否认，说自己是业务员不是楚总。
- **冲突**：同上（持续的身份确认矛盾）
- **动作**：楚靳寒短暂沉默后平静否认身份
- **反应**：李哲怔住，难以置信，再次打量楚靳寒
- **读者情绪**：悬念+憋屈，强度3
- **感官**：听觉

### 节拍3

- **反应**：（群像反应）
- **读者情绪**：悬念缓解，强度2
- **感官**：视觉（群像）

## 二、源文结构特征

- **冲突类型**：身份误解（核心），商业利益，职场竞争，内心道德
- **节奏曲线**：紧张升级（身份暴露危机）→ 意外解决（侄女脑补）→ 红利爆发（订单爆炸）→ 回报兑现（升职加薪）→ 温情落点（买房提议）
- **章节功能**：职场升级兑现章，展示主角能力带来的连锁商业红利，同时在关系层面埋下未来共同生活的伏笔
- **对抗方式**：外界角色主动追查身份，主角方被动应对，第三方（侄女）意外解围
- **爽点构造**：大人物一句话 = 底层跑断腿，实力被意外放大
- **情绪走向**：悬念→紧张→松一口气→爽→更爽→甜
- **核心反转**：身份暴露危机 → 订单爆炸红利的意外转化

## 三、新书融合设定

### 必须保留的元素
- 新书角色对应：识别者（新角色）、被识别者（对应楚靳寒）、遮掩者（对应宋云绯）、受益方（对应何总）、被冷落方（对应王新娟）
- 情绪曲线走向：悬念→紧张→松一口气→爽→更爽→甜
- 章节功能：展示主角（非传统意义）能力带来的连锁红利，并回到关系落点
- 反转结构：危机 → 意外解围 → 红利爆发

### 可替换元素
- 冲突类型：新书冲突类型必须与源文不同（源文=身份误解，新书=全新类型）
- 具体场景：源文展馆→新书任意符合设定的场景
- 识别方式：源文“长得像” → 新书用新书设定中的伏笔元素
- 红利形式：源文订单爆炸 → 新书用符合新书世界观的红利形式

## 四、本章排除项

1. ❌ 商业展会/展馆场景
2. ❌ “长得像某重要人物”的误会
3. ❌ 德国客户/外贸订单
4. ❌ “扮猪吃老虎泡妞”的认知框架
5. ❌ 制衣厂/代工厂行业
