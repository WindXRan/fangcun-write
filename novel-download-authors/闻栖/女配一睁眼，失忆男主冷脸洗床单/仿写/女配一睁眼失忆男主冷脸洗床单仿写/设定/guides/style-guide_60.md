### 2. 过渡与衔接

**源文写法**:  
[从第N章具体描述:场景切换如何过渡?段落间如何衔接?硬切换多还是软过渡多?使用什么衔接词或衔接方式?]

### 3. 描写精度

**源文写法**:  
[从第N章具体描述:描写细腻度如何?五感描写的分布?细节聚焦在哪些方面(动作/神态/环境/心理)?描写与叙事比例?]

**源文写法**:

**源文写法**:

## 五、避免AI写作痕迹核查清单

> 写章完成后,按此清单自查,不符合项返回修改。

| 检查项 | 判断标准 | 是否通过 |
|--------|----------|----------|
| **句长机械对齐** | 没有出现"每句都是15字"或人工感强的等距句长 | [ ] |
| **段落模式化** | 没有出现"每段3句"或"全部单句成段"的模板感 | [ ] |
| **过渡词重复** | 没有出现"突然/于是/接着/同时"堆砌在每段开头 | [ ] |
| **情绪直给** | 没有出现"她很伤心/他很生气"此类直接陈述情绪 | [ ] |
| **对话刻板** | 对话中没有"你说的对/我理解你的感受"类模板用语 | [ ] |
| **信息倾倒** | 没有一大段纯解释性旁白(超过3句连续说明) | [ ] |
| **节奏单一** | 全章没有出现"一直快"或"一直慢"的单调节奏 | [ ] |
| **五感空洞** | 全章至少出现2种以上感官描写(视觉/听觉/触觉/嗅觉) | [ ] |

**判定**: 全部通过→提交; 任一项未通过→定位问题段落→针对性修改→重新核查。
