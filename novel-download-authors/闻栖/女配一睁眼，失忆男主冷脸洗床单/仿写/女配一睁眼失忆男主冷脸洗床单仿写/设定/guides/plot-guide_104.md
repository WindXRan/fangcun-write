# 第N章写章指南

## 源文节拍拆解

### 节拍1：副线——刘茂才炒股成瘾
- **冲突**：刘茂才的个人嗜好与他正常家庭生活的矛盾（炒股沉迷vs妻子期待的丈夫角色）
- **动作**：刘茂才天天盯着股票看，连女朋友也不管了
- **反应**：妻子看到他的绿光脸，蹙眉头，犹豫后下楼打电话询问转移财产的事
- **读者情绪**：悬念（妻子要干什么？）
- **强度**：3
- **感官**：视觉——电脑屏幕绿光、K线图、妻子的脸

### 节拍2：宋云绯接单成功
- **冲突**：宋云绯的经济压力（接不到单）与她即将离开无法带电脑的矛盾
- **动作**：她收到第一个设计单，露出笑容，但想到电脑无法带走又开始犯愁
- **反应**：听到开门声赶紧关掉网站，开电视剧伪装
- **读者情绪**：憋屈（她还得伪装）
- **强度**：2
- **感官**：听觉——电视剧声音、开门声

### 节拍3：散步遛狗（生活日常）
- **冲突**：无直接冲突（日常温馨段落）
- **动作**：两人牵着圈圈散步，树叶落在宋云绯头上，楚靳寒取下
- **反应**：两人感叹时间过得快
- **读者情绪**：甜（温馨互动）
- **强度**：1
- **感官**：触觉——树叶落在头上、楚靳寒的手

### 节拍4：圈圈挣脱
- **冲突**：宋云绯对圈圈的掌控力不足（狗乱跑vs主人控制）
- **动作**：圈圈挣脱牵引绳追树叶，宋云绯追赶不上，弯腰抓绳
- **反应**：楚靳寒扶住她胳膊，她冲他笑说没事
- **读者情绪**：甜（关心时刻）
- **强度**：2
- **感官**：动觉——追赶、气喘、踉跄、扶胳膊

### 节拍5：烤红薯互动
- **冲突**：宋云绯想吃烤红薯的欲望vs她意识到楚靳寒只买了一个的不好意思
- **动作**：楚靳寒买红薯、剥皮、递给她，她感动接过咬一口
- **反应**：楚靳寒拍照，她挡脸说技术差，分享红薯给他吃
- **读者情绪**：甜（日常情侣感）
- **强度**：2
- **感官**：味觉——红薯的香味、甜味、软糯口感

### 节拍6：间接亲吻的暧昧
- **冲突**：宋云绯的羞怯（不好意思让他吃自己咬过的地方）vs楚靳寒的直白（都亲过了）
- **动作**：楚靳寒直接咬她咬过的地方，反问“有什么区别”，提“我们不是都亲过了”
- **反应**：宋云绯无言以对，转移话题问“甜吗”
- **读者情绪**：甜（暧昧撩人）
- **强度**：3
- **感官**：视觉+触觉——楚靳寒擦她嘴角、两人眼神对视

### 节拍7：离开消息的悬念
- **冲突**：宋云绯即将离开的现实vs两人此刻的亲密气氛
- **动作**：她突然说“大后天就走”，楚靳寒擦嘴角的手指顿住
- **反应**：他脸上没有表情，只剩无声注视
- **读者情绪**：悬念+虐（她真要走了？他会怎样？）
- **强度**：4
- **感官**：视觉——顿住的手指、无声注视的眼神

## 新书事件设计

### 节拍1全新事件
- **冲突类型**：商业/职场——副线角色王建国的公司被恶意收购，他决定反击
- **动作**：王建国连夜整理公司财务数据，发现对手的收购资金来源可疑
- **反应**：他妻子看到他在书房通宵，端咖啡进来，偷偷瞥到屏幕上“反收购方案”的文件名，脸色一变退出书房
- **读者情绪**：悬念（妻子为什么变脸？她知道什么？）
- **强度**：3
- **感官**：视觉——电脑屏幕的文件名、妻子偷瞥的眼神

### 节拍2全新事件
- **冲突类型**：职业尊严vs生活现实——林雨桐第一次直播带货成功，但想到妈妈要她明天去相亲
- **动作**：她下播后数着今天赚的300块开心，收到妈妈发来的相亲对象照片，笑容僵住
- **反应**：听到室友开门声，赶紧关掉对话框，假装在剪辑视频
- **读者情绪**：憋屈（她为自己赚钱开心，却得藏起来，还要被逼相亲）
- **强度**：2
- **感官**：听觉——直播结束的提示音、室友钥匙开门声

### 节拍3全新事件
- **冲突类型**：无直接冲突——建立温暖日常
- **动作**：两人在夜市摊吃烧烤，火星溅到林雨桐手背上
- **反应**：秦墨立刻抓住她的手检查，用冰啤酒瓶贴她被烫的地方，她低头看他的手指，两人都没说话
- **读者情绪**：甜（他紧张她的样子）
- **强度**：2
- **感官**：触觉——火星烫、冰凉啤酒瓶、他的手抓着她

### 节拍4全新事件
- **冲突类型**：意外打扰浪漫——突然下雨破坏气氛
- **动作**：暴雨突然落下，两人没带伞，秦墨脱下外套披她头上
- **反应**：林雨桐拽着他一起躲进公交站台，两人衣服都湿了，她看他头发滴水，从包里抽出纸巾递过去，他接过却先擦她脸上的水
- **读者情绪**：甜（笨拙的照顾）
- **强度**：2
- **感官**：动觉+触觉——雨砸、跑、披外套、擦水的动作

### 节拍5全新事件
- **冲突类型**：分享欲vs消费观——林雨桐想吃路边棉花糖，但觉得幼稚不好意思
- **动作**：她停下来盯着棉花糖摊看，又拽他走，被秦墨拉回来，买了一根粉色的
- **反应**：她咬了一口，递给他尝，秦墨看着她咬过的地方，低头咬另一边，说“你这边更甜”，她耳朵红了，说“明明一样”
- **读者情绪**：甜（撩人小动作）
- **强度**：3
- **感官**：味觉——棉花糖的甜、白糖在嘴里化开的沙沙感

### 节拍6全新事件
- **冲突类型**：害羞vs直球——林雨桐被秦墨的直接弄得不自在
- **动作**：秦墨突然凑近，亲掉她嘴唇上沾的糖丝，退开后说“确实更甜”
- **反应**：林雨桐愣住，捂嘴瞪他，转头不看他，耳尖通红地说“你……你偷袭”
- **读者情绪**：甜+惊讶（突然亲上来）
- **强度**：4
- **感官**：触觉+味觉——亲的触感、糖和嘴唇的甜、心跳加速

### 节拍7全新事件
- **冲突类型**：内心秘密vs此刻甜蜜——林雨桐手机震动，收到医院的复诊通知
- **动作**：她看完消息，沉默几秒，转头对秦墨说“我下周可能要住院做检查”
- **反应**：秦墨手里的棉花糖棍“啪”的一声断了，他怔怔看着她，脸上的笑意全部消失，只剩雨声在两人之间响着
- **读者情绪**：悬念+虐（她生病了？什么病？两人刚甜蜜就有变故？）
- **强度**：5
- **感官**：听觉——手机震动声、棉花糖棍断裂的声音、雨声

## 读者情绪曲线

悬念(3) → 憋屈(2) → 甜(2) → 甜(2) → 甜(3) → 惊讶+甜(4) → 悬念+虐(5)

情绪走向：副线悬念开场 → 主线憋屈 → 连续四段甜美日常建立情感 → 突然亲密升级 → 秘密爆出，急转直下

## 感官分布
节拍1：视觉 → 节拍2：听觉 → 节拍3：触觉 → 节拍4：动觉+触觉 → 节拍5：味觉 → 节拍6：触觉+味觉 → 节拍7：听觉

## 排除项
1. ❌ 禁止出现“树叶落在头上”的浪漫暗示
2. ❌ 禁止出现“吃烤红薯”的秋冬路边摊场景
3. ❌ 禁止出现“只买一个分享吃”的间接亲吻暗示
4. ❌ 禁止出现“擦嘴角”的亲昵动作
5. ❌ 禁止出现“说走的时间”引发对方表情变化的告别场景（可用其他方式引爆）

## 关键提醒
- 节拍3-6是日常甜蜜堆叠，必须写足四段不同情境的互动（夜市、躲雨、棉花糖、亲吻），让读者充分沉浸后才捅刀
- 节拍7的秘密爆出，秦墨的“棉花糖棍断裂”替代源文“手指顿住”，用物体破坏象征情绪破裂，更具画面冲击力
- 副线王建国妻子变脸，要埋下伏笔，后续可引出她与收购方的关联
