## 2. 八维定性分析

### 2.1 节奏控制
**源文写法**：日常互动（做饭、洗床单）与内心戏交织，形成一种“慢中有紧”的节奏。没有强情节冲突，靠人物微妙的情绪变化（嫌弃、无奈、一丝在意）驱动。场景切换清晰但不生硬。

**仿写指令**：
- **场景内节奏**：维持“慢动作”式的细节铺陈（如做饭的每个步骤），用角色的微小反应（皱眉、顿一下、别开脸）制造阅读的锚点。不要急于推进情节，让情绪在细节里发酵。
- **场景外节奏**：通过白描式的环境或动作进行场景切换，不写“与此同时”这类转场词。

### 2.2 过渡与衔接
**源文写法**：多用动作衔接，即“以一个角色的动作带出另一个角色的反应”。例如，他放下碗→她看了一眼。少用“时间过得很快”、“接下来说到”等生硬过渡。

**仿写指令**：
- **段落衔接**：上一个段落的末尾动作或物件，在下一个段落开头得到回应或延续，形成镜头感。
- **时间过渡**：用环境变化（天光、温度）或人物的状态变化（犯困、肚子饿）来暗示，避免精确的时间词。

### 2.3 描写精度
**源文写法**：聚焦于“动作的细节”而非“外表的铺陈”。不写“他穿着高定的黑色西装”，而是写“他挽起袖子时，露出的手腕骨节分明”。描写服务于2个目的：①展示人物状态；②暗示人物关系（疏离与拉扯）。

**仿写指令**：
- **动作分解**：把一个完整动作拆成2-3个步骤写。比如，不写“他洗床单”，写成“他放水，倒洗衣液，然后把床单塞进去。”
- **物件特写**：给关键的、承载情绪的小物件特写（如床单上的印记、做饭时溅出的水珠），通过角色如何看待它来传递情绪。

### 2.4 对话功能
**源文写法**：对话功能性极强，不用于交换信息，而用于1. 确立权力关系；2. 掩盖真实情绪；3. 制造潜台词。简短，多有停顿，常有“答非所问”和“沉默回应”。

**仿写指令**：
- **减少信息量**：对话中，角色只说必须要说的，甚至必须说的也只说一半。
- **增加动作伴随**：几乎每句对白都附带一个动作或神态，用动作“翻译”对白的真实含义。
- **禁止问答式对话**：拒绝“一问一答”的模式，要写“一问，一默，再一答”或“一问，一行动”。

### 2.5 情绪表达
**源文写法**：情绪高度内敛。角色从不直说“我很生气/难过”，而是通过“刻意的沉默”、“更用力的动作”、“保持距离的站位”来外化。读者的情绪体验是“推测”出来的，而非被“告知”的。

**仿写指令**：
- **禁止心理标签**：严禁出现“他愤怒地想”、“她觉得委屈”这类直接解释情绪的词汇。
- **使用生理与行为信号**：
  - 压抑/不耐烦：下颌线绷紧、加快的语速、重复的无意义小动作。
  - 示弱/关心：眼神回避、故作冷漠的给予（如“顺手”递水）、放轻的脚步。
- **必须有“压抑”的痕迹**：情绪不是没有，而是被强行压住了。要写出那种“快压不住”的临界感。

### 2.6 句子结构
**源文写法**：陈述句为主，偶有长短句交替。在描写心理活动或连续动作时，句子会变短、变碎；在观察或描写环境时，句子会变长、变缓。多为顺承复句，少用关联词。

**仿写指令**：
- **结构比**：简单句占主导（70%），复合句主要用来描述场景和细微感受（30%）。
- **切割长句**：当一个句子超过35字时，评估是否应切成2句，以增加阅读的喘息感。
- **节奏变化**：在连续3-4个短句后，用一个小长句来总结或承接，形成起伏。
  - *差示范（全是短句）*：她走过去。她看了他一眼。他没动。她也没说话。
  - *好示范（长短结合）*：她走过去，看了他一眼。他没动，就那么靠在门边，视线落在不远处的洗衣机上，不知道在想什么。她也就没说话。

### 2.7 信息释放节奏
**源文写法**：信息释放极其克制。一次只给一点，且常常不是通过叙述直接说明，而是通过角色的行为意外暴露。例如，失忆的设定不是开头交代的，而是通过男主的行为异常让读者自己感知到的。

**仿写指令**：
- **藏问于答**：不要让叙述替角色问问题，然后回答。直接写发生的事，让读者自己产生疑问。
- **细节钩子**：在平淡的日常中，随机掉落一个不寻常的小细节（男主记得她不吃香菜，但想不起任何过去），作为吸引读者往下看的钩子。这个细节必须点到为止，不做解释。

### 2.8 叙事结构
**源文写法**：典型的三段式结构：日常生活→小事件打破平衡→新平衡建立。本章可能聚焦于“小事件打破平衡”（例如，一件不得不一起做的事）。主角为女主，叙事视角紧密跟随女主的所见、所感、所思。

**仿写指令**：
- **严格限制视角**：所有叙述都必须从女主的眼睛和心灵出发。男主的信息（他的过去、他的想法）是绝对的“盲区”，只能通过他的行为和女主的猜测来展现。
- **章内结构**：1. 相对平静的开场 → 2. 一个微小的接触或摩擦（言语/动作） → 3. 情绪积累/波动 → 4. 一个暂时性的、不稳定的平静收尾。

## 3. 本章重点（1-3条）

1.  **描写代替解释**：本章情感张力在于“男主嘴上说不要，身体却很诚实”的反差。务必通过他“冷着脸洗床单”的一系列动作细节（如洗得格外仔细、用力）来体现，一个字都不要写他的心理活动。
2.  **对话的“失忆”感**：在对话中，要体现男主的疏离和试探。他会说出一些基于过去习惯的话，但又立刻意识到“不该知道”，从而戛然而止或转移话题。这种断裂感是本章独特的爽点。
3.  **保持“烦与在意的平衡”**：女主对他的态度不能是单纯的厌恶，也不能是单纯的心疼。而是一种“看不惯他又有点拿他没办法”的复杂情绪，混合着好奇和被打扰的烦躁。
