# 第N章 写章指南

### 全书大弧线衔接
本章处于【阶段一：矛盾建立期】的开始阶段。女主林瑶刚从医院被男主陆止川接回家中（失忆后遗症设定），两人处于一种极度不稳定的“表面夫妻”状态。本章要展现陆止川因愧疚和占有欲而采取的“冷脸照顾”，以及林瑶在陌生环境中的试探与不适。核心是建立“冰山下”的第一层矛盾：**陆止川单方面强加的、令人窒息的“照顾”与林瑶对真实过去被动且懵懂的探寻之间的矛盾。**

### 本章鼓点
1.  **（甜蜜的困惑）** 陆止川反常的居家照顾引发林瑶的疑问。
2.  **（蛰伏的记忆）** 林瑶无意间触碰到与过去有关的事物，产生既视感。
3.  **（冷酷的条件）** 陆止川提出一个交换条件，暴露了他温柔之下的控制欲。
4.  **（虚与委蛇）** 林瑶表面答应，内心计划用自己的方式寻找真相。

### 排除项
1.  **电影/电影院**：源文的核心场景，我们换成超市和家庭内部。
2.  **照片/表情包威胁**：换成更贴合本书设定的“记忆篡改”和“物品替换”，威胁是软性的，源于信息垄断。
3.  **剥虾**：示好与服务的具体动作，我们换成“买衣服”和“纠正记忆”，更具控制意味。
4.  **穿越话题讨论**：换成“巧克力品牌”的认知分歧，直接导向“被篡改的过去”这一核心谜团。
5.  **“你赢了”式的土味情话收尾**：换成更具悬念感和对抗决心的“藏巧克力”动作收尾。
