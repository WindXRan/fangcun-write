## 第一部分：风格指南（8维度+排除项）

### 一、叙事声音与语气

**第三人称限制视角**，贴近楚靳寒的孤寂与卫岢的担忧。叙述声音在孤寂与日常之间切换。

- **语气基调**：孤寂→日常→脱敏→等待。从楚靳寒的孤寂，到日常琐事，再到脱敏治疗和等待。
- **叙述者人格**：善于用细节传递情感，用对话推进情节。

**例句**：
> 精心准备的一桌菜肴，最终只有一条狗，陪他品尝这份盛大又无声的孤寂（第44行）
> "等到，我看见这张椅子，只是张椅子"（第158行）
> 卫岢一时间都分不清，他到底是在改掉习惯，还是在一遍遍加深记忆（第154行）

### 二、对话风格

**对话比例较低**（18.5%），对话用于传递情感和推进情节。

- 王义的对话热情、关心
- 楚靳寒的对话沉默、简短
- 卫岢的对话担忧、不解

**例句**：
> "楚兄弟，你怎么在这？"（第56行）
> "改掉某些习惯。"（第140行）
> "等到，我看见这张椅子，只是张椅子"（第158行）

### 三、场景描写特征

**孤寂型白描**，用空间和动作传递情感。

- 厨房场景：用做饭、碗筷传递孤寂
- 超市场景：用人群、食材传递日常
- 长椅场景：用等待、沉默传递脱敏

**例句**：
> 他站在桌前，看着手里的两副碗筷，再次陷入冗长的沉默（第34行）
> 超市里人声鼎沸，年货堆积如山（第24行）
> 他依旧垂着眸，修长的手指梳理着圈圈背上光滑的毛发（第146行）

### 四、转折与衔接手法

**时间递进式衔接**：用时间推移推动场景转换。

- **空间转换**：厨房→超市→老房子→长椅
- **时间递进**：做饭→购物→散步→等待
- **动作驱动**：用做饭、购物、散步推动情节

**例句**：
> 四十分钟后，楚靳寒拎着超市买回来的食材，走进了厨房（第28行）
> 连着一个星期，楚靳寒都独自生活在这套小房子里（第106行）

### 五、节奏特征

**"孤寂-日常-脱敏-等待"四段结构**：

- 孤寂场景用**细节描写**（碗筷、沉默）
- 日常场景用**动作描写**（做饭、购物）
- 脱敏场景用**对话推进**（习惯、改掉）
- 等待场景用**情绪描写**（沉默、等待）

**段落极短**：98.8%的段落为短段落（<50字），73.2%为单句段落。节奏舒缓，情绪饱满。

### 六、词汇偏好

| 类型 | 偏好 | 例句 |
|------|------|------|
| 口语词 | 日常口语为主 | "习惯""脱敏""等待" |
| 动词 | 偏好日常动词 | "做饭""购物""散步" |
| 形容词 | 孤寂色彩 | "孤寂""沉默""冗长" |
| 连接词 | 偏好"然后""接着" | 时间递进 |
| 情感词 | 使用心理活动 | "孤寂""等待""沉默" |

### 七、情绪表达方式

**孤寂场景+细节反应**，通过：

- **孤寂场景**：用碗筷、沉默传递情感
- **日常场景**：用做饭、购物传递生活
- **等待场景**：用脱敏、等待传递希望

**例句**：
> 精心准备的一桌菜肴，最终只有一条狗，陪他品尝这份盛大又无声的孤寂（第44行）
> "等到，我看见这张椅子，只是张椅子"（第158行）

### 八、独特习惯

1. **"两副碗筷"细节**：用物品传递思念
2. **"超市购物"场景**：用日常传递生活
3. **"脱敏治疗"**：用心理术语传递情感
4. **"椅子只是椅子"**：用哲理传递希望
5. **"圈圈陪伴"**：用宠物传递温暖

### ⚠️ 风格排除项（写章时必须回避）

1. **"两副碗筷"这个细节**：过于标志性，**应创造新的思念细节**
2. **"超市购物"这个场景**：不能照搬，**应换其他日常场景**
3. **"脱敏治疗"这个术语**：过于具体，**应换其他心理术语**
4. **"椅子只是椅子"这句台词**：不能复用，**应换其他哲理台词**
5. **"圈圈陪伴"这个细节**：过于经典，**应换其他宠物陪伴**

## 第二部分：节奏骨架

### 1. 情绪强度变化

```
[高-孤寂] → [中-日常] → [中-脱敏] → [低-等待]
```

整章情绪从高位孤寂开始，降到中位日常和脱敏，最后以低位等待收尾。

### 2. 钩子位置和类型

- **章首钩子**（第44行）：孤寂→**情感悬念**（会怎样？）
- **章中钩子**（第56行）：怎么在这→**悬念**（为什么？）
- **章中钩子**（第140行）：改掉习惯→**悬念**（什么习惯？）
- **章末钩子**（第158行）：椅子只是椅子→**哲理悬念**（什么意思？）

### 3. 爽点公式

**孤寂做饭 → 日常琐事 → 脱敏等待 → 哲理希望**

本章为"孤寂等待章"，爽点在于**楚靳寒的孤寂日常**和**脱敏治疗**形成的情感张力。

### 4. 节奏模式

```
[孤寂做饭] × 6段
+ [日常琐事] × 6段
+ [脱敏等待] × 8段
+ [哲理希望] × 4段
```

具体比例：
- 孤寂做饭线：约25%
- 日常琐事线：约25%
- 脱敏等待线：约30%
- 哲理希望线：约20%

### 5. 场景切换频率

全章**4次场景切换**：
1. 厨房（第4-48行）
2. 超市（第24-28行）
3. 老房子（第52-98行）
4. 长椅（第100-164行）

场景切换频繁，节奏舒缓，情绪递进明显。
