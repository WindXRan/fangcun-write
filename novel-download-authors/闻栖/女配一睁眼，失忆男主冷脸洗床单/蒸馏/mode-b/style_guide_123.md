## 第一部分：风格指南（8维度+排除项）

### 一、叙事声音与语气

**第三人称限制视角**，交替贴近宋云绯和楚靳寒的意识流动。叙述声音口语化，但更内敛、细腻。

- **语气基调**：细腻、敏感、略带忧伤。面对感情变化，情绪是"察觉→困惑→反思"的波动。
- **叙述者人格**：敏感细腻，能捕捉细微的情感变化，像在记录内心日记。

**例句**：
> 果然，两个人相处久了，某些细微的变化轻易就能察觉到。（第80行）
> 自己，好像不知不自觉间，陷得有点深了。（第118行）
> 仿佛昨晚床上留下的那道沟壑突然就被填满了。（第114行）

### 二、对话风格

**对话比例中等**（24.6%），对话用于展现人物关系和内心变化。

- 对话简短，但充满试探和关心
- 何总对话幽默、八卦，带有说教意味
- 人物对话前后常接心理活动或动作描写

**例句**：
> "你回来的挺准时嘛，我还准备给你打电话呢。"（第8行）
> "你还站着干嘛，去洗手啊。"（第14行）
> "那你怎么离婚了？"（第156行）

### 三、场景描写特征

**情感型白描**，用物理细节映射情感状态。

- 空间描写带情感色彩：床中间的缝隙、餐桌上的早餐
- 人物动作细腻：搓手指、咬筷子、托腮发呆
- 环境描写映射内心：冷意从缝隙钻进来、日光挡不住

**例句**：
> 两人中间空着，冷意从中间的缝隙里往里钻，半小时过去了被窝里都还是冷的。（第88行）
> 看到桌上的早餐，仿佛昨晚床上留下的那道沟壑突然就被填满了。（第114行）
> 她缓缓走到餐桌坐了下来，托着腮，看着桌上的早餐发呆。（第122行）

### 四、转折与衔接手法

**情感变化式衔接**：用情感变化推动场景转换。

- **时间流逝**：从晚饭到睡觉，再到第二天中午
- **情感对比**：楚靳寒的异常→宋云绯的困惑→何总的建议
- **细节呼应**：床中间的缝隙→早餐填补沟壑

**例句**：
> 不知道是电视太无聊，还是太困，看着看着睡着了。（第50行）
> 再次醒来，外面已经大亮，窗帘都挡不住外面的日光。（第102行）
> 会议室里，何总喊了楚靳寒好几声，那家伙低垂着眼，也不知道想什么，愣是没反应。（第126行）

### 五、节奏特征

**"细腻-幽默-反思"三段结构**：

- 家庭场景用**短句细腻描写**（情感丰富）
- 会议室场景用**中句对话**（幽默轻松）
- 反思场景用**独立短句**（制造停顿）

**段落极短**：70.5%的段落为单句，平均段落长度19.8字。整章节奏舒缓，情感细腻。

### 六、词汇偏好

| 类型 | 偏好 | 例句 |
|------|------|------|
| 口语词 | 大量使用日常口语 | "说不上来""无精打采""有气无力" |
| 动词 | 偏好具象小动作 | "搓手指""咬筷子""托腮" |
| 形容词 | 少用，用则细腻 | "细微的变化""漆黑的方向""普通不过" |
| 连接词 | 偏好"然后""至于""但是""所以" | 口语化衔接，少用书面连词 |
| 情感词 | 使用情感映射物理细节 | "沟壑""冷意""填补" |

### 七、情绪表达方式

**细节映射+内心独白**，不直接描写情绪，而是通过：

- **物理细节**：用床中间的缝隙、冷意等细节映射情感距离
- **内心独白**：用"说不上来的感觉""陷得有点深"表达情感
- **对比手法**：用楚靳寒的异常和早餐的温暖形成对比

**例句**：
> 两人中间空着，冷意从中间的缝隙里往里钻，半小时过去了被窝里都还是冷的。（第88行）
> 看到桌上的早餐，仿佛昨晚床上留下的那道沟壑突然就被填满了。（第114行）
> 自己，好像不知不自觉间，陷得有点深了。（第118行）

### 八、独特习惯

1. **情感物理化**：用床中间的缝隙、冷意等物理细节映射情感状态
2. **时间流逝描写**：从晚饭到睡觉，再到第二天中午，时间线清晰
3. **幽默对话**：何总的对话幽默、八卦，带有说教意味
4. **细节呼应**：床中间的缝隙→早餐填补沟壑，形成情感呼应
5. **双线叙事**：宋云绯的家庭线和楚靳寒的工作线交替进行

### ⚠️ 风格排除项（写章时必须回避）

以下特征过于独特，模仿会直接暴露"抄源文"，**仿写时必须替换为其他表达方式**：

1. **"床中间的缝隙"作为情感距离象征**：这个意象过于标志性，**应创造新的情感映射方式**
2. **"沟壑被早餐填补"的比喻**：这个比喻不能照搬，**应设计新的情感修复象征**
3. **何总"两个完美的人走不到一起"的理论**：这个观点过于独特，**应创造新的感情建议**
4. **"凌晨三点"的具体时间**：这个时间点不能复用，**应选择其他时间点**
5. **"托腮发呆"的具体动作**：这个动作不能照搬，**应设计新的发呆方式**

## 第二部分：节奏骨架

### 1. 情绪强度变化

```
[低-日常] → [中-敏感] → [低-困惑] → [中-幽默] → [中-反思]
```

整体情绪从日常逐步敏感，中间有幽默调节，最后以反思收尾。情绪曲线是"细腻波动"。

### 2. 钩子位置和类型

- **章首钩子**（第4-6行）：宋云绯做饭等楚靳寒→**日常悬念**（关系如何？）
- **章中钩子**（第34行）：楚靳寒饭没吃完就走→**情感悬念**（发生了什么？）
- **章中钩子**（第86行）：睡觉不搂着→**情感悬念**（关系变化？）
- **章末钩子**（第118行）：意识到陷得深→**情感悬念**（会怎样发展？）

### 3. 爽点公式

**情感变化 → 幽默调节 → 自我反思**

本章为"情感细腻描写"，爽点在于**情感变化的细腻捕捉**和**幽默对话的调节**。从敏感察觉到幽默调节，再到深刻反思。

### 4. 节奏模式

```
[家庭场景-晚饭] × 3段
+ [家庭场景-睡觉] × 4段
+ [工作场景-会议室] × 4段
+ [反思场景-早餐] × 2段
```

具体比例：
- 家庭场景：约45%
- 工作场景：约30%
- 反思场景：约25%

### 5. 场景切换频率

全章**3次场景切换**：
1. 开场：家庭晚饭（第4-46行）
2. 切换：家庭睡觉（第48-120行）
3. 切换：工作会议室（第124-175行）

场景切换较少，本章是"双线交替"的结构，属于典型的**情感变化章**。
