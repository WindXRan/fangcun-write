# 第3章 风格指南

---

## 第一部分：风格指南（8维度+排除项）

### 1. 叙事声音与语气

第三人称有限视角，叙事者像一个"热心旁白"，带有轻微的调侃语气，偶尔替角色内心OS做翻译。不是冷冰冰的上帝视角，而是带着人情味的"解说员"。

- 叙事者会主动替角色做心理推测：「宋云绯暗自松了口气，总算蒙混过去了，楚靳寒心里估计也松了口气吧？」——"估计也松了口气吧？"是叙事者在替读者猜测对方心理。
- 叙事者会用口语化判断句介入：「不怪宋云绯不放心，毕竟她知道楚靳寒的身份。」——这个"不怪"是叙事者在为角色辩护。
- 叙事者偶尔带轻微感叹：「这要是被人抢走了，估计真会气的上吊。」——"真会气的上吊"是叙事者的夸张修辞，不是角色内心独白。

**语气定位**：不严肃、不文艺、不煽情，像朋友在给你讲八卦，偶尔插嘴点评两句。

### 2. 对话风格

对话占比极低（约7%），但每句对话都承担功能——要么推进情节，要么暴露人物关系。

- **对话极简**：单句对话为主，很少有超过两句的连贯对话。角色说完就走，不啰嗦。
  > 「晚上要送外卖，没空。」——说完就接起电话走了，零废话。
- **对话后紧跟动作/心理，不给读者喘息**：
  > 「知道就好，赶紧睡觉！你明天不是还要上班吗？」说完，宋云绯重新关了灯，躺在床上，又往墙壁方向挪了挪。
- **女性角色对话带"酸味"时用语气词和感叹词**：
  > 「你女朋友也太幸福了，不像我……」「哎。」
- **对话不加过多修饰语**：不说"她语带酸意地说"，而是直接写对话内容，让读者自己品。

### 3. 场景描写特征

极简白描，只写"必要的道具"，不铺陈氛围。

- **环境描写不超过一两句**，且都服务于情绪或情节：
  > 「外面繁密的蝉鸣声，伴随着屋里转动的风扇声，让人有种莫名的烦闷。」——蝉鸣+风扇=烦闷，功能明确。
  > 「阳光透过生锈的铁窗落进房间，整个房间一片明亮。」——"生锈的铁窗"交代出租屋的破旧，不展开。
- **人物外貌只写"当下穿着"**，不写固定特征：
  > 「她扎着丸子头，穿着鹅黄色T恤，下半身穿着牛仔短裤。」
- **道具描写是情节触发器**：掉出来的东西→引发情绪→驱动决策，不是为了"好看"而写。

### 4. 转折与衔接手法

**时间跳跃 + 地点标签**，不用过渡句。

- 「不知道过多久，宋云绯才迷迷糊糊的睡着。」→ 噩梦 → 天亮
- 「公司里。」——两个字完成场景切换，干净利落。
- 「因为一整天宋云绯没有查岗，下午，楚靳寒下班就回了租房。」——用因果关系串联时间跳跃，既是衔接也是情节推进。
- **心理活动作为转折枢纽**：角色的想法直接导致下一个行动。「跑路！必须要赶紧跑路！但得先赚到足够的路费才行。」→ 直接跳到找工作。

### 5. 节奏特征

数据佐证：平均句长21字，88.8%段落为单句段，25.3%句子短于15字。

- **短句驱动，几乎没有长段落**。每个段落=一个信息点，读起来像"刷屏"。
- **节奏公式：叙述→心理→动作→叙述**，不断轮转：
  > 叙述：「楚靳寒沉默地看着她。」
  > 动作：「下意识捏了捏手里的东西，并往身后藏了藏。」
  > 心理：「生怕宋云绯看见似的。」
- **信息密度高**：一段话同时完成"交代现状+暗示关系+制造悬念"。例如桌上的鸡蛋和电饭锅的粥，一句话同时交代了楚靳寒的体贴和他的不在场。
- **场景切换时节奏突然加快**，用短句群快速推进。

### 6. 词汇偏好

TTR 0.267，词汇复用率高，偏好口语化、生活化词汇。

- **高频口语词**：「压根」「果然」「说起来」「看来看去」「够呛」「估计」
- **动作词偏好**：「捏了捏」「藏了藏」「擦了又擦」——叠词动词表示重复动作
- **情绪词偏好**：「暗自松了口气」「酸溜溜」「气的牙痒痒」「满头大汗」
- **网络口语**：「查岗」「提桶跑路」「鹤立鸡群」——混用成语和网络词
- **数字夸张**：「十万八千里」「隔半小时」「五六十个」

### 7. 情绪表达方式

**不直接写"她很紧张"，而是写身体反应或内心判断。**

- 用动作泄露情绪：「下意识捏了捏手里的东西，并往身后藏了藏」——紧张+隐瞒。
- 用内心推理表达情绪：「看来他心里果然厌恶自己，不然也不会把这东西藏起来了。」——不是写"她难过"，而是写她的推理过程。
- 用环境烘托：「让人有种莫名的烦闷」——不说"她很烦"，让环境替她说。
- 用夸张口语外化情绪：「这要是被人抢走了，估计真会气的上吊。」——叙事者替角色夸张表达。
- **情绪转折用短句强调**：「跑路！必须要赶紧跑路！」——感叹号+重复，情绪爆发。

### 8. 独特习惯

- **"心理吐槽+叙事"无缝切换**：不加"她想"之类的引导词，直接从叙事滑入角色内心，再滑出来。
- **叙事者会做"读者服务"**：解释背景、点评角色行为，像在跟读者对话。
  > 「也不怪宋云绯不放心，毕竟她知道楚靳寒的身份。」
  > 「这已经是全公司皆知的事情了，现在私底下别人都喊他查岗哥。」
- **段落极短但信息量不小**，每段控制在1-2个信息点。
- **喜欢用"似想到什么"作为动作触发**：
  > 「似想到什么，楚靳寒收起了手机」
  > 「胡瑶眼珠子一转，似想到什么。」
- **章节结尾留画面感**：不以对话或心理结尾，而是以一个具象画面收束，留悬念。

---

### ⚠️ 风格排除项

以下特征过于独特，属于"闻栖签名"，仿写时必须回避，否则会被识别为抄源文：

1. **「十万八千里」形容两人距离**——太标志性，换个说法（"两人中间隔着一条银河"、"恨不得贴到墙上去"）。
2. **「查岗哥」这个绰号**——太具体太记忆点，仿写中不能复用这个梗，可以用类似的但必须换词。
3. **「提桶跑路」**——网络热梗但和角色绑定太紧，换其他口语表达。
4. **「似想到什么」连续使用**——本章出现两次，是作者高频习惯，仿写时最多用一次，或换成"忽然想起"、"念头一闪"。
5. **叙事者直接为角色辩护的口吻**（"也不怪XX不放心"）——用多了会暴露，仿写时偶尔用一次即可。
6. **「他女朋友不上班，那些衣服和手机，不用想就知道是谁买的。」这种"叙事者替读者推理"的句式**——是闻栖的标志性手法，仿写时要减少使用频率。
7. **章节标题风格**（"查岗哥"）——三个字+网络感绰号，太有辨识度，仿写章节标题要换风格。

## 第二部分：节奏骨架

### 1. 情绪强度变化

```
[低] 尴尬沉默 → [中低] 松了口气/日常 → [中] 烦闷睡不着 → [中高] 噩梦惊醒 → [低] 平静早餐 → [中] 决心跑路 → [中低] 公司日常/胡瑶酸 → [中] 被拒/生气 → [中低] 找工作规划 → [中] 回家/悬念画面
```

整体呈**微波状**，没有大起大落，但有两次明显的情绪跳动（噩梦、跑路决心）。

### 2. 钩子位置和类型

- **章末钩子类型：画面悬念+甜**
  > 她扎着丸子头……手里拎着大包小包的菜，热的满头大汗，连身上的T恤胸前也湿了一片。
  - 宋云绯买菜回来=反常行为（以前不干活），楚靳寒视角看到她=好奇+暧昧，留悬念给下一章。
  - 既是"她变了"的反转暗示，也有视觉上的暧昧甜感。

### 3. 爽点公式

```
弱者展示变化 → 强者困惑/好奇
```

具体模式：宋云绯突然不查岗→楚靳寒不习惯→公司同事注意到变化→胡瑶嫉妒但被拒→宋云绯独立找工作/买菜→楚靳寒看到反常的她。

**爽点来源**：信息差——读者知道宋云绯在"跑路计划"，但楚靳寒以为她在变好，这种错位制造阅读快感。

### 4. 节奏模式

```
[1段尴尬/收尾上章] + [2段日常/心理] + [1段噩梦/情绪冲击] + [2段日常/早餐+家务] + [1段发现道具/情绪] + [2段心理决策] + [4段公司副线/胡瑶] + [3段找工作规划] + [1段回家/悬念收尾]
```

**比例**：约3段情绪/冲突 + 6段日常/心理 + 3段副线 + 1段钩子

**节奏特征**：日常为主，冲突为辅，用"反常行为"制造期待感。

### 5. 场景切换频率

本章共**4次场景切换**：

1. 卧室（夜晚同床）→ 卧室（天亮醒来）——时间跳跃
2. 卧室 → 公司（楚靳寒视角）——地点+视角双切
3. 公司 → 出租屋（宋云绯找工作）——地点切换
4. 出租屋 → 出租屋（傍晚回家）——时间推进

**切换手法**：用短句标签（"公司里。"）或因果连接（"因为一整天宋云绯没有查岗，下午……"），平均约每450字切一次场景。
