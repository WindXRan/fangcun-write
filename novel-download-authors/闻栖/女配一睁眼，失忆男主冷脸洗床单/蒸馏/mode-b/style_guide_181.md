## 第一部分：风格指南（8维度+排除项）

### 一、叙事声音与语气

**第三人称限制视角**，贴近宋云绯的噩梦与楚靳寒的关心。叙述声音在噩梦与关心之间切换。

- **语气基调**：噩梦→关心→搞笑→回忆。从宋云绯的噩梦，到楚靳寒的关心，再到搞笑和回忆瑶瑶。
- **叙述者人格**：善于用细节传递情感，用对话推进情节。

**例句**：
> 宋云绯猛然睁开眼，剧烈地咳嗽起来（第172行）
> "怎么了？做噩梦了？"（第178行）
> "都让你别抱这么紧了，害我做噩梦！"（第190行）

### 二、对话风格

**对话比例适中**（约35.2%），对话用于传递情感和推进情节。

- 楚靳寒的对话关心、搞笑
- 宋云绯的对话惊恐、吐槽
- 柏庾的对话调侃、回应

**例句**：
> "怎么了？做噩梦了？"（第178行）
> "都让你别抱这么紧了，害我做噩梦！"（第190行）
> "怎么？才分开几天，就想我了？"（第42行）

### 三、场景描写特征

**噩梦型白描**，用空间和动作传递情感。

- 卧室场景：用噩梦、惊醒传递恐惧
- 电话场景：用调侃、回应传递日常
- 回忆场景：用瑶瑶、落水传递悲伤

**例句**：
> 宋云绯猛然睁开眼，剧烈地咳嗽起来（第172行）
> "怎么？才分开几天，就想我了？"（第42行）
> 她缓缓开口，"我梦到，柏庾和他妹妹了。"（第200行）

### 四、转折与衔接手法

**情绪递进式衔接**：用情绪逐步升级推动场景转换。

- **空间转换**：卧室→电话→卧室
- **情绪递进**：噩梦→关心→搞笑→回忆
- **对话驱动**：用噩梦、关心、回忆推动情节

**例句**：
> "怎么了？做噩梦了？"（第178行）
> 她缓缓开口，"我梦到，柏庾和他妹妹了。"（第200行）

### 五、节奏特征

**"噩梦-关心-搞笑-回忆"四段结构**：

- 噩梦场景用**身体反应**（惊醒、咳嗽）
- 关心场景用**对话推进**（关心、安慰）
- 搞笑场景用**对话描写**（调侃、吐槽）
- 回忆场景用**情绪描写**（噩梦、回忆）

**段落极短**：约95%的段落为短段落（<50字）。节奏紧凑，情绪饱满。

### 六、词汇偏好

### 七、情绪表达方式

**噩梦场景+身体反应**，通过：

- **噩梦场景**：用惊醒、咳嗽传递恐惧
- **关心场景**：用关心、安慰传递情感
- **回忆场景**：用噩梦、回忆传递悲伤

**例句**：
> 宋云绯猛然睁开眼，剧烈地咳嗽起来（第172行）
> "怎么了？做噩梦了？"（第178行）

### 八、独特习惯

1. **"噩梦惊醒"动作**：用动作传递恐惧
2. **"抱太紧"吐槽**：用吐槽传递搞笑
3. **"瑶瑶噩梦"**：用噩梦传递回忆
4. **"验孕试纸"**：用道具传递日常
5. **"柏庾妹妹"**：用回忆传递悲伤

### ⚠️ 风格排除项（写章时必须回避）

1. **"噩梦惊醒"这个动作**：过于标志性，**应创造新的恐惧动作**
2. **"抱太紧"这句吐槽**：不能照搬，**应换其他搞笑吐槽**
3. **"瑶瑶噩梦"这个场景**：过于具体，**应换其他噩梦场景**
4. **"验孕试纸"这个道具**：不能复用，**应换其他日常道具**
5. **"柏庾妹妹"这个回忆**：过于经典，**应换其他回忆场景**

## 第二部分：节奏骨架

### 1. 情绪强度变化

```
[高-噩梦] → [中-关心] → [中-搞笑] → [中-回忆]
```

整章情绪从高位噩梦开始，降到中位关心、搞笑和回忆。

### 2. 钩子位置和类型

- **章首钩子**（第6行）：等了一个多小时→**悬念**（会怎样？）
- **章中钩子**（第172行）：噩梦→**恐惧悬念**（梦到什么？）
- **章中钩子**（第84行）：有了？→**悬念**（怀孕了吗？）
- **章末钩子**（第200行）：柏庾妹妹→**回忆悬念**（梦到什么？）

### 3. 爽点公式

**噩梦惊醒 → 关心安慰 → 搞笑吐槽 → 瑶瑶回忆**

本章为"噩梦回忆章"，爽点在于**宋云绯的噩梦**和**瑶瑶回忆**形成的情感张力。

### 4. 节奏模式

```
[噩梦惊醒] × 6段

+ [瑶瑶回忆] × 6段
```

- 噩梦惊醒线：约25%
- 关心安慰线：约25%
- 搞笑吐槽线：约25%
- 瑶瑶回忆线：约25%
