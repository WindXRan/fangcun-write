> 宋云绯的心狠狠一沉，连忙伸手按亮了墙上的灯（第38行）

- 对话前后常接心理描写

### 四、转折与衔接手法

**情绪递进式衔接**：用情绪逐步升级推动场景转换。

- **空间转换**：浴室→客厅→医院
- **情绪递进**：恐惧→苏醒→感动→结束
- **动作驱动**：用开灯、苏醒、离开推动情节

**例句**：

### 五、节奏特征

**"恐惧-苏醒-感动-结束"四段结构**：

- 恐惧场景用**身体反应**（心沉、开灯）
- 苏醒场景用**对话推进**（瑶瑶、苏醒）
- 感动场景用**情绪描写**（感动、虚弱）
- 结束场景用**动作描写**（离开、结束）

**段落极短**：约95%的段落为短段落（<50字）。节奏紧凑，情绪饱满。

## 第二部分：节奏骨架

### 1. 情绪强度变化

```
[高-恐惧] → [中-苏醒] → [高-感动] → [中-结束]
```

整章情绪从高位恐惧开始，降到中位苏醒，再升到高位感动，最后以中位结束收尾。

### 2. 钩子位置和类型

- **章首钩子**（第6行）：密码打开→**悬念**（会怎样？）
- **章中钩子**（第38行）：开灯→**恐惧悬念**（会看到什么？）
- **章中钩子**（第82行）：瑶瑶→**感动悬念**（会苏醒吗？）
- **章末钩子**（第162行）：瑶瑶回来了→**悬念**（会怎样？）

**密码打开 → 浴缸恐惧 → 瑶瑶呼唤 → 苏醒感动**

本章为"恐惧感动章"，爽点在于**浴缸恐惧**和**瑶瑶呼唤**形成的情感张力。

