## 第一部分：风格指南（8维度+排除项）

### 一、叙事声音与语气

**第三人称限制视角**，贴近宋云绯的恐惧与楚靳寒的复杂。叙述声音在恐惧与温柔之间切换。

- **语气基调**：恐惧→亲密→离别→质问。从宋云绯的恐惧，到亲密接触，再到离别和楚太太质问。
- **叙述者人格**：善于用身体反应传递情感，用对话推进情节。

**例句**：
> 宋云绯紧抿着唇，屏住呼吸，紧张地盯着他（第26行）
> 他缓慢地亲吻着，一寸寸巡弋她的唇齿（第42行）
> "妈，这么着急叫我回来，有什么事？"（第174行）

### 二、对话风格

**对话比例较高**（41.3%），对话用于推进情节和传递情感。

- 楚靳寒的对话温柔、强势交替
- 宋云绯的对话恐惧、撒娇
- 楚太太的对话质问、失望

**例句**：
> "忘了你不方便，我帮你。"（第12行）
> "我舍不得你呀。"（第68行）
> "你为什么要骗我们？"（第184行）

### 三、场景描写特征

**亲密型白描**，用空间和动作传递情感。

- 洗手间场景：用靠近、亲吻传递亲密
- 卧室场景：用离别、不舍传递情感
- 客厅场景：用质问、失望传递冲突

**例句**：
> 两人靠得极近，近到能数清对方的睫毛（第36行）
> 他缓慢地亲吻着，一寸寸巡弋她的唇齿（第42行）
> 楚太太的脸色依旧难看（第176行）

### 四、转折与衔接手法

**情绪递进式衔接**：用情绪逐步升级推动场景转换。

- **空间转换**：洗手间→卧室→门外→客厅
- **情绪递进**：恐惧→亲密→离别→质问
- **动作驱动**：用亲吻、离开、跪下推动情节

**例句**：
> 他揽着她腰的手臂微微用力（第40行）
> 楚靳寒没再继续说下去，站起身转身离开了卧室（第90行）
> "你给我跪下。"（第176行）

### 五、节奏特征

**"恐惧-亲密-离别-质问"四段结构**：

- 恐惧场景用**身体反应**（屏住呼吸、紧张）
- 亲密场景用**动作描写**（亲吻、拥抱）
- 离别场景用**对话推进**（不舍、撒娇）
- 质问场景用**情绪描写**（失望、愤怒）

**段落极短**：98.9%的段落为短段落（<50字），76.8%为单句段落。节奏紧凑，情绪饱满。

### 七、情绪表达方式

**亲密场景+身体反应**，通过：

- **亲密场景**：用亲吻、拥抱传递情感
- **离别场景**：用撒娇、不舍传递情感
- **质问场景**：用失望、愤怒传递冲突

**例句**：
> 他缓慢地亲吻着，一寸寸巡弋她的唇齿（第42行）
> "我舍不得你呀。"（第68行）
> 楚太太的脸色依旧难看（第176行）

### 八、独特习惯

1. **"睫毛数清"描写**：用细节传递亲密
2. **"一寸寸巡弋"**：用缓慢动作传递占有欲
3. **"撒娇不舍"**：用语言传递情感
4. **"跪下"命令**：用强制传递权威
5. **"西红柿炒鸡蛋"禁令**：用奇怪禁令传递控制欲

## 第二部分：节奏骨架

### 1. 情绪强度变化

```
[中-恐惧] → [高-亲密] → [中-离别] → [高-质问]
```

整章情绪从中位恐惧开始，升级到高位亲密，再降到中位离别，最后以高位质问收尾。

### 2. 钩子位置和类型

- **章首钩子**（第12行）：帮忙→**暧昧悬念**（帮什么？）
- **章中钩子**（第42行）：亲吻→**亲密悬念**（会怎样？）
- **章中钩子**（第90行）：离开→**离别悬念**（去哪？）
- **章末钩子**（第176行）：跪下→**冲突悬念**（会怎样？）

### 3. 爽点公式

**恐惧亲密 → 撒娇离别 → 权威质问 → 家庭冲突**

本章为"亲密冲突章"，爽点在于**楚靳寒的亲密占有**和**楚太太的权威质问**形成的对比。

### 4. 节奏模式

```
[恐惧亲密] × 6段
+ [撒娇离别] × 8段
+ [权威质问] × 8段
+ [家庭冲突] × 4段
```

具体比例：
- 恐惧亲密线：约25%
- 撒娇离别线：约30%
- 权威质问线：约30%
- 家庭冲突线：约15%

### 5. 场景切换频率

全章**4次场景切换**：
1. 洗手间（第4-46行）
2. 卧室（第56-116行）
3. 门外（第120-168行）
4. 客厅（第172-190行）

场景切换频繁，节奏紧凑，情绪递进明显。
