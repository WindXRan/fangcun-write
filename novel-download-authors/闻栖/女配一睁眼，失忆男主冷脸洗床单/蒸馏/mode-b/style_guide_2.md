# 第2章 风格指南

---

## 第一部分：风格指南（8维度+排除项）

### 叙事声音与语气

第三人称有限视角，紧贴女主宋云绯。语气轻松调侃，带吐槽感，内心OS直接外化，有"旁白解说"的味道。

- "他声音依然没什么情绪，像个人机一样，估计心里都快吐了吧？"
- "说实话，宋云绯现在挺佩服楚靳寒这心理素质的，这样他都还能让原主怀上孩子？"
- "不得不说，这信念感实在是太强了。"

叙述者与女主视角高度融合，偶尔跳出来"点评"，但不跳出故事。

### 对话风格

对话比例极低（8.7%），以叙事驱动为主。对话简短，符合人物性格：

- 楚靳寒：冷淡、被动、惜字如金——"吃完了就早点睡，明天我还要早起上班。"
- 宋云绯：慌张中带狡黠，急中生智型——"我只是一个人睡得有点害怕，让你陪我睡觉，没让你睡我！"

对话不承载大量信息，更多是推动情绪转折。

### 场景描写特征

**极度精简**，几乎不写环境全貌。只在必要时用一两句细节暗示：

- "大夏天，屋里没有空调，每天只能吹风扇。"
- "镜子上的水印都被擦干净了。"

场景服务于人物行为，不独立成段。环境信息通过角色动作和感受间接带出。

### 转折与衔接手法

1. **动作触发转折**：用具体动作打断当前状态——"忽然身上一沉，男人直接翻身压了上来"
2. **心理急转**：内心纠结→外部事件打断——"宋云绯脑子疯狂急转...这时，楚靳寒忍不住开口"
3. **时间词快速切换**："这时"、"下一秒"、"片刻后"
4. **对话制造反转**：用一句话改变局势走向

### 节奏特征

**碎片化快节奏**：

- 单句段落占比85.9%，几乎每段只说一件事
- 平均段落22字，最长段落不超过48字
- 动作→心理→动作→对话，层层推进
- 亲密场景用短句叠加制造紧张感

节奏公式：**日常铺垫（慢）→ 事件触发（快）→ 情绪高潮（密）→ 落点（缓）**

### 词汇偏好

- **口语化**：臭臭的、磨磨蹭蹭、乱七八糟、嗡嗡的
- **现代感**：人机、信念感、母胎单身
- **身体感知词**：心跳、呼吸、肌肤、细汗、炙热
- **程度副词**：疯狂、猛地、忙不迭

### 情绪表达方式

**动作+身体反应外化**，极少直接写"她很紧张"：

- 紧张："宋云绯身子一僵"、"心跳的更快了"
- 心虚："宋云绯有些心虚"（偶尔直接点明）
- 慌张："说话都有些不利索"、"慌张的提起裤子"
- 害羞："满脸通红"、"额头和脖颈上布满了细汗"

内心活动通过"她知道/她清楚/她想"引出，不做大段心理描写。

### 独特习惯

1. **"了"字结尾**：大量句子以"了"收尾，制造口语松弛感——"她先睡了"、"将饭盒收拾好"
2. **反问式内心独白**：用反问带出信息——"拿什么养啊？让它跟我们一起挤在这个破出租屋里吗？"
3. **旁白式补充**：叙述者插入一句"客观评价"——"只从表面上来说，她现在这副模样是诱人的"
4. **破折号引出转折**：用在心理活动末尾引出动作

### ⚠️ 风格排除项

以下特征过于独特，模仿容易暴露，写章时必须回避：

1. **"人机"**：网络用语，过于时代特定
2. **"信念感太强"**：饭圈/演艺圈术语，不通用
3. **"母胎单身"**：网络流行语，过于特定
4. **"初恋的味道"**：形容长相，过于标志性
5. **具体外貌描写组合**（巴掌大的小脸、小翘鼻、樱桃小嘴）：模板化过强
6. **"跟宋云绯原本的长相有七分相似"**：穿越设定特有表达
7. **直接评价男主"心理素质"**：叙述者跳出来点评的特定角度

## 第二部分：节奏骨架

### 1. 情绪强度变化

```
低（吃饭纠结）→ 中（洗澡观察）→ 高（床上亲密）→ 中高（推开解释）
```

整体呈**上升曲线**，章末落在中高点，留悬念。

### 2. 钩子位置和类型

**章末钩子**：悬念+反转型

女主推开男主后给出"理性理由"，男主反应未交代，读者想知道：
- 男主会生气吗？
- 男主会相信她的理由吗？
- 两人关系会怎么变？

### 3. 爽点公式

```
被动者突然掌握主动权 → 强势者被反制 → 理由正当化行为
```

具体模式：女主拒绝亲密 → 男主被动 → 女主用现实理由合理化拒绝 → 暧昧被打破

### 4. 节奏模式

```
日常铺垫段（吃饭、观察）× 3-4段
+ 转折触发段（洗澡、准备睡觉）× 2-3段
+ 冲突高潮段（亲密、推开）× 5-6段
+ 落点收束段（解释、对峙）× 2段
```

比例约为：**日常30% + 过渡20% + 冲突40% + 收束10%**

### 5. 场景切换频率

全章共**3个场景**，切换2次：

1. 饭桌（吃饭、对话）
2. 洗手间（洗澡、照镜子）
3. 床上（睡觉、亲密、冲突）

场景切换通过时间流动自然衔接，无硬切。核心场景（床上）占全章60%以上篇幅。
