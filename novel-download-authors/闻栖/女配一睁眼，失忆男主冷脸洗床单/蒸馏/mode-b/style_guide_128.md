## 第一部分：风格指南（8维度+排除项）

### 一、叙事声音与语气

**第三人称限制视角**，交替贴近宋云绯和楚靳寒的意识流动。叙述声音口语化，但更富有诗意和深情。

- **语气基调**：深情、试探、略带哲理。面对感情抉择，情绪是"紧张→深情→释然"的波动。
- **叙述者人格**：在深情中展现哲理，像在记录一段充满诗意的感情。

**例句**：
> 像分割出两个不同的她。（第70行）
> 因为只有到了山顶，才能看清来时的路，也才能知道，该往哪里去。（第126行）
> 就像饥肠辘辘的乞丐，手里只有一个馒头，是现在吃，还是等死的时候再吃？（第186行）

### 二、对话风格

**对话比例较低**（14.4%），对话用于展现深情和哲理。

- 楚靳寒对话深情、试探，略带哲理
- 宋云绯对话紧张、回应，略带羞涩
- 对话前后常接心理活动或动作描写

**例句**：
> "你和我一起上山吗？"（第122行）
> "好啊。"（第142行）
> "感冒了，不能传染给你。"（第154行）

### 三、场景描写特征

**情感型白描**，用环境细节映射情感状态。

- 空间描写带情感色彩：卧室、夜灯、阴影
- 人物动作细腻：擦拭胸膛、拥抱、对视
- 环境描写映射内心：昏黄灯光、阴影分割、屏障消散

**例句**：
> 暖色的灯光映照在她的侧脸，一半隐匿在阴影中。（第68行）
> 仿佛这两人弥漫在两人之间那层无形屏障消散，灯光也明亮了许多。（第144行）
> 宋云绯脸颊贴着他比平时热的胸膛，听着他的心跳声，也伸手抱住了他的腰。（第156行）

### 四、转折与衔接手法

**情感升华式衔接**：用情感升华推动场景转换。

- **试探与回应**：楚靳寒试探→宋云绯回应→情感升华
- **时间流逝**：从夜晚到深夜，时间跨度小
- **情感升华**：从紧张到深情，再到释然

**例句**：
> "想起来一点。"（第102行）
> "好啊。"（第142行）
> 人在纠结想不通的时候，最简单的办法就是不去想，怎么能让心里好受，就去怎么做。（第182行）

### 五、节奏特征

**"紧张-深情-释然"三段结构**：

- 试探场景用**短句对话**（紧张试探）
- 深情场景用**中句描写**（情感丰富）
- 释然场景用**独立短句**（哲理升华）

**段落极短**：69.5%的段落为单句，平均段落长度18.1字。整章节奏舒缓，情感丰富。

### 六、词汇偏好

| 类型 | 偏好 | 例句 |
|------|------|------|
| 口语词 | 大量使用日常口语 | "心惊肉跳""不打自招""饥肠辘辘" |
| 动词 | 偏好具象小动作 | "擦拭胸膛""拥抱""对视" |
| 形容词 | 少用，用则深情 | "暖色的灯光""无形屏障""比平时热" |
| 连接词 | 偏好"然后""至于""但是""所以" | 口语化衔接，少用书面连词 |
| 哲理词 | 使用哲理表达情感升华 | "山顶""来时的路""乞丐的馒头" |

### 七、情绪表达方式

**动作细节+哲理比喻**，不直接描写情绪，而是通过：

- **动作细节**：用擦拭胸膛、拥抱等动作展现深情
- **哲理比喻**：用"山顶""乞丐的馒头"等比喻表达情感
- **环境映射**：用灯光、阴影等环境细节映射情感

**例句**：
> 因为只有到了山顶，才能看清来时的路，也才能知道，该往哪里去。（第126行）
> 就像饥肠辘辘的乞丐，手里只有一个馒头，是现在吃，还是等死的时候再吃？（第186行）
> 暖色的灯光映照在她的侧脸，一半隐匿在阴影中。（第68行）

### 八、独特习惯

1. **深情与哲理交织**：对话深情，结尾哲理升华
2. **环境情感化**：用灯光、阴影等环境细节映射情感状态
3. **动作描写丰富**：擦拭胸膛、拥抱等动作描写丰富
4. **哲理比喻**：用"山顶""乞丐的馒头"等比喻增强表现力
5. **情感升华**：从紧张到深情，再到释然，情感逐步升华

### ⚠️ 风格排除项（写章时必须回避）

以下特征过于独特，模仿会直接暴露"抄源文"，**仿写时必须替换为其他表达方式**：

1. **"半山腰"的比喻**：这个比喻过于标志性，**应设计新的比喻方式**
2. **"山顶看清来时的路"的哲理**：这个哲理不能照搬，**应创造新的哲理表达**
3. **"乞丐的馒头"的比喻**：这个比喻不能复用，**应设计新的比喻方式**
4. **"无形屏障消散"的描写**：这个描写不能照搬，**应设计新的情感描写**
5. **"暖色灯光映照侧脸"的细节**：这个细节不能复用，**应设计新的环境描写**

## 第二部分：节奏骨架

### 1. 情绪强度变化

```
[中-紧张] → [高-深情] → [中-释然] → [低-平静] → [中-哲理]
```

整体情绪从紧张到深情，再到释然，最后以哲理收尾。情绪曲线是"波动上升后回落"。

### 2. 钩子位置和类型

- **章首钩子**（第4-6行）：楚靳寒开口→**情感悬念**（会说什么？）
- **章中钩子**（第102行）：想起来一点→**情节悬念**（想起了什么？）
- **章中钩子**（第122行）：你和我一起上山吗→**情感悬念**（会怎样回答？）
- **章末钩子**（第186行）：乞丐的馒头→**哲理悬念**（会怎样选择？）

### 3. 爽点公式

**试探与回应 → 深情告白 → 哲理升华**

本章为"深情与哲理交织"，爽点在于**楚靳寒的深情试探**和**哲理升华**。从紧张到深情，再到哲理，情感逐步深化。

### 4. 节奏模式

```
[试探场景-量体温] × 3段
+ [深情场景-擦拭胸膛] × 4段
+ [情感升华-拥抱] × 3段
+ [哲理场景-反思] × 2段
```

具体比例：
- 试探场景：约20%
- 深情场景：约30%
- 情感升华：约25%
- 哲理场景：约25%

### 5. 场景切换频率

全章**1次场景切换**：
1. 开场：卧室照顾（第4-190行）

场景切换极少，本章是"单场景深化"的结构，属于典型的**深情哲理章**。
