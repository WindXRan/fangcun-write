## 第一部分：风格指南（8维度+排除项）

### 一、叙事声音与语气

**第三人称限制视角**，贴近楚靳寒的压抑与萧娴的直率。叙述声音在压抑与直率之间切换。

- **语气基调**：压抑→直率→反思→孤寂。从楚靳寒的压抑，到萧娴的直率，再到反思和孤寂。
- **叙述者人格**：善于用对话传递情感，用细节推进情节。

**例句**：
> 楚靳寒穿着深色西装，外面套着一件黑色大衣（第20行）
> "跟个哑巴一样，整天摆张臭脸，谁受得了你？"（第92行）
> 楚靳寒就那么坐在沙发上，一动不动（第154行）

### 二、对话风格

**对话比例适中**（28.2%），对话用于传递情感和推进情节。

- 萧娴的对话直率、吐槽
- 楚靳寒的对话沉默、反思
- 对话前后常接心理描写

**例句**：
> "宋云绯是不是把你甩了？"（第50行）
> "跟个哑巴一样，整天摆张臭脸，谁受得了你？"（第92行）
> "我真有这么讨厌？"（第110行）

### 三、场景描写特征

**压抑型白描**，用空间和动作传递情感。

- 客厅场景：用沙发、沉默传递压抑
- 对话场景：用直率、吐槽传递冲突
- 孤寂场景：用黑暗、烟花传递氛围

**例句**：
> 楚靳寒穿着深色西装，外面套着一件黑色大衣（第20行）
> 房间里的一切都没变，和当初一模一样（第42行）
> 外界的光芒越是璀璨喧闹，就越衬得房间里冷清孤寂（第162行）

### 四、转折与衔接手法

**情绪递进式衔接**：用情绪逐步升级推动场景转换。

- **空间转换**：门口→客厅→黑暗房间
- **情绪递进**：压抑→直率→反思→孤寂
- **对话驱动**：用甩了、讨厌、反思推动情节

**例句**：
> "宋云绯是不是把你甩了？"（第50行）
> "我真有这么讨厌？"（第110行）
> 楚靳寒就那么坐在沙发上，一动不动（第154行）

### 五、节奏特征

**"压抑-直率-反思-孤寂"四段结构**：

- 压抑场景用**环境描写**（西装、大衣）
- 直率场景用**对话推进**（甩了、讨厌）
- 反思场景用**心理描写**（沉默、反思）
- 孤寂场景用**情绪描写**（黑暗、烟花）

**段落极短**：93.5%的段落为短段落（<50字），67.4%为单句段落。节奏紧凑，情绪饱满。

### 六、词汇偏好

| 类型 | 偏好 | 例句 |
|------|------|------|
| 口语词 | 日常口语为主 | "甩了""讨厌""哑巴" |
| 动词 | 偏好压抑动词 | "坐着""沉默""盯着" |
| 形容词 | 压抑色彩 | "阴沉""孤寂""冷清" |
| 连接词 | 偏好"可是""但是" | 转折递进 |
| 情感词 | 使用心理活动 | "压抑""反思""孤寂" |

### 七、情绪表达方式

**压抑场景+心理反应**，通过：

- **压抑场景**：用西装、大衣传递压抑
- **直率场景**：用吐槽、直率传递冲突
- **孤寂场景**：用黑暗、烟花传递氛围

**例句**：
> 楚靳寒穿着深色西装，外面套着一件黑色大衣（第20行）
> "跟个哑巴一样，整天摆张臭脸，谁受得住你？"（第92行）
> 外界的光芒越是璀璨喧闹，就越衬得房间里冷清孤寂（第162行）

### 八、独特习惯

1. **"西装大衣"描写**：用穿着传递压抑
2. **"哑巴臭脸"吐槽**：用直率传递冲突
3. **"朋友圈点赞"比喻**：用讽刺传递反思
4. **"黑暗烟花"**：用环境传递孤寂
5. **"照片视频"**：用回忆传递思念

### ⚠️ 风格排除项（写章时必须回避）

1. **"西装大衣"这个描写**：过于标志性，**应创造新的穿着描写**
2. **"哑巴臭脸"这个吐槽**：不能照搬，**应换其他直率吐槽**
3. **"朋友圈点赞"这个比喻**：过于具体，**应换其他讽刺比喻**
4. **"黑暗烟花"这个场景**：不能复用，**应换其他孤寂场景**
5. **"照片视频"这个细节**：过于经典，**应换其他回忆细节**

## 第二部分：节奏骨架

### 1. 情绪强度变化

```
[中-压抑] → [高-直率] → [中-反思] → [高-孤寂]
```

整章情绪从中位压抑开始，升到高位直率，再降到中位反思，最后以高位孤寂收尾。

### 2. 钩子位置和类型

- **章首钩子**（第20行）：西装大衣→**悬念**（会怎样？）
- **章中钩子**（第50行）：被甩了→**情感悬念**（真的吗？）
- **章中钩子**（第110行）：讨厌→**反思悬念**（真的吗？）
- **章末钩子**（第162行）：孤寂→**情感悬念**（会怎样？）

### 3. 爽点公式

**压抑重逢 → 直率吐槽 → 反思自我 → 孤寂思念**

本章为"反思孤寂章"，爽点在于**萧娴的直率吐槽**和**楚靳寒的孤寂思念**形成的情感张力。

### 4. 节奏模式

```
[压抑重逢] × 6段
+ [直率吐槽] × 8段
+ [反思自我] × 6段
+ [孤寂思念] × 6段
```

具体比例：
- 压抑重逢线：约25%
- 直率吐槽线：约30%
- 反思自我线：约25%
- 孤寂思念线：约20%

### 5. 场景切换频率

全章**3次场景切换**：
1. 门口（第4-40行）
2. 客厅（第42-150行）
3. 黑暗房间（第152-182行）

场景切换较少，节奏紧凑，情绪递进明显。
