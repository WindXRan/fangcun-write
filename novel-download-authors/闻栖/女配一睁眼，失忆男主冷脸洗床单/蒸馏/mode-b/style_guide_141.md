## 第一部分：风格指南（8维度+排除项）

### 一、叙事声音与语气

**第三人称限制视角**，贴近宋云绯和楚靳寒的双线意识。叙述声音口语化，兼具轻松调侃与紧张悬念。

- **语气基调**：轻松→孤独→紧张→悬念递进。从何总的八卦调侃，到宋云绯独居的空落，再到楚靳寒家族的暗流涌动。
- **叙述者人格**：善于捕捉生活细节，用日常物件传递情感温度。

**例句**：
> 习惯是真可怕。（第26行）
> 备注是女朋友。是宋云绯拿手机自己改的。（第116-118行）
> 楚靳寒淡淡地看着他，"怎么，你想要？"（第150行）

### 二、对话风格

**对话比例偏低**（24.1%），对话用于推动情节和展现人物关系。

- 何总对话轻松八卦，带试探意味
- 宋云绯与圈圈的自言自语传递孤独感
- 楚靳寒与楚修野对话简短、暗藏锋芒

**例句**：
> "没有啊，他这么跟你说的吗？"（第7行）
> "不知道他什么时候回来，现在只有我们两个相依为命了。"（第20行）
> "怎么，你想要？"（第150行）

### 三、场景描写特征

**生活化白描**，用日常细节传递情感。

- 空间描写带情感色彩：冷清的屋子、空落落的感觉
- 人物动作生活化：蹲地摸狗脑袋、弯腰抱狗、反锁门
- 环境描写简洁：夜色、三轮车突突突

**例句**：
> 回家后屋里一片冷清，但当圈圈扑过来的时候，那种空落落的感觉淡了不少。（第18行）
> 吴伟杵着拐杖，艰难地从车斗里下来。（第38行）
> 突突突的消失在夜色里。（第66行）

### 四、转折与衔接手法

**场景跳切式衔接**：用空行和场景切换推进叙事。

- **空间跳切**：从宋云绯家→街上遇吴伟→回家→楚家别墅
- **时间压缩**："晚上""第二天"等时间词快速推进
- **信息落差**：楚靳寒的家人团聚与宋云绯的孤独形成对比

**例句**：
> ——（第72行，场景切换标记）
> 更大的别墅里，客厅里坐了一圈人。（第74行）

### 五、节奏特征

**"轻松-孤独-紧张-悬念"四段结构**：

- 何总八卦场景用**轻松对话**（调笑调侃）
- 宋云绯独居场景用**短句细腻描写**（孤独感）
- 吴伟出场用**快节奏对话**（紧张信息）
- 楚家场景用**暗流涌动的对话**（悬念铺垫）

**段落极短**：98.8%的段落为短段落（<50字），60.7%为单句段落。整章节奏明快，信息密度高。

### 六、词汇偏好

| 类型 | 偏好 | 例句 |
|------|------|------|
| 口语词 | 大量日常口语 | "裤衩子都快亏没了""牛逼""突突突" |
| 动词 | 偏好具象小动作 | "扑过来""摸了摸脑袋""杵着拐杖" |
| 形容词 | 少用，用则生活化 | "冷清""空落落""雍容华贵" |
| 连接词 | 偏好"但""所以""然后" | 口语化衔接 |
| 情感词 | 用日常感受表达 | "习惯是真可怕""相依为命" |

### 七、情绪表达方式

**日常细节+内心独白**，不直接描写情绪，而是通过：

- **生活细节**：用冷清的屋子、扑过来的狗传递孤独
- **内心独白**：用"习惯是真可怕"表达对依赖的感慨
- **对比手法**：楚家的热闹与宋云绯的冷清形成反差

**例句**：
> 回家后屋里一片冷清，但当圈圈扑过来的时候，那种空落落的感觉淡了不少。（第18行）
> 之前自己独居并没什么感觉，但现在和另外一个人生活久了，自己吃饭总觉得莫名孤独。（第24行）

### 八、独特习惯

1. **空行分隔**：用"——"标记场景切换
2. **备注式细节**："备注是女朋友。是宋云绯拿手机自己改的。"
3. **自言自语**：宋云绯对圈圈说话传递孤独
4. **三轮车意象**："突突突的消失在夜色里"带烟火气
5. **悬念收尾**：兄弟对话戛然而止，留白

### ⚠️ 风格排除项（写章时必须回避）

以下特征过于独特，模仿会直接暴露"抄源文"，**仿写时必须替换为其他表达方式**：

1. **"裤衩子都快亏没了"**：这个口语过于标志性，**应创造新的夸张表达**
2. **"突突突的消失在夜色里"**：三轮车拟声词不能照搬，**应换其他交通工具**
3. **"备注是女朋友"这个设定**：这个细节过于标志性，**应创造新的甜蜜细节**
4. **"雍容华贵"形容母亲**：这个形容词组合不能复用，**应换其他描写**
5. **"怎么，你想要？"这句台词**：这句反问过于经典，**应设计新的对峙台词**

## 第二部分：节奏骨架

### 1. 情绪强度变化

```
[低-轻松] → [中-孤独] → [中-紧张] → [高-悬念]
```

整体情绪从轻松八卦逐步过渡到孤独，再因吴伟出现而紧张，最后在楚家悬念中收尾。

### 2. 钩子位置和类型

- **章首钩子**（第6-8行）：何总试探宋云绯→**八卦悬念**（楚靳寒的朋友是谁？）
- **章中钩子**（第44行）：吴伟被打→**情节悬念**（刘茂才会怎样？）
- **章中钩子**（第90行）：楚靳寒手机响→**身份悬念**（谁在找他？）
- **章末钩子**（第162行）：兄弟对峙→**权力悬念**（董事长之争？）

### 3. 爽点公式

**轻松调侃 → 孤独共鸣 → 紧张信息 → 权力悬念**

本章为"多线铺垫章"，爽点在于**生活细节的真实感**和**悬念的层层递进**。

### 4. 节奏模式

```
[何总八卦] × 3段
+ [宋云绯独居] × 5段
+ [吴伟出场] × 8段
+ [楚家暗流] × 10段
```

具体比例：
- 何总八卦线：约15%
- 宋云绯孤独线：约25%
- 吴伟信息线：约25%
- 楚家权力线：约35%

### 5. 场景切换频率

全章**4次场景切换**：
1. 开场：何总办公室（第4-12行）
2. 切换：宋云绯家（第14-30行）
3. 切换：街上遇吴伟（第32-70行）
4. 切换：楚家别墅（第74-168行）

场景切换频繁，本章属于典型的**多线铺垫章**，为后续情节埋下伏笔。
