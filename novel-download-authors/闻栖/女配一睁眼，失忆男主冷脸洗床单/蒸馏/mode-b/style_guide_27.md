## 八维度风格指南

### 1. 句式节奏
- **短句比例更高**：37.9%句子<15字（五章最高）
- **单句段落**：67.7%
- **节奏特点**：对话密集 → 叙述短句缓冲 → 内心独白长句
- **本章独有**：大量电话对话，形成"电话体"节奏

### 2. 对话风格
- **占比提升至20.9%**，电话对话为主
- **宋云绯台词**：质问式、连珠炮（"不是，你买电视干什么啊？多少钱买的，你哪来的钱？"）
- **楚靳寒台词**：选择性回答（只答第一个问题）、搪塞（"打折的样品机"）
- **何总台词**：吐槽担当（"你小子！"）
- **对话节奏**：宋问一串 → 楚答一句 → 宋追问 → 楚沉默或转移

### 3. 叙述视角
- **以宋云绯为主视角**，楚靳寒段落仅一小段（何总办公室）
- **视角切换用场景分割**：洗手间打电话 → 办公室 → 回家 → 安装现场
- **楚靳寒段落**：外部行为描写，不深入内心

### 4. 情绪密度
- **情绪层次**：震惊 → 质问 → 无奈接受 → 感动 → 孤独共鸣
- **转折点**：从"败家男人"到"遭不住了"的心理转变
- **情绪高潮**：回忆父母 → 幻象消失 → "无边无际的孤独和黑暗"
- **收尾升华**：理解楚靳寒的孤独处境

### 5. 物质细节
- **本章核心意象**：电视、空调、洗衣机——三件家电
- **家电作为情感载体**：不是物品，是"家的气息"
- **感官描写**：洗衣机转动声 + 电视机声 = 烟火气
- **对比手法**：虚假热闹 vs 真实寂静

### 6. 时间处理
- **时间跳跃**：当晚 → 第二天下午 → 傍晚 → 夜晚
- **时间标记**：用事件锚定（"下班前""半小时后""外面已经完全黑了"）
- **不写具体钟点**，用光线和行为暗示

### 7. 内心独白
- **本章独白最长最深**：从物质到精神的层层递进
- **比喻手法**："像个人贩子，把人卖了，还要哄着对方给自己数钱"
- **共情机制**：先写自己的孤独 → 映射楚靳寒的处境
- **独白不解释**，让读者自己体会

### 8. 章末钩子
- **情绪钩子**："厌恶，却只能相依为命，彼此折磨又无法分离"
- **类型**：主题升华式收尾
- **效果**：让读者重新审视两人关系的本质
