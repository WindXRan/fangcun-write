## 八维度风格指南

### 1. 句式节奏
- **短句比例**：40.2%<15字（十章最高）
- **单句段落**：80%
- **节奏特点**：心理活动密集，用短句堆叠紧张感
- **本章独有**：装睡场景用脚步声节奏制造悬念

### 2. 对话风格
- **极简克制**：楚靳寒台词极少（"嗯""你头发没干"）
- **宋云绯内心吐槽**：比对话更丰富
- **潜台词**：浴室玻璃事件后两人尴尬，对话更短更冷

### 3. 叙述视角
- **宋云绯主视角**：内心戏最丰富的一章
- **心理活动外化**：用动作和反应代替内心独白
- **视角功能**：读者跟着宋云绯一起紧张、尴尬、心跳加速

### 4. 情绪密度
- **情绪层次**：尴尬 → 害羞 → 慌乱 → 装睡 → 心跳 → 梦境 → 更尴尬
- **尴尬高潮**：发现浴室玻璃能看到外面
- **暧昧高潮**：梦境场景（擦边但不越线）
- **情绪反复**：装死→被叫醒→继续装→更尴尬

### 5. 外貌描写
- **朦胧轮廓**：浴室玻璃上的身影（若隐若现）
- **不写具体**：只写"朦胧的笔挺的轮廓和身姿"
- **感官叠加**：沐浴露清香+呼吸声+温度

### 6. 场景描写
- **浴室玻璃**：半透明磨砂、暖黄灯光、若隐若现
- **卧室**：黑暗、被窝、床下陷
- **场景功能**：浴室=暧昧尴尬，卧室=装睡紧张

### 7. 人物关系
- **尴尬升级**：浴室玻璃事件后两人更小心翼翼
- **宋云绯心理**：想看又不敢看，想装又装不像
- **楚靳寒表现**：若无其事，但"眼底闪烁着意味不明的情绪"

### 8. 章末钩子
- **梦境回忆**：继续回味昨晚的梦
- **类型**：心理状态延续
- **效果**：暧昧气氛延续到下一章
