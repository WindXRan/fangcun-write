# 第20章 风格指南

---

## 第一部分：8维度风格分析

### 1. 叙事声音与语气

**视角**：第三人称限制视角，主要跟宋云绯，偶尔切楚靳寒内心（末尾余额段落）。

**语气**：轻快、调侃、都市烟火气。叙述者像一个"会吐槽的朋友"，带着对角色的善意揶揄。

| 维度 | 特征 | 原文例句 |
|------|------|----------|
| 叙述立场 | 贴女主视角，偶尔跳男主 | `她的一条腿还搭在男人的腿上` → 末尾 `楚靳寒默默地收回视线` |
| 调侃密度 | 高，几乎每个动作都有轻微吐槽 | `要不是还在喘气，都以为他已经驾鹤西去了` |
| 信息投喂 | 通过角色行动自然带出，不做百科式铺垫 | `夏天天气热，就算平时没有人在家，也会有不少虫子飞进来`（生活细节而非设定说明） |

**核心语调公式**：正经叙述 + 突然一句"不正经"的内心独白或比喻，制造微反差。

---

### 2. 对话风格

**特征**：口语化、短句为主、充满"言不由衷"的推拉感。

| 类型 | 特征 | 原文例句 |
|------|------|----------|
| 夫妻拌嘴 | 一来一回，每轮不超过两句，收束快 | `"你在看什么？"` → `"我刚看见有只耗子钻进床底了"` |
| 双关/擦边 | 用日常词汇暗藏暧昧，表面正经实际擦边 | `"好大一只耗子"` → `"你确定只有这么大？"` |
| 角色区分 | 宋云绯话多、活泼、爱演；楚靳寒话少、冷、偶尔一击致命 | 宋：`"好了！"攥起拳头展示自己并没有的二肱肌` / 楚：`"那就是还没好。"` |
| 邻里对话 | 礼貌但有距离感，服务于信息传递和人物关系暗示 | `"不了嫂子，我们去买点东西就回来了"` |

**对话节奏公式**：提问 → 短答 → 反转/吐槽 → 切话题。极少超过3轮同一话题。

---

### 3. 场景描写特征

**原则**：极简白描，只写与情绪或动作相关的物件，不做氛围铺陈。

| 手法 | 说明 | 原文例句 |
|------|------|----------|
| 一笔带过 | 环境只用半句交代，不独立成段 | `这会儿已经凌晨三点` / `夏天天气热` |
| 物件即情绪 | 用物件暗示角色状态，不直接写心理 | `两百二十三块八毛钱`（无需说"他很穷很窘迫"） |
| 五感克制 | 只用触觉和视觉，不用嗅觉/听觉渲染 | `感觉身上黏糊糊的` / `外面刺眼的太阳` |
| 拒绝长段 | 场景描写永远不超过两句 | `商场里冷气真足，一进门冷的宋云绯打了个哆嗦` |

---

### 4. 转折与衔接手法

| 手法 | 说明 | 原文例句 |
|------|------|----------|
| 时间跳切 | 用"第二天"等时间词硬切，不用过渡段 | `第二天楚靳寒没有去送外卖，所以起来的有点晚` |
| 动作衔接 | 上一段的动作结果直接接下一段的动作发起 | `宋云绯这才稍微放心下来` → `三下五除吃完面条` |
| 话题硬转 | 对话中突然换话题，制造"话没说完就被打断"的真实感 | 楚靳寒没有再继续这个话题，`"你今天感觉怎么样？"` |
| 信息悬念 | 用角色的回避制造未解之谜，不做解释 | `楚靳寒并没有回答她的问题，率先下了楼`（王哥的事留悬念） |

---

### 5. 节奏特征

**整体节奏**：快。短段落 + 短句 + 高对话比 = 极速推进。

| 指标 | 数据 | 含义 |
|------|------|------|
| 平均句长 | 19.2字 | 偏短，口语化 |
| 短句占比 | 30.7%（<15字） | 大量短句穿插 |
| 对话占比 | 22.9% | 中等偏高 |
| 单句段落占比 | 59.1% | 超过一半段落只有一句，节奏极快 |
| 长段占比 | 0% | 几乎没有超过200字段落 |

**节奏公式**：
```
[短叙述] → [对话1-2轮] → [短动作] → [下一个beat]
```
每4-6行完成一个小转折或笑点。

---

### 6. 词汇偏好

**总体**：口语化、日常、不文艺。词汇TTR 0.27，用词重复度高（符合口语特征）。

| 类别 | 偏好词 | 回避词 |
|------|--------|--------|
| 动作 | 撇嘴、小跑、盯着、收回、攥着 | 端详、凝视、踱步 |
| 程度 | 真、挺、有点、特别 | 格外、分外、极为 |
| 感叹 | 啧、哦、嗯 | — |
| 身体 | 脸颊、鼻子、下巴、胸膛、腿 | 肌肤、胴体、纤腰 |
| 状态 | 黏糊糊、板板正正、黑麻麻、刺眼 | 昏黄、氤氲、静谧 |

**ABB/AABB式叠词偏好**：黏糊糊、板板正正、黑麻麻、婀娜多姿。用叠词制造口语质感。

---

### 7. 情绪表达方式

**原则**：Show, don't tell 的"轻量版"——用微动作替代心理独白。

| 情绪 | 表达方式 | 原文例句 |
|------|----------|----------|
| 尴尬 | 摸鼻子、移开视线、干笑 | `宋云绯心虚的摸了摸鼻子` |
| 害羞 | 脸热、反应慢半拍 | `顿时脸颊一热` |
| 心虚 | 演戏、故作惊讶 | `故作惊讶道，"我刚看见有只耗子钻进床底了"` |
| 羡慕 | 盯着看、不说话 | `看着那些漂亮衣服，宋云绯也只能望衣兴叹了` |
| 窘迫/自卑 | 沉默、看手机、垂眼 | `黯然垂下长睫` |
| 感动 | 接过东西、道谢 | `宋云绯感动的接过水杯，道了声谢` |

**关键手法**：从不写"她感到很尴尬""他心里一阵酸楚"，永远用一个动作代替。

---

### 8. 独特习惯

| 习惯 | 说明 | 原文例句 |
|------|------|----------|
| **成语/俗语改造** | 把经典表达改成口语版或反差版 | `驾鹤西去`（正经词用在不正经语境）、`望衣兴叹`（改自望洋兴叹） |
| **"并没有的"修饰** | 用否定句自嘲，制造反差萌 | `攥起拳头展示自己并没有的二肱肌` |
| **数字穷酸法** | 用精确数字暗示窘迫，不说"穷" | `两百二十三块八毛钱` |
| **"似的/般"比喻** | 用夸张比喻强化动作的画面感 | `演的跟真的似的` |
| **末句留白** | 章末不总结、不煽情，用一个安静的动作收束 | `黯然垂下长睫`（无内心独白，纯动作） |
| **邻家烟火** | 穿插邻里互动（李娇）制造生活质感 | 隔壁嫂子出场、约吃饭 |

---

### ⚠️ 风格排除项

以下元素**绝不出现在本风格中**，仿写时必须避免：

| 排除项 | 说明 |
|--------|------|
| ❌ 华丽辞藻 | 不用"氤氲""缱绻""旖旎""婆娑"等文艺词 |
| ❌ 长段心理独白 | 不写超过两句的心理描写，情绪用动作替代 |
| ❌ 说教/感悟 | 角色不总结人生道理，不"顿悟" |
| ❌ 环境渲染段 | 不单独写一段环境描写，环境只嵌入动作句中 |
| ❌ 慢镜头描写 | 不写"时间仿佛静止""那一刻世界只剩下…" |
| ❌ 感叹号轰炸 | 全章感叹号极少，情绪通过内容而非标点传达 |
| ❌ 旁白解说 | 不用"殊不知""岂料""命运弄人"等上帝视角词 |
| ❌ 过度修辞 | 不连续使用比喻，一章最多1-2个比喻 |
| ❌ 章末钩子悬念 | 不用"他不知道，更大的危机正在逼近"这类套路收尾 |

## 第二部分：节奏骨架

### 1. 情绪强度变化

```
强度
 5 │                              ┌─╮
 4 │              ┌─╮            │ │
 3 │  ┌─╮        │ │    ┌─╮    │ │
 2 │  │ │  ┌─╮  │ │    │ │    │ │  ┌─╮
 1 │──╯ ╰─╯  ╰─╯  ╰──╯  ╰──╯  ╰─╯
   └──────────────────────────────────
   起床   耗子  洗漱  邻居  商场  余额
   尴尬   调侃  温馨  微妙  惬意  窘迫
```

| 节点 | 情绪 | 强度 | 手法 |
|------|------|------|------|
| 开场（起床） | 尴尬+暧昧 | 2 | 腿搭腿 + 心虚收腿 |
| 耗子段 | 调侃+害羞 | 4 | 双关擦边，脸颊一热 |
| 日常照顾 | 温馨 | 3 | 泡感冒冲剂、做早餐 |
| 邻居互动 | 微妙+悬念 | 3 | 楚靳寒拒绝，留下未解 |
| 商场蹭空调 | 惬意+轻松 | 2 | 望衣兴叹，生活化 |
| 余额收尾 | 窘迫+心酸 | 5（章内最高） | 精确数字 + 无言动作 |

**情绪走向**：低→高→中→中→低→最高。章末情绪陡升，用窘迫收束，留余韵。

### 2. 钩子位置和类型

| 位置 | 类型 | 内容 |
|------|------|------|
| **第36-37行** | 擦边钩子 | `这薄薄的，宽松的裤子，啧。看着不小……` → 读者好奇接下来发生什么 |
| **第52-63行** | 幽默钩子 | "耗子"双关对话，读者被逗笑想看怎么收场 |
| **第120-121行** | 悬念钩子 | 楚靳寒打断拒绝，`"不了，晚上我们有事"` → 为什么拒绝？跟王哥什么矛盾？ |
| **第172-174行** | 情绪钩子 | `两百二十三块八毛钱` → 读者心疼男主，想看后续经济困境怎么解决 |

**钩子手法总结**：不用"大事悬念"，用**日常细节中的微妙矛盾**勾人。

---

### 3. 爽点公式

```
公式：欲看 → 被打断 → 尬演 → 被反杀 → 羞
```

**第20章实例**：
1. 宋云绯偷看楚靳寒身材 → 被发现 → 编"有耗子" → 楚靳寒反问"你确定只有这么大？" → 脸颊一热

**通用爽点结构**：
| 步骤 | 功能 | 对应原文 |
|------|------|----------|
| ① 欲望/意图 | 角色想做某事（通常是偷看/偷摸/偷想） | 目光顺着胸膛下移 |
| ② 被抓包 | 意图暴露 | "你在看什么？" |
| ③ 找借口 | 角色急中生智，编一个蹩脚理由 | "我刚看见有只耗子" |
| ④ 对方配合演 | 不拆穿，顺着演 | "有多大？""你确定只有这么大？" |
| ⑤ 反转擦边 | 借口反噬，变成暧昧暗示 | 宋云绯反应过来，脸颊一热 |

### 4. 节奏模式

**本章节奏模板**：

```
[日常动作 × 2-3句] → [对话交锋 × 1-2轮] → [微转折/笑点] → [新场景硬切]
```

**场景内节奏**：
```
叙述(1句) → 动作(1句) → 心理(1句，带吐槽) → 对话(2句) → 动作收束(1句)
```

**全章节奏分布**：

| 段落 | 行数 | 节奏 | 类型 |
|------|------|------|------|
| 起床段 | 4-36 | 中速 | 日常+暧昧铺垫 |
| 耗子段 | 38-63 | 快速 | 对话密集，笑点爆发 |
| 照顾段 | 65-82 | 中速 | 温馨日常 |
| 出门段 | 84-100 | 快速 | 短对话，推动行动 |
| 邻居段 | 102-132 | 中速 | 信息+悬念 |
| 商场段 | 134-174 | 慢→快 | 惬意→窘迫，情绪陡转 |

### 5. 场景切换频率

| 切换 | 位置 | 手法 | 间隔 |
|------|------|------|------|
| 卧室→厨房 | 第10行 | 动作衔接 | ~6行 |
| 厨房→浴室→床 | 第12行 | 时间词硬切 | ~2行 |
| 晚上→第二天早上 | 第14行 | "第二天"时间跳 | 2行 |

| 走廊→电动车 | 第134行 | 动作衔接 | ~2行 |
| 路上→商场内 | 第136行 | 到达即切入 | ~2行 |

**切换频率**：全章7次场景切换，平均每24行切一次。

**切换手法**：
- 90% 硬切（直接写新场景的第一个动作）
- 10% 时间词过渡（"第二天"）
- **从不用**"与此同时""另一边"等过渡词
