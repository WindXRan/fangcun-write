## 第一部分：风格指南（8维度+排除项）

### 一、叙事声音与语气

**第三人称限制视角**，贴近宋云绯的忐忑与苏婉华的温和。叙述声音在忐忑与温和之间切换。

- **语气基调**：忐忑→温和→回忆→请求。从宋云绯的忐忑，到苏婉华的温和，再到回忆和请求。
- **叙述者人格**：善于用对话传递情感，用细节推进情节。

**例句**：
> 宋云绯瞬间又开始打退堂鼓，声音发虚（第6行）
> 苏婉华目光灼灼地看着她，眼神里带着一丝不易察觉的怅惘（第102行）
> "我想请你，帮我开导开导他。"（第174行）

### 二、对话风格

**对话比例较高**（约27.5%），对话用于传递情感和推进情节。

- 苏婉华的对话温和、回忆
- 宋云绯的对话忐忑、回应
- 楚靳寒的对话简短、关心

**例句**：

**温和型白描**，用空间和对话传递情感。

- 别墅场景：用沙发、聊天传递日常

- 对话场景：用请求、回应传递情感

**例句**：

### 四、转折与衔接手法

**情绪递进式衔接**：用情绪逐步升级推动场景转换。

- **空间转换**：别墅→咖啡厅
- **情绪递进**：忐忑→温和→回忆→请求
- **对话驱动**：用相像、回忆、请求推动情节

**例句**：
> "你长得，真的和瑶瑶很像。"（第128行）
> "我想请你，帮我开导开导他。"（第174行）

### 五、节奏特征

**"忐忑-温和-回忆-请求"四段结构**：

- 忐忑场景用**身体反应**（打退堂鼓、声音发虚）
- 温和场景用**对话描写**（温和、聊天）
- 回忆场景用**对话推进**（怀表、回忆）
- 请求场景用**情绪描写**（请求、回应）

**段落极短**：约95%的段落为短段落（<50字）。节奏紧凑，情绪饱满。

## 第二部分：节奏骨架

### 1. 情绪强度变化

```
[中-忐忑] → [中-温和] → [中-回忆] → [中-请求]
```

整章情绪持续中位运行，从忐忑到温和再到回忆和请求。

### 2. 钩子位置和类型

- **章首钩子**（第6行）：打退堂鼓→**紧张悬念**（会怎样？）
- **章中钩子**（第102行）：相像→**悬念**（像谁？）
- **章中钩子**（第128行）：瑶瑶→**回忆悬念**（是谁？）
- **章末钩子**（第174行）：开导→**悬念**（会答应吗？）

### 3. 爽点公式

**忐忑见面 → 温和聊天 → 怀表回忆 → 开导请求**

本章为"温情回忆章"，爽点在于**苏婉华的温和回忆**和**开导请求**形成的情感张力。

### 4. 节奏模式

```
[忐忑见面] × 6段
+ [温和聊天] × 6段
+ [怀表回忆] × 8段
+ [开导请求] × 6段
```

具体比例：
- 忐忑见面线：约25%
- 温和聊天线：约25%
- 怀表回忆线：约30%
- 开导请求线：约20%

### 5. 场景切换频率

全章**2次场景切换**：
1. 别墅（第4-52行）
2. 咖啡厅（第54-180行）
