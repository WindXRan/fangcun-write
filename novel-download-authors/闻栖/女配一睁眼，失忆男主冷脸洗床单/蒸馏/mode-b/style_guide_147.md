## 第一部分：风格指南（8维度+排除项）

### 一、叙事声音与语气

**第三人称限制视角**，贴近宋云绯的羞涩与楚靳寒的温柔。叙述声音细腻暧昧，充满亲密感。

- **语气基调**：暧昧→亲密→坦白→甜蜜。从浴室的暧昧气氛，到楚靳寒的温柔照顾，再到宋云绯的坦白和楚靳寒的回应。
- **叙述者人格**：善于用细节描写传递亲密感，用对话制造甜蜜。

**例句**：
> "你不是没力气吗？"他说着，已经伸手帮她解扣子（第54行）
> "想亲你。"（第178行）
> 他忽然低头，吻在了她唇上。（第182行）

### 二、对话风格

**对话比例极低**（15.3%），对话用于传递亲密和甜蜜。

- 宋云绯的对话羞涩、试探、带有心虚
- 楚靳寒的对话温柔、直接、充满宠溺
- 对话前后常接身体接触描写

**例句**：
> "我跟你说正事呢，你先不要急。"（第12行）
> "你这样看着我干什么？"（第144行）
> "想亲你。"（第178行）

### 三、场景描写特征

**暧昧型白描**，用身体细节传递亲密感。

- 浴室场景用雾气、水声、暖黄灯光营造氛围
- 身体描写细腻：手指穿过发丝、脸颊发烫、嘴唇微肿
- 空间描写私密：浴室、卧室、床头

**例句**：
> 水声哗哗响起，不过片刻，浴室里雾气蒸腾，模糊了镜子，也模糊了两人的身影。（第28行）
> 他的手指穿过她有些打结的发丝，仔细地揉搓着。（第62行）
> 楚靳寒盯着她低垂的眼睛，低声问（第172行）

### 四、转折与衔接手法

**情感递进式衔接**：用情感逐步升温推动场景转换。

- **空间跳切**：从浴室→沙发→厨房→卧室
- **情感递进**：暧昧→亲密→坦白→甜蜜
- **时间压缩**：从洗澡到睡觉再到下午六点

**例句**：
> 等洗完澡，宋云绯几乎快睡着了。（第68行）
> 彻底清醒已经是下午六点了。（第138行）

### 五、节奏特征

**"暧昧-亲密-坦白-甜蜜"四段结构**：

- 浴室场景用**细腻感官描写**（暧昧氛围）
- 沙发吹头发用**温柔动作描写**（亲密接触）
- 坦白场景用**紧张对话**（情感转折）
- 接吻场景用**甜蜜短句**（情感高潮）

**段落极短**：98.9%的段落为短段落（<50字），64.1%为单句段落。节奏舒缓暧昧，情感细腻。

### 六、词汇偏好

| 类型 | 偏好 | 例句 |
|------|------|------|
| 口语词 | 日常口语为主 | "没力气""不好意思""正儿八经" |
| 动词 | 偏好亲密动词 | "揉搓""穿过发丝""吻在唇上" |
| 形容词 | 暧昧色彩 | "雾气氤氲""暖黄""昏暗" |
| 连接词 | 偏好"不过""然后""说完" | 口语化衔接 |
| 情感词 | 使用感官表达 | "脸颊发烫""气息不稳""心跳" |

### 七、情绪表达方式

**感官描写+身体反应**，通过：

- **感官描写**：用雾气、水声、暖黄灯光传递暧昧
- **身体反应**：用脸颊发烫、气息不稳传递亲密
- **对话甜蜜**：用"想亲你"直接表达情感

**例句**：
> 浴室里雾气蒸腾，模糊了镜子，也模糊了两人的身影。（第28行）
> 宋云绯脸颊有点发烫（第20行）
> "想亲你。"（第178行）

### 八、独特习惯

1. **浴室雾气**：用雾气模糊视线营造暧昧
2. **手指穿过发丝**：用洗头动作传递亲密
3. **"想亲你"告白**：用直接台词制造甜蜜
4. **噩梦描写**：用踩缝纫机等噩梦传递焦虑
5. **"僵尸"比喻**：用搞笑比喻缓解尴尬

---

### ⚠️ 风格排除项（写章时必须回避）

1. **"浴室雾气模糊镜子"这个场景**：过于标志性，**应换其他暧昧场景**
2. **"手指穿过发丝"这个动作**：不能照搬，**应换其他亲密动作**
3. **"想亲你"这句告白**：过于经典，**应设计新的甜蜜台词**
4. **"踩缝纫机"这个噩梦**：过于具体，**应换其他焦虑表现**
5. **"僵尸"这个比喻**：不能复用，**应换其他搞笑比喻**

## 第二部分：节奏骨架

### 1. 情绪强度变化

```
[中-暧昧] → [中-亲密] → [中-紧张] → [高-甜蜜]
```

整体情绪从暧昧逐步升温，在坦白时略有紧张，最后在接吻处达到甜蜜高峰。

### 2. 钩子位置和类型

- **章首钩子**（第6-8行）：浴室场景→**暧昧悬念**（会发生什么？）
- **章中钩子**（第54行）：帮她解扣子→**亲密悬念**（会继续？）
- **章中钩子**（第162行）：宋云绯要坦白→**情感悬念**（她会说什么？）
- **章末钩子**（第178行）："想亲你"→**甜蜜悬念**（会接吻？）

### 3. 爽点公式

**暧昧氛围 → 亲密接触 → 紧张坦白 → 甜蜜告白**

本章为"亲密升级章"，爽点在于**浴室的暧昧氛围**和**楚靳寒的甜蜜告白**。

### 4. 节奏模式

```
[浴室暧昧] × 6段
+ [沙发亲密] × 6段
+ [坦白紧张] × 5段
+ [甜蜜接吻] × 3段
```

具体比例：
- 浴室暧昧线：约30%
- 沙发亲密线：约30%
- 坦白紧张线：约25%
- 甜蜜接吻线：约15%

### 5. 场景切换频率

全章**4次场景切换**：
1. 开场：浴室暧昧（第4-28行）
2. 切换：沙发吹头发（第68-100行）
3. 切换：厨房做饭（第104-110行）
4. 切换：卧室坦白（第112-184行）

场景切换频繁，本章属于典型的**亲密升级章**，从暧昧到甜蜜逐步升温。
