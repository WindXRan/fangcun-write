# 第7章 风格指南

## 第一部分：8维度风格分析

### 1. 叙事声音与语气

第三人称限制视角，频繁在男女主之间切换内心聚焦。叙述者表面冷淡旁观，偶尔滑入角色主观评价，形成一种"带点毒舌的上帝视角"。

- **冷静旁白切入角色主观**：先客观叙述，再无缝滑入角色内心吐槽。
  > "楚靳寒闻言，敲击键盘的手指微微一顿，很快又恢复正常。"（纯叙述）
  > "没错，宋云绯之前就是这么恐怖。"（滑入角色视角评价）

- **轻喜剧底色**：即便写冲突也不沉重，用日常化语气消解紧张感。
  > "宋云绯越看他越像个人机。"
  > "她恼怒地指了指不远处的大米，'那你去拿一袋吧！'"

- **感叹句/反问句做情绪出口**：叙述者不直接评判，让角色内心独白承担情绪。
  > "这算不算冷暴力？太过分了！"

### 2. 对话风格

对话占比约17.9%，偏低。对话极简，短句为主，大量信息藏在潜台词和动作里。

- **男主台词极简**：一个字到半句话，用最少的字传递距离感。
  > "骑车方便。""嗯。""你来开，我去坐公交。"

- **女主台词活泼+试探**：长句带语气词，但不是话痨，有明确目的（试探男主态度）。
  > "等会你来找我汇合，对面商场抽奖，一等奖送空调！"
  > "楚靳寒，我来了，你怎么把电动车骑回来了？"

- **对话后紧跟动作/心理反应**：不让人物干说，每句对话后都有反馈层。
  > "宋云绯蹙了蹙眉，不知道这家伙答应了没有。"
  > "楚靳寒看着她兴高采烈的模样，眸光微动。"

### 3. 场景描写特征

极简白描，几乎不写环境细节，场景靠人物动作和对话自然带出。

- **场景靠"动线"带出**：不写"商场很热闹"，而是"宋云绯花了好大功夫才挤进去"。
  > "商场人特别多，尤其是门口抽奖的地方，挤都挤不进去。"

- **物件描写服务于情节**：冰箱大小用"跟床头柜差不多大"这种生活化类比，不写精确尺寸。
  > "这冰箱很小，跟床头柜差不多大。宋云绯一个人都能搬起来。"

- **空间转换一笔带过**：不用过渡段，直接切。
  > "半小时后，她终于收到男人的回复了。"

### 4. 转折与衔接手法

- **时间跳跃用短句过渡**：不写"过了很久"，直接给时间节点。
  > "下午，快要下班的时候""半小时后"

- **情绪转折藏在动作里**：不写"他有些在意"，而是写"摸出了手机"然后发现没消息。
  > "片刻后，他摸出了手机。消息一大堆，但没有一条是宋云绯发来的。"

- **内外反差制造张力**：表面冷淡+内心在意，用并置呈现。
  > "楚靳寒终于抬起眼，目光清淡地瞥向她"（外冷）→"楚靳寒也重新将目光投向眼前的电脑屏幕……他却在失神。"（内动）

### 5. 节奏特征

- **单句段落占73.6%**：几乎每段只有一句话，形成快节奏碎片感。
- **短句为主**：30.8%的句子在15字以内，平均句长20.2字。
- **无长段落**：最长段落51字，无超过200字段落。
- **叙事→对话→叙事→对话**交替推进，像镜头切换。
- **情节密度中等**：本章只发生两件事（办公室试探+商场抽奖），但不显拖沓，因为每个小节都有微冲突或微甜。

### 6. 词汇偏好

- **口语化动词**：凑近、打起、挤、搓手、跨下来、犯愁
- **日常化名词**：破电驴、床头柜、小票、纸箱
- **高频副词/程度词**：微微、赶紧、迫不及待、兴高采烈
- **"了"字高频**：用于标记状态变化和完成态，几乎每句都有
- **拟声/状态词少**：不用"砰""哗"之类，靠动词本身传递动态

### 7. 情绪表达方式

- **动作外化**：不写"她很开心"，写"欣喜的站了起来，拿上包就跑了出去"。
  > "宋云绯看到他回复的消息，神色一喜，又急忙给他发了条语音。"

- **微表情/微动作**：用细微身体反应传递情绪。
  > "楚靳寒闻言，敲击键盘的手指微微一顿"
  > "胡瑶脸上的笑容一僵"
  > "楚靳寒看着她兴高采烈的模样，眸光微动"

- **内心独白简短**：用一两句话直接呈现想法，不展开分析。
  > "这个女人。为什么突然转变这么大？"
  > "不知道这家伙答应了没有，去不去倒是给个话啊。"

### 8. 独特习惯

- **人物名做段首主语**：97.7%段落以"楚靳寒""宋云绯""她""他"开头。
- **破折号做场景分隔**：用"——"分割场景转换。
- **旁白补充"背景信息"**：用简短一句话补充前情，像画外音。
  > "没错，宋云绯之前就是这么恐怖。"
  > "这辆电动车是他买来送外卖的二手车，电池不怎么好。"

- **章末停在"未完成动作"上**：不写结果，停在动作进行中。
  > "还不忘回头催促他，'快上来啊。'"（戛然而止）

### ⚠️ 风格排除项（写章时必须回避）

1. **"人机"比喻**：这是本文对男主的专属梗，模仿会被识别为同源。
2. **单句段落占比过高**：73.6%单句段落是本文标志性节奏，仿写时需降到50-60%，否则节奏指纹太明显。
3. **"没错，XX之前就是这么XX"句式**：这是作者的招牌旁白插入方式，需替换为其他补充背景的手法。
4. **破折号场景分隔符**：本文专用格式，仿写应改用空行或时间标记。
5. **抽奖/冰箱等具体情节元素**：这些是情节层，不是风格层，仿写时不得复用。
6. **"微微一顿""微微一怔"**：作者高频用词，每章必出，仿写需换用其他程度副词。

## 第二部分：节奏骨架

### 1. 情绪强度变化

```
[低] 冷淡试探 → [中低] 内心在意但不表露 → [中] 收到消息小欣喜 → [中高] 抽奖紧张期待 → [高] 中奖喜悦 → [甜] 亲密接触
```

整体走势：**低开→缓升→峰值在章末**

### 2. 钩子位置和类型

- **章末钩子**：甜向+悬念。女主蹲在电动车前面催男主上车，亲密场景戛然而止。
- **类型**：甜钩（读者期待下一章的互动发展）+行为悬念（他会上来吗？）

### 3. 爽点公式

```
冷淡男主被女主自然亲近打动 → 内心动摇但表面不显
↓
女主无意间展示"变了" → 男主产生困惑/在意
↓
日常小事中男主暗中配合 → 读者get到隐性甜
```

**抽象公式**：一方改变→另一方困惑→日常互动中暗流涌动→章末亲密升级

### 4. 节奏模式

```
[日常冲突] 办公室胡瑶试探（约30%）
    ↕ 情绪：冷→微恼→内心在意
[过渡] 收消息+出发（约15%）
    ↕ 情绪：平淡→小期待
[日常甜] 超市购物（约25%）
    ↕ 情绪：微吐槽→配合→小满足
[爽点] 抽奖中奖（约20%）
    ↕ 情绪：期待→失望→惊喜
[甜尾] 电动车亲密（约10%）
    ↕ 情绪：心动→戛然而止
```

**比例**：3段日常 + 1段小冲突 + 2段甜 + 1段爽点

### 5. 场景切换频率

本章共 **3个场景**，切换 **2次**：
1. 办公室 → （时间跳跃）→ 手机对话
2. 手机对话 → （空间跳跃）→ 商场/超市/电动车

切换方式均为**时间/空间直接跳跃**，无过渡段，靠一句话带过。平均每800字切一次场景。
