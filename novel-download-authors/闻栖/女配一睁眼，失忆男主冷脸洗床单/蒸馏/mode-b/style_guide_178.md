## 第一部分：风格指南（8维度+排除项）

### 一、叙事声音与语气

**第三人称限制视角**，贴近宋云绯的感动与楚靳寒的深情。叙述声音在感动与深情之间切换。

- **语气基调**：感动→深情→承诺→离开。从宋云绯的感动，到楚靳寒的深情，再到承诺和离开青城。
- **叙述者人格**：善于用动作传递情感，用对话推进情节。

**例句**：
> "你愿不愿意，收下我，嫁给我？"（第12行）
> "现在，我可以光明正大地叫你老婆了么？"（第30行）
> "你自己说过，过完年就结婚"（第36行）

### 二、对话风格

**对话比例适中**（约18.3%），对话用于传递情感和推进情节。

- 楚靳寒的对话深情、承诺
- 宋云绯的对话感动、回应
- 对话前后常接心理描写

**例句**：
> "你愿不愿意，收下我，嫁给我？"（第12行）
> "现在，我可以光明正大地叫你老婆了么？"（第30行）
> "你自己说过，过完年就结婚"（第36行）

### 三、场景描写特征

**浪漫型白描**，用空间和动作传递情感。

- 沙发场景：用跪下、戒指传递浪漫
- 对话场景：用承诺、回应传递深情
- 离开场景：用收拾、离开传递结束

**例句**：
> 他站起身，在宋云绯呆滞的目光中，缓缓屈膝，单膝跪地（第10行）
> "你愿不愿意，收下我，嫁给我？"（第12行）
> 两人下楼，宋云绯终于见到了那辆属于自己的迈巴赫（第118行）

### 四、转折与衔接手法

**情绪递进式衔接**：用情绪逐步升级推动场景转换。

- **空间转换**：沙发→阳台→机场
- **情绪递进**：感动→深情→承诺→离开
- **动作驱动**：用跪下、戴戒指、离开推动情节

**例句**：
> 他站起身，在宋云绯呆滞的目光中，缓缓屈膝，单膝跪地（第10行）
> 两人下楼，宋云绯终于见到了那辆属于自己的迈巴赫（第118行）

### 五、节奏特征

**"感动-深情-承诺-离开"四段结构**：

- 感动场景用**动作描写**（跪下、戴戒指）
- 深情场景用**对话推进**（嫁给我、老婆）
- 承诺场景用**情绪描写**（承诺、回应）
- 离开场景用**动作描写**（收拾、离开）

**段落极短**：约95%的段落为短段落（<50字）。节奏紧凑，情绪饱满。

**浪漫场景+动作反应**，通过：

- **感动场景**：用跪下、戴戒指传递浪漫
- **深情场景**：用嫁给我、老婆传递深情
- **离开场景**：用收拾、离开传递结束

**例句**：

### ⚠️ 风格排除项（写章时必须回避）

1. **"跪下求婚"这个动作**：过于标志性，**应创造新的浪漫动作**
2. **"戴戒指"这个细节**：不能照搬，**应换其他承诺细节**
3. **"老婆"这个称呼**：过于具体，**应换其他甜蜜称呼**
4. **"迈巴赫"这个礼物**：不能复用，**应换其他惊喜礼物**
5. **"离开青城"这个场景**：过于经典，**应换其他离开场景**

## 第二部分：节奏骨架

### 1. 情绪强度变化

```
[高-感动] → [高-深情] → [中-承诺] → [低-离开]
```

整章情绪从高位感动和深情开始，降到中位承诺，最后以低位离开收尾。

### 2. 钩子位置和类型

- **章首钩子**（第10行）：跪下→**浪漫悬念**（会怎样？）
- **章中钩子**（第12行）：嫁给我→**深情悬念**（会答应吗？）
- **章中钩子**（第30行）：老婆→**甜蜜悬念**（会怎样？）
- **章末钩子**（第118行）：迈巴赫→**惊喜悬念**（会怎样？）

### 3. 爽点公式

**跪下求婚 → 戴戒指承诺 → 老婆称呼 → 迈巴赫惊喜**

本章为"浪漫求婚章"，爽点在于**楚靳寒的跪下求婚**和**迈巴赫惊喜**形成的浪漫氛围。

### 4. 节奏模式

```
[跪下求婚] × 6段
+ [戴戒指承诺] × 6段
+ [老婆称呼] × 6段
+ [迈巴赫惊喜] × 6段
```

具体比例：
- 跪下求婚线：约25%
- 戴戒指承诺线：约25%
- 老婆称呼线：约25%
- 迈巴赫惊喜线：约25%

### 5. 场景切换频率

全章**3次场景切换**：
1. 沙发（第4-50行）
2. 阳台（第52-72行）
3. 机场（第118-166行）

场景切换频繁，节奏紧凑，情绪递进明显。
