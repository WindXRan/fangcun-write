## 第一部分：风格指南（8维度+排除项）

### 一、叙事声音与语气

**第三人称限制视角**，贴近楚靳寒的矛盾与宋云绯的等待。叙述声音在幽默喜剧与紧张危机之间剧烈切换。

- **语气基调**：矛盾→喜剧→等待→恐慌。从楚靳寒的左右脑互搏，到卫岢的喜剧 relief，再到宋云绯的机场等待和坠机恐慌。
- **叙述者人格**：善于用幽默化解紧张，再用突发危机制造情感冲击。

**例句**：
> 刚才在胡说八道什么？（第26行）
> 他不会人格分裂了吧？（第34行）
> #银州飞往青州的XXX航班，下午四点三十分，在XX坠毁#（第138行）

### 二、对话风格

**对话比例偏低**（20.6%），对话用于展现人物关系和推动情节。

- 卫岢对话幽默、紧张、带有恐惧感
- 冯医生对话专业、轻松、带有调侃
- 宋云绯的等待用沉默和动作表达

**例句**：
> "楚、楚总，您没事吧？"（第31行）
> "把你的破铃声换了。"（第54行）
> "你是说，他在左右脑互搏？"（第76行）

### 三、场景描写特征

**喜剧与危机并置**，用反差制造情感冲击。

- 喜剧场景用夸张动作：手忙脚乱、手机差点飞出去
- 等待场景用时间流逝：五点、六点、七点、十一点
- 危机场景用感官描写：耳鸣、血液凝固、心脏攥紧

**例句**：
> 卫岢手忙脚乱的掏出手机（第52行）
> 六点。七点。晚上十一点。（第114-118行）
> 霎时间，浑身的血液仿佛凝固一般（第142行）

### 四、转折与衔接手法

**情绪反转式衔接**：用情绪剧烈变化推动场景转换。

- **空间跳切**：从书房→机场→事故新闻→书房
- **情绪反转**：喜剧→等待→恐慌→震惊
- **时间压缩**：用时间点快速推进等待过程

**例句**：
> 周四。（第88行，时间跳切）
> 一条热搜新闻弹了出来。（第136行，危机突降）

### 五、节奏特征

**"矛盾-喜剧-等待-恐慌"四段结构**：

- 楚靳寒矛盾场景用**短句内心独白**（心理冲突）
- 卫岢喜剧场景用**快节奏对话**（幽默 relief）
- 机场等待场景用**时间递进**（焦虑累积）
- 坠机恐慌场景用**感官密集描写**（情感爆发）

**段落极短**：97.9%的段落为短段落（<50字），79.8%为单句段落。节奏变化剧烈，情绪冲击力强。

### 六、词汇偏好

| 类型 | 偏好 | 例句 |
|------|------|------|
| 口语词 | 大量日常口语 | "破铃声""左右脑互搏""悻悻" |
| 动词 | 偏好夸张动作 | "手忙脚乱""差点飞出去""猛然站起身" |
| 形容词 | 感官色彩 | "刺眼而扭曲""冰冷""沉闷钝痛" |
| 连接词 | 偏好"但""不过""然后" | 口语化衔接 |
| 情感词 | 使用感官表达 | "耳鸣""血液凝固""心脏攥紧" |

### 七、情绪表达方式

**感官描写+时间流逝**，通过：

- **感官描写**：用耳鸣、血液凝固、心脏攥紧传递恐慌
- **时间流逝**：用具体时间点传递等待的煎熬
- **喜剧反差**：用卫岢的幽默缓解紧张，再用危机制造冲击

**例句**：
> 霎时间，浑身的血液仿佛凝固一般，世界安静了下来，只能听见一连串的耳鸣（第142行）
> 六点。七点。晚上十一点。（第114-118行）

### 八、独特习惯

1. **时间点递进**：用具体时间点传递等待焦虑
2. **热搜新闻**：用弹窗新闻制造危机感
3. **卫岢的铃声**：用流行歌曲歌词制造喜剧效果
4. **"左右脑互搏"**：用医学解释制造幽默
5. **感官密集描写**：在危机场景集中使用感官词汇

---

### ⚠️ 风格排除项（写章时必须回避）

1. **"酒入喉，添新愁"这个铃声歌词**：过于标志性，**应换其他歌曲**
2. **"左右脑互搏"这个说法**：不能照搬，**应创造新的医学解释**
3. **热搜新闻弹窗这个设定**：过于具体，**应换其他危机发现方式**
4. **"银州飞往青州"这个航班信息**：不能复用，**应换其他航班**
5. **"血液凝固"这个感官描写**：过于经典，**应创造新的恐慌描写**

## 第二部分：节奏骨架

### 1. 情绪强度变化

```
[中-矛盾] → [低-喜剧] → [中-等待] → [高-恐慌]
```

整体情绪从矛盾开始，经过喜剧缓解，在等待中逐步焦虑，最后在坠机新闻处达到恐慌高峰。

### 2. 钩子位置和类型

- **章首钩子**（第26行）：楚靳寒自问"胡说八道什么"→**心理悬念**（他怎么了？）
- **章中钩子**（第34行）：卫岢怀疑人格分裂→**喜剧悬念**（真会分裂？）
- **章中钩子**（第130行）：电话关机→**情感悬念**（为什么关机？）
- **章末钩子**（第186行）：楚靳寒看到新闻→**危机悬念**（他会怎么做？）

### 3. 爽点公式

**心理矛盾 → 喜剧 relief → 等待焦虑 → 危机爆发**

本章为"危机转折章"，爽点在于**喜剧与危机的剧烈反差**和**坠机新闻的突发冲击**。

### 4. 节奏模式

```
[楚靳寒矛盾] × 4段
+ [卫岢喜剧] × 6段
+ [机场等待] × 8段
+ [坠机恐慌] × 6段
+ [书房震惊] × 3段
```

具体比例：
- 楚靳寒矛盾线：约15%
- 卫岢喜剧线：约20%
- 机场等待线：约30%
- 坠机恐慌线：约25%
- 书房震惊线：约10%

### 5. 场景切换频率

全章**4次场景切换**：
1. 开场：楚靳寒书房矛盾（第4-28行）
2. 切换：卫岢喜剧 relief（第30-86行）
3. 切换：宋云绯机场等待（第88-155行）
4. 切换：楚靳寒看到新闻（第158-189行）

场景切换频繁，本章属于典型的**危机转折章**，用喜剧缓冲后突降危机。
