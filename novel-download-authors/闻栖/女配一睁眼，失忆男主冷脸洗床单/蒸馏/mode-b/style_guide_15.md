## 第一部分：风格指南（8维度+排除项）

### 1. 叙事声音与语气

**第三人称限制视角**，紧贴当前焦点角色（宋云绯→楚靳寒），叙述者语气克制、不评论，偶尔带一丝冷幽默旁白。

- 叙述者不解释角色心理，只写行为和对话，让读者自行推断。如：「楚靳寒静静地在旁边看着她溜须拍马，眼底神色难辨。」——只写了"看"和"神色难辨"，不说他想什么。
- 冷幽默穿插在叙述中，不是角色说的，是叙述者的调侃口吻。如：「这马屁算是拍到重点了」「差点咬到自己舌头」。
- 语气始终维持在"旁观者轻描淡写"的基准线上，不煽情、不渲染。

### 2. 对话风格

**对话占比约34%**，但对话承担了绝大部分情节推进和性格展示功能。

- **口语化、短句为主**：角色说话不超过两句话就切换，节奏极快。如：「"你先跟他们走，我骑车。""好吧，那你自己小心点。"」
- **方言/口音制造喜剧效果**：「副四长」→「副市长」→「护士长」，一个口音梗用了三层反转。
- **对话不加修饰语**：大量对话只有引号内容，不写"XX说""XX道"，靠上下文和位置区分说话人。如第46-48行男孩说的话，直接接动作。
- **宋云绯的对话特征**：会说话、能圆场、擅长恭维。面对尴尬局面用话术化解，如打车那段自圆其说。
- **楚靳寒的对话特征**：极简、冷静、不废话。如「追上你了。」「说啊。」
- **吴伟的对话特征**：粗口、威胁、色厉内荏。如「你他妈不想活了？」「你知道我爸是谁吗？」

### 3. 场景描写特征

**极简白描，只写"有用的"**，不做氛围铺陈。

- 环境描写只服务于情节需要。如开头写老小区：「不能说是小区了，更像个村」「墙上大大的'拆'字」——所有描写都指向"拆迁"这个关键信息。
- 不写天气的感受性描写，只写角色对天气的反应：「甚至都不觉得天上的太阳热了」——用心理反应代替景物描写。
- 暴力场景写动作不写感受：「楚靳寒又抬脚朝他膝盖踹过去」「踩住他先前拿手机的那只手」——动词精准，不加"狠狠地""用力地"等副词。

### 4. 转折与衔接手法

- **场景硬切**：段落之间直接跳转，不用过渡句。如第91-92行：「很快就见不到彼此的踪影。不过，他骑车穿过某条巷子时，忽然停了下来。」一个"不过"就切了。
- **信息差制造悬念**：「如果宋云绯在这，一定能认出这两人。」——读者知道但角色不知道，拉读者入局。
- **伏笔回扣**：「一个是她的同事小霜，另外一个，就是之前在宾利上跟她搭讪的那个男人。」——直接点明前文关联，不做暗示。
- **用动作打断对话**推进情节：吴伟话没说完就被踹，「他话还未说完，屁股上猛然传来一股撞击」。

### 5. 节奏特征

**短段落+快节奏，段落平均22字，57%是单句段落。**

- 大量一句话一段，制造阅读速度感。如：「"说啊。"」单独成段。
- 长段只在需要展示连续对话（宋云绯圆场）或动作连续（踹人踩手）时出现。
- 情绪紧张时段落更短，如最后的冲突场景几乎每句话一段。
- 轻松场景（看房对话）段落稍长，允许两三句合一段。

### 6. 词汇偏好

- **动词选择精准且日常**：「攀谈」「松口」「讪讪一笑」「哀嚎」——不生僻但比口语词汇高半格。
- **成语/俗语混用**：「冤家路窄」「狗吃屎」「蚍蜉撼树」「溜须拍马」——用常见成语但不堆砌。
- **不使用形容词堆叠**：描写人物只用一两个关键特征，如「穿白衬衫的男人」「身材高大，目光低垂」。
- **数字和细节具体化**：「十岁左右的男孩」「十套房」——具体数字增加真实感。

### 7. 情绪表达方式

**不写情绪词，写行为外化。**

- 紧张：「宋云绯嘴角保持着微笑」——用"保持微笑"外化紧张。
- 尴尬：「尴尬归尴尬，她还是硬着头皮上前」——直接写"尴尬"但紧跟行动。
- 愤怒：「吴伟骂骂咧咧的从地上爬起来」——用动作而非心理描写。
- 震慑：「这眼神和身上散发出来的气质，着实震慑了吴伟两秒」——写效果不写原因。
- 宋云绯的内心偶尔直接写出：「想到这，宋云绯心情都好了不少」——但很克制，只用一两句。

### 8. 独特习惯

- **章节标题即本章核心台词**：「你完了！」取自吴伟的台词，点题式标题。
- **喜欢用"XX归XX"句式**：「尴尬归尴尬」「不行归不行」（推测全文风格）。
- **喜欢用"还真是"作为心理转折**：「还真是，冤家路窄。」
- **动作描写用"猛然""忽然"制造突发感**：「屁股上猛然传来一股撞击」「忽然停了下来」。
- **叙述中插入反差对比**：楚靳寒的冷静 vs 吴伟的暴躁，宋云绯的圆滑 vs 楚靳寒的沉默。

### ⚠️ 风格排除项

以下是**绝对不能出现**的写法：

| 排除项 | 说明 |
|--------|------|
| 心理独白大段倾泻 | 不允许超过3句连续心理描写 |
| 环境氛围渲染 | 不写"夕阳西下""微风拂面"等抒情景物 |
| 感叹号过度使用 | 对话中感叹号仅限角色愤怒/惊呼，叙述不用 |
| 副词修饰动作 | 不写"狠狠地""用力地""轻轻地"，动词本身要精准 |
| 破折号解释说明 | 不用破折号做补充解释，信息融入叙事 |
| 省略号制造悬念 | 不用"……"代替情节，该写什么写什么 |
| 诗意化/散文化语言 | 不写排比、不写对仗、不写意象堆叠 |
| "仿佛""好像""似乎"过多 | 比喻不超过1处/章，叙述不做模糊推断 |
| 角色内心OS解释情节 | 不让角色在心里复盘"原来如此是因为……" |
| 过度使用"了"结尾 | 避免连续多个句子以"了"结尾 |

## 第二部分：节奏骨架

### 1. 情绪强度变化

```
强度
 5 │                                              ┌─踩手质问
 4 │                                  ┌─踹人───┘
 3 │            ┌─口音梗反转──圆场──┘
 2 │  ┌─看房尴尬─┘
 1 │──┘  到达
   └────────────────────────────────────────────────→ 行数
    1    20     40       60       80    100   130  160
```

- **低开**（1-22行）：到达老小区，平淡叙事+轻微好奇（拆迁信息）
- **小升**（23-88行）：看房尴尬→口音梗喜剧→圆场化解，情绪在3左右浮动，有喜剧缓冲
- **硬切**（89-92行）：楚靳寒单独行动，叙事节奏骤然收紧
- **攀升**（93-158行）：小霜被骚扰→楚靳寒出手→踹人→踩手→质问，情绪阶梯式上升至5，**章末断在最高点**

### 2. 钩子位置和类型

| 位置 | 类型 | 内容 |
|------|------|------|
| 第1行（章标题） | **情绪钩子** | 「你完了！」——读者好奇谁完了 |
| 第12行 | **信息钩子** | 墙上「拆」字——拆迁=钱，拉住读者 |
| 第92行 | **悬念钩子** | 「忽然停了下来」——他发现了什么？ |
| 第114-116行 | **命运钩子** | 「如果宋云绯在这，一定能认出这两人」——信息差，读者想知道后续 |
| 第140行 | **人格钩子** | 「追上你了。」——楚靳寒的冷酷一面首次完整展示 |
| 第158行（章末） | **断点钩子** | 「说啊。」——断在施压最高点，读者必须翻页 |

### 3. 爽点公式

**「弱者被欺负 → 强者出手 → 恶人受辱」三层递进**

1. **铺垫委屈**：小霜被吴伟纠缠骚扰（弱者困境）
2. **强者介入**：楚靳寒不说话直接踹人（无废话动手）
3. **碾压展示**：踩手+质问「你爸是谁？」（以其人之道还治其人之身——吴伟刚炫耀完"我爸"就被反过来问）
4. **留白断章**：不给结果，让读者脑补更爽的画面

**附加爽点**：宋云绯的社交能力也是一种"爽"——她用嘴皮子化解尴尬，跟楚靳寒用拳头解决问题形成对照。

### 4. 节奏模式

```
[叙事铺垫] → [对话推进] → [喜剧缓冲] → [硬切转折] → [动作升级] → [断章]

具体拆解：
L1-22:   叙事铺垫（到达+环境+拆迁信息）
L23-88:  对话推进+喜剧缓冲（看房尴尬→口音梗→圆场）
L89-92:  硬切转折（楚靳寒独行）
L93-158: 动作升级（发现→介入→踹→踩→质问）
L158:    断章（最高点戛然而止）
```

**关键特征**：前半章是"文戏"（社交/对话/喜剧），后半章是"武戏"（暴力/冲突/爽感），中间用一个"不过"硬切，不拖泥带水。

### 5. 场景切换频率

| 场景 | 行数 | 时长占比 |
|------|------|----------|
| 老小区巷口（等赵先生） | 1-23 | 14% |
| 出租车+路上 | 80-91 | 7% |
| 巷子内（吴伟骚扰小霜） | 92-158 | 42% |

- **场景数**：3个
- **切换频率**：低。本章以两个主场景为核心，不做频繁跳转。
- **场景一（看房）**和**场景二（巷子冲突）**形成平行对照：前者用"嘴"解决问题，后者用"手"解决问题。
- 楚靳寒在场景一中是旁观者，在场景二中是主导者——**角色功能在同一章内完成切换**。
