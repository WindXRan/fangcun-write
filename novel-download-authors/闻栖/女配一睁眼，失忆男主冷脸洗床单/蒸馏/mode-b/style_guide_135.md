## 第一部分：风格指南（8维度+排除项）

### 一、叙事声音与语气

**第三人称限制视角**，交替贴近楚靳寒、卫岢和宋云绯的意识流动。叙述声音口语化，充满戏剧性和幽默。

- **语气基调**：戏剧、幽默、略带紧张。面对重逢和买房，情绪是"紧张→幽默→温馨"的波动。
- **叙述者人格**：在戏剧中展现幽默，像在记录一段既戏剧又搞笑的故事。

**例句**：
> 卫岢闻言，立马回头看去。（第19行）
> 楚靳寒沉吟了一下，缓缓道，"你既然口口声声说是我最得力的助理，那我应该很有钱，你先给我一百万。"（第74行）
> 在看到他骑上一辆粉色电动车的时候，眼睛瞪得像铜铃，瞬间感觉天塌了。（第124行）

### 二、对话风格

**对话比例中等**（28.4%），对话用于展现戏剧和幽默。

- 卫岢对话激动、委屈，略带无奈
- 楚靳寒对话冷静、试探，略带幽默
- 宋云绯对话日常、好奇，略带紧张
- 对话前后常接心理活动或动作描写

**例句**：
> "楚总，您…不认识我了？"（第42行）
> "你先给我一百万。"（第74行）
> "买下来，也省的搬了。"（第162行）

### 三、场景描写特征

**戏剧型白描**，用动作细节展现戏剧。

- 空间描写带戏剧气息：咖啡厅、包厢、餐桌
- 人物动作细腻：激动站起、跌坐沙发、骑电动车
- 环境描写映射内心：表情变换、眼神淡漠、脸颊发热

**例句**：
> 淡定的脸上瞬间表情变换，激动的直接站了起来。（第20行）
> 卫岢跌坐回沙发，脸上写满了失落，担忧和心疼。（第48行）
> 在看到他骑上一辆粉色电动车的时候，眼睛瞪得像铜铃，瞬间感觉天塌了。（第124行）

### 四、转折与衔接手法

**戏剧变化式衔接**：用戏剧变化推动场景转换。

- **戏剧重逢**：激动→冷静→试探
- **幽默互动**：要钱→买房→认怂
- **时间流逝**：从晚上到深夜，时间跨度小

**例句**：
> "楚总，您…不认识我了？"（第42行）
> "你先给我一百万。"（第74行）
> "买下来，也省的搬了。"（第162行）

### 五、节奏特征

**"戏剧-幽默-温馨"三段结构**：

- 戏剧场景用**中句对话**（紧张戏剧）
- 幽默场景用**短句对话**（幽默搞笑）
- 温馨场景用**独立短句**（情感升华）

**段落较短**：60%的段落为单句，平均段落长度20.4字。整章节奏紧张，对话密集。

### 六、词汇偏好

| 类型 | 偏好 | 例句 |
|------|------|------|
| 口语词 | 大量使用日常口语 | "天塌了""认怂""假笑" |
| 动词 | 偏好具象小动作 | "激动站起""跌坐""骑电动车" |
| 形容词 | 少用，用则戏剧 | "剑眉星目""一身正气""面无表情" |
| 连接词 | 偏好"然后""至于""但是""所以" | 口语化衔接，少用书面连词 |
| 戏剧词 | 使用戏剧表达增强氛围 | "重逢""试探""要钱" |

### 七、情绪表达方式

**动作细节+对话内容**，不直接描写情绪，而是通过：

- **动作细节**：用激动站起、跌坐等动作展现戏剧
- **对话内容**：用对话内容展现人物性格和关系
- **环境映射**：用表情变换、眼神淡漠等细节映射情感

**例句**：
> 淡定的脸上瞬间表情变换，激动的直接站了起来。（第20行）
> 卫岢跌坐回沙发，脸上写满了失落，担忧和心疼。（第48行）
> 在看到他骑上一辆粉色电动车的时候，眼睛瞪得像铜铃，瞬间感觉天塌了。（第124行）

### 八、独特习惯

1. **戏剧与幽默交织**：戏剧场景中带有幽默元素
2. **环境情感化**：用表情变换、眼神淡漠等细节映射情感状态
3. **动作描写丰富**：激动站起、跌坐等动作描写丰富
4. **对话密集**：对话比例较高，推动情节发展
5. **多线叙事**：楚靳寒和卫岢的重逢线，楚靳寒和宋云绯的买房线

### ⚠️ 风格排除项（写章时必须回避）

以下特征过于独特，模仿会直接暴露"抄源文"，**仿写时必须替换为其他表达方式**：

1. **"给我一百万"的具体要求**：这个要求过于标志性，**应设计新的试探方式**
2. **"眼睛瞪得像铜铃"的比喻**：这个比喻不能照搬，**应设计新的比喻方式**
3. **"粉色电动车"的具体细节**：这个细节不能复用，**应设计新的交通工具**
4. **"认怂"的具体表现**：这个表现不能照搬，**应设计新的妥协方式**
5. **"天塌了"的夸张**：这个夸张不能复用，**应设计新的夸张表达**

## 第二部分：节奏骨架

### 1. 情绪强度变化

```
[中-紧张] → [高-戏剧] → [中-幽默] → [低-温馨] → [中-紧张]
```

整体情绪从紧张到戏剧，再到幽默，最后以温馨收尾。情绪曲线是"波动起伏"。

### 2. 钩子位置和类型

- **章首钩子**（第4-6行）：咖啡厅见面→**戏剧悬念**（会怎样重逢？）
- **章中钩子**（第42行）：不认识我了→**情节悬念**（会怎样发展？）
- **章中钩子**（第74行）：给我一百万→**幽默悬念**（会怎样回应？）
- **章末钩子**（第162行）：买下来→**情感悬念**（会怎样发展？）

### 3. 爽点公式

**戏剧重逢 → 幽默试探 → 温馨买房**

本章为"戏剧与幽默交织"，爽点在于**戏剧重逢的紧张感**和**幽默试探的趣味性**。从戏剧到幽默，再到温馨，情感逐步深化。

### 4. 节奏模式

```
[戏剧场景-重逢] × 4段
+ [幽默场景-要钱] × 4段
+ [温馨场景-买房] × 3段
```

具体比例：
- 戏剧场景：约35%
- 幽默场景：约35%
- 温馨场景：约30%

### 5. 场景切换频率

全章**2次场景切换**：
1. 开场：咖啡厅重逢（第4-128行）
2. 切换：家庭买房（第130-190行）

场景切换较少，本章是"双线交替"的结构，属于典型的**戏剧幽默章**。
