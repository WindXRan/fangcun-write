## 第一部分：风格指南（8维度+排除项）

### 一、叙事声音与语气

**第三人称限制视角**，贴近宋云绯的感动与楚靳寒的深情。叙述声音在感动与深情之间切换。

- **语气基调**：感动→深情→回忆→离开。从宋云绯的感动，到楚靳寒的深情，再到回忆和离开伦敦。
- **叙述者人格**：善于用对话传递情感，用细节推进情节。

**例句**：
> "从来都不是你需要我，而是我需要你。"（第114行）
> "我们的相遇，难道不是更匪夷所思吗？"（第68行）
> "又说这种话。"（第76行）

### 二、对话风格

**对话比例适中**（约28.7%），对话用于传递情感和推进情节。

- 楚靳寒的对话深情、认真
- 宋云绯的对话感动、调皮
- 对话前后常接心理描写

**例句**：
> "从来都不是你需要我，而是我需要你。"（第114行）
> "我们的相遇，难道不是更匪夷所思吗？"（第68行）
> "又说这种话。"（第76行）

### 三、场景描写特征

**深情型白描**，用空间和动作传递情感。

- 酒店场景：用拥抱、对话传递深情
- 回忆场景：用童年、孤独传递感动
- 离开场景：用收拾、离开传递结束

**例句**：
> "你一定没体会过，小学一年级就排满行程的生活"（第84行）
> "从来都不是你需要我，而是我需要你。"（第114行）
> 她的东西其实挺多的，大多都是柏庾买的（第160行）

### 四、转折与衔接手法

**情绪递进式衔接**：用情绪逐步升级推动场景转换。

- **空间转换**：酒店→伦敦街头→公寓
- **情绪递进**：感动→深情→回忆→离开
- **对话驱动**：用需要、回忆、离开推动情节

**例句**：
> "从来都不是你需要我，而是我需要你。"（第114行）
> 她的东西其实挺多的，大多都是柏庾买的（第160行）

### 五、节奏特征

**"感动-深情-回忆-离开"四段结构**：

- 感动场景用**对话描写**（感动、回应）
- 深情场景用**动作描写**（拥抱、亲吻）
- 回忆场景用**对话推进**（童年、孤独）
- 离开场景用**情绪描写**（离开、结束）

**段落极短**：约95%的段落为短段落（<50字）。节奏紧凑，情绪饱满。

### 六、词汇偏好

| 类型 | 偏好 | 例句 |
|------|------|------|
| 口语词 | 日常口语为主 | "需要""回忆""离开" |
| 动词 | 偏好深情动词 | "拥抱""亲吻""离开" |
| 形容词 | 深情色彩 | "感动""深情""认真" |
| 连接词 | 偏好"因为""所以" | 因果递进 |
| 情感词 | 使用心理活动 | "感动""深情""回忆" |

### 七、情绪表达方式

**深情场景+动作反应**，通过：

- **感动场景**：用感动、回应传递情感
- **深情场景**：用拥抱、亲吻传递深情
- **离开场景**：用离开、收拾传递结束

**例句**：
> "从来都不是你需要我，而是我需要你。"（第114行）
> 她的东西其实挺多的，大多都是柏庾买的（第160行）

### 八、独特习惯

1. **"需要"告白**：用告白传递深情
2. **"童年回忆"**：用回忆传递感动
3. **"伦敦雨伞"**：用道具传递浪漫
4. **"收拾离开"**：用动作传递结束
5. **"奢侈品挑出"**：用细节传递占有欲

### ⚠️ 风格排除项（写章时必须回避）

1. **"需要"这句告白**：过于标志性，**应创造新的告白台词**
2. **"童年回忆"这个场景**：不能照搬，**应换其他回忆场景**
3. **"伦敦雨伞"这个道具**：过于具体，**应换其他浪漫道具**
4. **"收拾离开"这个动作**：不能复用，**应换其他离开动作**
5. **"奢侈品挑出"这个细节**：过于经典，**应换其他占有欲细节**

## 第二部分：节奏骨架

### 1. 情绪强度变化

```
[中-感动] → [高-深情] → [中-回忆] → [低-离开]
```

整章情绪从中位感动开始，升到高位深情，再降到中位回忆，最后以低位离开收尾。

### 2. 钩子位置和类型

- **章首钩子**（第10行）：坦白穿越→**悬念**（会信吗？）
- **章中钩子**（第114行）：需要→**深情悬念**（需要什么？）
- **章中钩子**（第84行）：童年→**回忆悬念**（什么童年？）
- **章末钩子**（第176行）：挑出奢侈品→**悬念**（会怎样？）

### 3. 爽点公式

**感动回应 → 深情告白 → 童年回忆 → 离开伦敦**

本章为"深情回忆章"，爽点在于**楚靳寒的深情告白**和**童年回忆**形成的情感张力。

### 4. 节奏模式

```
[感动回应] × 6段
+ [深情告白] × 8段
+ [童年回忆] × 6段
+ [离开伦敦] × 6段
```

具体比例：
- 感动回应线：约25%
- 深情告白线：约30%
- 童年回忆线：约25%
- 离开伦敦线：约20%

### 5. 场景切换频率

全章**3次场景切换**：
1. 酒店（第4-40行）
2. 伦敦街头（第42-136行）
3. 公寓（第138-178行）

场景切换频繁，节奏紧凑，情绪递进明显。
