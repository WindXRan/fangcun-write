## 第一部分：风格指南（8维度+排除项）

### 一、叙事声音与语气

第三人称限知视角，交替聚焦宋云绯和楚靳寒。叙述语气轻松口语化，带轻微自嘲和吐槽感，像一个会讲段子的朋友在转述故事。叙述者不端着，常用网络化短句制造节奏感。

- 女主视角的吐槽式内心独白是核心调味料：
  > "宋云绯嘴角狂抽。"
  > "坏了，这家伙好像真被那个女人刺激到了，开始怀疑人生了。"
- 叙述中夹带轻微调侃，但不过分：
  > "长得像鲶鱼，但他开宝马。"
  > "小城市就是这点好，不禁电动车，汽车和电动车乱窜。"
- 男主视角的叙述偏内敛、克制，用短句和留白：
  > "楚靳寒在电动车上沉默着。"
  > "仿佛自己拥有过一样。"

### 二、对话风格

对话极简，每句不超过20字为常态，像日常聊天的碎片。对话不承载大量信息，更多是情绪试探和关系博弈。角色通过对话"摸底"对方，而非直接表达。

- 试探型对话，一问一答节奏快：
  > "这位是你男朋友？" / "嗯，是的。"
  > "他是外卖员？" / "兼职的。"
- 关键情感对话用短句+反问制造压力：
  > "如果我一直这么穷下去，你还会跟着我吗？"
  > "真的么？"
- 女主的内心独白和嘴上说的形成反差，对话背后藏着"不能说的秘密"：
  > 心里想：大佬你不穷啊！你可是宴金集团的总裁！
  > 嘴上说："我不委屈，我们穷只是暂时的。"

### 三、场景描写特征

极简白描，几乎不用比喻和修饰。场景描写只保留"动作+必要物件"，环境细节靠一两个词带过，不做氛围渲染。人物外貌通过碎片式细节而非整体描写。

- 环境极简，只点出关键物件：
  > "电动车的电池不太行，两天没充电了。"
  > "冰箱通电，正常运行。"
- 人物外貌用碎片细节，不做全景：
  > "白皙的后颈，几缕碎发贴在肌肤上，还有一层细汗。"
  > "一头卷发铺在脑后，长得很漂亮，气质冷艳。"
- 感官细节偏视觉和触觉，极少用听觉/嗅觉：
  > "衬衣布料下的肌肤。"
  > "大热天的，走了两公里，身上都湿透了。"

### 四、转折与衔接手法

场景切换用"—"分隔符硬切。段落间衔接靠动作链自然过渡，不用"突然""忽然"等词频繁开头（偶尔用）。转折多通过"对话打断"或"视角切换"实现，而非叙述性过渡。

- 硬切场景用"—"：
  > "—"
  > "等把东西搬回家后，宋云绯已经累得瘫倒在沙发上了。"
- 对话打断推进节奏，不给读者喘息：
  > 楚靳寒内心独白后 → 宋云绯察觉 → 直接问"你怎么了？"
- 视角切换制造信息差（侧写车辆里的神秘女人）：
  > 从主CP推车 → 切到奔驰车内女人的视角 → 留悬念

### 五、节奏特征

短段落为主体（73.4%为单句段），形成"一击一停"的节奏感。长段落仅在关键情感/心理描写时出现。整体节奏：日常铺垫（轻快）→ 外部刺激（中）→ 情感试探（高）→ 悬念收尾（留白）。

- 单句段密集，制造阅读惯性：
  > "油门启动，电动车缓缓驶了出去。"
  > "狗子也很热情的凑过来舔她的脸。"
- 心理描写是唯一允许"拉长"的地方，但也不超过50字：
  > "宋云绯能品出来她这眼神的含义，优越感带着几分轻蔑。好像在说，长得帅有什么用，还不是骑个破电动车。"
- 节奏器：对话短句 → 叙述推进 → 内心吐槽 → 再对话，形成循环。

### 六、词汇偏好

口语化现代词汇为主，不用四字成语堆砌。偏好身体动作类动词（盯、抬、推、捏、攥）和状态形容词（沉默、坦然、复杂）。避免文学化修辞。

- 高频动词：盯、看、推、捏、攥、收回、抬、吐出
- 口语化表达："没招了""坏了""分币没有""手到擒来"
- 状态词偏克制：沉默、坦然、复杂、低落、认真
- 网络化用语适度穿插："小鸡逐米似的点头""鲶鱼"

### 七、情绪表达方式

核心手法是"外冷内热"——外在动作克制，内心独白剧烈。情绪通过身体反应传递（手抖、吐气、嘴角抽搐），不用"他很感动""她很紧张"等直述。

- 身体反应代替情绪直述：
  > "她的手不禁抖了抖，小心脏也跟着抖了抖。"（紧张+心动）
  > "宋云绯偷偷的吐出一口气。"（如释重负）
- 内心独白用短句+感叹号爆发：
  > "大佬你不穷啊！你可是宴金集团的总裁，怎么会穷！"
- 男主情绪通过"沉默"和"目光"传递，不直接写心理：
  > "楚靳寒没再说话，收回视线，继续推着电动车往前走。"
  > "目光炯炯的盯着她，好似要将她看穿。"

### 八、独特习惯

- **反差结构**：频繁使用"心里想A vs 嘴上说B"的模式，制造喜剧张力和戏剧讽刺。
- **第三者旁观视角**：每章安排一个"旁观者"视角（本章是宝马女、奔驰女），通过他人眼光折射主角处境。
- **物件作为情绪锚点**：冰箱、电动车等日常物件承载情感重量，而非作为背景道具。
- **章末留悬念**：用一个未回答的问题或未解决的困境收尾，不做总结。

### ⚠️ 风格排除项（写章时必须回避）

1. **萨摩耶段落的巧合设计**：路边偶遇一只狗、位置一模一样、电动车颜色也一样——这种"过于精巧的巧合"是本文的标志性桥段，模仿会直接暴露。写章时用其他方式制造轻松插曲。
2. **"宝马vs电动车"的阶层对比母题**：本章用宝马女+电动车构成核心冲突，这是特定场景设计，不能照搬"豪车对比穷车"的框架。可以用其他形式的阶层/处境对比。
3. **"鲶鱼"比喻**：形容丑男"长得像鲶鱼"是作者的个人标签，不可复用。
4. **"小鸡逐米似的点头"**：这类非标准比喻是作者的辨识度用语，写章时用更通用的表达。
5. **"嘴角狂抽"**：过于口语化的面部描写，是作者习惯用语，避免直接使用。
6. **奔驰女的悬念设置方式**：神秘女人"攥起手指+眼底闪过恨意"是本文反派登场的标准模板，不能用相同的出场方式设计新反派。
7. **"大佬你不穷啊！"式内心独白**：用感叹号+直接称谓的内心吐槽是女主专属口吻，过于独特，仿写时调整语气频率和感叹号密度。

## 第二部分：节奏骨架

### 1. 情绪强度变化

```
[低: 日常骑车/轻松插曲] → [中低: 被旁人注视的尴尬] → [中: 外部刺激(宝马女)] → [中高: 情感试探(穷的问题)] → [低: 回家日常] → [中高: 再次情感追问] → [悬念收尾]
```

整体呈"双峰"结构：前半段外部冲突（宝马女）为第一峰，后半段内部情感冲突（穷的追问）为第二峰，中间用日常家务做缓冲。

### 2. 钩子位置和类型

- **章首钩子**：延续上章情境（骑电动车），无独立钩子，靠惯性带入。
- **中段钩子**：奔驰女的出现（信息差型悬念——读者知道有神秘势力，主角不知道）。
- **章末钩子**：情感悬念——"如果自己嫌弃他穷，他会不会提分手？"（开放式问题，未回答，驱动读者翻页）。类型：**情感困境型**。

### 3. 爽点公式

```
外部轻蔑/质疑 → 女主坦然应对（不卑不亢）→ 男主内心触动 → 男主主动表白/示弱 → 女主心软但嘴硬
```

本章爽点不在"打脸"，而在"反差甜"——别人看不起他们穷，但两人之间的信任和默契反而因此加深。

### 4. 节奏模式

```
[2段日常铺垫: 骑车/萨摩耶插曲]
→ [3段外部冲突: 宝马女的轻蔑]
→ [1段日常缓冲: 电动车没电/推车]
→ [3段情感高光: 穷的对话/男主示弱]
→ [2段日常回归: 搬家/冰箱]
→ [3段情感追问: 再次问穷/章末悬念]
→ [1段旁观者悬念: 奔驰女]
```

结构：日常-冲突-日常-甜-日常-甜更深-悬念，"三段日常+两段甜+一段悬念"的分布。

### 5. 场景切换频率

全章共4个场景：
1. 骑电动车路上（含等红灯插曲）→ 用"绿灯亮起"自然过渡
2. 电动车没电/推车 → 硬切"—"
3. 回到家/冰箱 → 用"等把东西搬回家后"一句话跳切
4. 奔驰女旁观（穿插在场景2中，非独立段落）

切换频率：约每400-500字切一次场景，属于中等频率。场景切换不频繁，以"一镜到底"的单场景内推进为主，偶尔插入旁观者视角打破封闭感。
