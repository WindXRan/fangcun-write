## 第一部分：风格指南（8维度+排除项）

### 一、叙事声音与语气

**第三人称限制视角**，交替贴近宋云绯、楚靳寒和卫岢的意识流动。叙述声音口语化，充满心理张力和紧张。

- **语气基调**：心理、紧张、略带悬疑。面对买房和袭击，情绪是"心理→紧张→悬疑"的波动。
- **叙述者人格**：在心理中展现紧张，像在记录一段充满心理张力的故事。

**例句**：
> 她一直以为，以两人的工资，买房子是件很遥远的事。（第20行）
> 楚靳寒瞳孔骤缩，扑面而来的气浪，带着烧焦的味道，同时也带来了无数破碎的画面。（第146行）
> 像是从牙缝里挤出这三个字，瞬间又被脑海里的画面冲击淹没，痛苦的闭上了眼。（第172行）

### 二、对话风格

**对话比例较低**（14.2%），对话用于展现心理和紧张。

- 宋云绯对话困惑、担忧，略带无奈
- 楚靳寒对话深沉、坚定，略带痛苦
- 卫岢对话焦急、恳求，略带无奈
- 对话前后常接心理活动或动作描写

**例句**：
> "你…为什么？"（第50行）
> "我想给你。"（第52行）
> "刘、茂、才。"（第170行）

### 三、场景描写特征

**心理型白描**，用心理细节展现心理。

- 空间描写带心理气息：办事大厅、车内、园区外
- 人物动作细节：攥紧安全带、捏着安全带、痛苦闭眼
- 环境描写映射内心：阳光暖洋洋、空气稀薄、强光刺目

**例句**：
> 今天阳光很好，照在身上暖洋洋的，但宋云绯却感觉不到多少暖意。（第34行）
> 车里再次沉静下来，连空气都得稀薄起来，呼吸有些困难。（第88行）
> 楚靳寒瞳孔骤缩，扑面而来的气浪，带着烧焦的味道，同时也带来了无数破碎的画面。（第146行）

### 四、转折与衔接手法

**心理变化式衔接**：用心理变化推动场景转换。

- **心理变化**：困惑→担忧→无奈
- **紧张变化**：平静→紧张→悬疑
- **时间流逝**：从白天到晚上，再到凌晨

**例句**：
> "我想给你。"（第52行）
> "我们去领证吧。"（第80行）
> "刘、茂、才。"（第170行）

### 五、节奏特征

**"心理-紧张-悬疑"三段结构**：

- 心理场景用**短句描写**（心理细腻）
- 紧张场景用**中句对话**（紧张氛围）
- 悬疑场景用**独立短句**（悬疑升华）

**段落较短**：70.8%的段落为单句，平均段落长度19.7字。整章节奏紧张，心理描写丰富。

### 六、词汇偏好

| 类型 | 偏好 | 例句 |
|------|------|------|
| 口语词 | 大量使用日常口语 | "炸毛""天人交战""石沉大海" |
| 动词 | 偏好具象小动作 | "攥紧安全带""捏着安全带""痛苦闭眼" |
| 形容词 | 少用，用则心理 | "沉甸甸""泛酸""黯然" |
| 连接词 | 偏好"然后""至于""但是""所以" | 口语化衔接，少用书面连词 |
| 心理词 | 使用心理表达增强氛围 | "预感""保障""枷锁" |

### 七、情绪表达方式

**心理细节+动作细节**，不直接描写情绪，而是通过：

- **心理细节**：用"沉甸甸""泛酸"等心理描写展现情感
- **动作细节**：用攥紧安全带、痛苦闭眼等动作展现紧张
- **环境映射**：用阳光、空气等环境细节映射情感

**例句**：
> 宋云绯心里沉甸甸的，心头也有些泛酸。（第58行）
> 楚靳寒不断的喘息，余光看到车里下来的人，他试图起身，但又无力地跌了下去。（第168行）
> 像是从牙缝里挤出这三个字，瞬间又被脑海里的画面冲击淹没，痛苦的闭上了眼。（第172行）

### 八、独特习惯

1. **心理与紧张交织**：心理场景中带有紧张元素
2. **环境情感化**：用阳光、空气等环境细节映射情感状态
3. **动作描写丰富**：攥紧安全带、痛苦闭眼等动作描写丰富
4. **对话比例较低**：对话比例14.2%，以心理描写为主
5. **情感升华**：从心理到紧张，再到悬疑，情感逐步升华

## 第二部分：节奏骨架

### 1. 情绪强度变化

```
[中-心理] → [高-紧张] → [中-悬疑] → [低-痛苦] → [高-危险]
```

整体情绪从心理到紧张，再到悬疑，最后以危险收尾。情绪曲线是"波动上升"。

### 2. 钩子位置和类型

- **章首钩子**（第4-6行）：过户在名下→**心理悬念**（会怎样回应？）
- **章中钩子**（第80行）：领证→**情感悬念**（会怎样回应？）
- **章中钩子**（第146行）：强光→**紧张悬念**（会怎样发展？）
- **章末钩子**（第170行）：刘茂才→**危险悬念**（会怎样发展？）

### 3. 爽点公式

**心理变化 → 紧张氛围 → 危险降临**

本章为"心理与紧张交织"，爽点在于**心理变化的细腻感**和**紧张氛围的紧张感**。从心理到紧张，再到危险，情感逐步深化。

### 4. 节奏模式

```
[心理场景-买房过户] × 4段
+ [紧张场景-领证提议] × 3段
+ [悬疑场景-卫岢等待] × 3段
+ [危险场景-刘茂才袭击] × 3段
```

具体比例：
- 心理场景：约30%
- 紧张场景：约25%
- 悬疑场景：约20%
- 危险场景：约25%

### 5. 场景切换频率

全章**4次场景切换**：
1. 开场：买房过户（第4-32行）
2. 切换：车内对话（第34-106行）
3. 切换：卫岢等待（第108-140行）
4. 切换：刘茂才袭击（第142-178行）

场景切换较多，本章是"多线交替"的结构，属于典型的心理悬疑章。
