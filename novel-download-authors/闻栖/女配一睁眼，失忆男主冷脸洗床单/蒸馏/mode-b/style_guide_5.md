# 第5章 风格指南

---

## 第一部分：风格指南（8维度+排除项）

### 叙事声音与语气

- **第三人称限制视角**，交替贴近男女主内心，但以女主视角为主（约7:3）
- **语气轻快日常**，带轻微吐槽感，像朋友在讲八卦
- **叙述者偶尔插入主观评价**，口吻随意自然：
  > "怎么说也是个正常男人，血气方刚，哪里受得了这个。"
  > "说不定又是三分钟热度，上两天就不去了。"
- **不用书面化长句**，几乎全部是口语节奏的短句

### 对话风格

- **对话极简**，每句不超过15字居多，口语化：
  > "你今天起这么早？"
  > "我昨天都跟你说了，今天我要去上班。"
- **对话后紧跟心理活动或动作**，不单独成段铺陈
- **对话标签极简**：用"说""道""开口问"，不用"低吼""冷笑"等夸张标签
- **语气词丰富**："吧""呢""了""吗"，口语感强
- **对话占比极低（8.8%）**，以叙述推动为主

### 场景描写特征

- **白描为主**，不堆砌形容词，用最简洁的笔触勾勒：
  > "床头亮着暖橘色的台灯，低头看去，能清楚地看见毛茸茸的头顶。"
- **感官描写点到即止**：触觉（"柔软的触感"）、嗅觉（"闻到了煎蛋的味道"）、视觉（"暖橘色的台灯"），每个场景只突出1-2个感官
- **环境描写嵌入动作中**，不单独成段：
  > "洗了个澡，躺在床上，吹着风扇，刷着手机。"（四个动作连写，环境信息即动作信息）
- **空间感靠动作串联**，不靠方位词堆砌

### 转折与衔接手法

- **时间跳跃直接切**：用"等第二天宋云绯醒来"一句话跳过整夜，不写过渡段
- **场景切换靠动作触发**：从家→公交，用"吃过饭→换了身衣服→出门"自然串联
- **内部转折用"不过/但是"一句话翻转**：
  > "不过气质这东西，也不是靠穿着打扮就能完全改变的。"
  > "不过她睡得正香，困意占据大脑，她也没管。"
- **情绪转折不铺垫**，直接给结果

### 节奏特征

- **段落极短**：77.4%的段落是单句，平均段落长度21.6字
- **句子中等偏短**：平均21.1字，26.7%的句子低于15字
- **无超长句**（>60字占比0%）
- **节奏模式**：日常铺垫（算账、睡觉）→ 暧昧升温（拥抱）→ 日常过渡（起床、早餐）→ 冲突（公交）→ 轻微钩子
- **暧昧段落节奏放缓**，用更多细节描写拉长时间感；日常段落快速推进

### 词汇偏好

- **动词偏好日常动作词**：洗、躺、吹、刷、算、抓、拖、端、挤
- **程度词克制**：用"更""越发""有些"，不用"非常""极其"
- **身体部位词高频**：肩窝、鼻息、手臂、胸膛、小腹、喉结、脖颈、膝盖——暧昧段落密集出现
- **经济类词汇自然嵌入**：余额、房租、水电、工资、生活费——不解释，直接用
- **口语化表达**："东买买西买买""热死""遭这份罪"

### 情绪表达方式

- **动作暗示代替心理独白**：
  > "楚靳寒抬起手，小心翼翼地捏着女人的手腕"（克制+小心翼翼=隐忍）
  > "宋云绯口中溢出不满的哼唧声"（不满=动作，不写"她感到不满"）
- **身体反应写生理不写心理**：
  > "喉结不受控制地滚动了一下，呼吸骤然变得沉重，一股热意从小腹窜起"
- **情绪词极简**：用"不乐意""诧异""无语""不自然"，不用"愤怒""震惊""羞涩"等强烈词
- **微表情替代大段心理**："眸色暗了暗""脸色越发的不自然""白着脸摇了摇头"

### 独特习惯

- **"怎么说也是……"句式**：叙述者插入感慨，拉近读者距离
- **连续短动词堆叠**："洗了个澡，躺在床上，吹着风扇，刷着手机"——四个动作一句说完
- **反差对比一笔带过**："这身装扮，和平日里邋里邋遢的形象判若两人"——不展开，点到即止
- **对话后接"不说话了"作为沉默表达**：
  > "楚靳寒不说话了，但那表情显然是不信她能坚持下去。"

### ⚠️ 风格排除项

1. **"说道"仅出现1次**，属于偶发AI痕迹，写章时用"说""道""开口""问"替代
2. **不要模仿"哪里受得了这个"这种直白生理描写**——过于标志性，容易被识别为抄源文，改用更含蓄的表达
3. **不要模仿"像八爪鱼一样缠绕着他"这个比喻**——过于独特，易暴露
4. **不要复用"暖橘色的台灯"这个意象**——已出现，再用会暴露
5. **不要在暧昧段落中出现"血气方刚"这类成语**——过于显眼
6. **"怎么说也是个正常男人"这种叙述者直接感慨**——本章特色，但高频使用会暴露来源
7. **不要重复"公交车剧烈颠簸"导致身体接触的具体设定**——属于本章独有的巧合设计

## 第二部分：节奏骨架

### 1. 情绪强度变化

```
[低:日常琐碎] → [中低:算账叹气] → [低:入睡] → [中高:暧昧肢体接触] → [高:生理反应升级] → [急降:逃离浴室] → [低:次日清晨] → [中低:早餐对话] → [中:换装惊艳] → [中低:公交日常] → [中:身体碰撞] → [中低:争执+钩子]
```

整体走势：**低→中高→急降→低→中→中低**，核心高点在暧昧段，之后回归日常。

### 2. 钩子位置和类型

- **章中钩子（~行54）**：暧昧/甜——男主被女主无意识撩拨，逃去浴室。类型：**暧昧悬念**（他怎么处理这份冲动？）
- **章末钩子（~行166）**：轻度悬念——男主不信女主能坚持上班，但女主最后一句话让他"信了"。类型：**人物态度反转+日常期待**（她能坚持多久？）

### 3. 爽点公式

- **无意识撩拨→对方失控**：女主睡着后无意识贴上来→男主生理反应→逃离。公式：[被动方无意施加亲密接触] → [主动方忍耐极限被突破] → [主动方被迫回避]
- **被看不起→反驳→对方态度松动**：公式：[一方轻视] → [另一方反驳+给出合理理由] → [轻视者态度微变]

### 4. 节奏模式

```
日常铺垫（算账+经济压力）≈ 30% → 暧昧高点（夜间接触）≈ 25% → 日常过渡（起床+早餐）≈ 15% → 微冲突（公交+争执）≈ 30%
```

比例：**3段日常 + 1段暧昧 + 1段微冲突**，甜/虐比约3:0（纯日常章，无虐）。

### 5. 场景切换频率

全章共 **4个场景**：
1. 卧室（算账+入睡）
2. 卧室（夜间暧昧）——同空间不同时间
3. 家中（早餐+换装）
4. 公交车上

切换频率：约每400-600字切一次，**低频切换**，单场景停留时间长，靠内部情绪变化制造节奏感而非场景跳转。
