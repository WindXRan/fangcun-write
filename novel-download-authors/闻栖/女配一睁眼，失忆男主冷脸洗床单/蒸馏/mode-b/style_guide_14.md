## 第一部分：风格指南（8维度+排除项）

### 一、叙事声音与语气

**第三人称限制视角，紧贴女主宋云绯的心理活动。** 叙述者语气轻快、带点吐槽感，像一个"旁观的闺蜜"在实时解说。不煽情、不铺陈，点到即止。

| 特征 | 原文例句 |
|------|---------|
| 吐槽式内心旁白 | "他这脑回路也是无敌了。" |
| 自嘲式心理活动 | "好吧，原主好像是这样的。" |
| 轻描淡写的悲剧感 | "真相大白那天，就是她的死期，这才是真的。" |
| 不做全知解释，留白给读者 | "楚靳寒脸上没什么表情，斜了眼宋云绯。"（不解释他在想什么） |

**语气光谱：** 70% 轻松吐槽 → 20% 暗涌心酸 → 10% 认真正经

---

### 二、对话风格

- **对话占比：25.5%**，不算高，但每句对话都推动情节或暴露人物关系。
- **对话标签极简**：多用"道""说""开口"，不用花哨的动词（如"怒吼""低喃"）。
- **大量省略对话标签**：靠上下文和换行自然区分说话人。

**典型对话节奏——短句交锋，你来我往：**

> "美女，加个好友啊。"
> 宋云绯看着男人眼里的戏谑，哼了一声，别过头不去看他。
> "美女，这破电动车坐着不热吗？要不要我载你？"
> 话音刚落，楚靳寒猛地一捏刹车。

**人物语言分化：**
| 角色 | 语言特征 |
|------|---------|
| 宋云绯 | 话多、情绪外露、爱用反问和感叹号："你有病啊？""你！" |
| 楚靳寒 | 话少、冷淡、陈述句为主："你终于说出来了。""那倒不是，我怕你饿死。" |
| 路人男 | 油腻、挑衅、口语化："哥们好像不服气啊？有本事追上来干一架。" |

---

### 三、场景描写特征

**极简白描，只抓一个核心感官细节，不做环境铺陈。**

| 手法 | 原文例句 |
|------|---------|
| 用动作代替环境描写 | "走出这条破烂的路后，电动车终于平稳了。"（不说路多烂，用"平稳"反衬） |
| 用身体感受代替天气描写 | "身上大汗淋漓，被太阳晒得头晕目眩。" |
| 用物件推进场景转换 | "楚靳寒从后备箱里取出一杯奶茶递给她。" |
| 社会身份暗示用道具 | "宾利""破电动车"——贫富对比不靠说教，靠物件自显 |

**原则：场景服务于情绪，不独立存在。**

---

### 四、转折与衔接手法

**核心手法：心理活动截断 + 反转式对话。**

1. **心理预期→现实打脸**
   > "要是不会刺激到他的话，借着这个机会分手也不错。" → 紧接楚靳寒冷淡回应："你终于说出来了。"

2. **情绪铺垫→一个动作打断**
   > 宋云绯内心感慨原主的不幸 → "见她沉默，楚靳寒从后备箱里取出一杯奶茶递给她。"

3. **对话中的逻辑跳跃制造意外感**
   > "所以你才没加他好友？" ——完全出乎女主和读者意料的脑回路。

4. **段落间硬切，不用过渡句**
   > 上一段：宋云绯气呼呼走了。下一段："楚靳寒没有追上去，站在电动车旁，看着她气冲冲的背影。" ——直接切换视角对象。

### 五、节奏特征

**数据佐证：**
- 平均句长：17.4字（极短）
- 短句占比（<15字）：38.6%
- 单句段占比：63.4%
- 无长段落（>200字）：0%

**节奏公式：短句交锋（快）→ 内心独白（慢）→ 短句交锋（快）**

典型节奏波形：
```
快：对话连发 3-5 句
慢：插入 1-2 句心理活动
快：一个动作/对话收束
停：空行留白
```

**"三拍子"节奏单元：** 动作→反应→内心，反复循环。
> 楚靳寒猛地一捏刹车。（动作）
> 电动车稳稳停在路边。（反应补充）
> 他冷眼朝着那辆宾利车看去。（动作延伸）

### 六、词汇偏好

**高频词汇风格：口语化 > 书面化，具体 > 抽象。**

| 类别 | 偏好词 | 避免词 |
|------|--------|--------|
| 情绪词 | 气呼呼、无语、讪讪、狐疑 | 愤怒、悲伤、惆怅 |
| 动作词 | 扯了扯、瞥着、瞪了眼、别过头 | 凝视、端详、审视 |
| 口语词 | 破、装逼、神经病、无敌了 | — |
| 转折词 | 紧接着、结果、但、不过 | 然而、尽管如此、殊不知 |
| 程度词 | 挺、确实、还真、几分 | 格外、分外、极其 |

**叠词/拟态词偏好：** "气呼呼""讪讪""缓缓"

---

### 七、情绪表达方式

**核心原则：用行为外化情绪，极少直接命名情绪。**

| 情绪 | 表达方式 | 原文例句 |
|------|---------|---------|
| 愤怒 | 动作+对话 | "气呼呼的下了电动车，朝着前面走去。" |
| 尴尬 | 身体动作 | "讪讪地把手收了回来。" |
| 无奈 | 内心独白+苦笑 | "她无奈的一笑，假的终究是假的。" |
| 心动/感动 | 克制叙述 | "宋云绯心里还挺感动的。"（"还挺"——刻意降调） |
| 复杂心情 | 直接说"复杂"但不展开 | "她心情也跟着复杂起来。" |

**情绪降调技巧：** 在高情绪节点用轻描淡写的句子收尾，制造"欲言又止"的留白感。
> "即便是宋云绯自己，此刻也……" ——省略号代替千言万语。

---

### 八、独特习惯

1. **问号轰炸**：用连续追问制造紧迫感
   > "你什么时候买的？""所以你这么半天才追上来，是买奶茶去了？"

2. **"??""……"独占一行**：用标点符号做独立段落，制造节奏停顿和情绪张力。

3. **反问句密度极高**：几乎每段对话都有反问，推动冲突升级。
   > "我在你心里就是这样的人吗？！""什么意思？舍不得我？"

4. **"好吧"作为心理转折标志词**：出现时意味着角色认输/接受现实。
   > "好吧，原主好像是这样的。"

5. **用"破"字做万能贬义修饰**：破电动车、破地方。

6. **结尾用日常小事化解大冲突**：分手→奶茶→和好。大起大落收在一个微小的温暖动作上。

### ⚠️ 风格排除项

| 排除项 | 说明 |
|--------|------|
| ❌ 华丽辞藻 | 不用"旖旎""缱绻""氤氲"等网文高频美词 |
| ❌ 长段心理剖析 | 单段不超过45字，不做意识流 |
| ❌ 环境渲染超过2句 | 场景描写一笔带过，不铺陈 |
| ❌ 全知视角泄露 | 不写楚靳寒的内心想法，只写外在表现 |
| ❌ 直接命名深层情绪 | 不写"她感到一阵撕心裂肺的痛"，用行为代替 |
| ❌ 成语堆砌 | 偶尔用"头晕目眩"，但绝不连续使用 |
| ❌ 感叹号泛滥 | 仅在宋云绯爆发时使用，其他角色几乎不用 |
| ❌ "仿佛""似乎""好像"过度使用 | 只在必要时用1次，不做模糊化处理 |
| ❌ 说教式旁白 | 不做"人生就是……"式的哲理总结 |

### 3. 爽点公式

**本章爽点公式：「委屈积累 → 爆发宣泄 → 被默默守护」**

| 阶段 | 内容 | 爽感类型 |
|------|------|---------|
| 积累 | 宾利男骚扰、被误解出轨 | 憋屈感/代入感 |
| 爆发 | "分手吧！""他比你有钱比你帅！" | 宣泄爽/嘴替爽 |
| 反转 | 楚靳寒冷静说出"你终于说出来了" | 意外爽/悬念爽 |

| 回收 | 他追上来+奶茶+"我应当对你负责到底" | 甜爽/安全感爽 |

**单章爽点密度：** 约每400字一个爽点/钩子，节奏紧凑。

### 4. 节奏模式

**整体节奏：快—快—慢—快—慢**

```
[快] 宾利事件（外部冲突，对话密集，节奏紧凑）
[快] 分手争吵（情绪爆发，短句交锋）
[慢] 暴走独白（内心反思，节奏放缓，抒情留白）
[快] 楚靳寒追来（对话交锋，节奏回升）
[慢] 奶茶和好（温情收尾，节奏归于平静）
```

**段落节奏单位：**
- 冲突段：3-5行/节奏单元
- 独白段：1-2行/节奏单元
- 对话段：1行/节奏单元（极快）
