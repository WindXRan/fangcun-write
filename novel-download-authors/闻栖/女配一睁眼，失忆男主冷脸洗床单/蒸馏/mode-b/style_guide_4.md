# 第4章 风格指纹分析

## 第一部分：风格指南（8维度+排除项）

### 叙事声音与语气

第三人称有限视角，主要锁定宋云绯的心理，偶尔短暂切入楚靳寒视角。语气轻松、生活化，带轻微吐槽感，像在跟读者唠嗑。

> "楚靳寒直勾勾盯着她，也没有上去帮忙的意思。"（开篇直接、不铺垫）

> "宋云绯在心中暗自叹气。看来这半年，原主是真把他心里那点愧疚都快作没了。"（内心吐槽式叙事）

> "这家伙怕是有强迫症吧？"（用反问句吐槽，语气随意）

### 对话风格

台词极简，男主台词尤其短冷，女主台词稍长但也不啰嗦。对话后紧接心理活动或动作描写，形成"台词+内心反应"的节奏。

> 楚靳寒："等会我要出去送外卖，做饭来不及了，还是点外卖吧。"（陈述句，不解释）

> 宋云绯："你看着我干嘛？"（反问，带小脾气）

> 楚靳寒："很好。" / "不胖。"（两个字敷衍，冷淡标配）

> 宋云绯叹了口气，继续吃饭。（对话后的沉默处理）

### 场景描写特征

环境描写克制，只在推动情节或暗示人物关系时才出现。偏好感官细节（热、汗、风扇声），用具体物件替代抽象描述。

> "外面好热啊。"她打开风扇，坐在椅子上对着风扇吹了起来。（动作带场景）

> "屋里除了风扇转动的声音之外，安静的有些诡异了，还有一丝尴尬。"（用声音反衬沉默）

> "这厨房很小，而且非常之简陋，就搭了两个台子而已，也没有安装天然气和油烟机。"（具体物件罗列，不抒情）

> "几缕碎发黏在颈侧，T恤紧绷，露出白皙的后腰"（身体细节，暧昧暗示）

### 转折与衔接手法

用短句制造停顿感，用"忽然""不知怎的"做意识流转折。段落间靠动作或心理推进，不靠连接词。

> "只是看得正入神，忽然一阵带着热意的风吹了过来。"（感官打断注意力）

> "不知怎的，昨晚的画面和手感，忽然跃入脑海。"（意识流闪回）

> "楚靳寒古怪地看着她。"（一个短句就是一个转折点）

### 节奏特征

短段落主导（83.8%为单句段落），叙述-对话-内心独白交替，形成"三拍子"节奏。段落间留白多，阅读体感轻快。

> 平均句长23.3字，段落长23.0字，几乎等长——说明一句一段是常态。

> 对话比例仅8.9%，大量篇幅用于心理活动和动作描写。

### 词汇偏好

口语化词汇为主，避免书面腔。偏好"自顾自""敷衍""古怪""咂舌"等带态度的词。身体词汇高频出现（手臂、后颈、小腹、腿）。

> "楚靳寒自顾自的看起了手机。"

> "看他敷衍的模样，宋云绯想起来什么。"

> "宋云绯咂舌，看看人家多有觉悟，再看看原来的宋云绯。真是狗看了都摇头。"

### 情绪表达方式

不直接写"她很紧张/他很尴尬"，而是用动作、环境、内心吐槽间接传达。情绪常藏在反问句或感叹句里。

> "她也抬头看了过去，'你看着我干嘛？'"（用反问掩饰尴尬）

> "楚靳寒又是一愣，诧异地看了她一眼"（"一愣"比"惊讶"更口语）

> "这么冷淡，难怪原主会对他控制欲越来越强。"（用因果分析代替情绪描写）

### 独特习惯

1. **"自顾自"高频使用**：楚靳寒的标志性动作词，暗示他的疏离和独立。
2. **心理活动用"看来""实际上"引导**：宋云绯的内心戏常以分析口吻展开。
3. **段末留白**：很多段落结尾是句号收束，不做延伸，制造"说完就走"的干脆感。
4. **身体凝视**：男主视角常落在女主身体部位（后颈、腰、腿），用克制的笔触写暧昧。

### ⚠️ 风格排除项

1. **"自顾自"不能每章都用**：这是本文的标志性词，但高频使用会暴露模仿痕迹，每章限用1-2次。
2. **"不知怎的"+"忽然跃入脑海"**：意识流闪回句式过于独特，写章时换用其他闪回方式。
3. **"狗看了都摇头"**：这种网络梗式吐槽过于标志性，不宜照搬。
4. **"切片研究""送去监狱"**：女主内心对男主的极端想象是本文特有设定，不能移植到其他角色。
5. **身体凝视的固定节奏**：男主看女主→回忆昨晚→移开视线，这个三拍模式太规律，需要打乱。

## 第二部分：节奏骨架

### 1. 情绪强度变化

```
[低-平静] → [中-微妙尴尬] → [中-暧昧暗涌] → [低-平淡收束]
```

开篇日常引入 → 两人独处的微妙张力 → 做饭/吃饭的渐进亲近 → 洗碗的平淡结局

### 2. 钩子位置和类型

**章末钩子：甜/日常向**——男主主动洗碗，女主感叹"有觉悟的男人"，用轻松吐槽收尾，不设悬念，靠氛围感留人。

### 3. 爽点公式

```
女主改变行为 → 男主感到意外/困惑 → 男主被动接受好意 → 女主内心吐槽男主冷淡
```

本章无强冲突爽点，核心是"反差感"：女主突然变正常，男主不适应但默默配合。

### 4. 节奏模式

```
[2段日常引入] + [3段微妙尴尬] + [4段做饭场景] + [5段吃饭对话] + [2段收束吐槽]
```

日常→暧昧→互动→收尾，无明显冲突段，属于"甜日常"节奏。

### 5. 场景切换频率

**低频切换**：全章只有一个场景（出租屋），空间从客厅→厨房→客厅，共3次微切换，不跨场景。
