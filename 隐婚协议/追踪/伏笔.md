# 伏笔管理

## 已埋设的伏笔

### 第4章
1. 陆宴舟对苏念说"去结个婚"（助理视角）→ 表现他对这场婚姻的重视程度
2. 陆宴舟把第二天上午的会全部推到下午 → 他专门空出领证的时间

### 第5章
3. 陆宴舟知道苏念下午要回公司（"你简历上写了"）→ 第6章证明他连她的生活细节都知道
4. 大哥陆宴霖发消息"终于把人骗到手了"→ 暗示陆宴舟早有预谋
5. 陆宴舟把大哥消息删除 → 他在隐瞒什么

### 第6章
6. 苏念常用的牙膏和洗发水牌子出现在公寓洗手间 → 陆宴舟怎么知道的？
7. 陆宴舟准备了苏念尺码的睡衣和毛巾 → 暗示他关注她很久了
8. 玄关同款情侣拖鞋 → 陆宴舟在营造"家"的氛围
9. 床头便利贴"都是你的码"→ 陆宴舟细心到可怕，苏念开始困惑

### 第22章
10. 书房整洁到不像有人用（苏念认为"样板间"）→ 陆宴舟在刻意隐藏生活痕迹
11. 空白相框放在书房桌上 → 苏念怀疑是故意留下的线索
12. 相框背板后贴着模糊的照片复印件 → 苏念和一个人的大学咖啡店合照
13. 陆宴舟记得苏念说过"蛋糕挺好吃"还买了整块芝士蛋糕 → 暗示他对苏念的每句话都记在心上
14. 陆宴舟特意告诉苏念书房备用钥匙在玄关 → 他知道苏念翻过，但不说破

### 第23章
15. 照片里苏念穿白色卫衣扎头发（读研时期）→ 时间线在三年前
16. 照片里的男人侧脸像陆宴舟 → 苏念不敢确认
17. "遇见"咖啡店（学校后门，已倒闭）→ 线索断了
18. 苏念完全不记得拍过这张照片 → 暗示照片是偷拍的，或者她当时注意力完全不在旁边的人身上

### 第24章
19. 陆宴舟是周小棠表哥公司的大客户（采购额几千万）→ 滨江圈子里排名前五
20. 陆宴舟已婚的消息在圈子内传开 → 社交圈的认知开始扩散
21. 苏念意识到陆宴舟"出现在她身边每一个环节都太巧了"→ 怀疑累积
