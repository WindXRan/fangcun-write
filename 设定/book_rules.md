# 《穿成黑粉后，落魄顶流住隔壁》写作规则

## 写作规则（基于闻栖风格）

### 核心原则
1. **困境即钩子**：开局必须“天崩”，女主穿书成黑粉+欠债+被封杀，男主顶流伪装+落魄+豪门私生子。
2. **口嫌体正直**：角色性格通过动作展示，嘴硬心软、言行不一。
3. **延迟满足**：情感高潮前用3-5个步骤层层递进，直接写“在一起”爽感减半。
4. **具体细节＞抽象情感**：用物件承载情感（50块奶茶钱、一碗面条）。
5. **虐不过三章**：甜宠文核心是“甜”，虐只是调剂，最多2-3章必须“回甜”。

### 开篇铁律
- **第1章前500字**：建立困境（女主穿书+欠债+被封杀）。
- **第1章结尾**：男女主相遇（听到隔壁男人声音）。
- **第3章结束**：关系确立（被迫同居/假装情侣）。

## 禁用词列表

### 绝对禁用词（AI痕迹）
- **连接词**：`此外`、`因而`、`综上所述`、`值得注意的是`、`与此同时`
- **副词**：`必然`、`显然`、`无疑`、`必定`
- **高频副词**：`微微`、`轻轻`、`缓缓`
- **古早网文词**：`俊美无俦`、`美得不可方物`、`倾国倾城`
- **武侠遗留词**：`说时迟那时快`、`电光火石之间`

### 替代方案
- `然而` → `但是`、`不过`、`结果`
- `因此` → `所以`、`反正`
- `微微一笑` → `一笑`
- `缓缓开口` → `开口`
- `必然` → `肯定`、`必须`（口语化）
- `显然` → `明显`、`明摆着`

## 句式规范

### 短句动作链
- **规则**：用3-8字的短句串联连续动作，每个动作独立成句，句号断开。
- **示例**：`宋云绯三下五除吃完面条，去厨房把碗洗了。`
- **禁止**：连续5句以上用相同句式开头。

### 长句情感描写
- **规则**：情感描写用长句（15-25字），但超过25字必须拆分。
- **示例**：`他身上带着阳光混合着茉莉花的味道。`
- **禁止**：心理描写超过3句不打断。

### 长短句交替
- **规则**：动作链用短句（3-8字）推进，情感描写用长句（15-25字）渲染。
- **节奏**：连续短句不超过5句，连续长句不超过3句。
- **示例**：
  ```
  她倏地的睁开眼。（短句）
  入目是一面斑驳老旧的墙壁，有的地方墙皮脱落开裂，上面还有不知谁用水彩笔画的五颜六色的涂鸦和文字。（长句）
  宋云绯有些懵。（短句）
  ```

## 对话规范

### 反问链推进
- **规则**：用连续反问推进对话深度，每轮增加一层情感。
- **限制**：反问链不超过3轮，最后一轮必须得到明确答案。
- **示例**：
  ```
  “你刚才，为什么亲我。”
  “我是个男人。”
  “所以，只因为我是个女人，刚才换做任何一个人……你都会那样做？”
  “因为你是我女朋友。”
  ```

### 对话潜台词
- **规则**：对话要有潜台词层，不是直白表达。
- **示例**：嘴上说“调查她”，实际关注的是“她和柏庾的关系”。
- **禁止**：角色直接说“我爱你”，用极端选择/歪理正说/价值观宣言/行动代替。

### 对话节奏
- **规则**：关键对话用短句（4-12字）结束，冗长对话解释是失败的。
- **禁止**：对话原地踏步不推进，每一句对话都要增加一层张力。

## 节奏规范

### 章节结构
- **开篇方式**：对话/声音开场（首选，46%）或感官/环境开场或内心独白开场（穿书/重生）。
- **铺垫节奏**：三幕式——开头建立情境（1-2段），中段发展推进（主要篇幅），结尾制造转折或悬念（最后3-5行）。
- **高潮位置**：情感高潮用3-5个步骤层层递进（延迟满足），71%压在章尾1/5处。

### 章节类型
1. **日常章**：用小事展示关系（做饭、洗碗、遛狗），在日常中加入一个“发现”。
2. **冲突章**：用对话推进冲突，每一句对话都要增加一层张力。
3. **高潮章**：延迟满足——情感高潮前用3-5个步骤层层递进。
4. **过渡章**：用日常琐事消解尴尬，用自嘲和幽默缓冲情绪。

### 钩子类型
1. **对话钩**：章节第一句是对话（引号开头），46%概率。
2. **叙述钩**：章节第一句是叙述，25%概率。
3. **动作钩**：章节第一句是动作描写，13%概率。
4. **悬念钩**：章节第一句制造疑问，8%概率。
5. **环境钩**：章节第一句是环境描写，4%概率。

### 信息差节奏
- **第1-100章**：双方互相隐瞒，读者全知视角。
- **第100章（1/2处）**：男主顶流身份暴露。
- **第150章（3/4处）**：全部信息差揭开（女主穿越者身份+男主豪门背景）。

## 去AI策略

### 执行步骤
1. **删连接词**：扫描「然而」「此外」「因此」「综上所述」「值得注意的是」，删除或替换为「但是」「不过」「结果」「所以」。
2. **拆长句**：扫描超过25字的句子，在逗号处或「并且」「而且」「但是」前拆成2-3个短句。
3. **得地的**：将「得」「地」统一改为「的」。
4. **加微重复**：在情绪激动处加入词语重复（"完了完了"）或字词重复（"穿、穿书了？"）。
5. **加模糊词**：在心理活动前加「好像」「似乎」「大概」「估计」，删除「必然」「显然」「无疑」「必定」。
6. **降级副词**：将「微微一笑」改为「一笑」，「缓缓开口」改为「开口」。
7. **标点后处理**：情绪激动处用「？！」，犹豫处用省略号，反问处用问号。

### 改动示例
**AI原文**：
> 然而，宋云绯并没有因此而感到害怕，反而在心中涌起了一股强烈的求生欲望。她迅速地环顾四周，试图寻找可以逃脱的路线，但是却发现所有的出口都被堵死了。

**闻栖风格改写**：
> 宋云绯没害怕，反而更想活了。
>
> 她飞快地扫了一圈，想找个出口跑。
>
> 结果，全堵死了。

**改动说明**：删除"然而""因此""反而"→直接断句；拆分长句→3个短句；"迅速地环顾四周"→"飞快地扫了一圈"（口语化）；"试图寻找"→"想找"（精简）；"但是却发现"→"结果"（口语化连接）

## 表达DNA

### 短句动作链
- **规则**：用3-8字的短句串联连续动作，每个动作独立成句，句号断开。
- **场景**：角色执行日常动作、需要快速推进时间、章节开头需要快速进入状态。

### 口语化内心独白
- **规则**：内心独白使用网络热梗和口语化表达，制造“接地气”的亲切感。
- **场景**：角色遇到荒诞/离谱情况、需要制造喜剧效果、展示角色真实性格。
- **示例**：`林安此刻的表情和地铁老人看手机一模一样。`

### 感官细节替代直述
- **规则**：用嗅觉、触觉等感官细节替代“她很喜欢”“他很温暖”的直述描写。
- **场景**：亲密场景的氛围营造、角色产生好感/心动、需要增强画面感。
- **示例**：`他身上带着阳光混合着茉莉花的味道。`

### 具体细节传递情感
- **规则**：用具体物件/金额/动作传递情感，比“我爱你”更有力。
- **场景**：男主默默付出的展示、情感高潮的具象化表达、需要制造记忆点。
- **示例**：`楚靳寒转了四十万给宋云绯。`

### 打断节奏制造转折
- **规则**：在情感/剧情即将到达高潮时，用一个“意外事件”打断，制造转折。
- **场景**：亲密场景即将失控、告白被打断、关键信息即将揭露。
- **示例**：`眼看要走火，紧张之下，宋云绯猛地抬手一推。下一秒，听到闷哼声，以及重物落地的声音。`

### 留白比明说更有力
- **规则**：最重要的情感用“不说”代替“说”——让读者自己填充。
- **场景**：情感爆发后的克制、角色有难言之隐、结局的余韵处理。
- **示例**：`到嘴的话，到底还是咽了下去。`

### 环境情绪映射
- **规则**：环境描写与角色情绪同步，用具体细节映射情绪。
- **场景**：情感转折后的“静默”段落、章节结尾收束情绪。
- **示例**：`窗外烈日炎炎，屋内风扇轻转动，伴随着阵阵蝉鸣声，并不吵闹，反而有种诡异的静谧。`

### 微表情捕捉
- **规则**：用微小的身体反应代替语言表达复杂情绪。
- **场景**：角色强忍情绪、需要“无声胜有声”的时刻。
- **示例**：`他的睫毛剧烈地颤抖着，胸口起伏得越来越剧烈。`

### 假理由掩盖真情感
- **规则**：让角色用一个“假理由”掩盖真实情感（孤独），读者知道真相但角色不说。
- **场景**：角色不愿暴露脆弱、需要制造“读者比角色更心痛”的效果。
- **示例**：`“哭什么？很难受么？”她低着头，瓮声瓮气道，“没有。”又找了个蹩脚的理由，“药太苦了。”`

## 写作检查清单

### 写之前问自己
- 这章的核心事件是什么？（不能是「关系升温」这种空话，必须是具体事件）
- 开篇用什么钩子？（对话开场/感官开场/内心独白开场，三选一）
- 结尾断在哪里？（悬念断章/情感落点断章/动作中断断章，三选一）
- 信息差是什么？（谁知道了什么？谁不知道？）
- 这章属于什么类型？（日常章/冲突章/高潮章/过渡章）

### 写完后检查
- [ ] 章名在4-7字范围内，口语化，疑问句占比≥20%
- [ ] 开篇第一句是对话或声音或感官描写（不是叙述）
- [ ] 结尾有钩子（悬念/情感/动作，不能用句号收尾）
- [ ] 核心事件2个（双事件节拍），不是流水账
- [ ] 没有使用反模式词汇（此外/因而/综上所述/必然/显然/无疑/必定/俊美无俦/说时迟那时快）
- [ ] 没有超过25字的长句未拆分
- [ ] 心理描写不超过3句，用身体反应替代
- [ ] 对话有潜台词层，不是直白表达
- [ ] 没有角色直接说“我爱你”
- [ ] 情绪基调与章纲一致
- [ ] 动作描写具体到可拍摄程度
- [ ] 经济数字具体到个位（50块，不是“几十块”）
- [ ] 去AI检查：无正式连接词、长句已拆、有微重复、有模糊词、得地的已统一

## 情绪曲线控制

### 三段式情绪过山车
- **规则**：先用负面情绪建立张力，再用正面情绪释放，最后用感悟升华。
- **示例**：`宋云绯揉了揉胸口，‘气死我了，世上怎么会有这样的人！’...楚靳寒握住她的手...「少看网络上的毒鸡汤，你只是做了一件，你自己认为对的事。」`

### 甜虐比
- **比例**：1.8:1（甜宠为主，虐心调剂）
- **虐心长度**：最多2-3章，之后必须“回甜”
- **日常缓冲**：每3-5章安排一个日常章

## 空间对比

### 规则
- 用不同空间（出租屋/别墅、城中村/豪门宅邸）强化情感落差。
- 空间对比能直观展示阶级差异、情感距离。

### 应用
- **城中村**：象征男女主的“低谷起点”，也是他们感情的“安全屋”。
- **豪门宅邸**：象征男主的“过去”和“枷锁”，也是他必须面对的“真相”。
- **游戏工作室**：象征男主的“新起点”，也是他翻红的“战场”。

---

## 信息差设计

### 规则
- 让读者知道角色A的秘密，但角色B不知道，制造“我替主角着急”的紧张感。
- 信息差是网文最核心的钩子之一。

### 应用
- **女主隐瞒**：穿越者身份、原主黑历史。
- **男主隐瞒**：顶流身份、豪门背景。
- **读者视角**：全知视角，知道双方都在隐瞒。

### 揭穿节奏
- **第100章（1/2处）**：男主顶流身份暴露。
- **第150章（3/4处）**：全部信息差揭开（女主穿越者身份+男主豪门背景）。

- 绝对不用：`俊美无俦`、`美得不可方物`、`倾国倾城`
- 绝对不用：`说时迟那时快`、`电光火石之间`

### 句式反模式
- 绝对不做：连续3句以上用相同句式开头
- 绝对不做：写超过25字的长句不拆分
- 绝对不做：用「首先/其次/此外/因此」排列观点
- 绝对不做：心理描写超过3句不打断

### 结构反模式
- 绝对不做：一次性交代所有背景设定
- 绝对不做：情感高潮“一步到位”不延迟
- 绝对不做：在高潮前插入回忆

### 情节反模式
- 绝对不做：靠“你不听我解释”制造冲突
- 绝对不做：反派纯粹邪恶没有人性
- 绝对不做：所有线完美收束的大团圆结局
- 绝对不做：拖长误会循环，误会最多拖1-2章

### 角色反模式
- 绝对不做：女主善良无瑕/为爱无条件牺牲
- 绝对不做：第一次亲密戏写得完美浪漫
- 绝对不做：角色直接说“我爱你”
- 绝对不做：写标准霸总人设，男主必须有「落魄/缺陷/困境」

### 对话反模式
- 绝对不做：角色犯错后长篇解释/道歉
- 绝对不做：对话原地踏步不推进
- 绝对不做：所有角色用同一种说话方式
- 绝对不做：冗长对话解释，关键对话用短句（4-12字）结束
