# 新书设定

## 书名候选
1. 《攀权附势》（推荐）
2. 《权门娇客》
3. 《名门弃女》

## 类型定位
- 标签组合：现代言情|豪门世家|商战|逆袭

## 核心卖点（保留的成功因子）
| 源文成功因子 | 新书保留方式 |
|-------------|-------------|
| 落魄女主搭上强势男主各取所需 | 女主父亲破产入狱，为救家人接近男主 |
| 女主是男主侄子的未婚妻 | 女主是男主侄子女朋友，被抛弃后转投男主 |
| 女主聪明务实能屈能伸 | 保留，女主是金融/法律背景，擅长谈判 |
| 男主风流残暴有魅力 | 改为商界大佬，手段狠辣，私生活混乱 |
| 叔侄争斗权力博弈 | 改为家族企业继承权争夺 |
| 虐→爽节奏 | 保留，女主从被羞辱到站稳脚跟 |
| 男主女儿（萧珠）作为纽带 | 改为男主侄女/妹妹，性格刁蛮但喜欢女主 |

## 人物设定（与源文差异化）

### 女主
- **身份**：顾念之，前金融集团千金，父亲因商业诈骗入狱
- **背景**：曾留学英国学金融，回国后家族崩塌，母亲重病，弟弟年幼
- **性格**：冷静、务实、善于察言观色，能屈能伸，有商业头脑
- **目标**：救父亲出狱、照顾家人、在商界站稳脚跟
- **差异点**：从医学背景改为金融/法律背景；从民国闺秀改为现代职场女性

### 男主
- **身份**：陆景琛，陆氏集团实际控制人，商界枭雄
- **背景**：白手起家，手段狠辣，控制着半个城市的商业命脉
- **性格**：表面风流倜傥，实则心狠手辣，野心勃勃，护短
- **标签**：商界疯狗、护短、占有欲强
- **差异点**：从军阀改为商界大佬；无子女（改为有刁蛮妹妹）

### 关键配角
- **陆子轩**：男主侄子，女主前男友，软弱但有野心（对应萧珩，但无心理疾病）
- **陆子萱**：男主妹妹，17岁，刁蛮任性但讲义气（对应萧珠）
- **顾父**：女主父亲，因商业诈骗入狱（对应徐白父亲卷款私奔）
- **林婉清**：陆子轩新女友，名门千金（对应罗绮）
- **陈助理**：男主心腹助理（对应石铖/石锋）

## 世界观
- **时代背景**：现代都市，经济发达的沿海城市
- **地理**：海城（虚构，类似上海/深圳）
- **势力**：陆氏集团（商业帝国）、林氏集团（竞争对手）、政府关系
- **核心设定**：商界权贵家族的继承权争夺，各方势力明争暗斗
- **与源文差异化**：从民国军阀割据改为现代商战，更注重商业谈判、资本运作、媒体公关等现代元素

## 简介（嵌入设定文件末尾，三种风格各150-250字）

### 标准版（叙事型，背景+冲突+钩子）
顾念之家道中落，父亲因商业诈骗锒铛入狱，母亲重病卧床，弟弟年幼无依。前男友陆子轩奉家族之命另结新欢，却不愿放过她："等我掌权，会给你名分。"顾念之为救父亲、保家人，不得不接近陆子轩的小叔叔——海城商界有名的"疯狗"陆景琛。这男人风流成性、手段狠辣、野心勃勃。顾念之与他各取所需，竟是在风雨飘摇中站稳了脚跟。"你回头是岸。"陆子轩说。"他说他是岸。"男人俯身耳语，"把这岸炸了吧。"

### 悬念版（反问/留白）
父亲入狱，母亲病危，弟弟年幼。前男友说："等我掌权，会给你名分。"可他的新欢已经登堂入室。顾念之没有等。她转身走向了那个男人——陆子轩的小叔叔，海城商界最危险的存在。他们各取所需。她要救父，他要夺权。只是，这场交易里，谁先动了真心？"你回头是岸。"陆子轩说。"他说他是岸。"男人笑了，"把这岸炸了吧。"

### 甜宠版（强调互动甜感）
顾念之以为，接近陆景琛只是权宜之计。可这男人护短得过分——"我的人，外人没资格说。"他给她夹菜，替她挡酒，甚至为了她得罪整个林家。"四爷，您对每个女人都这样？"她问。"不。"他俯身，"只对你。因为你是我的人。"顾念之心想：这场交易，似乎亏大了。可他说："不亏。我赔给你一辈子。"
