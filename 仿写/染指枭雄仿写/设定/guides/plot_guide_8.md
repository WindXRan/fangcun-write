# 写章指南：第8章

## 写章目标
甜蓄力章：顾念之用故事和耐心打动陆子萱，两人关系破冰——展现女主教育能力，赢得陆景琛初步认可，同时引入林婉清作为情感竞争者，为后续三角关系埋线。

## 角色声音
本章出场角色的说话风格：
| 角色 | 说话特征 | 示例句式 |
|------|---------|---------|
| 顾念之 | 节奏不快不慢，微拖尾音却不做作，冷静克制 | "写得很不错。""复习一下今天认识的字。""这是分内事，不要加工钱。" |
| 陆景琛 | 慵懒散漫，语调轻佻带讽刺，护短 | "怎么我三天没回家，祖坟还冒青烟了？""瞎子居然睁眼了。""我得给顾小姐双倍工钱。" |
| 陆子萱 | 牙尖嘴利、叛逆、学大人说话、爱翻白眼 | "等你老得不能动了，我直接把你埋祖坟里。""伯母，你留着带回去给自己孩子玩吧。" |
| 陈助理 | 恭敬简练、不废话 | "顾小姐很老实，与小姐相处也很好。" |
| 林婉清 | 娇媚主动、话术圆滑、暗藏心机 | "四爷，您落了东西。""晚上可有空？我买了两张票。" |

## 鼓点节奏
按字数分段，每段写明场景内容和情绪走向（目标字数：2200字）：
- **章首0-400字**：陆景琛多日未归，进门问陈助理顾念之近况，得知顾念之与陆子萱相处融洽。情绪：冷淡→微微好奇。主感官：听觉。
- **发展400-900字**：陆景琛去后花园，旁观顾念之教陆子萱认字/讲故事，意外发现妹妹竟学会了不少，内心触动。情绪：意外→暗赞。主感官：视觉/听觉。
- **转折900-1400字**：陆景琛现身与两人互动，调侃陆子萱，顾念之打圆场解释教学方法，陆景琛认可并请她留下吃饭。情绪：轻佻→认可。主感官：听觉/嗅觉。
- **高潮1400-1900字**：林婉清突然来访，送礼讨好陆子萱，陆子萱态度刁蛮讽刺，陆景琛不冷不热应对，林婉清邀约被拒。情绪：紧张→微妙敌意。主感官：嗅觉/视觉。
- **钩子最后300字**：林婉清离开时回头看顾念之一眼，眼神意味深长——暗示她已将顾念之视为威胁。悬念：这个女人会不会成为麻烦？情绪：暗流涌动。主感官：视觉。

## 暗线埋设
- **陆景琛的认可**：这章是陆景琛第一次正面认可顾念之的能力——从"监视对象"转为"有用的人"，为后续重用铺垫
- **林婉清的竞争意识**：林婉清离开时那一眼，暗示她已注意到顾念之的存在，为后续骚扰埋线
- **陆子萱的接纳**：陆子萱开始接受顾念之的教育，关系破冰，为后续依赖她做铺垫
- **陆景琛的私生活**：通过林婉清来访，侧面展现陆景琛风流的表面形象，与后续他对顾念之的"特殊"形成对比
