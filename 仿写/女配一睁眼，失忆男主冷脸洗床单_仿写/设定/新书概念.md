# 新书概念

## 基本信息
- **书名**：穿成恶毒女配后，失忆大佬非要当我老公
- **类型**：现代言情 · 穿书 · 甜宠 · 豪门
- **标签**：穿书|甜宠|先婚后爱|双洁|豪门
- **体量**：188章，约47万字（对标源文）
- **一句话梗概**：穿书成恶毒女配的美食博主，为了活命假装失忆大佬的老婆，结果大佬越陷越深，她却在策划跑路。

4. **追妻火葬场**：男主恢复记忆后发现，自己才是被耍的那个

## 故事弧线（粗线）

### 第一幕：穿书求生（1-30章）
- 苏棠穿书成恶毒女配，发现自己正在相亲局上
- 原主设计陷害陆衍舟，导致他失忆
- 苏棠为了活命，假冒陆衍舟的妻子
- 两人在城中村出租屋开始"夫妻"生活
- 陆衍舟白天在夜市摆摊卖烧烤，晚上帮苏棠打下手做美食直播

### 第二幕：假戏真做（31-80章）
- 同居日常：从互相防备到渐生好感
- 苏棠的美食直播意外走红，开始有收入
- 陆衍舟的烧烤摊也火了，被人拍视频传上网
- 两人在生活中互相照顾，感情升温
- 原书女主沈念薇开始寻找失踪的陆衍舟

### 第三幕：危机浮现（81-130章）
- 陆衍舟的记忆开始出现碎片
- 沈念薇找到了陆衍舟的下落
- 苏棠发现原书剧情在加速推进
- 陆衍舟恢复部分记忆，开始怀疑苏棠的身份
- 苏棠准备跑路，但发现自己已经动心

### 第四幕：真相大白（131-170章）
- 陆衍舟完全恢复记忆
- 得知真相后愤怒又痛苦
- 沈念薇的复仇计划浮出水面
- 苏棠被卷入陆家的家族争斗
- 陆衍舟在愤怒和爱之间挣扎

### 第五幕：追妻火葬场（171-188章）
- 苏棠离开，陆衍舟疯狂寻找
- 陆衍舟放下身段追妻
- 沈念薇的阴谋被揭穿
- 大结局：HE，陆衍舟和苏棠在一起

## 世界观设定

### 主要场景
1. **城中村出租屋**：两人同居的地方，破旧但有烟火气
2. **夜市**：陆衍舟摆摊的地方，苏棠也在这里做美食直播
3. **陆氏集团**：陆衍舟的公司，恢复记忆后的主战场
4. **陆家老宅**：陆老太太住的地方，家族争斗的核心

### 关键道具
1. **苏棠的美食直播间**：她赚钱独立的工具
2. **陆衍舟的烧烤摊**：他失忆后的伪装身份
3. **原书小说**：苏棠知道剧情的金手指
4. **陆家传家玉佩**：关键道具，贯穿全书

### 金手指
- 苏棠知道原书剧情，可以提前规避风险
- 但原书剧情会因为她的介入而发生偏移
- 需要不断调整策略
