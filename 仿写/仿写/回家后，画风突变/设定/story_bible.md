# 世界观设定

## 时代背景

**时间**：当代中国，202X年

**城市**：虚构的南方二线城市"锦城"——经济发达，新旧城区并存。城东是CBD和高端住宅区（豪门势力范围），城西是老城区夜市街（烟火气最浓的地方）。两个世界直线距离不到十公里，但生活方式天差地别。

**季节起点**：夏末秋初（生日宴认亲发生在八月底），贯穿秋冬至次年春天。季节变化对应主角从"外来者"到"融入者"的身份转变。

**社会背景**：
- 锦城新城区近年高速发展，商业地产、互联网新贵崛起，传统豪门面临转型压力
- 老城区夜市经济繁荣，是本地特色名片，但正面临城市改造拆迁的威胁
- 社交媒体（短视频、直播、小红书）深度渗透日常生活，太太圈攀比从线下扩展到线上
- "接地气"成为网络热词，与"高大上"形成天然对立，为主角的"地气哲学"提供社会语境

## 主要势力

### 沈家（豪门）

**地位**：锦城老牌豪门，三代经商，以商业地产起家，近年涉足文旅、餐饮连锁。

**家族结构**：
- **沈老爷子**（已故）：家族奠基人，白手起家，留下"做人要实诚"的家训。他的创业故事是沈鹿鸣后来"地气商业学"的精神源头
- **沈天泽**（家主/沈鹿鸣生父）：商场铁腕，表面威严，实则被女儿的"地气智慧"颠覆商业观。与妻子陈清竹因当年送走女儿一事心存芥蒂
- **陈清竹**（生母/豪门主母）：出身书香门第，性格高冷克制，内心对女儿有强烈愧疚。年轻时曾与丈夫在夜市摆摊，嫁入豪门后被迫隐藏这段经历
- **沈鹿宸**（亲哥哥）：28岁，家族企业副总裁，商业精英，理性冷静，被妹妹的"地气商业学"颠覆三观
- **沈鹿薇**（亲妹妹）：16岁，学霸，高冷话少，被姐姐带偏后变成"夜市砍价王"
- **沈鹿泽**（豪门小少爷）：被惯坏的高冷小屁孩，被姐姐用夜市美食收买后主动喊姐
- **顾砚深**（养子/契约兄妹）：22岁，沈家收养的孤儿，身份敏感，高冷社恐，被沈鹿鸣带偏后反差萌。亲生父母留下的旧手表是唯一遗物
- **顾砚泽**（顾砚深表弟/纨绔富二代）：嘴炮纨绔，被沈鹿鸣的地气智慧折服后变成小弟，在校园冲突中负责录视频直播"今日打脸进度"
- **沈清雅**（养母/陈清竹闺蜜）：优雅克制，一直对沈鹿鸣冷淡，后被女主直率打破心防。当年因愧疚代为照顾顾砚深
- **顾老太太**（顾家老太君）：毒舌老太太，一眼看穿沈鹿鸣本色并力挺，成为她在豪门的靠山

**家族规则**：
- 每月家族聚餐必须全员到场，座次严格按辈分排列（沈鹿鸣的"随便坐"引发第一次冲突）
- 家族企业决策需大房二房三房共同签字
- 子女婚嫁讲究门当户对（顾砚深被逼相亲的根源）

**内部矛盾**：
- 沈天泽与陈清竹因当年送走女儿一事心存芥蒂，夫妻关系疏离
- 沈鹿鸣回归引发的资源重新分配焦虑
- "养子"顾砚深的身份尴尬——既不是亲生，又承担家族责任

### 夜市街（平民势力）

**地位**：锦城老城区最热闹的夜市，三百多个摊位，是本地人的深夜食堂，也是沈鹿鸣的精神根据地。

**核心人物**：
- **沈大强**（养父/烤串摊主）：江湖义气老炮儿，烤串手艺一绝，摊位是夜市的"地标"。对豪门用烧烤摊规矩考验
- **苏糖**（夜市闺蜜）：从小一起摆摊长大，游戏宅，用"游戏视角"吐槽豪门生活（"这波是降维打击""豪门副本太难了"）
- **夜市众摊主**：卖麻辣烫的老王、套圈摊的老李、卖糖葫芦的张婶——构成沈鹿鸣的"平民智囊团"

**夜市生存法则**（沈鹿鸣的"地气哲学"来源）：
- 砍价三原则：先夸再砍、对半起砍、转身就走
- 摆摊四要素：位置、吆喝、回头客、应急处理
- 人情世故：邻居有难大家帮、收摊后轮流请客、孩子放假互相照看

**面临威胁**：
- 城市改造计划，夜市街可能被拆迁（中期冲突线）
- 商业地产商想把夜市改造成商业广场（与沈家企业的利益冲突）

### 太太圈（社交势力）

**地位**：锦城豪门太太们的社交圈层，表面下午茶插花，实际是攀比战场。

**核心人物**：
- **赵美兰**（主要对手）：女儿留学海外，热衷攀比，每次被沈鹿鸣用"地气"打脸
- **林诗雅**（假闺蜜）：精致利己主义者，表面关心实则背后造谣，每次算计都被沈鹿鸣用直球打脸
- **周子轩**（纨绔子弟）：自恋孔雀型，骚扰沈鹿鸣被一脚踹飞后成为笑柄
- **周太太、钱太太**（跟风者）：赵美兰的应声虫，后期转向支持沈鹿鸣
- **方太太**（中立派）：低调务实，暗中欣赏沈鹿鸣

**攀比项目**：子女成绩、留学学校、老公职位、包包品牌、下午茶地点、朋友圈点赞数

**社交规则**：
- 每周三下午固定聚会
- 新人入场需"老人"引荐
- 攀比不能太直白，要用"凡尔赛"话术

## 核心设定

### "画风突变"机制

这是全书的核心喜剧引擎——沈鹿鸣用"夜市生存法则"改造豪门生活，每次改造都遵循固定节奏：

1. **碰撞**：豪门规矩 vs 地气做法，双方都不适应
2. **发酵**：地气做法在豪门环境中产生意想不到的喜剧效果
3. **扩散**：一个家庭成员被"带偏"，开始主动使用地气做法
4. **固化**：地气做法成为家族新常态，但保留豪门的精致底色

**改造层次**：
- **生活习惯层**：外卖、路边摊、砍价、菜市场（最先被改造）
- **社交方式层**：太太圈话术、待客之道、人情往来（中期改造）
- **商业思维层**：摆摊经验应用于企业经营、直播营销、接地气的品牌策略（最终改造）

### "契约兄妹"关系规则

沈鹿鸣与顾砚深的关系是全书感情线主轴，其发展遵循以下规则：

1. **互助期**（1-40章）：各取所需——她帮他挡相亲，他教她豪门规矩
2. **默契期**（41-80章）：无需言语的配合，外人开始误会
3. **危机期**（81-120章）：身份秘密揭露，信任动摇
4. **确认期**（121-148章）：从契约到真心，双向奔赴

**契约条款**（初期）：
- 沈鹿鸣帮顾砚深挡相亲，顾砚深帮沈鹿鸣应对豪门社交
- 对外称"兄妹"，私下保持距离
- 任何一方想终止，提前一周通知

### "地气商业学"体系

沈鹿鸣的商业智慧来自夜市实战，后期应用于家族企业，形成独特的方法论：

| 夜市经验 | 商业应用 | 对应源文模式 |
|---------|---------|-------------|
| 砍价话术 | 谈判策略 | 用话术碾压对手 |
| 吆喝引流 | 直播营销 | 用接地气吸引流量 |
| 回头客维护 | 客户关系管理 | 用真诚打动人心 |
| 应急处理 | 危机公关 | 用摆摊经验化解危机 |
| 位置选择 | 选址策略 | 用直觉判断市场 |

### 身世悬念线

沈鹿鸣被送走的真相贯穿全书，信息碎片逐步释放：

- **第1-3章**：生日宴认亲，不解释原因
- **第20-25章**：沈大强酒后暗示"当年有苦衷"
- **第50-55章**：陈清竹旧照片泄露，暗示与家族斗争有关
- **第80-90章**：沈老爷子遗嘱曝光，揭示当年被迫送走的真相
- **第120-130章**：全部真相揭露——涉及家族利益、保护孩子、被迫妥协

## 关键道具

1. **烤串夹子**：沈鹿鸣的"身份象征"。生日宴认亲时手里攥着的烤串夹子，后来成为她与豪门碰撞的符号——用它夹过豪门牛排、在家族聚餐上烤串、教顾砚深翻烤串时被烫到手。每次出现都承载"地气"与"豪门"的碰撞。

2. **沈老爷子的旧算盘**：家族传承信物，原本由大房保管。沈鹿鸣发现算盘上的刻痕是爷爷当年夜市摆摊时的记账痕迹——原来沈家也是白手起家。这个发现成为她"地气商业学"的精神支柱，也是说服家族接受她理念的关键道具。

3. **夜市VIP卡**：沈大强给沈鹿鸣的十八岁生日礼物，夜市所有摊位通用的打折卡。表面是塑料卡片，实际承载着整个夜市街的人情网络。后期成为沈鹿鸣调动"平民资源"的信物——太太圈打脸时，她用这张卡组织夜市摊主配合直播，三天扭转舆论。

4. **陈清竹的旧围裙**：陈清竹年轻时与丈夫在夜市摆摊的围裙，被她藏在衣柜深处。沈鹿鸣意外发现后，理解了母亲当年的处境——她也曾是"地气"的人，嫁入豪门后被迫隐藏。围裙成为母女和解的情感纽带。

5. **顾砚深的旧手表**：顾砚深亲生父母留下的唯一遗物，走时不准但从不摘下。沈鹿鸣帮他修好手表（用夜市修表摊的手艺），成为两人关系从"契约"转向"真心"的象征物。
