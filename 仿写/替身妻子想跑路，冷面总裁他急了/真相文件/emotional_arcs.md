# 情绪弧线

## 全局情绪基调

轻松搞笑为主，虐心为辅。前期日常喜剧，中期误会拉扯，后期真相大白甜收尾。

## 第1-10章情绪弧线

```
第1章：懵逼→恐惧→接受→制定计划（穿书觉醒）
第2章：试探→发现豪宅→震惊→冷静分析（环境认知）
第3章：尴尬→紧张→意外发现→小得意（第一次跟男主互动）
第4章：好奇→意外→微甜→警觉（男主态度变化）
第5章：危机感→应对→小聪明→暂时安全（外部威胁）
第6章：日常→搞笑→温馨→意外心动（日常甜点）
第7章：焦虑→计算→规划→坚定（跑路计划推进）
第8章：误会→吃醋（自己都没察觉）→否认→自我安慰
第9章：意外→紧张→感动→强行理智（男主示好）
第10章：试探→震惊→纠结→决定加速跑路（提离婚）
```

## 情绪节奏原则

- 每3章一个小高潮（甜/搞笑/危机）
- 每5章一个中型转折
- 连续2章日常后必须插入冲突
- 每章结尾必须有情绪钩子（好奇/期待/紧张/心动）
