# 第3章 他没回来

- 源文对应：源文第3章 查岗哥
- 新书事件：苏念念穿书第一夜独守空房，发现陆景琛彻夜未归，从林姨口中得知他从不回景苑过夜；她翻遍衣帽间确认自己的"替身"处境，决定加速攒钱跑路；陆景琛在公司反常地查看手机，助理和女高管们议论"陆太太从不查岗"，陆景琛莫名烦躁（原创）
- 因果逻辑：苏念念穿书醒来→发现枕边冰凉、陆景琛彻夜未归→林姨送早餐时无意透露"先生住公司附近的公寓，三年来景苑只回来过七次"→苏念念确认自己在原书中就是个摆设→翻衣帽间发现满柜奢侈品但没有一件私人痕迹，梳妆台上连合照都没有→彻底清醒：这段婚姻就是空壳，白月光回来之前必须跑→打开手机投简历，用室内设计专业找设计公司实习→场景硬切到陆氏集团→助理陈薇发现陆景琛今天频繁看手机，以为在等重要电话→女高管宋总监打趣"陆总，太太今天没给您送汤？"→陆景琛冷脸说"她不管这些"→宋总监私下跟陈薇说"陆总结婚三年，太太从没来过公司，也太可怜了"→陆景琛听到后脸色更冷→傍晚苏念念从外面回景苑，发现客厅灯亮着，陆景琛坐在沙发上，冷冷问她"去哪了"
- 情绪弧线：惊醒独处（空旷不安）→ 林姨点破真相（心凉确认）→ 翻衣帽间（绝望中的清醒）→ 决定跑路+投简历（务实的紧迫感）→ 公司群像：陆景琛的反常关注（暗流涌动）→ 回家撞见陆景琛（意外紧张）
- 钩子：苏念念以为自己是透明人，陆景琛根本不在乎——但他今晚破天荒回了景苑，还质问她的行踪。她不知道的是，他在公司一整天都心不在焉。
