## 第1章 穿成替身妻子？

- 源文对应：源文第1章《穿、穿书了？》
- 新书事件：苏念念从陌生的豪华大平层卧室醒来，发现自己穿书成了陆景琛的替身妻子沈薇薇。通过涌来的原主记忆，她得知自己三个月后白月光回国就要被扫地出门送精神病院，于是立刻制定了"攒钱→提离婚→跑路"的生存计划。然而当晚，那个三年没回过家的合约丈夫陆景琛，却突然推门走进了卧室。（原创）
- 因果逻辑：苏念念原本在加班后打车回家 → 出租车被一辆闯红灯的货车撞上 → 她失去意识，醒来时身处一间陌生的豪华卧室 → 脑海中涌入原主沈薇薇的记忆：三年前因长相酷似陆景琛的白月光顾清漪，被陆家用联姻名义娶进门当替身，陆景琛从未碰过她，每月打十万生活费当补偿，三年来回家次数一只手数得过来 → 苏念念同时想起原书剧情：白月光三个月后回国，陆景琛立刻提离婚，原主沈薇薇不肯，纠缠作妖陷害白月光，最终被陆家送进精神病院惨死 → 她判断自己的处境极度危险，必须在白月光回国前主动离婚跑路 → 算了下三年攒的钱加上离婚补偿，勉强够在二线城市安身 → 正盘算着怎么开口提离婚，卧室门突然被推开，陆景琛回来了
- 情绪弧线：[迷糊] 陌生环境醒来，浑身酸痛 → [震惊] 涌入原主记忆，意识到穿书 → [恐惧] 得知原书结局是精神病院惨死 → [冷静] 分析局势，制定跑路计划 → [庆幸] 发现还有三个月时间窗口，来得及 → [紧张] 卧室门被推开，三年不回家的陆景琛突然出现
- 钩子：陆景琛站在卧室门口，目光落在她身上，语气平淡地说了句"还没睡？"——三年不回家的人，今天怎么回来了？
