# 项目截图和演示

本目录用于存放项目截图和演示 GIF。

## 建议的截图内容

### 1. 项目架构图
- 六阶段流水线架构图
- 双执行模式对比图

### 2. 使用流程截图
- 配置文件示例
- 运行命令示例
- 输出结果示例

### 3. 质量检查截图
- 七维检查报告
- 对比报告示例

### 4. 演示 GIF
- 完整流程演示
- 单章生成演示
- 质量检查演示

## 如何添加截图

1. 将截图或 GIF 文件放入此目录
2. 在 README.md 中引用：

```markdown
![项目架构](docs/screenshots/architecture.png)
![演示 GIF](docs/screenshots/demo.gif)
```

## 截图命名规范

- `architecture.png` - 项目架构图
- `pipeline-flow.png` - 流程图
- `config-example.png` - 配置文件示例
- `output-example.png` - 输出结果示例
- `quality-report.png` - 质量检查报告
- `demo.gif` - 完整演示 GIF
- `quick-start.gif` - 快速开始演示

## 工具推荐

### 截图工具
- Windows: Snipping Tool, ShareX
- macOS: Screenshot (Cmd+Shift+5)
- Linux: GNOME Screenshot, Flameshot

### GIF 录制工具
- Windows: ScreenToGif, LICEcap
- macOS: Kap, GIPHY Capture
- Linux: Peek, Byzanz

### 图片编辑
- 轻量级: Canva, Pixlr
- 专业级: Photoshop, GIMP
