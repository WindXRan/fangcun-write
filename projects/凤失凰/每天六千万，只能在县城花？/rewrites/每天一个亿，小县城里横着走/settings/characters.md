# 角色设定

## 主角团

### 林深（男主）
- **年龄**：26岁
- **职业/身份**：
  - 前：深城某互联网大厂游戏策划，月薪两万，996社畜
  - 现：南山村“深林创投”公司创始人，被系统绑定的“天选打工人”
- **外貌**：身高178cm，体重135斤，常年熬夜导致脸色苍白，略显清瘦。辞职后气色好转。戴黑框眼镜，长相斯文干净，属于典型的“闷骚理科男”长相。
- **性格特征**：
  - 前期：疲惫、压抑、有野心但被现实磨平棱角，吐槽欲旺盛（内心戏丰富）
  - 后期：沉稳、有魄力、格局打开，但依然保持接地气的幽默感，擅长扮猪吃虎
  - 核心性格：表面随和好说话，内心极其有主见和底线，一旦下决定十头牛都拉不回来
- **背景故事**：
  - 华南理工大学软件工程毕业，在深城南山科技园某游戏大厂干了三年策划
  - 月薪两万看似光鲜，但在深城根本存不下钱——房租四千五（合租单间）、吃饭三千、通勤八百、社交两千，每个月基本月光
  - 连续加班三个月后，体检出心律不齐、高血压临界值，在凌晨两点下班时差点晕倒在家门口
  - 辞职前最后一个月，因项目上线连续通宵，绩效被评C，年终奖泡汤，还背了五万块信用卡债
  - 辞职后回到从小长大的南山村（城中村），准备躺平混吃等死
- **经济状态**：
  - 初期：负债5万（信用卡），存款8千
  - 系统到账后：日收1800万+（随人口增长递增）
- **专属技能/金手指**：村域消费金系统（详见world.md）
- **人物弧线**：从“逃离内卷的失败者” → “误打误撞的村里首富” → “真正有社会责任感的企业家”
- **口头禅**：“（内心OS）这系统是不是有病？”

### 苏念（女主）
- **年龄**：25岁
- **职业/身份**：
  - 前：深城某建筑设计院建筑师，年薪十五万
  - 现：回南山村接手父亲的小五金店，后被林深挖角为“深林创投”运营总监
- **外貌**：
  - 身高165cm，短发到肩，身材纤细但干活利落
  - 喜欢穿工装连体裤、帆布鞋，常戴一顶鸭舌帽
  - 长相清冷，但笑起来有两个酒窝，反差萌明显
- **性格特征**：
  - 外冷内热，做事雷厉风行，典型ISTJ人格
  - 在深城工作两年，被甲方的奇葩需求和无休止的改稿搞得心累，辞职回村
  - 相信“好设计能改变生活”，对城中村的建筑和空间有独到见解
  - 面对林深时，从最开始的高冷疏离 → 职业认可 → 伙伴信任 → 情感升温
- **背景故事**：
  - 同济大学建筑系毕业，在深城某知名建筑设计院干了两年
  - 参与过几个南山的商业综合体设计，但对“只服务富人”的地产逻辑深感厌倦
  - 父亲苏大强在南山村经营“强哥五金店”二十年，最近摔伤了腿，苏念辞职回村照顾
  - 高中时和林深是隔壁班同学，但基本没说过话（林深是理科班闷葫芦，苏念是文科班高岭之花）
- **家庭**：
  - 父亲苏大强：南山村土著，开五金店，老实厚道，手艺好
  - 母亲早逝，父女俩相依为命
- **能力与专长**：
  - 建筑设计、空间规划、项目管理
  - 对城中村的“握手楼”和“一线天”有深入研究和改造热情
- **与男主的关系发展**：
  - 源文模式：高中同学重逢 → 女主主动求职 → 事业伙伴 → 情感萌芽
  - 新书模式：高中隔壁班 → 因五金店业务产生交集 → 女主应聘运营总监 → 并肩作战改造南山村 → 情感逐渐升温

## 核心配角

### 林深的关系网

**胖子（刘全）**
- **年龄**：27岁
- **身份**：林深大学室友兼好兄弟，南山村“全村人的希望”二手车行老板
- **外貌**：真·胖子，180斤，总是笑嘻嘻的，穿花衬衫戴大金链子（假货），一副社会人打扮但人怂心善
- **性格**：话痨、爱面子、胆小怕事、但关键时刻很讲义气
- **背景**：大学毕业后在深城混了几年，最后回村啃老，用家里的两间铺面开了个二手车行，生意半死不活
- **作用**：类似源文陆建军的“引路人”角色，带林深完成第一笔系统消费（买车），后续成为林深在村里的“百事通”和跟班
- **出场节奏**：第1章出场，之后每5-10章出现一次

**苏大强**
- **年龄**：52岁
- **身份**：苏念的父亲，南山村“强哥五金店”老板
- **性格**：老实巴交、手艺精湛、有点固执、疼女儿
- **作用**：女主的情感纽带，也是展示南山村“新旧冲突”的典型代表（老一辈对改造的抵触 → 逐渐理解）

### 南山村势力

**陈伟强（强哥）**
- **年龄**：45岁
- **身份**：南山村本地土霸王，南山物业管理公司老板，实际控制村里80%的出租屋和商铺
- **外貌**：矮胖，光头，脖子挂一条粗金链（真货），手腕上盘着菩提籽，常年穿人字拖和大短裤
- **性格**：笑面虎，表面热情好说话，实际上心狠手辣，控制欲极强
- **背景**：南山村土著，从父辈开始就经营村里的物业租赁，手下有一批本地青年充当“保安队”，垄断了村里所有建材、装修、废品回收等生意
- **作用**：前期核心反派，类似源文胡奎的角色，代表“旧势力”
- **冲突点**：
  - 林深的系统消费冲击了陈伟强的垄断利润
  - 林深想改造城中村，动了陈伟强的蛋糕（租金定价权、商户控制权）
- **出场节奏**：第8-10章出场，之后作为主要反派持续出现

**白松（松哥）**
- **年龄**：58岁
- **身份**：南山村党支部书记、村委会主任、股份公司董事长
- **外貌**：瘦高个，头发花白但精神矍铄，永远穿白衬衫和深色西裤，戴老花镜
- **性格**：老派干部，讲究“规矩”和“稳定”，对新生事物持谨慎态度，但对村里发展有责任心
- **背景**：在南山村当了三十年书记，看着村子从渔村变成城中村，经历过大大小小的拆迁和改造，对“外人来村里搞事”有天然戒备心
- **作用**：类似源文白局长的角色，代表“体制内”力量，既是阻碍也是可争取的对象
- **冲突点**：
  - 对林深的改造方案持怀疑态度，担心“搞乱村子”
  - 在陈伟强和林深之间摇摆，后期逐渐被林深的诚意和成果打动

### 官方力量

**王局（王志宏）**
- **年龄**：50岁
- **身份**：深城南山区城市更新和土地整备局局长
- **性格**：典型的基层官员，谨慎、讲程序，但对真正干实事的人会网开一面
- **作用**：类似源文王局长，第一个被林深搞定的官方人物，为后续改造提供政策支持
- **出场节奏**：第5-8章出场

### 女性配角

**陈思思**
- **年龄**：23岁
- **身份**：南山村一家美甲店的店员，来自江西农村，村里典型的“深漂”底层女孩
- **外貌**：长相甜美，但穿着廉价，总是化浓妆掩盖疲惫
- **背景**：高中辍学来深城打工，在城中村租住鸽子笼大小的单间，被陈伟强的侄子纠缠
- **作用**：类似源文陈思甜，展示“深漂”底层女性的生存困境；也是林深决定改造城中村的触发点之一
- **关键情节**：参与了盛世皇朝事件（对应源文第30章），给林深发求救短信

**方瑜**
- **年龄**：35岁
- **身份**：南山区某律师事务所合伙人，主要负责城市更新和商业地产法律事务
- **性格**：干练、专业、有少许理想主义，是林深的法律顾问和智囊
- **作用**：类似源文方瑜，提供法律支持，也是林深了解政策边界的窗口
