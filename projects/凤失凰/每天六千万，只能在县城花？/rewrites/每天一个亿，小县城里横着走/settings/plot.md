# 情节设定

## 主线弧线（一句话概括）
一个在城市内卷到崩溃的边缘青年，被迫绑定只能在城中村消费的无限资金系统，从改造一碗猪脚饭开始，逐步将脏乱差的城中村建成大湾区青年的梦想孵化器，最终打破系统和地理的边界，证明限制本身即是最大的自由。

## 核心冲突

### 冲突类型

| 冲突维度 | 类型 | 描述 | 升级方向 |
|----------|------|------|----------|
| **世界观冲突** | 人vs系统 | 林深有无限资金但被限制在城中村，如何在“鸡肋”的限制下最大化系统价值？ | 从被动接受到主动利用规则，最终打破/超越规则 |
| **利益冲突** | 新势力vs旧势力 | 林深要改造南山村，动了以陈伟强为代表的包租公和垄断势力的蛋糕 | 从停车场争夺 → 建材采购战 → 物业管理权争夺 → 村股份公司控制权 |
| **阶层冲突** | 原住民vs深漂族 | 靠收租坐享其成的原住民与拼命工作却被压榨的深漂族之间的矛盾 | 从个人恩怨（陈思思事件）上升到制度层面（村规民约改革） |
| **理念冲突** | 旧改vs有机更新 | 开发商想推平重建，林深想保留城中村活力并改造升级 | 从私下较量到街道办听证会，最终影响城市更新政策 |
| **情感冲突** | 林深vs苏念 | 林深想商业最大化，苏念坚持设计的人文关怀，两人在“利润vs品质”上反复拉扯 | 从职业分歧到价值观碰撞，最终融合为“有温度的商业” |

## 情感内核（源文vs新书，必须本质不同）

| 对比维 | 源文《每天六千万》 | 新书《我的系统只认城中村》 |
|----------|------------------|--------------------------|
| **女主付出方式** | 沈璃主动求职、熬夜加班、做陆明的事业助理 | 苏念从五金店偶遇开始，逐渐被林深的改造理念打动，用自己的建筑设计专业能力帮助林深 |
| **女主被伤害原因** | 暂无明确情感伤害线（慢热，感情线较淡） | 苏念在深城设计院被甲方和领导不尊重专业，她的设计被改得面目全非还要承担骂名，回到村后父亲又不理解她“读了名校回来卖五金” |
| **情感内核本质** | 事业伙伴衍生情感 | **“两个逃离内卷的理想主义者，在城中村找到了彼此，也找到了重新定义成功的方式”** |
| **核心情感冲突** | 从高中同学重逢到并肩作战 | 从“理工男vs高冷学姐”的刻板印象，到逐渐发现彼此都是被城市伤害过的理想主义者，在共同改造城中村的过程中互相治愈 |
| **本质差异** | 外向型情感（通过事业产生羁绊） | **内向型情感（通过共同价值观和伤痕互相吸引）** |

**新书情感核心**：两个被“深城速度”抛弃的年轻人，在城中村这个“城市伤疤”里，用无限资金和好设计，为同样被城市抛弃的深漂族建了一个家，也找到了自己的归属。

### 源文关系网
```
陆明（男主）
  ├─ 陆建军（叔父引路人）
  ├─ 沈璃（高中同学→事业伙伴→爱情线）
  ├─ 胡奎（本地建材商→前期对手→后期合作关系）
  ├─ 白局长（住建局→官商博弈对象）
  ├─ 王局长（财政局→官方助力）
  └─ 陈思甜（底层悲剧→触发人性思考）
```

### 新书关系网（结构类似，具体关系模式不同）
```
林深（男主）
  ├─ 刘全（大学兄弟引路人，非叔父）
  ├─ 苏念（高中隔壁班→五金店偶遇→事业伙伴→爱情线）
  ├─ 陈伟强（本地包租公+物业垄断→前期对手→持续博弈）
  ├─ 白松书记（村支书→新旧理念博弈对象）
  ├─ 王志宏局长（城市更新局→官方助力）
  └─ 陈思思（深漂底层女孩→悲剧触发→人性思考）

关系对比：
- 源文叔父 → 新书兄弟（亲属→朋友，关系模式不同）
- 源文同班同学 → 新书隔壁班不认识（重逢方式不同）
- 源文建材商垄断 → 新书物业+建材双垄断（冲突面更广）
```

## 觉醒契机（源文vs新书，触发方式和情感路径必须本质不同）

### 源文觉醒契机
- **触发事件**：辞职回乡后第三天，躺在床上刷短视频时系统自动绑定
- **心境**：迷茫、疲惫、不知道自己要干什么，被动接受系统
- **觉醒方式**：被系统“砸”中，从测试消费边界开始，逐步接受“只能在县城花”的现实
- **情感路径**：从“这破系统真鸡肋” → “先花着玩” → “好像真能干点事” → “我要把这破县城建成乌托邦”

### 新书觉醒契机
- **触发事件**：辞职回南山村躺平第七天，在村口吃猪脚饭时，手机弹出系统提示【村域消费金系统已绑定】
- **心境**：比源文更惨（负债5万+信用卡催债+高血压边缘），从一开始的“彻底摆烂”到被系统激活
- **核心差异**：林深的觉醒是因为在城中村看到了太多“像自己一样被城市压榨的人”——外卖员凌晨三点还在跑单、深漂女孩租住3平米鸽子笼、大学生创业者在城中村租不起办公室——他意识到自己有无限资金却只想躺平，是一种特权式的傲慢
- **觉醒场景**：第5章，林深在村口看到外卖员李哥（35岁，有妻有子，每天跑单16小时）在猪脚饭店门口累得睡着，手机还亮着接单页面。那一刻他终于明白：**“系统不是让我当首富的，是让我当工具人的——为这群没有选择的人，创造选择的工具。”**
- **情感路径**：从“这系统有病吧限制我” → “躺平算了，每天白嫖1800万也挺好” → “妈的这钱花在村里确实能改变点什么” → “我林深不是首富，我是南山村的首席改造官” → “我要让所有在深城漂泊的人，都有一个体面的落脚点”

**本质差异**：
- 源文觉醒：个人英雄主义（“我要当县城首富”）→ 逐渐演变为社会责任
- **新书觉醒：共情驱动（“我和他们一样是失败者”）→ 从利己到利他的自然过渡，情感颗粒度更细**

## 决裂场景（源文vs新书，冲突起点和对抗性质必须本质不同）

### 源文决裂触发
- **核心对手**：胡奎（建材商）
- **冲突起点**：万家福改造时的建材采购权之争
- **决裂方式**：胡奎通过白局长施压，陆明当面硬刚，把底牌摊开
- **对抗性质**：商业垄断vs自由竞争

### 新书决裂触发
- **核心对手**：陈伟强（包租公+物业垄断）
- **冲突起点**：林深想改造南山村的一栋烂尾农民房做“共享办公楼”，但陈伟强控制着全村的建材供应和装修队，强行要分一杯羹
- **新书具体事件**：林深公开招标建材和装修，陈伟强派保安队阻挠施工，在村口堵住林深的车，要求“按规矩办事”
- **决裂场景**（对应源文第10章）：林深在南山村村委会，当着白松书记和陈伟强的面，直接说出：
  > “强哥，你的规矩我懂。但我林深的规矩只有一条：公平。建材招标贴在村务公开栏上，所有符合资质的公司都能投。你陈伟强的伟强建材当然可以来，但中不中标，评标委员会说了算。至于村里的'装修保护费'——从今天起，我深林创投的每一分钱工程款，都走公账，都开发票。你收，我报警；你不收，咱们公平竞争。”
- **对抗性质**：**村霸垄断vs规则公平**（比源文多了“体制内博弈”维度，因为陈伟强是村股份公司副董事长，有部分官方身份）

**核心差异**：
- 源文：胡奎是“认错可以，但利润必须分”的江湖逻辑
- **新书：陈伟强是“南山村我话事，要么合作要么滚”的土皇帝逻辑，更接近“改开早期的地方保护主义”，冲突更底层也更复杂**
