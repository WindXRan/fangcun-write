## 主线弧线
一句话：错位的守护与弥补——他为她守住了故宅，她要如何为他照亮长夜？

### 核心冲突

| 冲突类型 | 描述 | 升级方向 |
|----------|------|----------|
| **情感误解冲突** | 顾予安以为宋怀月当年软弱无能，在她被羞辱时不敢出面维护，是个旁观者。 | 揭示宋怀月的真实本性（并非软弱，而是为了她默默承担了更多不堪的后果）。 |
| **身份落差冲突** | 顾家小姐家破人亡，沦为平民；宋家子弟虽没落但清白。顾予安感到与过去割裂，不愿接受宋怀月的“愧疚式”守护。 | 揭示宋怀月并非出于愧疚，而是十二年如一日的暗恋与忠诚。 |
| **外部现实冲突** | 当年造成顾家悲剧的残余势力注意到顾予安的归来和重振家业的举动，试图打压。宋怀月的茶馆成为保护她的前哨。 | 从商业打压升级为威胁人身安全，逼二人联手面对共同的敌人。 |
| **内心秘密冲突** | 宋怀月隐瞒了十二年“喜欢她”这件事，以及他收殓顾家尸骨时发现的惊天秘密。 | 秘密的揭露将彻底颠覆顾予安对过去的认知，以及他对她家人的承诺。 |

### 情感内核

- **源文情感内核**：《春深锁惊鸿》是“男主因嘴贱伤害女主，后用行动和卑微的守护追回”。核心是**追妻火葬场**。
- **新书情感内核**：《长夜无明》是“女主以为男主懦弱，是情感上的旁观者，多年后发现他才是那个为她扛起一切的人”。核心是**错位的误解与迟来的发现**。本质不同：源文的伤害是主动言语造成的，新书的伤害是源于长期的沉默和误解。男主付出的方式是“沉默的牺牲”，是隐藏于幕后的守护。

### 人物关系架构

- **源文关系**：男主（纨绔将军之子）直接伤害女主（体弱太傅之女）→ 女主心死放弃 → 男主悔悟，疯狂追回。
- **新书关系**：女主（理性家变小姐）误解男主（温和没落子弟）是“旁观者” → 女主自我封闭，不接受他的守护 → 多年后，真相揭开，男主是“承担者” → 女主从误解到理解，双向治愈。

### 觉醒契机

- **源文契机**：男主看到女主与他人订亲，责任感与失去的前兆双重爆发，引发追回行动。
- **新书契机**：未来情节中，当顾予安为保护宋怀月，打算独自面对旧敌，重现当年“被抛弃”的局面时，宋怀月撕下“温和”的面具，用自己的方式（可能不太光彩，甚至阴狠）解决了问题，让顾予安第一次窥见他不为人知的一面，意识到他并非自己以为的那样懦弱无能。

### 决裂场景

- **源文决裂**：男主被误会与女二有染，且后期发现男主曾公开羞辱自己，女主心死另嫁，导致男主崩溃。
- **新书决裂**：未来情节中，因一个与当年顾家案有关的关键事件，宋怀月为保护顾予安，被迫在公开场合说出伤害她的话，承认自己这么多年只是出于“愧疚”，从未真心喜欢她。顾予安信以为真，彻底绝望，与他决裂。（与源文相反，新书是男主因“保护”而主动造成决裂，而非因自身犯错。）
