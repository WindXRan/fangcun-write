## 源文字数
3207字 → 新书目标 3207字（1:1对标）

## 角色（固定，不可改）
- 女主：顾予安
- 男主：宋怀月
- 配角：傅明远 → 替换为 温姑姑（女主的乳母）

## 信息释放对标（从源文观察推导）
- **源文第1章释放了**：
  - 男主闯祸（当众羞辱女主）
  - 男主家庭关系（父亲是将军，母亲强势）
  - 女主被羞辱的背景（体弱、不出门）
  - 女方家长态度（表面客气，实则拒绝）
  - 冲突因何而起（口无遮拦）
  
- **源文第1章未释放**：
  - 女主是否真如男主所说那样
  - 男主内心是否真的讨厌女主
  - 女主的真实想法和容貌
  - 具体发生了什么后续惩罚
  - 两人之前是否有交情

- **新书第1章释放对标**：
  - ✓ 男主做了什么导致冲突（换了具体事件）
  - ✓ 男主家庭关系（父亲/母亲态度）
  - ✓ 女主背景暗示（曾经家变）
  - ✓ 女方家长/长辈的态度（表面拒绝）
  - ✓ 冲突因何而起

- **新书第1章不释放**：
  - ✓ 男主真实心意（是否暗恋女主）
  - ✓ 女主的真实状态
  - ✓ 往事具体细节
  - ✓ 两人童年交情

## 时间线
- **本章时间**：顾家被平反后第三年，初秋（源文是暮春，必须换）
- **本章场景**：宣城西街茶馆（宋怀月的茶馆）
- **与前章衔接**：第1章为开篇，无前章
