# 剧情核心设定

### 主线和核心冲突

- **主线弧线（一句话）**: 被系统逼着去“温暖”全警局最不能碰的心机女犯和霸王花女警，郝仁在粉红陷阱中撕开了一张权与钱的犯罪黑幕。
- **核心冲突类型**:
  1.  **性别与权力的矛盾**: 全科室唯一的男领导，面对不服管的女警和有求于他的女犯，如何在压迫与反抗中建立权威与羁绊。
  2.  **系统与内心的矛盾**: 系统的任务是“安慰、接触、获得真心”，这与他内心“要破案、要正义”的原则不断冲突，他想当孤狼破案，系统却让他当妇女之友。
  3.  **悬疑与身世的矛盾**: 前几任科长的离奇遭遇与秦梦的案子逐渐交织，发现幕后的黑手似乎也在盯着郝仁的身体。
- **升级方向**: 从科室内的鸡毛蒜皮/系统任务→女犯遇袭/勾引科长→牵出局内人物孙文海→金融案浮出水面→危及郝仁父亲及更高层→郝仁用系统附加的“神技”破局（完成肉体与精神的双重反杀）。

### 情感内核（新书 vs 源文）
| 对比项 | 源文 | **新书设计（必须本质不同）** |
|:---|:---|:---|
| **女主类型** | 官二代娇女，从误会到归心 | **烈女型战友**：从绝对鄙视到生死同袍，共同破案。 |
| **付出方式** | 开篇即因父辈关系捆绑 | **由外而内**：先认可他的办案能力（帮她破了她父亲的悬案），再被系统的“强迫温柔”打动。 |
| **伤害原因** | 抢班夺权的小矛盾 | **原则对立**：她认为郝仁是在利用女犯/玩弄女性（系统要求），误会加深变成鄙视。 |

### 人物关系架构（新书 vs 源文）
| 对比项 | 源文 | **新书设计** |
|:---|:---|:---|
| **男主与女配1(秦梦)** | 监狱长与囚犯，单纯利用出狱 | **猪与老虎**：表面是科长与嫌犯，实际上都是在下棋博弈，互相利用破局，性张力中夹杂着智商对决。 |
| **男主与女配2(夏晴)** | 上下级帮忙 | **猎人与观察者**：夏晴把郝仁当“特殊的心理样本”观察，两人言语交锋不断，但夏晴在医学/尸检上给予核心支持。 |
| **男主与反派关系** | 直接因女犯而起 | **世仇+局内斗**：不仅是因为秦梦，更因为张文海是郝仁父亲的“政敌”，是害死林钰父亲的间接元凶。 |

### 数值与时间线对照表（底层锁死）

- **[时间线表] 关键事件前后顺序**
  1. 郝仁入职（D-Day）
  2. 秦梦案发（D-180 天）
  3. 系统激活（D-Day + 08:00）
  4. 郝仁与重案组起冲突（D-Day + 09:30）

- **[技能/等级表]**
  - 初期：无技能
  - 第1次任务后：格斗技能 Lv1（被动：关键时爆发，平时还菜）
  - 第3次任务后：唇语解读 Lv1

- **[伤病表] 角色状态记录**
  - D-Day + 1：郝仁左臂被重案组王队长压伤。
  - 恢复周期：3天。

- **擦边浓度对照表 (对标源文)**
  - **密度要求**: 每章必须包含至少1处“强制/意外肢体接触”或“男性视角下的女性特征描写（如制服腰线、手铐印记等）”。
  - **场景设计对标**:
    - 源文“医务室体检” -> 新书“夏晴给郝仁修复肩伤时的强制脱衣/上药”。
    - 源文“监区巡视” -> 新书“软包拘留室突然断水，秦梦开门湿身撞进郝仁怀里”。
