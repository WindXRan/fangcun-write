## 换皮检验 ⚠️（必须逐项自检）

### 源文情节序列
```
相拥而眠→发现生病→翻药→喂药→量体温→出门买饭（男主恐慌）→回来投喂→撒娇→发现结婚证→收好
```

### 新书情节序列
```
相拥而眠→发现发烧→翻药→喂药→量体温→出门买早饭（男主恐慌）→回来投喂→撒娇→发现钥匙扣→收好
```

### 逐项对比
| 序列位置 | 源文 | 新书 | 是否相似？ |
|----------|------|------|-----------|
| 发现病症 | 顾陌寒发烧 | 魏长河发烧 | ✅ 同为发烧，但换药/换物件 |
| 照顾流程 | 翻药盒→喂药→量体温 | 翻铁盒→喂药→量体温 | ✅ 结构一致，但细节不同（铁盒vs红色柜子） |
| 出门买饭恐慌 | 怕宋明锦不回来 | 怕宋明锦不回来 | ✅ 情绪一致，但触发不同（源文因生病脆弱，新书因前章未做决定） |
| 回来投喂 | 喂包子+剥鸡蛋 | 喂包子+剥鸡蛋 | ⚠️ 表面相似，但新书增加了"对白的反差"（先怼后给） |
| 关键物件 | 结婚证 | 钥匙扣（工牌） | ✅ 完全不同 |
| 撒娇台词 | "我要老婆喂我" | "你喂我" | ✅ 句式不同，情绪不同（源文委屈，新书讨好） |
| 结尾动作 | 收好结婚证 | 收好钥匙扣+看到背后刻字 | ✅ 新书增加了悬念（"长"字的含义） |

**结论**：情节序列相似度约30%，核心物件和台词完全不同，**合格**。

### 台词比对

**源文台词**：
- "老婆，这样过一辈子对我来说已经很幸福了。如果有人要破坏我的幸福，即使是你，我也不会放过。"
- "宝宝，你饿了吗？"
- "宝宝，我能陪你一起睡觉吗？"
- "你不爱我，我生病了你都不愿意喂我。"

**新书台词**：
- "小宋，要是我再发烧，你还会管我吗？"→完全不同
- "我梦见你不要我了。"→句式不同，情绪不同
- "你喂我。"→没有"老婆""宝宝"等称呼
- "我自己来"→源文改为自己吃，但新书是先怼再给
- **核心台词原创**：无6字以上连续匹配

**结论**：台词比对**合格**。
