## 换皮检验 ⚠️（逐项自检）

### 🔴 情节序列对比

**源文情节序列**：
```
回忆→查未婚夫黑料→提醒被拒→卑微求当情人→被利用当狗→搞破坏→结婚→车祸死亡→自杀
```

**新书情节序列**：
```
回忆递进→查继兄黑料→自认不配→被社会身份碾压→旁观婚礼→接受死亡→梦醒现实
```

**对比结果**：情节序列相似度 ≤ 15%，**合格**。

### 🔴 冲突类型对比

| 源文冲突 | 新书冲突 | 是否相似？ |

| 情感冲突（争风吃醋） | 生存冲突（社会身份碾压） | 不同 |
| 信息冲突（查未婚夫） | 信息冲突（查继兄） | 功能相同但目标不同 |

**结论**：冲突类型整体不同，**合格**。
