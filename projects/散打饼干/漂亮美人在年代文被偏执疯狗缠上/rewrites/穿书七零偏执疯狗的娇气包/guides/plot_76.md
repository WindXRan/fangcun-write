# 第76章 改革

广播里的新闻播报声从门缝里挤进来，像一盆冷水浇在宋明锦后颈上。

她用铅笔尖在草稿纸戳了个洞，抬头。办公室墙上挂着那台老式收音机，女播音员的嗓音正念着“允许部分个体工商户经营”。窗外，厂区里的梧桐树叶子落了大半，风把最后几片刮下来，贴在玻璃上啪嗒响。

“听见了没？个体户可以申请营业执照了！”对面工位上的裴爱华抬起头，眼睛亮得像捡到了钱，压低声音说，“我二舅昨天听说了，急得跟什么似的。”

宋明锦没接话，继续画她的设计图。铅笔划过纸面的沙沙声填满了沉默。

办公室里安静了几秒，坐在窗户边的何剑嗤了一声:“个体户？说得容易，地呢？钱呢？门路呢？你以为卖冰棍就能发财？”她边说边拿钢笔敲着搪瓷缸子，一下一下，节奏刺耳。

坐在门口的宋凤英也跟着接了话:“就是，毕业了国家给分配工作，铁饭碗不比风吹日晒的个体户强？”

裴爱华看了宋明锦一眼，眼神里全是“你说句话啊”的意思。宋明锦还是没抬头，只是把手中的铅笔转了半圈，轻声说了句:“枪打出头鸟。”

“你肯定有想法。”裴爱华凑过来，压低声音说得笃定。

宋明锦没否认。手在纸上游走，勾出一条设计线的轮廓。她的心不在纸上。

外头的风又大了，顶棚的铁皮被吹得嘎吱嘎吱响。设计科的暖气片只有一半是热的，她裹着那件灯芯绒外套，身子往桌上缩了缩，继续画。

她的思绪却飘到了上个月那个傍晚，城郊的废弃仓库里，魏长河掀开一块帆布，灰尘扬起来呛得她咳了好几声。她看见底下堆着小山高的铁管和齿轮，随口问了句“这些东西哪来的”，他没回答，只靠在墙边抽着烟说了一句话:“改革快了，这些东西马上就能见光。”

那个场景像是印在她脑子里了:仓库里的油灯被风吹得晃动，他的脸一半在光亮里，一半在暗处，烟头的火星忽明忽暗。他在等她说“好”，可她没说。她知道他的真实身份，一旦上了他的船，下面可就不是什么清新正经路子了。

她怕。

可是她也知道——她有脑子，他有货。他们各取所需，也许真能走在改革的风尖上。

可这个人情，她还不起。

下班铃响得像救命。

宋明锦收拾好图纸，把铅笔一根一根插回笔筒里，动作慢得跟故意拖延似的。裴爱华拎着包走过来，看了她一眼:“一块儿走？”

“你先走，我锁门。”

宋明锦等办公室的人差不多都走了，才把拉链拉上，背上包推着自行车出厂门。

铁门外的路边停着一辆解放牌卡车，灰色的车身，车头的油漆剥了一大块，露出底下锈迹。驾驶室的窗户半开着，里面有人抽烟，烟雾一缕一缕飘出来，被风吹散了。

车牌号她认得。

宋明锦的脚步滞了一下，心跳一下子加快了几分。但她的表情没有变，只是跨上自行车，低了一下头，假装什么也没看见，从卡车旁边骑了过去。

车没开走，也没人下车喊她。

她骑得更快了，迎面吹来的风几乎要把她的眼睛冻出水来。

第二天中午，食堂里人声和碗筷碰撞的声响混在一起，空气里飘着白菜炖粉条的味道。裴爱华端着盘子坐到何剑对面，夹起一块萝卜放在嘴里嚼，眼珠子却滴溜溜转着。

何剑看她那副样子:“又怎么了？”

裴爱华放下筷子，探着身子压低声音:“你们发现没有？宋明锦最近下班老是骑那条路回来，还总有一辆卡车搁厂门口待着。”

“卡车？”何剑眉毛一挑。

“解放牌，灰色的，车头漆都掉了。我昨天亲眼看见的，就在厂门口停着，里面有人抽烟，一直没动。等宋明锦骑过去了，那车才开走。”

宋凤英凑过来:“是不是运输队的？他们司机整天在外面跑，谁会专门跑我们厂门口候着？”

裴爱华捡着筷子敲了一下碗，低声说:“说不定就是运输队那个姓魏的。”

“魏长河？”何剑想了一下，“我听说那人不太好惹，拳头硬，路子也野，运输队里没人敢惹他。他怎么跟宋明锦扯上关系了？”

“谁知道呢。”裴爱华夹了一筷子白菜，“反正看着不像是正经朋友。”

那天傍晚，宋明锦回到家时，天已经麻黑了。

北方的冬天天黑得快，路灯也是隔两盏才亮一盏。她推着自行车进院子，刚把车锁好，低头一看，门口地上放着一个蛇皮袋。

袋子里是一袋子大米，还有一捆扎好的青菜，绿得发亮。

这些菜和米，光有钱可买不着，还得有票。没有门路的人，想买这种品相的菜都难。

她蹲下，摸了摸袋口，米粒从扎线缝里漏了几颗出来，白得发亮。

她的心一沉，抬头四下里看了看。院里的槐树光秃秃的，墙角晾着几件衣服。隔壁王婶家的灯已经亮了，厨房里传来炒菜的嗞啦声。

她深吸一口气，转身就想往巷子里走。

“怎么了？不认识人了？”

声音从巷口的方向传来，低沉，带着一丝漫不经心的笑。宋明锦身子一僵，脚步下意识地就顿住了。

魏长河从墙角的阴影里走出来，夜里看不大清他的脸，只有烟头的那一点火星在黑暗中明灭。他走到路灯下，穿着一件深蓝色的工装，领口敞着，露出里面的白背心。嘴角叼着一根烟，表情像是在说“你这反应挺有意思”。

宋明锦火一下子窜上来，攥着车把的手发紧:“魏长河，你能不能别老是往我家门口放东西？”

他掐了烟，用鞋底捻灭烟头:“怕你饿死。”

“我不缺你这点东西。”她瞪他。

“知道你不缺。”他笑了一下，“这是我刚才收的货，卸多了，顺道带过来的。”

“你——”

“行了，别瞪了。”魏长河收住笑，走近两步，声音压低了些，“跟你说正事。政策下来了，我的店可以开了。城郊那个仓库我租了三年，马上就开始收拾。做五金件批发生意，光明正大的。”

宋明锦愣住了。

他继续说:“有个忙求你。你替我管账，我不识几个字，算账更是一塌糊涂。你设计图画得那么好，账本肯定管得明白。年底我给你分红。”

“我有工作。”她条件反射一样脱口而出，声音却有些虚。

“你那工作一个月多少钱？”魏长河歪了歪头，伸手比了个数，“32块吧？转正后也就这个数。我一个月的利润，能给你这个数。”

他伸出三根手指。

三百块。

这个数字不是她能轻视的。纺织厂的模具设计工作做一辈子，也未必能存出这个数来。何况现在厂里已经在传裁员的消息了，铁饭碗不再那么牢靠。

宋明锦的目光落在地上的那袋大米上。白生生的米粒在昏黄的路灯光下，像是路边捡来的碎银。

她想说“好”，但喉头像被人掐住了，咽了口唾沫才压下那股冲动:“我再想想。”

“随你。”魏长河的语气很轻，像是这件事根本不需要她纠结，“东西提进去吧，外面凉。米放地上容易招老鼠。”

他说完就转身走了。卡车引擎响了两声，车灯在巷口亮起，像两只黄色的眼睛，然后缓缓驶远了。

宋明锦站在门口，看着那袋大米发了很久的呆。

凉意从鞋底渗上来，她的脚被冻得有些发麻。她终于弯腰提起袋子，推开门，把东西搬进屋里。

屋里的暖气烧得不够热，她泡了个热水袋捧在手心，靠着墙坐了很久。

她知道拒绝是对的——魏长河这个人沾上了就甩不掉。可是她也知道，自己早晚得做这个选择。纺织厂不是她的归宿。改革开放后大批的个体经营者发了财，买得起房子和粮食，不再看铁饭碗的脸色。

她怕的是自己上了他的船，就再也下不来了。钱能防身，也能绑人。

她躺在床上，关了灯，房间里黑漆漆的，只有窗外路灯的光从窗帘缝隙里透进来。

远处传来卡车引擎远去的声音，慢慢消失在这个冬天的夜里。

她闭上眼，翻了个身。

明天再说吧。
