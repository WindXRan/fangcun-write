# 第36章 风格速查

## 定量锚点

| 指标 | 源文值 | 仿写范围 |
|------|--------|----------|
| 总字数 | 2023字 | 1820~2225字 |
| 平均句长 | 21.2字 | ±20% |
| 句长标准差 | 13.0 | - |
| 平均段长 | 24字 | - |
| 词汇多样性(TTR) | 0.31 | - |
| 比喻句 | 1句 | ±2 |
| AI路标词 | 4次 | **必须0** |
| 省略号 | 7个 | 源文7个 → 仿写最多8个 |

## 修辞特征

短句节奏(5处)

## 句首高频模式

蒲雨(9次), 原溯(8次), “这(4次)

## 执行规则

### 句式
- 平均句长21.2字，标准差13.0
- 段落长度6~67字
- 段落结尾偏好：句号

### 对话
- 对话标签：源文用「XX说」→ 仿写可用动作/神态替代

### 描写
- 情绪表达：源文无直抒情 → 仿写禁止（感到/心中涌起/不由得/不禁）

### 防AI检测
- **禁止**：首先/其次/然后/最后/与此同时/此外/值得注意的是
- **禁止**：对仗句式（他不……也不……更不……）

### 字数控制
- 目标：2023字，范围：1820~2225字
