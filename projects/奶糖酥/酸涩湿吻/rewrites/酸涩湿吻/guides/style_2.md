# 第2章 风格速查

## 定量锚点

| 指标 | 源文值 | 仿写范围 |
|------|--------|----------|
| 总字数 | 1818字 | 1636~1999字 |
| 平均句长 | 20.1字 | ±20% |
| 句长标准差 | 13.4 | - |
| 平均段长 | 24字 | - |
| 词汇多样性(TTR) | 0.352 | - |
| 比喻句 | 10句 | ±2 |
| AI路标词 | 2次 | **必须0** |
| 省略号 | 5个 | 源文5个 → 仿写最多6个 |

## 修辞特征

比喻(像/如/仿佛)(4处), 短句节奏(4处)

## 句首高频模式

蒲雨(10次), 原溯(5次)

## 执行规则

### 句式
- 平均句长20.1字，标准差13.4
- 段落长度6~63字
- 段落结尾偏好：句号

### 对话
- 对话标签：源文无「XX说」→ 仿写禁止，只用动作/神态

### 描写
- 情绪表达：源文无直抒情 → 仿写禁止（感到/心中涌起/不由得/不禁）

### 防AI检测
- **禁止**：首先/其次/然后/最后/与此同时/此外/值得注意的是
- **禁止**：对仗句式（他不……也不……更不……）

### 字数控制
- 目标：1818字，范围：1636~1999字
