# 第21章 风格速查

## 定量锚点

| 指标 | 源文值 | 仿写范围 |
|------|--------|----------|
| 总字数 | 2504字 | 2253~2754字 |
| 平均句长 | 21.3字 | ±20% |
| 句长标准差 | 15.2 | - |
| 平均段长 | 26字 | - |
| 词汇多样性(TTR) | 0.26 | - |
| 比喻句 | 2句 | ±2 |
| AI路标词 | 5次 | **必须0** |
| 省略号 | 6个 | 源文6个 → 仿写最多7个 |

## 修辞特征

短句节奏(8处)

## 句首高频模式

程司(12次), 蒲雨(11次), 原溯(8次)

## 执行规则

### 句式
- 平均句长21.3字，标准差15.2
- 段落长度6~73字
- 段落结尾偏好：句号

### 对话
- 对话标签：源文用「XX说」→ 仿写可用动作/神态替代

### 描写
- 情绪表达：源文有直抒情 → 仿写可少量用（≤2次）

### 防AI检测
- **禁止**：首先/其次/然后/最后/与此同时/此外/值得注意的是
- **禁止**：对仗句式（他不……也不……更不……）

### 字数控制
- 目标：2504字，范围：2253~2754字
