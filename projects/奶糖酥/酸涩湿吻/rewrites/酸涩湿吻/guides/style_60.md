# 第60章 风格速查

## 定量锚点

| 指标 | 源文值 | 仿写范围 |
|------|--------|----------|
| 总字数 | 2050字 | 1845~2255字 |
| 平均句长 | 16.3字 | ±20% |
| 句长标准差 | 11.0 | - |
| 平均段长 | 24字 | - |
| 词汇多样性(TTR) | 0.315 | - |
| 比喻句 | 12句 | ±2 |
| AI路标词 | 2次 | **必须0** |
| 省略号 | 6个 | 源文6个 → 仿写最多7个 |

## 修辞特征

比喻(像/如/仿佛)(6处), 短句节奏(13处)

## 句首高频模式

原溯(10次), 蒲雨(8次), “你(6次)

## 执行规则

### 句式
- 平均句长16.3字，标准差11.0
- 段落长度6~69字
- 段落结尾偏好：句号

### 对话
- 对话标签：源文无「XX说」→ 仿写禁止，只用动作/神态

### 描写
- 情绪表达：源文有直抒情 → 仿写可少量用（≤2次）

### 防AI检测
- **禁止**：首先/其次/然后/最后/与此同时/此外/值得注意的是
- **禁止**：对仗句式（他不……也不……更不……）

### 字数控制
- 目标：2050字，范围：1845~2255字
