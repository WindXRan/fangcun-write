|------|--------|----------|
| 总字数 | 2092字 | 1882~2301字 |
| 平均句长 | 16.5字 | ±20% |

| 平均段长 | 19字 | - |
| 词汇多样性(TTR) | 0.304 | - |

| AI路标词 | 1次 | **必须0** |

## 修辞特征

## 句首高频模式

蒲雨(15次), 原溯(10次), 许岁(7次)

## 执行规则

### 句式
- 平均句长16.5字，标准差9.8
- 段落长度6~61字
- 段落结尾偏好：句号

### 对话
- 对话标签：源文无「XX说」→ 仿写禁止，只用动作/神态

### 描写
- 情绪表达：源文无直抒情 → 仿写禁止（感到/心中涌起/不由得/不禁）

### 防AI检测
- **禁止**：首先/其次/然后/最后/与此同时/此外/值得注意的是
- **禁止**：对仗句式（他不……也不……更不……）

### 字数控制
- 目标：2092字，范围：1882~2301字
