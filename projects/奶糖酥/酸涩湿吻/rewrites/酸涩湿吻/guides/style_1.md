# 第1章 风格速查

## 定量锚点

| 指标 | 源文值 | 仿写范围 |
|------|--------|----------|
| 总字数 | 2202字 | 1981~2422字 |
| 平均句长 | 19.6字 | ±20% |
| 句长标准差 | 12.8 | - |
| 平均段长 | 26字 | - |
| 词汇多样性(TTR) | 0.35 | - |
| 比喻句 | 7句 | ±2 |
| AI路标词 | 1次 | **必须0** |
| 省略号 | 5个 | 源文5个 → 仿写最多6个 |

## 修辞特征

比喻(像/如/仿佛)(3处), 短句节奏(7处)

## 句首高频模式

蒲雨(11次), “我(4次), ”蒲(3次)

## 执行规则

### 句式
- 平均句长19.6字，标准差12.8
- 段落长度6~108字
- 段落结尾偏好：句号

### 对话
- 对话标签：源文无「XX说」→ 仿写禁止，只用动作/神态

### 描写
- 情绪表达：源文无直抒情 → 仿写禁止（感到/心中涌起/不由得/不禁）

### 防AI检测
- **禁止**：首先/其次/然后/最后/与此同时/此外/值得注意的是
- **禁止**：对仗句式（他不……也不……更不……）

### 字数控制
- 目标：2202字，范围：1981~2422字
