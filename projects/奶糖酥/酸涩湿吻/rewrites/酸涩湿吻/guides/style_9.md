# 第9章 风格速查

## 定量锚点

| 指标 | 源文值 | 仿写范围 |
|------|--------|----------|
| 总字数 | 2103字 | 1892~2313字 |
| 平均句长 | 20.5字 | ±20% |
| 句长标准差 | 13.1 | - |
| 平均段长 | 24字 | - |
| 词汇多样性(TTR) | 0.29 | - |
| 比喻句 | 7句 | ±2 |
| AI路标词 | 2次 | **必须0** |
| 省略号 | 4个 | 源文4个 → 仿写最多5个 |

## 修辞特征

比喻(像/如/仿佛)(4处), 短句节奏(8处)

## 句首高频模式

蒲雨(12次), 许岁(8次), ”许(5次)

## 执行规则

### 句式
- 平均句长20.5字，标准差13.1
- 段落长度6~78字
- 段落结尾偏好：句号

### 对话
- 对话标签：源文用「XX说」→ 仿写可用动作/神态替代

### 描写
- 情绪表达：源文有直抒情 → 仿写可少量用（≤2次）

### 防AI检测
- **禁止**：首先/其次/然后/最后/与此同时/此外/值得注意的是
- **禁止**：对仗句式（他不……也不……更不……）

### 字数控制
- 目标：2103字，范围：1892~2313字
