# 修复计划

## 目标
一次性修复所有致命问题和严重问题，避免越修越差。

## 问题分类

### 致命问题（5个）
1. **债务金额不一致** — 统一为"四百万"
2. **"系统→预知梦"能力转变缺乏过渡** — 在第5-8章插入系统"半苏醒"场景
3. **倒计时系统混乱** — 统一为"每天减1天"，补全缺失天数
4. **裴行舟"科研天才"人设弱化** — 加入科研能力场景
5. **"死亡倒计时"核心设定被遗忘** — 每5章出现一次预知验证

### 严重问题（8个）
1. **裴行舟"生活白痴"到"全能男友"转变过快** — 增加学习过程
2. **宋知渺"骄纵"特质消失** — 每3-4章加入一次作精表现
3. **悬疑线推进节奏失衡** — 甜宠章节穿插悬疑暗线
4. **周亦辰角色定位模糊** — 增加伏笔和行动线
5. **裴行舟"身份证"线索悬空** — 解释或删除
6. **裴行舟"被追踪"状态不一致** — 统一警惕程度
7. **配角工具人化严重** — 激活配角
8. **裴行舟"报复"动机悬念消耗过快** — 推迟坦白

## 修复策略

### 原则
1. **只修不删** — 不删除原有内容，只增加或修改
2. **保持情感线** — 情感线是亮点，不要破坏
3. **最小改动** — 只修改必要的部分，不要过度修改
4. **统一设定** — 所有修改必须符合统一设定

### 执行方式
1. 派子agent修复致命问题
2. 派子agent修复严重问题
3. 重新导出完整版
4. 重新审稿验证
