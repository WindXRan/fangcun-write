# 一、核心总论

## 1.1 期待感——网文的根本

**核心论断：网文的一切元素，都是为期待感服务的。**

**什么是期待感？**
读者内心深处有一个东西在吊着他——想知道主角能不能打赢、误会何时解开、身份何时暴露。这一个又一个"想知道"就是期待感。

**糖果路比喻：**
- 期待感 = 糖果的香气（勾引读者往前走）
- 爽点 = 吃到的糖果（满足期待）
- 追读 = 走一段路，吃一颗糖的循环
- 心理链条：期待 → 追读 → 满足 → 再期待 → 再追读 → 再满足

**读者离开的三种原因：**
1. **毒点**：吃到的糖果是苦的（期待主角打脸，结果被羞辱）
2. **期待感弱**：糖果气味太淡（情节平淡如水）
3. **未能满足期待**：草莓香但吃到薄荷糖（期待A得到B）

**核心公式：**
```
制造期待（抛钩） → 满足期待（收钩）
```
整本书就是无数个这样的循环单元——大的循环套小的循环，远的循环交织近的循环。

**钩子与收钩必须一一对应：** 抛出钩子A，读者产生期待A，后面必须用收钩A满足它。期待错位会导致读者弃书。

**期待感的分类：**
| 维度 | 类型 | 示例 |
|------|------|------|
| 时间距离 | 近期待 vs 远期待 | 打脸小反派(近) vs 三年之约(远) |
| 重要程度 | 小期待 vs 大期待 | 恢复天赋(大) vs 打败路人(小) |

**期待感的递减法则：**
- 远期待随时间淡去 → 需反复提醒、撩拨
- 近期待要"现杀现宰" → 快速强化后满足
- 同类型重复 → 审美疲劳 → 需引入支线更替类型

**大期待结束时的处理：**
卷末高潮不要一次性满足所有期待。做反转或留悬念，把一个"完全收尾"变成"阶段性高潮+新悬念的起点"。

**超预期满足：**
在满足基本期待的基础上，以意料之外、情理之中的方式超出预期——爽感翻倍。

**从核心期待过渡到代入期待——写长写稳的关键：**
开头靠核心期待吸引读者（代入感为零），中后期核心期待权重下降至五六成，剩下四五成靠代入感填补。如果核心期待衰退前没建立起坚实的代入感，读者流失不可避免。

**增强代入感六大技法：**
1. 角色行动逻辑合理
2. 动机与经历统一
3. 用具体事例彰显性格
4. 展现内心矛盾与自我抗争
5. 不同角色间要有张力
6. 角色性格生活化（小习惯、小癖好拉近距离）

**情绪不能脱离期待感独立存在：** 单纯的哭和笑读者内心毫无波澜。情绪低落 = 抛钩（勾起解决困难的期待），情绪上扬 = 收钩（解决上文抛出的期待）。主角是情绪锚点，所有情绪通过主角感受。

**期待感要围绕核心设计：**
所有期待感都要从核心卖点衍生出来。与核心无关的期待感会打乱读者预期。

## 1.2 剧情循环——让故事永不枯竭

**核心定义：**
剧情循环是一种稳定的、可重复的模式，让你能持续不断地产生剧情。它是一台"永动机"。

**核心公式（13字）：**
```
题材 + 金手指 + 主角身份 = 剧情循环模式
```

**三要素详解：**

**①题材决定路线：**
- 王朝探案 → 一个又一个的案件
- 玄幻升级 → 一层又一层的境界
- 都市生活 → 一个又一个的事件和人际关系
- 巫师学院 → 一阶又一阶的知识

**②金手指是核心工具：**
- 熟练度/加点 → 升级变强路线
- 信息差（重生知道未来）→ 探案/经商/宫斗路线
- 系统商城 → 资源积累路线

**③主角身份决定出发点：**
- 采药少年 → 底层生存+变强
- 世家少爷 → 家族争斗+博弈
- 穿越前警察 → 天然与"查案"绑定

**经典案例拆解：**

| 类型 | 代表作 | 核心循环 |
|------|--------|----------|
| 探案类 | 《大奉打更人》 | 案件→解谜→真相→更大谜团→新案件 |
| 升级类·扮猪吃虎 | 滚开系列 | 默默发育→升级→敌人不知→碾压→震惊→继续发育 |
| 升级类·资源积累 | 传统玄幻 | 资源→技能→实力→新资源（螺旋上升） |
| 都市反转类 | 《亏成首富》 | 亏钱→反转赚更多→拿更多去亏→反转 |
| 诡秘/克苏鲁 | 《诡秘之主》 | 成员冒险→塔罗会汇聚→衍生新剧情→执行→再汇聚 |
| 公路片 | 《剑来》《雪中》 | 出门→遇人→发生故事→继续前行→再遇人 |

**新手村必须包含四种势力（资源积累循环核心）：**
1. **学校/武馆**：学技能、提实力
2. **地主/豪绅/药行**：卖收获、得资源
3. **山贼/马匪**：展示成果的靶子（让读者看到"变强了"的效果）
4. **官府**：维持秩序和上升通道

四种势力形成完整闭环：采药 → 卖钱 → 学技能 → 打山贼 → 去更危险的地方采更好的药。

**开头最重要的功能——引导：** 像游戏新手村一样，尽快向读者展示本书的基本循环规则。让读者知道：主角是干什么的、靠什么变强、将通过什么方式一步步往上走。

**换地图掉追读的优化方法：**
- 新旧地图间有更紧密联系（新地图势力是旧地图的上级）
- 把旧地图重要人物带过去
- 提前铺垫新地图吸引力，让读者主动盼着去而非被动被赶
- 用更大利益驱动（如学院遗迹进修名额），不要用被追杀的老套路

**地位提升与环境破坏必须同步：** 学生→面对考试竞争；实验室助手→面对实验事故和同行嫉妒；教授弟子→面对学术争端和真正危险。地位每升一层，环境"破坏"就提高一个量级——这就是循环的节奏感。

**天才流解法：** 正常玄幻10个境界不够用 → 设50个境界，让他不断突破。**无敌流解法：** 靠信息差而非变强，永远不让其他角色知道主角的上限。

## 1.3 剧情设计——起承转合与设计感

**精彩故事具备起、承、转、合。** 网文能写好起承就足够，但设计感是区分平庸与优秀的关键。

**设计感三要素：**
1. **都市/轻小说背景**（强代入感）
2. **开篇反转**（有包袱、有意外的揭示）
3. **主角与众不同**（鲜明的记忆点、身体特征、前情故事）

**设计感的方法论：**
- **服化道**：偏好某颜色、独特配饰（如白衣+红花 = 长生者的孤独）
- **身体特征**：有故事性（如永远戴帽子的管家 → 秃头+戒疤+血莲花 = 前绿林龙头）
- **角色反转**：通过"奇情"式解法（荒诞不经又能自圆其说的转折）

**创作的金句：**
> "网文、故事，它的上限藏在细节里。"
> "你的人物、你的剧情没有设计感，就打动不了读者。"
> "你写这个人，这个人为什么是这个人——你要在脑海里过一遍。"

# 二、线纲体系（上）：主线、副线、旁线、支线

> 来源：网文写作理论课 1.0-1.1

## 2.1 线纲的概念

线纲是写作之前最重要的准备。**线**分主线、副线、辅线、旁线、支线；**纲**分总纲、大纲、详纲、细纲。好线纲完成，整本书完成一半。

## 2.2 线的五级体系

### 2.2.1 主线——全书核心

主线分**明线**和**暗线**。明线：玄幻中境界线最完美（14年沉淀验证）。暗线候选：心态线、技能线、装备线、身世线、地图线、复仇线、伙伴线、感情线、势力线。**暗线不宜超过两条**，一明一暗最佳。

### 2.2.2 明线：境界线设计法

境界线 = 横轴时间、纵轴境界、自由弯曲的斜线。
- 变陡→奇遇/消化奇遇 | 变缓→忙于其他/逃亡 | 变平→重伤/沉睡
- 下降→散功重修 | 高频起伏→走火入魔 | 巨幅涨落→天才变废柴/九死一生
- 起始点非零→重生流/旁观流

**线画得越波折复杂，情节越丰富越吸引读者。**

### 2.2.3 暗线

暗线串联所有情节点——大到战争小到冲突，都带来暗线变化。

### 2.2.4 副线与辅线

**关键法则**：主线越多越难驾驭，副线越多越轻松。一类线可同时是主线/副线/辅线的三组：细微变化=主线，关键变化=副线，重大转折=辅线。

### 2.2.5 旁线——时间线的魔力

三种旁线：**时间线**（最核心）、背景线、他人主线。

**时间线写作法**：罗列按时间发展的一切事件→细化→筛选要描写的剧情。一个简单行为可引发数十种剧情，**五分钟构思几十个情节，灵感永不枯竭。** vs 阅历写作法：写完就江郎才尽。

### 2.2.6 支线

四种支线：下副本（≤2个）、参活动（需完整铺垫，订阅主力）、长路行（辅助不限量）、兵刃见（核心高潮，冲突间平淡过渡，大高潮层层递进）。

## 2.3 线纲底层逻辑

**线纲 = 除法思维**：基础难度10000 → 层层线纲÷2÷2÷2 → 初文只剩100 → 阅历-100 → 完成。

设计好主线副线辅线，剧情自然设计好——对着线匹配原理、安排剧情。**卡文？不存在的。**

# 三、线纲体系（下）：纲与文

> 来源：网文写作理论课 1.2

## 3.1 纲的四级体系

- **总纲**：一句话几十字概括全书（时间地点人物起因经过结果+中心）
- **大纲**：辅线的所有转折点排列 = 大纲
- **详纲**：副线+旁线融合，每部分分几到十几个段
- **细纲**：主线+时间线填充，落实到每一情节

推荐工具：MindMaster思维导图。**从大纲→详纲→细纲层层细化，第一部分"学院篇"够写一百多万字。**

## 3.2 文的三步法

**初文→成文→精文**

**初文**：一气呵成，把脑子里想法全写出来。不完美没事，跳过没想法处，连着写十章八章。

**成文**：另开文档，照着初文重写。确立框架，安排情节点，添加过渡。此时才用阅历经验。

**精文**：逐字逐句修改用词。注意错别字（超万分之五很影响）、生僻字（常出现的不超两三个）。但修改幅度不宜过大——你已无法像刚看文的读者那样判断。

# 四、世界观构建：背景世界、人文世界、历史科技

> 来源：网文写作理论课 2.0-2.2

## 4.1 设定的重要性

**设定 = 地基，线 = 钢筋，纲 = 水泥，文 = 装修。** 大神开新书前构建设定：两到三个月。扑街与职业作家的区别之一，就是对设定的重视。

## 4.2 背景世界观

### 地界概况
层次划分+地图绘制。解决"城/国/大陆/世界"之间的过渡空白。可选国度制或宗门制（当前趋势）。

### 奇地险地
提前设计十多个，写作取用。**预先设计的自然；临时设计的"死"——优秀小说的条件：没写的比写出来的多。**

## 4.3 人文世界观

### 修炼方式
两大方向：吸纳外界灵气 / 开辟体内潜能。**修炼方式决定境界线的走势与波折种类。** 全新修炼设定是极大亮点。

### 境界命名四体系
华夏传统（天地玄黄/天干地支）、道教神话（练气→金丹）、尊称地位（x者→x神）、自创词汇（按修炼方式分阶段概括，目前最常见）。

### 元素属性
元素种类可超千种。**关键是元素应用**：风不适合风刃——风的特性是大、多、广，适合辅助大范围攻击。每种元素有独特优势。预先设定松开想象力束缚。

### 奇门职业
丹器符术阵 × 械灵魂。丹师链可开发127种职业（不只是炼丹师，指导服药的人更出名赚钱）；器师链可开发366种。**主角前期学的技能中后期不该废弃——融合为四五个核心技能。**

### 世界历史与天灾天劫
天灾天劫 = 元素 × 表现（多/少/暴）。土少=位面破碎，土多=位面融合——可作为中期大高潮背景。

**心魔劫正确设计**：不是一次性幻像，而是"强化欲望+淡化危害"，逐渐蚕食修为的过程，表现在生活方方面面，修士至死意识不到。

### 科技产物
空间戒指远超"起步"阶段——内部空间外是什么？稳定吗？时间同步吗？有空气吗？设计好就是亮点。

# 六、人物设定：创建、解析、应用、人际关系

> 来源：网文写作理论课 2.3-2.5

## 6.1 人物库设计法

与传统"设计主角配角龙套反派"不同：**为每一个角色都把他当主角来设计。**

方法：根据世界观每一个设定点创建一个角色。如"风属性"→设计三段人生经历框架→一个立体角色诞生（5-10分钟一个）。写两三百个放入人物库，正文直接取用。

**片段人物**：只有一个情节的灵感→放入片段人物库→三天后潜意识发酵→完善成完整人物。

## 6.2 人物解析

性格、装备技能、人际关系三者解析完，人物有血有肉。

**装备技能解析**：从设定库（武器库/建筑库/科技产品库）挑选匹配的，而非硬生生创造。先构建设定库再挑选 = 永不会江郎才尽。

## 6.3 人际关系

**亲、友、情、敌、仇五种关系。人际关系是区别"作家"与"写手"的核心。**

### 亲人
最难写，因为读者对亲缘逻辑极度敏感→导致大半主角是孤儿。但这只是逃避。**直面亲人，先积累"库"（如"弟"的200种状态），然后排列组合。四人可产生一亿多种情节。**

### 友人
核心原理：双方都能从对方身上得到价值（金钱价值+心理价值）。得到的价值远超失去的→关系牢固；得到≈失去→争吵；得到<失去→决裂。

### 情人
感情内容一旦写就好好写，写崩则整书崩。最关键的是**逻辑详略**——读者对感情极度敏感，不能有丝毫小说逻辑错误。绿帽不是禁忌，是需极高驾驭力。

### 敌人与仇人
争夺利益=敌人（可变盟友），侵犯利益=仇人（基本永远）。设计boss级反派时，像主角一样详细刻画他的人际关系。

## 6.4 人物应用

**分配人物地位**：主角唯一，配角可多。重要配角形象完整度不输主角。注：龙套不能加戏——出场三句话的龙套别写得多感动。

**设计经历**：依据性格+目的+需求+环境→产生应对行为→自然出剧情。**源于生活高于生活，前四个字同样重要。**

**反派设计革新**：不要全写悲惨反派——一个为了作恶而作恶的大魔头，比一百个模式化苦情反派都强。

# 七、开篇与签约

> 来源：新人写作资料 + 指南教程8/10

## 7.1 签约铁律（20条）

**前3万字的代入感决定成绩。**

签约模板（前10章节奏）：
- 第1章：简单介绍主角背景
- 第2章：矛盾加剧
- 第3章：出现外挂，重点介绍但留钩子
- 第4章：全文第一个爽点（装逼踩人）
- 第5章：继续装逼或低调隐藏，引出主线
- 第6章：都市出女主
- 第7章：主线剧情，结识新人脉
- 第8章：高级敌人出现
- 第9章：冲突加剧
- 第10章：主角再次爆发装逼

**20条铁律速查：**
1. 开局300字内必须出现主角
2. 第一章3000字必须爆发第一个矛盾冲突
3. 3章之内亮出功法体系或最大卖点
4. 30000字内爆发第一个高潮
5. 解决矛盾后展开3条支线
6. 全书拉3段大情节，第一个30万字前后解决
7. 每阶段至少3个不同成长要素
8. 每次高潮后主角得3件战利品
9. 主角异性至少3个
10. 七分铺垫三分打斗
11. 类似桥段全书不超过3次
12. 每章不超过3个场景
13. 每章对话3成+情节叙述3成+描写3成
14. 旁白/内涵每章不超过300字
15. 无关内容每章不超过300字
16. 严禁同时使用3种以上视角
17. 严禁主角打酱油超过3000字
18. 每章主要人物不超过3个
19. 严禁连续3章通篇打斗

## 7.2 读者流量漏斗理论

每个作者读者有上限→层层漏斗过滤：开篇漏斗、更新漏斗、上架漏斗、雷人情节漏斗……**开篇是最重要的漏斗，开篇越吸引人，后续读者流量越大。**

## 7.3 开篇六条铁律

1. **聚焦主角**：镜头一直跟主角，配角越少越好（最好≤3人），从日常小事入手
2. **开篇明志、确定主线**：一开始定下类型、背景、人设和故事动力
3. **简洁明快**：不要大篇幅介绍设定背景，不要模仿金庸延迟出场
4. **尽快挑起悬念**：迅速有悬念/金手指，让读者好奇接下来发生什么
5. **不要犯常识性错误**：错别字、语法、标点
6. **取好书名，写好简介**

## 7.4 冲突万能公式

```
渴望 + 阻力 = 冲突
```
主角想达成一个目标，但要克服很多阻力。

# 八、情节驾驭

> 来源：新人写作资料 + 指南教程 + 理论课 3.2-3.4

## 8.1 情节的本质

**情节 = 矛盾。高潮 = 解决矛盾。** 情节是按因果逻辑组织的事件，表现为人和人、人和环境的矛盾冲突。

**五步构建法**：找到结果（高潮）→ 设置开头（矛盾诞生）→ 分三条线路 → 每条线路分三个组成 → 设置关键点穿插

## 8.2 三线并行法（以《斗破》为例）

- **药老线**：伙伴+功法+技能（金手指相关）
- **家族线**：伙伴+后宫+势力（人际关系）
- **云岚宗线**：后宫+矛盾+BOSS（主线对抗）

每条线独立发展，三线交织产生化学反应。

## 8.3 对比——情节吸引力的底层逻辑

**对比与反差是商业化写作的核心引擎。** 对比越强，吸引力越大。

**武林高手描写公式**：不空泛写他强，先设定参照对手（已建立强印象的角色），通过参照对手的反应（崇敬/恐惧/震惊）来写出主角更强。

**打斗爽感对比链条**：打前刻画对手强大→对手轻蔑/旁观者不看好→此战重要性→打完后对手震惊/旁观者崇拜→对比越强越爽。

## 8.4 七步情节设定法

1. 确定世界观与背景
2. 确定主角目标
3. 确定达成目标的阻碍
4. 如何克服阻碍
5. 克服阻碍的结果（可正面可负面，负面引更多情节）
6. 对结果的反应
7. 循环产生新目标

**本质：目标-阻碍-行动-结果-反应-新目标的无限循环链。**

## 8.5 三大驱动力

```
三大驱动力 = 戏剧性 + 矛盾冲突 + 欲望与恐惧
```

**戏剧性**：巧合、误会、反转、秘密。公式：戏剧性→读者期待发酵→看到影响。
**矛盾冲突**（最根本）：人与人/人与环境/人与自身。
**欲望与恐惧**：欲望驱人前进，恐惧让人纠结。

## 8.6 剧情结构

**情节→剧情→阶段大剧情→全书大剧情**

| 层级 | 承载 | 篇幅 |
|------|------|------|
| 情节 | 情绪 | ~5章/1万字 |
| 剧情 | 气氛 | 五六情节/5-10万字 |
| 阶段大剧情 | 基调 | 50万字左右 |
| 全书大剧情 | 风格 | 500万字 |

**铺垫核心：把中心情节拆解成台阶，逐级铺开。越靠近中心情节的台阶越陡（描写越细）。** 土豆的铺垫堪称完美，番茄也是——扑街作者小看了他们。

## 8.7 故事卡系统

**事业线关键点套装**：阻碍→危机→装逼/打脸→收获→引出新矛盾

**事业线11种节点**：比试、任务/副本、拍卖会、奇遇、学院、势力建设、冒险之旅、战争、争夺战、揭秘、收服

**爱情线节点**：相遇→冲突→和解→暧昧→高潮（确认关系）→危机→守护

**CP行为（"发糖"）**：需容忍行为、特殊对待（主权/信赖/牺牲/特殊待遇/安抚）、关联行为（默契/分享/陪伴）

# 九、爽点体系

## 9.1 装逼与打脸

**打脸九大构件（完整流程）**：背景→导火索→立靶（确立反派）→前戏（积蓄期待）→看戏（围观者反应）→洗脸（开始打脸）→送脸（四种类型）→打脸（高潮）→踩脸（追加羞辱）

**三种打脸套路**：
| 套路 | 模式 | 代表 |
|------|------|------|
| AB式线性法 | "在座各位都是垃圾" | 李七夜 |
| ABC式三角法 | "五帝之下皆为蝼蚁" | 陈北玄 |
| ACB式代装逼法 | "来个捧哏代我装逼" | 陈北玄 |

**金句**："打脸的关键不在于如何打脸，而在于打脸前后的铺垫、渲染！"

## 9.2 三大心理快感

畅快感（随心所欲）、成就感（实现价值）、优越感（比别人强）

## 9.3 经典桥段与爽点精选

- 轰杀反派（越卑鄙越爽）、破坏规则/挑战权威、削装B犯/傲娇女（从最引以为豪处下手）
- 扮猪吃虎、变废为宝、化不可能为可能
- 释放绝技后淡淡一句"回家吃饭"
- 每次只赢别人一点点，永远追不上
- 别人求不到的主角硬塞都不想要
- **55条爽文桥段速查**：身份反转/宝物逆袭/情感义气/碾压展示/隐藏实力/关系互动/智慧型——覆盖都市/玄幻/仙侠全品类

# 十、节奏与情绪

## 10.1 节奏 = 矛盾和解决矛盾的冲突进程

**快节奏秘诀**：与矛盾无关内容一笔带过，展现能力和收获奖励重点写。围绕矛盾、找主角麻烦。

## 10.2 情绪 = 价值变化

**四种价值**：实物价值（金钱/功法）、非实物价值（亲情/友情/安全/尊严）。**情绪由"价值变化"产生**——没有价值变化就是平淡无趣的日常。

## 10.3 逻辑真实 vs 情绪真实

**核心论断：情绪真实远比逻辑真实重要。** 在二者并存时，毫不犹豫牺牲逻辑，追求更强烈的情绪。

## 10.4 水文 = 反应

人类天然本能：做任何事都渴望得到反馈。**唯一可以水且最出爽点的地方：反应。**

三种手法：反差（百看不厌）、化学反应（金手指与剧情匹配）、连锁反应（扩大事件影响力）。

## 10.5 节奏诊断与调整

**节奏太慢→放水**：直接删掉不重要场景 / 合并场景 / 用叙述代替描写
**节奏太快→注水**：穿插日常场景 / 一句话能说清的用一段话

男频：阶梯式上升+缓冲注水。女频：心电图的循环起伏。

# 十一、人物塑造

## 11.1 人物构成要素

```
人物 = 外观形象 + 行为模式 + 社会身份 + 资源禀赋
```

## 11.2 三维公式

```
标签化 + 真实感 + 深度 = 丰满人物
```

- **标签化**（辨识度）：网文中标签是好事——让读者快速识别记忆
- **追求真实感**：追溯成因，撰写前史。核心前史放第一卷后期揭示，一段回忆最多500字
- **追求深度**：有深度的特征思想+价值取向

## 11.3 增加人设戏剧性

- 去常态化的个人特质
- 具有反差（场景/身份/刻板印象三重反差）
- 自带矛盾的人物（晕血的医生、生在黑帮想做厨师的少爷）

## 11.4 为角色添加冲突

**冲突 = 渴望 + 阻碍**。如：小明渴望认识更多女性朋友 + 极度内向/社交恐惧 = 戏剧性反差。

设定好冲突后多问几个"为什么"（倒推性格成因），多做几个"假如"（测试角色反应→产生情节）。

## 11.5 角色层级分类

叙事主角（核心）→ 可攻略对象（感情线）→ 主要配角（重要推动）→ 主要反派（对抗核心）

特殊功能角色：引导者、谐星、变形者（立场不明/反复横跳）

## 11.6 外貌描写

**反直觉观点：外貌描写并没有那么重要。** 大多数读者记不住主角长相。

**方法**：抓住一个典型特征（关羽髯长二尺面如重枣、小龙女白衣轻纱），其余全"藏"起来。回忆生活中让你多看几眼的人，只写那一个吸引你的特征。

## 11.7 对话描写

**四种对话形式**：提示语在前/在中间/在后/无提示语。四种穿插=摇曳生姿，纯用第一种=偷懒小白文。

**潜文本概念**：把对白的真实意思藏起来。不说"我爱你别离开"，而是通过日常生活对话暗示矛盾——这才是人话。

**小说对话 ≠ 现实对话**：必须干净简练，只给读者有用信息。

## 11.8 心理描写四法

1. 直接描写（"想""寻思"等心理动词，注意换近义词）
2. 内心独白（自己对自己说话）
3. **多角度呈现**（从神态/外貌/动作/语言间接揭示——不用"想"字但心理完全外化）
4. 环境衬托（乐景写乐情/乐景写哀情）

# 十二、场景与分镜

## 12.1 场景布局

**硬性指标：每个场景必须承载一个期待点或爽点。** 否则是废场景。

三要素（缺一不可）：人物 + 情节（起因经过结果）+ 环境（时间地点）

技术规格：单场景300-500字，2000字一章1-5个场景（3个最舒服）。禁忌：不要蒙太奇、不要倒叙、尽量不插叙。

## 12.2 分镜画面感

**核心原则：小说是留白的艺术——只写最关键最特殊的点，交给读者脑补。**

5种生动技法：色彩、光影/明暗、细节动作、五感、比喻（不常见有新鲜感的）

**分镜衔接**：上下分镜必须有同一个人事物作为桥梁，且有衔接动作。

## 12.3 行文公式

```
行文 = 叙事 + 旁白
叙事 = (场景+分镜) + 对白 + 心理活动
```

原则：能用叙事就用叙事，少用旁白。人脑更喜具体"经验"而非抽象"指令"。

## 12.4 打斗场景五步法

1. **布局铺垫**：环境描写+万众瞩目+观战者议论+嘴炮
2. **正式开打前**：描写双方语言神态心理
3. **正式开打**：动作+语言+神态+心理+五感（视觉/听觉/触觉优先）
4. **打斗结束**：双方及亲友团五感描写+造成破坏
5. **配角反应清单**：吸凉气、咽口水、呆住、不可能不相信、喜极而泣……

# 十三、正文笔力：流畅度、画面感、立体感

> 来源：网文写作理论课 4.0-4.3

## 13.1 笔力的四个维度

笔力 = 文字流畅度 + 描述清晰度 + 文字画面感 + 场景立体化

写书首要做到让读者理解你写的是什么——读者不理解，情节设计再好也白搭。

## 13.2 文字流畅度

### 标点与断句
**大脑同时处理5-6个字。** 逗号间隔最佳长度5-10字。长句化短句不是简单加逗号——需要重新调整语句结构。

### 关联词使用
读者阅读有预判与验证机制。**关联词让读者提前知道后文方向**——"不过"=转折，"而且"=并列。职业作者普遍能用好关联词，扑街基本不会用。

### 病句与重复
**杜绝病句。** 最常见：主语错乱。十本未签约作品八九个都是病句连篇。

### 段落与布局
**一段 = 一个事件。** 四类基础事件：背景旁白、客观规律、与人互动（主体）、自我感触。每章 = 一个情节，章末必须有总结性结果。

### 前因与后果
一切事件必须有完整前因后果链条。检查方法：提取事件概述→依次检查递推关系→发现跳跃→补充。

## 13.3 文字画面感

### 词汇运用三要素
1. **用读者熟悉的词汇**——不用"侘傺"用"落魄"，不用"外焰"用"火焰"
2. **抓住重点详细描写**——略写部分一段文字讲述，详写部分不超三段
3. **自创词汇符合逻辑**——命名展示特点而非威势，"幽蓝鬼火"vs"玄冥妖火"，前者画面感远超后者

### "焚天烈火"的命名错误
命名应抓住客观可观察的特点（颜色、位置、变化、成因），而不是夸大虚无缥缈的威势。"幽蓝鬼火"抓住了三个特点：幽（暗、飘忽）+蓝（蓝色）+鬼（飘忽、阴坟）。"焚天烈火"只描述了性质，没特点。

## 13.4 环境立体感

**完整 vs 全面**：完整是广（不遗漏）、全面是细（所有细节）。小说需要完整而非全面。

**介绍清晰的三个技法**：
1. **内容完整**：设计第一梯队所有人物/建筑，一个不能少
2. **顺序合理+先略后详**：先总体简述再在具体剧情中展开
3. **每次只达到读者理解极限**：让读者以为世界观已经完整，然后再展示新层次

**核心原则**：在剧情中概述介绍 → 剧情后具体讲解。不要直白灌输设定，也不要完全不给设定让读者迷失。

## 13.5 五感描写法

通过主角的视听触嗅味来写环境，不用上帝视角。视觉几乎必用，听觉触觉紧随其后。推荐BBC纪录片《人类感官》——全是干货。

# 十四、现象描述与抽象描写

> 来源：网文写作理论课 4.1-4.3 + 指南教程21

## 14.1 四种现象的描述

| 现象 | 核心技法 | 禁忌 |
|------|---------|------|
| 静态环境 | 用词符合气氛 | 空泛修饰不服务于情绪 |
| 静态人物感受 | 具体动作代替概括心情 | 直接写"他很悲伤" |
| 人与环境互动 | 通过五感写，主角视角 | 上帝视角客观描述 |
| 人与人互动 | 质问+冲突+转折 | 平淡陈述无趣味 |

## 14.2 现象描述的精髓

**不要概括，要具体。** "他很烦躁"不如"他在客厅沉着头来回踱步、越走越快、突然顿住、最终颓废地一屁股坐在沙发上"。这个世界的本质是我们只能看到动作看不到心理——心理是人们为简化概括而创造的。

## 14.3 复杂现象处理

**拆分简化**：把同时发生的多件事错开时间线→一个时间只发生一件事。

**后置介绍**（激烈变化时）：先只写中心角色发生的事，等到其他事情影响中心角色时再总结回顾。

## 14.4 抽象现象处理

**特殊内觉**（疼/晕等）：用比喻——"一把锋刃阔刀插在胸口翻搅不停"

**超越感觉**（全新感知）：同样用比喻类比——"像是赤红色却比赤红更深更暗，偏向深邃的青蓝"

**核心原则**：抽象无法被常规描述的，用比喻/找相似、进一步形容来说明。

# 十五、作品信息：书名、简介、章节名、评论区

> 来源：网文写作理论课 3.0-3.1

## 15.1 作品"颜值"

**"始于颜值，忠于才华"。** 小说颜值=书名+封面+简介。颜值好→小说成功一半。负责吸引读者，正文负责留住读者。

## 15.2 书名

三大忌讳：太长不通顺、抹杀代入感（"我的认知世界"）、不明所以。

好的书名让读者直接知道全书风格基调：《百年孤独》悲凄、《冰与火之歌》绚烂、《飘邈之旅》仙逸、《生死时速》紧张。

## 15.3 简介

**简介要具体，不要抽象。** 别写"风云起天地动"这类华而不实的。

最佳方式：截取小说中一个有趣的点（设定或情节），进行具体完整讲述，直接展示风格。喜欢的会留下，不喜欢的会离开——不要惋惜。

**第二类**：传统介绍式——先介绍背景开篇，后续留悬念（需极高功底）。

## 15.4 章节名

标准格式：**"第x章 章节名"**（汉字小写+空格+简短通顺概括性强）。

必须有章节名——让读者相信你的书不毒。必须有章节序号——读者有"知道第几章"的习惯需求。

## 15.5 评论区

**打赏的、订阅的，都在看评论区的少部分人里。** 大多数人不看评论区，但大多数人也不付费。

五条管理法则：不骂人、删无故辱骂帖、发布自己的话（置顶沟通）、置顶大额打赏、可适当营造热烈评论区。

**金句**："沟通打破猜疑链，让你不再神秘。"

## 15.6 开篇剧情的常见问题（19条速查）

1. 出场人物太多 2. 不知所云不知所喻 3. 挖坑不填 4. 铺垫不足 5. 上层脑残长辈幼稚 6. 情节间无过渡 7. 段落太长 8. 无关杂乱描写多 9. 流水账 10. 主角瞎吹牛逼 11. 设定在应用前介绍 12. 剧情拖沓 13. 句尾不加标点 14. 旁白太多 15. 强硬引笑 16. 人称视角不当

# 十六、品类策略

## 16.1 无线阅读三大特点

| 特点 | 含义 | 要求 |
|------|------|------|
| 碎片化 | 随时阅读随时中断 | 即时爽点优先 |
| 细小化 | 手机小屏幕 | 行文简练 |
| 低龄化 | 年龄学历偏低 | 小白文，不需多知识储备 |

## 16.2 都市vs玄幻

| 维度 | 都市 | 玄幻 |
|------|------|------|
| 背景 | 现实天然代入 | 凭空虚构需设置 |
| 爽点类型 | 即时爽 | 期待爽 |
| 主线 | 相对模糊 | 清晰明确（升级变强） |
| 人物 | 重刻画互动 | 重做事简单即可 |
| 金手指 | 一个点隐性为主 | 更鲜明可多个 |
| 后期优势 | 天然代入稳定 | 主线清晰长线爆发 |

**玄幻是新人坟墓：** 已是红海，大神占据头部。新人建议"往小处写"——小异能+小故事+小人物+略显新意。

## 16.3 选题原则

**三原则**：写自己最擅长最熟悉的 + 找容易入门市场大的 + 有独特之处但不要过于小众。

职业就是优势：医生写医学、程序员走技术流。你的专业 = 你的写作壁垒 = 你的逼格。

## 16.4 玄幻设定三原则

格局要宏大、名字要霸气、架构要简单。**境界：简单分级有规律有辨识度。势力：尽量简单化——一个城池三大家族/四大宗门就够。**

# 十七、运营实战：更新、读者、签约

> 来源：新人写作资料 + 指南教程

## 17.1 基础功底十大自查

错字率<千分之五 / 标点规范 / 无逻辑错误 / 段落≤400字 / 无括号补充性文字 / 无作者旁白 / 章节前后无无关名言警句 / 无不合理情节 / 文章简洁 / 无绿帽虐主等郁闷情节

## 17.2 读者心理二分法

**上班族→关键词"满足"**：上班不爽→主角狠狠蹂躏老板→读者爽。
**学生族→关键词"发泄"**：制造让人恨的角色→主角踩死他→阅读快感。

**黄金法则：可以不让读者爽，但绝对不能让人不爽！**

## 17.3 更新策略

三千一天跑不了（稳）/ 六千一天不能少（好）/ 一万一天都嫌少（红）

**存稿策略："三分力气只出两分，留一分应万变"**——写一万更六千存四千。

**更新时间必须固定**，培养读者阅读习惯。**20万字后读者看书变成习惯（如吸鸦片）。**

## 17.4 掐高潮

关键时刻→"等下一章吧"→"请加入VIP不然不给你看"。掐得好不带来压力，掐不好被骂。

## 17.5 书名笔名原则

简单、易记、顺口、有含义。避免特殊符号，笔名不超过四字。

## 17.6 对待读者

不要随便怼读者——重视大部分人的共识，少数人意见直接忽略。**冲动撕逼得不偿失。**

## 17.7 签约注意事项

保证稳定更新，不断更不"太监"。**处理好与编辑的关系**（编辑决定推荐和运营）。有事直接留言把问题描述清楚，不要只发"在吗""你好"。

# 十九、换地图技法：莲花法与千层饼法

> 来源：宅猪写作心得

**换地图是掉订阅最大的杀手。原因：割裂感。**

**方法进化**：
1. **千丝万线法**（有改善但割裂感还在）：不断穿针引线
2. **不换大地图法**（完美解决）：在一个大地图里转，飞升时完本
3. **莲花地图法**：终极地图是花芯→花瓣有很多→不断出去回来→花芯秘密不断揭开

# 二十、高级开篇：七点合一与框架设计

> 来源：原归总高级写作资料

## 20.1 飞卢开篇七点合一法

**核心原则：书名、简介、主线、切入点、金手指、框架、脑洞创意——七点高度统一。**

| 序号 | 要素 | 作用 |
|------|------|------|
| 1 | 书名 | 筛选读者，建立初步期待 |
| 2 | 简介 | 进一步筛选，书名补充 |
| 3 | 主线 | 全书主要故事线 |
| 4 | 切入点 | 切在主线上，最好第一个小高潮起始点 |
| 5 | 金手指 | 必须匹配主线 |
| 6 | 框架 | 3-8个故事框架构成前期骨架 |
| 7 | 脑洞创意 | 冲刺万订的关键元素 |

**常见错误**：七点分散不匹配 → 你写得很嗨读者不感兴趣。

## 20.2 故事框架（框架思维）

框架是3-8个大的故事高潮单元。有框架→每个小爽点都直奔大高潮→读者有很强期待感。无框架→情节像碎片拼合→读者第三天就不想看。

# 二十一、情绪掌控：从YY核心到情绪设计

## 21.1 愤怒的香蕉：YY核心

**核心论断：YY的本质是情绪的波动。爽点从心而发，不从分析和结构而发。**

创作方法论：
1. 研究一切心情的细节变化——看热血书心里翻涌的瞬间，记住它反复咀嚼
2. 不研究文法只研究情绪传递——抓住情绪后自然找到手段传递
3. 代入所有角色——"我代入每一个角色开始演说一句话"
4. 挖掘禁忌心情——人类内心不敢说的，写出来读者会非常爽
5. 合理性是关键——同样心情历史背景也可以未来背景也可以，只要抓住核心

**爽点本质**：不在于升级本身，而在于升级后延伸出来的效果——别人说你年轻有为、老大巴结你、有人求你捞他。

> "你拿着刀腰斩一百个敌人读者看了也就那么回事，你一砖头砸在敌人脑门上砰的一下——读者反而会被震慑。"

## 21.2 盖世巨星：情绪设计情节法

**核心公式：情绪波动 + 对比反差 = 完整的高潮情节。**

**情绪对比五维度**：时间、地点、人物、事件、物品。

> "你只要拉动了情绪，高潮爆发出来能解决拉动的情绪，那就是爽点。否则就是毒点。"

## 21.3 情绪调动与装逼打脸的联动

- 十里剑神：多角度全方位立体装逼打脸开花
- 老鹰吃小鸡：大家一起装逼，主角拿好处被称赞，配角成笑料
- **愤怒释放法**：苦大仇深→调怒值到巅峰→期待感拉满→逐个报仇→怒释放→酣畅淋漓

# 二十二、人设进阶：反差、特点与灵魂注入

## 22.1 毒奶巨：反差快速立人设

**核心方法：从极致的反差让角色给人眼前一亮。**

示例：冰山女总裁→背地里的抖M / 柔柔软软的妹妹→实则是绿茶

## 22.2 养神：人设是书的灵魂

**核心论断：人设是书的灵魂。读者喜欢你这个人设，就算剧情烂也关系不大。**

鲜明人设：沈幼楚、萧容鱼、许七安、李长寿、李淳罡——极度鲜明到不再是角色，而是活在读者心里。

**写好人设的三步**：
1. 先让这个人物在自己心里活过来
2. 以自身视角代入人设，用其性格习惯思维逻辑开展剧情
3. **加入特点**：所有书女主都是可爱→加入"一下雨就悲伤抱着小腿蜷缩在床上"（因为父母在雨夜被撞死）
4. 人设塑造起来后不要轻易破坏

## 22.3 翅膀巨：角色的灵魂

**核心论断：千订到万订的差距在于小说欠缺了情感灵魂。**

每一个角色有了自己的灵魂→读者因为角色一举一动产生牵挂→订阅爆炸一波。

> "写小说首先要感动自己，才能感动读者。"
> "如果订阅卡在5000-6000，那就必须要在小说中加入情感。"

# 二十三、高级结构：对比、铺垫与心理补偿

## 23.1 盖世巨星：对比理论

**核心论断：一段情节的核心就是找或推敲出对比点。对比越多越丰富，高潮越爽。**

对比五维度：时间、地点、人物、事件、物品。对比一定要与高潮一致。

**错误对比（拉偏仇恨）**：女友找了个更帅的→主角暴富没用。打篮球时对方说你颜值低→赢球解决不了。

## 23.2 铺垫 = 润滑剂

**前置无敌条件**：装逼场景开始前，必须把道具/人物/实力都准备好。主角已经小无敌，读者才能轻松看主角装逼。

**铺垫要写什么**：与接下来的装逼相辅相成。你想装金钱逼，就铺垫反派拜金看不起人。

多层次铺垫+多层次报复=多层爽。

## 23.3 丧泽巨：读者心理补偿

**核心概念：主角主动帮助人时，需要对读者进行心理补偿。**

**补偿方法一：交易化**——主角帮女主→要钱（两千万）→补偿到位。
**补偿方法二：情绪替代**——死战不能杀也不能输→杀另一个重要敌人。
**补偿方法三：道德补偿**——主角道德亏欠→让配角更可恶让读者站主角这边。

> "写网络小说，本质上是在讨好大部分读者。"

# 二十五、飞卢实战

## 25.1 飞卢写书四原则

1. 多装逼少打脸
2. 少写无脑剧情（大街上跳出富二代嘲讽）
3. 多一些自己的剧情（流程不能烂大街）

## 25.2 连环爽+创意法

**连环爽八字诀：情理之中，意料之外。** 像抖音一样几十秒一个新刺激。

**脑洞拓展法**：精髓=拓展。时间、空间、背景。万物皆可扩展脑洞。

**爽点放大**：国术打败外国人→放大到中华民族不可战胜→教育经济军事改变→国内外轰动→上升到家国情怀。

## 25.3 十九巨：核心卖点=主线

**核心卖点就是贯穿全文的东西，就是主线。** 工作流：把卖点包装成爽点→通过剧情包装→简化到一个词=卖点爽点。

# 二十七、大神分享精华

> 来源：指南大神分享文件 + 原归总

## 27.1 逐没：矛盾冲突设计

**五大矛盾类型**：
1. 竞争和比赛（有名利就有矛盾）
2. 内心意愿和理想（主角强烈想要→有人跳出来阻挠）
3. 打破规则（与规则维护者冲突）
4. 价值观差异（不可调和的矛盾）
5. 因果关系（爱恨情仇）

**矛盾运用三原则**：迎合读者心理 / 矛盾升级 / 矛盾服务于主线（至关重要——不能为了制造矛盾而引出矛盾）

## 27.2 矛盾设计的总原则

**别让主角成长得太顺。** 玄幻小说矛盾 = "获美、得宝、打架、砍人"。

## 27.3 抓住读者的心——五大满足

| 满足类型 | 对应行为 |
|---------|---------|
| 满足阅读洁癖 | 没有基础性错误 |
| 满足阅读量 | 更新速度快 |
| 满足阅读习惯 | 更新时间稳定 |
| 满足好奇心理 | 学会掐高潮 |
| 满足猎艳心理 | 适当的艳遇/感情线 |

**糖葫芦式写法**：一个情节=一粒糖葫芦，用主线这根竹签串起来。不要写树枝分叉式。

**代入感维持**：尽量以单主角成长描绘，不要让副角戏份超过主角，不要让主角长时间不出现。

# 三十一、补充：开篇、签约与冲榜实战

> 来源：补充读取——准备篇/爆款方法/大纲教程

## 31.1 冲榜策略

**新书榜积分公式**：当周推荐票×20 + 当周点击数 + 总收藏数×1/5

**分段变速更新策略**：入库初期快冲3万字 → 上榜稳固期放慢 → 冲榜无望正常更 → 下榜临界点再放慢延长上榜时间

## 31.2 上架前后吸引力提升

**沙漠之水理论**：沙漠里最贵的是水，比水更贵的是"你渴了但别人瓶子里有水"。上架前掐好高潮点。

**耳光理论**：前几下打人最痛，一直打会麻木；停下来抹药水、吹风，再突然用力打，更痛。

**饥饿-饱食循环**：吊胃口→喂饱→消化→重新吊→再喂饱。一直高潮=读者阈值提高=失去期待感。

**爆发促销心理**：稳定日更3章 vs 日更2章+间隔爆发，总量相同但后者成绩更好。

## 31.3 签约核心规则

**签约分级**：通过→修改待定→枪毙。终审过稿经编辑安排签约。

**大纲=事件而非描写**：第1-2章至少写3个事件支撑才是过稿标准。最快过稿：1-2章写3个环环相扣的事件。

**17K签约三要素**：设定（简单明了等级分明）+ 主角（人类/人形，男频必须男主）+ 主线剧情（升级流/都市暧昧流/都市装逼流）

**内签黄金公式**：简练直白+直奔主线+情节不拖沓=高签约概率

## 31.4 编辑视角

**编辑签人不签书**：签约不是因为你当前这本书行，而是觉得你这个作者有潜力。

**三秒生死法则**：编辑3秒决定拒签。开头=省掉一切废话。

**编辑关注公式**：读者轨迹+作者自身网站行为轨迹=点亮编辑关注。真正首次出现在编辑面前的作品，都是先经过读者筛选过的。

# 三十二、补充：大纲技法进阶

## 32.1 大纲制作六步法

## 32.3 大纲阶梯递进法

笼统大纲(几千字) → 分卷大纲(每个大事件分卷细化) → 分卷详纲(大事件前后的小事件+人际关系)

用树形结构管理大纲：方框记录关键情节节点，可展开备注，用线连接前后关系。所有伏笔、支线都围绕主线有机添加。

# 三十三、补充：人物塑造进阶

> 来源：补充读取——人物塑造/大神分享

## 33.1 人物三面呈现法

职业生活(与主线交叉) + 个人生活(朋友/家人关系) + 私生活(内心独白) ——三者缺一不可。

## 33.2 配角塑造法则

**配角行为法**：表现配角的性格用他的习惯、行为、动作、表情、语言，**不要用心理描写**。

**配角脸谱化≠缺点**：从头到尾一个性格，不要让配角"霸道中带温柔"——读者记不住复合性格的配角。

## 33.3 对话三元素

对话必须有：自我(自我意识) + 情绪(表达于外的感觉) + 对他(一定有个主题)。缺一=对话如嚼蜡。

## 33.4 反派塑造黄金法则

**大忠大义的反面人物通常比一个邪恶的反角更能打动人。** 反派=任务的发布者——没有深入人心的反角就没有更吸引人的主角。

**写人不是写神**：写的是人，不是精灵或神。

## 33.5 作者中立立场

作者就应该站在一个中立的角度，不一味偏爱主角。最忌讳在文中夹带写手的个人因素。作家的旁白要尽量少。

## 33.6 女性角色塑造

**女主塑造"朋友法"**：把她当朋友，赋予你希望在朋友身上发现的品质。当她开始不受你控制时——恭喜你，写活了她。

**恋爱写作高级法则**：恋爱主体是人，不是行为。写恋爱=写人本身。色情内容泛滥=构思枯竭的典型表现。

**女性角色标签法**：每个女性配角=一个独特标签（弹琴/煮菜/素颜），每次出场强化标签。

## 33.7 人物辅助词汇法（罗霸道）

每个重要人物配一套专属辅助词汇。专属词汇反复使用→强烈印象→寥寥数语就能让读者记住一辈子。

# 三十四、补充：情节与节奏进阶

> 来源：补充读取——情节套路/主编指点/大神分享

## 34.1 创意翻新核心法

**创意翻新公式**：别人的创意+自己的加工处理=全新创意。不是抄袭照搬，是了解别人思想后进行加工。

**超能力+平凡职业反差法**：把修真者的能力放到普通职业——码头扛包/运动员/数学研究/围棋/行医/保镖/间谍……制造反差趣味。

## 34.2 情节单元自检工具

一句话概括→环境地点→涉及人物(关联+变化+冲击+好处)→故事开端(伏笔/悬念/动力)→发展阻力→故事高潮→结尾(主角变化+收获+他人改变+新悬念)

**情节驱动力二元模型**：主动型(主角主动探索) vs 被动型(主角被动卷入)

## 34.3 小剧情设计原则

每个小剧情解决一个小问题，但留下更大的悬念或通往更大的问题。小剧情=主线螺旋上升过程中的每一个台阶。

**有事则缓，无事则快**——与剧情相关详细写，吃喝拉撒略过。

## 34.4 节奏共识（多位大神共同结论）

| 来源 | 节奏公式 |
|------|---------|
| 月关 | 每章小亮点 + 三章小故事 + 3万字小高潮 + 30万字大高潮 |
| 罗霸道 | 前20万字起点低进展快，20万字后场面展开可放缓 |
| 骁骑校 | 波浪状起起伏伏，全是高潮=没有高潮 |
| 断刃天涯 | 3000-5000字立主角 + 3-5万字出第一个高潮 |

**共通结论**：高潮和低潮交替，压抑不超过三章，节奏比文笔重要。

## 34.5 伏笔收束法则

伏笔一定要有用，越在危急时刻出现越爽。收笔时机：十万字以内。情景对主角越不利，伏笔回收越爽。

## 35.1 军事小说六法则

1. 主体必须是解放军
2. **丰满胜利公式**：丰满人物+艰苦战斗+最终胜利+赢得尊严（不能轻松赢，不能最后失败，不能完美无缺）
3. 真实深挖——作者必须对解放军有足够了解
4. 政治一笔带过，爱情更加不需要
5. **人性化视角**：说大实话+淡化主旋律+避免高大全=引发共鸣
6. 军事主题必须有笑料调剂

## 35.2 官场小说六大爽点

招商对比爽+政治打败爽+惩治贪官爽+打黑除恶爽+商战爽+感情戏（网络小说核心=娱乐性=让读者看了"舒服"）

## 35.3 惊悚灵异核心法则

**离奇 > 恐怖**。读者排斥血腥恐怖（挖眼割喉），喜欢离奇探秘（古墓宝藏羊皮卷）。**写实基础上的离奇最有代入感**——牛魔王没人信，鬼打墙可能有真实基础。

## 35.4 玄幻YY七要素

奇遇、升级、泡妞、寻宝、发财、欺人、助人。**全用是SB，掌握1-2个即可成神。** 番茄只用2个（升级+欺人）。研究透彻=封推。

## 35.5 穿越文核心

**穿越=金手指，不只是噱头。** 穿越文开篇第一章就穿越成功。慢热（前三章还在现代）=签约失败。

## 35.6 不同类型难度分析

| 类型 | 难度 | 说明 |
|------|------|------|
| 奇幻 | 最低 | 自由度高，新人最佳 |
| 仙侠 | 低 | 需把握升级节奏 |
| 都市 | 中等 | 简单入门，写好最难 |
| 历史 | 高 | 需知识+古人尊重+YY合理 |
| 军事 | 高 | 读者中军迷比例高 |
| 科幻/推理 | 最高 | 读者少，逻辑要求高 |

**核心原则**：不了解专业知识可以写（读者也不懂），但不感兴趣绝对不行。

# 三十六、补充：悬疑与悬念技法

> 来源：补充读取——悬疑剧情/烟枪谈悬念

## 36.1 63种悬念套路速查

信息不对称(卧底内鬼)、死亡谜题、身份危机、预言成真、道德困境、记忆碎片、时空穿越……
- **反常即疑点法**：性情大变/暴富/行为古怪=有异常
- **遗言钩子法**：死者临死前留下含义模糊的遗言→贯穿全文线索
- **碎片拼图法**：记忆错乱/失忆+碎片化记忆=惊天秘密
- **罗生门法**：多人说法自相矛盾，真伪难辨

## 36.2 悬念三类型

**解谜型**（读者和主角都不知道）→ **紧张型**（读者知道，主角不知道）→ **结果型**（结果未知）

## 36.3 悬念组合技法（烟枪）

**横向悬念线**：悬念→解开→引出下一个悬念→一环扣一环形成全文悬念链条

**纵向悬念线**：N个基本悬念(建议2-3个)在同一时间/事件上同时爆发=惊心动魄

**悬念节制原则**：悬念不是越多越好，为提升吸引力胡乱堆砌=作茧自缚

**脱衣舞隐喻**：小说悬念如脱衣舞——过程比结果更有意思。每一次暴露都预示着还有更多的要暴露。

# 三十七、补充：开篇六法与打斗七式

> 来源：补充读取——六种开头方法/打斗七公式

## 37.1 六种故事开头方法

| 方法 | 核心 | 适用 |
|------|------|------|
| 平铺直述式 | 按时间顺序，标签法则让平铺反而有吸引力 | 常规 |
| 场景式 | 特定人物+时间地点+未解矛盾/难题 | 强情节 |
| 悬念式 | 困境/倒叙/反差三种，设置谜与钩子 | 悬疑推理 |
| 对比渲染式 | 两种相反的人事物并举 | 强对比题材 |
| 描述道具式 | 展示起线索作用的物件，充满暗示 | 伏笔型 |
| 引用式 | 假借故事外但与故事相关的内容 | 特殊风格 |

**悬念设置终极标准**：意料之外、情理之中。读者不用猜就知道结果=吸引力大打折扣。

## 37.2 打斗七段结构

开场(身法出场+全场沸腾)→前奏(暗暗下决心+防御奇招)→敌人塑造(被打压+新仇旧恨)→对打(被压制+观众聚焦)→反转点(对手出大招+主角有资格接)→高潮爆发(放新绝招+与巅峰平手)→结局(确认实力+名声传播)

**打斗观众反应线**：沸腾→聚焦→压抑→心跳加速→震惊→兴奋→称王称霸的感叹

## 37.3 打斗描写技术要点

- 开打时不写心理描写，结束后展现气场时再写反派心理
- **越直接越过瘾，越暴力越兴奋**
- 通过时间/距离制造紧张感："一秒，五十米，二秒，三十米……近了，出拳！"
- 段落分开，制造"停顿感"=时间落差

# 三十八、补充：十大忌讳与防扑街指南

> 来源：补充读取——新人忌讳/枪毙理由

## 38.1 新人十大忌讳及精确流失率

| # | 忌讳 | 读者流失 |
|---|------|---------|
| 1 | 正文夹杂作者碎碎念/括号互动 | 30% |
| 2 | **开头多角色名轰炸** | **70%** |
| 3 | 2个以上角色用生僻字名 | 40% |
| 4 | 开头长篇幅描写时代背景 | 40% |
| 5 | 频繁多视角切换 | 40% |
| 6 | 主角言行与设定严重不符 | 75%成年读者 |
| 7 | **主角说话幼稚/碎碎念装可爱** | **80%成年读者** |
| 8 | **自恋式外貌描写** | **80%** |
| 9 | 过多笔墨描写非重要环境 | 35% |
| 10 | 反复大量描写角色外貌 | 30% |

## 38.2 枪毙一本书的理由（正文八毒）

标点毒(通篇一个逗号)→句式毒/装B毒(故作深沉的句子)→**逻辑毒**(80%网文都存在)→拔高毒(一出场就牛X无比)→雷同毒(被雷劈穿越=雷同的"雷")→白开水毒(流水账无详略)→牛屎堆毒(不分段)→甩文毒(故意用生冷字考验读者)

**作者尊严法则**：自己都无法说清楚的句子不要塞进作品——对读者的不尊重，缺乏作者尊严的表现。

## 38.3 开篇检查清单

内容安全红线：涉黄(性行为/器官/心理涉及一个即屏蔽)、涉政(违反宪法、危害统一、文革钓鱼岛台湾)、抄袭(永久禁止)、一书多签(严重)。

# 四十、补充：金手指理论与日常素材

> 来源：补充读取——金手指/日常生活/借鉴素材

## 40.1 金手指终极定义

**预埋一个巨大的不合理(让读者接受)→用这个不合理解释一切后续不合理=自圆其说。** 一本小说好不好看，首先是要合理——哪怕是歪理也是理，只要你能说通就行。

**金手指使用节制**：开头三章(必须用)→之后有需要再用→不要两章三章就用一次。

**金手指强度**：不够用(憋屈) < 适度(最佳) < 过度(恶心)

## 40.2 日常生活素材提取

**提取公式**：观察身边人→分类提取共性特征→重组为角色原型

**普通人不普通化公式**：普通人设+极致冲突场景=不普通的故事。如：公交车司机+时间循环爆炸案(《开端》)

**灵感收集三渠道**：填素材(读书看报/社会热点/电影电视) + 填生活(七大姑八大姨和邻居) + 填转述(拉爆款讲给别人)

## 40.3 读者爽点四源泉（佛家四苦的反面）

不满得到发泄(痛骂无良老板) + 期待得到满足(报复小人) + 被赞叹(默默无闻被赏识) + 震撼他人(小人物登上舞台释放光芒)
