# 嘎子带货 — 情节模式

## 高频故事结构

### 结构1：翻车三部曲（适合短篇/单元剧）
```
第1章：主角带着童年滤镜开始直播
第2章：产品/话术出问题，弹幕开始质疑
第3章：翻车实锤，全网传播，主角回应
```

### 结构2：潘嘎式讽刺（适合中篇）
```
前期：前辈义正言辞劝诫主角
中期：前辈自己下场做同样的事
后期：前辈翻车比主角还惨，讽刺闭环
```

### 结构3：过气逆袭（适合长篇）
```
第一幕：童星成年，无戏可拍，被迫带货
第二幕：直播翻车，全网嘲讽，跌入谷底
第三幕：低谷反思，找到新方向
第四幕：绝地反击，用真诚/专业重新赢得认可
```

### 结构4：行业揭秘（适合现实向）
```
线1：主角的带货生涯（明线）
线2：直播行业的潜规则（暗线）
交汇：主角的翻车暴露了行业问题
```

## 爽点分布节奏

| 章节区间 | 爽点类型 | 密度 |
|----------|---------|------|
| 1-5章 | 滤镜碎裂、初入行业的荒诞感 | 每章1个小爽点 |
| 6-15章 | 连续翻车、前辈打脸、弹幕审判 | 每2章1个大爽点 |
| 16-30章 | 低谷、温情、行业反思 | 每3章1个情绪高点 |
| 31-50章 | 绝地反击、真相揭露、逆袭 | 每2章1个爽点 |

## 常见冲突类型

| 冲突类型 | 表现形式 |
|---------|---------|
| 情怀 vs 商业 | 想保持童年形象 vs 需要卖货赚钱 |
| 真诚 vs 套路 | 想说实话 vs MCN要求背话术 |
| 个人 vs 平台 | 想自由播 vs 平台限流/封号 |
| 过去 vs 现在 | 童年英雄 vs 成年小丑 |
| 理想 vs 现实 | 想当好演员 vs 没戏可拍 |
| 家人 vs 事业 | 家人反对 vs 需要收入 |

## 典型场景模板

### 直播间翻车场景
```
开场：主角热情开场，"家人们今天给大家带来..."
发展：产品出现质量问题/话术被质疑
高潮：弹幕爆炸，主角情绪失控
收尾：被迫下播/强行解释/直播间被封
```

### 前辈劝诫场景
```
开场：前辈以"过来人"身份联系主角
发展：语重心长地劝主角不要带货
转折：前辈话锋一转，开始推销自己的产品
反转：主角发现前辈也在卖同样的东西
```

### 低谷反思场景
```
开场：主角独自在家，刷到自己的负面新闻
发展：回忆童年拍戏的纯真时光
转折：家人/朋友的一个电话或到访
收尾：做出新的决定
```
