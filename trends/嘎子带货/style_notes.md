# 嘎子带货 — 风格备注

## 适合的文风

### 推荐：讽刺轻喜剧
- **基调**：7分搞笑 + 3分心酸
- **节奏**：快节奏、密集笑点、每章一个名场面
- **参考**：《赘婿》前期的讽刺感、《万万没想到》的荒诞感
- **核心**：笑着笑着就沉默了

### 也适合：现实向都市文
- **基调**：5分现实 + 3分温情 + 2分讽刺
- **节奏**：中等节奏，注重人物内心
- **参考**：《我是余欢水》的底层挣扎感
- **核心**：不是在嘲笑他，而是在看他怎么爬起来

## 对话风格

### 推荐：口语化 + 网络用语
- 大量使用直播间行话（家人们、上链接、321）
- 网友弹幕/评论穿插（增加真实感）
- 方言/口癖（如嘎子的河北口音感）
- 短句对轰，节奏快

### 示例对话风格
```
"家人们！今天嘎子给你们带来一个好东西！"他对着镜头比了个大拇指，
弹幕瞬间飘过一片"嘎子偷狗"。

"不是，你们能不能别提偷狗的事了？"他脸色一僵，
"今天这个是真的好东西，嘎子用自己的人格担保！"

"你还有人格呢？"助理小声嘀咕。

"闭嘴。"
```

## 叙事视角推荐

### 推荐：第三人称有限视角（跟随主角）
- 读者跟着主角体验直播翻车的全过程
- 可以穿插主角的内心吐槽（增加喜感）
- 偶尔切到弹幕/网友评论视角（增加信息量）

### 也适合：多视角
- 主角视角 + MCN老板视角 + 铁粉视角 + 黑粉视角
- 每个视角都有不同的信息差，制造讽刺效果

## 特殊写作技巧

### 1. 弹幕体穿插
在正文中穿插直播间弹幕，增加真实感和喜感：
```
他举起酒瓶："这酒，嘎子亲自品过！"
弹幕：
[用户A] 品过？品过假酒吧
[用户B] 嘎子偷狗喝的酒吧
[铁粉小李] 支持嘎子！
[用户C] 家人们快跑
```

### 2. 新闻/热搜体
用新闻报道/热搜标题推进剧情：
```
#嘎子谢孟伟穿警服直播# 冲上热搜第三。
#潘嘎之交# 再次被翻出来。
#童年滤镜碎了# 阅读量破3亿。
```

### 3. 评论区体
模拟网友评论，展现舆论：
```
热评第一："从抗日小英雄到直播间小丑，嘎子用20年完成了蜕变。"
热评第二："潘子，你把握住了吗？"
热评第三："心疼嘎子，但更心疼买了他酒的家人们。"
```

### 4. 时间线跳跃
用"XX天前""XX年后"快速推进时间，适合展现人物弧线：
```
2003年。8岁的谢孟伟站在片场，导演喊"过！"。
全场鼓掌。
2025年。30岁的谢孟伟站在直播间，运营喊"上链接！"。
弹幕爆炸。
```

## 写作禁忌

- ❌ 不能用过于文学化的语言（这不是散文，是网文）
- ❌ 不能节奏太慢（读者期待密集爽点）
- ❌ 不能纯搞笑没有情感内核（笑完就忘，没有回味）
- ❌ 不能回避行业的残酷性（读者想看真实感）
- ❌ 不能美化造假行为（讽刺可以，但立场要正）
