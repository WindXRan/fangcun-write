# 嘎子带货 — 核心机制/爽点

## 爽点循环

```
童年滤镜碎裂 → 荒诞带货名场面 → 被揭穿/翻车 → 试图挽回 → 再次翻车（循环）
```

升级版：
```
过气困境 → 直播翻车 → 全网嘲讽 → 低谷反思 → 绝地反击（逆袭线）
```

## 高频反转模式

### 1. 滤镜碎裂型
- 观众带着童年滤镜进入直播间
- 产品/话术/行为打破滤镜
- 经典台词："家人们，我嘎子什么时候骗过你们？"

### 2. 前辈翻车型
- 资深前辈义正言辞劝诫后辈
- 前辈自己也下场做同样的事
- 讽刺点：道貌岸然的伪善

### 3. 翻车连锁型
- 一次带货翻车 → 被扒出更多黑历史 → 信任崩塌 → 账号被封
- 每次道歉都是下一次翻车的铺垫

### 4. 自嘲洗白型
- 被全网嘲讽后，主动玩梗自嘲
- 反而获得部分好感
- 但很快又因为新事件翻车

## 读者期待的"名场面"

| 场景 | 为什么爽 |
|------|---------|
| 直播翻车现场 | 产品造假被当场揭穿，弹幕爆炸 |
| 前辈劝诫后辈打脸 | "我劝你别做"→自己做了→被打脸 |
| 穿帮时刻 | 露出破绽的瞬间（如穿帮镜头、矛盾话术） |
| 弹幕审判 | 观众/网友的集体嘲讽，形成舆论风暴 |
| 绝地反击 | 低谷后真正悔改/逆袭，情绪释放 |
| 资本博弈 | 平台/MCN/品牌方的利益纠葛 |

## 节奏特点

- **快节奏**：每个事件都是独立爆点，适合短章节密集输出
- **单线为主**：主角的带货生涯为主线，穿插个人生活
- **情绪波动大**：笑→怒→悲→爽，循环往复
- **适合碎片化阅读**：每章一个名场面或翻车事件

## 禁忌/雷区

- ❌ 不能纯搞笑没有内核（需要有对行业的反思）
- ❌ 不能美化造假/欺骗行为
- ❌ 不能把主角写成纯反派（需要有人性弧线）
- ❌ 不能太贴近真实人物以至于变成诽谤（需要虚构化处理）
- ❌ 不能回避"过气"的残酷性（读者想看真实感）
